# 《阿含經》白話翻譯與數位典藏計畫

本計畫旨在透過現代 AI 技術與網頁平台，將深奧的佛典核心《阿含經》轉化為易懂的白話文字，並提供舒適的數位閱讀環境，讓原始佛教的智慧能觸及更多現代讀者。

---

## 專案初衷
* **AI 助譯**：本站內容主要由人工智慧 (AI) 進行初步白話編譯與註解。
* **消除門檻**：打破古文經卷的隔閡，提供淺顯易懂的註解。
* **無私分享**：法寶應廣為流通，不應受限於科技或金錢。

**請注意：**計畫目的只為了將阿含經作白話翻譯，讓使用者讀經時，可以輕鬆閱讀，不會在遇到專有名詞時，還要花時間再去做查詢的動作，但翻譯內容不若專業佛經翻譯般嚴謹，類似的原文可能會有不統一的翻譯，畢竟硬體有限，無法RAG所有內容，達到翻譯都統一，但意義會傳達到。「法尚應捨，何況非法」，內容應該比文字的修飾更重要，而從中能得到什麼又比內容更重要。所以這個網站是基於這個目的為出發點，

## 經文分類
* **雜阿含經**：收集各類簡短且核心的教法。
* **中阿含經**：篇幅適中，深入探討義理。
* **長阿含經**：內容較長，多具備敘事背景。
