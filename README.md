# 《阿含經》白話翻譯與數位典藏計畫

本計畫旨在透過現代 AI 技術與網頁平台，將深奧的佛典核心《阿含經》轉化為易懂的白話文字，並提供舒適的數位閱讀環境，讓原始佛教的智慧能觸及更多現代讀者。

---

## 🎯 專案初衷
* **AI 助譯**：本站內容主要由人工智慧 (AI) 進行初步白話編譯與註解，並輔以人工校對。
* **消除門檻**：打破古文經卷的隔閡，提供淺顯易懂的註解。
* **無私分享**：法寶應廣為流通，不應受限於科技或金錢。
