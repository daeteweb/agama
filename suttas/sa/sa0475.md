# 雜阿含經 第 475 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，世尊告諸比丘：「毘婆尸如來未成佛時，獨一靜處，禪思思惟，作如是觀，觀察諸受。云何為受？云何受集？云何受滅？云何受集道跡？云何受滅道跡？云何受味？云何受患？云何受離？如是觀察，有三受——樂受、苦受、不苦不樂受。觸集是受集，觸滅是受滅。若於受愛樂、讚歎、染著、堅住，是名受集道跡。若於受不愛樂、讚歎、染著、堅住，是名受滅道跡。若受因緣生樂喜，是名受味，若受無常變易法，是名受患。若於受斷欲貪、越欲貪，是名受離。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如毘婆尸佛，如是式棄佛、毘濕波浮佛、迦羅迦孫提佛、迦那迦牟尼佛、迦葉佛，及我釋迦文佛，未成佛時思惟觀察諸受，亦復如是。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「毘婆尸如來未成佛時，獨一靜處，禪思思惟，作如是觀，觀察諸受。**
　　「毘婆尸如來在還未成佛之前，在一個安靜的地方獨處，進行禪定思考，並進行這樣的觀察，觀察各種受（感受）。

**◈ 云何為受？云何受集？云何受滅？云何受集道跡？云何受滅道跡？云何受味？云何受患？云何受離？**
　　什麼是受（感受）？受（感受）是如何產生的（集）？受（感受）是如何停止的（滅）？導致受（感受）產生的路徑（道跡）是什麼？導致受（感受）停止的路徑（道跡）是什麼？受（感受）的滋味（吸引力）是什麼？受（感受）的缺陷與危險（患）是什麼？如何脫離受（感受）？

**◈ 如是觀察，有三受——樂受、苦受、不苦不樂受。**
　　如此觀察後，發現受（感受）有三種：快樂的感受、痛苦的感受，以及既不痛苦也不快樂的感受。

**◈ 觸集是受集，觸滅是受滅。**
　　因為有了觸（感官與對象的接觸），所以產生了受（感受）；當觸（感官與對象的接觸）停止，受（感受）也就隨之停止。

**◈ 若於受愛樂、讚歎、染著、堅住，是名受集道跡。**
　　如果對感受產生貪愛、讚美、執著並堅定地停留其中，這就叫做導致受（感受）持續產生的路徑。

**◈ 若於受不愛樂、讚歎、染著、堅住，是名受滅道跡。**
　　如果對感受不產生貪愛、不讚美、不執著且不堅定停留，這就叫做導致受（感受）停止的路徑。

**◈ 若受因緣生樂喜，是名受味，若受無常變易法，是名受患。**
　　如果感受因為特定的因緣而產生快樂與喜悅，這就叫做受（感受）的滋味；如果意識到感受是無常（隨時在改變）且變易的現象，這就叫做受（感受）的缺陷與危險。

**◈ 若於受斷欲貪、越欲貪，是名受離。」**
　　如果能斷除對感受的貪欲，超越對感受的渴求，這就叫做脫離受（感受)。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如毘婆尸佛，如是式棄佛、毘濕波浮佛、迦羅迦孫提佛、迦那迦牟尼佛、迦葉佛，及我釋迦文佛，未成佛時思惟觀察諸受，亦復如是。**
　　正如毘婆尸佛，以及式棄佛、毘濕波浮佛、迦羅迦孫提佛、迦那迦牟尼佛、迦葉佛，以及我釋迦文佛，在還未成佛之前，對於各種受（感受）的思惟觀察，也同樣是這樣。

### [詞語註解（解釋重要佛教概念）]
1. **受 (Vedanā)**：指感官接觸對象後產生的情緒反應，分為苦、樂、不苦不樂三種。它是五蘊中的第二蘊。
2. **觸 (Phassa)**：指「感官」、「感官對象」與「識（意識）」三者同時具足的接觸狀態。沒有觸，就沒有受。
3. **集 (Samudaya) 與 滅 (Nirodha)**：集指現象的產生原因；滅指現象的消失或停止。
4. **味 (Rasa)、患 (Ādīnava)、離 (Nissaraṇa)**：這是原始佛教分析任何現象的標準邏輯框架。首先分析其吸引力（味），接著分析其潛在的危險或缺陷（患），最後尋找脫離該現象的方法（離）。
5. **道跡 (Magga/Path)**：指導致某種結果而產生的行為路徑或心理傾向。

### [現代實作指南]
本經提供了一套精準的「情緒管理與解構」操作流程，可將其轉化為以下生活實踐：

1. **標記感受（識別受）**：
   當強烈情緒出現時，不要立刻陷入情緒中，而是在心中標記：「現在有樂受」、「現在有苦受」或「現在是不苦不樂受」。將「我很痛苦」轉化為「有一個苦受在發生」，建立觀察者的距離。

2. **追溯觸點（分析集）**：
   思考這個感受是由什麼「觸」引起的？（例如：眼睛看到了某人的訊息 $\rightarrow$ 意識接收 $\rightarrow$ 產生憤怒的受）。意識到感受並非憑空而來，而是感官接觸的結果。

3. **辨識「味」與「患」**：
   - **分析味**：當我們追求快樂時，承認這種快樂的「滋味」確實吸引人。
   - **分析患**：立刻提醒自己，這種快樂是「無常變易」的。如果執著於此，當它消失時，必然會轉化為痛苦。

4. **實踐「離」的轉念**：
   當發現自己開始「讚歎」或「染著」於某種感受時，練習「不堅住」。告訴自己：「這個感受只是暫時的現象，不需要抓住它，也不需要推開它。」透過斷除對感受的貪著，達到心理上的自由。

### 段落大意 
本經記述佛陀說明過去諸佛（如毘婆尸佛等）在成道前，如何透過觀察「受（感受）」來達成覺悟。文中詳細分析了受的三種類型（苦、樂、捨），以及受的產生（由觸而生）與滅盡（由不執著而止）。透過對受的「滋味（吸引力）」、「缺陷（無常）」與「脫離（斷貪）」的邏輯分析，指導修行者如何從情緒的束縛中解脫。
