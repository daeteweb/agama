# 雜阿含經 第 1045 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有相習近法。諦聽，善思，當為汝說。何等為相習近法？謂殺生者、殺生者習近，盜、婬、妄語、兩舌、惡口、綺語、貪、恚、邪見，各各隨類更相習近。譬如不淨物、不淨物自相和合。如是，殺生、殺生，乃至邪見、邪見自相習近。如是，比丘！不殺生、不殺生相習近，乃至正見、正見更相習近。譬如淨物、淨物自相和合，乳生酪，酪生酥，酥生醍醐，醍醐自相和合。如是，不殺、不殺更相習近，乃至正見、正見更相習近，是名比丘相習近法。」

佛說是經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有相習近法。**
　　那時，世尊告訴在場的眾位比丘說：「有一種『相習近法』（指事物因性質相近而相互吸引、聚集的法則）。

**◈ 諦聽，善思，當為汝說。**
　　請你們仔細聽，好好思考，我現在為你們解說。

**◈ 何等為相習近法？**
　　什麼是相習近法？

**◈ 謂殺生者、殺生者習近，盜、婬、妄語、兩舌、惡口、綺語、貪、恚、邪見，各各隨類更相習近。**
　　指的是殺生的人會與殺生的人聚集在一起；同樣地，盜竊、淫蕩、妄語（說謊）、兩舌（挑撥離間）、惡口（辱罵）、綺語（花言巧語）、貪婪、嗔恚（憤怒）、邪見（錯誤的見解），每一類惡行的人都會與同類的人相互吸引而聚集。

**◈ 譬如不淨物、不淨物自相和合。**
　　這就像是不潔淨的物質與不潔淨的物質會自然地結合在一起。

**◈ 如是，殺生、殺生，乃至邪見、邪見自相習近。**
　　同樣地，殺生的人會聚集在殺生的人身邊，直到邪見的人聚集在邪見的人身邊。

**◈ 如是，比丘！不殺生、不殺生相習近，乃至正見、正見更相習近。**
　　同樣地，比丘們！不殺生的人會與不殺生的人相互吸引，直到正見（正確的見解）的人與正見的人相互吸引。

**◈ 譬如淨物、淨物自相和合，乳生酪，酪生酥，酥生醍醐，醍醐自相和合。**
　　這就像是純淨的物質與純淨的物質自然結合，例如牛奶產生酪，酪產生酥，酥產生醍醐（最精純的乳製品），而醍醐與醍醐會相互融合。

**◈ 如是，不殺、不殺更相習近，乃至正見、正見更相習近，是名比丘相習近法。」**
　　同樣地，不殺生的人與不殺生的人相互吸引，直到正見的人與正見的人相互吸引，比丘們，這就叫做『相習近法』。」

**◈ 佛說是經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **相習近法 (Samanatā/Association)**：指具有相似性質、傾向或心態的人或事物，在因緣作用下會自然地相互吸引並聚集在一起的法則。
2. **邪見 (Micchā-diṭṭhi)**：錯誤的見解，特別是指不承認因果業報、不認同四聖諦的認知，會導致錯誤的行為與痛苦。
3. **正見 (Sammā-diṭṭhi)**：正確的見解，指對四聖諦（苦、集、滅、道）的正確認知，是八正道之首，也是解脫的起點。
4. **醍醐**：古印度將牛奶經過多次精煉後得到的最高級精華，在此經文中象徵修行達到最純淨、最高層次的境界。

### [現代實作指南]
本經揭示了「環境與同類吸引」的心理與社會機制。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **社交圈審視（環境覺察）**：
   ➔ 觀察自己經常接觸的人。如果周圍的人傾向於抱怨、批評、貪婪或持有偏激的邪見，根據「相習近法」，自己很容易在潛移默化中被同化。
   ➔ 意識到「我所吸引的人，反映了我目前的心態」。

2. **主動建立「正見」場域**：
   ➔ 刻意尋找「善知識」（Kalyāṇa-mitta），即那些具有正見、慈悲且能給予正向指引的人。
   ➔ 透過閱讀正向典籍、參加正念修行社群，將自己置於「淨物」的環境中，讓正向的習慣（如不殺生、正語）自然地被強化。

3. **由淺入深的精進（乳 ➔ 酪 ➔ 酥 ➔ 醍醐）**：
   ➔ 認清修行是漸進的。先從簡單的「不殺生」、「不妄語」等基礎戒律開始（如牛奶），逐步深化為內心的平靜與智慧（如醍醐）。
   ➔ 不要期待立即達到最高境界，而應專注於與「正確的類別」相習近，讓正向的因緣自然推動成長。

### 段落大意 
本經說明「相習近法」，即性質相近者會相互吸引。惡行者（如殺生、邪見者）會聚集在一起，如同不淨物之和合；而行善者（如不殺生、正見者）則會相互吸引，如同純淨物質由乳精煉至醍醐的過程。佛陀以此勉勵比丘應遠離惡友，親近正見之人，以促進修行的純淨與進步。
