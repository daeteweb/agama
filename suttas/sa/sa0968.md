# 雜阿含經 第 968 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，給孤獨長者日日出見世尊，禮事供養，給孤獨長者作是念：「我今出太早，世尊及諸比丘禪思未起，我寧可過諸外道住處。」即入外道精舍，與諸外道共相問訊慰勞已，退坐一面。

時，彼外道問言：「長者！汝見沙門瞿曇，云何見？何所見？」

長者答言：「我亦不知云何見世尊，世尊何所見。」

諸外道言：「汝言見眾僧，云何見眾僧？眾僧何所見？」

長者答言：「我亦不知云何見僧，僧何所見。」

外道復問：「長者！汝今云何自見？自何所見？」

長者答言：「汝等各各自說所見，然後我說所見，亦不難。」

時，有一外道作如是言：「長者！我見一切世間常，是則真實，餘者虛妄。」

復有說言：「長者！我見一切世間無常，此是真實，餘則虛妄。」

復有說言：「長者！世間常無常，此是真實，餘則虛妄。」

復有說言：「世間非常非無常，此是真實，餘則虛妄。」

復有說言：「世有邊，此是真實，餘則虛妄。」

復有說言：「世無邊，此是真實，餘則虛妄。」

復有說言：「世有邊無邊。」

復有說言：「世非有邊非無邊。」

復有說言：「命即是身。」

復有說言：「命異身異。」

復有說言：「如來死後有。」

復有說言：「如來死後無。」

復有說言：「如來死後有無。」

復有說言：「如來死後非有非無，此是真實，餘則虛妄。」

諸外道語長者言：「我等各各已說所見，汝復應說汝所見。」

長者答言：「我之所見真實、有為、思量、緣起，若復真實、有為、思量、緣起者，彼則無常，無常者是苦。如是知已，於一切見都無所得。如汝所見：『世間常，此是真實，餘則虛妄。』者，此見真實、有為、思量、緣起；若真實、有為、思量、緣起者，是則無常，無常者是苦。是故汝等習近於苦，唯得於苦，堅住於苦，深入於苦。如是汝言世間無常，此是真實，餘則虛妄。有如是咎。」

世間常、無常、非常非無常，世有邊、世無邊、世有邊無邊、世非有邊非無邊，命即是身、命異身異，如來死後有、如來死後無、如來死後有無、如來死後非有非無，此是真實，餘則虛妄，皆如上說。

有一外道語給孤獨長者言：「如汝所說，若有見彼，則真實、有為、思量、緣起者，是無常法，若無常者是苦。是故，長者所見亦習近苦，得苦，住苦，深入於苦？」

長者答言：「我先不言所見者，是真實、有為、思量、緣起法，悉皆無常，無常者是苦，知苦已，我於所見無所得耶？」

彼外道言：「如是，長者！」

爾時，給孤獨長者於外道精舍伏彼異論，建立正論，於異學眾中作師子吼已，往詣佛所，稽首禮足，退坐一面，以向與諸外道共論事向佛廣說。

佛告給孤獨長者：「善哉！善哉！宜應時時摧伏愚癡外道，建立正論。」

佛說是語已，給孤獨長者歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，給孤獨長者日日出見世尊，禮事供養，給孤獨長者作是念：「我今出太早，世尊及諸比丘禪思未起，我寧可過諸外道住處。」即入外道精舍，與諸外道共相問訊慰勞已，退坐一面。**
　　那時，給孤獨長者每天都出來拜見世尊，進行禮拜與供養。有一天，給孤獨長者心裡想：「我今天出發太早了，世尊以及眾位比丘可能還在禪定思考中尚未起身，我不如先去拜訪那些外道（非佛教的修行者）的住處。」於是，他進入外道的精舍，與那些外道互相問候慰勞之後，退到一邊坐下。

**◈ 時，彼外道問言：「長者！汝見沙門瞿曇，云何見？何所見？」**
　　這時，那些外道問道：「長者！您看待沙門瞿曇（佛陀）的看法如何？您在他身上看到了什麼？」

**◈ 長者答言：「我亦不知云何見世尊，世尊何所見。」**
　　長者回答說：「我也不知道該如何描述對世尊的看法，也不知道世尊看到了什麼。」

**◈ 諸外道言：「汝言見眾僧，云何見眾僧？眾僧何所見？」**
　　外道們又說：「您說您見過眾僧，那麼您對眾僧的看法如何？眾僧看到了什麼？」

**◈ 長者答言：「我亦不知云何見僧，僧何所見。」**
　　長者回答說：「我也不知道該如何描述對僧團的看法，也不知道僧團看到了什麼。」

**◈ 外道復問：「長者！汝今云何自見？自何所見？」**
　　外道再次問道：「長者！那麼您現在對您自己的看法如何？您自己看到了什麼？」

**◈ 長者答言：「汝等各各自說所見，然後我說所見，亦不難。」**
　　長者回答說：「請你們先各自說明自己的見解，然後我再說明我的見解，這樣並不困難。」

**◈ 時，有一外道作如是言：「長者！我見一切世間常，是則真實，餘者虛妄。」**
　　這時，有一位外道說：「長者！我看一切世間是恆常不變的，這才是真實的，其餘的看法都是虛假的。」

**◈ 復有說言：「長者！我見一切世間無常，此是真實，餘則虛妄。」**
　　另一個人說：「長者！我看一切世間是無常（變遷不恆定）的，這才是真實的，其餘的看法都是虛假的。」

**◈ 復有說言：「長者！世間常無常，此是真實，餘則虛妄。」**
　　又有人說：「長者！世間既是恆常的又是無常的，這才是真實的，其餘的看法都是虛假的。」

**◈ 復有說言：「世間非常非無常，此是真實，餘則虛妄。」**
　　又有人說：「世間既非恆常也非無常，這才是真實的，其餘的看法都是虛假的。」

**◈ 復有說言：「世有邊，此是真實，餘則虛妄。」**
　　又有人說：「世界是有邊界的，這才是真實的，其餘的看法都是虛假的。」

**◈ 復有說言：「世無邊，此是真實，餘則虛妄。」**
　　又有人說：「世界是沒有邊界的，這才是真實的，其餘的看法都是虛假的。」

**◈ 復有說言：「世有邊無邊。」**
　　又有人說：「世界既有邊界又沒有邊界。」

**◈ 復有說言：「世非有邊非無邊。」**
　　又有人說：「世界既非有邊界也非無邊界。」

**◈ 復有說言：「命即是身。」**
　　又有人說：「生命就是這個物質身體。」

**◈ 復有說言：「命異身異。」**
　　又有人說：「生命與物質身體是截然不同的兩者。」

**◈ 復有說言：「如來死後有。」**
　　又有人說：「如來在死後依然存在。」

**◈ 復有說言：「如來死後無。」**
　　又有人說：「如來在死後便不存在了。」

**◈ 復有說言：「如來死後有無。」**
　　又有人說：「如來在死後既存在又不存在。」

**◈ 復有說言：「如來死後非有非無，此是真實，餘則虛妄。」**
　　又有人說：「如來在死後既非存在也非不存在，這才是真實的，其餘的看法都是虛假的。」

**◈ 諸外道語長者言：「我等各各已說所見，汝復應說汝所見。」**
　　外道們對長者說：「我們已經各自說明了見解，現在您應該說明您的見解了。」

**◈ 長者答言：「我之所見真實、有為、思量、緣起，若復真實、有為、思量、緣起者，彼則無常，無常者是苦。如是知已，於一切見都無所得。如汝所見：『世間常，此是真實，餘則虛妄。』者，此見真實、有為、思量、緣起；若真實、有為、思量、緣起者，是則無常，無常者是苦。是故汝等習近於苦，唯得於苦，堅住於苦，深入於苦。如是汝言世間無常，此是真實，餘則虛妄。有如是咎。」**
　　長者回答說：「我的見解是：任何被認為是真實的看法，其實都是由條件組合而成（有為）、經過心智思量、且依因緣而生（緣起）的。而凡是真實存在、由條件組合、經過思量且依因緣而生的現象，必然是無常（變遷不恆定）的；而無常的現象必然會導致苦（不滿足與不安的狀態）。一旦這樣了解，就會發現對於任何一種固定的見解都無法得到永恆的滿足。例如你們認為『世間是恆常的，這才是真實的』，這個『見解』本身就是真實存在、由條件組合、經過思量且依因緣而生的；既然它是依因緣而生的，它就是無常的，而無常就是苦。因此，你們執著於這種見解，就是親近苦、獲得苦、堅守苦，並深入到苦之中。同樣地，如果你們說『世間是無常的，這才是真實的』，也會有同樣的過錯。」

**◈ 世間常、無常、非常非無常，世有邊、世無邊、世有邊無邊、世非有邊非無邊，命即是身、命異身異，如來死後有、如來死後無、如來死後有無、如來死後非有非無，此是真實，餘則虛妄，皆如上說。**
　　對於上述關於世間恆常或無常、有邊或無邊、生命與身體的關係，以及如來死後是否存在的所有見解，其分析邏輯都與前面所說的一樣。

**◈ 有一外道語給孤獨長者言：「如汝所說，若有見彼，則真實、有為、思量、緣起者，是無常法，若無常者是苦。是故，長者所見亦習近苦，得苦，住苦，深入於苦？」**
　　有一位外道對給孤獨長者說：「按照您的說法，如果任何見解都是真實、有為、思量且緣起的，那麼它就是無常法，而無常就是苦。那麼，長者您的見解不也同樣是在親近苦、獲得苦、堅守苦，並深入到苦之中嗎？」

**◈ 長者答言：「我先不言所見者，是真實、有為、思量、緣起法，悉皆無常，無常者是苦，知苦已，我於所見無所得耶？」**
　　長者回答說：「我剛才說的，是讓你們明白所有的『見解』都是真實、有為、思量且緣起的法，因此悉皆無常，而無常就是苦。但在我意識到這種苦之後，我對任何一種固定的見解都不再執著（無所得），這難道不是正確的嗎？」

**◈ 彼外道言：「如是，長者！」**
　　那位外道說：「確實如此，長者！」

**◈ 爾時，給孤獨長者於外道精舍伏彼異論，建立正論，於異學眾中作師子吼已，往詣佛所，稽首禮足，退坐一面，以向與諸外道共論事向佛廣說。**
　　那時，給孤獨長者在外道的精舍中擊破了他們的錯誤論點，建立了正確的論理，在異教徒之中展現了如獅子般威猛的真理宣告（師子吼）後，前往佛陀那裡，恭敬地頂禮佛足，然後退到一邊坐下，將剛才與外道討論的事情詳細地告訴佛陀。

**◈ 佛告給孤獨長者：「善哉！善哉！宜應時時摧伏愚癡外道，建立正論。」**
　　佛陀告訴給孤獨長者：「好極了！好極了！你應該時時摧毀愚癡外道的錯誤見解，建立正確的論理。」

**◈ 佛說是語已，給孤獨長者歡喜隨喜，作禮而去。**
　　佛陀講完這番話後，給孤獨長者心生法喜並隨喜讚嘆，向佛陀行禮後才離開。

### [詞語註解（解釋重要佛教概念）]
1. **有為 (Sankhata)**：指由多種條件（因緣）組合而成的現象。所有有為法都具有生、住、異、滅的過程，因此必然無常。
2. **緣起 (Paṭicca-samuppāda)**：指一切現象都不是獨立存在的，而是依賴於特定條件而生起。其核心邏輯是「此有故彼有，此生故彼生」。
3. **師子吼 (Sīhanāda)**：原指獅子吼聲威震山林，在佛教中比喻佛陀或其弟子以無畏、威嚴且不可撼動的真理論述，摧破外道的邪見。
4. **無所得 (Non-attainment/Non-clinging)**：在此指不對任何一種特定的、概念化的「見解」產生執著。因為任何概念化的見解都是有為法，執著於其中即是執著於無常，進而產生苦。

### [現代實作指南]
本經文提供了一個強大的邏輯工具，可用於處理現代生活中的「觀念衝突」與「認知執著」：

1. **識別「見解」的本質** ➔ 當我們在爭論某件事「對或錯」、「恆常或變遷」時，首先意識到：我的這個「看法」本身就是一個「有為法」。它是基於我過去的經驗、教育、情緒（因緣）而產生的思量。
2. **打破「正確性」的陷阱** ➔ 很多人追求「絕對正確的答案」。但本經告訴我們，只要這個答案是以「文字、概念、思維」形式存在的，它就是無常的。執著於「我才是正確的」這件事，本身就是一種「苦」的來源。
3. **從「對立」轉向「觀察」** ➔ 
   - **錯誤模式**：我的觀點 $\text{vs}$ 你的觀點 → 爭論 → 憤怒（苦）。
   - **實作模式**：觀察我的觀點 → 意識到它是緣起的 → 意識到它會改變 → 放下對該觀點的絕對執著 → 平靜。
4. **實踐「無所得」的心理狀態** ➔ 這不是指沒有想法，而是「有想法但不被想法綁架」。當你能對自己的見解保持一種「這只是目前的暫時認知」的開放態度時，你便能像給孤獨長者一樣，在面對不同意見時依然能保持從容且具說服力。
