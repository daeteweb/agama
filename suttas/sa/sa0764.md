# 雜阿含經 第 764 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「我今當說修八聖道。諦聽，善思。何等為修八聖道？是比丘修正見，依遠離、依無欲、依滅、向於捨，修正志、正語、正業、正命、正方便、正念、正定，依遠離、依無欲、依滅、向於捨，是名修八聖道。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「我今當說修八聖道。**
　　那時，世尊告訴在場的眾位比丘說：「我現在要為你們說明如何修行八聖道（通往解脫的八種正確路徑）。

**◈ 諦聽，善思。何等為修八聖道？**
　　請你們仔細聽，好好思考。什麼叫做修行八聖道呢？

**◈ 是比丘修正見，依遠離、依無欲、依滅、向於捨，**
　　也就是比丘修行正見（正確的見地與認知），依循遠離（對世俗執著的脫離）、依循無欲（消除貪欲）、依循滅（苦的止息），並趨向於捨（不執著於任何一方的平等心），

**◈ 修正志、正語、正業、正命、正方便、正念、正定，**
　　同時修行正思惟（正確的思考方向）、正語（正確的言語）、正業（正確的行為）、正命（正確的生活方式）、正精進（正確的努力）、正念（正確的覺知）、正定（正確的專注），

**◈ 依遠離、依無欲、依滅、向於捨，是名修八聖道。」**
　　同樣依循遠離、依循無欲、依循滅，並趨向於捨，這就叫做修行八聖道。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **八聖道 (Ariya Aṭṭhaṅgika Magga)**：佛法中為了達到涅槃而設定的八種正確修行路徑，分為慧（正見、正思惟）、戒（正語、正業、正命）、定（正精進、正念、正定）三學。
2. **遠離 (Viveka)**：指在心理上與導致痛苦的貪、嗔、痴等染污法保持距離，或在環境上遠離喧囂以利修行。
3. **無欲 (Virāga)**：指對感官對象的貪愛逐漸淡化，不再被欲望所驅使的狀態。
4. **滅 (Nirodha)**：指苦因（貪愛）的完全止息，進而使苦果不再產生。
5. **捨 (Upekkhā)**：一種高度平衡且中立的心理狀態，不對任何現象產生貪著或厭惡，是正定與智慧的基礎。

### [現代實作指南]
本經文揭示了修行八聖道的「核心動能」並非單純的條目執行，而是一種「由執著 ➔ 脫離 ➔ 平衡」的心理轉化過程。在現代生活中可採取以下實踐步驟：

1. **建立「遠離」的覺察**：
   當你陷入強烈的情緒（如憤怒或極度渴望）時，不要試圖強行壓抑，而是先在心中建立一個「觀察者」的視角。告訴自己：「現在有一股憤怒在升起」，透過這種覺知將自己與情緒分離，這就是心理上的「遠離」。

2. **練習「無欲」的轉念**：
   在面對物質欲望（如想要購買不必要的高價品）時，思考該物品能帶來的快感是短暫的，而隨之而來的維護壓力與失去的恐懼才是長久的。將焦點從「擁有」轉向「足夠」，降低對特定結果的依賴。

3. **體會「滅」的寧靜**：
   當你成功地從欲望中抽離，你會發現心靈中原本緊繃的壓力突然消失，這是一種短暫的「滅」。記錄這種寧靜感，讓大腦習慣「不執著反而更輕鬆」的體驗。

4. **培養「捨」的平衡**：
   在面對生活中的讚美與批評時，練習將兩者視為同樣的「現象」。讚美不讓我們膨脹，批評不讓我們沮喪。將心維持在一個不偏不倚的中間點，這能極大地降低精神內耗。

### 段落大意 
本經由世尊向比丘說明修行八聖道的具體路徑。強調八聖道的實踐必須依循「遠離、無欲、滅、捨」這四個心理轉化階段，將正確的見地、思考、言語、行為、生活、努力、覺知與專注，全部置於脫離執著並趨向平等心的框架下，方能稱為真正的修行。
