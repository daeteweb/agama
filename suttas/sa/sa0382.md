# 雜阿含經 第 382 經

### [原文]
如是我聞：

一時，佛住波羅㮈仙人住處鹿野苑中。

爾時，世尊告諸比丘：「有四聖諦。何等為四？謂苦聖諦、苦集聖諦、苦滅聖諦、苦滅道跡聖諦。若比丘於苦聖諦當知、當解，於集聖諦當知、當斷，於苦滅聖諦當知、當證，於苦滅道跡聖諦當知、當修。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住波羅㮈仙人住處鹿野苑中。**
　　在某一個時間，佛住在波羅㮈仙人住處的鹿野苑中。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「有四聖諦。何等為四？**
　　「有四個關於痛苦的聖なる真理。這四個真理是什麼呢？

**◈ 謂苦聖諦、苦集聖諦、苦滅聖諦、苦滅道跡聖諦。**
　　也就是：苦聖諦（關於痛苦本質的真理）、苦集聖諦（關於痛苦原因的真理）、苦滅聖諦（關於痛苦止息的真理）、以及苦滅道跡聖諦（關於導向痛苦止息之道路的真理）。

**◈ 若比丘於苦聖諦當知、當解，**
　　如果比丘對於苦聖諦，應該去認知並深刻理解（這就是苦的實相）。

**◈ 於集聖諦當知、當斷，**
　　對於集聖諦（導致痛苦的貪愛與執著），應該去認知並將其斷除。

**◈ 於苦滅聖諦當知、當證，**
　　對於苦滅聖諦（痛苦完全止息的涅槃境界），應該去認知並親自證悟。

**◈ 於苦滅道跡聖諦當知、當修。」**
　　對於苦滅道跡聖諦（通往解脫的八正道），應該去認知並在生活中實踐修行。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四個真理，涵蓋了問題（苦）、原因（集）、目標（滅）與方法（道）。
2. **苦 (Dukkha)**：指一切不穩定、不圓滿、令人不安的狀態，包含出生、衰老、疾病、死亡及與不愛之人共處等。
3. **集 (Samudaya)**：指導致苦產生的原因，核心在於對五蘊（物質身體、感受、想像與認知、心的造作、意識）的貪愛與執取。
4. **滅 (Nirodha)**：指苦因被完全消除後，達到絕對寧靜、不再輪迴的涅槃狀態。
5. **道 (Magga)**：指消除苦因、實現苦滅的具體路徑，即「八正道」。

### [現代實作指南]
本經文提出了面對生命問題的「四個對應動作」，可轉化為現代心理實踐的四步驟：

1. **認知與理解（對應苦）** $\rightarrow$ **正視壓力**：當感到痛苦或焦慮時，不要試圖逃避或壓抑，而是客觀地觀察它。告訴自己：「現在這種不安的感覺就是『苦』」，將其視為一種自然現象而非個人失敗。
2. **分析與切斷（對應集）** $\rightarrow$ **找出執著點**：追問自己「我為什麼痛苦？」。通常是因為我們希望現實符合某種特定的想像（執取）。意識到「我的痛苦來自於我想控制不可控的事物」，並嘗試在心念上切斷這種強求。
3. **願景與體會（對應滅）** $\rightarrow$ **體驗放下的寧靜**：嘗試在短暫的冥想或深呼吸中，感受當「不再強求」時，內心瞬間產生的輕鬆感。這讓大腦建立「放下即是快樂」的正向經驗。
4. **實踐與習慣（對應道）** $\rightarrow$ **建立健康生活模式**：將八正道簡化為日常習慣。例如：正語（不說傷人的話）、正念（專注於當下），透過持續的小習慣改變，將解脫的邏輯內化為生活方式。

### [段落大意]
本經概述了佛法核心的「四聖諦」及其對應的修行任務：對「苦」要認知理解，對「集（苦因）」要認知並斷除，對「滅（苦止）」要認知並證悟，對「道（解脫路徑）」要認知並實踐。
