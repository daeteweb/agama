# 雜阿含經 第 746 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若比丘修習安那般那念，多修習已，得大果大福利。云何修習安那般那念，多修習已，得大果大福利？是比丘心與安那般那念俱，修念覺分，依遠離、依無欲、依滅、向於捨；乃至修捨覺分，依遠離、依無欲、依滅、向於捨。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若比丘修習安那般那念，多修習已，得大果大福利。**
　　「如果比丘修行安那般那念（對呼吸的覺知），在長時間且深入地修習之後，將會獲得巨大的成就與深遠的利益。

**◈ 云何修習安那般那念，多修習已，得大果大福利？**
　　究竟應該如何修習安那般那念（對呼吸的覺知），才能在多加修習後獲得巨大的成就與深遠的利益呢？

**◈ 是比丘心與安那般那念俱，修念覺分，依遠離、依無欲、依滅、向於捨；**
　　這位比丘讓心與安那般那念（對呼吸的覺知）保持一致，藉此修行「念覺分」（能導向覺悟的正念要素），其修行過程依據於遠離雜念、依據於沒有貪欲、依據於熄滅煩惱，並趨向於捨（心靈的平衡與不偏不倚）；

**◈ 乃至修捨覺分，依遠離、依無欲、依滅、向於捨。」**
　　乃至於修行到「捨覺分」（能導向覺悟的平等心要素）時，同樣依據於遠離雜念、依據於沒有貪欲、依據於熄滅煩惱，並趨向於捨（心靈的平衡與不偏不倚）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **安那般那念 (Ānāpānasati)**：指對「吸入（安那）」與「呼出（般那）」之呼吸過程的持續覺知。這是一種將心專注於當下生理現象，進而達到心定與智慧的修行方法。
2. **覺分 (Bojjhaṅga)**：指導向覺悟（Bodhi）的七種心理要素。本經提及的「念覺分」是指清晰的覺知力，「捨覺分」是指在面對對象時不產生貪愛或厭惡的中立平衡心。
3. **遠離 (Viveka)**：指在心理上與導致煩惱的對象（如五欲、雜念）保持距離，使心不再受其牽引。
4. **滅 (Nirodha)**：指煩惱的止息或熄滅。在修行過程中，是指透過覺知使貪、嗔、癡等心理造作停止運作。
5. **捨 (Upekkhā)**：並非指冷漠或放棄，而是一種高度穩定、不偏袒、不反應的心理狀態，是面對所有經驗時的平等心。

### [現代實作指南]
本經揭示了從「呼吸覺知」進階到「覺悟要素」的具體路徑。在現代生活中，可採取以下步驟實踐：

1. **建立呼吸錨點**：
   - 每天抽出 10-20 分鐘，僅僅觀察呼吸的自然進出。不刻意控制呼吸，而是像觀察者一樣看著呼吸發生。這就是「心與安那般那念俱」。

2. **將覺知轉化為「念覺分」**：
   - 當你發現心在呼吸中安定後，嘗試將這種「覺知力」擴展到日常生活中。例如：在行走時覺知行走，在工作時覺知工作。當你能清晰地知道「現在正在發生什麼」而沒有被情緒牽引時，即是在修習念覺分。

3. **實踐「遠離 $\text{➔}$ 無欲 $\text{➔}$ 滅」的心理轉念**：
   - **遠離**：當強烈的憤怒或貪婪升起時，告訴自己：「我現在觀察到憤怒升起了」，將自己從情緒中抽離，成為觀察者。
   - **無欲**：意識到該情緒是由於「想要改變現狀」或「想要得到某物」而起，進而對該欲望保持中立。
   - **滅**：當你不再認同該情緒，也不再餵養該欲望時，情緒會自然地熄滅。

4. **趨向「捨」的平衡狀態**：
   - 在面對生活中的稱讚或批評時，練習不隨之興奮或沮喪，而是維持一種「這只是現象的生滅」的平等心。這能減少心理內耗，提升心理韌性。

---

### [段落大意]
本經說明修習安那般那念（呼吸覺知）能帶來巨大的利益。其核心路徑在於將呼吸的專注力轉化為覺悟的要素（覺分），透過遠離雜念、消除貪欲與熄滅煩惱，最終使心靈達到一種平等、不偏不倚的「捨」之狀態。
