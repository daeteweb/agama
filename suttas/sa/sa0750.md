# 雜阿含經 第 750 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若比丘！諸惡不善法，比丘，一切皆以無明為根本，無明集、無明生、無明起。所以者何？無明者無知，於善、不善法不如實知，有罪、無罪，下法、上法，染污，不染污，分別、不分別，緣起、非緣起不如實知；不如實知故，起於邪見，起於邪見已，能起邪志、邪語、邪業、邪命、邪方便、邪念、邪定。

「若諸善法生，一切皆明為根本，明集、明生、明起。明，於善、不善法如實知者，罪、無罪，親近、不親近，卑法、勝法，穢污、白淨，有分別、無分別，緣起、非緣起悉如實知，如實知者，是則正見。正見者，能起正志、正語、正業、正命、正方便、正念、正定。正定起已，聖弟子得正解脫貪、恚、癡，貪、恚、癡解脫已，是聖弟子得正智見：『我生已盡，梵行已立，所作已作，自知不受後有。』」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若比丘！諸惡不善法，比丘，一切皆以無明為根本，無明集、無明生、無明起。**
　　「比丘們！所有惡劣且不善的法（現象），全部都是以無明（對真理的無知）為根本，由無明而聚集、由無明而產生、由無明而興起。

**◈ 所以者何？無明者無知，於善、不善法不如實知，有罪、無罪，下法、上法，染污，不染污，分別、不分別，緣起、非緣起不如實知；**
　　為什麼呢？所謂的無明就是缺乏智慧，對於善法與不善法無法如實地認知；對於什麼是有罪的、什麼是無罪的，什麼是低劣的法、什麼是高尚的法，什麼是染污的、什麼是不染污的，什麼是有分別的、什麼是無分別的，以及什麼是緣起（因緣而生）的、什麼是非緣起的，都無法如實地認知。

**◈ 不如實知故，起於邪見，起於邪見已，能起邪志、邪語、邪業、邪命、邪方便、邪念、邪定。**
　　因為無法如實認知，所以產生了錯誤的見解（邪見）；在產生邪見之後，進而產生錯誤的志向（邪思惟）、錯誤的言語（邪語）、錯誤的行為（邪業）、錯誤的生活方式（邪命）、錯誤的精進（邪方便）、錯誤的覺察（邪念）以及錯誤的專注（邪定）。

**◈ 若諸善法生，一切皆明為根本，明集、明生、明起。**
　　相反地，如果諸多善法產生，一切都是以明（智慧與覺知）為根本，由明而聚集、由明而產生、由明而興起。

**◈ 明，於善、不善法如實知者，罪、無罪，親近、不親近，卑法、勝法，穢污、白淨，有分別、無分別，緣起、非緣起悉如實知，如實知者，是則正見。**
　　所謂的明，就是對於善法與不善法能如實認知的人；對於有罪、無罪，親近、不親近，卑劣的法、優勝的法，穢污的、白淨的，有分別的、無分別的，以及緣起（因緣而生）與非緣起的現象，全部都能如實地認知。能夠如實認知，這就是正見（正確的見解）。

**◈ 正見者，能起正志、正語、正業、正命、正方便、正念、正定。**
　　有了正見，就能進而產生正確的志向（正思惟）、正確的言語（正語）、正確的行為（正業）、正確的生活方式（正命）、正確的精進（正方便）、正確的覺察（正念）以及正確的專注（正定）。

**◈ 正定起已，聖弟子得正解脫貪、恚、癡，貪、恚、癡解脫已，是聖弟子得正智見：**
　　當正定（正確的專注）興起之後，聖弟子就能獲得正確的解脫，擺脫貪愛、嗔恚與愚癡；在從貪、恚、癡中解脫之後，這位聖弟子便獲得了正確的智慧見解：

**◈ 『我生已盡，梵行已立，所作已作，自知不受後有。』」**
　　『我的出生（輪迴）已經結束，清淨的修行（梵行）已經建立，我應做的事已經完成，自知不再受後世的出生（輪迴）。』」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **無明 (Avijjā)**：指對四聖諦以及事物本質（如無常、苦、無我）的無知，是導致輪迴與痛苦的根本原因。
2. **明 (Vijjā)**：指對真理的如實覺知，即般若智慧，能破除無明，是通往解脫的必要條件。
3. **緣起 (Paṭicca-samuppāda)**：指一切現象皆是依因緣而生，沒有獨立永恆的實體。
4. **正見 (Sammā-diṭṭhi)**：八正道之首，指正確地看待世界與生命的運作方式，如實認知善惡與因果。
5. **貪、恚、癡 (Lobha, Dosa, Moha)**：稱為「三毒」。貪是指對對象的執著渴求；恚是指對不滿意對象的厭惡與憤怒；癡是指對真理的迷失與無知。
6. **梵行 (Brahmacariya)**：指清淨的修行生活，旨在消除煩惱，達到精神上的絕對純淨。

### [現代實作指南]
本經文揭示了「認知 → 傾向 → 行為 → 結果」的心理鏈條。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **覺察「認知偏差」（對治無明）**：
　　當我們感到痛苦或憤怒時，首先詢問自己：「我現在對這件事的看法是否『如實』？」例如，將對方的批評視為「攻擊」（無明），還是視為「對方當下的情緒反應」（如實知）。

2. **校準「價值判斷」（建立正見）**：
　　練習區分「染污」與「白淨」的行為。在做決定前，分析該行為是基於貪婪（染污）還是基於慈悲與智慧（白淨）。將注意力從「我想要什麼」轉向「什麼才是正確的因果」。

3. **同步「言行一致」（實踐八正道）**：
　　一旦認知校準為正見，立即將其延伸至生活細節：
　　➔ **正語**：不再用言語傷害他人以獲取快感。
　　➔ **正業**：採取不損害他人且對自己有益的行動。
　　➔ **正命**：選擇一個不違背道德且能讓心靈平靜的職業。

4. **觀察「情緒消融」（達成解脫）**：
　　透過持續的正念觀察，我們會發現當「正見」確立時，貪愛（想要掌控）與嗔恚（不滿現狀）會自然減少。這不是強行壓抑，而是因為「看穿了真相」而不再執著。

### [段落大意]
本經論述了「無明」與「明」作為身心狀態之根本的對立影響。無明導致對善惡與緣起的錯誤認知，進而引發八種邪徑，導致輪迴痛苦；而「明」則使人能如實認知真理，建立正見，進而啟動八正道，最終消除貪、恚、癡三毒，達成不再輪迴的涅槃解脫。
