# 雜阿含經 第 946 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有異婆羅門來詣佛所，恭敬問訊，問訊已，退坐一面，白佛言：「瞿曇！未來世當有幾佛？」

佛告婆羅門：「未來佛者，如無量恒河沙。」

爾時，婆羅門作是念：「未來當有如無量恒河沙三藐三佛陀，我當從彼修諸梵行。」

爾時，婆羅門聞佛所說，歡喜隨喜，從坐起去。

時，婆羅門隨路思惟：「我今唯問沙門瞿曇未來諸佛，不問過去。」即隨路還，復問世尊：「云何？瞿曇！過去世時，復有幾佛？」

佛告婆羅門：「過去世佛亦如無量恒河沙數。」

時，婆羅門即作是念：「過去世中有無量恒河沙等諸佛世尊，我曾不習近。設復未來如無量恒河沙三藐三佛陀，亦當不與習近娛樂。我今當於沙門瞿曇所修行梵行。」即便合掌白佛言：「唯願聽我於正法、律出家修梵行。」

佛告婆羅門：「聽汝於正法、律出家修梵行，得比丘分。」

爾時，婆羅門即出家受具足。出家已，獨一靜處思惟：「所以善男子正信非家，出家學道……。」乃至得阿羅漢。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有異婆羅門來詣佛所，恭敬問訊，問訊已，退坐一面，白佛言：「瞿曇！未來世當有幾佛？」**
　　那時，有一位特殊的婆羅門前來拜訪佛陀，恭敬地問候，問候完畢後，退到一邊坐下，對佛陀恭敬地說道：「瞿曇！在未來的世間，將會出現多少位佛？」

**◈ 佛告婆羅門：「未來佛者，如無量恒河沙。」**
　　佛陀告訴這位婆羅門：「未來將會出現的佛，數量就像無量恒河中的沙子一樣多。」

**◈ 爾時，婆羅門作是念：「未來當有如無量恒河沙三藐三佛陀，我當從彼修諸梵行。」**
　　那時，婆羅門在心中思考：「未來將會有如無量恒河沙般多的三藐三佛陀（最高、最圓滿的智慧與覺悟之佛），我應該在那些佛陀的指導下修習清淨的梵行（修行）。」

**◈ 爾時，婆羅門聞佛所說，歡喜隨喜，從坐起去。**
　　那時，婆羅門聽聞佛陀所說的話，心中生起法喜並隨喜讚嘆，於是從座位上起身離開。

**◈ 時，婆羅門隨路思惟：「我今唯問沙門瞿曇未來諸佛，不問過去。」即隨路還，復問世尊：「云何？瞿曇！過去世時，復有幾佛？」**
　　這時，婆羅門在路途中思考：「我剛才只詢問了沙門瞿曇關於未來諸佛的情況，卻沒有詢問過去。」於是便沿路返回，再次請問世尊：「請問瞿曇，在過去的世間，又有多少位佛？」

**◈ 佛告婆羅門：「過去世佛亦如無量恒河沙數。」**
　　佛陀告訴婆羅門：「過去世間的佛，數量同樣如無量恒河沙般之多。」

**◈ 時，婆羅門即作是念：「過去世中有無量恒河沙等諸佛世尊，我曾不習近。設復未來如無量恒河沙三藐三佛陀，亦當不與習近娛樂。我今當於沙門瞿曇所修行梵行。」**
　　這時，婆羅門立刻在心中思考：「過去世間有如此多如恒河沙般的諸佛世尊，但我當時都沒有機會親近學習。如果等到未來同樣有如恒河沙般多的三藐三佛陀出現，我也未必能親近他們並從中獲益。我現在就應該在沙門瞿曇這裡修習清淨的梵行（修行）。」

**◈ 即便合掌白佛言：「唯願聽我於正法、律出家修梵行。」**
　　於是，他立即雙手合十向著佛陀恭敬地說道：「唯願您允許我在正法（正確的真理）與律（戒律）的指導下，出家修習清淨的梵行（修行）。」

**◈ 佛告婆羅門：「聽汝於正法、律出家修梵行，得比丘分。」**
　　佛陀告訴婆羅門：「我允許你在正法（正確的真理）與律（戒律）的指導下出家修習清淨的梵行（修行），讓你獲得比丘（出家男僧）的身份。」

**◈ 爾時，婆羅門即出家受具足。**
　　那時，婆羅門立即出家並受具足戒（完整的出家戒律）。

**◈ 出家已，獨一靜處思惟：「所以善男子正信非家，出家學道……。」乃至得阿羅漢。**
　　出家之後，他獨自到一個安靜的地方思考：「之所以作為一名善男子，要建立正確的信仰而離開家庭，出家學習解脫之道……」直到最後證得阿羅漢（完全斷除煩惱的聖者）的果位。

### [詞語註解（解釋重要佛教概念）]
1. **婆羅門 (Brahmana)**：古印度種姓制度中的最高階級，傳統上負責祭祀與研習吠陀經。
2. **三藐三佛陀 (Samyak-saṃbuddha)**：指正等覺者，即能獨立發現真理並教導眾生解脫的佛陀。
3. **梵行 (Brahmacariya)**：原指婆羅門的禁欲修行，在佛教中轉義為清淨的修行生活，旨在斷除貪欲以達成解脫。
4. **正法 (Saddhamma)**：指佛陀所開示的正確真理，能導向滅苦與涅槃的教法。
5. **比丘 (Bhikkhu)**：指經過正式授戒、完全出家並依循戒律生活的男僧侶。
6. **阿羅漢 (Arahant)**：指已完全斷除所有煩惱（漏盡），不再輪迴，達到最終解脫境界的聖者。

### [現代實作指南]
本經文的核心在於「對時機的覺察」與「果斷的行動」。在現代生活中，我們可以將其轉化為以下實踐技巧：

1. **克服「拖延心理」與「完美主義」**：
   婆羅門最初想等到「未來」的佛陀出現才修行，這是一種典型的拖延心理（認為未來會有更好的條件）。實作時，當你意識到某個正確的目標（如健康、學習、心理成長）時，請提醒自己：**「最好的時機就是現在」**，而非等待一個理想化的未來。

2. **分析「機會成本」的稀有性**：
   婆羅門意識到過去錯失了無數次機會，未來則充滿不確定性。在面對人生抉擇時，嘗試列出「如果現在不開始，我將錯過什麼」的清單，將對「失去」的覺察轉化為「行動」的動力。

3. **建立「靜處思惟」的習慣**：
   經文中婆羅門出家後「獨一靜處思惟」，這說明了行動之後需要深度的內省。建議每天撥出 15-30 分鐘的「數位斷食」時間，在安靜的空間中反思自己的行為是否符合自己的核心價值（正信），將外部的行動內化為深層的認知轉變。

4. **從「詢問」到「實踐」的跳躍**：
   婆羅門從詢問佛陀數量（理論認知） → 意識到稀有性（價值評估） → 請求出家（採取行動） → 靜處思惟（深化實踐）。這是一個完整的轉念路徑：**認知 ➔ 評估 ➔ 行動 ➔ 內化**。

### [段落大意]
一位婆羅門詢問佛陀關於過去與未來佛陀的數量，得知佛陀數量如恒河沙般多後，意識到能親近佛陀是極其稀有的機會。他不再等待未來，果斷決定在當下的佛陀指導下出家修行，最終透過靜處思惟與實踐，證得阿羅漢果。
