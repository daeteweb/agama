# 雜阿含經 第 855 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有難提優婆塞來詣佛所，稽首佛足，退坐一面，白佛言：「世尊！若聖弟子於此五根一切時不成就者，為放逸？為不放逸？」

佛告難提：「若於此五根一切時不成就者，我說此等為凡夫數。若聖弟子不成就者，為放逸，為不放逸。難提！若聖弟子於佛不壞淨成就，而不上求，不於空閑林中，若露地坐，晝夜禪思，精勤修習，勝妙出離，饒益隨喜；彼不隨喜已，歡喜不生，歡喜不生已，身不猗息，身不猗息已，苦覺則生，苦覺生已，心不得定，心不得定者，是聖弟子名為放逸。於法、僧不壞淨，聖戒成就亦如是說。如是，難提！若聖弟子成就於佛不壞淨，其心不起知足想。於空閑林中，樹下露地，晝夜禪思，精勤方便，能起勝妙出離隨喜；隨喜已，生歡喜，生歡喜已，身猗息，身猗息已，覺受樂，覺受樂已，心則定。若聖弟子心定者，名不放逸，法、僧不壞淨，聖戒成就亦如是說。」

佛說此經已，難提優婆塞聞佛所說，歡喜隨喜，從座起，禮佛足而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 時，有難提優婆塞來詣佛所，稽首佛足，退坐一面，白佛言：**
　　那時，有一位名叫難提的優婆塞（男居士）來到佛陀這裡，恭敬地向佛陀的足下頂禮，然後退到一邊坐下，對佛陀恭敬地說道：

**◈ 「世尊！若聖弟子於此五根一切時不成就者，為放逸？為不放逸？」**
　　「世尊！如果一位聖弟子（已進入聖道、追求解脫的修行者）在所有時間裡都沒有成就這五根（信、精進、念、定、慧這五種修行能力），這應該被稱為放逸（懈怠、不警覺），還是不放逸（精勤、警覺）呢？」

**◈ 佛告難提：「若於此五根一切時不成就者，我說此等為凡夫數。**
　　佛陀告訴難提：「如果一個人完全沒有成就這五根，我說這樣的人被歸類為凡夫（尚未覺悟的普通眾生）。

**◈ 若聖弟子不成就者，為放逸，為不放逸。**
　　但如果是一位聖弟子卻沒有成就這些能力，那麼這就涉及他是放逸還是不放逸的問題。

**◈ 難提！若聖弟子於佛不壞淨成就，而不上求，**
　　難提！如果一位聖弟子雖然對佛陀保持著純淨且堅定的信仰，但卻不追求更高的覺悟境界，

**◈ 不於空閑林中，若露地坐，晝夜禪思，精勤修習，勝妙出離，饒益隨喜；**
　　不在安靜的森林中或露天地上坐禪，不日夜地禪思冥想，不精勤地修行，也不對那種勝妙的解脫（脫離輪迴的痛苦）以及對眾生的饒益而感到隨喜（欣悅）；

**◈ 彼不隨喜已，歡喜不生，歡喜不生已，身不猗息，身不猗息已，苦覺則生，苦覺生已，心不得定，心不得定者，是聖弟子名為放逸。**
　　因為他沒有隨喜，所以內心無法產生歡喜；因為沒有歡喜，所以身體無法放鬆（猗息）；因為身體無法放鬆，所以會產生苦的覺受（感受）；因為苦的覺受產生，心識（意識）就無法進入定（心意識集中且不散亂的狀態）。這樣的心態，對於聖弟子來說就稱為放逸。

**◈ 於法、僧不壞淨，聖戒成就亦如是說。**
　　對於對正法、僧團保持純淨信仰，以及成就聖潔戒律的情況，也是同樣的道理。

**◈ 如是，難提！若聖弟子成就於佛不壞淨，其心不起知足想。**
　　同樣地，難提！如果一位聖弟子成就了對佛陀的純淨信仰，且心中不產生「這樣就夠了」的知足錯覺（不滿足於現狀而持續精進），

**◈ 於空閑林中，樹下露地，晝夜禪思，精勤方便，能起勝妙出離隨喜；**
　　他在安靜的森林中、樹下或露天地上，日夜禪思，運用精勤的方便方法，能夠對勝妙的解脫產生隨喜之情；

**◈ 隨喜已，生歡喜，生歡喜已，身猗息，身猗息已，覺受樂，覺受樂已，心則定。**
　　在隨喜之後，內心會產生歡喜；產生歡喜之後，身體會感到放鬆；身體放鬆之後，會產生快樂的覺受（感受）；在感受到快樂之後，心意識（意識）就能進入定（集中且不散亂的狀態）。

**◈ 若聖弟子心定者，名不放逸，法、僧不壞淨，聖戒成就亦如是說。」**
　　如果聖弟子的心能進入定，就稱為不放逸。對於對正法、僧團保持純淨信仰，以及成就聖潔戒律的情況，也是同樣的道理。」

**◈ 佛說此經已，難提優婆塞聞佛所說，歡喜隨喜，從座起，禮佛足而去。**
　　佛陀講完這部經後，難提優婆塞聽聞佛陀所說的教法，心生法喜並依照教導去實踐，他從座位上站起來，向佛陀的足下行禮後才離開。

### [詞語註解（解釋重要佛教概念）]
1. **五根 (Indriya)**：指修行者在心中培育的五種主導能力，分別為：信根（對正法的信心）、精進根（不懈努力的意志）、念根（正念的覺知）、定根（心的專注穩定）、慧根（洞察真理的智慧）。
2. **放逸 (Pamāda)**：指心靈的懈怠、疏忽或對修行缺乏警覺，容易被感官欲望牽引而失去對目標的專注。
3. **不放逸 (Appamāda)**：指一種高度的警覺與精勤，持續地將心維持在正念中，不讓心散亂或墮入懈怠。
4. **不壞淨 (Asammisa/Purity)**：指對佛、法、僧三寶保持純淨、不染污的信心，不因外界干擾或內心懷疑而損毀。
5. **定 (Samādhi)**：心意識集中於單一對象且不散亂的狀態，是產生智慧（慧根）的必要基礎。

### [現代實作指南]
本經文揭示了一個關於「心理正向循環」的機制：**隨喜 ➔ 歡喜 ➔ 身體放鬆 ➔ 快樂覺受 ➔ 心定**。在現代生活中，我們可以將其轉化為以下實作步驟：

1. **練習「隨喜」 (Positive Appreciation)**：
   - **操作**：每天記錄三件他人成功或環境美好的小事，並真心為之感到高興。
   - **轉念**：將「比較心」（為什麼他有我沒有）轉化為「隨喜心」（他能獲得這份好處，真是太好了）。這能打破內心的匱乏感。

2. **建立「身心連結」的放鬆 (Somatic Relaxation)**：
   - **操作**：當內心產生歡喜感時，有意識地觀察身體的反應（如肩膀下沉、呼吸深長）。
   - **技巧**：利用「歡喜」作為切入點來放鬆身體，而非在壓力大時強迫自己放鬆。

3. **從「樂受」進入「專注」 (From Pleasure to Focus)**：
   - **操作**：在心情愉悅且身體放鬆的狀態下，嘗試進行 5-10 分鐘的單一任務專注（如專注呼吸或閱讀）。
   - **原理**：當身體處於「非威脅」且「愉悅」的狀態時，大腦更容易進入深度專注（Flow state），而非在焦慮或壓力（苦覺）中強行專注。

4. **警惕「偽知足」 (Avoiding False Contentment)**：
   - **操作**：區分「對現狀的感恩」與「對成長的停止」。
   - **轉念**：感恩目前的擁有，但同時保持對更高層次心靈品質（如慈悲、智慧）的追求，避免陷入停滯不前的舒適區。

### [段落大意]
本經透過難提優婆塞的提問，由佛陀闡明「放逸」與「不放逸」的區別。佛陀指出，聖弟子若僅有信仰而缺乏精勤修行與隨喜心，會導致身心緊繃、產生苦受，最終無法心定，此為「放逸」；反之，若能以隨喜心驅動歡喜，進而使身體放鬆並產生樂受，則能輕易進入心定之境，此為「不放逸」。
