# 雜阿含經 第 864 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若比丘若行、若形、若相，離欲、惡不善法，有覺有觀，離生喜樂，初禪具足住。彼不憶念如是行、如是形、如是相，然於彼色、受、想、行、識法；作如病、如癰、如刺、如殺、無常、苦、空、非我思惟，於彼法生厭、怖畏、防護；生厭、怖畏、防護已，以甘露門而自饒益，如是寂靜，如是勝妙，所謂捨離，餘愛盡、無欲、滅盡、涅槃。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若比丘若行、若形、若相，離欲、惡不善法，有覺有觀，離生喜樂，初禪具足住。**
　　「如果一名比丘無論是在行為、形貌或外相上，都能遠離對感官欲望的渴求，厭離不善的法，具足覺知（正念）與觀察（正定），並在遠離感官欲望而產生的喜悅與快樂中，完全進入初禪的定境。

**◈ 彼不憶念如是行、如是形、如是相，**
　　他不會對這種修行行為、這種形貌或這種外相產生執著的憶念，

**◈ 然於彼色、受、想、行、識法；作如病、如癰、如刺、如殺、無常、苦、空、非我思惟，**
　　而是針對構成身心的五種要素，即色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）這五法，進行思惟：認為它們就像疾病一樣痛苦，像腫瘤一樣險惡，像刺入身體一樣折磨，像死亡一樣毀滅；並體悟到它們是無常（變遷不居）、苦（不圓滿）、空（缺乏恆常實體）且無我（沒有永恆不變的自我）。

**◈ 於彼法生厭、怖畏、防護；**
　　對這些五蘊之法產生厭離心，意識到其危險而感到怖畏，進而採取防護，不再讓心被其牽引；

**◈ 生厭、怖畏、防護已，以甘露門而自饒益，**
　　在產生厭離、怖畏並完成防護之後，透過這扇如同甘露般能消除痛苦的解脫之門，使自己獲得真正的利益，

**◈ 如是寂靜，如是勝妙，所謂捨離，餘愛盡、無欲、滅盡、涅槃。」**
　　如此的狀態才是真正的寂靜，如此的境界才是最勝妙的。這就叫做捨離，讓殘餘的貪愛完全消失，不再有欲望，達到煩惱的滅盡，進入涅槃（超越生死、絕對寧靜的解脫境界）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **初禪 (First Jhāna)**：禪定修行中的第一個階段。其特徵是離欲（遠離感官欲望）與離惡不善法，並伴隨由定力產生的喜（Pīti，強烈的法喜）與樂（Sukha，寧靜的快樂），同時維持著覺（Mindfulness）與觀（Clear Comprehension）。
2. **五蘊 (Five Aggregates)**：構成生命個體的五種動態要素，包括色（物質）、受（感受）、想（認知）、行（意志造作）、識（意識）。本經強調對五蘊進行「病苦」的思惟，以打破對自我的執著。
3. **甘露門 (Amṛta-dvāra)**：甘露（Amṛta）在佛教中象徵不死與永恆的解脫，對比世間的苦與死。甘露門是指通往涅槃的解脫路徑。
4. **厭 (Nibbida)**：並非世俗的厭惡或反感，而是一種基於智慧的「厭離」。是指在深刻觀察到現象的無常與苦後，心不再對其產生貪著的冷靜狀態。

### [現代實作指南]
本經提供了一套從「定力」轉向「智慧」的實踐路徑，可用於處理現代人的精神內耗與執著：

1. **建立心理緩衝區（對應初禪）**：
   當面對強烈的情緒或欲望時，先不急於反應，嘗試進入一種「離欲」的觀察狀態。練習深呼吸，讓心安定在一個不被外界干擾的寧靜點，感受當下純粹的平靜（喜樂），而非追求外部刺激的快感。

2. **對執著對象進行「去美化」思惟（對應病、癰、刺、殺）**：
   當我們對某種身份、名聲或物質產生強烈執著時，嘗試將其視為「潛在的壓力源」：
   ➔ **如病**：思考這個執著是否像疾病一樣，在潛意識中讓我感到不安？
   ➔ **如癰/刺**：思考維持這個執著是否像身體長腫瘤或被刺一樣，需要不斷地維護、擔心，反而帶來折磨？
   ➔ **如殺**：思考這種執著是否正在扼殺我的內心自由與真實的平靜？

3. **從「厭離」轉向「防護」**：
   意識到執著帶來的是痛苦而非快樂後，產生一種「不想再被這樣牽著走」的覺悟（厭離）。接著建立「防護」機制 ➔ 每當執著再次升起時，立刻提醒自己：「這又是那個『刺』在折磨我」，從而迅速抽離。

4. **進入寂靜狀態**：
   當不再試圖掌控或擁有（捨離）後，體驗那種不需要依賴外部條件而產生的內在安定感，這便是生活中的微型「涅槃」體驗。

### [段落大意]
本經闡述了從初禪定境進階至解脫的過程。佛陀教導比丘，即便進入初禪，也不應執著於定境的狀態，而應將智慧轉向觀察五蘊（色受想行識）的本質。透過將五蘊視為如疾病、腫瘤般的苦患，體悟其無常、苦、空、無我的特性，進而產生厭離心並採取防護，最終透過捨離貪愛，達到滅盡煩惱的涅槃境界。
