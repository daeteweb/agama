# 雜阿含經 第 804 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「當修安那般那念。安那般那念修習多修習者，斷諸覺想。云何安那般那念修習多修習斷諸覺想？若比丘依止聚落、城邑住……」如上廣說，乃至「於出息滅善學，是名安那般那念修習多修習斷諸覺想。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如斷覺想，如是不動搖，得大果大福利，如是得甘露、究竟甘露，得二果、四果、七果，一一經亦如上說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「當修安那般那念。**
　　那時，世尊告訴在場的眾位比丘說：「應當修行安那般那念（即對呼吸出入的覺知）。

**◈ 安那般那念修習多修習者，斷諸覺想。**
　　那些對安那般那念（對呼吸的覺知）進行大量修習的人，能夠斷除各種覺想（指對外界現象的感知與內心的妄想）。

**◈ 云何安那般那念修習多修習斷諸覺想？**
　　為什麼說對安那般那念（對呼吸的覺知）進行大量修習能斷除各種覺想呢？

**◈ 若比丘依止聚落、城邑住……」如上廣說，**
　　（佛陀接著說明）如果比丘住在村落或城市中……」如上所述詳細地說明。

**◈ 乃至「於出息滅善學，是名安那般那念修習多修習斷諸覺想。」**
　　直到說到「在呼氣停止的時刻能善於學習覺知，這就叫做對安那般那念（對呼吸的覺知）進行大量修習而斷除了各種覺想」。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如斷覺想，如是不動搖，得大果大福利，**
　　就像這樣斷除覺想（妄想），且心不動搖，就能獲得巨大的果報與利益。

**◈ 如是得甘露、究竟甘露，**
　　如此便能獲得甘露（指解脫之法）、究竟的甘露（指超越生死、絕對寧靜的解脫境界，即涅槃）。

**◈ 得二果、四果、七果，一一經亦如上說。**
　　能夠證得二果、四果或七果（指不同層次的聖者果位），每一部經文也都是如此說明。

### [詞語註解（解釋重要佛教概念）]
1. **安那般那念 (Ānāpānasati)**：指對呼吸（入息與出息）的持續覺知。這是一種正念修行法，透過觀察呼吸的自然過程，將散亂的心凝聚在當下，進而達到心靈的寧靜與洞察。
2. **覺想 (Saññā/Vitakka)**：在此處指對感官對象的認知與隨之而來的思維。當修行者過度關注外界的「覺想」時，心會產生波動；而透過呼吸修行，可以切斷這些干擾，使心進入定境。
3. **甘露 (Amṛta)**：原意為「不死之藥」，在佛教中象徵涅槃 (Nibbāna)，即徹底斷除煩惱、不再輪迴的絕對解脫狀態。
4. **二果、四果、七果**：指修行者在解脫道路上達到的不同階段。四果通常指：初果（須陀洹）、二果（須陀洹果之進階）、三果（阿那含）、四果（阿羅漢）。七果則是指更細分的聖者層級。

### [現代實作指南]
本經文的核心在於透過「呼吸」作為錨點，來切斷「覺想」（即不斷跳轉的念頭與外界干擾）。在現代生活中，可採取以下步驟實作：

1. **設定錨點 (Anchoring)**：
   在感到壓力或思緒混亂時，將全部注意力集中在鼻尖或腹部。不要試圖控制呼吸，而是「觀察」呼吸。
   ➔ 實作：感受空氣進入鼻腔的涼意 ➔ 感受空氣呼出時的溫暖。

2. **標記覺想 (Labeling)**：
   當意識中出現「覺想」（例如：想起工作、感到焦慮、聽到噪音）時，不要對抗它，而是輕輕地在心中標記「這是想」、「這是覺」。
   ➔ 轉念：將「我現在很煩」轉化為「我觀察到心中升起了一種煩躁的覺想」。

3. **溫柔回歸 (Returning)**：
   一旦意識到自己被「覺想」帶走，立即溫柔地將注意力帶回呼吸的起點。
   ➔ 關鍵：回歸的過程本身就是「修習多修習」，每一次回歸都在強化心靈的肌肉，增加不動搖的能力。

4. **觀察「滅」的瞬間**：
   嘗試觀察呼氣結束、吸氣開始之前那個極短暫的「空隙」（即經文中的出息滅）。在那個瞬間，心最容易達到寂靜，能體驗到短暫的「斷除覺想」之狀態。

### [段落大意]
本段經文闡述了修行「安那般那念」（呼吸覺知）的重要性。佛陀教導比丘透過大量修習呼吸觀察，能有效斷除干擾心的各種覺想與妄想。當心能保持不動搖且斷除雜念時，修行者將能獲得巨大的利益，最終證得從初果到阿羅漢等不同層次的聖果，達成究竟的涅槃解脫。
