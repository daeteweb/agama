# 雜阿含經 第 597 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「云何諸眾生，　　受身得妙色？ 　云何修方便，　　而得乘出道？ 　眾生住何法，　　為何所修習， 　為何等眾生，　　諸天所供養。」

爾時，世尊說偈答言：

「持戒明智慧，　　自修習正受， 　正直心繫念，　　熾然憂悉滅， 　得平等智慧，　　其心善解脫， 　斯等因緣故，　　受身得妙色， 　成就乘出道，　　心住於中學， 　如是德備者，　　為諸天供養。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其端正美麗的天子，在深夜時分來到佛陀面前，恭敬地頂禮佛陀的足部，然後退到一邊坐下。他身體散發出的光明，將整個祇樹給孤獨園照得通亮。

**◈ 時，彼天子說偈問佛：**
　　那時，這位天子用偈頌的形式請問佛陀：

**◈ 「云何諸眾生，受身得妙色？**
　　「請問眾生是如何在輪迴中受生，而獲得絕妙的色（物質身體）相的？

**◈ 云何修方便，而得乘出道？**
　　如何修行方便的方法，才能乘上解脫（從苦輪中解脫）之道的車輛？

**◈ 眾生住何法，為何所修習，為何等眾生，諸天所供養。」**
　　眾生應安住於什麼樣的法（真理與規律），修習什麼樣的內容，才能成為被諸天供養的那類眾生呢？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答道：

**◈ 「持戒明智慧，自修習正受，**
　　「持守戒律並明瞭智慧，且親自修習正受（正確的感受與專注），

**◈ 正直心繫念，熾然憂悉滅，**
　　以正直的心繫於正念（對當下的覺知），使內心熾熱的憂慮與煩惱悉數熄滅，

**◈ 得平等智慧，其心善解脫，**
　　獲得平等看待一切的智慧，使心靈獲得善巧的解脫（超越束縛），

**◈ 斯等因緣故，受身得妙色，**
　　正是因為這些因緣（因緣：條件的聚合而生）的關係，受生時才能獲得絕妙的色（物質身體）相，

**◈ 成就乘出道，心住於中學，**
　　並能成就乘上解脫之道的果位，使心安住於中學（不偏不倚的中道修行），

**◈ 如是德備者，為諸天供養。」**
　　如此德行完備的人，才會受到諸天的供養。」

**◈ 時，彼天子復說偈言：**
　　那時，這位天子再次說偈頌道：

**◈ 「久見婆羅門，逮得般涅槃，**
　　「我長期以來看見許多婆羅門，最終達到了涅槃（超越生死、絕對寧靜的解脫境界），

**◈ 一切怖已過，永超世恩愛。」**
　　他們已經超越了一切恐懼，永遠超脫了世間的情愛執著。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，這位天子聽聞佛陀的教法後，內心感到法喜並隨喜讚嘆，再次頂禮佛陀的足部，隨即消失不見。

### [詞語註解（解釋重要佛教概念）]
1. **色 (Rūpa)**：指物質身體或物質現象。在本經中，妙色是指因過去善業而獲得的端正相貌。
2. **解脫 (Vimokkha)**：指心靈從貪、嗔、癡等煩惱的束縛中完全釋放，不再受苦輪轉動的狀態。
3. **因緣 (Paṭicca-samuppāda)**：指事物並非單一原因產生，而是由「因」（主因）與「緣」（助緣）共同聚合而生。
4. **正受 (Samyak-vedanā)**：指在修行中保持正確的感受，不對樂受產生貪著，不對苦受產生厭惡，維持心的平衡。
5. **中學 (Majjhima-sikkhā)**：指中道的修行訓練，避免走入極端的享樂或極端的苦行，實踐八正道。
6. **涅槃 (Nibbāna)**：指熄滅一切煩惱之火，達到絕對寧靜、不再輪迴的最高解脫境界。

### [現代實作指南]
本經將「外在相貌」、「內在解脫」與「他人（天人）的尊重」統一在同一套修行邏輯中。現代人可將其轉化為以下實踐步驟：

1. **外在美與內在德的連結（轉念：美由心生）**：
   - 經文指出「妙色」源於持戒與智慧。在現代生活中，這意味著一個人的氣質與吸引力，並非僅靠化妝或整形，而是來自於道德的純淨（持戒）與心靈的開闊（智慧）。
   - **實作**：每日練習一件「利他」的小事，減少自私心，讓內心的平和轉化為外在的溫潤氣質。

2. **管理情緒的「正受」練習（轉念：不被感受牽著走）**：
   - 佛陀提到「自修習正受」與「熾然憂悉滅」。
   - **實作**：當強烈的負面情緒（如焦慮、憤怒）升起時，不要試圖壓抑，而是以「觀察者」的身分看待它（正念）。告訴自己：「現在有一種焦慮的感受在升起」，而非「我很焦慮」。透過這種分離，讓情緒的「熾熱」自然冷卻。

3. **追求「平等智慧」以降低衝突（轉念：放下評判）**：
   - 平等智慧是指不再將人分為「我喜歡的」與「我不喜歡的」。
   - **實作**：在面對衝突對象時，試著思考對方也同樣渴望快樂、恐懼痛苦。當評判心消失，心靈會感到一種巨大的寬鬆感，這便是「心善解脫」的初步體驗。

4. **建立「中道」的生活節奏（轉念：適度即是正道）**：
   - 「心住於中學」提醒我們避免極端。
   - **實作**：檢查生活是否陷入「過度努力導致崩潰」或「過度懶散導致空虛」的極端。在工作與休息、欲望與節制之間尋找平衡點。

### [段落大意]
一位容貌絕美且身發光芒的天子在深夜請問佛陀：眾生如何獲得美貌、如何獲得解脫以及如何得到天人的供養？佛陀回答，這三者皆源於相同的修行：持戒、發展智慧、修習正受、保持正念、熄滅憂慮並獲得平等心。當一個人德行完備且安住於中道修行時，自然會獲得美貌、解脫與尊崇。天子聞後深感歡喜，讚嘆涅槃之境界後消失。
