# 雜阿含經 第 849 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有四種諸天天道。未淨眾生令淨，已淨者增其淨。何等為四？謂聖弟子念如來事，如是：如來、應、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊。彼如是念如來事已，則斷惡貪，及斷心惡不善過。念如來故，心生隨喜；心隨喜已，則歡悅；歡悅已，身猗息；身猗息已，覺受樂；覺受樂已，三昧定；三昧定已，聖弟子作如是學：『何等為諸天天道？』復作是念：『我聞無恚為上諸天天道。我從今日，於諸世間若怖若安，不起瞋恚，但當受持純一滿淨諸天天道。』如是法、僧、聖戒成就亦如是說。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「有四種諸天天道。未淨眾生令淨，已淨者增其淨。何等為四？**
　　那時，世尊告訴在場的眾位比丘說：「有四種通往天界（高尚心靈境界）的道途。這些道途能讓尚未清淨的眾生變得清淨，並讓已經清淨的人更加清淨。這四種道途是什麼呢？

**◈ 謂聖弟子念如來事，如是：如來、應、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊。**
　　也就是聖弟子去憶念如來（成就不變真理而來到世間的人）的殊勝功德，如此憶念：如來、應供（應受世間供養的人）、正遍知（對真理有全面覺知的人）、明行足（智慧與行為圓滿）、善逝（已走上正道且到達彼岸）、世間解（通達世間一切真理）、無上士（至高無上的聖者）、調御丈夫（能調伏心靈的導師）、天人師（天與人的導師）、佛世尊。

**◈ 彼如是念如來事已，則斷惡貪，及斷心惡不善過。**
　　聖弟子如此憶念如來的功德之後，就能斷除惡性的貪愛（對物質與情慾的執著），以及斷除心中不善的過錯。

**◈ 念如來故，心生隨喜；心隨喜已，則歡悅；歡悅已，身猗息；身猗息已，覺受樂；覺受樂已，三昧定；**
　　因為憶念如來，心中會生起隨喜（對正法感到歡喜）的心；心生隨喜之後，便會感到歡悅；歡悅之後，身體會感到放鬆與安適；身體安適之後，受（感受）會變得快樂；受（感受）快樂之後，便能進入三昧定（心意識高度集中且寧靜的狀態）。

**◈ 三昧定已，聖弟子作如是學：『何等為諸天天道？』**
　　進入三昧定之後，聖弟子會如此學習思考：『什麼是通往天界的道途？』

**◈ 復作是念：『我聞無恚為上諸天天道。我從今日，於諸世間若怖若安，不起瞋恚，但當受持純一滿淨諸天天道。』**
　　接著會如此念想：『我聽說不生瞋恚（不憤怒）是最高尚的天道。我從今天起，無論在世間遇到令人恐懼或令人安心的情況，都不再起瞋恚之心，而要受持純淨、圓滿且唯一的天道。』

**◈ 如是法、僧、聖戒成就亦如是說。」**
　　對於憶念佛法、僧伽以及聖潔戒律的成就，也是同樣的道理。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **念如來事 (Buddhanussati)**：指「佛隨念」，一種修行方法，透過反覆憶念佛陀的十種殊勝特質（如正遍知、世間解等），將心靈從雜念中抽離，轉向純淨與喜悅。
2. **三昧定 (Samādhi)**：指心意識高度集中於單一對象，不再散亂的狀態。在此經文中，它是從情緒喜悅 ➔ 身體放鬆 ➔ 心理快樂之後自然產生的定力。
3. **無恚 (Adosa)**：指沒有瞋恨心。在原始佛教中，瞋恚被視為最劇烈的煩惱之一，而「無恚」則是通往高尚境界（天道）的核心條件。
4. **隨喜 (Muditā)**：指看到他人獲得成功或憶念聖者功德時，心中產生的純粹歡喜，不含嫉妒。

### [現代實作指南]
本經提供了一套從「認知」到「生理」再到「心理」的轉念路徑，可用於現代壓力管理與情緒調節：

1. **正向錨定（憶念功德）**：
   當心中充滿焦慮或負面情緒時，不要強行壓抑，而是將注意力轉移到一個「完美的榜樣」或「崇高的價值」上（如佛陀的慈悲與智慧）。這在心理學上類似於「正向聚焦」，用高品質的認知取代低品質的煩惱。

2. **生理反饋鏈條（隨喜 ➔ 歡悅 ➔ 猗息）**：
   意識到正向價值 ➔ 產生喜悅感 ➔ 身體肌肉放鬆（猗息） ➔ 產生愉悅的感官體驗。實作時，可嘗試在深呼吸中感受這種從心到身的放鬆過程，讓身體進入「安全模式」。

3. **建立情緒防火牆（無恚實踐）**：
   在心靈平靜（三昧定）的狀態下，對自己下達一個明確的指令：「無論環境是恐懼（壓力）還是安穩（舒適），我都不採取瞋恚的反應」。這是一種預先設定的心理機制，將「不憤怒」作為一種生活準則而非臨時反應。

4. **擴展對象**：
   將此方法從憶念佛陀，擴展到憶念正確的價值觀（法）、良師益友（僧）或自律的準則（戒），建立多維度的心理支持系統。

### [段落大意]
本經闡述了透過「憶念佛陀功德」來達到心靈清淨的修行路徑。聖弟子藉由憶念佛陀的十種特質，引發隨喜與歡悅，進而使身體放鬆並進入三昧定。在定境中，修行者體悟到「無瞋恚」是最高尚的道途，並誓願在任何環境下都不起瞋心，以維持純淨的心靈境界。此方法同樣適用於對法、僧、戒的憶念。
