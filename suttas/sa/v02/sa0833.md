# 雜阿含經 第 833 經

### [原文]
如是我聞：

一時，佛住毘舍離國獼猴池側重閣講堂。

時，有善調象師離車，名曰難陀，來詣佛所，稽首佛足，退坐一面。

爾時，世尊告離車難陀言：「若聖弟子成就四不壞淨者，欲求壽命，即得壽命，求好色、力、樂、辯、自在即得。何等為四？謂佛不壞淨成就，法、僧不壞淨，聖戒成就。我見是聖弟子於此命終，生於天上，於天上得十種法。何等為十？得天壽、天色、天名稱、天樂、天自在，天色、聲、香、味、觸。若聖弟子於天上命終，來生人中者，我見彼十事具足。何等為十？人間壽命、人好色、名稱、樂、自在、色、聲、香、味、觸。我說彼多聞聖弟子不由他信、不由他欲、不從他聞、不取他意、不因他思，我說彼有如實正慧知見。」

爾時，難陀有從者，白難陀言：「浴時已到，今可去矣！」

難陀答言：「我今不須人間澡浴，我今於此勝妙法以自沐浴，所謂於世尊所得清淨信樂。」

爾時，離車調象師難陀聞佛所說，歡喜隨喜，從座起，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住毘舍離國獼猴池側重閣講堂。**
　　在某一個時間，佛陀住在毘舍離國獼猴池旁邊的重閣講堂中。

**◈ 時，有善調象師離車，名曰難陀，來詣佛所，稽首佛足，退坐一面。**
　　那時，有一位來自離車族、擅長馴象的調象師，名叫難陀，他來到佛陀這裡，頂禮佛陀的足部，然後退到一邊坐下。

**◈ 爾時，世尊告離車難陀言：「若聖弟子成就四不壞淨者，欲求壽命，即得壽命，求好色、力、樂、辯、自在即得。**
　　那時，世尊告訴離車族的難陀說：「如果一位聖弟子成就了『四不壞淨』（四種不可毀壞的清淨信仰），若他想要長壽，就能獲得長壽；若他追求良好的相貌、強健的力量、快樂、辯才以及自在（隨心所欲的能力），也能夠獲得。」

**◈ 何等為四？謂佛不壞淨成就，法、僧不壞淨，聖戒成就。**
　　這四種清淨是什麼呢？指的是：對佛陀的清淨信仰成就、對法（真理）的清淨信仰成就、對僧伽（出家眾）的清淨信仰成就，以及聖戒（清淨的戒律）的成就。

**◈ 我見是聖弟子於此命終，生於天上，於天上得十種法。**
　　我觀察到，這樣的聖弟子在這次生命結束後，會投生在天界，並在天界中獲得十種利益。

**◈ 何等為十？得天壽、天色、天名稱、天樂、天自在，天色、聲、香、味、觸。**
　　這十種利益是什麼？即是獲得天界的壽命、天界的相貌、天界的名聲、天界的快樂、天界的自在，以及天界中極其美好的色（物質身體）、聲（聲音）、香（氣味）、味（味道）、觸（接觸感受）。

**◈ 若聖弟子於天上命終，來生人中者，我見彼十事具足。**
　　如果這位聖弟子在天界壽命結束後，再次投生在人間，我觀察到他依然能具足十種利益。

**◈ 何等為十？人間壽命、人好色、名稱、樂、自在、色、聲、香、味、觸。**
　　這十種利益是指：人間的長壽、良好的相貌、名聲、快樂、自在，以及美好的色（物質身體）、聲（聲音）、香（氣味）、味（味道）、觸（接觸感受）。

**◈ 我說彼多聞聖弟子不由他信、不由他欲、不從他聞、不取他意、不因他思，我說彼有如實正慧知見。」**
　　我說，這位博學的聖弟子不再依賴他人的信仰、不依賴他人的願望、不盲從他人的聽聞、不採取他人的想法、也不因為他人的思考而決定，我說他擁有如實的正確智慧與見解。」

**◈ 爾時，難陀有從者，白難陀言：「浴時已到，今可去矣！」**
　　那時，難陀的一位隨從對難陀說：「洗澡的時間到了，現在可以離開了！」

**◈ 難陀答言：「我今不須人間澡浴，我今於此勝妙法以自沐浴，所謂於世尊所得清淨信樂。」**
　　難陀回答說：「我現在不需要人間的身體洗浴，我現在正用這勝妙的法（真理）來為自己洗浴，也就是指我從世尊這裡獲得的清淨信仰與法喜。」

**◈ 爾時，離車調象師難陀聞佛所說，歡喜隨喜，從座起，作禮而去。**
　　那時，離車族的調象師難陀聽完佛陀的教導後，內心感到歡喜並隨喜，從座位上站起來，向佛陀行禮後才離開。

### [詞語註解]
1. **四不壞淨 (Catur-ariya-suddhi)**：指對佛、法、僧三寶的堅定信仰，以及對聖戒（道德律條）的嚴格實踐。這四者構成了一種不可被毀壞的心理清淨狀態，使修行者在面對誘惑或壓力時能保持內心的穩定。
2. **自在 (Vimokkha/Independence)**：在此處指一種心理上的自由與能力，能夠不受環境限制地達成目標或保持心境的安詳。
3. **如實正慧知見 (Yathābhūta-ñāṇadassana)**：指能夠如實地觀察現象的本質（如無常、苦、無我），而不被主觀的偏見、妄想或他人的意見所遮蔽的正確智慧。
4. **隨喜 (Anumodanā)**：指看到他人獲得正法或行善時，內心生起的純粹歡喜之情，這是一種消除嫉妒心、增加功德的修行方法。

### [現代實作指南]
本經文將「洗澡」從生理層面提升至心理與靈性層面，提供以下實踐方法：

1. **建立「內在洗浴」的習慣**：
   ➔ 當我們感到壓力、焦慮或被世俗瑣事（如名聲、外貌、物質欲望）困擾時，意識到這些是「心理的污垢」。
   ➔ 練習將注意力從外部的追求（如追求他人的認可）轉向內在的真理（如觀察事物的無常），將這種覺知視為一種「心靈的沐浴」，洗淨心中的貪婪與執著。

2. **練習「獨立思考」的正見**：
   ➔ 經文中提到「不由他信、不取他意」，在現代資訊爆炸的時代，我們極易被社交媒體或他人意見左右。
   ➔ 實作技巧：在做重大決定或形成價值觀前，先問自己：「這個想法是基於事實的觀察，還是因為我想獲得他人的認同？」嘗試從「依賴他人」轉向「依賴對真理的觀察」。

3. **培養「四不壞淨」的現代對應**：
   ➔ **對導師/真理的信任**：尋找一個能引領自己向善、向覺悟的對象或準則。
   ➔ **對實踐的信任**：相信正確的行動（如正念、正業）會帶來正向的結果。
   ➔ **對社群的信任**：與志同道合、能互相激勵的正向群體共處。
   ➔ **對自律的堅持**：建立一套個人的道德底線（聖戒），在任何情況下不違背良知。

### [段落大意]
佛陀向離車族調象師難陀說明，若聖弟子能成就對佛、法、僧的清淨信仰及聖戒（四不壞淨），不僅在現世能獲得長壽、美貌與自在，在天界與未來的人間生命中亦能具足十種利益，並擁有獨立於他人意見之外的如實正慧。難陀深受觸動，認為佛法帶來的清淨心比身體的洗浴更為殊勝。
