# 雜阿含經 第 542 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。時，尊者阿那律在舍衛國松林精舍。

時，有眾多比丘詣尊者阿那律所，共相問訊慰勞已，於一面坐，問尊者阿那律：「若比丘在於學地，上求安隱涅槃住，聖弟子云何修習多修習，於此法、律得盡諸漏，無漏心解脫、慧解脫，現法自知作證：『我生已盡，梵行已立，所作已作，自知不受後有。』？」

尊者阿那律語諸比丘言：「若比丘在於學地，上求安隱涅槃心住，聖弟子云何修習多修習？於此法、律得盡諸漏，無漏心解脫、慧解脫，現法自知作證：『我生已盡，梵行已立，所作已作，自知不受後有。』者？當住四念處。何等為四？謂內身身觀念處，乃至法法觀念處。如是四念處修習多修習，於此法、律得盡諸漏，無漏心解脫、慧解脫，現法自知作證：『我生已盡，梵行已立，所作已作，自知不受後有。』」

時，諸比丘共聞尊者阿那律所說，歡喜隨喜，各從座起而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。時，尊者阿那律在舍衛國松林精舍。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。那時，尊者阿那律住在舍衛國的松林精舍。

**◈ 時，有眾多比丘詣尊者阿那律所，共相問訊慰勞已，於一面坐，問尊者阿那律：**
　　那時，有許多比丘前往尊者阿那律那裡，在彼此問候慰勞之後，大家坐在一邊，請問尊者阿那律：

**◈ 「若比丘在於學地，上求安隱涅槃住，聖弟子云何修習多修習，於此法、律得盡諸漏，無漏心解脫、慧解脫，現法自知作證：『我生已盡，梵行已立，所作已作，自知不受後有。』？」**
　　「如果一名比丘處於學習階段（學地），心中追求著安穩且超越生死、絕對寧靜的解脫境界（涅槃），聖弟子應該如何進行大量的修行與修習，才能在佛法與律法中消除所有煩惱的滲漏（盡諸漏），達到心靈的解脫與智慧的解脫，並在目前的生命中親自證悟：『我的輪迴出生已經結束，清淨的修行生活已經建立，應該做的事已經完成，自知不再有後世的出生』？」

**◈ 尊者阿那律語諸比丘言：「若比丘在於學地，上求安隱涅槃心住，聖弟子云何修習多修習？於此法、律得盡諸漏，無漏心解脫、慧解脫，現法自知作證：『我生已盡，梵行已立，所作已作，自知不受後有。』者？**
　　尊者阿那律告訴眾位比丘說：「關於一名比丘處於學習階段，心中追求著安穩且超越生死、絕對寧靜的解脫境界（涅槃），聖弟子應該如何進行大量的修行與修習，才能在佛法與律法中消除所有煩惱的滲漏，達到心靈的解脫與智慧的解脫，並在目前的生命中親自證悟：『我的輪迴出生已經結束，清淨的修行生活已經建立，應該做的事已經完成，自知不再有後世的出生』這件事，

**◈ 當住四念處。何等為四？謂內身身觀念處，乃至法法觀念處。**
　　答案是：應當安住於四種正念的確立（四念處）。這四種是什麼？指的是：觀照內在身體的身體念處、感受念處、心念處，以及觀照一切現象的法（現象）念處。」

**◈ 如是四念處修習多修習，於此法、律得盡諸漏，無漏心解脫、慧解脫，現法自知作證：『我生已盡，梵行已立，所作已作，自知不受後有。』」**
　　「如此對這四種正念的確立進行大量的修行與修習，就能在佛法與律法中消除所有煩惱的滲漏，達到心靈的解脫與智慧的解脫，並在目前的生命中親自證悟：『我的輪迴出生已經結束，清淨的修行生活已經建立，應該做的事已經完成，自知不再有後世的出生』。」

**◈ 時，諸比丘共聞尊者阿那律所說，歡喜隨喜，各從座起而去。**
　　那時，眾比丘一同聽聞了尊者阿那律的教導，心中感到歡喜並隨喜贊嘆，隨後各自從座位上起身離開。

### [詞語註解（解釋重要佛教概念）]
1. **學地 (Sekha)**：指尚未達到阿羅漢果，但已進入聖道（如須陀洹、須陀洹果等），仍需在佛法指導下繼續學習與修行的聖弟子。
2. **漏 (Āsava)**：字面意為「滲漏」，在學術上指深層的、潛在的染污煩惱，主要分為欲漏（對感官欲望的執著）、有漏（對存在與生存的執著）、無明漏（對真理的無知）。
3. **四念處 (Satipaṭṭhāna)**：正念修行的四個核心對象，旨在透過覺察來破除對「我」的執著。包括：
    - **身念處**：覺察身體的動作、呼吸與物質組成。
    - **受念處**：覺察苦、樂、不苦不樂的感受。
    - **心念處**：覺察心的狀態（如貪、嗔、痴或專注、散亂）。
    - **法念處**：覺察心法運作的規律與現象（如五蘊、十二因緣）。
4. **梵行 (Brahmacariya)**：指清淨的修行生活，原意為「如梵天般的生活」，在佛教中指遠離世俗欲望、致力於解脫的聖潔生活。

### [現代實作指南]
本經文的核心在於將「四念處」作為解脫的唯一路徑。在現代生活中，我們可以將其轉化為「全方位的覺察練習」：

1. **身體覺察（身念處）** $\rightarrow$ **實作：身體掃描**
   - 在感到壓力或焦慮時，將注意力從大腦的思考轉移到身體。感受腳底接觸地面的感覺，或觀察呼吸時腹部的起伏。這能將意識從「想像的恐懼」拉回「當下的現實」。

2. **感受標記（受念處）** $\rightarrow$ **實作：情緒標記法**
   - 當強烈情緒升起時，不要說「我很生氣」，而要說「現在有一個『生氣』的感受在發生」。將「我」與「感受」分離，將感受視為一種暫時的生理與心理現象，而非自我的本質。

3. **心態觀察（心念處）** $\rightarrow$ **實作：心靈鏡像**
   - 像觀察天氣一樣觀察自己的心。當發現心在散亂時，輕輕地對自己說：「心現在散亂了」；當發現心在貪婪時，說：「心現在起貪心了」。不評判，僅僅是觀察。

4. **現象分析（法念處）** $\rightarrow$ **實作：因緣分析**
   - 當面對挫折時，分析這個現象是如何構成的。例如：這場爭吵是由於「對方的誤解（因）」加上「我的急躁（緣）」共同產生的。意識到一切現象都是因緣而生，而非某個人的錯，能有效降低對立感。

### [段落大意]
本經描述眾比丘請教尊者阿那律，關於處於學習階段的修行者如何才能消除煩惱、達成心慧解脫並終結輪迴。尊者阿那律明確指出，唯一的途徑是大量修習「四念處」（觀照身、受、心、法），透過對身心現象的持續覺察，方能證悟解脫，不再受後世輪迴之苦。
