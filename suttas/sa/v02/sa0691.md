# 雜阿含經 第 691 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有七力。何等為七？信力、精進力、慚力、愧力、念力、定力、慧力。何等為信力？於如來所起信心，深入堅固，諸天、魔、梵、沙門、婆羅門及餘同法所不能壞，是名信力。何等為精進力？謂四正斷……」如上廣說。「何等為慚力？謂恥惡不善法……」如上說。「何等為愧力？於可愧事愧，愧起惡不善法……」如上說。「何等為念力？謂四念處……」如上說。「何等為定力？謂四禪……」如上說。「何等為慧力？謂四聖諦……」如上說。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「有七力。何等為七？信力、精進力、慚力、愧力、念力、定力、慧力。」**
　　那時，世尊告訴在場的眾位比丘說：「有七種內在的精神力量（七力）。這七種是什麼？分別是：信力（堅定的信心）、精進力（不懈的努力）、慚力（對自己的羞愧）、愧力（對他人的愧疚）、念力（持續的覺知）、定力（專注的心念）以及慧力（洞察真理的智慧）。」

**◈ 何等為信力？於如來所起信心，深入堅固，諸天、魔、梵、沙門、婆羅門及餘同法所不能壞，是名信力。**
　　什麼是信力？是指對成就不變真理而來到世間的人（如來）所產生的信心，這種信心深入且堅固，無論是天神、魔王、梵天、沙門、婆羅門或其他同樣修行法的人，都無法將其摧毀，這就稱為信力。

**◈ 何等為精進力？謂四正斷……」如上廣說。**
　　什麼是精進力？是指四正斷（即：斷除已生之惡、防止未生之惡、生起未生之善、增長已生之善）……佛陀就此詳細地說明。

**◈ 「何等為慚力？謂恥惡不善法……」如上說。**
　　什麼是慚力？是指對自己心中產生的惡劣、不善之法感到羞愧……佛陀就此說明。

**◈ 「何等為愧力？於可愧事愧，愧起惡不善法……」如上說。**
　　什麼是愧力？是指對於應該感到愧疚的事情而感到愧疚，並因此而對產生惡劣、不善之法感到愧疚……佛陀就此說明。

**◈ 「何等為念力？謂四念處……」如上說。**
　　什麼是念力？是指四念處（即：對身體的覺知、對感受的覺知、對心的覺知、對法之運作的覺知）……佛陀就此說明。

**◈ 「何等為定力？謂四禪……」如上說。**
　　什麼是定力？是指四禪（即：四種深度的禪定狀態）……佛陀就此說明。

**◈ 「何等為慧力？謂四聖諦……」如上說。**
　　什麼是慧力？是指對四聖諦（即：苦、苦因、苦止、解脫之道）的深刻洞察……佛陀就此說明。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **信力 (Saddhā-bala)**：並非盲信，而是基於對真理的初步認可與信任，形成一種能對抗外界干擾、穩定心神的精神力量。
2. **慚 (Hiri)**：一種內在的道德自律，指對自己的不善行為感到羞愧，是自省的力量。
3. **愧 (Ottappa)**：一種對外部規範或他人評價的敬畏，指因擔心不善行為會帶來惡果或被他人知曉而感到愧疚。
4. **四念處 (Satipaṭṭhāna)**：修行覺知的四個對象，包含觀身、觀受、觀心、觀法，旨在打破對「我」的執著。
5. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四個真理，即：苦（生命本質的不滿足）、集（苦的根源是貪愛）、滅（苦的止息即是涅槃）、道（達成止息的八正道）。

### [現代實作指南]
本經提出的「七力」可視為一套完整的心理韌性與道德建設系統，可在生活中如此實踐：

1. **建立核心價值（信力）**：在混亂的資訊時代，尋找一個經過驗證的真理或核心價值觀（如：慈悲、誠實），將其深植於心，使其在面對外界壓力或誘惑時不被輕易動搖。
2. **建立正向反饋迴路（精進力）**：實踐「四正斷」。當發現壞習慣時立即停止（斷已生惡），在誘惑出現前建立防線（止未生惡），主動尋找正確的行為模式（生未生善），並持續強化好的習慣（增已生善）。
3. **內在道德監控（慚與愧）**：
    *   **慚** ➔ 建立「自我誠實」機制。在沒人看見時，依然對自己的不誠實或自私感到不安，以此驅動自我修正。
    *   **愧** ➔ 建立「社會責任」意識。意識到自己的行為會對他人與環境產生影響，利用這種責任感來抑制衝動。
4. **練習正念覺知（念力）**：在日常生活中練習「標記」。例如，當憤怒升起時，在心中輕聲標記「這是感受（受）」，而非陷入「我很憤怒」的認同中。
5. **深度專注訓練（定力）**：每天撥出 10-20 分鐘，將注意力完全集中於呼吸，練習在心念飄移時溫柔地將其拉回，培養心靈的穩定度。
6. **邏輯分析苦因（慧力）**：面對壓力或痛苦時，不採取情緒化反應，而是運用四聖諦分析：目前的痛苦是什麼（苦） ➔ 導致這個痛苦的執著是什麼（集） ➔ 如果放下這個執著會如何（滅） ➔ 現在我該採取什麼具體行動（道）。

### [段落大意]
本經由佛陀向比丘說明修行者應具備的七種精神力量（七力）：信、精進、慚、愧、念、定、慧。這七力涵蓋了從建立信心、道德自律、正念覺知到最終洞察真理的完整路徑，旨在幫助修行者在面對外界干擾時能保持堅定，最終透過對四聖諦的體悟達成解脫。
