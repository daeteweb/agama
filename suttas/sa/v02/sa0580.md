# 雜阿含經 第 580 經

### [原文]
如是我聞：

一時，佛在舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。

時，彼天子而說偈言：

「以法善調伏，　　不隨於諸見， 　雖復著睡眠，　　則能隨時悟。」

爾時，世尊說偈答言：

「若以法調伏，　　不隨餘異見， 　無知已究竟，　　能度世恩愛。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛在舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。**
　　那時，有一位相貌極其出眾的天子，在深夜時分來到佛陀所在的住所，頂禮佛陀的足部，他的身體散發出各種光明，照亮了整個祇樹給孤獨園。

**◈ 時，彼天子而說偈言：**
　　那時，那位天子說了一首偈頌：

**◈ 「以法善調伏，不隨於諸見，雖復著睡眠，則能隨時悟。」**
　　「透過佛法（真理）來良好地調伏心靈，不隨順於各種錯誤的見解，這樣即使陷入睡眠之中，也能在任何時刻保持覺悟。」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「若以法調伏，不隨餘異見，無知已究竟，能度世恩愛。」**
　　「如果能以佛法（真理）來調伏心靈，不隨順於其他不同的錯誤見解，且不自以為已經達到究竟（終點），就能度過世間的恩愛（執著與貪愛）。」

**◈ 時，彼天子復說偈言：**
　　那時，那位天子再次說了一首偈頌：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「長久以來看見那些婆羅門，直到他們獲得了涅槃（超越生死、絕對寧靜的解脫境界），所有的恐懼都已經過去，永遠超越了世間的恩愛（執著與貪愛）。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，那位天子聽聞佛陀所說的教法，心生法喜並隨喜，頂禮佛陀的足部後，立即消失不再出現。

### [詞語註解]
1. **法 (Dhamma)**：指佛陀所悟出的宇宙真理，以及以此真理所建立的教導，旨在指引眾生脫離苦海。
2. **見 (Diṭṭhi)**：指對事物本質的看法或觀點。在原始佛教中，通常指那些導致執著、阻礙覺悟的「錯見」或「偏見」。
3. **調伏 (Dama)**：指透過修行與正念，將躁動、貪婪或嗔恨的心靈馴服，使其回歸平靜與清明。
4. **涅槃 (Nibbāna)**：超越生死、絕對寧靜的解脫境界，指貪、嗔、痴三毒完全熄滅的狀態。
5. **世恩愛 (Samsaric attachment)**：指在輪迴世間中對親人、財產、自我等產生的強烈情感依戀與執著，這是導致輪迴苦楚的核心原因。

### [現代實作指南]
本經文強調「調伏心靈」與「不執著於見解」的重要性。在現代生活中，我們可以透過以下步驟實踐：

1. **覺察「我的看法」 (解構見解)**：
   當我們在爭論或感到憤怒時，通常是因為我們執著於某個「見解」（例如：「我認為這才是正確的」）。實作時，請在心中對自己說：「這只是一個看法，而不是絕對的真理。」將「我的看法」轉化為「一個觀察到的念頭」，從而打破對特定見解的隨順。

2. **建立「隨時悟」的正念 (調伏心靈)**：
   天子提到「雖著睡眠，則能隨時悟」。這在現代可轉化為「在自動導航模式中覺醒」。我們每天有許多時間處於無意識的習慣中（如刷手機、通勤）。嘗試在這些時刻設定「覺醒錨點」，例如每當觸碰門把或喝水時，立刻意識到自己的呼吸，將心靈從慣性中調伏回來。

3. **放下「已究竟」的傲慢 (持續精進)**：
   世尊提醒「無知已究竟」。在學習或修行中，最危險的是認為自己「懂了」或「達到了」。實作技巧是保持「初學者之心」(Beginner's Mind)，對任何經驗保持開放與謙卑，避免因自以為是的成就感而陷入新的執著。

4. **處理「恩愛」的執著 (轉化依戀)**：
   將對他人的「佔有式愛」轉化為「慈悲式愛」。意識到所有關係都是緣起（因緣而生）且無常的，練習在愛情的同時，接納對方的獨立性與最終的離別，減少因執著而產生的恐懼與痛苦。

### [段落大意]
一位天子深夜拜訪佛陀，透過偈頌討論以佛法調伏心靈、不執著於錯誤見解的重要性，認為如此能使人在任何狀態下保持覺悟。世尊進一步補充，若能不自以為達到終點且不隨順異見，方能超越世間的執著與貪愛。天子隨後讚嘆獲得涅槃者能永離恐懼與執著，最後在法喜中頂禮消失。
