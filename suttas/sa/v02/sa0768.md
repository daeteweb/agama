# 雜阿含經 第 768 經

### [原文]
如是我聞：

一時，佛住王舍城山谷精舍。

時，尊者阿難獨一靜處，作如是念：「半梵行者，謂善知識、善伴黨、善隨從。」乃至。佛告阿難：「純一滿淨具梵行者，謂善知識。所以者何？我為善知識故，令諸眾生修習正見，依遠離、依無欲、依滅、向於捨，乃至修正定，依遠離、依無欲、依滅、向於捨。」

佛說此經已，尊者阿難聞佛所說，歡喜奉行。

---

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住王舍城山谷精舍。**
　　在某一個時間，佛陀住在王舍城的山谷精舍中。

**◈ 時，尊者阿難獨一靜處，作如是念：「半梵行者，謂善知識、善伴黨、善隨從。」乃至。**
　　那時，尊者阿難獨自處於一個安靜的地方，產生了這樣的念頭：「那些修行尚未圓滿的人（半梵行者），是指那些被稱為善知識、好的同伴、好的隨從的人。」（後略）。

**◈ 佛告阿難：「純一滿淨具梵行者，謂善知識。**
　　佛陀告訴阿難說：「真正純淨、圓滿且具足梵行（清淨修行）的人，才稱得上是善知識（良師益友）。

**◈ 所以者何？我為善知識故，令諸眾生修習正見，**
　　為什麼呢？因為我（佛陀）就是真正的善知識，我引導所有有情（擁有意識與生命的眾生）修習正見（正確的見地）。

**◈ 依遠離、依無欲、依滅、向於捨，乃至修正定，依遠離、依無欲、依滅、向於捨。」**
　　讓他們依循遠離（對煩惱的遠離）、依循無欲（對貪愛的捨離）、依循滅（苦的止息），並趨向於捨（平靜且不執著的狀態），直到修習正定（專注且正確的禪定），同樣地依循遠離、依循無欲、依循滅，並趨向於捨。」

**◈ 佛說此經已，尊者阿難聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，尊者阿難聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **梵行 (Brahmacariya)**：原意為「梵天般的行持」，在原始佛教中指清淨的修行生活，旨在遠離感官欲望，追求精神的絕對純淨與解脫。
2. **善知識 (Kalyāṇa-mitta)**：指能引導修行者走向正道、提供正確指導且能激勵其進步的良師益友。
3. **正見 (Sammā-diṭṭhi)**：八正道之首，指對四聖諦（苦、集、滅、道）的正確認知與理解。
4. **正定 (Sammā-samādhi)**：八正道之末，指心意識集中於單一對象而不再散亂的正確禪定狀態。
5. **捨 (Upekkhā)**：一種中立、平靜且不偏不倚的心態，不對愉快的感受產生貪愛，也不對痛苦的感受產生厭惡。

### [現代實作指南]
本經文強調了「真正的導師」與「表面陪伴者」的區別。在現代生活中，我們可以將其轉化為以下實踐：

1. **重新定義「良友」**：
   ➔ 區分「情緒價值提供者」與「成長導引者」。真正的善知識並非僅僅是讓我們感到舒適、認同我們的人，而是能指出我們的盲點，引導我們走向「遠離煩惱」與「內心平靜」的人。

2. **建立「遠離 ➔ 無欲 ➔ 止息」的心理路徑**：
   ➔ 當面對強烈的情緒衝擊（如憤怒或貪婪）時，練習以下步驟：
     - **遠離**：意識到情緒的產生，在心理上與該情緒拉開距離（不認同該情緒就是「我」）。
     - **無欲**：觀察自己想要掌控局面或獲取的欲望，並意識到這種欲望正是痛苦的來源，嘗試放下對結果的執著。
     - **止息**：當欲望停止，情緒自然會趨於平息。
     - **趨向捨**：維持一種「無論結果如何，我都能保持內心平靜」的觀照狀態。

3. **從正見到正定的實踐**：
   ➔ 先建立正確的認知（正見：理解事物是無常且因緣而生的），再將注意力集中於當下的呼吸或覺知（正定），使心靈不再被外界的雜訊牽引。

### [段落大意]
本段記載阿難對「善知識」的定義產生疑問，佛陀隨即予以糾正。佛陀指出，真正的善知識必須是具足圓滿清淨修行的人，其核心功能在於引導眾生透過正見與正定，經歷遠離、無欲與滅苦的過程，最終達到平靜不執著的「捨」之境界。
