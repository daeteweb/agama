# 雜阿含經 第 947 經

### [原文]
如是我聞：

一時，佛住王舍城毘富羅山。

爾時，世尊告諸比丘：「有一人於一劫中生死輪轉，積累白骨不腐壞者，如毘富羅山。若多聞聖弟子此苦聖諦如實知，此苦集聖諦如實知，此苦滅聖諦如實知，此苦滅道跡聖諦如實知，彼如是知、如是見，斷三結，謂身見、戒取、疑，斷此三結，得須陀洹，不墮惡趣法，決定正向三菩提，七有天人往生，究竟苦邊。」

爾時，世尊即說偈言：

「一人一劫中，　　積聚其身骨， 　常積不腐壞，　　如毘富羅山。 　若諸聖弟子，　　正智見真諦， 　此苦及苦因，　　離苦得寂滅。 　修習八道跡，　　正向般涅槃， 　極至於七有，　　天人來往生， 　盡一切諸結，　　究竟於苦邊。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城毘富羅山。**
　　在某一個時間，佛陀住在王舍城的毘富羅山。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「有一人於一劫中生死輪轉，積累白骨不腐壞者，如毘富羅山。**
　　「有一個人在一劫（極長的時間單位）之中，在生死輪迴（Saṃsāra）中不斷地出生與死亡，如果他累計的所有白骨不腐爛，其堆積的高度將會像毘富羅山一樣高。

**◈ 若多聞聖弟子此苦聖諦如實知，此苦集聖諦如實知，此苦滅聖諦如實知，此苦滅道跡聖諦如實知，**
　　如果一位博學的聖弟子能如實地知道這「苦聖諦（關於苦的真理）」、如實地知道這「苦集聖諦（關於苦之原因的真理）」、如實地知道這「苦滅聖諦（關於苦之止息的真理）」以及如實地知道這「苦滅道跡聖諦（關於解脫之道的真理）」。

**◈ 彼如是知、如是見，斷三結，謂身見、戒取、疑，**
　　他若能這樣地認知、這樣地觀察，就能斷除三種結（束縛心的煩惱），也就是：身見（對物質身體與心的錯誤執著）、戒取（對錯誤戒律或儀軌的執著）、以及疑（對正法與修行路徑的懷疑）。

**◈ 斷此三結，得須陀洹，不墮惡趣法，決定正向三菩提，七有天人往生，究竟苦邊。」**
　　斷除這三種結之後，便能成為須陀洹（入流果位，即初步進入解脫之流的人），不再墮入惡趣（如地獄、餓鬼、畜生等痛苦的生存狀態），確定會正向著最高、最圓滿的智慧與覺悟 (阿耨多羅三藐三菩提) 前進，最多在七次天人道的往生之中，最終將到達苦的邊緣（即徹底結束痛苦）。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即說了偈頌：

**◈ 「一人一劫中，積聚其身骨，常積不腐壞，如毘富羅山。**
　　「一個人在一個劫的時間裡，累計其身體的白骨，若這些骨頭不腐爛，將會像毘富羅山一樣高。

**◈ 若諸聖弟子，正智見真諦，此苦及苦因，離苦得寂滅。**
　　如果聖弟子們能以正確的智慧見到真理，了解這苦以及苦的原因，就能離開苦而獲得寂滅（涅槃）。

**◈ 修習八道跡，正向般涅槃，極至於七有，天人來往生，**
　　修習八正道（解脫之道），正向著超越生死、絕對寧靜的解脫境界 (涅槃) 前進，最多在七次天人的往生之中。

**◈ 盡一切諸結，究竟於苦邊。」**
　　盡除一切煩惱的束縛，最終到達苦的盡頭。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **劫 (Kappa)**：佛教中用來衡量極長時間的單位，用以比喻輪迴之久遠。
2. **聖諦 (Ariya-sacca)**：指佛陀覺悟的四個真理（苦、集、滅、道），是修行解脫的核心框架。
3. **三結 (Samyojana)**：指將眾生束縛在輪迴中的煩惱。本經提到的三結是進入須陀洹果位必須斷除的最低限度束縛 ➔ 身見（執著我）、戒取（執著外在形式）、疑（對真理的猶豫）。
4. **須陀洹 (Sotāpanna)**：意為「入流者」，指修行者已初步證悟真理，確定不再退轉，且最多在七次輪迴內達成完全解脫。
5. **七有 (Satta-bhava)**：指須陀洹果位者在完全解脫前，最多在七個不同的生存狀態（通常指天人道）中往生。

### [現代實作指南]
本經透過「白骨如山」的強烈視覺意象，旨在打破我們對「恆常自我」的幻覺。在現代生活中，可採取以下實踐方法：

1. **觀照身體的無常（對治身見）**：
   - 當我們過度在意外貌、追求身體的永恆或陷入對身體疾病的極度恐懼時，可以嘗試想像本經中的「白骨堆」。
   - 意識到目前的身體僅是因緣而生的暫時組合，而非一個永恆不變的「我」。這種轉念能有效降低對物質身體的執著，減輕因老化與病痛帶來的心理壓力。

2. **邏輯化分析痛苦（實踐四聖諦）**：
   - 面對生活中的壓力或痛苦時，不要僅僅地感受痛苦，而要嘗試將其「客體化」：
     - **苦** ➔ 承認現在正處於痛苦狀態（不逃避）。
     - **集** ➔ 追溯這個痛苦是由於什麼貪愛或執著引起的（找原因）。
     - **滅** ➔ 相信只要原因消失，痛苦必然會止息（建立希望）。
     - **道** ➔ 採取具體的正向行動（如正念、正語）來處理問題（執行方案）。

3. **減少對形式主義的依賴（對治戒取）**：
   - 檢視自己是否在生活中過度執著於某些「必須這樣做才能成功/獲救」的僵化儀式或迷信，將注意力從「形式」轉移到「心靈的淨化」與「智慧的增長」上。

### [段落大意]
本經以一劫中累計的白骨如山來比喻輪迴之苦的深重。佛陀教導比丘，若能如實知曉四聖諦並修習八正道，即可斷除身見、戒取、疑三結，證得須陀洹果位。此後將不再墮入惡趣，且在最多七次往生中徹底結束輪迴，達到涅槃的解脫境界。
