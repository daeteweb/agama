# 雜阿含經 第 985 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有從愛生愛，從愛生恚；從恚生愛，從恚生恚。

「云何為從愛生愛？謂有一於眾生有喜、有愛、有念、有可意，他於彼有喜、有愛、有念、有可意隨行此，作是念：『我於彼眾生有喜、有愛、有念、有可意隨行故，我於他人復生於愛。』是名從愛生愛。

「云何從愛生恚？謂有一於眾生有喜、有愛、有念、有可意，而他於彼不喜、不愛、不念、不可意隨行此，作是念：『我於眾生有喜、有愛、有念、有可意，而他於彼不喜、不愛、不念、不可意隨行故，我於他而生瞋恚。』是名從愛生恚。

「云何為從恚生愛？謂有一於眾生不喜、不愛、不念、不可意，他復於彼不喜、不愛、不念、不可意隨行故，我於他而生愛念，是名從恚生愛。

「云何從恚生恚？謂有一於眾生不喜、不愛、不念、不可意，而他於彼有喜、有愛、有念、有可意隨行此，作是念：『我於彼眾生不喜、不愛、不念、不可意，而他於彼有喜、有愛、有念、有可意隨行，我於他所問起瞋恚。』是名從恚生恚。

「若比丘離欲、惡不善法，有覺有觀，乃至初禪、第二、第三、第四禪具足住者，從愛生愛、從恚生恚、從恚生愛、從愛生恚已斷已知，斷其根本，如截多羅樹頭，無復生分，於未來世成不生法。

「若彼比丘盡諸有漏，無漏心解脫、慧解脫，現法自知作證：『我生已盡，梵行已立，所作已作，自知不受後有。』當於爾時，不自舉，不起塵，不熾然，不嫌彼。

「云何自舉？謂見色是我、色異我、我中色、色中我。受、想、行、識亦復如是，是名自舉。云何不自舉？謂不見色是我、色異我、我中色、色中我。受、想、行、識亦復如是，是名不自舉。云何還舉？謂於罵者還罵，瞋者還瞋，打者還打，觸者還觸，是名還舉。云何不還舉？謂罵者不還罵，瞋者不還瞋，打者不還打，觸者不還觸，是名不還舉。

「云何起塵？謂有我、我欲，乃至十八種愛，是名起塵。云何不起塵？謂無我、無我欲，乃至十八愛不起，是名不起塵。

「云何熾然？謂有我所、我所欲，乃至外十八愛行，是名熾然。云何不熾然？謂無我所、無我所欲，乃至無外十八愛行，是名不熾然。

「云何嫌彼？謂見我真實起，於我慢、我欲、我使不斷不知，是名嫌彼。云何不嫌彼？謂不見我真實，我慢、我欲、我使已斷已知，是名不嫌彼。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有從愛生愛，從愛生恚；從恚生愛，從恚生恚。**
　　那時，世尊告訴在場的眾位比丘說：「世間存在著因為貪愛（愛）而產生貪愛、因為貪愛而產生瞋恚（恚）；以及因為瞋恚而產生貪愛、因為瞋恚而產生瞋恚的情況。

**◈ 云何為從愛生愛？謂有一於眾生有喜、有愛、有念、有可意，他於彼有喜、有愛、有念、有可意隨行此，作是念：『我於彼眾生有喜、有愛、有念、有可意隨行故，我於他人復生於愛。』是名從愛生愛。**
　　什麼叫做「從貪愛而產生貪愛」？是指一個人對某個眾生感到歡喜、產生貪愛、心心念念且覺得對方很合意，而對方對他同樣感到歡喜、產生貪愛、心心念念且覺得他很合意。於是這個人這樣想：『因為我對那個眾生有歡喜與貪愛，而他對我也同樣有歡喜與貪愛，所以我對他產生了更深的貪愛。』這就叫做從貪愛而產生貪愛。

**◈ 云何從愛生恚？謂有一於眾生有喜、有愛、有念、有可意，而他於彼不喜、不愛、不念、不可意隨行此，作是念：『我於眾生有喜、有愛、有念、有可意，而他於彼不喜、不愛、不念、不可意隨行故，我於他而生瞋恚。』是名從愛生恚。**
　　什麼叫做「從貪愛而產生瞋恚」？是指一個人對某個眾生感到歡喜、產生貪愛、心心念念且覺得對方很合意，但對方對他卻不歡喜、沒有貪愛、不掛念且覺得他不合意。於是這個人這樣想：『我對這個眾生有歡喜與貪愛，他卻對我不歡喜且不貪愛，因此我對他產生了瞋恚。』這就叫做從貪愛而產生瞋恚。

**◈ 云何為從恚生愛？謂有一於眾生不喜、不愛、不念、不可意，他復於彼不喜、不愛、不念、不可意隨行故，我於他而生愛念，是名從恚生愛。**
　　什麼叫做「從瞋恚而產生貪愛」？是指一個人對某個眾生不歡喜、沒有貪愛、不掛念且覺得對方不合意，而對方對他同樣不歡喜、沒有貪愛、不掛念且覺得他不合意。因為這種共同的厭惡感，我反而對他產生了親近的貪愛念頭。這就叫做從瞋恚而產生貪愛。

**◈ 云何從恚生恚？謂有一於眾生不喜、不愛、不念、不可意，而他於彼有喜、有愛、有念、有可意隨行此，作是念：『我於彼眾生不喜、不愛、不念、不可意，而他於彼有喜、有愛、有念、有可意隨行，我於他所問起瞋恚。』是名從恚生恚。**
　　什麼叫做「從瞋恚而產生瞋恚」？是指一個人對某個眾生不歡喜、沒有貪愛、不掛念且覺得對方不合意，但對方對他卻感到歡喜、產生貪愛、心心念念且覺得他很合意。於是這個人這樣想：『我對這個眾生不歡喜且不貪愛，他卻對我產生貪愛與歡喜，這讓我對他更加瞋恚。』這就叫做從瞋恚而產生瞋恚。

**◈ 若比丘離欲、惡不善法，有覺有觀，乃至初禪、第二、第三、第四禪具足住者，從愛生愛、從恚生恚、從恚生愛、從愛生恚已斷已知，斷其根本，如截多羅樹頭，無復生分，於未來世成不生法。**
　　如果比丘能夠遠離欲望、厭離不善法，具足覺知與觀察（覺有觀），並能完整地安住在初禪、第二禪、第三禪、第四禪之中，那麼上述的「從愛生愛、從恚生恚、從恚生愛、從愛生恚」這些情緒循環就已被斷除並被認知。這就像砍斷多羅樹的頂端一樣，徹底斷除其根本，使其不再生長，在未來世中成就不再輪迴的法。

**◈ 若彼比丘盡諸有漏，無漏心解脫、慧解脫，現法自知作證：『我生已盡，梵行已立，所作已作，自知不受後有。』當於爾時，不自舉，不起塵，不熾然，不嫌彼。**
　　如果那位比丘盡除了所有有漏（被煩惱染污）的狀態，達到心靈的解脫與智慧的解脫，在現法中自知並作證：『我的輪迴出生已盡，清淨的梵行已建立，應該做的事已做完，自知不再受後世的出生。』在那時，他不會自舉（執著自我），不會起塵（產生我欲），不會熾然（被貪欲焚燒），也不會嫌彼（執著我慢）。

**◈ 云何自舉？謂見色是我、色異我、我中色、色中我。受、想、行、識亦復如是，是名自舉。**
　　什麼叫做「自舉」？是指認為色（物質身體）就是我、色（物質身體）與我不同、我在色（物質身體）之中、或色（物質身體）在我之中。對於受（感受）、想（想像與認知）、行（心的造作）、識（意識）也同樣這樣認為，這就叫做自舉。

**◈ 云何不自舉？謂不見色是我、色異我、我中色、色中我。受、想、行、識亦復如是，是名不自舉。**
　　什麼叫做「不自舉」？是指不認為色（物質身體）是我、色（物質身體）與我不同、我在色（物質身體）之中、或色（物質身體）在我之中。對於受（感受）、想（想像與認知）、行（心的造作）、識（意識）也同樣不這樣認為，這就叫做不自舉。

**◈ 云何還舉？謂於罵者還罵，瞋者還瞋，打者還打，觸者還觸，是名還舉。**
　　什麼叫做「還舉」？是指對於辱罵自己的人回以辱罵，對於瞋怒自己的人回以瞋怒，對於打擊自己的人回以打擊，對於觸碰自己的人回以觸碰，這就叫做還舉。

**◈ 云何不還舉？謂罵者不還罵，瞋者不還瞋，打者不還打，觸者不還觸，是名不還舉。**
　　什麼叫做「不還舉」？是指對於辱罵自己的人不回以辱罵，對於瞋怒自己的人不回以瞋怒，對於打擊自己的人不回以打擊，對於觸碰自己的人不回以觸碰，這就叫做不還舉。

**◈ 云何起塵？謂有我、我欲，乃至十八種愛，是名起塵。**
　　什麼叫做「起塵」？是指心中還存在著「我」的概念以及「我的欲望」，乃至於十八種貪愛，這就叫做起塵。

**◈ 云何不起塵？謂無我、無我欲，乃至十八愛不起，是名不起塵。**
　　什麼叫做「不起塵」？是指心中沒有「我」的概念，也沒有「我的欲望」，乃至於十八種貪愛都不再升起，這就叫做不起塵。

**◈ 云何熾然？謂有我所、我所欲，乃至外十八愛行，是名熾然。**
　　什麼叫做「熾然」？是指心中有「我的所有物」以及「我所渴望的東西」，乃至於對外在十八種貪愛的追求行為，這就叫做熾然。

**◈ 云何不熾然？謂無我所、無我所欲，乃至無外十八愛行，是名不熾然。**
　　什麼叫做「不熾然」？是指心中沒有「我的所有物」，也沒有「我所渴望的東西」，乃至於沒有對外在十八種貪愛的追求行為，這就叫做不熾然。

**◈ 云何嫌彼？謂見我真實起，於我慢、我欲、我使不斷不知，是名嫌彼。**
　　什麼叫做「嫌彼」？是指認為有一個真實的「我」存在而升起執著，對於我慢（傲慢）、我欲（自我欲望）、我使（自我驅使）無法斷除且不自覺，這就叫做嫌彼。

**◈ 云何不嫌彼？謂不見我真實，我慢、我欲、我使已斷已知，是名不嫌彼。**
　　什麼叫做「不嫌彼」？是指不認為有一個真實的「我」存在，且我慢（傲慢）、我欲（自我欲望）、我使（自我驅使）已經被斷除並被認知，這就叫做不嫌彼。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **愛 (Taṇhā/Pema)**：在此指貪愛或情感上的依戀。它不僅是喜好，而是一種渴望獲得或維持某種狀態的心理驅動力，是導致輪迴與苦受的核心原因。
2. **恚 (Dosa)**：指瞋恚、厭惡或反感。它是對不合意對象的排斥反應，與貪愛相對，但兩者在心理機制上都屬於對現象的強烈反應。
3. **覺有觀 (Sati-sampajañña)**：指正念（Sati）與正知（Sampajañña）。「覺」是覺知當下的狀態，「觀」是以智慧觀察現象的無常、苦、無我本質。
4. **有漏 (Āsrava)**：指被煩惱（如貪、瞋、癡）所染污的心態。如同容器漏水，這些「漏」將眾生從解脫之門拉回輪迴之中。
5. **自舉 (Attā-upādāna/Self-lifting)**：一種將五蘊（色受想行識）誤認為「我」或「我的」之執著行為。
6. **多羅樹 (Tala tree)**：一種棕櫚科植物，其頂端被砍斷後仍可能發芽。佛陀以此比喻，若僅是壓制煩惱而未斷其根本，煩惱仍會復發；而真正的斷除則是如截斷樹頭使其再也無法生長。

### [現代實作指南]
本經文揭示了人類情感反應的四種邏輯循環，以及如何透過「去自我化」來終結情緒內耗。

**1. 識別情感循環 (Emotional Loop Recognition)**
當你感到強烈的情緒時，請嘗試將其對號入座：
- **正向強化**：我喜歡他 → 他也喜歡我 → 我更依戀他（容易導致過度依賴）。
- **期待落空**：我喜歡他 → 他不喜歡我 → 我感到憤怒（典型的受挫反應）。
- **同類共鳴**：我不喜歡他 → 他也不喜歡我 → 我覺得我們是一類人而產生好感（基於共同敵人的虛假親密）。
- **價值衝突**：我不喜歡他 → 他卻很喜歡我 → 我感到反感或壓力（對不合意關注的排斥）。
➔ **實作技巧**：在情緒升起時，對自己說：「這只是『從愛生恚』的循環」或「這是『從恚生愛』的循環」，將情緒客體化，從參與者變為觀察者。

**2. 練習「不還舉」 (Non-Reaction Practice)**
面對外界的攻擊（辱罵、瞋怒）時，意識到「還舉」只會延長衝突的因緣。
➔ **實作技巧**：當被攻擊時，在心中建立一個緩衝區。意識到對方的攻擊是他的「有漏」表現，而你的反應決定了是否要接手這個負面能量。練習「不還舉」即是切斷因果鏈條的開始。

**3. 拆解「自舉」的錯覺 (Deconstructing the Self)**
經文提到的「色是我、色異我」等，是在分析我們如何定義自我。
➔ **實作技巧**：當你感到「我被冒犯了」或「我的東西被搶了」時，分析這個「我」是由什麼構成的？
- 是物質身體（色）感到不適？
- 是某種不愉快感受（受）在運作？
- 是某種對對方的認知（想）在作祟？
- 是某種反擊的衝動（行）在驅使？
- 還是意識（識）在評判？
當你發現「我」只是五種要素的暫時組合，而非一個實體時，「自舉」的執著會自然減輕。

### [段落大意]
本經詳細分析了貪愛與瞋恚如何相互轉化並形成四種情緒循環，導致眾生在情感中受苦。佛陀指出，唯有透過離欲、覺觀及禪定，才能從根本上斷除這些情緒根源。最後，佛陀定義了「自舉」、「還舉」、「起塵」、「熾然」與「嫌彼」等執著自我的表現，強調解脫的關鍵在於不再將五蘊視為「我」，並停止對外界刺激的對立反應，從而達到無漏的解脫境界。
