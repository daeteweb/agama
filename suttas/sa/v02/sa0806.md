# 雜阿含經 第 806 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊於晨朝時著衣持鉢，入舍衛城乞食。食已，還精舍，舉衣鉢，洗足已，持尼師檀入安陀林，坐一樹下，晝日禪思。

時，尊者罽賓那亦晨朝時著衣持鉢，入舍衛城乞食。還，舉衣鉢，洗足已，持尼師檀入安陀林，於樹下坐禪，去佛不遠，正身不動，身心正直，勝妙思惟。

爾時，眾多比丘晡時從禪覺，往詣佛所，稽首禮佛足，退坐一面。

佛語諸比丘：「汝等見尊者罽賓那不？去我不遠，正身端坐，身心不動，住勝妙住。」

諸比丘白佛：「世尊！我等數見彼尊者正身端坐，善攝其身，不傾不動，專心勝妙。」

佛告諸比丘：「若比丘修習三昧，身心安住，不傾不動，住勝妙住者，此比丘得此三昧，不勤方便，隨欲即得。」

諸比丘白佛：「何等三昧，比丘得此三昧，身心不動，住勝妙住？」

佛告諸比丘：「若比丘依止聚落，晨朝著衣持鉢，入村乞食已，還精舍，舉衣鉢，洗足已，入林中，若閑房、露坐，思惟繫念，乃至息滅觀察善學，是名三昧，若比丘端坐思惟，身心不動，住勝妙住。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊於晨朝時著衣持鉢，入舍衛城乞食。**
　　那時，世尊在早晨穿好袈裟、拿著缽，進入舍衛城乞食。

**◈ 食已，還精舍，舉衣鉢，洗足已，持尼師檀入安陀林，坐一樹下，晝日禪思。**
　　用餐完畢後，回到精舍，收拾好衣缽，洗淨雙足，帶著尼師檀（僧侶用以接納供養的布）進入安陀林，坐在的一棵樹下，在白天進行禪思修行。

**◈ 時，尊者罽賓那亦晨朝時著衣持鉢，入舍衛城乞食。**
　　同時，尊者罽賓那也在早晨穿好袈裟、拿著缽，進入舍衛城乞食。

**◈ 還，舉衣鉢，洗足已，持尼師檀入安陀林，於樹下坐禪，去佛不遠，正身不動，身心正直，勝妙思惟。**
　　回來後，收拾衣缽，洗淨雙足，帶著尼師檀進入安陀林，在樹下坐禪，距離佛陀不遠，身體端正不動，身心正直，進行著極其精妙的思惟。

**◈ 爾時，眾多比丘晡時從禪覺，往詣佛所，稽首禮佛足，退坐一面。**
　　那時，許多比丘在傍晚從禪定中覺醒，前往佛陀所在之處，頂禮佛足，然後退到一邊坐下。

**◈ 佛語諸比丘：「汝等見尊者罽賓那不？去我不遠，正身端坐，身心不動，住勝妙住。」**
　　世尊告訴在場的眾位比丘說：「你們看到尊者罽賓那了嗎？他在離我不遠的地方，身體端正地坐著，身心不動，處於一種極其精妙的定境之中。」

**◈ 諸比丘白佛：「世尊！我等數見彼尊者正身端坐，善攝其身，不傾不動，專心勝妙。」**
　　眾比丘對佛陀恭敬地說道：「世尊！我們多次看到那位尊者身體端正地坐著，很好地攝持自己的身體，不傾斜也不動搖，專心地處於精妙的思惟中。」

**◈ 佛告諸比丘：「若比丘修習三昧，身心安住，不傾不動，住勝妙住者，此比丘得此三昧，不勤方便，隨欲即得。」**
　　佛陀告訴眾位比丘說：「如果比丘修習三昧（心意識的專注統一），使身心安住，不傾斜也不動搖，處於這種精妙的定境中，這位比丘就能獲得這種三昧，不需要費力地尋找複雜的方法，只要隨願就能獲得。」

**◈ 諸比丘白佛：「何等三昧，比丘得此三昧，身心不動，住勝妙住？」**
　　眾比丘對佛陀恭敬地說道：「是什麼樣的三昧，能讓比丘獲得這種身心不動、處於精妙定境的狀態？」

**◈ 佛告諸比丘：「若比丘依止聚落，晨朝著衣持鉢，入村乞食已，還精舍，舉衣鉢，洗足已，入林中，若閑房、露坐，思惟繫念，乃至息滅觀察善學，是名三昧，若比丘端坐思惟，身心不動，住勝妙住。」**
　　佛陀告訴眾位比丘說：「如果比丘依止於村落，早晨穿好袈裟、拿著缽，進入村莊乞食完畢後，回到精舍，收拾衣缽，洗淨雙足，進入林中，無論是在幽靜的房間或在露天坐著，思惟並繫念（將心專注於特定的對象），直到呼吸平息並觀察善學（對法義的正確學習與體悟），這就稱為三昧。如果比丘端正地坐著思惟，身心不動，就處於這種精妙的定境之中。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **三昧 (Samādhi)**：指心意識的專注統一，使心不再散亂，能穩定地停留在單一的對象上，是進入深層定境與產生智慧的基礎。
2. **尼師檀 (Nisadhana)**：原指僧侶在乞食時用來承接供養物的布，象徵著比丘對物質的簡樸需求與對供養者的尊重。
3. **繫念 (Sati/Upacara)**：指將心意識繫縛在特定的對象（如呼吸或法義）上，不讓心隨外境漂移，即現代所稱的「正念」之實踐。
4. **息滅 (Passaddhi/Nirodha)**：在此處指呼吸變得細膩平穩，以及內心躁動的平息，使身心達到一種極度寧靜的狀態。

### [現代實作指南]
本經文揭示了一個極其重要的修行邏輯：**「三昧（定力）並非脫離生活的孤立狀態，而是建立在規律生活與正念行動之上的結果。」**

**1. 建立「儀式感」的生活規律 (Routine as Foundation)**
佛陀與罽賓那尊者的行程完全一致：著衣 ➔ 持缽 ➔ 乞食 ➔ 還精舍 ➔ 洗足 ➔ 入林。
*   **實作**：在現代生活中，設定一套簡單且固定的早晨或晚間儀式（例如：起床後的簡單整理、固定的步行時間）。當身體進入規律的模式時，心會更容易從混亂中安定下來。

**2. 從「動」中轉化為「靜」 (Transition from Action to Stillness)**
經文中強調「洗足已」後才「入林坐禪」。洗足是一個轉折點，象徵將世俗的塵垢（外部世界的干擾）洗淨，準備進入內在空間。
*   **實作**：在工作與休息之間，設定一個「切換儀式」。例如，回家後洗手或更換衣服時，意識地告訴自己：「現在我將放下外部的壓力，進入內在的寧靜。」

**3. 實踐「繫念」與「觀察」 (Mindful Contemplation)**
三昧的達成路徑是：思惟 → 繫念 → 息滅 → 觀察善學。
*   **實作**：
    *   **思惟**：選擇一個簡單的議題（如：無常或感恩）。
    *   **繫念**：當心分心時，溫柔地將注意力拉回到該議題或呼吸上。
    *   **息滅**：不強求平靜，而是觀察呼吸如何自然地變得深長。
    *   **觀察善學**：在寧靜中反思今日的行為是否符合正法，將經驗轉化為智慧。

### [段落大意]
本經描述佛陀與尊者罽賓那在日常生活中展現出高度一致的規律修行。佛陀藉此向比丘說明，真正的三昧（定力）並非透過艱澀的技巧，而是源於將規律的日常生活（如乞食、洗足）與深度的正念思惟相結合。當修行者能將心專注於法義且身心安定時，自然能進入精妙的定境。
