# 雜阿含經 第 926 經

### [原文]
如是我聞：

一時，佛住那梨聚落深谷精舍。

爾時，世尊告詵陀迦旃延：「當修真實禪，莫習強良禪，如強良馬，繫槽櫪上，彼馬不念：『我所應作、所不應作。』但念穀草。如是，丈夫於貪欲纏多所修習故，彼以貪欲心思惟，於出離道不如實知，心常馳騁，隨貪欲纏而求正受；瞋恚、睡眠、掉悔、疑多修習故，於出離道不如實知，以疑蓋心思惟，以求正受。

「詵陀！若真生馬繫槽櫪上，不念水草，但作是念：『駕乘之事。』如是，丈夫不念貪欲纏，住於出離如實知，不以貪欲纏而求正受，亦不瞋恚、睡眠、掉悔、疑纏，多住於出離；瞋恚、睡眠、掉悔、疑纏如實知，不以疑纏而求正受。

「如是，詵陀！比丘如是禪者，不依地修禪，不依水、火、風、空、識、無所有、非想非非想而修禪。不依此世、不依他世，非日、月，非見、聞、覺、識，非得非求，非隨覺，非隨觀而修禪。

「詵陀！比丘如是修禪者，諸天主、伊濕波羅、波闍波提恭敬合掌，稽首作禮而說偈言：

「『南無大士夫，　　南無士之上， 　　以我不能知，　　依何而禪定？』」

爾時，有尊者跋迦利住於佛後，執扇扇佛。時，跋迦利白佛言：「世尊！若比丘云何入禪，而不依地、水、火、風，乃至覺觀，而修禪定？云何比丘禪，諸天主、伊濕波羅、波闍波提合掌恭敬，稽首作禮而說偈言：

「『南無大士夫，　　南無士之上， 　　以我不能知，　　依何而禪定？』」

佛告跋迦利：「比丘於地想能伏地想，於水、火、風想、無量空入處想、識入處想、無所有入處、非想非非想入處想。此世他世，日、月、見、聞、覺、識，若得若求，若覺若觀，悉伏彼想。跋迦利！比丘如是禪者，不依地、水、火、風，乃至不依覺、觀而修禪。跋迦利！比丘如是禪者，諸天主、伊濕波羅、波闍波提恭敬合掌，稽首作禮而說偈言：

「『南無大士夫，　　南無士之上， 　　以我不能知，　　何所依而禪？』」

佛說此經時，詵陀迦旃延比丘遠塵離垢，得法眼淨。跋迦利比丘不起諸漏，心得解脫。

佛說此經已，跋迦利比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住那梨聚落深谷精舍。**
　　在某一個時間，佛陀住在那梨聚落深谷中的精舍裡。

**◈ 爾時，世尊告詵陀迦旃延：「當修真實禪，莫習強良禪，**
　　那時，世尊告訴詵陀迦旃延說：「你應該修行真實的禪定，不要練習那種強行偽裝的禪定。

**◈ 如強良馬，繫槽櫪上，彼馬不念：『我所應作、所不應作。』但念穀草。**
　　就像一匹被強行馴服的馬，雖然被繫在馬槽邊，但牠並不思考：『我應該做什麼，或不應該做什麼』，而僅僅是想著穀草（食物）。

**◈ 如是，丈夫於貪欲纏多所修習故，彼以貪欲心思惟，於出離道不如實知，心常馳騁，隨貪欲纏而求正受；**
　　同樣地，如果一個人經常在貪欲的束縛中修行，他會用貪欲的心來思考，對於出離道（脫離輪迴的道路）無法如實地認知，心念經常奔馳，隨著貪欲的牽引而追求一種舒適的感受（正受）。

**◈ 瞋恚、睡眠、掉悔、疑多修習故，於出離道不如實知，以疑蓋心思惟，以求正受。**
　　因為經常在瞋恨、昏沉睡眠、躁動悔恨以及疑惑中修行，導致對出離道無法如實認知，被疑惑遮蔽了心智而思考，僅僅是為了追求舒適的感受。

**◈ 「詵陀！若真生馬繫槽櫪上，不念水草，但作是念：『駕乘之事。』**
　　詵陀！如果是一匹天生純良的馬被繫在馬槽邊，牠不會想著水草，而只會思考：『關於被駕馭與奔馳的事務。』

**◈ 如是，丈夫不念貪欲纏，住於出離如實知，不以貪欲纏而求正受，**
　　同樣地，修行者不再想著貪欲的束縛，處於出離（脫離執著）的狀態並如實地認知真理，不再利用貪欲的束縛來追求舒適的感受。

**◈ 亦不瞋恚、睡眠、掉悔、疑纏，多住於出離；瞋恚、睡眠、掉悔、疑纏如實知，不以疑纏而求正受。**
　　他也不再被瞋恨、睡眠、躁動悔恨與疑惑所束縛，而多地處於出離的狀態；他能如實地認知到瞋恨、睡眠、躁動悔恨與疑惑的束縛，不再利用疑惑的束縛來追求舒適的感受。

**◈ 「如是，詵陀！比丘如是禪者，不依地修禪，不依水、火、風、空、識、無所有、非想非非想而修禪。**
　　詵陀！比丘如果這樣禪修，就不會依賴地（物質身體的元素）來修禪，也不依賴水、火、風（物質元素）、空（虛空）、識（意識）、無所有處（極其微細的空寂狀態）、非想非非想處（超越想與非想的極微細狀態）而修禪。

**◈ 不依此世、不依他世，非日、月，非見、聞、覺、識，非得非求，非隨覺，非隨觀而修禪。**
　　他不會依賴這個世界或其他世界，不依賴日、月，不依賴見、聞、覺、識（感官認知），不依賴於獲得或追求，也不隨著覺知或觀察（對對象的執著）而修禪。

**◈ 「詵陀！比丘如是修禪者，諸天主、伊濕波羅、波闍波提恭敬合掌，稽首作禮而說偈言：**
　　詵陀！比丘如果這樣修禪，連諸天之主、伊濕波羅（自在天）與波闍波提（大梵天）都會恭敬地合掌，頂禮而說偈頌：

**◈ 「『南無大士夫，　　南無士之上， 　　以我不能知，　　依何而禪定？』」**
　　『頂禮偉大的修行者，頂禮修行者之首，因為我無法得知，您是依據什麼而進入禪定的？』

**◈ 爾時，有尊者跋迦利住於佛後，執扇扇佛。時，跋迦利白佛言：**
　　那時，有尊者跋迦利站在佛陀身後，為佛陀扇風。這時，跋迦利對佛陀恭敬地說道：

**◈ 「世尊！若比丘云何入禪，而不依地、水、火、風，乃至覺觀，而修禪定？**
　　世尊！比丘究竟如何進入禪定，卻能不依賴地、水、火、風，乃至不依賴覺知與觀察而修習禪定呢？

**◈ 云何比丘禪，諸天主、伊濕波羅、波闍波提合掌恭敬，稽首作禮而說偈言：**
　　什麼樣的比丘禪修，能讓諸天之主、伊濕波羅與波闍波提合掌恭敬，頂禮而說偈頌：

**◈ 「『南無大士夫，　　南無士之上， 　　以我不能知，　　依何而禪定？』」**
　　『頂禮偉大的修行者，頂禮修行者之首，因為我無法得知，您是依據什麼而進入禪定的？』

**◈ 佛告跋迦利：「比丘於地想能伏地想，於水、火、風想、無量空入處想、識入處想、無所有入處、非想非非想入處想。**
　　佛陀告訴跋迦利：「比丘對於地想（對物質元素的認知）能消除地想，對於水、火、風想、無量空入處想、識入處想（意識的認知）、無所有入處想、非想非非想入處想也能同樣消除。

**◈ 此世他世，日、月、見、聞、覺、識，若得若求，若覺若觀，悉伏彼想。**
　　對於這個世界或其他世界，對於日、月，對於見、聞、覺、識，無論是獲得、追求，或是覺知、觀察，都能將這些想（想像與認知）全部消除。

**◈ 跋迦利！比丘如是禪者，不依地、水、火、風，乃至不依覺、觀而修禪。**
　　跋迦利！比丘如果這樣禪修，就不會依賴地、水、火、風，乃至不依賴覺知與觀察而修禪。

**◈ 跋迦利！比丘如是禪者，諸天主、伊濕波羅、波闍波提恭敬合掌，稽首作禮而說偈言：**
　　跋迦利！比丘如果這樣禪修，諸天之主、伊濕波羅與波闍波提都會恭敬地合掌，頂禮而說偈頌：

**◈ 「『南無大士夫，　　南無士之上， 　　以我不能知，　　何所依而禪？』」**
　　『頂禮偉大的修行者，頂禮修行者之首，因為我無法得知，您是依據什麼而禪修的？』

**◈ 佛說此經時，詵陀迦旃延比丘遠塵離垢，得法眼淨。跋迦利比丘不起諸漏，心得解脫。**
　　佛陀講這部經時，詵陀迦旃延比丘遠離了世俗的塵垢，獲得了清淨的法眼（能見真理的智慧）。跋迦利比丘不再產生煩惱的滲漏（漏），心獲得了解脫（Vimokkha）。

**◈ 佛說此經已，跋迦利比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，跋迦利比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **強良禪 (Forced/Fake Zen)**：指一種強行壓制或偽裝的禪定。修行者並非真正地放下執著，而是利用某種特定的心理狀態（如對寧靜的貪愛）來獲取舒適感，本質上仍被貪欲所驅動。
2. **出離 (Renunciation/Liberation)**：指心靈脫離對五蘊（物質身體、感受、想像與認知、心的造作、意識）的執著，不再追求世俗的滿足，轉而追求徹底的解脫。
3. **伏想 (Subduing Perception/Saññā)**：指透過覺知，消除對特定對象的「想（想像與認知）」。這裡的「伏」並非強行壓制，而是透過如實觀察，使對對象的執著與認知自然消融。
4. **法眼淨 (Pure Dharma Eye)**：指修行者在實踐中突破了無明，能夠如實地看到萬法緣起、無常、無我的真理。
5. **漏 (Āsava)**：指像滲漏一樣不斷流出的煩惱，主要包括煩惱漏（Kāmāsava）、有漏（Bhavāsava）與無明漏（Avijjāsava）。「不起諸漏」即指徹底斷除這些潛在的煩惱根源。

### [現代實作指南]
本經文揭示了「追求狀態」與「觀察本質」之間的關鍵差異。在現代生活中，我們常陷入「強良禪」的陷阱，例如：追求冥想時的平靜感、追求心靈雞湯帶來的暫時寬慰，這本質上仍是「隨貪欲纏而求正受」。

**實作步驟：**
1. **辨識「舒適陷阱」**：
   當你在靜坐或思考時，若發現自己正渴望進入某種「很舒服」、「很空靈」的狀態，請立刻意識到：這就是「強良馬」在想穀草。這種對舒適感的追求，正是阻礙出離的貪欲。

2. **執行「伏想」練習**：
   - **對象化觀察**：當心中出現「我想讓身體放鬆」或「我想消除焦慮」的念頭時，不要試圖推開它，而是將這個「想（認知）」本身視為觀察對象。
   - **去標籤化**：將「地、水、火、風」等元素類比為現代的「身體感覺（酸、麻、痛）」或「情緒標籤（憤怒、悲傷）」。當感覺出現時，僅僅認知「有這個感覺」，而不依賴這個感覺來定義自己的狀態。

3. **轉念技巧 ➔ 從「依賴」轉向「不依賴」**：
   - **錯誤路徑**：我需要環境安靜 → 我才能心平氣和（依賴外部環境）。
   - **正確路徑**：環境嘈雜 → 認知到「嘈雜」這個想 → 觀察這個想的生滅 → 不依賴安靜而獲得平靜（不依地水火風）。

### [段落大意]
佛陀以「強良馬」與「真生馬」的比喻，區分了偽裝的禪定（追求舒適感）與真實的禪定（追求出離）。真實的禪定不依賴任何物質元素、感官認知或外部世界，而是透過「伏想」消除對所有對象的執著與認知。最終，詵陀與跋迦利兩位比丘透過此教導，分別獲得法眼淨與解脫。
