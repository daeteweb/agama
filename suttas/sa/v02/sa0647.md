# 雜阿含經 第 647 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有五根。何等為五？謂信根、精進根、念根、定根、慧根。

「何等為信根？若比丘於如來所起淨信心，根本堅固，餘沙門、婆羅門、諸天、魔、梵、沙門、婆羅門，及餘世間，無能沮壞其心者，是名信根。

「何等為精進根？已生惡不善法令斷，生欲、方便、攝心、增進；未生惡不善法不起，生欲、方便、攝心、增進；未生善法令起，生欲、方便、攝心、增進；已生善法住不忘，修習增廣，生欲、方便、攝心、增進，是名精進根。

「何等為念根？若比丘內身身觀住，慇懃方便，正念正智，調伏世間貪憂；外身、內外身，受、心、法法觀念住亦如是說，是名念根。

「何等為定根？若比丘離欲惡不善法，有覺有觀，離生喜樂，乃至第四禪具足住，是名定根。

「何等為慧根？若比丘苦聖諦如實知，苦集聖諦、苦滅聖諦、苦滅道跡聖諦如實知，是名慧根。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有五根。何等為五？謂信根、精進根、念根、定根、慧根。**
　　那時，世尊告訴在場的眾位比丘說：「有五種修行根基（五根）。什麼是這五種根基？指的是：信根（信心的力量）、精進根（努力的能量）、念根（正念的覺知）、定根（專注的定力）以及慧根（洞察的智慧）。」

**◈ 「何等為信根？若比丘於如來所起淨信心，根本堅固，餘沙門、婆羅門、諸天、魔、梵、沙門、婆羅門，及餘世間，無能沮壞其心者，是名信根。**
　　「什麼是信根？如果一位比丘對成就不變真理而來到世間的人（如來）產生了純淨的信心，且這種信心根基堅固，使得其他的沙門、婆羅門、諸天、魔王、梵天，以及世間上任何其他的人或力量，都無法摧毀或動搖他的信心，這就稱為信根。」

**◈ 「何等為精進根？已生惡不善法令斷，生欲、方便、攝心、增進；**
　　「什麼是精進根？對於已經產生的惡劣不善之法（負面心態或行為），他生起願望、運用方法、攝受心念並持續進步，致力於將其斷除；

**◈ 未生惡不善法不起，生欲、方便、攝心、增進；**
　　對於尚未產生的惡劣不善之法，他生起願望、運用方法、攝受心念並持續進步，致力於使其不生起；

**◈ 未生善法令起，生欲、方便、攝心、增進；**
　　對於尚未產生的善法（正向心態或行為），他生起願望、運用方法、攝受心念並持續進步，致力於使其生起；

**◈ 已生善法住不忘，修習增廣，生欲、方便、攝心、增進，是名精進根。**
　　對於已經產生的善法，他生起願望、運用方法、攝受心念並持續進步，使其能安住而不忘失，並透過修習來擴展增廣。這就稱為精進根。」

**◈ 「何等為念根？若比丘內身身觀住，慇懃方便，正念正智，調伏世間貪憂；**
　　「什麼是念根？如果一位比丘對自己的身體（內身）進行觀照並安住其中，以殷勤的方法，運用正念（清晰的覺知）與正智（正確的理解），來調伏世間的貪婪與憂愁；

**◈ 外身、內外身，受、心、法法觀念住亦如是說，是名念根。**
　　對於他人的身體（外身）、以及內外身體、受（感受）、心（意識狀態）、法（心理現象）也同樣地進行觀照與安住。這就稱為念根。」

**◈ 「何等為定根？若比丘離欲惡不善法，有覺有觀，離生喜樂，乃至第四禪具足住，是名定根。**
　　「什麼是定根？如果一位比丘遠離了欲望與惡劣不善之法，在心中保有覺知與觀察，體驗遠離感官欲望而產生的喜悅與快樂，直到完全進入第四禪的定境中安住，這就稱為定根。」

**◈ 「何等為慧根？若比丘苦聖諦如實知，苦集聖諦、苦滅聖諦、苦滅道跡聖諦如實知，是名慧根。」**
　　「什麼是慧根？如果一位比丘能如實地認知苦聖諦（關於苦的真理）、苦集聖諦（關於苦之原因的真理）、苦滅聖諦（關於苦之止息的真理）以及苦滅道跡聖諦（關於解脫之道的真理），這就稱為慧根。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **五根 (Indriya)**：指在修行過程中起主導作用的五種心理能力。它們如同植物的根一樣，是成長與解脫的基礎。
2. **第四禪 (Fourth Jhāna)**：定修的高階境界。此階段已捨棄喜與樂，進入一種極其純淨、平穩且捨離的狀態，心意識達到最高程度的集中與清明。
3. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四項真理 ➔ 苦（生命的本質是苦）、集（苦的原因是貪愛與無明）、滅（苦可以被止息）、道（達成止息的八正道）。
4. **有覺有觀 (Sampajañña)**：指在定中或生活中，不僅有意識到當下的狀態（覺），還能對該狀態的性質有清晰的洞察（觀）。

### [現代實作指南]
本經文提出的「五根」可視為一套心智管理系統，現代人可將其轉化為以下實踐步驟：

1. **建立核心價值（信根）**：
   - **實作**：在面對外界紛擾或自我懷疑時，找回一個能讓自己心安的真理或目標（如：追求內心的平靜）。當這個核心價值足夠堅固，外界的批評或誘惑將不再能輕易動搖你的情緒。

2. **動態調整行為（精進根）**：
   - **轉念技巧**：將「努力」細分為四個方向 ➔ 
     - 發現壞習慣 → 立即採取行動切斷。
     - 預見潛在錯誤 → 提前建立防線防止發生。
     - 發現好的方法 → 主動嘗試將其導入生活。
     - 擁有好的習慣 → 透過重複練習使其成為本能。

3. **提升覺察力（念根）**：
   - **實作**：練習「身體掃描」。每天花5分鐘觀察呼吸、肌肉的緊繃感（內身），以及觀察他人情緒對自己的影響（外身）。當貪婪或憂慮升起時，僅僅是「標記」它（例如：心中默念「憂慮升起了」），而不陷入其中。

4. **創造深度專注（定根）**：
   - **實作**：練習「單一任務法」。在工作或閱讀時，刻意遠離手機等干擾源（離欲），將注意力完全集中在當下的一件事上，體驗那種心靈不再跳躍的寧靜感。

5. **邏輯分析真相（慧根）**：
   - **轉念技巧**：面對壓力時，運用四聖諦邏輯分析 ➔ 
     - 承認現在感到痛苦（苦）。
     - 分析痛苦是因為對某事有執著或期待（集）。
     - 意識到如果放下執著，痛苦會消失（滅）。
     - 制定具體的行動計畫來改變現狀（道）。

### [段落大意]
本經詳細定義了修行解脫所需的五種核心能力（五根）：堅定不移的信心（信根）、對善惡法能靈活調控的努力（精進根）、對身心現象的持續覺察（念根）、深度的專注定力（定根），以及對四聖諦真理的如實洞察（慧根）。這五者相輔相成，共同構成從凡夫轉化為覺者的心理基礎。
