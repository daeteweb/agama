# 雜阿含經 第 910 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

時，有兇惡聚落主來詣佛所，稽首佛足，退坐一面，白佛言：「世尊！不修何等法故，於他生瞋恚；生瞋恚故，口說惡言，他為其作惡性名字？」

佛告聚落主：「不修正見故，於他生瞋；生瞋恚已，口說惡言，他為其作惡性名字。不修正志、正語、正業、正命、正方便、正念、正定故，於他生瞋；生瞋恚故，口說惡言，他為其作惡性名字。」

復問：「世尊！修習何法，於他不瞋，不瞋恚故，口說善言，他為其作賢善名字？」

佛告聚落主：「修正見故，於他不瞋；不瞋恚故，口說善言，他為其作賢善名字。修習正志、正語、正業、正命、正方便、正念、正定故，於他不瞋；不瞋恚故，口說善言，他為其作賢善名字。」

兇惡聚落主白佛言：「奇哉！世尊！善說此言。我不修正見故，於他生瞋；生瞋恚已，口說惡言，他為我作惡性名字。我不修正志、正語、正業、正命、正方便、正念、正定故，於他生瞋；生瞋恚故，口說惡言，他為我作惡性名字。是故，我今當捨瞋恚、剛強、麁澁。」

佛告聚落主：「此真實要。」

佛說此經已，兇惡聚落主歡喜隨喜，作禮而去。

**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園中。

**◈ 時，有兇惡聚落主來詣佛所，稽首佛足，退坐一面，白佛言：**
　　那時，有一位性格兇惡的村落首領來到佛陀這裡，向佛陀的足下頂禮，然後退到一邊坐下，對佛陀恭敬地說道：

**◈ 「世尊！不修何等法故，於他生瞋恚；生瞋恚故，口說惡言，他為其作惡性名字？」**
　　「世尊！是因為沒有修行什麼樣的法，才會對他人產生瞋恚（憤怒與厭惡的心情）；而因為產生了瞋恚，所以口中說出惡劣的話，導致他人為他起惡劣的稱號（壞名聲）呢？」

**◈ 佛告聚落主：「不修正見故，於他生瞋；生瞋恚已，口說惡言，他為其作惡性名字。」**
　　佛陀告訴這位村落首領：「因為沒有修行正見（正確的見地與認知）的緣故，所以對他人產生瞋心；在產生瞋恚之後，口中說出惡劣的話，他人因此為他起惡劣的稱號。」

**◈ 「不修正志、正語、正業、正命、正方便、正念、正定故，於他生瞋；生瞋恚故，口說惡言，他為其作惡性名字。」**

　　他再次請問：「世尊！應該修習什麼樣的法，才能對他人不產生瞋心；因為不產生瞋恚，所以口說良善的話，使他人為他起賢善的稱號（好名聲）呢？」

**◈ 佛告聚落主：「修正見故，於他不瞋；不瞋恚故，口說善言，他為其作賢善名字。」**
　　佛陀告訴這位村落首領：「因為修行正見（正確的見地與認知）的緣故，所以對他人不產生瞋心；因為不產生瞋恚，所以口說良善的話，他人因此為他起賢善的稱號。」

**◈ 「修習正志、正語、正業、正命、正方便、正念、正定故，於他不瞋；不瞋恚故，口說善言，他為其作賢善名字。」**

**◈ 兇惡聚落主白佛言：「奇哉！世尊！善說此言。」**
　　這位兇惡的村落首領對佛陀恭敬地說道：「太奇妙了！世尊！您說得太好了。」

**◈ 「我不修正見故，於他生瞋；生瞋恚已，口說惡言，他為我作惡性名字。」**

　　「我之前因為沒有修行正思惟（正確的思考方向與志向）、正語（正確的言語表達）、正業（正確的行為造作）、正命（正確的生存方式）、正精進（正確的努力與修行方向）、正念（正確的覺知與專注）、正定（正確的心念專注與安定），所以對他人產生瞋心；因為產生瞋恚，所以口說惡言，他人因此為我起惡劣的稱號。」

**◈ 「是故，我今當捨瞋恚、剛強、麁澁。」**

### [詞語註解]
1. **正見 (Sammā-diṭṭhi)**：八正道之首，指正確的見地。在原始佛教中，核心是指對四聖諦（苦、集、滅、道）的正確理解，能讓人洞察因緣生滅的真理，從而消除錯誤的認知。
2. **瞋恚 (Kodha/Dosa)**：指內心對不順意之物或人的厭惡、憤怒與排斥。它是三毒（貪、瞋、癡）之一，是導致口業（惡言）與身業（暴力）的直接心理驅動力。
3. **八正道 (Ariya-aṭṭhaṅgika-magga)**：佛陀提出的解脫之道，包含正見、正思惟（正志）、正語、正業、正命、正精進（正方便）、正念、正定。這八項要素相互支持，旨在消除煩惱並達到涅槃。
4. **正方便 (Sammā-vāyāma)**：即正精進。指正確的努力，包含：防止未生之惡、斷除已生之惡、生起未生之善、增長已生之善。

### [現代實作指南]
本經揭示了一個清晰的「心理因果鏈」：**錯誤認知（缺乏正見） ➔ 情緒反應（瞋恚） ➔ 外在行為（惡言） ➔ 社會反饋（惡名）**。在現代生活中，我們可以透過以下步驟實踐轉念：

1. **覺察因果鏈**：當你發現自己被他人評價為「脾氣差」或「難相處」時，不要急於反擊他人，而應回溯至源頭。意識到「惡名」是「惡言」的結果，而「惡言」是「瞋心」的結果，而「瞋心」則是源於對現實的「錯誤認知（缺乏正見）」。
2. **修正「正見」的實作**：
   - **挑戰定見**：當憤怒升起時，問自己：「我現在對這個人的看法是否絕對化了？」或「我是否將對方的行為過度解讀為對我的攻擊？」
   - **因緣觀察**：試著思考對方的行為是由哪些壓力、習慣或無明（缺乏智慧）所驅動，而非單純的惡意。將「他很壞」轉化為「他正處於痛苦與無明之中」。
3. **切斷反應路徑**：
   - 在「瞋心」與「惡言」之間建立一個緩衝區。練習在想說惡言前深呼吸三次，運用「正念」觀察憤怒在身體上的感受（如胸口發熱、呼吸急促），僅僅觀察而不採取行動。
4. **建立正向反饋**：
   - 刻意練習「正語」。即使內心仍有微小憤怒，先嘗試用中性或溫和的言語表達需求。當你發現溫和的言語能帶來更好的社交結果（賢善名字）時，這種正向反饋會反過來強化你的正見。

### 段落大意 
一位性格兇惡的村落首領向佛陀請教產生瞋心與惡名的原因。佛陀指出，這是因為缺乏正見及八正道的修行，導致內心生瞋並口說惡言。反之，修習八正道能消除瞋心，使人說善言並獲得好名聲。首領領悟後決定捨棄剛強粗魯，心生歡喜。
