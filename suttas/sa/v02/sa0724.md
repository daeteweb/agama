# 雜阿含經 第 724 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若比丘持戒、修德、慚、愧，成真實法，見此人者，多得果報。若復聞者，若隨憶念者、隨出家者，多得功德，況復親近恭敬奉事。所以者何？親近奉事如是人者，時時得聞深妙之法；得聞深法已，成就二正，身正及心正，方便修習定覺分；修習已，修習滿足，乃至捨覺分修習滿足。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若比丘持戒、修德、慚、愧，成真實法，見此人者，多得果報。**
　　「如果有一位比丘能夠持守戒律、修行德行，並且擁有慚（對自己內心的羞愧感）與愧（對他人評價的羞恥感），進而成就真實的法（真理），那麼只要是見到這樣的人，就能獲得許多正向的果報。

**◈ 若復聞者，若隨憶念者、隨出家者，多得功德，況復親近恭敬奉事。**
　　如果是聽聞其名的人，或是隨之憶念、隨之出家修行的人，都能獲得許多功德，更不用說那些親近、恭敬並奉事這樣的人了。

**◈ 所以者何？親近奉事如是人者，時時得聞深妙之法；**
　　為什麼呢？因為親近並奉事這樣的人，能夠時時聽聞深奧微妙的法（真理）；

**◈ 得聞深法已，成就二正，身正及心正，方便修習定覺分；**
　　在聽聞深奧的法之後，能成就兩種正向的狀態：身體的端正（身正）以及心的端正（心正），如此便能方便地修習定覺分（導向覺悟的禪定要素）；

**◈ 修習已，修習滿足，乃至捨覺分修習滿足。」**
　　修習之後，直到修習圓滿，甚至連捨覺分（對一切不執著、平等的覺悟要素）也修習圓滿。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **慚 (Hiri)**：指內在的自省，因自覺行為不端而產生的羞愧感，是一種自我監控的道德力量。
2. **愧 (Ottappa)**：指對外在評價的恐懼，因擔心行為不端會被他人指責或導致惡果而產生的羞恥感。
3. **真實法 (Saddhamma)**：指符合宇宙真理、能導向解脫的正確教法。
4. **定覺分 (Samādhi-sambojjhaṅga)**：七覺支（導向覺悟的七個要素）之一，指心專注於單一對象而產生的安定狀態。
5. **捨覺分 (Upekkhā-sambojjhaṅga)**：七覺支之最後一項，指在覺知中保持中立、不偏不倚、不執著於任何感受的平等心。

### [現代實作指南]
本經強調了「善知識」（Kalyana-mitta）對修行者的決定性影響。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **篩選社交圈（親近善友）**：
   ➔ 觀察身邊的人是否具備「慚」與「愧」的特質（即：是否對錯誤有自覺，且不隨波逐流地行惡）。
   ➔ 刻意減少與傾向於抱怨、毀謗或鼓勵不道德行為的人往來，增加與具有正向價值觀、自律且謙卑之人的互動時間。

2. **建立「身正心正」的觀察機制**：
   ➔ **身正**：在日常行動中，練習「正業」，例如不妄語、不傷害他人。
   ➔ **心正**：在面對情緒波動時，練習「正思惟」，將注意力從「我想要什麼」轉向「現在發生了什麼」。

3. **從模仿到內化（憶念與奉事）**：
   ➔ 當遇到值得尊敬的榜樣時，不應僅止於崇拜，而應「隨憶念」➔ 將其處理問題的理智方式、面對困難的平靜態度作為心智模型進行模仿。
   ➔ 透過「奉事」（提供協助、傾聽、學習），在互動中自然地吸收對方的正向能量與智慧。

4. **培養「捨」的心理韌性**：
   ➔ 在面對生活中的得失時，練習「捨覺分」的邏輯 ➔ 告訴自己：「這件事的發生是由因緣而生，我觀察它，但不被它牽引。」從而達到心靈的安定。

### [段落大意]
本段經文闡述了親近具備持戒、德行及慚愧心之聖者的益處。佛陀指出，僅僅是見到、聽聞或憶念這樣的人都能獲益，而親近奉事則能使人時時聞法，進而達成身心端正，並圓滿從定覺分到捨覺分的修行過程。
