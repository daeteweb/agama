# 雜阿含經 第 541 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。時，尊者阿那律在舍衛國松林精舍，病差未久。

時，有眾多比丘往詣阿那律所，問訊慰勞已，於一面坐，問尊者阿那律：「安隱樂住不？」

阿那律言：「安隱樂住，身諸苦痛漸已休息。」

諸比丘問尊者阿那律：「住何所住，身諸苦痛得安隱？」

尊者阿那律言：「住四念處，身諸苦痛漸得安隱。何等為四？謂內身身觀念處，乃至法法觀念處，是名四念處。住此四念處故，身諸苦痛漸得休息。」

時，諸正士共論議已，歡喜隨喜，各從座起而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，尊者阿那律在舍衛國松林精舍，病差未久。**
　　那時，尊者阿那律住在舍衛國的松林精舍中，剛從病中康復不久。

**◈ 時，有眾多比丘往詣阿那律所，問訊慰勞已，於一面坐，問尊者阿那律：「安隱樂住不？」**
　　那時，有許多比丘（出家修行者）前往拜訪阿那律尊者，在問候並慰問他的病情之後，大家坐在旁邊，詢問阿那律尊者：「您現在是否住於安穩與快樂之中？」

**◈ 阿那律言：「安隱樂住，身諸苦痛漸已休息。」**
　　阿那律尊者回答：「我是安穩且快樂的，身體所感受到的種種苦痛已經逐漸平息。」

**◈ 諸比丘問尊者阿那律：「住何所住，身諸苦痛得安隱？」**
　　眾位比丘詢問阿那律尊者：「您是依止於什麼樣的心態或修行方法，才能讓身體的苦痛獲得安穩？」

**◈ 尊者阿那律言：「住四念處，身諸苦痛漸得安隱。何等為四？謂內身身觀念處，乃至法法觀念處，是名四念處。住此四念處故，身諸苦痛漸得休息。」**
　　阿那律尊者回答：「我是依止於四念處（四種正念的觀察對象），因此身體的苦痛才逐漸獲得安穩。所謂的四種對象是什麼呢？指的是：觀察內在身體的『身觀念處』、觀察受（感受）的『受觀念處』、觀察心（意識狀態）的『心觀念處』，以及觀察法（一切現象）的『法觀念處』，這就稱為四念處。正因為依止這四念處的修行，身體的苦痛才逐漸平息。」

**◈ 時，諸正士共論議已，歡喜隨喜，各從座起而去。**
　　那時，在場的各位正士（對佛法有信仰的在家修行者）在共同討論之後，心中感到歡喜並隨喜讚嘆，隨後各自從座位上起身離開。

### [詞語註解（解釋重要佛教概念）]
1. **四念處 (Satipaṭṭhāna)**：佛教修行中最重要的正念觀察法，旨在透過對四種對象的覺知，破除對「我」的執著。
2. **身觀念處 (Kāyānupassanā)**：對身體的觀察，包括呼吸、姿勢、身體組成（如四大元素）以及身體的無常。
3. **受觀念處 (Vedanānupassanā)**：對感受的觀察，將感受區分為苦、樂、不苦不樂，並覺知感受的生起與滅去。
4. **心觀念處 (Cittānupassanā)**：對心態的觀察，覺知當下心中是否有貪、嗔、痴或定、亂等狀態。
5. **法觀念處 (Dhammānupassanā)**：對法（現象/真理）的觀察，例如觀察五蘊的運作、十二因緣的鏈條或四聖諦。

### [現代實作指南]
本經文揭示了如何透過「正念」將「生理疼痛」與「心理痛苦」分離。在現代生活中，當我們面對身體不適或劇烈疼痛時，可採取以下實作步驟：

1. **第一階段：身觀（客觀標記）**
   - 不要將疼痛定義為「我的痛」，而將其視為「一種感覺」。
   - 嘗試在心中標記：「現在有灼熱感」、「現在有抽痛感」，將注意力集中在感覺的物理特性上，而非對疼痛的抗拒。

2. **第二階段：受觀（分離感受）**
   - 觀察疼痛帶來的「不快感」。意識到「疼痛」是生理訊號，而「痛苦」是心靈對疼痛產生的排斥反應。
   - 告訴自己：「身體在疼痛，但我的心不需要跟著痛苦。」

3. **第三階段：心觀（覺察情緒）**
   - 觀察心中是否產生了焦慮、煩躁或恐懼（例如：「為什麼我還沒好？」、「這痛會持續多久？」）。
   - 僅僅是「看著」這些情緒的升起，不與其認同，不試圖強行壓抑。

4. **第四階段：法觀（體悟無常）**
   - 觀察疼痛的波動。疼痛並非一個恆定的整體，而是由許多微小的感覺組成，且在不斷地變動（生起 → 停留 $\rightarrow$ 消失）。
   - 體悟到疼痛也是一種「緣起」的現象，沒有一個永恆的「我」在承受痛苦。

### [段落大意]
本經敘述阿那律尊者在病後康復之際，告知眾比丘其能於病痛中獲得安穩之秘訣，在於實踐「四念處」的修行，透過對身體、感受、心態與法現象的正念觀察，使生理苦痛逐漸平息。
