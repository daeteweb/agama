# 雜阿含經 第 632 經

### [原文]
如是我聞：

一時，佛住巴連弗邑鷄林精舍。爾時，尊者阿難、尊者跋陀羅亦在彼住。

尊者跋陀羅問尊者阿難：「頗有法修習多修習，得阿羅漢？」

尊者阿難語尊者跋陀羅：「有法修習多修習，而得阿羅漢，謂四念處。何等為四？謂身身觀念住，受、心、法法觀念住。」

時，二正士共論議已，各還本處。

---

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住巴連弗邑鷄林精舍。**
　　在某一個時間，佛陀住在巴連弗邑的雞林精舍。

**◈ 爾時，尊者阿難、尊者跋陀羅亦在彼住。**
　　那時，尊者阿難與尊者跋陀羅也住在在那裡。

**◈ 尊者跋陀羅問尊者阿難：「頗有法修習多修習，得阿羅漢？」**
　　尊者跋陀羅問尊者阿難：「是否有一種修行方法，只要經過大量的修習，就能成就阿羅漢（完全斷除煩惱、獲得解脫的聖者）？」

**◈ 尊者阿難語尊者跋陀羅：「有法修習多修習，而得阿羅漢，謂四念處。」**
　　尊者阿難告訴尊者跋陀羅：「確實有一種修行方法，只要多加修習就能成就阿羅漢，那就是所謂的『四念處』（四種正念的確立）。」

**◈ 「何等為四？謂身身觀念住，受、心、法法觀念住。」**
　　「這四種是什麼呢？即是：對色（物質身體）的觀照、對受（感受）的觀照、對心（意識狀態）的觀照，以及對法（心理現象與真理）的觀照。」

**◈ 時，二正士共論議已，各還本處。**
　　那時，這兩位修行者討論完畢後，各自回到了原處。

### [詞語註解（解釋重要佛教概念）]
1. **阿羅漢 (Arahant)**：指已完全斷除貪、嗔、痴三毒，不再受輪迴之苦，達到最終解脫境界的聖者。
2. **四念處 (Satipaṭṭhāna)**：原始佛教中最重要的修行方法，指將正念（Mindfulness）確立在四個對象上，以破除對「我」的執著。
3. **身觀 (Kāyānupassanā)**：對色（物質身體）的觀察，包括呼吸、身體的姿勢以及身體的組成，體悟其無常與非我。
4. **受觀 (Vedanānupassanā)**：對受（感受）的觀察，覺知當下產生的苦、樂或不苦不樂的感受，而不被其牽引。
5. **心觀 (Cittānupassanā)**：對心（意識狀態）的觀察，覺知心的狀態（如：有貪心或無貪心、散亂或專注）。
6. **法觀 (Dhammānupassanā)**：對法（心理現象與真理）的觀察，分析心法運作的規律（如五蘊、十二因緣），體悟緣起空性。

### [現代實作指南]
本經文提出的「四念處」是極具操作性的心理觀察練習，可轉化為以下生活實踐：

1. **身體覺察（對應身觀）**：
   ➔ 技巧：在行走或工作時，將注意力轉向呼吸或腳掌接觸地面的感覺。
   ➔ 轉念：當身體感到疲累或疼痛時，不將其視為「我的痛苦」，而僅將其視為「色（物質身體）正在產生疼痛的現象」。

2. **情緒標記（對應受觀）**：
   ➔ 技巧：當強烈情緒（如憤怒、焦慮）升起時，在心中快速標記：「現在有『不快』的感受升起了」。
   ➔ 轉念：將自己從「我很生氣」轉變為「觀察到生氣的感受正在發生」，建立一個觀察者的空間，避免被情緒牽著走。

3. **心態監控（對應心觀）**：
   ➔ 技巧：定期檢查當下的心境。例如問自己：「我現在的心是散亂的，還是專注的？」
   ➔ 轉念：不對心境做道德評判（不責備自己的散亂），僅僅是「如實地覺知」當下的心理狀態。

4. **模式分析（對應法觀）**：
   ➔ 技巧：觀察反應的因果鏈。例如：看到某人（觸）➔ 產生厭惡感（受）➔ 產生想爭吵的衝動（行）。
   ➔ 轉念：體悟到所有的心理反應都是由因緣而生，而非一個永恆的「我」在操控，從而降低對特定結果的執著。

### [段落大意]
本經記錄了尊者跋陀羅與尊者阿難的對話。跋陀羅詢問能否透過特定修行成就阿羅漢，阿難回答應修習「四念處」，即對身體（色）、感受（受）、心態（心）與心理現象（法）進行正念觀照，以此達成解脫。
