# 雜阿含經 第 614 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有異比丘來詣佛所，稽首佛足，退坐一面，白佛言：「世尊！如所說『大丈夫』。云何名大丈夫、非大丈夫？」

佛告比丘：「善哉！善哉！比丘能問如來大丈夫義。諦聽，善思，當為汝說。若比丘身身觀念住，彼身身觀念住已，心不離欲，不得解脫，盡諸有漏，我說彼非為大丈夫。所以者何？心不解脫故。若比丘受、心、法法觀念住，心不離欲，不得解脫，盡諸有漏，我不說彼為大丈夫。所以者何？心不解脫故。

「若比丘身身觀念住，心得離欲，心得解脫，盡諸有漏，我說彼為大丈夫也。所以者何？心解脫故。若受、心、法法觀念住，受、心、法法觀念住已，心離貪欲，心得解脫，盡諸有漏，我說彼為大丈夫也。所以者何？心解脫故。是名，比丘！大丈夫及非大丈夫。」

佛說此經已，諸比丘聞佛所說，歡喜隨喜，禮足而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有異比丘來詣佛所，稽首佛足，退坐一面，白佛言：「世尊！如所說『大丈夫』。云何名大丈夫、非大丈夫？」**
　　那時，有一位特殊的比丘來到佛陀這裡，頂禮佛陀的足部，退到一邊坐下，對佛陀恭敬地說道：「世尊！關於您所提到的『大丈夫』，請問什麼樣的人才稱得上是大丈夫，而什麼樣的人則不是大丈夫？」

**◈ 佛告比丘：「善哉！善哉！比丘能問如來大丈夫義。諦聽，善思，當為汝說。**
　　佛陀告訴比丘說：「好極了！做得好！比丘能夠詢問如來關於大丈夫的義理。請仔細聽，好好思考，我現在為你們解說。

**◈ 若比丘身身觀念住，彼身身觀念住已，心不離欲，不得解脫，盡諸有漏，我說彼非為大丈夫。**
　　如果一位比丘雖然在修行中對色（物質身體）進行觀照並安住其中，但在觀照身體之後，內心依然沒有離開貪欲，無法獲得解脫，且未能盡除所有煩惱的漏盡，我說這樣的人並非大丈夫。

**◈ 所以者何？心不解脫故。**
　　為什麼呢？因為他的心沒有獲得解脫。

**◈ 若比丘受、心、法法觀念住，心不離欲，不得解脫，盡諸有漏，我不說彼為大丈夫。**
　　如果一位比丘對受（感受）、心（意識狀態）、法（心理現象）進行觀照並安住其中，但心依然沒有離開貪欲，無法獲得解脫，且未能盡除所有煩惱的漏盡，我不說他為大丈夫。

**◈ 所以者何？心不解脫故。**
　　為什麼呢？因為他的心沒有獲得解脫。

**◈ 若比丘身身觀念住，心得離欲，心得解脫，盡諸有漏，我說彼為大丈夫也。**
　　如果一位比丘對色（物質身體）進行觀照並安住其中，且心能離開貪欲，獲得解脫，並盡除所有煩惱的漏盡，我說他才是大丈夫。

**◈ 所以者何？心解脫故。**
　　為什麼呢？因為他的心已經獲得了解脫。

**◈ 若受、心、法法觀念住，受、心、法法觀念住已，心離貪欲，心得解脫，盡諸有漏，我說彼為大丈夫也。**
　　如果一位比丘對受（感受）、心（意識狀態）、法（心理現象）進行觀照並安住其中，在觀照之後，心能離開貪欲，獲得解脫，並盡除所有煩惱的漏盡，我說他才是大丈夫。

**◈ 所以者何？心解脫故。**
　　為什麼呢？因為他的心已經獲得了解脫。

**◈ 是名，比丘！大丈夫及非大丈夫。」**
　　比丘！這就是所謂的大丈夫與非大丈夫的區別。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜隨喜，禮足而去。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐，向佛陀行禮後才離開。

### [詞語註解]
1. **身、受、心、法 (Kāya, Vedanā, Citta, Dhamma)**：此處指「四念處」的修行對象。身指物質身體（色蘊）；受指對刺激產生的感受（苦、樂、捨）；心指意識的狀態（如貪、嗔、癡的運作）；法指一切心理現象或自然法則。
2. **有漏 (Āsava)**：指像滲漏一樣不斷流出的煩惱，主要包括欲漏（對感官欲望的執著）、有漏（對生存與存在的執著）以及無明漏（對真理的無知）。「盡諸有漏」即指徹底清除這些染污。
3. **解脫 (Vimokkha)**：指心靈從貪、嗔、癡的束縛中完全釋放，不再受輪迴之苦的狀態。
4. **大丈夫 (Mahāpurisa)**：在原始佛教脈絡中，並非指體格強壯或社會地位高，而是指在心靈修行上能透過觀照而離欲、斷漏，最終獲得解脫的覺悟者。

### [現代實作指南]
本經文揭示了一個關鍵的修行誤區：**「僅有觀照，而無離欲」**。許多現代人在練習正念（Mindfulness）時，僅止於「觀察」自己的情緒或身體感覺，但若觀察的目的是為了控制情緒，或在觀察中依然潛藏著對某種狀態的渴望（貪）或厭惡（嗔），則僅是形式上的觀照，而非真正的解脫。

**實作步驟：**
1. **覺察觀照 (Observation)** ➔ 當你感到憤怒或焦慮時，不要試圖壓抑，而是將其視為「受（感受）」或「心（意識狀態）」。告訴自己：「現在有一個憤怒的感受在升起」，而非「我很憤怒」。
2. **辨識欲求 (Identifying Desire)** ➔ 在觀照的同時，覺察內心是否在期待這個感受快點消失（厭離心），或期待某種快感能持續（貪著心）。
3. **練習離欲 (Practicing Detachment)** ➔ 意識到「感受」與「我」是分開的。感受像雲朵一樣飄過，而你只是觀察者。不試圖改變它，也不試圖抓住它。
4. **轉念技巧** ➔ 將「我想要解脫」的欲望，轉化為「我僅僅是觀察這一切的本質是無常的」之智慧。當你不再對觀照的結果產生執著時，心才會真正趨向解脫。

---

### [段落大意]
本經透過比丘的詢問，定義了真正的「大丈夫」。佛陀指出，無論是觀照身體、感受、心態或心理現象，若僅止於觀照而心不離貪欲，則無法解脫，不能稱作大丈夫；唯有在觀照中能真正離欲、盡除煩惱漏盡並獲得心靈解脫的人，才是真正的大丈夫。
