# 雜阿含經 第 817 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「亦復有三學。何等為三？謂增上戒學、增上意學、增上慧學。何等為增上戒學？若比丘住於戒波羅提木叉律儀，威儀行處具足，見微細罪則生怖畏，受持學戒。何等為增上意學？若比丘離欲、惡不善法，乃至第四禪具足住。何等為增上慧學？是比丘此苦聖諦如實知，集、滅、道聖諦如實知，是名增上慧學。」

爾時，世尊即說偈，如上所說。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「亦復有三學。何等為三？謂增上戒學、增上意學、增上慧學。**
　　那時，世尊告訴在場的眾位比丘說：「另外還有三種修行學習（三學）。這三種是什麼呢？即是指增上的戒學（卓越的道德規範）、增上的意學（卓越的心念專注）以及增上的慧學（卓越的真理洞察）。

**◈ 何等為增上戒學？若比丘住於戒波羅提木叉律儀，威儀行處具足，見微細罪則生怖畏，受持學戒。**
　　什麼是增上的戒學呢？如果比丘安住於波羅提木叉（比丘戒本）的律儀規範中，在舉止與行處上都能做到圓滿具足，即使看到微小的過失也會產生敬畏之心，並虔誠地受持並實踐這些學戒。

**◈ 何等為增上意學？若比丘離欲、惡不善法，乃至第四禪具足住。**
　　什麼是增上的意學呢？如果比丘能遠離貪欲、厭惡不善的法（心理狀態），直到能圓滿地安住在第四禪的定境之中。

**◈ 何等為增上慧學？是比丘此苦聖諦如實知，集、滅、道聖諦如實知，是名增上慧學。」**
　　什麼是增上的慧學呢？這位比丘能如實地認知到「苦聖諦（關於苦的真理）」，並且能如實地認知到「集聖諦（苦的因緣）」、「滅聖諦（苦的止息）」以及「道聖諦（解脫之道的實踐）」，這就稱為增上的慧學。」

**◈ 爾時，世尊即說偈，如上所說。**
　　那時，世尊便以偈頌的形式，將上述內容再次闡述。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **三學 (Tisso sikkhā)**：佛教修行的三大核心基礎，包含戒（Sīla）、定（Samādhi，文中稱意學）與慧（Paññā）。三者相輔相成，戒為定之基，定為慧之基。
2. **波羅提木叉 (Pātimokkha)**：指比丘與比丘尼必須遵守的戒律清單，是僧團維持純淨與和諧的制度基礎。
3. **第四禪 (Catuttha Jhāna)**：禪定修行的最高階段，特徵是捨（Upekkhā）與純淨的意識，心靈達到極度的平靜與清明，不再受樂或苦的擾動。
4. **四聖諦 (Cattāri Ariyasaccāni)**：佛法最核心的真理，包含苦（人生本質的不滿足）、集（苦的產生原因）、滅（苦的完全止息）與道（達成止息的具體路徑）。

### [現代實作指南]
本經提出的「三學」可轉化為現代人的心理成長路徑：

1. **實踐「增上戒學」➔ 建立生活秩序與底線**：
   - **操作方法**：設定一套個人行為準則（如：不欺騙、不傷害他人）。當意識到自己有微小的失誤時，不採取「沒關係」的敷衍態度，而是誠實面對並反省。
   - **轉念技巧**：將「遵守規則」視為「保護心靈的純淨」，而非被束縛。

2. **實踐「增上意學」➔ 培養深度專注與情緒調節**：
   - **操作方法**：練習每日 10-20 分鐘的正念觀察，刻意地將注意力從對物慾的渴求（離欲）或對負面情緒的糾結（惡不善法）中抽離。
   - **轉念技巧**：當雜念生起時，不評判它，僅僅將其視為「一種心理現象」而讓其流過，直到心靈進入平靜狀態。

3. **實踐「增上慧學」➔ 邏輯分析痛苦的因果**：
   - **操作方法**：面對壓力或痛苦時，運用四聖諦分析法：
     - **苦** ➔ 承認目前的痛苦狀態（不逃避）。
     - **集** ➔ 找出痛苦的根源（是否源於對某事的執著或期待？）。
     - **滅** ➔ 想像如果放下該執著，心靈將會如何平靜。
     - **道** ➔ 制定具體的行動計畫來改變現狀。
   - **轉念技巧**：將痛苦視為「覺察的機會」，而非「不幸的遭遇」。

### 段落大意 
本經闡述了佛教修行的核心「三學」：首先是「戒學」，強調透過嚴格遵守律儀與對微小過失的敬畏來淨化行為；其次是「意學」，強調透過遠離欲望與不善法來達到深層的禪定；最後是「慧學」，強調對四聖諦的如實認知以達成智慧的覺悟。三者共同構成從行為、心念到真理洞察的完整解脫路徑。
