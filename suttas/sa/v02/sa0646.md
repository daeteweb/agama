# 雜阿含經 第 646 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有五根。何等為五？謂信根、精進根、念根、定根、慧根。信根者，當知是四不壞淨；精進根者，當知是四正斷；念根者，當知是四念處；定根者，當知是四禪；慧根者，當知是四聖諦。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有五根。何等為五？**
　　那時，世尊告訴在場的眾位比丘說：「有五種修行根基（五根）。什麼是這五種根基呢？

**◈ 謂信根、精進根、念根、定根、慧根。**
　　即是指信根（信仰的力量）、精進根（努力的動力）、念根（正念的覺知）、定根（專注的定力）以及慧根（洞察的智慧）。

**◈ 信根者，當知是四不壞淨；**
　　關於信根，應當了解它是指四不壞淨（四種不可毀壞的清淨心）。

**◈ 精進根者，當知是四正斷；**
　　關於精進根，應當了解它是指四正斷（四種正確地斷除煩惱的方法）。

**◈ 念根者，當知是四念處；**
　　關於念根，應當了解它是指四念處（對色（物質身體）、受（感受）、心、法四個面向的持續覺察）。

**◈ 定根者，當知是四禪；**
　　關於定根，應當了解它是指四禪（四種由淺入深、心境絕對寧靜的禪定狀態）。

**◈ 慧根者，當知是四聖諦。」**
　　關於慧根，應當了解它是指四聖諦（關於苦（痛苦的本質）、集（苦因）、滅（苦止）、道（解脫之道）的四個聖妙真理）。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **五根 (Indriya)**：指在修行過程中起主導作用的五種心理能力，如同植物的根基，決定了修行能否成長。
2. **四不壞淨 (Cattāri asankhatā visuddhi)**：指信、精進、念、定四種清淨的心理狀態，一旦建立便不易被外界染污或毀壞。
3. **四正斷 (Sammappadhāna)**：又稱四正勤。包括：防止未生之惡、斷除已生之惡、生起未生之善、增長已生之善。
4. **四念處 (Satipaṭṭhāna)**：正念修行的四個對象，分別是觀身、觀受、觀心、觀法。
5. **四禪 (Jhāna)**：指透過專注修行達到的四個層次的定境，從初步的喜樂到最終的捨心與平靜。
6. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四個真理，揭示了生命痛苦的真相、原因、目標以及達成目標的路徑。

### [現代實作指南]
本經將「五根」與具體的「修行方法」對接，可轉化為以下生活實踐步驟：

1. **建立正向信念（信根 ➔ 四不壞淨）**：
   - 實作：不要盲目相信，而是在生活中觀察「因果」的運作。當你發現透過改變心念能改善情緒時，對佛法的信心會從「盲信」轉為「理信」，形成穩固的心理基調。

2. **建立良性習慣（精進根 ➔ 四正斷）**：
   - 實作：採取「微習慣」策略。意識到負面情緒（如憤怒）萌芽時立即截斷（防止未生），對已有的壞習慣進行分析並逐步取代（斷除已生），並刻意練習感恩與寬容（生起與增長善法）。

3. **練習覺知生活（念根 ➔ 四念處）**：
   - 實作：在日常活動中加入「標記法」。行走時覺知腳底接觸（觀身），感到不適時標記「這是受」（觀受），發現心煩時標記「心現在煩躁」（觀心），分析此現象是因緣而生（觀法）。

4. **訓練深度專注（定根 ➔ 四禪）**：
   - 實作：每天設定 10-20 分鐘的單一任務時間（如專注呼吸）。排除所有數位干擾，將心意識集中於一點，體驗從躁動到平靜的心理轉變。

5. **運用邏輯分析（慧根 ➔ 四聖諦）**：
   - 實作：面對壓力時，運用四聖諦分析法：
     - 苦 ➔ 承認目前的壓力狀態。
     - 集 ➔ 分析壓力的根源是否來自於對某事的執著（貪）或不滿（嗔）。
     - 滅 ➔ 確認如果放下執著，壓力將會消失。
     - 道 ➔ 採取具體的行動（如正念、正思惟）來解決問題。

### [段落大意]
本經由佛陀闡述修行的五種核心能力（五根），並將其對應至具體的實踐法門：信根對應四不壞淨，精進根對應四正斷，念根對應四念處，定根對應四禪，慧根則對應四聖諦。旨在告訴修行者，能力（根）必須透過具體的實踐（法）才能真正成長。
