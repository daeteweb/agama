# 雜阿含經 第 761 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「我當說學及無學。諦聽，善思念之。何等為學？謂學正見成就，學正志、正語、正業、正命、正方便、正念、正定成就，是名為學。何等為無學？謂無學正見成就，無學正志、正語、正業、正命、正方便、正念、正定成就，是名無學。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如學、無學，如是正士、如是大士，亦如是說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「我當說學及無學。諦聽，善思念之。**
　　那時，世尊告訴在場的眾位比丘說：「我現在要為你們說明關於『學』與『無學』這兩種修行狀態。請仔細聽，好好思考並銘記在心。

**◈ 何等為學？**
　　什麼叫做『學』？」

**◈ 謂學正見成就，學正志、正語、正業、正命、正方便、正念、正定成就，是名為學。**
　　「也就是指在修行過程中，致力於成就正見（正確的見解）、正志（正確的思維）、正語（正確的言語）、正業（正確的行為）、正命（正確的生活方式）、正方便（正確的精進）、正念（正確的覺知）、正定（正確的專注），這種仍在訓練與實踐階段的狀態，就稱為『學』。」

**◈ 何等為無學？**
　　「什麼叫做『無學』？」

**◈ 謂無學正見成就，無學正志、正語、正業、正命、正方便、正念、正定成就，是名無學。」**
　　「也就是指已經完全成就了正見（正確的見解）、正志（正確的思維）、正語（正確的言語）、正業（正確的行為）、正命（正確的生活方式）、正方便（正確的精進）、正念（正確的覺知）、正定（正確的專注），達到不再需要進一步學習的圓滿境界，這就稱為『無學』。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如學、無學，如是正士、如是大士，亦如是說。**
　　關於『學』與『無學』的定義，對於『正士』（正直的修行者）與『大士』（偉大的修行者）也是同樣的說明。

### [詞語註解]
1. **學 (Sekha)**：指尚未達到阿羅漢果位，仍需透過修行來消除煩惱的修行者（包含須陀洹、須陀洹果、阿那含等）。
2. **無學 (Asekha)**：指已經完全斷除所有煩惱，達到解脫境界，不再需要任何進一步修習的阿羅漢。
3. **八正道 (Ariya Aṭṭhaṅgika Magga)**：佛法中解脫苦難的唯一路徑，包含正見、正思惟（正志）、正語、正業、正命、正精進（正方便）、正念、正定。
4. **正士/大士**：在此脈絡下指不同層次或類型的修行實踐者。

### [現代實作指南]
本經文將修行分為「過程（學）」與「結果（無學）」，在現代生活中可採取以下實踐步驟：

1. **建立「學習者」的謙卑心態**：
   認清自己目前處於「學」的階段。當我們在生活中遇到衝突或情緒起伏時，不要急於評判自己是否「開悟」，而應將其視為練習八正道的機會。例如：在憤怒時練習「正念（覺知）」與「正語（不惡口）」。

2. **八正道的日常審核表**：
   每天結束前，對照八正道進行自我檢視：
   ➔ **正見**：我今天是否能客觀看待事情，而非陷入偏見？
   ➔ **正志**：我的意圖是否出於善意，而非貪婪或嗔恨？
   ➔ **正語/正業**：我的言行是否對他人造成傷害？
   ➔ **正方便**：我是否在正確的方向上努力，而非盲目忙碌？

3. **從「追求結果」轉向「專注過程」**：
   「無學」是自然而然的結果，而非強求的目標。實作重點在於將「學」的過程（八正道的實踐）融入每一件小事中。當你不再執著於「我要變成聖人」，而僅僅專注於「現在這一刻我要正確地說話/思考」時，你便是在實踐真正的「學」。

### [段落大意]
本經由佛陀定義「學」與「無學」兩種狀態：前者是指仍在實踐八正道以追求成就的修行過程；後者是指已完全成就八正道、不再需要修習的圓滿境界。此定義同樣適用於所有層次的修行者。
