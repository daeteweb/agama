# 雜阿含經 第 611 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有善法聚、不善法聚。云何善法聚？所謂四念處，是為正說。所以者何？純一滿淨聚者，所謂四念處。云何為四？謂身身觀念處，受、心、法法觀念處。云何不善聚？不善聚者，所謂五蓋，是為正說。所以者何？純一逸滿不善聚者，所謂五蓋。何等為五？謂貪欲蓋、瞋恚蓋、睡眠蓋、掉悔蓋、疑蓋。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有善法聚、不善法聚。**
　　那時，世尊告訴在場的眾位比丘說：「世間存在著『善法的聚集』以及『不善法的聚集』。

**◈ 云何善法聚？所謂四念處，是為正說。**
　　什麼是善法的聚集？也就是所謂的『四念處』（四種正念的確立），這是正確的說法。

**◈ 所以者何？純一滿淨聚者，所謂四念處。**
　　為什麼這麼說呢？因為四念處是純淨、完整且統一的正面心理狀態聚集。

**◈ 云何為四？謂身身觀念處，受、心、法法觀念處。**
　　這四種是什麼？是指對色（物質身體）的觀照、對受（感受）的觀照、對心（心理狀態）的觀照，以及對法（一切現象的本質）的觀照。

**◈ 云何不善聚？不善聚者，所謂五蓋，是為正說。**
　　那麼，什麼是不善法的聚集？不善法的聚集是指『五蓋』（五種遮蔽智慧的障礙），這是正確的說法。

**◈ 所以者何？純一逸滿不善聚者，所謂五蓋。**
　　為什麼這麼說呢？因為五蓋是純粹由懈怠、混亂且充滿不善特質的心理狀態所聚集而成的。

**◈ 何等為五？謂貪欲蓋、瞋恚蓋、睡眠蓋、掉悔蓋、疑蓋。」**
　　這五種障礙是什麼？分別是：貪欲（對感官欲望的渴求）、瞋恚（對不滿之事的憤怒）、睡眠（精神昏沉與懈怠）、掉悔（心神不安與後悔），以及疑（對正法產生懷疑）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四念處 (Satipaṭṭhāna)**：佛教修行中建立正念的四個對象，包含觀照身體（身）、感受（受）、心靈狀態（心）以及心理現象的本質（法），旨在透過覺知打破對「我」的執著。
2. **五蓋 (Nīvaraṇa)**：指五種遮蔽心靈清明、阻礙禪定與智慧產生的心理障礙，如同雲層遮蔽太陽一般，使修行者無法看清真理。
3. **善法/不善法 (Kusala/Akusala)**：在原始佛教中，「善」是指能減少痛苦、導向解脫的心理狀態；「不善」則是會增加痛苦、導致輪迴的心理狀態。

### [現代實作指南]
本經文提供了一套簡單的「心理狀態對照表」，可用於日常的情緒管理與心智訓練：

1. **識別「蓋」的出現（覺察障礙）**：
   當你在工作或學習中感到效率低下時，試著辨識目前被哪一種「蓋」遮蔽：
   ➔ 是在幻想未來的獎賞（貪欲）？
   ➔ 是在對同事的錯誤感到憤怒（瞋恚）？
   ➔ 是感到精神萎靡、想逃避（睡眠）？
   ➔ 是在擔心明天會出錯而焦慮（掉悔）？
   ➔ 還是對目前的方法是否正確感到猶豫（疑）？

2. **啟動「念處」的轉化（建立正念）**：
   一旦識別出「蓋」，不要對其產生反感，而是將注意力轉移至四念處的觀照：
   ➔ **身念處**：感受呼吸的起伏，或感受身體在椅子上的壓力，將心拉回當下。
   ➔ **受念處**：覺察此刻心中產生的「不舒服感」或「焦慮感」，僅僅觀察它，而不被它捲走。
   ➔ **心念處**：觀察心靈目前的狀態（例如：我的心現在很混亂），像旁觀者一樣看待自己的心。
   ➔ **法念處**：體悟到「憤怒」或「焦慮」本身也是無常的，它們隨緣而生，隨緣而滅。

3. **實踐目標**：
   將「不善法聚」（五蓋）轉化為「善法聚」（四念處），使心靈從混亂狀態（逸滿）轉向清明狀態（純一滿淨）。

### [段落大意]
本經由世尊定義了「善法聚」與「不善法聚」的區別。善法聚是指能導向清淨的四念處（觀照身、受、心、法）；不善法聚則是遮蔽智慧的五蓋（貪、瞋、睡眠、掉悔、疑）。修行者應透過覺知，將心從五蓋的障礙中解脫，轉向四念處的正念修行。
