# 雜阿含經 第 846 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘……如上說。差別者：「何等為聖道如實知見？謂八聖道，正見，乃至正定。」

次經亦如是說。差別者：「何等為聖道如實知見？謂十二支緣起如實知見。如所說：『是事有故是事有，是事起故是事起。如緣無明行，緣行識，緣識名色，緣名色六入處，緣六入處觸，緣觸受，緣受愛，緣愛取，緣取有，緣取有，緣有生，緣生老、病、死、憂、悲、苦、惱。』是名聖弟子如實知見。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘……如上說。**
　　那時，世尊告訴在場的眾位比丘說……（內容）如前文所述。

**◈ 差別者：「何等為聖道如實知見？謂八聖道，正見，乃至正定。」**
　　而不同的部分在於：「什麼樣的認知才算是對聖道（解脫之道的真理）的如實知見？也就是指對八正道（通往解脫的八種正確路徑）的如實知見，從正見開始，直到正定為止。」

**◈ 次經亦如是說。**
　　接下來的經文也同樣如此地說。

**◈ 差別者：「何等為聖道如實知見？謂十二支緣起如實知見。」**
　　而不同的部分在於：「什麼樣的認知才算是對聖道（解脫之道的真理）的如實知見？也就是指對十二因緣（十二個環環相扣的因果鏈條）的如實知見。」

**◈ 如所說：『是事有故是事有，是事起故是事起。』**
　　正如經中所說：『因為這個條件存在，所以那個結果才會存在；因為這個條件興起，所以那個結果才會興起。』

**◈ 『如緣無明行，緣行識，緣識名色，緣名色六入處，緣六入處觸，緣觸受，緣受愛，緣愛取，緣取有，緣有生，緣生老、病、死、憂、悲、苦、惱。』**
　　『如同因為無明（對真理的無知）而導致行（心的造作）➔ 因為行（心的造作）而導致識（意識）➔ 因為識（意識）而導致名色（精神與物質的組合）➔ 因為名色（精神與物質的組合）而導致六入（六種感官接收處）➔ 因為六入（六種感官接收處）而導致觸（接觸）➔ 因為觸（接觸）而導致受（感受）➔ 因為受（感受）而導致愛（貪愛）➔ 因為愛（貪愛）而導致取（執取）➔ 因為取（執取）而導致有（存在）➔ 因為有（存在）而導致生（出生）➔ 因為生（出生）而導致老、病、死，以及隨之而來的憂、悲、苦、惱。』

**◈ 是名聖弟子如實知見。**
　　這就稱為聖弟子對真理的如實知見。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **八正道 (Ariya aṭṭhaṅgika magga)**：佛法中消除苦諦的實踐路徑，包含正見、正思惟、正語、正業、正命、正精進、正念、正定。
2. **十二因緣 (Paṭicca-samuppāda)**：描述生命如何因無明而陷入輪迴苦海的十二個環節，強調一切現象皆是依因緣而生，並無永恆不變的自我。
3. **無明 (Avijjā)**：對四聖諦（苦、集、滅、道）以及緣起真理的無知，是輪迴苦難的最根本原因。
4. **名色 (Nāmarūpa)**：名指精神功能（如受、想、行、識），色指物質身體。兩者共同構成個體的生命形式。
5. **六入 (Saḷāyatana)**：指六種感官接收處，即眼、耳、鼻、舌、身、意。

### [現代實作指南]
本經文揭示了苦難產生的「連鎖反應」。在現代生活中，我們可以透過觀察「受 ➔ 愛 ➔ 取」這個關鍵環節來實踐轉念：

1. **覺察「受」的發生**：當你感到憤怒或悲傷時（受），不要立刻反應，而是先在心中標記：「現在有一個『不舒服的感受』產生了」。
2. **切斷「愛」的貪著**：通常在感受之後，我們會產生「希望這個不舒服消失」或「希望對方道歉」的強烈渴望（愛/貪愛）。此時提醒自己，感受是無常的，不需要強行控制它。
3. **停止「取」的執著**：當我們試圖透過爭吵或自責來解決感受時，就是「取（執取）」。實作方法是：僅僅觀察感受的起伏，而不對其進行「我的憤怒」、「我的痛苦」這樣的認同。
4. **回溯至「無明」**：思考這份痛苦是否源於對「事物本質是變動不居（無常）」的誤解。當意識到「這一切只是因緣的組合」而非「我」的失敗時，連鎖反應將會停止，從而獲得短暫的寧靜與解脫。

### [段落大意]
本段經文強調聖弟子必須具備兩種核心的如實知見：一是對「八正道」的實踐路徑之認知，二是對「十二因緣」因果鏈條的深刻體悟。文中詳細列舉了從無明開始，經過意識、感官接觸、感受、貪愛、執取，最終導致出生與老病死苦的運作過程，說明苦難是如何由因緣遞進而產生的。
