# 雜阿含經 第 710 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「聖弟子清淨信心，專精聽法者，能斷五法，修習七法，令其滿足。何等為五？謂貪欲蓋，瞋恚、睡眠、掉悔、疑，此蓋則斷。何等七法？謂念覺支，擇法、精進、猗、喜、定、捨覺支。此七法修習滿足，淨信者，謂心解脫；智者，謂慧解脫；貪欲染心者，不得不樂；無明染心者，慧不清淨。是故，比丘！離貪欲者心解脫，離無明者慧解脫。若彼比丘離貪欲，心解脫，得身作證；離無明，慧解脫，是名比丘斷愛縛、結，慢無間等，究竟苦邊。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「聖弟子清淨信心，專精聽法者，能斷五法，修習七法，令其滿足。**
　　「聖弟子若擁有純淨的信心，並且專注精進地聆聽法（真理）的人，能夠斷除五種障礙，並修習七種覺悟的要素，使之圓滿。

**◈ 何等為五？謂貪欲蓋，瞋恚、睡眠、掉悔、疑，此蓋則斷。**
　　這五種障礙是什麼？指的是貪欲蓋（對感官欲望的執著）、瞋恚（憤怒與厭惡）、睡眠（昏沉與懈怠）、掉悔（心神不安與後悔）、以及疑（對正法的不確定），這些遮蔽心靈的蓋（障礙）就必須被斷除。

**◈ 何等七法？謂念覺支，擇法、精進、猗、喜、定、捨覺支。**
　　那七種法是什麼？指的是念覺支（正念的覺悟要素）、擇法（辨別真理的覺悟要素）、精進（努力修行的覺悟要素）、猗（輕安自在的覺悟要素）、喜（法喜的覺悟要素）、定（專注不亂的覺悟要素）、以及捨（平等心地的覺悟要素）。

**◈ 此七法修習滿足，淨信者，謂心解脫；智者，謂慧解脫；**
　　當這七種覺悟要素修習圓滿後，擁有純淨信心的人，稱為心解脫（心靈從執著中解脫）；而具備智慧的人，稱為慧解脫（透過智慧洞察真理而解脫）。

**◈ 貪欲染心者，不得不樂；無明染心者，慧不清淨。**
　　那些心被貪欲染污的人，無法獲得真正的快樂；而那些心被無明（對真理的無知）染污的人，其智慧是不清淨的。

**◈ 是故，比丘！離貪欲者心解脫，離無明者慧解脫。**
　　因此，比丘們！遠離貪欲的人能獲得心解脫，遠離無明（對真理的無知）的人能獲得慧解脫。

**◈ 若彼比丘離貪欲，心解脫，得身作證；離無明，慧解脫，是名比丘斷愛縛、結，慢無間等，究竟苦邊。」**
　　如果那位比丘遠離了貪欲，達到心解脫，並以自身的經驗證實；同時遠離了無明，達到慧解脫，這就稱為比丘斷除了愛縛（貪愛的束縛）、結（繫縛心靈的結）、以及慢（傲慢）等無間斷的煩惱，最終到達苦（Dukkha）的邊際（即終結苦難）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **五蓋 (Pañca Nīvaraṇa)**：指遮蔽心靈、妨礙禪定與智慧的五種心理障礙，包括貪欲、瞋恚、睡眠（昏沉）、掉悔（躁動與悔恨）、以及疑。
2. **七覺支 (Satta Bojjhaṅga)**：指導向覺悟的七種心理因素，透過修習可使心靈從迷茫走向清明。
3. **心解脫 (Citta-vimutti)**：指心靈透過遠離貪欲與執著，達到一種不再被煩惱牽引的自由狀態。
4. **慧解脫 (Paññā-vimutti)**：指透過對無常、無我、苦的深刻洞察（智慧），徹底根除無明而獲得的解脫。
5. **無明 (Avijjā)**：對四聖諦及緣起真理的無知，是所有苦難與輪迴的根本原因。
6. **結 (Samyojana)**：比喻將眾生繫縛在輪迴中的心理鏈條，如對自我的執著、對感官的渴求等。

### [現代實作指南]
本經文提供了一套從「清除障礙」到「建立正向心理」的完整路徑，可轉化為以下生活實踐：

1. **識別並標記「五蓋」 (障礙排除)**：
   - 當你在工作或學習中感到分心時，試著在心中標記目前的狀態：
     - 想要刷手機 ➔ 標記為「貪欲蓋」。
     - 對同事感到憤怒 ➔ 標記為「瞋恚蓋」。
     - 感到精神萎靡、想睡覺 ➔ 標記為「睡眠蓋」。
     - 心煩意亂、對過去決定後悔 ➔ 標記為「掉悔蓋」。
     - 對目標感到迷茫、懷疑意義 ➔ 標記為「疑蓋」。
   - **轉念技巧**：一旦標記，即意識到這只是暫時的「遮蔽物」，而非你的本質，從而降低其對你的控制力。

2. **啟動「七覺支」 (正向建構)**：
   - **擇法 ➔ 精進**：先分辨出目前對自己有益的行為（擇法），然後投入努力（精進）。
   - **喜 ➔ 猗 (輕安)**：在努力過程中，觀察到微小的進步並感到喜悅（喜），讓身體與心靈進入放鬆且清明的狀態（輕安）。
   - **念 ➔ 定 ➔ 捨**：持續覺察當下（念），使心專注於單一對象（定），最終對所有結果保持平等心，不隨好壞而波動（捨）。

3. **區分「心」與「慧」的清理**：
   - **清理情緒 (心解脫)**：練習「不執著」。當欲望升起時，告訴自己：「這只是暫時的感受」，練習放下對特定結果的強烈渴求。
   - **清理認知 (慧解脫)**：練習「看真相」。當你對某事感到痛苦時，分析其「緣起」 ➔ 思考這件事是由哪些條件構成的？它是否在改變（無常）？它是否真的屬於我（無我）？

### [段落大意]
本經闡述了聖弟子解脫的具體路徑：首先必須斷除五蓋（貪、瞋、睡眠、掉悔、疑）等心理障礙，隨後修習七覺支（念、擇法、精進、輕安、喜、定、捨）以圓滿覺悟。透過遠離貪欲可獲得心解脫，遠離無明可獲得慧解脫。當心與慧雙雙解脫時，比丘能斷除愛縛、結與慢等煩惱，最終終結苦難。
