# 雜阿含經 第 675 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘……如上說。差別者：「彼信力，當知是四不壞淨。精進力者，當知是四正斷。念力者，當知四念處。定力者，當知是四禪。慧力者，當知是四聖諦。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

---

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘……如上說。**
　　那時，世尊告訴在場的眾位比丘說……（內容）如前文所述。

**◈ 差別者：「彼信力，當知是四不壞淨。**
　　不同之處在於：「所謂的信力（信仰的力量），應當了解是指『四不壞淨』（四種不被摧毀的清淨心）。

**◈ 精進力者，當知是四正斷。**
　　所謂的精進力（努力修行的力量），應當了解是指『四正斷』（四種正確地斷除煩惱的方法）。

**◈ 念力者，當知四念處。**
　　所謂的念力（正念覺知的力量），應當了解是指『四念處』（對身體、感受、心法、心法之對象的四種覺知修行）。

**◈ 定力者，當知是四禪。**
　　所謂的定力（心專一的力量），應當了解是指『四禪』（四種深層的禪定狀態）。

**◈ 慧力者，當知是四聖諦。」**
　　所謂的慧力（智慧洞察的力量），應當了解是指『四聖諦』（苦、集、滅、道這四個神聖的真理）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四不壞淨 (Cattāri Ariyasaṭṭhānī)**：指四種不可摧毀的清淨信仰，通常指對佛、法、僧以及解脫道（或正法）的堅定信心，使心不被世俗染污所動搖。
2. **四正斷 (Sammappadhāna)**：又稱四正勤。包括：防止未生之惡、斷除已生之惡、生起未生之善、增長已生之善。
3. **四念處 (Satipaṭṭhāna)**：正念修行的四個對象，分別為：身念處（觀察身體）、受念處（觀察感受）、心念處（觀察心念）、法念處（觀察心法對象）。
4. **四禪 (Jhāna)**：指由淺入深四個階段的禪定狀態，透過捨棄感官欲望與心靈雜染，達到極度的寧靜與專注。
5. **四聖諦 (Cattāri Ariyasaccāni)**：佛教核心真理，包含苦（人生本質的苦）、集（苦的因 ➔ 貪愛與執著）、滅（苦的止 ➔ 涅槃）、道（解脫之途 ➔ 八正道）。

### [現代實作指南]
本經文將抽象的「力」具體化為可操作的修行路徑。在現代生活中，我們可以將其轉化為以下心理實踐：

1. **建立信力（價值錨定）**：
   ➔ 找出心中絕對認同的真理或價值觀（如：誠實、慈悲），在面對壓力或誘惑時，將其作為心理錨點，不隨波逐流。
2. **實踐精進力（習慣管理）**：
   ➔ 運用「四正斷」邏輯：覺察並阻止壞習慣的萌芽 ➔ 採取行動切斷現有的壞習慣 ➔ 建立正向的新習慣 ➔ 持之以恆地強化好習慣。
3. **培養念力（正念覺察）**：
   ➔ 在日常中練習「四念處」：身體緊繃時覺察身體 ➔ 情緒起伏時覺察感受 ➔ 雜念紛飛時覺察心念 ➔ 分析此現象的因緣（法）。
4. **鍛鍊定力（深度專注）**：
   ➔ 練習單一任務處理（Single-tasking），在特定時間內將意識完全集中於當下的一件事，減少心靈的碎片化。
5. **啟動慧力（邏輯分析）**：
   ➔ 面對問題時，運用「四聖諦」分析法：定義問題（苦） ➔ 找出問題根源（集） ➔ 想像解決後的理想狀態（滅） ➔ 制定具體的執行步驟（道）。

### 段落大意 
本段經文將修行所需的五種力量（信、精進、念、定、慧）分別對應到具體的修行法門：信力對應四不壞淨，精進力對應四正斷，念力對應四念處，定力對應四禪，慧力對應四聖諦，為修行者提供明確的實踐指標。
