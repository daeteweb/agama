# 雜阿含經 第 989 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，尊者大目揵連住耆闍崛山，後夜起經行，見有光明遍照迦蘭陀竹園。見已，作是念：「今夜或有大力鬼神詣世尊所，故有此光明。」

時，尊者大目揵連晨朝往詣佛所，稽首佛足，退坐一面，白佛言：「世尊！我於昨暮後夜出房經行，見勝光明普照迦蘭陀竹園。見已，作是念：『有何大力鬼神詣世尊所，故有此光明？』」

佛告尊者大目揵連：「昨暮後夜，釋提桓因來詣我所，稽首作禮，退坐一面。」

如上修多羅廣說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，尊者大目揵連住耆闍崛山，後夜起經行，見有光明遍照迦蘭陀竹園。**
　　那時，尊者大目揵連住在耆闍崛山，在深夜時分起床進行經行（行走禪修），看見有一道強烈的光芒遍照著迦蘭陀竹園。

**◈ 見已，作是念：「今夜或有大力鬼神詣世尊所，故有此光明。」**
　　看見之後，他心裡如此想：「今晚或許是有能力強大的鬼神來到世尊這裡，所以才產生這樣的光芒。」

**◈ 時，尊者大目揵連晨朝往詣佛所，稽首佛足，退坐一面，白佛言：**
　　於是，尊者大目揵連在早晨前往佛陀所在之處，向佛陀的足下頂禮，然後退到一邊坐下，對佛陀恭敬地說道：

**◈ 「世尊！我於昨暮後夜出房經行，見勝光明普照迦蘭陀竹園。見已，作是念：『有何大力鬼神詣世尊所，故有此光明？』」**
　　「世尊！我在昨晚深夜走出房舍進行經行（行走禪修）時，看見極其殊勝的光芒普照著迦蘭陀竹園。看見之後，我心裡如此想：『是有哪一位能力強大的鬼神來到世尊這裡，所以才產生這樣的光芒？』」

**◈ 佛告尊者大目揵連：「昨暮後夜，釋提桓因來詣我所，稽首作禮，退坐一面。」**
　　佛陀告訴尊者大目揵連：「在昨晚深夜，天主釋提桓因來到我這裡，向我頂禮，然後退到一邊坐下。」

**◈ 如上修多羅廣說，歡喜隨喜，作禮而去。**
　　如上所述的修多羅（經文）詳細闡述了這件事，（大目揵連）心生歡喜並隨喜佛陀的教導，向佛陀行禮後才離開。

### [詞語註解]
1. **經行 (Cankamana)**：一種行走禪修法。修行者在限定的範圍內緩慢行走，將注意力集中在足部的觸感與身體的移動上，旨在結合動靜修行，提升正念（Sati）。
2. **釋提桓因 (Śakra)**：印度神話與早期佛教中的天主（因陀羅），居住在忉利天（Trayastrimsa），在佛教典籍中常扮演護法者的角色，對佛陀深懷敬意。
3. **修多羅 (Sutta)**：巴利語 Sutta 的音譯，意指「線」或「經線」，比喻佛陀的教法如線般將真理串聯起來，後指稱佛陀的講經紀錄。
4. **隨喜 (Muditā)**：一種正向的心理狀態，指看到他人獲得快樂、成就或聽聞正法時，心中生起純粹的歡喜，而不產生嫉妒心。

### [現代實作指南]
本經文雖記載超自然現象，但其核心在於「觀察 ➔ 質疑 ➔ 驗證」的認知過程，可轉化為以下生活實踐：

1. **覺察外部異象（正念觀察）**：
   當生活中出現不尋常的現象（如突如其來的機會或危機）時，學習大目揵連的第一步：**單純觀察**。不要立即陷入恐慌或過度興奮，而是先記錄下「我看到了什麼」。

2. **建立理性假設（善思）**：
   面對異象時，在心中建立合理的假設（如大目揵連推測有大力鬼神到訪），而非盲目地將其視為吉兆或凶兆。這能幫助我們在面對未知時保持心靈的穩定。

3. **尋求權威驗證（正見）**：
   當個人推論無法得出結論時，尋找具備智慧的導師或可靠的事實來源進行核實。這體現了「不盲信」的科學精神，將主觀猜測 ➔ 客觀事實。

4. **轉化為隨喜心（情緒轉念）**：
   當得知真相（如天主到訪）後，將好奇心轉化為「隨喜」的喜悅。在生活中，練習將關注點從「為什麼這件事發生在我身上」轉移到「這件事展現了什麼樣的真理/美好」，以提升內心的正能量。

### 段落大意 
尊者大目揵連在深夜經行時，見到強光普照竹園，推測有強大鬼神拜訪佛陀。次日早晨向佛陀請益，佛陀證實昨夜天主釋提桓因曾前來頂禮。大目揵連聞後心生歡喜，行禮離去。
