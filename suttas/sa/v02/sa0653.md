# 雜阿含經 第 653 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘……如上說。差別者：「若比丘於彼五根增上明利滿足者，得阿羅漢俱分解脫。若軟、若劣者，得身證。於彼若軟、若劣，得見到。於彼若軟、若劣，得信解脫。於彼若軟、若劣，得一種。於彼若軟、若劣，得斯陀含。於彼若軟、若劣，得家家。於彼若軟、若劣，得七有。於彼若軟、若劣，得法行。於彼若軟、若劣，得信行。是名比丘根波羅蜜因緣知果波羅蜜，果波羅蜜因緣知人波羅蜜。如是滿足者作滿足事，減少者作減少事，彼諸根則不空無果，若無此諸根者，我說彼為作凡夫數。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘……如上說。**
　　那時，世尊告訴在場的眾位比丘說……（內容如前文所述）。

**◈ 差別者：「若比丘於彼五根增上明利滿足者，得阿羅漢俱分解脫。**
　　不同的之處在於：「如果比丘在五根（信、精進、念、定、慧）方面能增長、明晰且利落得圓滿，就能成就阿羅漢，獲得完全的解脫」。

**◈ 若軟、若劣者，得身證。**
　　如果根基較弱或較劣，則能獲得身證（以身體感受證悟）。

**◈ 於彼若軟、若劣，得見到。**
　　如果根基更弱或更劣，則能獲得見到（初步見到真理）。

**◈ 於彼若軟、若劣，得信解脫。**
　　如果根基更弱或更劣，則能獲得透過信心的解脫。

**◈ 於彼若軟、若劣，得一種。**
　　如果根基更弱或更劣，則能獲得一種（特定的果位）。

**◈ 於彼若軟、若劣，得斯陀含。**
　　如果根基更弱或更劣，則能成就斯陀含（一次回者）。

**◈ 於彼若軟、若劣，得家家。**
　　如果根基更弱或更劣，則能成就家家（指不還者，即阿那含）。

**◈ 於彼若軟、若劣，得七有。**
　　如果根基更弱或更劣，則能獲得七有（七種存在狀態的證悟）。

**◈ 於彼若軟、若劣，得法行。**
　　如果根基更弱或更劣，則能獲得法行（依據法而修行）。

**◈ 於彼若軟、若劣，得信行。**
　　如果根基更弱或更劣，則能獲得信行（依據信心而修行）。

**◈ 是名比丘根波羅蜜因緣知果波羅蜜，果波羅蜜因緣知人波羅蜜。**
　　這就稱為比丘以根波羅蜜（根基的圓滿）作為因緣而生，進而知曉果波羅蜜（果位的圓滿），並以果波羅蜜作為因緣而生，進而知曉人波羅蜜（對個體能力的圓滿認知）。

**◈ 如是滿足者作滿足事，減少者作減少事，彼諸根則不空無果，**
　　如此，對於根基圓滿的人，就給予圓滿的教導；對於根基不足的人，就給予相應的教導，這樣一來，那些根基就不會白費而沒有結果。

**◈ 若無此諸根者，我說彼為作凡夫數。」**
　　如果完全沒有這些根基的人，我說他們被歸類為凡夫。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **五根 (Indriya)**：指修行者內在的五種能力，分別為信（對正法的信任）、精進（努力修行）、念（正念覺知）、定（心不散亂）、慧（洞察真理的智慧）。
2. **斯陀含 (Sakadagami)**：聖果中的第二階段，稱為「一次回者」，指已斷除貪、嗔、癡中的部分染污，最多僅回人間一次即證涅槃。
3. **家家 (Anagami)**：此處指「不還者」，即阿那含，指已斷除欲界之貪欲與瞋心，不再回到欲界（家）而是在色界或無色界證果。
4. **波羅蜜 (Pāramitā)**：原意為「到達彼岸」，指為了達到覺悟與解脫而實踐的圓滿修行方法。
5. **根波羅蜜 ➔ 果波羅蜜 ➔ 人波羅蜜**：這是一種教學邏輯。首先觀察對方的根基（根波羅蜜），進而預判其能達到的成就（果波羅蜜），最後對該個體的特質做出精準的指導（人波羅蜜）。

### [現代實作指南]
本經文揭示了「因材施教」與「接納差異」的核心邏輯，可用於現代生活中的自我成長與人際溝通：

1. **停止橫向比較，建立縱向成長**：
   - 經文提到不同根基的人會得到不同的果位（如有的得阿羅漢，有的得信行）。在生活中，我們常因他人進步快而感到挫折。實作時應意識到每個人目前的「五根」狀態不同，將目標設定在「比昨天的自己更進步」，而非與他人競爭。

2. **識別自身的「根基」強項**：
   - 檢視自己在五根中的分佈：我是屬於「信」較強（容易相信並投入）？還是「慧」較強（傾向邏輯分析）？
   - **轉念技巧**：若發現自己「定」力不足，不要強求立即進入深定，而可從「念」（覺知當下）開始，採取「減少者作減少事」的漸進策略。

3. **實踐「人波羅蜜」的溝通法**：
   - 在指導他人或與家人溝通時，先觀察對方的接收能力（根波羅蜜），而非強加自己的標準。
   - **操作步驟**：
     - 觀察 → 判斷對方目前的理解層級 → 提供相應的資訊量 → 確保對方能獲得「果」（理解或改變）。

### [段落大意]
本段經文說明佛陀如何根據比丘不同的根基（五根）程度，給予相應的教導，使每個人都能根據其能力獲得不同層次的解脫或證悟。佛陀強調，透過對根基、果位與個體特質的精準掌握，能確保修行不落空，而完全缺乏根基者則被視為凡夫。
