# 雜阿含經 第 556 經

### [原文]
如是我聞：

一時，佛住娑祇城安禪林中。

爾時眾多比丘尼詣佛所，稽首禮足，退住一面。

爾時，世尊為眾多比丘尼種種說法，示、教、照、喜；示、教、照、喜已，默然住。

時，諸比丘尼白佛言：「世尊！若無相心三昧，不涌、不沒，解脫已住，住已解脫，此無相心三昧，世尊說是何果、何功德？」

佛告諸比丘尼：「若無相心三昧，不涌、不沒，解脫已住，住已解脫，此無相心三昧智果、智功德。」

時，諸比丘尼聞世尊所說，歡喜隨喜，作禮而去。

時，眾多比丘尼往詣尊者阿難所，稽首禮足，退坐一面，白尊者阿難：「若無相心三昧，不涌、不沒，解脫已住，住已解脫，此三昧說，是何果、何功德？」

尊者阿難語諸比丘尼：「姊妹！若無相心三昧，不涌、不沒、解脫已住，住已解脫，世尊說是智果、智功德。」

諸比丘尼言：「奇哉！尊者阿難！大師及弟子同句、同味、同義，所謂第一句義。今諸比丘尼詣世尊所，以如是句、如是味、如是義問世尊，世尊亦已如是句、如是味、如是義為我等說，如尊者阿難所說不異，是故奇特，大師及弟子同句、同味、同義。」

時，諸比丘尼聞尊者阿難所說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住娑祇城安禪林中。**
　　在某一個時間，佛陀住在娑祇城的安禪林中。

**◈ 爾時眾多比丘尼詣佛所，稽首禮足，退住一面。**
　　那時，許多比丘尼（女性出家修行者）來到佛陀這裡，恭敬地頂禮佛陀的足部，然後退到一邊站立。

**◈ 爾時，世尊為眾多比丘尼種種說法，示、教、照、喜；示、教、照、喜已，默然住。**
　　那時，世尊為許多比丘尼講解各種法（真理），透過示現、教導、照破迷妄與給予鼓勵；在完成這些示現、教導、照破與鼓勵後，世尊便進入了沈默的狀態。

**◈ 時，諸比丘尼白佛言：「世尊！若無相心三昧，不涌、不沒，解脫已住，住已解脫，此無相心三昧，世尊說是何果、何功德？」**
　　這時，比丘尼們對佛陀恭敬地說道：「世尊！如果進入一種無相心三昧（一種不執著於任何相狀、標籤的定境），處於不波動、不沉淪的穩定狀態，在解脫（從煩惱中釋放）後安住，在安住後再次確認解脫，請問世尊，這種無相心三昧所產生的果報是什麼？具有什麼樣的功德？」

**◈ 佛告諸比丘尼：「若無相心三昧，不涌、不沒，解脫已住，住已解脫，此無相心三昧智果、智功德。」**
　　佛陀告訴比丘尼們：「如果進入無相心三昧（不執著於任何相狀的定境），處於不波動、不沉淪的狀態，在解脫後安住，在安住後再次確認解脫，這種無相心三昧本身就是智慧的果報，也是智慧的功德。」

**◈ 時，諸比丘尼聞世尊所說，歡喜隨喜，作禮而去。**
　　這時，比丘尼們聽聞世尊所說的教法，內心感到法喜並隨喜而生，向佛陀行禮後才離開。

**◈ 時，眾多比丘尼往詣尊者阿難所，稽首禮足，退坐一面，白尊者阿難：「若無相心三昧，不涌、不沒，解脫已住，住已解脫，此三昧說，是何果、何功德？」**
　　接著，許多比丘尼來到尊者阿難這裡，恭敬地頂禮他的足部，退到一邊坐下，對尊者阿難說：「關於無相心三昧（不執著於任何相狀的定境），處於不波動、不沉淪的狀態，在解脫後安住，在安住後再次確認解脫，請問這種三昧（定境）的果報與功德是什麼？」

**◈ 尊者阿難語諸比丘尼：「姊妹！若無相心三昧，不涌、不沒、解脫已住，住已解脫，世尊說是智果、智功德。」**
　　尊者阿難告訴比丘尼們：「姊妹們！關於無相心三昧（不執著於任何相狀的定境），處於不波動、不沉淪的狀態，在解脫後安住，在安住後再次確認解脫，世尊說這就是智慧的果報與智慧的功德。」

**◈ 諸比丘尼言：「奇哉！尊者阿難！大師及弟子同句、同味、同義，所謂第一句義。今諸比丘尼詣世尊所，以如是句、如是味、如是義問世尊，世尊亦已如是句、如是味、如是義為我等說，如尊者阿難所說不異，是故奇特，大師及弟子同句、同味、同義。」**
　　比丘尼們說：「太奇妙了！尊者阿難！大師（佛陀）與弟子在用詞、體悟與義理上完全一致，這就是所謂的『第一句義』（最深奧的終極真理）。我們剛才去請問世尊，用同樣的措辭、同樣的體悟與義理去詢問，世尊也用同樣的措辭、體悟與義理為我們解答，這與尊者阿難所說的完全沒有差異。因此這真是太奇特了，大師與弟子在措辭、體悟與義理上竟然如此一致。」

**◈ 時，諸比丘尼聞尊者阿難所說，歡喜隨喜，作禮而去。**
　　這時，比丘尼們聽聞尊者阿難所說的教法，內心感到法喜並隨喜而生，向尊者行禮後才離開。

### [詞語註解（解釋重要佛教概念）]
1. **無相心三昧 (Animitta-samādhi)**：一種高度專注的定境，其核心在於不再將現象視為具有固定、獨立的「相」（標籤或特徵）。當修行者不再對對象產生「這是什麼」的認知執著時，心便進入無相狀態，從而消除貪愛與厭離。
2. **不涌、不沒**：比喻心靈的極度穩定。 「不涌」是指心不被激烈的情緒或妄想所驅使而波動；「不沒」是指心不陷入昏沈、遲鈍或絕望的低迷狀態。
3. **解脫已住，住已解脫**：描述一種動靜合一的狀態。解脫後能安住在寧靜中，而這種安住本身就是持續的解脫，不存在「獲得解脫」與「維持解脫」之間的斷層。
4. **第一句義**：指佛法中最核心、最直接的終極真理，不需透過比喻或漸進的推論，而是直接體悟到萬法空性或涅槃的實相。

### [現代實作指南]
本經文強調的是「去標籤化」的認知訓練，可用於緩解現代人的精神壓力與情緒內耗：

1. **覺察「相」的產生**：
　　當你感到憤怒或焦慮時，意識到你的心正在為這個經驗貼標籤（例如：「這是一個糟糕的狀況」、「這個人很討厭」）。這個「標籤」就是「相」。

2. **實踐「無相」轉念**：
　　嘗試將標籤剝離。將「我現在很痛苦」轉化為「現在有一種緊繃的感受在發生」。不定義它是「痛苦」或「糟糕」，僅僅將其視為一種生理或心理的現象。當你不再定義它，你就不會被這個定義所牽引。

3. **維持「不涌不沒」的平衡**：
　　在面對壓力時，練習不讓情緒像波浪一樣「涌」起來（過度反應），也不讓自己陷入沮喪的「沒」之中（自我否定）。在呼吸之間，尋找一個能觀察情緒但又不被情緒捲入的中心點。

4. **確認「解脫與安住」**：
　　當你成功地將標籤剝離，感受到短暫的輕鬆時（解脫），請有意識地在那個輕鬆的狀態中停留幾秒鐘（安住）。透過反覆練習「標籤剝離 $\rightarrow$ 感受輕鬆 $\rightarrow$ 安住其中」，將這種智慧轉化為習慣。

### [段落大意]
比丘尼們向佛陀詢問「無相心三昧」的果報與功德，佛陀開示這種不執著相狀、穩定且解脫的定境，其本身即是智慧的果報與功德。隨後比丘尼們向阿難尊者詢問相同問題，得到完全一致的回答。比丘尼們因此驚嘆佛陀與弟子在真理體悟上達到高度統一，體現了最深奧的「第一句義」。
