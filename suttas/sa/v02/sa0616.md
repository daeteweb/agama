# 雜阿含經 第 616 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「當取自心相，莫令外散。所以者何？若彼比丘愚癡、不辨、不善，不取自心相而取外相，然後退減，自生障閡。譬如厨士愚癡、不辨，不善巧便調和眾味，奉養尊主，酸醎酢淡，不適其意。不能善取尊主所嗜，酸醎酢淡，眾味之和。不能親侍尊主左右，司其所須，聽其所欲，善取其心，而自用意調和眾味，以奉尊主。若不適其意，尊主不悅，不悅故不蒙爵賞，亦不愛念。愚癡比丘亦復如是，不辨、不善，於身身觀住，不能除斷上煩惱，不能攝取其心，亦復不得內心寂靜，不得勝妙正念正知，亦復不得四種增上心法、現法樂住、本所未得安隱涅槃，是名比丘愚癡、不辨、不善，不能善攝內心之相而取外相，自生障閡。

「若有比丘黠慧才辯，善巧方便，取內心已，然後取於外相，彼於後時終不退減，自生障閡。譬如厨士黠慧聰辯，善巧方便，供養尊主，能調眾味，酸醎酢淡，善取尊主所嗜之相，而和眾味，以應其心，聽其尊主所欲之味，數以奉之，尊主悅已，必得爵祿，愛念倍重。如是黠慧厨士善取尊主之心，比丘亦復如是。身身觀念住，斷上煩惱，善攝其心，內心寂止，正念正知，得四增心法，現法樂住，得所未得安隱涅槃，是名比丘黠慧辯才，善巧方便，取內心相，攝持外相，終無退減，自生障閡，受、心、法觀亦復如是。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「當取自心相，莫令外散。**
　　那時，世尊告訴在場的眾位比丘說：「你們應當觀察並把握自己內心的狀態，不要讓心神向外分散於外界環境。

**◈ 所以者何？若彼比丘愚癡、不辨、不善，不取自心相而取外相，然後退減，自生障閡。**
　　為什麼呢？如果有的比丘愚癡、缺乏辨別力且不善於修行，不先觀察內心的狀態而只追求外在的相狀，隨後修行會退步，並為自己製造障礙。

**◈ 譬如厨士愚癡、不辨，不善巧便調和眾味，奉養尊主，酸醎酢淡，不適其意。**
　　就如同一個愚癡、缺乏辨別力且不擅長烹飪的廚師，不能巧妙地調和各種味道來供養尊主，做出的食物酸、鹹、苦、淡，不符合尊主的口味。

**◈ 不能善取尊主所嗜，酸醎酢淡，眾味之和。**
　　他不能正確掌握尊主喜好的酸、鹹、苦、淡等各種味道的調和比例。

**◈ 不能親侍尊主左右，司其所須，聽其所欲，善取其心，而自用意調和眾味，以奉尊主。**
　　他不能在尊主身邊親自侍奉，照顧其需求，聆聽其願望並掌握其心意，反而只用自己的主觀想法來調味奉獻給尊主。

**◈ 若不適其意，尊主不悅，不悅故不蒙爵賞，亦不愛念。**
　　如果食物不合口味，尊主就不會高興；因為不高興，所以不能得到賞賜，也不會受到寵愛與重視。

**◈ 愚癡比丘亦復如是，不辨、不善，於身身觀住，不能除斷上煩惱，不能攝取其心，亦復不得內心寂靜，不得勝妙正念正知，亦復不得四種增上心法、現法樂住、本所未得安隱涅槃，**
　　愚癡的比丘也是這樣，缺乏辨別力且不善修行，在對色（物質身體）進行持續的觀察與專注時，不能除斷深層的內心染污與執著（煩惱），不能攝受並掌控自己的心，因此無法獲得內心的寂靜，無法獲得卓越的正念（對當下覺知）與正知（清楚地領悟現狀），也無法獲得四種能提升心靈境界的卓越法門（四種增上心法），無法在現法中安樂居住，更無法獲得原本尚未得到的、絕對寧靜的解脫境界（涅槃）。

**◈ 是名比丘愚癡、不辨、不善，不能善攝內心之相而取外相，自生障閡。**
　　這就叫做比丘愚癡、缺乏辨別力且不善修行，不能良好地攝受內心狀態而追求外在相狀，導致自生障礙。

**◈ 「若有比丘黠慧才辯，善巧方便，取內心已，然後取於外相，彼於後時終不退減，自生障閡。**
　　「如果有的比丘聰慧且具備辨別力，懂得運用善巧的修行方法，在先把握內心狀態之後，再觀察外在相狀，他往後將永遠不會退步，也不會為自己製造障礙。

**◈ 譬如厨士黠慧聰辯，善巧方便，供養尊主，能調眾味，酸醎酢淡，善取尊主所嗜之相，而和眾味，以應其心，聽其尊主所欲之味，數以奉之，尊主悅已，必得爵祿，愛念倍重。**
　　就如同一個聰慧且具備辨別力、懂得善巧方法的廚師，在供養尊主時能調和各種味道，精準掌握尊主喜好的酸、鹹、苦、淡，並以此調味來符合尊主的心意，聆聽尊主想要的口味並經常奉上。尊主感到滿意後，必然會得到爵位與俸祿，且受到倍加的寵愛與重視。

**◈ 如是黠慧厨士善取尊主之心，比丘亦復如是。**
　　就像聰慧的廚師能掌握尊主的心意一樣，聰慧的比丘也是如此。

**◈ 身身觀念住，斷上煩惱，善攝其心，內心寂止，正念正知，得四增心法，現法樂住，得所未得安隱涅槃，**
　　他在對色（物質身體）進行持續的觀察與專注時，能斷除深層的內心染污與執著（煩惱），良好地攝受其心，使內心達到寂靜，具備正念（對當下覺知）與正知（清楚地領悟現狀），獲得四種提升心靈的卓越法門，在現法中安樂居住，並獲得原本尚未得到的、絕對寧靜的解脫境界（涅槃）。

**◈ 是名比丘黠慧辯才，善巧方便，取內心相，攝持外相，終無退減，自生障閡，受、心、法觀亦復如是。」**
　　這就叫做比丘聰慧且具備辨別力，懂得善巧方便，先把握內心狀態，再攝持外在相狀，最終不會退步，也不會自生障礙。對於受（感受）、心（意識狀態）、法（心理現象）的觀察也是同樣的道理。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **正念 (Sati)**：對當下所觀察的對象保持清醒的覺知，不使其遺忘或散亂。
2. **正知 (Sampajañña)**：在正念的基礎上，清楚地領悟當前行為的目的、適切性以及心靈的真實狀態。
3. **四種增上心法 (Catu-catur-appamāṇa/Higher Mental States)**：指能將心靈提升至更高境界的修行法門（通常指四無量心：慈、悲、喜、捨，或特定的禪定狀態），使修行者在現法中獲得安樂。
4. **身、受、心、法觀 (Satipaṭṭhāna)**：即「四念處」。分別是對色（物質身體）、受（感受）、心（意識狀態）、法（心理現象/真理）的四種覺知修行法。
5. **內心相 (Citta-lakkhaṇa)**：指心靈在面對對象時所呈現的真實特徵與狀態（如貪、嗔、癡或定、慧）。

### [現代實作指南]
本經的核心在於「先內後外」的覺知順序。在現代生活中，我們經常陷入「取外相」的陷阱，即過度關注外界的評價、環境的變遷或他人的反應，而忽略了自己內心的真實反應。

**實作步驟：**
1. **覺察「外散」狀態** ➔ 當你感到焦慮、憤怒或壓力時，觀察自己是否正專注於「外相」（例如：對方說了什麼、環境有多混亂、結果會如何）。
2. **轉向「取自心相」** ➔ 立即將注意力從外界撤回，詢問自己：「現在我的內心是什麼狀態？」
   - 身體是否有緊繃感？（身觀）
   - 現在的感受是痛苦、快樂還是中性？（受觀）
   - 心中是否升起了嗔恨或恐懼？（心觀）
3. **停止「自用意調和」** ➔ 像愚癡廚師一樣，我們常用自己的「主觀偏見」去解讀外界（例如：他這樣說一定是討厭我）。實作時，請嘗試僅僅「觀察」內心狀態，而不對其進行評判或試圖用主觀意志去強行改變。
4. **建立「內心寂止」** ➔ 當你能單純觀察內心的起伏而不被其牽引時，內心會自然趨向寂靜。在這種寂靜中，你才能產生「正知」，明白當下最適切的應對方式，而非被動地反應。

### [段落大意]
本經以廚師供養尊主的比喻，強調修行中「內觀」優先於「外觀」的重要性。比丘若不先攝受內心狀態而追求外在相狀，如同不察尊主口味而隨意烹飪的廚師，無法獲得解脫。唯有先透過對身、受、心、法的觀察，斷除煩惱並獲得內心寂靜與正念正知，才能在現法中安樂並最終達成涅槃。
