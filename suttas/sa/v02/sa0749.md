# 雜阿含經 第 749 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若無明為前相，故生諸惡不善法。時，隨生無慚、無愧；無慚、無愧生已，隨生邪見；邪見生已，能起邪志、邪語、邪業、邪命、邪方便、邪念、邪定。若起明為前相，生諸善法。時，慚愧隨生，慚愧生已，能生正見，正見生已，起正志、正語、正業、正命、正方便、正念、正定，次第而起。正定起已，聖弟子得正解脫貪欲、瞋恚、愚癡。如是聖弟子得正解脫已，得正知見：『我生已盡，梵行已立，所作已作，自知不受後有。』」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若無明為前相，故生諸惡不善法。**
　　「如果以無明（對真理的迷失與不覺）作為開端，因此會產生各種惡劣且不善的法（心理狀態與行為）。

**◈ 時，隨生無慚、無愧；**
　　此時，隨之而生出無慚（對自己的不羞愧）與無愧（對他人的不羞愧）；

**◈ 無慚、無愧生已，隨生邪見；**
　　當無慚與無愧產生後，隨之產生邪見（錯誤的認知與見解）；

**◈ 邪見生已，能起邪志、邪語、邪業、邪命、邪方便、邪念、邪定。**
　　邪見產生後，能引起錯誤的志向（邪志）、錯誤的言語（邪語）、錯誤的行為（邪業）、錯誤的生活方式（邪命）、錯誤的精進方法（邪方便）、錯誤的覺知（邪念）以及錯誤的專注（邪定）。

**◈ 若起明為前相，生諸善法。**
　　如果以明（對真理的覺悟與智慧）作為開端，則會產生各種善法。

**◈ 時，慚愧隨生，**
　　此時，慚（對自己的羞愧）與愧（對他人的羞愧）隨之而生，

**◈ 慚愧生已，能生正見，**
　　當慚愧產生後，能產生正見（正確的認知與見解），

**◈ 正見生已，起正志、正語、正業、正命、正方便、正念、正定，次第而起。**
　　正見產生後，會次第地引起正確的志向（正志）、正確的言語（正語）、正確的行為（正業）、正確的生活方式（正命）、正確的精進方法（正方便）、正確的覺知（正念）以及正確的專注（正定）。

**◈ 正定起已，聖弟子得正解脫貪欲、瞋恚、愚癡。**
　　當正定產生後，聖弟子能獲得正解脫（從束縛中解脫），擺脫貪欲、瞋恚與愚癡。

**◈ 如是聖弟子得正解脫已，得正知見：**
　　聖弟子如此獲得正解脫後，會獲得正確的知見：

**◈ 『我生已盡，梵行已立，所作已作，自知不受後有。』」**
　　『我的輪迴出生已經結束，清淨的修行（梵行）已經建立，應該做的事已經做完了，自知不再受後世的輪迴。』」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **無明 (Avijjā)**：對四聖諦等真理的無知，是導致輪迴與痛苦的根本原因。
2. **慚愧 (Hiri-Ottappa)**：慚是指對自己內心的羞愧，愧是指對他人或社會規範的羞愧。這兩者被視為「天人之道」，是防止惡行發生的心理防線。
3. **邪見 (Micchā-diṭṭhi)**：錯誤的見解，例如認為業報不存在或否定因果，是導致錯誤行為的根源。
4. **正見 (Sammā-diṭṭhi)**：正確的見解，指對緣起法與四聖諦的正確理解，是八正道的開端。
5. **梵行 (Brahmacariya)**：原指獨身修行，在廣義上指清淨的、聖者的生活方式，即完全遠離染污的修行。
6. **不受後有 (Anupādāna)**：指不再產生執取，因此在肉身死亡後不再進入輪迴的出生過程。

### [現代實作指南]
本經揭示了一個「心理連鎖反應」的機制：**認知 ➔ 情感 ➔ 行為 ➔ 習慣 ➔ 命運**。

1. **覺察「前相」（觸發點）**：
   - 當你發現自己開始產生憤怒、貪婪或嫉妒時，請立刻回溯：我的「前相」是否是「無明」？（例如：我是否誤以為這個東西永遠屬於我？我是否誤以為對方必須按照我的意願行動？）
   - **轉念技巧**：將「為什麼他這樣對我」轉化為「這是一個因緣生滅的過程」，用「明（覺知）」取代「無明」。

2. **建立「心理防線」（慚愧心）**：
   - 在衝動採取惡行（如辱罵、欺騙）之前，練習喚起「慚愧心」。
   - **操作方法**：問自己：「如果我的行為被所有人看見，我會感到羞愧嗎？」或「這個行為符合我想成為的理想人格嗎？」利用這種健康的羞愧感來截斷從「無明」到「邪業」的連鎖反應。

3. **正向連鎖的啟動**：
   - 意識到「正見」是所有正面改變的起點。不要強求直接達到「正定」或「解脫」，而應從「正確的認知」開始。
   - **實作步驟**：正確認知（正見） ➔ 決定採取正確的態度（正志） ➔ 實踐正確的言行（正語、正業） ➔ 形成正向的生活習慣（正命、正方便）。

### [段落大意]
本經闡述了兩種截然不同的心理因果鏈：一種是以「無明」為起點，導致失去慚愧心、產生邪見，進而引發八正道的反面（邪道），最終陷入痛苦；另一種是以「明（智慧）」為起點，激發慚愧心並建立正見，次第導向八正道的實踐，最終消除貪瞋癡，達成不再輪迴的完全解脫。
