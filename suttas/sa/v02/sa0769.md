# 雜阿含經 第 769 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。爾時，尊者阿難晨朝著衣持鉢，入舍衛城乞食。

時，有生聞婆羅門乘白馬車，眾多年少翼從，白馬、白車、白控、白鞭，頭著白帽、白傘蓋，手執白拂，著白衣服、白瓔珞，白香塗身，翼從皆白，出舍衛城，欲至林中教授讀誦，眾人見之咸言：「善乘！善乘！謂婆羅門乘。」

時，尊者阿難見婆羅門眷屬、眾具一切皆白，見已，入城乞食。還精舍，舉衣鉢，洗足已，往詣佛所，稽首禮足，退坐一面，白佛言：「世尊！今日晨朝著衣持鉢，入舍衛城乞食，見生聞婆羅門乘白馬車，眷屬、眾具一切皆白，眾人唱言：『善乘！善乘！謂婆羅門乘。』云何？世尊！於正法、律，為是世人乘？為是婆羅門乘？」

佛告阿難：「是世人乘，非我法、律婆羅門乘也。阿難！我正法、律乘、天乘、婆羅門乘、大乘，能調伏煩惱軍者，諦聽，善思，當為汝說。阿難！何等為正法、律乘、天乘、婆羅門乘、大乘，能調伏煩惱軍者？謂八正道——正見乃至正定。阿難！是名正法、律乘、天乘、梵乘、大乘，能調伏煩惱軍者。」

爾時，世尊即說偈言：

「信戒為法軛，　　慚愧為長縻， 
　　正念善護持，　　以為善御者， 
　　捨三昧為轅，　　智慧精進輪， 
　　無著忍辱鎧，　　安隱如法行， 
　　直進不退還，　　永之無憂處， 
　　智士乘戰車，　　摧伏無智怨。」

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，尊者阿難晨朝著衣持鉢，入舍衛城乞食。**
　　那時，尊者阿難在早晨穿好衣服，拿著缽，進入舍衛城中乞食。

**◈ 時，有生聞婆羅門乘白馬車，眾多年少翼從，白馬、白車、白控、白鞭，頭著白帽、白傘蓋，手執白拂，著白衣服、白瓔珞，白香塗身，翼從皆白，出舍衛城，欲至林中教授讀誦，眾人見之咸言：「善乘！善乘！謂婆羅門乘。」**
　　當時，有一位名叫生聞的婆羅門乘坐著白色的馬車，隨行有許多年輕的隨從。馬是白的、車是白的、馬缰是白的、鞭子也是白的；他頭戴白帽，撐著白傘，手持白拂，穿著白色衣服，佩戴白色瓔珞，身上塗著白香，連隨從們全部都是白色的。他走出舍衛城，準備前往森林中教授讀誦經文。路上的眾人見到這景象，都讚嘆道：「好漂亮的乘車啊！好漂亮的乘車啊！這真是婆羅門的豪華乘車。」

**◈ 時，尊者阿難見婆羅門眷屬、眾具一切皆白，見已，入城乞食。**
　　那時，尊者阿難看到這位婆羅門的眷屬以及所有隨行器具全部都是白色的，看完之後，才進入城中乞食。

**◈ 還精舍，舉衣鉢，洗足已，往詣佛所，稽首禮足，退坐一面，白佛言：**
　　回到精舍後，阿難收好衣鉢，洗淨雙足，隨即前往佛陀所在之處，頂禮佛足，然後退到一邊坐下，對佛陀恭敬地說道：

**◈ 「世尊！今日晨朝著衣持鉢，入舍衛城乞食，見生聞婆羅門乘白馬車，眷屬、眾具一切皆白，眾人唱言：『善乘！善乘！謂婆羅門乘。』**
　　「世尊！我今天早晨穿好衣服拿著缽，進入舍衛城乞食時，看到生聞婆羅門乘坐白馬車，他的眷屬和所有器具全部都是白色的，眾人都在讚嘆：『好漂亮的乘車啊！這真是婆羅門的乘車。』」

**◈ 云何？世尊！於正法、律，為是世人乘？為是婆羅門乘？」**
　　「請問世尊，就正法（正確的真理）與律（戒律）而言，這種豪華的馬車，究竟是世俗之人所追求的乘載工具，還是真正的婆羅門之乘？」

**◈ 佛告阿難：「是世人乘，非我法、律婆羅門乘也。**
　　佛陀告訴阿難：「那是世俗之人追求的乘載工具，並非我所教導的、基於正法與律的真正婆羅門之乘。」

**◈ 阿難！我正法、律乘、天乘、婆羅門乘、大乘，能調伏煩惱軍者，諦聽，善思，當為汝說。**
　　「阿難！關於我所說的正法之乘、律之乘、天之乘、婆羅門之乘以及大乘，也就是那些能夠調伏內心染污與執著（煩惱）這支軍隊的修行路徑，你請仔細聽，好好思考，我現在為你解說。」

**◈ 阿難！何等為正法、律乘、天乘、婆羅門乘、大乘，能調伏煩惱軍者？**
　　「阿難！究竟什麼才是正法之乘、律之乘、天之乘、婆羅門之乘以及大乘，且能調伏內心染污與執著（煩惱）這支軍隊的工具呢？」

**◈ 謂八正道——正見乃至正定。**
　　「那就是八正道（包含正見、正思惟、正語、正業、正命、正精進、正念、正定這八項正確的修行路徑）。」

**◈ 阿難！是名正法、律乘、天乘、梵乘、大乘，能調伏煩惱軍者。」**
　　「阿難！這才叫做正法之乘、律之乘、天之乘、梵之乘以及大乘，是真正能調伏內心染污與執著（煩惱）這支軍隊的工具。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即說了偈頌：

**◈ 「信戒為法軛，慚愧為長縻，**
　　「以對真理的信心與持守戒律作為控制馬匹的軛（橫木），以對過錯的慚愧心作為牽引的長繩，」

**◈ 正念善護持，以為善御者，**
　　「以正念（時刻覺知當下）來善加護持，作為優秀的駕車者，」

**◈ 捨三昧為轅，智慧精進輪，**
　　「以捨三昧（不偏不倚、平等的定心）作為車轅，以智慧與精進作為前進的車輪，」

**◈ 無著忍辱鎧，安隱如法行，**
　　「以無著（不執著）與忍辱作為防禦的鎧甲，如此便能安穩地依照正法而行，」

**◈ 直進不退還，永之無憂處，**
　　「筆直前進不再後退，最終到達永遠沒有憂慮的境界（涅槃），」

**◈ 智士乘戰車，摧伏無智怨。」**
　　「智慧之士乘坐著這輛修行戰車，能摧毀並調伏由無知所引起的仇恨與對立。」

### [詞語註解]
1. **八正道 (Ariya aṭṭhaṅgika magga)**：原始佛教的核心修行路徑，分為三學（戒、定、慧）。正見、正思惟屬「慧」；正語、正業、正命屬「戒」；正精進、正念、正定屬「定」。
2. **煩惱 (Kilesa)**：指染污心靈的心理狀態，如貪、嗔、癡，使人無法看清真理而陷入輪迴。
3. **捨三昧 (Upekkhā-samādhi)**：一種高度平穩的禪定狀態，對所有對象保持中立、不貪不憎的平衡心。
4. **慚愧 (Hiri-Ottappa)**：慚是指對自己的過錯感到羞愧；愧是指害怕他人知道自己的過錯而產生的不安。在佛教中，這被視為防止墮落的「天之法」。
5. **正念 (Sati)**：對當下身心狀態的持續覺知，不評判、不迷失，是調伏心念的關鍵。

### [現代實作指南]
本經文將「修行」比喻為「駕車」，將內心的調伏視為一場戰鬥。我們可以將其轉化為以下現代心理實踐：

1. **建立「心靈軛與繩」（價值觀與底線）**：
   - **實作**：定義你的核心價值（信）與行為底線（戒）。當衝動產生時，用「如果我這樣做，是否違背我的原則」來作為牽引（慚愧），防止心念失控。

2. **培養「心靈御者」（覺察力）**：
   - **實作**：練習「正念標記」。當憤怒或焦慮升起時，在心中輕聲標記「憤怒升起了」、「焦慮升起了」。不要試圖壓抑，而是像駕車者觀察馬匹一樣，客觀觀察情緒的起伏，而不被其牽走。

3. **安裝「心靈鎧甲」（情緒韌性）**：
   - **實作**：面對批評或挫折時，練習「無著」與「忍辱」。告訴自己：「這只是一個現象，並非我本身」。將外界的衝擊視為鍛鍊心靈的重量訓練，而非對自我的攻擊。

4. **驅動「心靈車輪」（行動力與洞察力）**：
   - **實作**：將「智慧」定義為對因果的理解，將「精進」定義為持續的小步改進。每天問自己：「目前的做法是否能讓我更接近平靜？」並採取具體行動。

### [段落大意]
阿難見到一名婆羅門擁有極其奢華的白色馬車而受眾人讚嘆，遂請問佛陀何謂真正的「乘」。佛陀指出，世俗的豪華僅是外在裝飾，真正的「大乘」是能調伏煩惱的「八正道」。佛陀進一步以戰車為喻，說明以信心、戒律、慚愧、正念、捨心、智慧、精進與忍辱作為修行的組成部分，方能摧伏無明，達到永無憂慮的涅槃境界。
