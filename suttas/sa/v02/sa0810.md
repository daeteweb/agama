# 雜阿含經 第 810 經

### [原文]
如是我聞：

一時，佛住金剛跋求摩河側薩羅梨林中。

爾時，尊者阿難獨一靜處，思惟禪思，作如是念：「頗有一法，修習多修習，令四法滿足；四法滿足已，七法滿足；七法滿足已，二法滿足？」

時，尊者阿難從禪覺已，往詣佛所，稽首禮足，退坐一面，白佛言：「世尊！我獨一靜處，思惟禪思，作是念：『頗有一法，多修習已，令四法滿足，乃至二法滿足？我今問世尊，寧有一法，多修習已，能令，乃至二法滿足耶？』」

佛告阿難：「有一法，多修習已，乃至能令二法滿足。何等為一法？謂安那般那念。多修習已，能令四念處滿足；四念處滿足已，七覺分滿足；七覺分滿足已，明、解脫滿足。云何修安那般那念，四念處滿足？是比丘依止聚落，乃至如滅出息念學。阿難！如是聖弟子入息念時如入息念學，出息念時如出息念學；若長若短，一切身行覺知，入息念時如入息念學，出息念時如出息念學；身行休息入息念時，如身行休息入息念學；身行休息出息念時，如身行休息出息念學。聖弟子爾時身身觀念住，異於身者，彼亦如是隨身比思惟。

「若有時聖弟子喜覺知，樂覺知，心行覺知，心行息覺知，入息念時如心行息入息念學，心行息出息念時如心行息出息念學。是聖弟子爾時受受觀念住，若復異受者，彼亦受隨身比思惟。有時聖弟子心覺知，心悅、心定、心解脫覺知，入息念時如入息念學，心解脫出息念時如心解脫出息念學，是聖弟子爾時心心觀念住，若有異心者，彼亦隨心比思惟。

「若聖弟子有時觀無常、斷、無欲、滅，如無常、斷、無欲、滅觀住學，是聖弟子爾時法法觀念住異於法者，亦隨法比思惟，是名修安那般那念，滿足四念處。」

阿難白佛：「如是修習安那般那念，令四念處滿足。云何修四念處，令七覺分滿足？」

佛告阿難：「若比丘身身觀念住，念住已，繫念住不忘。爾時方便修念覺分，修念覺分已，念覺分滿足；念覺滿足已，於法選擇思量。爾時方便修擇法覺分，修擇法覺分已，擇法覺分滿足，於法選擇分別思量已，得精勤方便。爾時方便修習精進覺分，修精進覺分已，精進覺分滿足，方便精進已，則心歡喜。爾時方便修喜覺分，修喜覺分已，喜覺分滿足，歡喜已，身心猗息。爾時方便修猗覺分，修猗覺分已，猗覺分滿足，身心樂已，得三昧。爾時修定覺分，修定覺分已，定覺分滿足，定覺分滿足已，貪憂則滅，得平等捨。爾時方便修捨覺分，修捨覺分已，捨覺分滿足，受、心、法法念處亦如是說，是名修四念處，滿足七覺分。」

阿難白佛：「是名修四念處，滿足七覺分。云何修七覺分，滿足明、解脫？」

佛告阿難：「若比丘修念覺分，依遠離、依無欲、依滅、向於捨；修念覺分已，滿足明、解脫。乃至修捨覺分，依遠離、依無欲、依滅、向於捨，如是修捨覺分已，明、解脫滿足。阿難！是名法法相類、法法相潤。如是十三法，一法為增上，一法為門，次第增進，修習滿足。」

佛說此經已，尊者阿難聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住金剛跋求摩河側薩羅梨林中。**
　　在某一個時間，佛陀住在金剛跋求摩河側的薩羅梨林中。

**◈ 爾時，尊者阿難獨一靜處，思惟禪思，作如是念：「頗有一法，修習多修習，令四法滿足；四法滿足已，七法滿足；七法滿足已，二法滿足？」**
　　那時，尊者阿難獨自待在安靜的地方，在禪思中思考，心裡想著：「是否有一種修行方法，只要勤加修習，就能讓四種正念（四念處）圓滿；四種正念圓滿後，能讓七種覺悟要素（七覺分）圓滿；而七種覺悟要素圓滿後，能讓兩種最終境界（明與解脫）圓滿？」

**◈ 時，尊者阿難從禪覺已，往詣佛所，稽首禮足，退坐一面，白佛言：「世尊！我獨一靜處，思惟禪思，作是念：『頗有一法，多修習已，令四法滿足，乃至二法滿足？我今問世尊，寧有一法，多修習已，能令，乃至二法滿足耶？』」**
　　隨後，尊者阿難從禪定中醒來，走到佛陀面前，恭敬地低頭頂禮觸足，然後退到一邊坐下，對佛陀恭敬地說道：「世尊！我獨自在靜處中禪思，心裡在想：『是否有一種法，只要多加修習，就能讓四念處圓滿，乃至讓明與解脫圓滿？』我現在請問世尊，是否真的有一種法，能透過多加修習，達成從四念處到二法圓滿的過程呢？」

**◈ 佛告阿難：「有一法，多修習已，乃至能令二法滿足。何等為一法？謂安那般那念。多修習已，能令四念處滿足；四念處滿足已，七覺分滿足；七覺分滿足已，明、解脫滿足。」**
　　佛陀告訴阿難：「確實有一種法，只要多加修習，就能讓最終的二法圓滿。這唯一的方法是什麼呢？就是安那般那念（觀察呼吸的修行）。勤加修習呼吸觀察，能讓四念處（對身、受、心、法的正念觀察）圓滿；四念處圓滿後，能讓七覺分（覺悟的七個階段）圓滿；而七覺分圓滿後，則能讓明（對真理的洞察）與解脫（從束縛中釋放）圓滿。」

**◈ 云何修安那般那念，四念處滿足？是比丘依止聚落，乃至如滅出息念學。**
　　如何透過修習安那般那念（呼吸觀察）來讓四念處圓滿呢？比丘住在村落中，學習觀察呼吸的進入與呼出直到其平息。

**◈ 阿難！如是聖弟子入息念時如入息念學，出息念時如出息念學；若長若短，一切身行覺知，入息念時如入息念學，出息念時如出息念學；身行休息入息念時，如身行休息入息念學；身行休息出息念時，如身行休息出息念學。聖弟子爾時身身觀念住，異於身者，彼亦如是隨身比思惟。**
　　阿難！這樣的聖弟子在吸氣時，就專注於吸氣的覺知；在呼氣時，就專注於呼氣的覺知。無論呼吸是長或短，都能覺知到所有色（物質身體）的運作，吸氣時如實觀察吸氣，呼氣時如實觀察呼氣。當身體的運作趨於平靜時，在吸氣與呼氣中同樣如實觀察。聖弟子在此時便處於對身（物質身體）的正念觀察之中，若發現身體有任何變化，也會隨之觀察身體的狀態。

**◈ 「若有時聖弟子喜覺知，樂覺知，心行覺知，心行息覺知，入息念時如心行息入息念學，心行息出息念時如心行息出息念學。是聖弟子爾時受受觀念住，若復異受者，彼亦受隨身比思惟。」**
　　「如果聖弟子有時覺知到喜悅、覺知到快樂、覺知到心的造作（行）以及心造作的平息，在吸氣與呼氣時，都能如實觀察這些心的造作與平息。聖弟子在此時便處於對受（感受）的正念觀察之中，若出現不同的感受，也會隨之觀察這些感受。」

**◈ 「有時聖弟子心覺知，心悅、心定、心解脫覺知，入息念時如入息念學，心解脫出息念時如心解脫出息念學，是聖弟子爾時心心觀念住，若有異心者，彼亦隨心比思惟。」**
　　「有時聖弟子覺知到識（意識）的狀態，覺知到心的歡喜、心的安定以及心的解脫，在吸氣與呼氣時，都能如實觀察這些意識的狀態。聖弟子在此時便處於對心（意識）的正念觀察之中，若意識狀態有所改變，也會隨之觀察心的變化。」

**◈ 「若聖弟子有時觀無常、斷、無欲、滅，如無常、斷、無欲、滅觀住學，是聖弟子爾時法法觀念住異於法者，亦隨法比思惟，是名修安那般那念，滿足四念處。」**
　　「如果聖弟子有時觀察到無常（一切現象在變）、斷（執著的斷除）、無欲（貪愛的消失）與滅（苦的止息），並依照這些真理來觀察與學習，聖弟子在此時便處於對法（心理現象與真理）的正念觀察之中，若發現不同的法，也會隨之觀察。這就叫做透過修習安那般那念（呼吸觀察），使四念處圓滿。」

**◈ 阿難白佛：「如是修習安那般那念，令四念處滿足。云何修四念處，令七覺分滿足？」**
　　阿難對佛陀說：「這樣修習呼吸觀察可以讓四念處圓滿。那麼，如何透過修習四念處，讓七覺分（覺悟的七要素）圓滿呢？」

**◈ 佛告阿難：「若比丘身身觀念住，念住已，繫念住不忘。爾時方便修念覺分，修念覺分已，念覺分滿足；**
　　佛陀告訴阿難：「如果比丘處於對身（物質身體）的正念觀察中，且能維持這種正念而不忘失，此時便能方便地修習念覺分（正念要素），修習之後，念覺分便圓滿了；

**◈ 念覺滿足已，於法選擇思量。爾時方便修擇法覺分，修擇法覺分已，擇法覺分滿足，**
　　當念覺分圓滿後，會開始對法（現象）進行選擇與思量。此時便能方便地修習擇法覺分（審視真理的要素），修習之後，擇法覺分便圓滿了；

**◈ 於法選擇分別思量已，得精勤方便。爾時方便修習精進覺分，修精進覺分已，精進覺分滿足，**
　　在對法進行分別思量後，會獲得精進的動力。此時便能方便地修習精進覺分（努力的要素），修習之後，精進覺分便圓滿了；

**◈ 方便精進已，則心歡喜。爾時方便修喜覺分，修喜覺分已，喜覺分滿足，**
　　在精進修行之後，內心會感到歡喜。此時便能方便地修習喜覺分（法喜的要素），修習之後，喜覺分便圓滿了；

**◈ 歡喜已，身心猗息。爾時方便修猗覺分，修猗覺分已，猗覺分滿足，**
　　在歡喜之後，身心會感到安適與寧靜。此時便能方便地修習猗覺分（輕安的要素），修習之後，猗覺分便圓滿了；

**◈ 身心樂已，得三昧。爾時修定覺分，修定覺分已，定覺分滿足，**
　　在身心安樂之後，能進入三昧（高度專注的定境）。此時修習定覺分（專注的要素），修習之後，定覺分便圓滿了；

**◈ 定覺分滿足已，貪憂則滅，得平等捨。爾時方便修捨覺分，修捨覺分已，捨覺分滿足，**
　　當定覺分圓滿後，貪愛與憂慮會消失，獲得平等的捨心。此時便能方便地修習捨覺分（平等心要素），修習之後，捨覺分便圓滿了。

**◈ 受、心、法法念處亦如是說，是名修四念處，滿足七覺分。」**
　　對於受（感受）、心（意識）、法（真理）的念處，其運作過程也同樣如此。這就叫做透過修習四念處，使七覺分圓滿。」

**◈ 阿難白佛：「是名修四念處，滿足七覺分。云何修七覺分，滿足明、解脫？」**
　　阿難對佛陀說：「這就是修習四念處以圓滿七覺分。那麼，如何透過修習七覺分，讓明（真理覺知）與解脫圓滿呢？」

**◈ 佛告阿難：「若比丘修念覺分，依遠離、依無欲、依滅、向於捨；修念覺分已，滿足明、解脫。**
　　佛陀告訴阿難：「如果比丘在修習念覺分時，依止於遠離（脫離煩惱）、依止於無欲（消除貪愛）、依止於滅（苦的止息），並趨向於平等捨心；修習念覺分之後，就能讓明（真理覺知）與解脫圓滿。

**◈ 乃至修捨覺分，依遠離、依無欲、依滅、向於捨，如是修捨覺分已，明、解脫滿足。**
　　乃至於修習捨覺分時，同樣依止於遠離、無欲、滅，並趨向於平等捨心；如此修習捨覺分之後，明與解脫便圓滿了。

**◈ 阿難！是名法法相類、法法相潤。如是十三法，一法為增上，一法為門，次第增進，修習滿足。」**
　　阿難！這就叫做法與法相互類比、法與法相互滋潤。這十三種法（1個呼吸念 ➔ 4個念處 ➔ 7個覺分 ➔ 2個最終果位），其中一種法作為增上緣（推動力量），一種法作為進入的門戶，次第遞進，修習直到圓滿。」

**◈ 佛說此經已，尊者阿難聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，尊者阿難聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **安那般那念 (Ānāpānasati)**：指對呼吸（入息與出息）的正念觀察。它是進入深層禪定與開發智慧的基礎方法。
2. **四念處 (Satipaṭṭhāna)**：四種正念的確立，包括：身念處（觀察物質身體）、受念處（觀察感受）、心念處（觀察意識狀態）、法念處（觀察心理現象與真理）。
3. **七覺分 (Bojjhaṅga)**：導致覺悟的七種心理要素，依序為：念（正念）、擇法（審視）、精進（努力）、喜（法喜）、猗（輕安）、定（專注）、捨（平等）。
4. **明 (Vijjā)**：對四聖諦與緣起真理的直接洞察，對立面為無明（Avijjā）。
5. **解脫 (Vimokkha)**：心靈從貪、嗔、癡等束縛中完全釋放，達到絕對自由的狀態。
6. **法法相類、法法相潤**：指佛教的修行路徑具有邏輯上的連貫性，前一個階段的成就為後一個階段提供必要的條件與滋養。

### [現代實作指南]
本經文提供了一套極其精準的「由淺入深」心理訓練路徑，可轉化為以下生活實踐步驟：

1. **第一階段：建立錨點（呼吸觀察）**
   - **操作**：每天抽出5-10分鐘，僅僅觀察呼吸。不強求改變呼吸，而是「如實覺知」吸氣時的感覺與呼氣時的感覺。
   - **轉念**：將呼吸視為與當下連接的錨點，每當心神分散時，溫柔地將注意力帶回呼吸。

2. **第二階段：擴展覺知（四念處）**
   - **身體覺知**：在呼吸中，觀察身體的緊繃或放鬆（身念處）。
   - **情緒標記**：當心中升起喜悅或不適時，輕輕標記「這是快樂」或「這是不安」，而不陷入其中（受念處）。
   - **意識監控**：觀察現在的心是散亂的、專注的，還是平靜的（心念處）。
   - **洞察本質**：意識到上述所有感覺與狀態都在不斷變化，沒有一個永恆不變的「我」（法念處 ➔ 無常）。

3. **第三階段：優化心理狀態（七覺分）**
   - **從覺知到選擇**：當你能穩定覺知後，開始思考「什麼是導致我痛苦的因」（擇法）。
   - **從思考到行動**：帶著這種洞察，努力減少負面習慣（精進）。
   - **從努力到放鬆**：在進步中感受喜悅（喜），並讓身心進入一種不勉強的輕盈狀態（猗）。
   - **從放鬆到安定**：在輕安中進入深層的專注（定），最終達到對任何境遇都能平靜面對的狀態（捨）。

4. **最終目標：獲得自由**
   - 當我們不再被情緒牽著走（捨），且能清晰看穿事物的真相（明），自然會感受到一種不再被外界定義或束縛的自由感（解脫）。

### [段落大意]
本經記述尊者阿難詢問佛陀是否有一種修行法能次第達成多重圓滿。佛陀指出「安那般那念（呼吸觀察）」即是此法。透過呼吸觀察可圓滿四念處（身、受、心、法），四念處進而推動七覺分（念、擇法、精進、喜、猗、定、捨）的圓滿，最終導向明與解脫。這展示了佛法修行中「一法為門，次第增進」的邏輯結構。
