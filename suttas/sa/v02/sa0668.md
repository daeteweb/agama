# 雜阿含經 第 668 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘……如上說。差別者：「若最勝施者，謂法施。最勝愛語者，謂善男子樂聞，應時說法。行利最勝者，諸不信者能令入信，建立於信；立戒者以淨戒，慳者以施，惡智者以正智令入建立。同利最勝者，謂阿羅漢以阿羅漢、阿那含以阿那含、斯陀含以斯陀含、須陀洹以須陀洹、淨戒者以淨而授於彼。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘……如上說。**
　　那時，世尊告訴在場的眾位比丘說……（內容如前所述）。

**◈ 差別者：「若最勝施者，謂法施。**
　　不同之處在於：「若說到最殊勝的布施，指的是法施（分享佛法真理的布施）。

**◈ 最勝愛語者，謂善男子樂聞，應時說法。**
　　最殊勝的愛語，是指對樂於聽聞佛法的善男子（修學佛法的男子），在適當的時機為其說法。

**◈ 行利最勝者，諸不信者能令入信，建立於信；**
　　行利（利益他人）最殊勝的做法，是能讓那些原本不相信的人產生信心，並使他們在信心上建立起來；

**◈ 立戒者以淨戒，慳者以施，惡智者以正智令入建立。**
　　對於需要建立戒律的人，以清淨的戒律來引導；對於吝嗇的人，以布施來感化；對於智慧偏差的人，以正智（正確的智慧）來令其建立正確的認知。

**◈ 同利最勝者，謂阿羅漢以阿羅漢、阿那含以阿那含、斯陀含以斯陀含、須陀洹以須陀洹、淨戒者以淨而授於彼。」**
　　同利（共同獲益）最殊勝的做法，是指阿羅漢（已解脫煩惱者）指導阿羅漢，阿那含（不還果者）指導阿那含，斯陀含（一來果者）指導斯陀含，須陀洹（入流果者）指導須陀洹，以及清淨戒律的人以清淨的戒律來指導對方。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **法施 (Dhamma-dāna)**：指分享佛法、真理或正確的見解給他人，被認為比物質布施更殊勝，因為其能從根本上解決苦楚。
2. **應時說法 (Kāla-desanā)**：指根據對方的根機（理解能力）、心理狀態以及時機的成熟度，選擇最合適的內容與方式進行教導。
3. **須陀洹 (Sotāpanna)**：初果聖者，意為「入流」，指已進入聖道之流，確定在七次輪迴之內必能達到涅槃。
4. **斯陀含 (Sakadāgāmī)**：二果聖者，意為「一次來」，指煩惱已大幅減少，僅需再回到人間一次即可解脫。
5. **阿那含 (Anāgāmī)**：三果聖者，意為「不還」，指已斷除貪欲與瞋心，死後不再回到欲界。
6. **阿羅漢 (Arahant)**：四果聖者，指已完全斷除所有煩惱，證得解脫，不再輪迴。

### [現代實作指南]
本經文的核心在於「對症下藥」的利他精神，可轉化為以下生活實踐：

1. **精準利他（對應「行利最勝」）**：
   - **觀察需求** ➔ 在幫助他人前，先觀察對方的「匱乏點」。
   - **針對性行動** ➔ 面對憤怒或吝嗇的人，不要用道理說服，而要用寬容與慷慨（布施）來感化；面對迷茫的人，則提供邏輯清晰的正確資訊（正智）。
   - **轉念技巧** ➔ 將「我想幫他」轉化為「他現在需要什麼樣的幫助」。

2. **適時溝通（對應「應時說法」）**：
   - **時機優先** ➔ 避免在對方情緒激動或心不在焉時傳達重要訊息。
   - **共情聆聽** ➔ 先確認對方是否「樂聞」（願意聽），在對方產生好奇或需求時再切入核心議題。

3. **同儕支持（對應「同利最勝」）**：
   - **尋找同頻者** ➔ 在學習或修行中，尋找處於相同階段或具有相似經驗的人互相勉勵。
   - **經驗共享** ➔ 意識到同等級別的經驗分享（如：同樣經歷過挫折的人互相支持）往往比高對低的指令式指導更具療癒力。

### 段落大意 
本段經文闡述了最殊勝的利他方式：最殊勝的布施是分享真理（法施）；最殊勝的愛語是適時地為樂於聽聞者說法；最殊勝的利益他人則是根據對方的缺失（如缺乏信心、戒律、慷慨或智慧）提供對應的引導；而最殊勝的共同獲益則是相同修行境界的人彼此指導。
