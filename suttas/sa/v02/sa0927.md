# 雜阿含經 第 927 經

### [原文]
如是我聞：

一時，佛住迦毘羅衛國尼拘律園中。

時，有釋種名摩訶男，來詣佛所，稽首佛足，退坐一面，白佛言：「世尊！云何名為優婆塞？」

佛告摩訶男：「在家清白修習淨住，男相成就，作是說言：『我今盡壽歸佛、歸法、歸比丘僧，為優婆塞，證知我！』是名優婆塞。」

摩訶男白佛言：「世尊！云何名為優婆塞信具足？」

佛告摩訶男：「優婆塞者，於如來所正信為本，堅固難動，諸沙門、婆羅門、諸天、魔、梵，及餘世間所不能壞。摩訶男！是名優婆塞信具足。」

摩訶男白佛言：「世尊！云何名優婆塞戒具足？」

佛告摩訶男：「優婆塞離殺生、不與取、邪婬、妄語、飲酒，不樂作。摩訶男！是名優婆塞戒具足。」

摩訶男白佛言：「世尊！云何名優婆塞聞具足？」

佛告摩訶男：「優婆塞聞具足者，聞則能持，聞則積集，若佛所說初、中、後善，善義善味，純一滿淨，梵行清白，悉能受持。摩訶男！是名優婆塞聞具足。」

摩訶男白佛言：「世尊！云何名優婆塞捨具足？」

佛告摩訶男：「優婆塞捨具足者，為慳垢所纏者，心離慳垢，住於非家，修解脫施、勤施、常施，樂捨財物，平等布施。摩訶男！是名優婆塞捨具足。」

摩訶男白佛言：「世尊！云何名優婆塞智慧具足？」

佛告摩訶男：「優婆塞智慧具足者，謂此苦如實知，此苦集如實知，此苦滅如實知，此苦滅道跡如實知。摩訶男！是名優婆塞慧具足。」

爾時，釋氏摩訶男聞佛所說，歡喜隨喜，從坐起，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住迦毘羅衛國尼拘律園中。**
　　在某一個時間，佛陀住在迦毘羅衛國的尼拘律園中。

**◈ 時，有釋種名摩訶男，來詣佛所，稽首佛足，退坐一面，白佛言：「世尊！云何名為優婆塞？」**
　　那時，有一位釋迦族的人名叫摩訶男，來到佛陀這裡，恭敬地向佛陀的足下頂禮，然後退到一邊坐下，對佛陀恭敬地說道：「世尊！什麼樣的人才稱得上是優婆塞（在家的隨從弟子）呢？」

**◈ 佛告摩訶男：「在家清白修習淨住，男相成就，作是說言：『我今盡壽歸佛、歸法、歸比丘僧，為優婆塞，證知我！』是名優婆塞。」**
　　佛陀告訴摩訶男：「在家庭生活中保持清白，修習純淨的住持（生活方式），且具備端正的特質，並如此宣告：『我從今直到生命結束，歸依佛、歸依法、歸依比丘僧，成為一名優婆塞，請各位為我作證！』這樣的人就稱為優婆塞。」

**◈ 摩訶男白佛言：「世尊！云何名為優婆塞信具足？」**
　　摩訶男對佛陀恭敬地說道：「世尊！什麼樣的優婆塞才稱得上是『信（對真理的堅定信心）』具足呢？」

**◈ 佛告摩訶男：「優婆塞者，於如來所正信為本，堅固難動，諸沙門、婆羅門、諸天、魔、梵，及餘世間所不能壞。摩訶男！是名優婆塞信具足。」**
　　佛陀告訴摩訶男：「優婆塞若能以對如來（成就不變真理而來到世間的人）的正信為根本，其信念堅固且難以動搖，無論是沙門、婆羅門，或是諸天、魔王、梵天，以及世間其他任何力量都無法摧毀。摩訶男！這就稱為優婆塞的信具足。」

**◈ 摩訶男白佛言：「世尊！云何名優婆塞戒具足？」**
　　摩訶男對佛陀恭敬地說道：「世尊！什麼樣的優婆塞才稱得上是『戒（道德自律）』具足呢？」

**◈ 佛告摩訶男：「優婆塞離殺生、不與取、邪婬、妄語、飲酒，不樂作。摩訶男！是名優婆塞戒具足。」**
　　佛陀告訴摩訶男：「優婆塞若能遠離殺生、不偷盜（不與取）、不從邪淫、不說妄語、不飲酒，且不樂於從事這些行為。摩訶男！這就稱為優婆塞的戒具足。」

**◈ 摩訶男白佛言：「世尊！云何名優婆塞聞具足？」**
　　摩訶男對佛陀恭敬地說道：「世尊！什麼樣的優婆塞才稱得上是『聞（對教法的學習與記憶）』具足呢？」

**◈ 佛告摩訶男：「優婆塞聞具足者，聞則能持，聞則積集，若佛所說初、中、後善，善義善味，純一滿淨，梵行清白，悉能受持。摩訶男！是名優婆塞聞具足。」**
　　佛陀告訴摩訶男：「優婆塞的聞具足，是指聽聞教法後能持守，並能積累學習。無論是佛陀所說的初、中、後階段的善法，以及其中精妙的義理與深遠的滋味，只要是純淨無雜、符合梵行（清淨修行）的教導，都能悉數受持。摩訶男！這就稱為優婆塞的聞具足。」

**◈ 摩訶男白佛言：「世尊！云何名優婆塞捨具足？」**
　　摩訶男對佛陀恭敬地說道：「世尊！什麼樣的優婆塞才稱得上是『捨（對財物的捨離）』具足呢？」

**◈ 佛告摩訶男：「優婆塞捨具足者，為慳垢所纏者，心離慳垢，住於非家，修解脫施、勤施、常施，樂捨財物，平等布施。摩訶男！是名優婆塞捨具足。」**
　　佛陀告訴摩訶男：「優婆塞的捨具足，是指原本被吝嗇（慳垢）所束縛的人，能讓心離於吝嗇，即便身處世俗家庭（住於非家），仍能實踐解脫的布施、勤勉的布施與恆常的布施，樂於捨棄財物，且平等地布施給眾生。摩訶男！這就稱為優婆塞的捨具足。」

**◈ 摩訶男白佛言：「世尊！云何名優婆塞智慧具足？」**
　　摩訶男對佛陀恭敬地說道：「世尊！什麼樣的優婆塞才稱得上是『智慧（對真理的洞察）』具足呢？」

**◈ 佛告摩訶男：「優婆塞智慧具足者，謂此苦如實知，此苦集如實知，此苦滅如實知，此苦滅道跡如實知。摩訶男！是名優婆塞慧具足。」**
　　佛陀告訴摩訶男：「優婆塞的智慧具足，是指能如實地知道什麼是苦（Dukkha）、如實地知道苦的集（Samudaya/苦因）、如實地知道苦的滅（Nirodha/苦止），以及如實地知道苦滅的道跡（Magga/解脫之道）。摩訶男！這就稱為優婆塞的慧具足。」

**◈ 爾時，釋氏摩訶男聞佛所說，歡喜隨喜，從坐起，作禮而去。**
　　那時，釋迦族的摩訶男聽完佛陀的教導後，心生法喜並依照教導去實踐，他從座位上站起來，向佛陀行禮後才離開。

### [詞語註解]
1. **優婆塞 (Upāsaka)**：原意為「親近者」，指在家庭生活中修行、皈依三寶的男性在家弟子。
2. **正信 (Saddhā)**：並非盲信，而是基於對真理的觀察與體驗後，產生的堅定信心與信任。
3. **五戒 (Pañca-sīla)**：文中提到的離殺、盜、淫、妄、酒，是在家修行者最基本的道德底線，旨在減少對他人的傷害並穩定內心。
4. **慳垢 (Macchariya)**：指對財產、名聲或知識的吝嗇與執著，是阻礙慈悲心與捨離心的主要心理障礙。
5. **四聖諦 (Cattāri Ariyasaccāni)**：
    - **苦 (Dukkha)**：體認生命本質中的不安與不滿足。
    - **集 (Samudaya)**：分析導致苦的根源（如貪愛與無明）。
    - **滅 (Nirodha)**：體悟苦的止息狀態（涅槃）。
    - **道 (Magga)**：實踐導致苦止的具體路徑（八正道）。

### [現代實作指南]
本經文為在家修行者提供了一套完整的「成長指標」，可將其轉化為現代生活中的五個實踐維度：

1. **建立核心價值（信具足）** ➔ 
   在資訊碎片化的時代，尋找一個經過驗證的真理體系（如佛法），並在面對外界壓力或價值觀衝擊時，保持內心的定力，不隨波逐流。
2. **設定行為底線（戒具足）** ➔ 
   將五戒轉化為「不傷害原則」。在職場與人際關係中，練習不欺騙、不剽竊、不傷害他人，透過減少負面行為來降低心理內疚感，獲得內在的平靜。
3. **建立持續學習系統（聞具足）** ➔ 
   不滿足於碎片化的金句，而是系統性地閱讀與研究教法。練習「聞、思、修」：聽聞正確的資訊 ➔ 邏輯分析其義理 ➔ 將其應用於生活實踐。
4. **練習心理鬆綁（捨具足）** ➔ 
   意識到「擁有」並不等於「幸福」。透過定期的捐贈或無私地分享知識與時間，對抗內心的匱乏感與吝嗇心，將「佔有」轉化為「貢獻」。
5. **培養問題分析能力（慧具足）** ➔ 
   面對生活壓力時，不採取逃避或情緒化反應，而是運用四聖諦分析法：
   - 承認目前的壓力（苦）。
   - 找出壓力背後的執著或渴望（集）。
   - 想像壓力消失後的平靜狀態（滅）。
   - 制定具體的行動計畫來解決問題（道）。
