# 雜阿含經 第 874 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有三種子。何等為三？有隨生子、有勝生子、有下生子、何等為隨生子？謂子父母不殺、不盜、不婬、不妄語、不飲酒，子亦隨學不殺、不盜、不婬、不妄語、不飲酒，是名隨生子。何等為勝生子？若子父母不受不殺、不盜、不婬、不妄語、不飲酒戒，子則能受不殺、不盜、不婬、不妄語、不飲酒戒，是名勝生子。云何下生子？若子父母受不殺、不盜、不婬、不妄語、不飲酒戒，子不能受不殺、不盜、不婬、不妄語、不飲酒戒，是名下生子。」

爾時，世尊即說偈言：

「生隨及生上，　　智父之所欲， 
 　生下非所須，　　以不紹繼故， 
 　為人法之子，　　當作優婆塞。 
 　於佛法僧寶，　　勤修清淨心， 
 　雲除月光顯，　　光榮眷屬眾。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如五戒，如是信、戒、施、聞、慧經，亦如是說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「有三種子。何等為三？**
　　那時，世尊告訴在場的眾位比丘說：「（在道德修行上）有三種子女的類別。這三種是指什麼呢？

**◈ 有隨生子、有勝生子、有下生子、**
　　分別是：隨行之子（繼承父母美德的子女）、超越之子（比父母更具德行的子女）、以及墮落之子（未能繼承父母美德的子女）。

**◈ 何等為隨生子？謂子父母不殺、不盜、不婬、不妄語、不飲酒，子亦隨學不殺、不盜、不婬、不妄語、不飲酒，是名隨生子。**
　　什麼是隨行之子？是指父母實踐不殺生、不偷盜、不邪淫、不妄語、不飲酒（這五項基本的道德戒律），而子女也隨之學習並實踐這五項戒律，這就稱為隨行之子。

**◈ 何等為勝生子？若子父母不受不殺、不盜、不婬、不妄語、不飲酒戒，子則能受不殺、不盜、不婬、不妄語、不飲酒戒，是名勝生子。**
　　什麼是超越之子？是指父母沒有受持不殺生、不偷盜、不邪淫、不妄語、不飲酒的戒律，但子女卻能自主地受持並實踐這些戒律，這就稱為超越之子。

**◈ 云何下生子？若子父母受不殺、不盜、不婬、不妄語、不飲酒戒，子不能受不殺、不盜、不婬、不妄語、不飲酒戒，是名下生子。」**
　　什麼是墮落之子？是指父母受持了不殺生、不偷盜、不邪淫、不妄語、不飲酒的戒律，但子女卻不能受持並實踐這些戒律，這就稱為墮落之子。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即用詩偈的形式說道：

**◈ 「生隨及生上，智父之所欲，**
　　「能夠隨行美德或超越父母德行的子女，是具有智慧的父母所希望的，

**◈ 生下非所須，以不紹繼故，**
　　而墮落的子女則不是父母所需要的，因為他們無法繼承並延續家族的善法。

**◈ 為人法之子，當作優婆塞。**
　　若要成為真正符合法理（正法）的子女，應當成為一名優婆塞（在家修行的男信徒）。

**◈ 於佛法僧寶，勤修清淨心，**
　　對於佛、法、僧三寶，應勤奮地修行以保持內心的清淨，

**◈ 雲除月光顯，光榮眷屬眾。」**
　　就像雲朵散去後月光顯現一樣，如此修行將會讓自己的親屬與眷屬們都感到光榮。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如五戒，如是信、戒、施、聞、慧經，亦如是說。**
　　如同關於五戒的說明一樣，關於信（信心）、戒（戒律）、施（布施）、聞（聞法）、慧（智慧）的經文，也是同樣的道理。

### [詞語註解（解釋重要佛教概念）]
1. **五戒 (Pañca-sīla)**：原始佛教中在家信徒應遵守的五項基本道德準則，包括不殺生、不偷盜、不邪淫、不妄語、不飲酒。其目的是為了防止對他人的傷害並維持心靈的安定。
2. **優婆塞 (Upāsaka)**：指在家修行的男信徒。他們雖然不出家，但對三寶有堅定的信仰，並在日常生活中實踐佛法。
3. **三寶 (Tiratana)**：指佛（覺悟者）、法（真理/教法）、僧（修行共同體）。這是佛教信仰的核心依止對象。
4. **紹繼 (Succession)**：在本文中指對善法、德行與正見的傳承，而非單純的血緣或財產繼承。

### [現代實作指南]
本經文將「孝道」從單純的物質供養或血緣服從，提升到了「德行傳承」的層次。在現代生活中，我們可以透過以下方式實踐：

1. **重新定義「好孩子」**：
   ➔ 意識到真正的孝順不是盲目地順從父母的所有意願，而是透過自身的道德成長（如實踐五戒），讓父母在精神上感到欣慰。如果你發現父母有不良習慣，你選擇不隨之而行並活出正向的榜樣，這便是「勝生子」的實踐。

2. **建立個人道德底線（五戒實作）**：
   ➔ **不殺生** ➔ 練習慈悲，減少對生命的傷害。
   ➔ **不偷盜** ➔ 尊重他人的權利與財產，不佔便宜。
   ➔ **不邪淫** ➔ 在親密關係中保持忠誠與尊重。
   ➔ **不妄語** ➔ 練習誠實，不以謊言操縱他人。
   ➔ **不飲酒** ➔ 保持心智清明，避免因物質成癮而喪失自我控制力。

3. **轉念：從「繼承財產」到「繼承善法」**：
   ➔ 當我們思考如何回饋父母時，除了物質照顧，更應思考如何將「正見」與「清淨心」帶入家庭。當一個家庭成員開始修行清淨心，其正面影響力會像「雲除月光顯」一樣，改善整個家庭的氛圍。

### [段落大意]
本經由佛陀定義三種子女：隨行之子（繼承父母美德）、超越之子（比父母更具德行）與墮落之子（不繼承美德）。佛陀強調，真正的子女應透過受持五戒、皈依三寶並修行清淨心，成為「法之子」，如此不僅能成就自身，更能令家族光榮。此邏輯同樣適用於信、戒、施、聞、慧等所有善法。
