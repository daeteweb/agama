# 雜阿含經 第 729 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「當修七覺分。何等為修七覺分？謂念覺分，乃至捨覺分，若比丘修念覺分，依遠離、依無欲、依滅、向於捨。如是修擇法、精進、喜、猗、定、捨覺分，依遠離、依無欲、依滅、向於捨。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「當修七覺分。**
　　那時，世尊告訴在場的眾位比丘說：「你們應該修行七種覺悟的要素（七覺分）。

**◈ 何等為修七覺分？**
　　什麼是修行七種覺悟的要素？

**◈ 謂念覺分，乃至捨覺分，**
　　指的是：念覺分（正念）、擇法覺分（審慎抉擇）、精進覺分（持續努力）、喜覺分（法喜）、輕安覺分（身心安適）、定覺分（專注安定）、以及捨覺分（平等捨離）。

**◈ 若比丘修念覺分，依遠離、依無欲、依滅、向於捨。**
　　如果比丘在修行念覺分（正念）時，應該依據遠離（遠離感官誘惑）、依據無欲（沒有貪求）、依據滅（熄滅煩惱），並趨向於捨（一種平等且不執著的狀態）。

**◈ 如是修擇法、精進、喜、猗、定、捨覺分，依遠離、依無欲、依滅、向於捨。」**
　　同樣地，在修行擇法、精進、喜、輕安（猗）、定、捨這六種覺分時，也都要依據遠離、依據無欲、依據滅，並趨向於捨。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **七覺分 (Satta Bojjhaṅga)**：指導致覺悟的七種心理因素，包含念（正念）、擇法（分析真理）、精進（努力修行）、喜（法喜）、輕安（身心放鬆）、定（專注）、捨（平等心）。
2. **遠離 (Viveka)**：指遠離導致煩惱的環境或對感官欲望的抽離。
3. **無欲 (Virāga)**：指對對象失去貪愛與渴求的狀態，是解脫的關鍵步驟。
4. **滅 (Nirodha)**：指煩惱與苦因（貪、嗔、癡）的完全熄滅。
5. **捨 (Upekkhā)**：指一種不偏不倚、不執著於好壞、保持心靈平衡的中立狀態。
6. **猗 (Passaddhi)**：即輕安，指在修行過程中，身體與心理的緊張感消失，達到極其舒適且寧靜的狀態。

### [現代實作指南]
本經文提供了一套將「心理正向狀態」轉化為「解脫工具」的具體路徑。在現代生活中，我們常將「喜悅」或「專注」視為目的，但佛陀提醒，若這些狀態伴隨著執著，則仍是束縛。

**實作步驟：**
1. **正念覺察 ➔ 遠離**：當你意識到自己在憤怒或貪婪時（念），不要試圖對抗它，而是意識到這種情緒是「不適的」，從而心理上與該情緒產生距離（遠離）。
2. **分析與努力 ➔ 無欲**：分析該情緒產生的原因（擇法），並努力維持覺察（精進）。在分析過程中，意識到該欲望並不能帶來真正的滿足，從而降低對該欲望的渴求（無欲）。
3. **體驗輕安 ➔ 熄滅**：當欲望降低，心靈會自然產生一種放鬆感（喜與輕安）。在這種狀態下，觀察煩惱如何逐漸消失（滅）。
4. **穩定心境 ➔ 平等捨離**：將這種寧靜維持在專注狀態（定），最後達到無論外界環境如何變動，內心都能保持不動搖、不評判的平衡狀態（捨）。

**轉念技巧：**
將「我想獲得快樂」轉化為「我觀察快樂的生起，並在快樂中保持不執著」。這樣能防止「喜」變成「貪」，將心理能量從「追求」轉向「覺悟」。

### [段落大意]
本經闡述了修行「七覺分」的正確方法。佛陀強調，無論是在培養正念、分析法理、精進努力、感受法喜、獲得輕安、進入定境或實踐捨離時，都必須以「遠離、無欲、滅」為基礎，最終目標是趨向於「捨」的平等狀態，而非執著於修行過程中的正向感受。
