# 雜阿含經 第 824 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有二學。何等為二？謂上威儀學、上波羅提木叉學。」

爾時，世尊即說偈言：

「學者學戒時，　　直道隨順行， 　專審勤方便，　　善自護其身， 　得初漏盡智，　　次究竟無知， 　得無知解脫，　　知見悉已度。 　成不動解脫，　　諸有結滅盡， 　彼諸根具足，　　諸根寂靜樂， 　持此後邊身，　　摧伏眾魔怨。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有二學。何等為二？謂上威儀學、上波羅提木叉學。」**
　　那時，世尊告訴在場的眾位比丘說：「有兩種修行學習。什麼是這兩種？指的是最高層級的威儀（關於舉止禮節的規範）修行，以及最高層級的波羅提木叉（比丘必須遵守的根本戒律）修行。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即以偈頌的形式說道：

**◈ 「學者學戒時，直道隨順行，**
　　修行者在學習戒律（規範行為）的時候，應當正直地依照正法隨順而行，

**◈ 專審勤方便，善自護其身，**
　　專注地審視並勤奮地運用方便法門（適當的修行方法），善於保護自己的身體與心靈不被染污，

**◈ 得初漏盡智，次究竟無知，**
　　如此便能獲得初步的漏盡智（消除煩惱之漏的智慧），接著達到究竟的無知（指不再有對世俗執著的認知），

**◈ 得無知解脫，知見悉已度。**
　　獲得這種不再執著於認知而產生的解脫，所有的錯誤知見都已超越。

**◈ 成不動解脫，諸有結滅盡，**
　　成就了不動的解脫（不再受外界干擾的境界），所有存在的結（束縛心靈的煩惱結）全部滅盡，

**◈ 彼諸根具足，諸根寂靜樂，**
　　那時，所有的根（感官與心靈能力）都達到圓滿具足，所有的根都處於寂靜且快樂的狀態，

**◈ 持此後邊身，摧伏眾魔怨。」**
　　持守這最後階段的身體（指達到最後果位後的狀態），能摧毀並降伏所有魔障與怨恨。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **威儀學 (Sekhiya)**：指關於日常生活中的禮儀、舉止與社交規範的訓練，旨在透過外在行為的端正來安定內心。
2. **波羅提木叉 (Pātimokkha)**：原意為「一次全部地」，指比丘與比丘尼必須共同誦持並遵守的根本戒律清單。
3. **漏盡智 (Āsavakkhaya-ñāṇa)**：指能將「漏」（āsava，指如貪欲、存在欲、無明等像滲漏一樣不斷流出的煩惱）徹底清除的智慧。
4. **結 (Samyojana)**：指將眾生束縛在輪迴（Saṃsāra）之中的心理枷鎖，如色欲結、有結、慢結等。
5. **根 (Indriya)**：指主導心靈功能的要素，包括五根（眼耳鼻舌身）與心靈根（如信根、精進根、念根、定根、慧根）。

### [現代實作指南]
本經強調「外在規範」與「內在解脫」的遞進關係。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **從「微習慣」開始（對應威儀學）**：
   不要試圖立刻達到深奧的禪定。先從調整生活細節開始，例如：吃飯時專注於食物、走路時保持脊椎挺直、說話時保持溫和。透過對外在行為的微小掌控，訓練心靈的覺知力。

2. **建立底線原則（對應波羅提木叉學）**：
   為自己設定幾項不可逾越的道德底線（如：不欺騙、不傷害他人）。當行為有了明確的規範，心靈就不會因過多的選擇或愧疚感而產生劇烈波動，為後續的心理清理創造穩定環境。

3. **審視與調整（對應專審勤方便）**：
   每天花 5-10 分鐘回顧一天的行為。問自己：「剛才的反應是隨順正法，還是隨順情緒？」若發現偏差，不責備自己，而是尋找更適當的方法（方便法門）來修正。

4. **追求「不動」的心理狀態（對應不動解脫）**：
   當外在行為端正、內在原則明確後，練習在面對衝突時，觀察內心升起的「結」（如憤怒、執著）。意識到這些結只是暫時的現象而非「我」，從而達到不被外界干擾的心理韌性。

### [段落大意]
本經闡述修行需兼顧外在的威儀規範與內在的戒律修行。透過正直地實踐戒律與勤奮的自我修正，修行者能逐步消除煩惱之漏，超越錯誤知見，最終滅盡束縛心靈的結，使感官與心靈達到圓滿寂靜，成就不可動搖的解脫境界。
