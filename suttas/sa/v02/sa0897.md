# 雜阿含經 第 897 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

時，尊者羅睺羅來詣佛所，稽首禮足，退坐一面，白佛言：「世尊！云何知、云何見，我此識身及外境界一切相不憶念，於其中間盡諸有漏？」

佛告羅睺羅：「有內六入處。何等為六？謂眼入處。耳、鼻、舌、身、意入處，此等諸法，正智觀察，盡諸有漏，正智心善解脫，是名阿羅漢，盡諸有漏，所作已作，已捨重擔，逮得己利，盡諸有結，正智心得解脫。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如內六入處，如是外六入處乃至五陰，亦如是說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 時，尊者羅睺羅來詣佛所，稽首禮足，退坐一面，白佛言：**
　　那時，尊者羅睺羅來到佛陀這裡，頂禮佛陀的足部，退到一邊坐下，對佛陀恭敬地說道：

**◈ 「世尊！云何知、云何見，我此識身及外境界一切相不憶念，於其中間盡諸有漏？」**
　　「世尊！我該如何知道、如何觀察，才能讓我對這個識（意識）所構成的身體，以及外部環境的所有現象不再產生執著的記憶，並在這些過程之中，徹底消除所有有漏（被煩惱污染）的狀態？」

**◈ 佛告羅睺羅：「有內六入處。何等為六？謂眼入處。耳、鼻、舌、身、意入處，**
　　佛陀告訴羅睺羅：「有六種內在的感官入口（內六入處）。什麼是這六種？指的是眼、耳、鼻、舌、身、意這六個感官的入口。

**◈ 此等諸法，正智觀察，盡諸有漏，正智心善解脫，是名阿羅漢，**
　　對於這些感官所接觸到的法（現象），若能用正確的智慧去觀察，就能消除所有被煩惱污染的狀態（盡諸有漏）；當正確智慧的心獲得圓滿的解脫，這就稱為阿羅漢。

**◈ 盡諸有漏，所作已作，已捨重擔，逮得己利，盡諸有結，正智心得解脫。」**
　　他徹底消除了所有有漏的煩惱，完成了所有應做的事業，捨棄了輪迴的沉重擔子，達到了自我的利益，斷除了所有將眾生繫縛在輪迴中的結（結使），使正確智慧的心獲得了真正的解脫。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如內六入處，如是外六入處乃至五陰，亦如是說。**
　　正如對內六入處的說明，對於外六入處（外部的色聲香味觸法）乃至五陰（構成身心的五種要素：色、受、想、行、識），也同樣如此說明。

### [詞語註解]
1. **內六入處 (Salāyatanā)**：指六種內在的感官接收系統，包括眼、耳、鼻、舌、身、意。它們是意識與外部世界接觸的門戶。
2. **有漏 (Āsava)**：指被煩惱（如貪、嗔、痴）所污染的狀態。在原始佛教中，漏是指像滲漏一樣不斷流出的煩惱，導致眾生在輪迴中受苦。
3. **阿羅漢 (Arahant)**：指已徹底斷除所有煩惱、不再受輪迴之苦的覺悟者。
4. **結 (Samyojana)**：指將眾生繫縛在輪迴（Saṃsāra）中的心理紐帶，如對自我的執著、對感官欲望的追求等。
5. **五陰 (Pañca-khandha)**：即五蘊，指構成生命個體的五種要素：色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）。

### [現代實作指南]
本經的核心在於「觀察感官的入口」，將其轉化為現代生活中的「正念覺察」練習：

1. **建立「感官門衛」意識**：
   當你感到情緒波動（如憤怒或焦慮）時，立即將注意力從「事件本身」移回「感官入口」。
   ➔ 練習：問自己「現在是哪個入口在運作？」
   ➔ 例如：聽到批評時 → 覺察到「耳入處」接收到了聲音 → 意識到這僅僅是一個聲音現象，而非對我人格的定義。

2. **切斷「憶念」的連鎖反應**：
   羅睺羅提到的「不憶念」是指不對過去的經驗產生執著或反芻。
   ➔ 實作：當大腦開始回想過去的委屈或未來的擔憂時，溫柔地標記「正在憶念」，然後將注意力重新拉回到當下的呼吸或身體感覺上。

3. **從「我」轉向「法」**：
   將「我的身體」、「我的感受」轉化為「色（物質身體）」、「受（感受）」等客觀現象。
   ➔ 轉念技巧：將「我很痛苦」 → 轉化為「現在有一個『受（感受）』在運作」。透過這種客觀化（Objectification），能有效降低情緒對心靈的掌控力，達成初步的「解脫」。

### 段落大意 
本經記載羅睺羅詢問佛陀如何消除對身心與外境的執著以斷除煩惱。佛陀指導應透過正智觀察「內六入處」（六根），當能以正確智慧覺察感官現象而不產生執著時，即可消除有漏，達成阿羅漢的解脫境界。此修行方法同樣適用於外六入處及五蘊。
