# 雜阿含經 第 983 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，尊者阿難住舍衛國祇樹給孤獨園。獨一靜處。如是思惟：「或有一人作如是念，我於此識身及外境界一切相，無有我、我所見、我慢繫著使，及心解脫、慧解脫，現法自知作證具足住；於此識身及外境界一切相，無有我、我所見、我慢繫著使，我當於彼心解脫、慧解脫，現法自知作證具足住。」

爾時，尊者阿難晡時從禪覺，詣世尊所，稽首禮足，退坐一面，白佛言：「世尊！我獨一靜處，作是思惟：『若有一人作如是言：「我此識身及外境界一切相……」乃至自知作證具足住。』」

佛告阿難：「如是！如是！有一人作如是念：『我此識身及外境界一切相，無有我、我所見、我慢繫著使，及心解脫、慧解脫，現法自知作證具足住。』阿難！彼比丘於此識身及外境界一切相，無有我、我所見、我慢繫著使，及心解脫、慧解脫，現法自知作證具足住；於此識身及外境界一切相，無有我、我所見、我慢繫著使，及彼心解脫、慧解脫，現法自知作證住。

「阿難！若復比丘於此識身及外境界一切相，乃至自知作證具足住，是名比丘斷愛縛結、慢無間等，究竟苦邊。

「阿難！我於此有餘說，答波羅延憂陀耶所問：

「斷於愛欲想，　　憂苦亦俱離， 
　覺悟於睡眠，　　滅除掉悔蓋， 
　捨貪恚清淨，　　現前觀察法， 
　我說智解脫，　　滅除無明闇。」

佛說是經已，尊者阿難聞佛所說，歡喜隨喜，禮佛而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，尊者阿難住舍衛國祇樹給孤獨園。獨一靜處。**
　　那時，尊者阿難也住在舍衛國的祇樹給孤獨園中，獨自處於一個安靜的地方。

**◈ 如是思惟：「或有一人作如是念，我於此識身及外境界一切相，無有我、我所見、我慢繫著使，及心解脫、慧解脫，現法自知作證具足住；於此識身及外境界一切相，無有我、我所見、我慢繫著使，我當於彼心解脫、慧解脫，現法自知作證具足住。」**
　　他如此思考：「是否可能有一個人產生這樣的念頭：對於這個識（意識）之身以及外部境界的一切現象，都沒有『我』或『我的』這種認知（我所見），也沒有被我慢（自我傲慢）的繫著使（束縛）所牽引，並且在心解脫（心的自由）與慧解脫（智慧的自由）中，於現法（當下）能自我覺知並證悟地安住；而我也應該在對於這個識（意識）之身及外部境界的一切現象，沒有『我』或『我的』認知與我慢束縛的情況下，達到心解脫與慧解脫，並於現法中自我覺知地安住。」

**◈ 爾時，尊者阿難晡時從禪覺，詣世尊所，稽首禮足，退坐一面，白佛言：「世尊！我獨一靜處，作是思惟：『若有一人作如是言：「我此識身及外境界一切相……」乃至自知作證具足住。』」**
　　那時，尊者阿難在傍晚從禪定中醒來，走到世尊面前，頂禮觸足，退到一邊坐下，對佛陀恭敬地說道：「世尊！我獨自在安靜處思考：『若有一人說：我對此識（意識）之身及外部境界的一切相……』直到自我覺知地安住。」

**◈ 佛告阿難：「如是！如是！有一人作如是念：『我此識身及外境界一切相，無有我、我所見、我慢繫著使，及心解脫、慧解脫，現法自知作證具足住。』**
　　佛陀告訴阿難：「確實如此！確實如此！確實有一個人會產生這樣的念頭：『我對此識（意識）之身及外部境界的一切相，沒有『我』或『我的』認知，也沒有我慢的束縛，且在心解脫與慧解脫中，於現法中自我覺知地安住。』

**◈ 阿難！彼比丘於此識身及外境界一切相，無有我、我所見、我慢繫著使，及心解脫、慧解脫，現法自知作證具足住；於此識身及外境界一切相，無有我、我所見、我慢繫著使，及彼心解脫、慧解脫，現法自知作證具足住。**
　　阿難！那位比丘對於這個識（意識）之身及外部境界的一切相，沒有『我』或『我的』認知，也沒有我慢的束縛，且在心解脫與慧解脫中，於現法中自我覺知地安住；正因為對此識（意識）之身及外部境界的一切相，沒有『我』或『我的』認知與我慢束縛，他才能在心解脫與慧解脫中，於現法中自我覺知地安住。」

**◈ 「阿難！若復比丘於此識身及外境界一切相，乃至自知作證具足住，是名比丘斷愛縛結、慢無間等，究竟苦邊。**
　　「阿難！如果一位比丘對於這個識（意識）之身及外部境界的一切相，達到自我覺知地安住，這就稱為比丘斷除了愛（貪愛）的束縛與我慢等障礙，達到了苦的終結（究竟苦邊）。」

**◈ 「阿難！我於此有餘說，答波羅延憂陀耶所問：**
　　「阿難！關於這一點，我還有其他的教導，是用來回答波羅延憂陀耶所問的：

**◈ 斷於愛欲想，憂苦亦俱離，覺悟於睡眠，滅除掉悔蓋，捨貪恚清淨，現前觀察法，我說智解脫，滅除無明闇。」**
　　『斷除對於愛欲的想像，憂苦也會隨之離去；從睡眠（無明之眠）中覺悟，消除悔恨的遮蔽；捨棄貪婪與嗔恚以達到清淨，在當下觀察法（真理），我所說的智解脫（智慧的解脫），能滅除無明（根本無知）的黑暗。』」

**◈ 佛說是經已，尊者阿難聞佛所說，歡喜隨喜，禮佛而去。**
　　佛陀講完這部經後，尊者阿難聽聞佛陀所說的教法，心生法喜並依照教導去實踐，向佛陀行禮後才離開。

### [詞語註解]
1. **識身 (Viññāṇa-kāya)**：指意識的總體運作，即將感知對象轉化為認知經驗的心理過程。
2. **我所見 (Sakkāya-diṭṭhi)**：一種錯誤的認知，認為身體或心理現象中存在一個永恆、獨立的「我」或「我的」所有物。
3. **我慢 (Māna)**：基於對「我」的執著而產生的自我傲慢或對比心態（認為自己比他人優或劣）。
4. **繫著使 (Samyojana)**：將眾生束縛在輪迴（Saṃsāra）中的心理枷鎖，如貪愛、嗔恨、無明等。
5. **心解脫 (Citta-vimutti)**：透過定力（Samādhi）使心遠離煩惱，達到寂靜與自由的狀態。
6. **慧解脫 (Paññā-vimutti)**：透過對無常、苦、無我的洞察（Vipassanā），從根本上切斷執著而獲得的自由。
7. **現法自知作證 (Paccattaṃ abhiññā)**：指不需要依賴他人的描述或理論，而在當下的生命經驗中直接體證真理。
8. **無明 (Avijjā)**：對四聖諦（苦、集、滅、道）的根本無知，是所有痛苦與輪迴的源頭。

### [現代實作指南]
本經文的核心在於「拆解自我認同」。在現代生活中，我們經常被「我」這個概念綁架（例如：「我的」名聲受損」、「我」覺得被冒犯）。可採取以下實作步驟：

1. **觀察「我所見」的產生**：
   當強烈的情緒（如憤怒或委屈）升起時，不要立刻進入情緒，而是觀察心念中是否出現了「我」這個主體。
   ➔ 轉念技巧：將「我很生氣」改為「此刻有『生氣』這種感受（受）在運作」。將主體（我）抽離，觀察現象。

2. **辨識「識身」與「外境界」的互動**：
   意識到我們感受到的世界，其實是「外部刺激 ➔ 意識處理 ➔ 產生認知」的過程。
   ➔ 實作方法：當你對某件事產生強烈評價時，提醒自己：「這只是我的意識（識身）對外部現象（外境界）的一種反應，而非事實本身。」

3. **練習「捨棄我慢」**：
   當你感到優越或自卑時，意識到這正是「我慢」在起作用。
   ➔ 轉念技巧：意識到無論是優越感還是自卑感，都是基於一個虛構的「我」在做比較。試著將注意力從「比較」轉移到「當下的觀察」。

4. **從「睡眠」中覺悟**：
   經文提到的「睡眠」是指對真理的麻木。
   ➔ 實作方法：每天花5分鐘練習「正念觀察」，意識到身體的呼吸與心念的起伏，打破慣性反應的自動導航模式，這便是初步的「覺悟於睡眠」。

### [段落大意]
本經記載尊者阿難在禪定後，向佛陀請教關於一個人能否在意識與外部境界中完全消除「我」與「我的」認知及我慢束縛，進而達到心解脫與慧解脫的狀態。佛陀肯定了這一可能，並指出這種狀態即是斷除貪愛與我慢、終結痛苦的境界。最後，佛陀以詩偈強調，透過斷除愛欲、覺悟無明、捨棄貪嗔並觀察真理，方能獲得智解脫，滅除無明的黑暗。
