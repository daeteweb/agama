# 雜阿含經 第 801 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有五法，多所饒益修安那般那念。何等為五？住於淨戒波羅提木叉律儀，威儀行處具足，於微細罪能生怖畏，受持學戒，是名第一多所饒益修習安那般那念。復次，比丘！少欲、少事、少務，是名二法多所饒益修習安那般那念。復次，比丘！飲食知量，多少得中，不為飲食起求欲想，精勤思惟，是名三法多所饒益修安那般那念。復次，比丘！初夜、後夜不著睡眠，精勤思惟，是名四法多所饒益修安那般那念。復次，比丘！空閑林中，離諸憒閙，是名五法多種饒益修習安那般那念。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有五法，多所饒益修安那般那念。何等為五？**
　　那時，世尊告訴在場的眾位比丘說：「有五種方法，能讓修習安那般那念（對呼吸的正念覺知）獲得極大的利益。這五種方法是什麼呢？

**◈ 住於淨戒波羅提木叉律儀，威儀行處具足，於微細罪能生怖畏，受持學戒，是名第一多所饒益修習安那般那念。**
　　首先，住在清淨的波羅提木叉（比丘戒律）律儀之中，在舉止與行處上都能做到端正具足，對於微小的過錯也能產生警覺與畏怖之心，並虔誠地受持並實踐學戒，這被稱為第一種能讓修習安那般那念獲益良多的方法。

**◈ 復次，比丘！少欲、少事、少務，是名二法多所饒益修習安那般那念。**
　　另外，比丘！保持少欲（慾望少）、少事（瑣事少）、少務（雜務少），這被稱為第二種能讓修習安那般那念獲益良多的方法。

**◈ 復次，比丘！飲食知量，多少得中，不為飲食起求欲想，精勤思惟，是名三法多所饒益修安那般那念。**
　　另外，比丘！對飲食量有自覺的掌控，多少都能適中，不會因為飲食而生起貪求的想（想像與認知），並勤奮地進行思惟，這被稱為第三種能讓修習安那般那念獲益良多的方法。

**◈ 復次，比丘！初夜、後夜不著睡眠，精勤思惟，是名四法多所饒益修安那般那念。**
　　另外，比丘！在初夜與後夜不沉溺於睡眠，勤奮地進行思惟，這被稱為第四種能讓修習安那般那念獲益良多的方法。

**◈ 復次，比丘！空閑林中，離諸憒閙，是名五法多種饒益修習安那般那念。」**
　　另外，比丘！身處空閑的森林之中，遠離種種喧鬧與干擾，這被稱為第五種能讓修習安那般那念獲益良多的方法。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **安那般那念 (Ānāpānasati)**：指對呼吸（入息與出息）的持續覺知。這是一種透過觀察呼吸來穩定心神，進而發展正念與定力的修行方法。
2. **波羅提木叉 (Pātimokkha)**：指比丘與比丘尼必須遵守的根本戒律清單，是僧團維持純淨與和諧的基礎。
3. **律儀 (Sīla-saṃvara)**：指在行為上對戒律的謹慎守持，使身心不被不善的行為所染污。
4. **少欲、少事、少務**：一種簡約的生活態度。減少對物質與感官的渴求（少欲），減少不必要的社交與瑣事（少事），減少繁雜的行政或世俗事務（少務），以騰出心力專注於修行。

### [現代實作指南]
本經文提供了提升冥想（呼吸覺知）品質的五個外部與內部條件。現代人在繁忙生活中可將其轉化為以下實踐：

1. **建立生活底線（對應淨戒）**：
   ➔ 實作：設定自己的「生活準則」（如：不欺騙、不傷害他人）。當生活充滿愧疚感或道德衝突時，心神會不安，難以進入深層的呼吸覺知。保持誠實與善良是心靈平靜的基礎。
2. **精簡生活（對應少欲少事）**：
   ➔ 實作：執行「數位斷捨離」。減少不必要的社交媒體瀏覽與無意義的資訊接收，降低對物質消費的追逐。當大腦不再被過多瑣事佔據，在呼吸修行時才不會不斷跳出雜念。
3. **覺知飲食（對應飲食知量）**：
   ➔ 實作：練習「正念飲食」。在用餐時專注於食物的味道與飽足感，避免過量飲食導致的昏沉（食後嗜睡），這能讓修行時的意識保持清明。
4. **規律作息與專注（對應精勤思惟）**：
   ➔ 實作：在早晨或深夜設定一個「神聖時間」。在外界安靜時，撥出 15-30 分鐘專注於呼吸，避免在疲憊不堪或昏睡狀態下強行修行。
5. **創造物理空間（對應空閑林中）**：
   ➔ 實作：建立一個「冥想角落」。即便不在森林，也可以在房間內清理出一塊乾淨、安靜、不被干擾的區域，透過環境的暗示讓心靈迅速進入平靜狀態。

### [段落大意]
本經詳細說明了能讓「呼吸覺知修行」獲益良多的五種輔助條件：一是守持清淨戒律與端正威儀；二是簡約生活，減少慾望與瑣事；三是節制飲食並勤於思惟；四是保持清醒，不沉溺睡眠；五是選擇安靜遠離喧囂的環境。這五者共同構成了支持心靈定力的外部環境與內部基礎。
