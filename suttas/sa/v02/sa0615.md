# 雜阿含經 第 615 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，尊者阿難晨朝著衣持鉢，入舍衛城乞食，於路中思惟：「我今先至比丘尼寺。」即往比丘尼寺。

諸比丘尼遙見尊者阿難來，疾敷床座，請令就座。

時，諸比丘尼禮尊者阿難足，退坐一面，白尊者阿難：「我等諸比丘尼修四念處繫心住，自知前後昇降。」

尊者阿難告諸比丘尼：「善哉！善哉！姊妹！當如汝等所說而學，凡修習四念處善繫心住者，應如是知前後昇降。」

時，尊者阿難為諸比丘尼種種說法，種種說法已，從座起去。

爾時，尊者阿難於舍衛城中乞食還，舉衣鉢，洗足已，詣世尊所，稽首佛足，退坐一面。以比丘尼所說具白世尊。

佛告阿難：「善哉！善哉！應如是學四念處善繫心住，知前後昇降。所以者何？心於外求，然後制令求其心，散亂心、不解脫皆如實知。若比丘於身身觀念住，於彼身身觀念住已，若身躭睡，心法懈怠，彼比丘當起淨信，取於淨相，起淨信心；憶念淨相已，其心則悅，悅已生喜；其心喜已，身則猗息；身猗息已，則受身樂；受身樂已，其心則定。心定者，聖弟子當作是學：『我於此義，外散之心攝令休息，不起覺想及已觀想，無覺無觀，捨念樂住；樂住已，如實知。』受、心、法念亦如是說。」

佛說此經已，尊者阿難聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：一時，佛住舍衛國祇樹給孤獨園。**
　　我是這樣聽佛陀說的：在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，尊者阿難晨朝著衣持鉢，入舍衛城乞食，於路中思惟：「我今先至比丘尼寺。」即往比丘尼寺。**
　　那時，尊者阿難在早晨穿好衣服並拿著缽，進入舍衛城乞食。在路途中他思考著：「我現在先前往比丘尼（女性出家修行者）的寺院。」於是便前往比丘尼寺。

**◈ 諸比丘尼遙見尊者阿難來，疾敷床座，請令就座。**
　　眾位比丘尼從遠處看到尊者阿難走來，趕緊鋪好床座，邀請他就座。

**◈ 時，諸比丘尼禮尊者阿難足，退坐一面，白尊者阿難：「我等諸比丘尼修四念處繫心住，自知前後昇降。」**
　　那時，眾位比丘尼向尊者阿難的足部行禮，然後退到一邊坐下，對尊者阿難恭敬地說道：「我們這些比丘尼正在修行四念處（對身體、感受、心法、心法現象的四種正念觀察），將心專注地繫留在其中，因此能清楚地知道心念在修行過程中的前後進退與起伏。」

**◈ 尊者阿難告諸比丘尼：「善哉！善哉！姊妹！當如汝等所說而學，凡修習四念處善繫心住者，應如是知前後昇降。」**
　　尊者阿難告訴比丘尼們：「好極了！好極了！姊妹們！應該依照你們所說的方式來學習。凡是修習四念處並能將心專注繫留的人，就應該這樣地觀察心念的前後進退與起伏。」

**◈ 時，尊者阿難為諸比丘尼種種說法，種種說法已，從座起去。**
　　那時，尊者阿難為比丘尼們講解了各種法（真理與修行方法），講解完畢後，便從座位上起身離開。

**◈ 爾時，尊者阿難於舍衛城中乞食還，舉衣鉢，洗足已，詣世尊所，稽首佛足，退坐一面。以比丘尼所說具白世尊。**
　　那時，尊者阿難在舍衛城中乞食回來，收拾好衣鉢並洗淨雙足後，前往世尊（佛陀）那裡，向佛陀的足部頂禮，然後退到一邊坐下，將比丘尼們所說的話詳細地對世尊報告。

**◈ 佛告阿難：「善哉！善哉！應如是學四念處善繫心住，知前後昇降。所以者何？心於外求，然後制令求其心，散亂心、不解脫皆如實知。」**
　　佛陀告訴阿難：「好極了！好極了！應該這樣學習四念處並將心專注繫留，以觀察心念的前後進退與起伏。為什麼呢？因為心原本向外追求（對外境產生執著），接著再制約心使其回歸觀察自身，如此一來，心靈的散亂以及尚未解脫（從苦中脫離）的狀態，都能如實地被覺知。」

**◈ 「若比丘於身身觀念住，於彼身身觀念住已，若身躭睡，心法懈怠，彼比丘當起淨信，取於淨相，起淨信心；憶念淨相已，其心則悅，悅已生喜；其心喜已，身則猗息；身猗息已，則受身樂；受身樂已，其心則定。」**
　　「如果比丘在色（物質身體）上建立觀念並專注停留，在對身體進行觀念住之後，如果身體感到昏沉欲睡，心法（心的運作）感到懈怠，這位比丘應該喚起純淨的信心，採取純淨的相狀（正面的心理意象），生起純淨的信心；在憶念純淨相狀之後，內心會感到愉悅，愉悅之後會產生喜悅；心生喜悅後，身體會感到放鬆舒適；身體放鬆舒適後，便能感受到身體的快樂；感受到身體快樂後，心靈就會進入安定（定）的狀態。」

**◈ 「心定者，聖弟子當作是學：『我於此義，外散之心攝令休息，不起覺想及已觀想，無覺無觀，捨念樂住；樂住已，如實知。』受、心、法念亦如是說。」**
　　「當心進入安定狀態時，聖弟子應該這樣學習：『我在這個義理中，將向外散亂的心攝受並使其休息，不再起覺察的想像或刻意的觀想，達到沒有刻意覺知也沒有刻意觀想的狀態，在捨（中立不偏）與正念中安樂地停留；在安樂停留之後，便能如實地知曉一切。』對於受（感受）、心（意識狀態）、法（心法現象）的觀察，也是同樣這樣說明的。」

**◈ 佛說此經已，尊者阿難聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，尊者阿難聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四念處 (Satipaṭṭhāna)**：佛教修行中四種正念觀察的對象，分別是：身（物質身體）、受（感受）、心（意識狀態）、法（心法現象）。
2. **繫心住**：將心專注地繫留在特定的觀察對象上，不使其隨外境散亂，達到一種穩定的專注狀態。
3. **前後昇降**：指在修行過程中，心念狀態的起伏、進展或退轉，以及對心靈運作過程的動態觀察。
4. **淨相**：在心靈懈怠時，刻意憶念純淨、美好或神聖的意象，用以激發正向的心理能量，對治昏沉。
5. **捨 (Upekkhā)**：一種中立、不偏不倚的心理狀態，既不貪著於樂，也不排斥於苦，是進入深層定境的關鍵。

### [現代實作指南]
本經文提供了一套處理「修行倦怠」與「進入深度專注」的心理操作流程，可轉化為以下現代實踐步驟：

1. **覺察散亂與懈怠**：
   當你在專注工作或冥想時，若發現自己陷入「昏沉（躭睡）」或「心不在焉（懈怠）」，不要對此產生挫折感，而是先「如實知」目前的狀態。

2. **運用「淨相」對治（心理轉念）**：
   當意識低落時，嘗試在心中喚起一個「純淨且正向」的意象。例如：想像清晨純淨的陽光、心中最崇敬的對象，或回想一件讓你感到純粹喜悅且無私的小事。

3. **建立正向情緒鏈接**：
   遵循經文中的心理路徑：
   - **憶念淨相** ➔ **產生愉悅感**（心悅） ➔ **轉化為喜悅**（生喜）。
   - 這種心理上的喜悅會反饋給身體，使肌肉放鬆、呼吸平穩（身猗息）。

4. **從身體舒適進入安定**：
   當身體感到放鬆且快樂時，心靈會自然地停止掙扎，進入一種「定」的狀態。

5. **進入「無為」的觀察狀態**：
   在安定後，停止刻意地「想要覺察」或「強行觀想」。放下所有對結果的期待，僅僅在「中立（捨）」與「覺知（念）」中安住。這是一種不費力的專注，能讓你最真實地看到事物的本質。

### [段落大意]
本經記載尊者阿難與比丘尼討論修習四念處的經驗，並由佛陀進一步闡述如何透過觀察身心狀態來達到定境。佛陀指出，當修行者遇到昏沉懈怠時，可透過憶念「淨相」來激發喜悅，進而使身體放鬆並進入心定狀態。最終，聖弟子應在捨念中安住，不再刻意觀想，從而如實地知曉一切法。
