# 雜阿含經 第 589 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「賴吒槃提國，　　有諸商賈客， 　大富足財寶，　　各各競求富， 　方便欲財利，　　猶如然熾火， 　如是競勝心，　　欲貪常馳騁， 　云何當斷貪，　　息世間勤求？」

爾時，世尊說偈答言：

「捨俗出非家，　　妻子及財寶， 　貪恚癡離欲，　　羅漢盡諸漏， 　正智心解脫，　　愛盡息方便。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出眾的天子，在深夜時分來到佛陀這裡，恭敬地向佛陀的足下頂禮，然後退到一邊坐下。他身體散發出的各種光明，遍照了整個祇樹給孤獨園。

**◈ 時，彼天子說偈問佛：**
　　那時，那位天子用偈頌（詩歌形式）請問佛陀：

**◈ 「賴吒槃提國，有諸商賈客，大富足財寶，各各競求富，方便欲財利，猶如然熾火，如是競勝心，欲貪常馳騁，云何當斷貪，息世間勤求？」**
　　「在賴吒槃提國中，有許多經商的商人，雖然已經擁有豐富的財寶，卻依然各自競爭著追求更多財富。他們追求財利的方法與欲望，就像燃燒的熾火一樣猛烈。這種想要勝過他人的競爭之心，以及對物質的貪愛（愛/Taṇhā）總是奔馳不停。請問應該如何才能斷除這種貪心，使世間這種勤於追求財富的奔波停止？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「捨俗出非家，妻子及財寶，貪恚癡離欲，羅漢盡諸漏，正智心解脫，愛盡息方便。」**
　　「捨棄世俗的生活，離開家庭，不再執著於妻子、孩子以及財寶。遠離貪（貪欲）、恚（嗔恨）與癡（愚癡）的欲望。成為一名阿羅漢（證得解脫的聖者）以盡除所有漏（煩惱的滲漏）。以正智（正確的智慧）使心獲得解脫（Vimokkha），當貪愛（Taṇhā）完全熄滅，追求財利的手段與奔波也就隨之停止。」

**◈ 時，彼天子復說偈言：**
　　那時，那位天子再次說偈頌道：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「我長期以來觀察那些婆羅門，他們最終達到了涅槃（超越生死、絕對寧靜的解脫境界），所有恐懼都已經過去，永遠超越了世間的恩愛與執著。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，那位天子聽聞佛陀所說的教法，心中感到歡喜並隨喜，向佛陀的足下頂禮後，立即消失不再出現。

### [詞語註解]
1. **貪恚癡 (Lobha, Dosa, Moha)**：佛教定義的三毒。貪是指對對象的渴求與執著；恚是指對不滿意對象的嗔恨與排斥；癡是指對真理（如無常、無我）的無知與迷糊。
2. **羅漢 (Arahant)**：指已斷除所有煩惱、不再受輪迴之苦的聖者。
3. **漏 (Āsava)**：指像滲漏一樣不斷流出的煩惱，主要包括欲漏（對感官欲望的執著）、有漏（對存在的執著）與無明漏（對真理的無知）。
4. **解脫 (Vimokkha)**：指心靈從貪、嗔、癡等束縛中完全釋放，不再受制於業力的狀態。
5. **涅槃 (Nibbāna)**：指貪、嗔、癡完全熄滅後的絕對寧靜狀態，是輪迴的終結。

### [現代實作指南]
本經文揭示了「貪欲」的本質如同「熾火」，即便在物質富足的情況下，競爭心依然會驅使人陷入永無止境的奔波。在現代生活中，可採取以下實作步驟：

1. **識別「熾火」的瞬間**：
   當你感到「必須贏過他人」或「即便已有足夠，仍想要更多」的焦慮感升起時，立刻在心中標記：「這是熾火（貪欲）在燃燒」。將這種感受視為一種客觀的生理/心理現象，而非「我的需求」。

2. **練習「足夠」的認知轉向**：
   將注意力從「缺乏什麼（Lack）」轉移到「已擁有什麼（Enough）」。當競爭心升起時，問自己：「如果我不與他人比較，目前的狀態是否足以維持生存與基本的安寧？」

3. **觀察「追求」後的空虛**：
   記錄自己在達成某個物質目標（如升職、買新產品）後的快感持續時間。你會發現快感極短，而隨之而來的是新的欲望。透過觀察這種「快感 $\rightarrow$ 空虛 $\rightarrow$ 新欲望」的循環，降低對「追求」的期待感。

4. **建立「心理出家」的空間**：
   不需要真的離開家庭，而是在心中建立一個「不被財富定義」的空間。每天撥出10分鐘，練習放下所有社會身份（如：經理、父母、子女），僅僅作為一個觀察者存在，體會不依賴外部條件而產生的寧靜。

### [段落大意]
一位天子請問佛陀，為何人們在財富充足時仍會陷入激烈的競爭與貪欲之中，以及如何才能停止這種奔波。佛陀回答指出，唯有透過捨棄對世俗財富與親情的執著，遠離三毒，達到阿羅漢的境界，以正智斷除貪愛，才能獲得真正的解脫。天子對此深表贊同，並感嘆涅槃境界能超越所有恐懼與執著。
