# 雜阿含經 第 609 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「我今當說四念處集、四念處沒。諦聽，善思。何等為四念處集、四念處沒？食集則身集、食滅則身沒。如是隨身集觀住，隨身滅觀住，隨身集滅觀住，則無所依住，於諸世間永無所取。

「如是觸集則受集，觸滅則受沒。如是隨集法觀受住，隨滅法觀受住，隨集滅法觀受住，則無所依住，於諸世間都無所取。

「名色集則心集，名色滅則心沒。隨集法觀心住，隨滅法觀心住，隨集滅法觀心住，則無所依住，於諸世間則無所取。

「憶念集則法集，憶念滅則法沒。隨集法觀法住，隨滅法觀法住，隨集滅法觀法住，則無所依住，於諸世間則無所取。是名四念處集、四念處沒。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「我今當說四念處集、四念處沒。**
　　那時，世尊告訴在場的眾位比丘說：「我現在要為你們解說關於四念處（四種正念觀察的確立）是如何積聚產生，以及是如何消滅止息的。

**◈ 諦聽，善思。何等為四念處集、四念處沒？**
　　請你們仔細聽，好好思考。什麼叫做四念處的積聚與消滅呢？

**◈ 食集則身集、食滅則身沒。**
　　當食物（營養）積聚時，色（物質身體）也就隨之形成；當食物（營養）停止供應或消耗完畢時，色（物質身體）也就隨之消亡。

**◈ 如是隨身集觀住，隨身滅觀住，隨身集滅觀住，則無所依住，於諸世間永無所取。**
　　像這樣，隨時觀察色（物質身體）的產生而覺知，觀察色（物質身體）的消亡而覺知，觀察色（物質身體）從產生到消亡的整個過程而覺知，如此便不再執著於任何依託，在世間的一切現象中永遠不再產生執取（貪愛而抓取）。

**◈ 「如是觸集則受集，觸滅則受沒。**
　　同樣地，當觸（感官與對象的接觸）發生時，受（感受）也就隨之產生；當觸（感官與對象的接觸）停止時，受（感受）也就隨之消失。

**◈ 如是隨集法觀受住，隨滅法觀受住，隨集滅法觀受住，則無所依住，於諸世間都無所取。**
　　像這樣，隨其產生的法（現象）觀察受（感受）而覺知，隨其消滅的法（現象）觀察受（感受）而覺知，觀察受（感受）從產生到消滅的過程而覺知，如此便不再執著於任何依託，在世間的一切現象中都不再產生執取。

**◈ 「名色集則心集，名色滅則心沒。**
　　當名色（心理與物質的組合）積聚時，識（意識）也就隨之形成；當名色（心理與物質的組合）消亡時，識（意識）也就隨之消失。

**◈ 隨集法觀心住，隨滅法觀心住，隨集滅法觀心住，則無所依住，於諸世間則無所取。**
　　隨其產生的法（現象）觀察識（意識）而覺知，隨其消滅的法（現象）觀察識（意識）而覺知，觀察識（意識）從產生到消滅的過程而覺知，如此便不再執著於任何依託，在世間的一切現象中則不再產生執取。

**◈ 「憶念集則法集，憶念滅則法沒。**
　　當憶念（記憶與認知）積聚時，對法（一切心理現象）的認知也就隨之形成；當憶念（記憶與認知）消亡時，對法（一切心理現象）的認知也就隨之消失。

**◈ 隨集法觀法住，隨滅法觀法住，隨集滅法觀法住，則無所依住，於諸世間則無所取。**
　　隨其產生的法（現象）觀察法（心理現象）而覺知，隨其消滅的法（現象）觀察法（心理現象）而覺知，觀察法（心理現象）從產生到消滅的過程而覺知，如此便不再執著於任何依託，在世間的一切現象中則不再產生執取。

**◈ 是名四念處集、四念處沒。」**
　　這就叫做四念處的積聚與消滅。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四念處 (Satipaṭṭhāna)**：佛教修行中四種正念觀察的確立，分別為：觀身（色）、觀受（感受）、觀心（意識）、觀法（心理現象）。
2. **名色 (Nāma-rūpa)**：名（Nāma）指心理現象（如受、想、行、識），色（Rūpa）指物質身體。兩者共同構成個體的生命存在。
3. **觸 (Phassa)**：感官（眼耳鼻舌身意）與其對應的對象（色聲香味觸法）以及意識三者同時相遇的過程。
4. **取 (Upādāna)**：對特定對象產生貪愛而產生的執著與抓取，是導致輪迴（Saṃsāra）的核心原因。
5. **法 (Dhamma)**：在此處指一切心理現象或自然法則，特別是指在觀法念處中被觀察的心理狀態。

### [現代實作指南]
本經文的核心在於「觀察因果的生滅」，將其轉化為生活實踐可採取以下步驟：

1. **身體層面的覺察（食 → 身）**：
   - **實作**：在進食或感覺飢餓時，意識到身體的能量狀態。當感到疲憊或身體沉重時，覺察這是物質條件（營養、睡眠）積聚或缺乏的結果。
   - **轉念**：意識到「身體」並非恆常的「我」，而是一個依賴外部營養維持的物質過程。

2. **情緒層面的覺察（觸 → 受）**：
   - **實作**：當你感到憤怒或快樂時，立刻回溯：是什麼「觸」導致了這個「受」？（例如：聽到一句批評 → 觸 → 產生不悅 → 受）。
   - **轉念**：告訴自己：「這份不悅是因為觸發而產生的現象，觸消失了，不悅也會隨之消失」，從而降低對情緒的認同感。

3. **意識層面的覺察（名色 → 心）**：
   - **實作**：觀察念頭的起伏。當一個強烈的想法出現時，意識到這是心理與生理狀態（名色）共同作用的結果。
   - **轉念**：將意識從「我在思考」轉向「有一個思考的現象正在發生」，將主體感抽離。

4. **認知層面的覺察（憶念 → 法）**：
   - **實作**：當你對某件事產生定論（例如：「他這個人很壞」）時，覺察這是基於過去的記憶（憶念）而建構出的認知（法）。
   - **轉念**：意識到所有的認知都是基於記憶的碎片，並非絕對真理，從而放下對特定觀點的執著。

### [段落大意]
本經闡述了四念處（身、受、心、法）的生滅因果關係：物質身體依賴食物而生滅，感受依賴接觸而生滅，意識依賴名色而生滅，而對法的認知則依賴記憶而生滅。透過觀察這四者從積聚到消亡的過程，修行者能體悟到一切現象皆是緣起且無常，進而達到不依託、不執取的解脫狀態。
