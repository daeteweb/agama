# 雜阿含經 第 866 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘……如上說。差別者：「若不得解脫，以欲法、念法、樂法故，取中般涅槃。若不如是，或生般涅槃，若不如是，或有行般涅槃，若不如是，或無行般涅槃，若不如是，或上流般涅槃。若不如是，或復即以此欲法、念法、樂法功德生大梵天中，或生梵輔天中，或生梵身天中。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘……如上說。**
　　那時，世尊告訴在場的眾位比丘說……（內容如前文所述）。

**◈ 差別者：「若不得解脫，以欲法、念法、樂法故，取中般涅槃。**
　　不同之處在於：「如果沒有獲得解脫（Vimokkha，指從煩惱與輪迴中完全解脫），而是依仗對欲法（欲望之法）、念法（思惟之法）、樂法（快樂之法）的實踐，則會進入『取中般涅槃』。」

**◈ 若不如是，或生般涅槃，**
　　如果不是這樣，或者會進入『生般涅槃』。

**◈ 若不如是，或有行般涅槃，**
　　如果不是這樣，或者會進入『有行般涅槃』。

**◈ 若不如是，或無行般涅槃，**
　　如果不是這樣，或者會進入『無行般涅槃』。

**◈ 若不如是，或上流般涅槃。**
　　如果不是這樣，或者會進入『上流般涅槃』。

**◈ 若不如是，或復即以此欲法、念法、樂法功德生大梵天中，或生梵輔天中，或生梵身天中。」**
　　如果仍然不是這樣，則會憑藉這些欲法、念法、樂法的功德，投生到大梵天、梵輔天或梵身天之中。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **解脫 (Vimokkha)**：指心靈從貪、嗔、痴三毒以及對五蘊（色、受、想、行、識）的執著中完全解脫，不再受輪迴之苦的狀態。
2. **般涅槃 (Nibbāna)**：在此處的脈絡中，除了指最終的絕對解脫，亦指在禪定修行中達到的一種暫時的寂靜狀態或特定的定境。
3. **欲法、念法、樂法**：這通常對應於禪定（Jhāna）的初階特徵。欲法指對感官愉悅的處理，念法指專注的思維，樂法指禪定產生的內在喜樂。
4. **梵天 (Brahma)**：佛教宇宙觀中的高階天界。修行者若僅達到高深的禪定而未斷除無明，死後會依業報投生於此，雖壽命極長且快樂，但仍處於輪迴（Saṃsāra）之中。

### [現代實作指南]
本經文揭示了一個關鍵的修行陷阱：**「誤將定境的快樂當作最終的解脫」**。在現代生活中，我們可以將其轉化為以下實踐：

1. **區分「暫時的寧靜」與「根本的覺悟」**：
   當我們透過冥想、深呼吸或進入某種心流狀態而感到平靜、喜悅時，應意識到這僅僅是「樂法」或「念法」產生的暫時狀態（如同經文中的梵天境界），而非真正的解脫。
   ➔ **轉念技巧**：在感到極度平靜時，提醒自己：「這份平靜是因緣而生的，它是無常的，並非永恆的解脫。」

2. **避免對「正向感受」的執著**：
   許多修行者容易追求禪定中的喜悅感（樂法），這反而成了另一種形式的「取（執取）」。
   ➔ **實作方法**：觀察喜悅升起時，不要試圖抓住它或要求它持續，而是以「正念」觀察喜悅的生起與消失，將其視為一種「受（感受）」而非「我」。

3. **從「定」轉向「慧」**：
   單純的專注（定）可能導致投生高階天界，但唯有結合「無我 (Anattā)」的洞察（慧），才能達成真正的解脫。
   ➔ **操作步驟**：在心靈安定後，將注意力轉向觀察身體（色）與意識（識）的運作，體會一切現象皆是緣起且無常的，從而打破對「存在」的執著。

### [段落大意]
本段經文說明了修行者若未能達成完全的解脫，根據其對欲法、念法、樂法的實踐程度與功德，會進入不同層次的涅槃定境，或投生至大梵天等高階天界。強調了單純的禪定功德雖能帶來高層次的生存，但仍非最終的解脫。
