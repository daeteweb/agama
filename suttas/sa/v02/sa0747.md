# 雜阿含經 第 747 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若比丘修無常想，多修習已，得大果大福利。云何比丘修無常想，多修習已，得大果大福利？是比丘心口與無常想俱，修念覺分，依遠離、依無欲、依滅、向於捨；乃至得捨覺分，依遠離、依無欲、依滅、向於捨。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如無常想，如是無常苦想、苦無我想、觀食想、一切世間不可樂想、盡想、斷想、無欲想、滅想、患想、不淨想、青瘀想、膿潰想、膖脹想、壞想、食不盡想、血想、分離想、骨想、空想，一一經如上說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若比丘修無常想，多修習已，得大果大福利。**
　　「如果比丘修行思考一切現象不斷變遷、沒有永恆特質的『無常想』，在經過多次的修習之後，將會獲得巨大的修行果報與福德利益。

**◈ 云何比丘修無常想，多修習已，得大果大福利？**
　　比丘究竟如何修行這種思考現象變遷的『無常想』，才能在多次修習後獲得巨大的果報與福德利益呢？

**◈ 是比丘心口與無常想俱，修念覺分，依遠離、依無欲、依滅、向於捨；**
　　這位比丘在心中與言語上都與『無常想』保持一致，修行正念（對當下覺知）的覺悟要素，依止於遠離感官欲望、依止於消除貪愛、依止於苦因的止息，並趨向於內心的平靜與不執著（捨）；

**◈ 乃至得捨覺分，依遠離、依無欲、依滅、向於捨。」**
　　直到他獲得了平靜不偏不倚（捨）的覺悟要素，同樣依止於遠離感官欲望、依止於消除貪愛、依止於苦因的止息，並趨向於內心的平靜與不執著（捨）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如無常想，如是無常苦想、苦無我想、觀食想、一切世間不可樂想、盡想、斷想、無欲想、滅想、患想、不淨想、青瘀想、膿潰想、膖脹想、壞想、食不盡想、血想、分離想、骨想、空想，一一經如上說。**
　　就像修行『無常想』一樣，關於修行思考「無常且痛苦」的想、思考「痛苦且無我（沒有永恆主體）」的想、觀察食物的想、思考一切世間現象不可樂的想、思考盡（終結）的想、思考斷（截斷煩惱）的想、思考無欲的想、思考滅（苦止）的想、思考患（危險）的想、思考不淨的想、思考皮膚青瘀的想、思考膿瘡潰爛的想、思考身體膖脹的想、思考身體壞滅的想、思考食物未消化之想、思考血液之想、思考肢體分離之想、思考骨骸之想、以及思考空性（Suññatā）的想，每一項的修行方法都如上述內容一樣。

### [詞語註解（解釋重要佛教概念）]
1. **無常 (Anicca)**：指一切由因緣而生的現象（色、受、想、行、識）都在不斷地生起與滅盡，沒有任何事物能保持恆常不變。
2. **覺分 (Bodhyanga)**：指導向覺悟的七種心理要素（念、擇法、精進、喜、捨、定、正念），是修行解脫的關鍵心理狀態。
3. **捨 (Upekkha)**：一種中立、平靜且不偏不倚的心態，不對愉快的感受產生貪著，也不對痛苦的感受產生厭惡。
4. **遠離 (Viveka)**：指在心理上脫離對感官對象的執著，或在環境上遠離會引起煩惱的社交與喧囂。

### [現代實作指南]
本經文提供了一套從「認知」轉化為「心理狀態」的實作路徑，可轉化為以下現代心理實踐：

1. **建立「變遷意識」 (無常想的實作)**：
   ➔ 在日常生活中，當你對某件事物（如名聲、財富、一段關係）產生強烈執著時，有意識地提醒自己：「這件事目前的狀態是暫時的，它必然會改變。」這不是為了讓人悲觀，而是透過認知「變遷」來降低對結果的過度期待與恐懼。

2. **從「觀察」到「平靜」 (念 ➔ 捨 的路徑)**：
   ➔ **第一步（念）**：僅僅觀察當下的感受（例如：我現在感到憤怒）。
   ➔ **第二步（遠離與無欲）**：意識到這個憤怒只是暫時的心理現象，不需要試圖控制它，也不需要被它牽著走（遠離執著）。
   ➔ **第三步（捨）**：讓憤怒自然升起並自然消散，保持一個觀察者的位置，直到心靈恢復到不被情緒左右的平靜狀態。

3. **對抗「完美主義」與「美化傾向」 (不淨想、壞想的實作)**：
   ➔ 當我們過度執著於外貌或物質的完美時，可以練習思考事物的本質（例如：身體最終會老化、物質最終會損壞）。這種練習能幫助我們從「對表象的貪愛」轉向「對真實的接納」，減少因無法維持完美而產生的焦慮。

### [段落大意]
本段經文闡述了修行「無常想」及其它相關觀想（如苦、無我、不淨等）的具體路徑。佛陀指出，透過將心口與無常的認知保持一致，並依止於遠離、無欲與滅，修行者能從「正念」的覺知進階到「捨」的平靜狀態，最終獲得巨大的解脫利益。此修行模式適用於所有旨在破除執著的觀想練習。
