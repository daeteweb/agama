# 雜阿含經 第 929 經

### [原文]
如是我聞：

一時，佛住迦毘羅衛國尼拘律園中。

爾時，釋氏摩訶男來詣佛所，稽首佛足，退坐一面，白佛言：「世尊！云何名優婆塞？」

佛告摩訶男：「優婆塞者，在家清白，乃至盡壽，歸依三寶，為優婆塞，證知我！」

摩訶男白佛：「世尊！云何為滿足一切優婆塞事？」

佛告摩訶男：「若優婆塞有信無戒，是則不具，當勤方便，具足淨戒。具足信、戒而不施者，是則不具；以不具故，精勤方便，修習布施，令其具足滿。信、戒、施滿，不能隨時往詣沙門，聽受正法，是則不具；以不具故，精勤方便，隨時往詣塔寺，見諸沙門，不一心聽受正法，是不具足。信、戒、施、聞修習滿足，聞已不持，是不具足；以不具足故，精勤方便，隨時往詣沙門，專心聽法，聞則能持。不能觀察諸法深義，是不具足；不具足故，精勤方便，信、戒、施、聞，聞則能持，持已，觀察甚深妙義，而不隨順知法次法向，是則不具，以不具故，精勤方便，信、戒、施、聞，受持觀察，了達深義，隨順行法次法向。摩訶男！是名滿足一切種優婆塞事。」

摩訶男白佛：「世尊！云何名優婆塞能自安慰，不安慰他？」

佛告摩訶男：「若優婆塞能自立戒，不能令他立於正戒；自持淨戒，不能令他持戒具足；自行布施，不能以施建立於他；自詣塔寺見諸沙門，不能勸他令詣塔寺往見沙門；自專聽法，不能勸人樂聽正法；聞法自持，不能令他受持正法；自能觀察甚深妙義，不能勸人令觀深義；自知深法能隨順行法次法向，不能勸人令隨順行法次法向。摩訶男！如是八法成就者，是名優婆塞能自安慰，不安慰他。」

摩訶男白佛：「世尊！優婆塞成就幾法自安安他？」

佛告摩訶男：「若優婆塞成就十六法者，是名優婆塞自安安他。何等為十六？摩訶男！若優婆塞具足正信，建立他人；自持淨戒，亦以淨戒建立他人；自行布施，教人行施；自詣塔寺見諸沙門，亦教人往見諸沙門；自專聽法，亦教人聽；自受持法，教人受持；自觀察義，教人觀察；自知深義，隨順修行法次法向，亦復教人解了深義，隨順修行法次法向。摩訶男！如是十六法成就者，是名優婆塞能自安慰，亦安慰他人。

「摩訶男！若優婆塞成就如是十六法者，彼諸大眾悉詣其所，謂婆羅門眾、剎利眾、長者眾，沙門眾，於諸眾中威德顯曜。譬如日輪，初、中及後，光明顯照。如是，優婆塞十六法成就者，初、中及後，威德顯照。如是，摩訶男！若優婆塞十六法成就者，世間難得。」

佛說此經已，釋氏摩訶男聞佛所說，歡喜隨喜，即從坐起，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住迦毘羅衛國尼拘律園中。**
　　在某一個時間，佛陀住在迦毘羅衛國的尼拘律園中。

**◈ 爾時，釋氏摩訶男來詣佛所，稽首佛足，退坐一面，白佛言：「世尊！云何名優婆塞？」**
　　那時，釋迦族的摩訶男來到佛陀這裡，恭敬地向佛陀的足下頂禮，然後退到一邊坐下，對佛陀恭敬地說道：「世尊！什麼樣的人才稱得上是優婆塞（在家男信徒）呢？」

**◈ 佛告摩訶男：「優婆塞者，在家清白，乃至盡壽，歸依三寶，為優婆塞，證知我！」**
　　佛陀告訴摩訶男：「所謂的優婆塞，是指在家庭生活中保持清淨不染污，直到生命結束都虔誠歸依佛、法、僧三寶的人。這樣的人才是優婆塞，請以此作為我的見證。」

**◈ 摩訶男白佛：「世尊！云何為滿足一切優婆塞事？」**
　　摩訶男對佛陀恭敬地說道：「世尊！如何才算圓滿地具足了一位優婆塞應有的所有條件？」

**◈ 佛告摩訶男：「若優婆塞有信無戒，是則不具，當勤方便，具足淨戒。」**
　　佛陀告訴摩訶男：「如果一位優婆塞雖然有正信（對覺悟的信任），卻沒有持戒（遵守道德規範），這就是不完備的，應當勤奮地尋找方法，使自己具足清淨的戒律。」

**◈ 「具足信、戒而不施者，是則不具；以不具故，精勤方便，修習布施，令其具足滿。」**
　　「如果具足了正信與戒律，卻不實踐布施（慷慨地給予），這也是不完備的；因為不完備，所以應當精進地尋找方法，修習布施，使其達到圓滿。」

**◈ 「信、戒、施滿，不能隨時往詣沙門，聽受正法，是則不具；以不具故，精勤方便，隨時往詣塔寺，見諸沙門，不一心聽受正法，是不具足。」**
　　「即便正信、戒律、布施都圓滿了，但如果不能隨時前往拜訪沙門（修行者），聽聞並接受正法（真理），這仍是不完備的；因為不完備，所以應當精進地尋找方法，隨時前往塔寺拜訪沙門；但如果見到沙門後，不能一心專注地聽受正法，這依然是不具足的。」

**◈ 「信、戒、施、聞修習滿足，聞已不持，是不具足；以不具足故，精勤方便，隨時往詣沙門，專心聽法，聞則能持。」**
　　「即便正信、戒律、布施、聽法都修習圓滿了，但聽完法後如果不實踐受持，這是不具足的；因為不具足，所以應當精進地尋找方法，隨時拜訪沙門，專心聽法，並且在聽聞後能將其落實在生活中。」

**◈ 「不能觀察諸法深義，是不具足；不具足故，精勤方便，信、戒、施、聞，聞則能持，持已，觀察甚深妙義，而不隨順知法次法向，是則不具，以不具故，精勤方便，信、戒、施、聞，受持觀察，了達深義，隨順行法次法向。」**
　　「如果不能觀察一切法（現象）的深層義理，這是不具足的；因為不具足，所以應當精進地尋找方法，在具足正信、戒律、布施、聽法且能受持之後，進一步觀察甚深的妙義；但如果觀察後，不能依照修行法門的次第（由淺入深的步驟）來實踐，這仍是不完備的。因此，應當精進地尋找方法，在具足正信、戒律、布施、聽法、受持與觀察之後，徹底了達深層義理，並隨順地按照修行次第來實踐。」

**◈ 「摩訶男！是名滿足一切種優婆塞事。」**
　　「摩訶男！這才叫做圓滿地具足了一位優婆塞的所有條件。」

**◈ 摩訶男白佛：「世尊！云何名優婆塞能自安慰，不安慰他？」**
　　摩訶男對佛陀恭敬地說道：「世尊！什麼樣的優婆塞叫做能讓自己獲得安寧，卻不能讓他人獲得安寧？」

**◈ 佛告摩訶男：「若優婆塞能自立戒，不能令他立於正戒；自持淨戒，不能令他持戒具足；自行布施，不能以施建立於他；自詣塔寺見諸沙門，不能勸他令詣塔寺往見沙門；自專聽法，不能勸人樂聽正法；聞法自持，不能令他受持正法；自能觀察甚深妙義，不能勸人令觀深義；自知深法能隨順行法次法向，不能勸人令隨順行法次法向。」**
　　佛陀告訴摩訶男：「如果一位優婆塞能為自己建立戒律，卻不能引導他人建立正戒；自己持守清淨戒律，卻不能讓他人也具足持戒；自己實踐布施，卻不能引導他人建立布施的精神；自己前往塔寺拜訪沙門，卻不能勸導他人也去拜訪沙門；自己專注聽法，卻不能勸人樂於聽聞正法；聽法後自己能受持，卻不能讓他人也受持正法；自己能觀察甚深的妙義，卻不能勸人觀察深義；自己了解深法並能隨順修行次第而實踐，卻不能勸人隨順修行次第而實踐。」

**◈ 「摩訶男！如是八法成就者，是名優婆塞能自安慰，不安慰他。」**
　　「摩訶男！如此成就這八項法的人，就叫做能讓自己獲得安寧，卻不能讓他人獲得安寧的優婆塞。」

**◈ 摩訶男白佛：「世尊！優婆塞成就幾法自安安他？」**
　　摩訶男對佛陀恭敬地說道：「世尊！優婆塞需要成就多少項法，才能達到既讓自己安寧，也能讓他人安寧？」

**◈ 佛告摩訶男：「若優婆塞成就十六法者，是名優婆塞自安安他。何等為十六？」**
　　佛陀告訴摩訶男：「如果一位優婆塞能成就十六項法，就叫做能讓自己安寧且能讓他人安寧。這十六項法是什麼呢？」

**◈ 「摩訶男！若優婆塞具足正信，建立他人；自持淨戒，亦以淨戒建立他人；自行布施，教人行施；自詣塔寺見諸沙門，亦教人往見諸沙門；自專聽法，亦教人聽；自受持法，教人受持；自觀察義，教人觀察；自知深義，隨順修行法次法向，亦復教人解了深義，隨順修行法次法向。」**
　　「摩訶男！如果一位優婆塞具足正信，並能引導他人建立正信；自己持守清淨戒律，也能引導他人建立淨戒；自己實踐布施，也能教導他人行施；自己前往塔寺拜訪沙門，也能教導他人前往拜訪沙門；自己專注聽法，也能教導他人聽法；自己受持正法，也能教導他人受持；自己觀察義理，也能教導他人觀察；自己了達深義並隨順修行次第而實踐，也能教導他人了解深義並隨順修行次第而實踐。」

**◈ 「摩訶男！如是十六法成就者，是名優婆塞能自安慰，亦安慰他人。」**
　　「摩訶男！如此成就這十六項法的人，就叫做能讓自己獲得安寧，也能讓他人獲得安寧的優婆塞。」

**◈ 「摩訶男！若優婆塞成就如是十六法者，彼諸大眾悉詣其所，謂婆羅門眾、剎利眾、長者眾、沙門眾，於諸眾中威德顯曜。」**
　　「摩訶男！如果一位優婆塞能成就這十六項法，那麼各階層的大眾都會聚集在他身邊，包括婆羅門、剎利耶、長者以及沙門，他在所有人群之中，其威德將會顯耀光芒。」

**◈ 「譬如日輪，初、中及後，光明顯照。如是，優婆塞十六法成就者，初、中及後，威德顯照。如是，摩訶男！若優婆塞十六法成就者，世間難得。」**
　　「就像太陽一樣，無論是早晨、中午還是傍晚，光芒都清晰地照耀著。同樣地，成就十六法的人，在任何時候其威德都顯耀照人。摩訶男！像這樣成就十六法的優婆塞，在世間是非常罕見的。」

**◈ 佛說此經已，釋氏摩訶男聞佛所說，歡喜隨喜，即從坐起，作禮而去。**
　　佛陀講完這部經後，釋迦族的摩訶男聽聞佛陀所說的教法，心生法喜並依照教導去實踐，隨即從座位上站起來，向佛陀行禮後才離開。

### [詞語註解]
1. **優婆塞 (Upāsaka)**：指在家居住且虔誠信仰佛陀的男信徒。其核心特質在於「歸依」與「清淨」。
2. **三寶 (Tiratana)**：指佛（覺悟者）、法（覺悟的真理）、僧（實踐真理的社群）。
3. **正信 (Saddhā)**：基於對真理的理解而產生的堅定信任，而非盲目崇拜。
4. **淨戒 (Sīla)**：指清淨的道德規範，旨在防止對他人及自己的傷害，為心靈安定奠定基礎。
5. **法次法向 (Dhamma-pariyāya/Sequence)**：指修行中由淺入深、循序漸進的階段。在原始佛教中，這通常指從基本的道德修養（戒）➔ 心靈專注（定）➔ 智慧洞察（慧）的過程。
6. **威德 (Teja/Puñña)**：指因修行而自然散發出的精神力量與人格魅力，能對他人產生正向的影響力。

### [現代實作指南]
本經文提供了一套極其清晰的「漸進式成長路徑」，可用於現代生活中的自我提升與利他實踐：

1. **建立基礎（信 → 戒 → 施）**：
   - **轉念技巧**：不要試圖直接追求「開悟」或「深義」。先從「信任一個正向的價值觀（信）」開始，接著設定自己的「底線原則（戒）」，最後透過「對他人的慷慨（施）」來打破自我中心。
   - **實作**：每天設定一個小小的道德底線（如：今天不說任何貶低他人的話），並在生活中實踐一次無條件的幫助。

2. **深化認知（聞 → 持 → 觀）**：
   - **實作**：將「學習」轉化為「實踐」。閱讀一本好書（聞）後，挑選其中一點在生活中執行一週（持），最後反思這一點如何改變自己的心態（觀）。

3. **從「自安」轉向「安他」**：
   - **轉念技巧**：真正的成功不是自己獲得平靜，而是能將平靜的「方法」傳遞給他人。
   - **實作**：當你發現某種心態調整方法（如：正念呼吸）能讓自己緩解壓力時，不要僅僅私藏，而是在適當的時機，以溫和的方式分享給同樣處於壓力中的同事或家人。

4. **遵循次第（法次法向）**：
   - **實作**：避免跳級。如果生活還處於混亂（缺乏戒），不要強求深層的禪定或哲學思考，應先將重心放在整理生活與修復人際關係上。

### [段落大意]
本經由摩訶男詢問優婆塞的定義，佛陀詳細闡述了在家信徒圓滿修行的七個漸進階段：從正信、持戒、布施，到聽法、受持、觀察深義，最後隨順修行次第。佛陀進一步區分了僅能「自安」的八法與能「自安安他」的十六法，強調將個人修行轉化為利他引導的重要性。成就十六法者如日輪般光芒四射，在世間極為罕見。
