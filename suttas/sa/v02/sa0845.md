# 雜阿含經 第 845 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若比丘於五恐怖怨對休息，三事決定，不生疑惑，如實知見賢聖正道，彼聖弟子能自記說：『地獄、畜生、餓鬼惡趣已盡，得須陀洹，不墮惡趣法，決定正向三菩提，七有天人往生，究竟苦邊。』

「何等為五恐怖怨對休息？若殺生因緣罪怨對恐怖生，若離殺生者，彼殺生罪怨對因緣生恐怖休息。若偷盜、邪婬、妄語、飲酒罪怨對因緣生恐怖；彼若離偷盜、邪婬、妄語、飲酒罪怨對者，因緣恐怖休息，是名罪怨對因緣生五恐怖休息。

「何等為三事決定，不生疑惑？謂於佛決定離於疑惑，於法、僧決定離疑惑，是名三法決定離疑惑。

「何等名為聖道如實知見？謂此苦聖諦如實知，此苦集聖諦、此苦滅聖諦、此苦滅道跡聖諦如實知，是名聖道如實知見。

「若於此五恐怖罪怨對休息、於三法決定離疑惑、於聖意如實知見，是聖弟子能自記說：『我地獄盡，畜生、餓鬼惡趣盡，得須陀洹，不墮惡趣法，決定正趣三菩提，七有天人往生，究竟苦邊。』」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若比丘於五恐怖怨對休息，三事決定，不生疑惑，如實知見賢聖正道，**
　　「如果比丘能夠讓由五種罪業（殺、盜、淫、妄、酒）所引起的恐懼與業報怨對平息，對三種事物（佛、法、僧）有堅定的信念而不產生疑惑，並且如實地觀察並認知到賢聖的正確解脫之道，

**◈ 彼聖弟子能自記說：『地獄、畜生、餓鬼惡趣已盡，得須陀洹，不墮惡趣法，決定正向三菩提，七有天人往生，究竟苦邊。』**
　　那麼這位聖弟子就能對自己做出預言說：『我已經終結了地獄、畜生、餓鬼這三種惡劣的輪迴去向，證得了須陀洹（入流果），不再墮入惡道，決定會正向著最高、最圓滿的智慧與覺悟 (阿耨多羅三藐三菩提) 前進，最多在七次天人或人類的轉世中，最終達到苦的終結（涅槃）。』

**◈ 「何等為五恐怖怨對休息？**
　　「什麼叫做五種恐怖怨對的平息？

**◈ 若殺生因緣罪怨對恐怖生，若離殺生者，彼殺生罪怨對因緣生恐怖休息。**
　　如果因為殺生的因緣而產生罪業與怨對，進而導致內心的恐懼；若能遠離殺生，那麼由殺生罪業所引起的恐懼就會平息。

**◈ 若偷盜、邪婬、妄語、飲酒罪怨對因緣生恐怖；彼若離偷盜、邪婬、妄語、飲酒罪怨對者，因緣恐怖休息，是名罪怨對因緣生五恐怖休息。**
　　同樣地，如果因為偷盜、邪婬、妄語、飲酒的因緣而產生罪業與怨對，進而導致恐懼；若能遠離偷盜、邪婬、妄語、飲酒，那麼由這些因緣引起的恐懼就會平息。這就叫做由罪業怨對而生的五種恐怖得到了平息。

**◈ 「何等為三事決定，不生疑惑？**
　　「什麼叫做對三件事決定且不生疑惑？

**◈ 謂於佛決定離於疑惑，於法、僧決定離疑惑，是名三法決定離疑惑。**
　　也就是對佛陀、對法（真理）、對僧伽（聖社）這三者有絕對的信心而遠離疑惑，這就叫做對三法決定且離於疑惑。

**◈ 「何等名為聖道如實知見？**
　　「什麼叫做對聖道如實地知見？

**◈ 謂此苦聖諦如實知，此苦集聖諦、此苦滅聖諦、此苦滅道跡聖諦如實知，是名聖道如實知見。**
　　也就是如實地認知到：這是苦聖諦（人生本質的苦）、這是苦集聖諦（苦的產生原因）、這是苦滅聖諦（苦的止息狀態）、以及這是苦滅道跡聖諦（導向苦止息的解脫之道），這就叫做對聖道的如實知見。

**◈ 「若於此五恐怖罪怨對休息、於三法決定離疑惑、於聖意如實知見，**
　　「如果能平息這五種罪業恐懼、對三法決定不疑、且對聖道如實知見，

**◈ 是聖弟子能自記說：『我地獄盡，畜生、餓鬼惡趣盡，得須陀洹，不墮惡趣法，決定正趣三菩提，七有天人往生，究竟苦邊。』」**
　　這位聖弟子就能自記預言說：『我的地獄業盡，畜生與餓鬼的惡趣也盡，證得了須陀洹，不再墮入惡道，決定正向著最高、最圓滿的智慧與覺悟 (阿耨多羅三藐三菩提) 前進，最多在七次天人或人類的轉世中，最終達到苦的終結。』」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **須陀洹 (Sotāpanna)**：意為「入流者」，是四果聖者中的第一階段。證得此果者已斷除對自我的執著（身見）、對儀軌的盲信（疑）以及對感官快感的執著（戒禁見），決定不再墮入三惡道，最多在七次輪迴內達成涅槃。
2. **三菩提 (Sammāsambodhi)**：即阿耨多羅三藐三菩提，指佛陀所證得的最高、最圓滿的覺悟，完全斷除一切煩惱且具足一切智慧。
3. **惡趣 (Apāya)**：指不幸福的生存狀態，通常指地獄、餓鬼、畜生三道，是受業報驅使而墮入的痛苦境界。
4. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四個真理：苦（Dukkha）、集（Samudaya）、滅（Nirodha）、道（Magga）。

### [現代實作指南]
本經文提供了一個從「行為 ➔ 心理 ➔ 認知」的階梯式解脫路徑，可用於現代生活中的心理重建：

1. **行為層面：消除內在恐懼的根源 (五恐怖休息)**
   - **實作**：審視自己的生活中是否有違背基本倫理的行為（如欺騙、傷害他人、過度沉溺於成癮物質）。
   - **轉念**：意識到許多潛在的焦慮、不安與恐懼（現代心理學中的內疚感或壓力），往往源於過去不誠實或傷害他人的行為。透過停止傷害（持戒），可以從根源上消除這種「業報式」的心理恐懼，獲得心靈的寧靜。

2. **心理層面：建立穩定的信任基石 (三事決定)**
   - **實作**：在混亂的資訊時代，尋找一個經過驗證的真理體系或導師，並在實踐中驗證其有效性。
   - **轉念**：將「盲信」轉化為「基於經驗的信心」。當你發現某種正向的生活方式（法）確實能減少你的痛苦時，這種決定性的信心會成為你在面對人生危機時的心理支柱。

3. **認知層面：建立對現實的精準分析 (聖道如實知見)**
   - **實作**：練習「四聖諦分析法」：
     - **苦**：承認目前的壓力或痛苦是真實存在的，不逃避。
     - **集**：分析這個痛苦是由於什麼執著或貪念引起的（例如：因為想要掌控他人而感到痛苦）。
     - **滅**：確認這種痛苦是可以消失的，想像一個不再被該執著掌控的狀態。
     - **道**：制定具體的行動計畫（如正念、正語）來逐步消除該執著。

### [段落大意]
本經闡述了聖弟子能預言自己不再墮入惡道並最終解脫的三個必要條件：首先是透過遠離五種惡行（殺、盜、淫、妄、酒）來平息內心的罪業恐懼；其次是對佛、法、僧三寶建立堅定的信心以消除疑惑；最後是對四聖諦有如實的認知。具備這三者的人即證得須陀洹果，決定在七次輪迴內達成最高覺悟並終結所有痛苦。
