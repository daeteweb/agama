# 雜阿含經 第 721 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「轉輪聖王出世之時，有七寶現於世間，金輪寶、象寶、馬寶、神珠寶、玉女寶、主藏臣寶、主兵臣寶。如是，如來出世，亦有七覺分寶現，齋戒處樓觀上，大臣圍遶，有金輪寶從東方出，輪有千輻，齊轂圓輞，輪相具足，有此吉瑞，必是轉輪聖王，我今決定為轉輪王。即以兩手承金輪寶，著左手中，右手旋轉，而說是言：『若是轉輪聖王金輪寶者，當復轉輪聖王古道而去。』於是輪寶即發，王蕃前隨，而於東方乘虛而逝，向於東方，遊古聖王正直之道。王隨輪寶，四兵亦從，若所至方，輪寶住者，王於彼住，四兵亦住。東方諸國處處小王，見聖王來，悉皆歸伏。

「如來出興於世，有七覺分現於世間，所謂念覺分、擇法覺分、精進覺分、喜覺分、猗覺分、定覺分、捨覺分。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「轉輪聖王出世之時，有七寶現於世間，金輪寶、象寶、馬寶、神珠寶、玉女寶、主藏臣寶、主兵臣寶。**
　　「當轉輪聖王（以正法統治世界的理想君主）出世之時，世間會顯現出七種寶物：金輪寶、象寶、馬寶、神珠寶、玉女寶、主藏臣寶（掌管國庫的大臣）以及主兵臣寶（掌管軍隊的大臣）。

**◈ 如是，如來出世，亦有七覺分寶現，**
　　同樣地，當如來（成就不變真理而來到世間的人）出世時，世間也會顯現出七種覺悟的寶藏（七覺分）。

**◈ 齋戒處樓觀上，大臣圍遶，有金輪寶從東方出，輪有千輻，齊轂圓輞，輪相具足，有此吉瑞，必是轉輪聖王，我今決定為轉輪王。**
　　在齋戒處的樓閣觀景台上，大臣們圍繞在側，此時有一個金輪寶從東方出現。這個輪子有千根輻條，輪轂與輪輞完整圓滿，所有徵兆都具足。看到這個吉祥的徵兆，就知道必定是轉輪聖王降臨，於是（王）決定承接轉輪王的使命。

**◈ 即以兩手承金輪寶，著左手中，右手旋轉，而說是言：『若是轉輪聖王金輪寶者，當復轉輪聖王古道而去。』**
　　他立即用兩手承接金輪寶，將其置於左手，右手使其旋轉，並說道：『如果這確實是轉輪聖王的金輪寶，就請依照古代聖王所走的正直道路前行。』

**◈ 於是輪寶即發，王蕃前隨，而於東方乘虛而逝，向於東方，遊古聖王正直之道。**
　　於是金輪寶立即啟程，聖王在後隨行，金輪寶在東方虛空中飛逝，向著東方前往古代聖王所行走的正直之道。

**◈ 王隨輪寶，四兵亦從，若所至方，輪寶住者，王於彼住，四兵亦住。**
　　聖王跟隨金輪寶，四方軍隊也隨之跟從。無論到達哪個地方，只要金輪寶停下來，聖王與四方軍隊就一同停留在該處。

**◈ 東方諸國處處小王，見聖王來，悉皆歸伏。」**
　　東方各國的小國之王，看到聖王到來，全部都心悅誠服地歸順。」

**◈ 「如來出興於世，有七覺分現於世間，所謂念覺分、擇法覺分、精進覺分、喜覺分、猗覺分、定覺分、捨覺分。」**
　　「如來出現在世間，同樣有七種覺悟的要素顯現於世，也就是：念覺分（正念的覺悟）、擇法覺分（分析法義的覺悟）、精進覺分（勤奮修行的覺悟）、喜覺分（法喜的覺悟）、猗覺分（身心安寧的覺悟）、定覺分（專注統一的覺悟）、以及捨覺分（平等心地的覺悟）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **轉輪聖王 (Cakravartin)**：佛教宇宙觀中最高等級的世俗統治者，不透過暴力，而是以正法（Dhamma）統治世界，其出現伴隨著七種象徵權威與德行的寶物。
2. **七覺分 (Satta Bojjhaṅga)**：指導致覺悟的七種心法要素。這是一種漸進的心理狀態，從正念開始，經過分析、努力、喜悅、安寧，最終達到專注與平靜。
3. **擇法 (Dhamma-vicaya)**：指對法（真理/現象）進行觀察、分析與區分，能辨別何為善、何為不善，何為導致解脫的因素，何為導致束縛的因素。
4. **猗 (Passaddhi)**：指身心的寧靜與放鬆，是消除焦躁與緊張後的平穩狀態，為進入深層定力（Samādhi）的必要前提。

### [現代實作指南]
本經以「轉輪聖王的七寶」比喻「如來的七覺分」，旨在說明精神上的覺悟比世俗的權力更具價值。在現代生活中，我們可以將「七覺分」轉化為一套心理調節系統：

1. **啟動正念 (念覺分)** ➔ 當面對壓力或衝突時，先將意識拉回當下，覺察自己的呼吸或身體感覺，停止自動導航的反應模式。
2. **理性分析 (擇法覺分)** ➔ 詢問自己：「目前的情緒是由什麼因緣而生的？」、「這個反應對解決問題有幫助嗎？」，將問題客觀化。
3. **採取行動 (精進覺分)** ➔ 在分析後，以不強求、不焦慮的態度，採取微小但持續的正面行動。
4. **正向回饋 (喜覺分)** ➔ 留意行動中產生的微小進步，給予自己正向的鼓勵，讓修行與工作變成一種喜悅而非苦役。
5. **適度放鬆 (猗覺分)** ➔ 在努力之後，刻意創造「留白」時間，讓緊繃的神經放鬆，避免過度精進導致的倦怠。
6. **深度專注 (定覺分)** ➔ 將心念單一化，在單一任務中達到心流狀態，減少心靈的碎片化。
7. **保持中立 (捨覺分)** ➔ 對於結果（成功或失敗）保持一種平等的態度，不被得失心牽著走，維持內心的穩定。

### [段落大意]
本經透過對比轉輪聖王出世時顯現的「七寶」與如來出世時顯現的「七覺分」，強調精神覺悟之寶貴。轉輪聖王以金輪寶引領世間歸伏，而如來則以七種覺悟要素（念、擇法、精進、喜、猗、定、捨）引領眾生走向解脫。
