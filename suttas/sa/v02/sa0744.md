# 雜阿含經 第 744 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若比丘修習慈心，多修習已，得大果大福利。云何比丘修習慈心，得大果大福利？是比丘心與慈俱，修念覺分，依遠離、依無欲、依滅、向於捨；乃至修習捨覺分，依遠離、依無欲、依滅、向於捨。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

---

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若比丘修習慈心，多修習已，得大果大福利。**
　　「如果比丘修行慈心（對一切眾生生起無條件的關懷與愛），且在長時間的修行之後，將會獲得巨大的果報與福德利益。

**◈ 云何比丘修習慈心，得大果大福利？**
　　比丘應該如何修行慈心，才能獲得巨大的果報與福德利益呢？

**◈ 是比丘心與慈俱，修念覺分，依遠離、依無欲、依滅、向於捨；**
　　這位比丘在心中保持著慈心，並修行念覺分（覺悟的要素之一：正念），這種修行是依據遠離（遠離煩惱與雜染）、依據無欲（消除對世間的貪欲）、依據滅（熄滅苦因），並導向捨（不執著的平等心）；

**◈ 乃至修習捨覺分，依遠離、依無欲、依滅、向於捨。」**
　　直到修行捨覺分（覺悟的要素之一：捨），同樣也是依據遠離、依據無欲、依據滅，並導向捨。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **慈心 (Mettā)**：一種希望所有眾生都能獲得快樂、遠離痛苦的無條件善意。
2. **覺分 (Bojjhaṅga)**：指導向覺悟（菩提）的七種心理要素，包括念、擇法、精進、喜、捨、定、正念。
3. **遠離 (Viveka)**：指在心理上脫離感官欲望的牽引，或在環境上遠離嘈雜，以獲得內心的寧靜。
4. **無欲 (Virāga)**：對對象失去貪著與執著的狀態，是解脫苦痛的關鍵。
5. **滅 (Nirodha)**：指貪、嗔、痴等煩惱的完全熄滅，進而終止苦的產生。
6. **捨 (Upekkhā)**：一種中立、平衡且不偏不倚的心理狀態，既不貪著也不排斥。

### [現代實作指南]
本經揭示了「慈心」與「覺悟要素」的結合路徑。在現代生活中，可採取以下步驟實踐：

1. **建立慈心基調**：在面對衝突或壓力時，先在心中默念：「願我與對方都能獲得平安，遠離痛苦。」將慈心作為修行的底色，避免在追求寧靜時陷入冷漠。
2. **將慈心轉化為正念 (念覺分)**：當慈心生起後，觀察這個「關懷」的心理狀態。意識到慈心如何讓心變得柔軟，並將注意力集中在當下的感受上，而非對結果的期待。
3. **練習心理遠離 (依遠離)**：在繁忙的日常中，刻意創造短暫的「心理空白期」。例如在兩場會議之間，花一分鐘放下所有對工作的執著，僅僅維持慈心與覺知。
4. **培養不執著的平等心 (向於捨)**：練習對待「喜歡的人」與「討厭的人」使用同樣的慈心。當你能以同樣的關懷看待不同類型的對象時，便是在實踐從「慈」走向「捨」的過程，最終達到心靈的絕對平衡。

### [段落大意]
本段經文說明修行慈心能帶來巨大的福德，但關鍵在於將慈心與覺悟要素（如念、捨）結合。修行者應以慈心為基礎，透過遠離煩惱、消除貪欲與熄滅苦因，最終達到不執著的平等心（捨），從而實現真正的覺悟。
