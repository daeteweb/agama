# 雜阿含經 第 711 經

### [原文]
如是我聞：

一時，佛住王舍城耆闍崛山中。

時，有無畏王子，日日步涉，仿佯遊行，來詣佛所，與世尊面相問訊慰勞已，退坐一面，白佛言：「世尊！有沙門、婆羅門作如是見，作如是說：『無因、無緣眾生煩惱，無因、無緣眾生清淨。』世尊復云何？」

佛告無畏：「沙門、婆羅門為其說，不思而說，愚癡、不辨、不善，非知思，不知量，作如是說：『無因、無緣眾生煩惱，無因、無緣眾生清淨。』所以者何？有因、有緣眾生煩惱，有因、有緣眾生清淨故。何因、何緣眾生煩惱？何因、何緣眾生清淨？謂眾生貪欲增上，於他財物、他眾具而起貪言：『此物於我有者，好、不離、愛樂。』於他眾生而起恨心、兇心，計挍、欲打、欲縛、欲伏，加諸不道。為造眾難，不捨瞋恚，身睡眠、心懈怠、心掉動，內不寂靜，心常疑惑，過去疑、未來疑、現在疑。無畏。如是因、如是緣眾生煩惱。如是因、如是緣眾生清淨。」

無畏白佛：「瞿曇！一分之蓋，足煩惱心，況復一切？」無畏白佛：「瞿曇！何因、何緣眾生清淨？」

佛告無畏：「若婆羅門有一勝念，決定成就，久時所作，久時所說，能隨憶念，當於爾時習念覺支。修念覺已，念覺滿足，念覺滿足已，則於選擇分別思惟，爾時擇法覺支修習。修擇法覺支已，擇法覺支滿足，彼選擇分別思量法已，則精進方便。精進覺支於此修習，修精進覺支已，精進覺支滿足，彼精進方便已，則歡喜生，離諸食想。修喜覺支，修喜覺支已，則喜覺支滿足，喜覺支滿足已，身心猗息，則修猗覺支。修猗覺支已，猗覺滿足，身猗息已，則愛樂，愛樂已心定，則修定覺支。修定覺支已，定覺滿足，定覺滿足已，貪憂滅，則捨心生，修捨覺支。修捨覺支已，捨覺支滿足。如是，無畏！此因、此緣眾生清淨。」

無畏白瞿曇：「若一分滿足，令眾生清淨，況復一切？」無畏白佛：「瞿曇！當何名此經？云何奉持？」

佛告無畏王子：「當名此為覺支經。」

無畏白佛：「瞿曇！此為最勝覺分。瞿曇！我是王子，安樂亦常求安樂，而希出入，今來上山，四體疲極，得聞瞿曇說覺支經，悉忘疲勞。」

佛說此已，王子無畏聞佛所說，歡喜隨喜，從座起，稽首禮佛足而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城耆闍崛山中。**
　　在某一個時間，佛陀住在王舍城的耆闍崛山中。

**◈ 時，有無畏王子，日日步涉，仿佯遊行，來詣佛所，與世尊面相問訊慰勞已，退坐一面，白佛言：**
　　那時，有一位名叫無畏的王子，每天步行前往，隨意遊行，來到佛陀這裡。在與世尊面對面地問候並表達關心之後，他退到一邊坐下，對佛陀恭敬地說道：

**◈ 「世尊！有沙門、婆羅門作如是見，作如是說：『無因、無緣眾生煩惱，無因、無緣眾生清淨。』世尊復云何？」**
　　「世尊！有些沙門（修行者）與婆羅門持有這樣的見解，並這樣說：『眾生的煩惱（內心的染污與執著）是沒有原因、沒有條件而產生的；眾生的清淨（擺脫染污的狀態）也是沒有原因、沒有條件而達成的。』關於這一點，世尊您的看法如何？」

**◈ 佛告無畏：「沙門、婆羅門為其說，不思而說，愚癡、不辨、不善，非知思，不知量，作如是說：『無因、無緣眾生煩惱，無因、無緣眾生清淨。』**
　　佛陀告訴無畏：「那些沙門與婆羅門在說這些話時，是不經過思考就隨口而出的，他們愚癡、無法分辨是非且不善於分析。他們並非基於對真理的認知與思考，也不懂得衡量法義的分寸，才如此宣稱：『眾生的煩惱與清淨都是無因、無緣的。』」

**◈ 所以者何？有因、有緣眾生煩惱，有因、有緣眾生清淨故。**
　　為什麼這麼說呢？因為眾生的煩惱（內心的染污與執著）是有原因、有條件而產生的；而眾生的清淨（擺脫染污的狀態）同樣是有原因、有條件而達成的。

**◈ 何因、何緣眾生煩惱？何因、何緣眾生清淨？**
　　那麼，是什麼原因、什麼條件導致了眾生的煩惱？又是什麼原因、什麼條件帶來了眾生的清淨？

**◈ 謂眾生貪欲增上，於他財物、他眾具而起貪言：『此物於我有者，好、不離、愛樂。』**
　　也就是說，當眾生的貪欲（對感官對象的渴求）增加，對他人的財物或器具產生貪念，心想：『如果這個東西屬於我，那就太好了，我絕對不會離開它，我非常愛樂它。』

**◈ 於他眾生而起恨心、兇心，計挍、欲打、欲縛、欲伏，加諸不道。**
　　對其他眾生產生恨意與兇狠之心，心裡盤算著、想要毆打、想要束縛、想要壓制，並採取不公正的手段。

**◈ 為造眾難，不捨瞋恚，身睡眠、心懈怠、心掉動，內不寂靜，心常疑惑，過去疑、未來疑、現在疑。**
　　為他人製造困難，不捨棄瞋恚（憤怒與厭惡的情緒），身體在睡眠中懈怠，心靈懶散且躁動不安，內心無法獲得寂靜，心中經常充滿疑惑 ➔ 對過去懷疑、對未來懷疑、對現在懷疑。

**◈ 無畏。如是因、如是緣眾生煩惱。如是因、如是緣眾生清淨。」**
　　無畏，這些就是導致眾生煩惱的原因與條件。而相對地，也有對應的原因與條件能讓眾生獲得清淨。」

**◈ 無畏白佛：「瞿曇！一分之蓋，足煩惱心，況復一切？」**
　　無畏對佛陀說：「瞿曇！僅僅其中一部分的遮蔽（蓋著心靈的煩惱），就足以讓心充滿煩惱，更不用說全部的煩惱了。」

**◈ 無畏白佛：「瞿曇！何因、何緣眾生清淨？」**
　　無畏接著問佛陀：「瞿曇！那麼是什麼原因、什麼條件能讓眾生獲得清淨呢？」

**◈ 佛告無畏：「若婆羅門有一勝念，決定成就，久時所作，久時所說，能隨憶念，當於爾時習念覺支。**
　　佛陀告訴無畏：「如果一個人擁有一種卓越的念頭，並決定要成就它，經過長期的實踐與長期的宣說，能夠隨時憶起，這時他便在修習『念覺支』（以正念覺醒的要素）。」

**◈ 修念覺已，念覺滿足，念覺滿足已，則於選擇分別思惟，爾時擇法覺支修習。**
　　在修習念覺支之後，當念覺支達到滿足（圓滿）時，他便能進行選擇性的分辨與思惟，這時就是在修習『擇法覺支』（能分辨真偽法義的覺醒要素）。

**◈ 修擇法覺支已，擇法覺支滿足，彼選擇分別思量法已，則精進方便。**
　　在修習擇法覺支之後，當擇法覺支達到滿足時，他在對法義進行選擇性的思量後，便會採取精進的方便手段。

**◈ 精進覺支於此修習，修精進覺支已，精進覺支滿足，彼精進方便已，則歡喜生，離諸食想。**
　　這時他就在修習『精進覺支』（努力不懈的覺醒要素）。在修習精進覺支之後，當精進覺支達到滿足時，他便會產生歡喜心，並脫離對食物（感官欲望）的執著。

**◈ 修喜覺支，修喜覺支已，則喜覺支滿足，喜覺支滿足已，身心猗息，則修猗覺支。**
　　這時他就在修習『喜覺支』（法喜的覺醒要素）。在修習喜覺支之後，當喜覺支達到滿足時，身心感到安適與放鬆，接著便修習『猗覺支』（心身安適的覺醒要素）。

**◈ 修猗覺支已，猗覺滿足，身猗息已，則愛樂，愛樂已心定，則修定覺支。**
　　在修習猗覺支之後，當猗覺支達到滿足，身體完全安適後，便會產生愛樂（對法義的喜愛），心靈隨之安定，這時便在修習『定覺支』（專注不亂的覺醒要素）。

**◈ 修定覺支已，定覺滿足，定覺滿足已，貪憂滅，則捨心生，修捨覺支。**
　　在修習定覺支之後，當定覺支達到滿足時，貪欲與憂愁都滅除了，隨之產生捨心（平等心），這時便在修習『捨覺支』（平靜捨離的覺醒要素）。

**◈ 修捨覺支已，捨覺支滿足。如是，無畏！此因、此緣眾生清淨。」**
　　在修習捨覺支之後，當捨覺支達到滿足。無畏！這就是讓眾生獲得清淨的原因與條件。」

**◈ 無畏白瞿曇：「若一分滿足，令眾生清淨，況復一切？」**
　　無畏對瞿曇說：「如果僅僅其中一部分的覺支滿足，就能讓眾生獲得清淨，更不用說全部的覺支都滿足了。」

**◈ 無畏白佛：「瞿曇！當何名此經？云何奉持？」**
　　無畏問佛陀：「瞿曇！這部經應該如何命名？應該如何虔誠地接受並依照教導去實踐？」

**◈ 佛告無畏王子：「當名此為覺支經。」**
　　佛陀告訴無畏王子：「這部經應命名為《覺支經》。」

**◈ 無畏白佛：「瞿曇！此為最勝覺分。瞿曇！我是王子，安樂亦常求安樂，而希出入，今來上山，四體疲極，得聞瞿曇說覺支經，悉忘疲勞。」**
　　無畏對佛陀說：「瞿曇！這是最殊勝的覺悟要素。瞿曇！我身為王子，雖然生活安樂，但也經常追求更高的安樂，因此希望能出入修行。今天來到山上，四肢極其疲憊，但聽完您所說的《覺支經》，所有的疲勞都忘卻了。」

**◈ 佛說此已，王子無畏聞佛所說，歡喜隨喜，從座起，稽首禮佛足而去。**
　　佛陀講完這部經後，王子無畏聽聞佛陀所說的教法，心生法喜並依照教導去實踐，從座位上起身，向佛陀的足部頂禮後才離開。

### [詞語註解]
1. **煩惱 (Kilesa)**：指染污心靈的心理狀態，如貪、瞋、癡，使人無法看清真理而陷入痛苦。
2. **清淨 (Visuddhi)**：指心靈擺脫了煩惱的染污，恢復到純淨、明覺且不被執著牽引的狀態。
3. **緣起 (Paṭicca-samuppāda)**：指一切現象皆由條件（緣）而生，沒有無因之果。本經強調煩惱與清淨皆有其特定的因緣。
4. **覺支 (Bojjhaṅga)**：意為「覺悟的要素」，指導致覺悟的七種心理品質：
    - **念 (Sati)**：正念，對當下經驗的清晰覺知。
    - **擇法 (Dhamma-vicaya)**：對法義進行分析與分辨，區分善惡與真偽。
    - **精進 (Viriya)**：為了達到目標而持續努力的心理能量。
    - **喜 (Pīti)**：修行過程中產生的法喜，一種輕盈且愉悅的狀態。
    - **猗 (Passaddhi)**：心身安適、寧靜，消除躁動。
    - **定 (Samādhi)**：心靈專注於單一對象，不散亂。
    - **捨 (Upekkhā)**：對所有經驗保持中立、平等且平靜的態度。

### [現代實作指南]
本經提供了一套從「煩惱」轉向「清淨」的邏輯路徑。在現代生活中，我們可以將其轉化為以下心理實踐步驟：

1. **識別煩惱的「因緣」**：
   - 當你感到憤怒或焦慮時，不要僅僅感受情緒，而要像佛陀教導的那樣，分析其原因。
   - 詢問自己：我是否在起「貪心」（想要掌控某物）？是否在起「恨心」（對他人不滿）？是否處於「疑惑」之中？
   - **轉念技巧**：將「我很生氣」轉化為「現在心中產生了瞋恚的因緣」。

2. **啟動「覺支」的遞進修習**：
   - **第一步：正念 (Mindfulness)** ➔ 意識到目前的情緒狀態，不評判，僅僅觀察。
   - **第二步：擇法 (Investigation)** ➔ 分析這個情緒是從哪裡來的？它對我是否有益？
   - **第三步：精進 (Effort)** ➔ 決定放下這個負面情緒，將注意力轉向正面的思考或呼吸。
   - **第四步：法喜 (Joy)** ➔ 當意識到自己能掌控情緒時，感受那份解脫的喜悅。
   - **第五步：安適 (Tranquility)** ➔ 隨著喜悅而來，讓身體的緊繃感放鬆。
   - **第六步：專注 (Concentration)** ➔ 將心安定在這種平靜的狀態中。
   - **第七步：平等 (Equanimity)** ➔ 最終達到一種無論外界如何變動，內心都能保持平靜且不偏頗的狀態。

3. **對抗「無因論」的迷思**：
   - 避免陷入「我天生就這樣」或「這就是命」的消極想法（即經中提到的「無因、無緣」）。
   - 堅信只要改變「因」（修習覺支），就一定能改變「果」（獲得清淨）。

### [段落大意]
本經透過無畏王子與佛陀的對話，反駁了「煩惱與清淨是無因而生」的錯誤見解。佛陀明確指出，煩惱源於貪欲、瞋恚、懈怠與疑惑等負面因緣；而清淨則源於對「七覺支」（念、擇法、精進、喜、猗、定、捨）的循序漸進之修習。當這七種覺悟要素依次圓滿時，眾生即可從染污轉為清淨，最終獲得解脫。
