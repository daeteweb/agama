# 雜阿含經 第 919 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，世尊告諸比丘：「世間有三種良馬。何等為三？謂有馬捷疾具足、非色具足、非形體具足。有馬捷疾具足、色具足、非形體具足。有馬捷疾具足、色具足、形體具足。如是於此法、律有三種善男子。何等為三？有善男子捷疾具足、非色具足、非形體具足，有善男子捷疾具足、色具足、非形體具足，有善男子捷疾具足、色具足、形體具足。

「何等為善男子捷疾具足、非色具足、非形體具足？謂善男子此苦聖諦如實知，此苦集聖諦如實知，此苦滅聖諦如實知，此苦滅道跡聖諦如實知。如是知、如是見已，欲有漏心解脫、有有漏心解脫、無明有漏心解脫：『我生已盡，梵行已立，所作已作，自知不受後有。』是名捷疾具足。云何非色具足？若有問阿毘曇、律，乃至不能為決定解說，是名色不具足。何等非形體具足？謂非名聞大德，乃至不感湯藥、眾具，是名形體不具足，是名善男子捷疾具足、非色具足、非形體具足。

「何等為善男子捷疾具足、色具足、非形體具足？謂善男子此苦聖諦如實知，乃至不受後有，是名捷疾具足。云何色具足？謂若有問阿毘曇、毘尼，乃至能為決定解說，是名色具足。何等為非形體具足？謂非名聞大德，乃至不能感湯藥、眾具，是名善男子捷疾具足、色具足、非形體具足。

「何等為善男子捷疾具足、色具足、形體具足？謂善男子此苦聖諦如實知，乃至不受後有，是名捷疾具足。何等為色具足？謂善男子若有問阿毘曇、毘尼，乃至能為決定解說，是名色具足。何等為形體具足？謂善男子名聞大德，乃至能感湯藥、眾具，是名形體具足，是名善男子捷疾具足、色具足、形體具足。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，世尊告諸比丘：「世間有三種良馬。何等為三？**
　　那時，世尊告訴在場的眾位比丘說：「世間有三種優良的馬。所謂的三種是指什麼呢？

**◈ 謂有馬捷疾具足、非色具足、非形體具足。**
　　第一種馬是：速度快（捷疾），但毛色不亮麗（非色），且體格不健壯（非形體）。

**◈ 有馬捷疾具足、色具足、非形體具足。**
　　第二種馬是：速度快（捷疾），毛色亮麗（色），但體格不健壯（非形體）。

**◈ 有馬捷疾具足、色具足、形體具足。**
　　第三種馬是：速度快（捷疾），毛色亮麗（色），且體格健壯（形體）。

**◈ 如是於此法、律有三種善男子。何等為三？**
　　同樣地，在修習佛法與律法（戒律）的修行者中，也有三種善男子。所謂的三種是指什麼呢？

**◈ 有善男子捷疾具足、非色具足、非形體具足，有善男子捷疾具足、色具足、非形體具足，有善男子捷疾具足、色具足、形體具足。**
　　第一種善男子是：修行進展迅速，但缺乏教法知識，且沒有名聲；第二種是：修行進展迅速，精通教法知識，但沒有名聲；第三種是：修行進展迅速，精通教法知識，且名聲顯赫。

**◈ 「何等為善男子捷疾具足、非色具足、非形體具足？**
　　「什麼樣的善男子叫做『修行進展迅速、缺乏知識、沒有名聲』呢？

**◈ 謂善男子此苦聖諦如實知，此苦集聖諦如實知，此苦滅聖諦如實知，此苦滅道跡聖諦如實知。**
　　是指這位善男子能如實地知道苦聖諦（關於生命本質是苦的真理）、集聖諦（關於苦之原因的真理）、滅聖諦（關於苦之止息的真理），以及道聖諦（關於導向苦止之解脫之道的真理）。

**◈ 如是知、如是見已，欲有漏心解脫、有有漏心解脫、無明有漏心解脫：『我生已盡，梵行已立，所作已作，自知不受後有。』是名捷疾具足。**
　　在如此地認知與見證之後，他渴望讓心中充滿煩惱的『有漏心』獲得解脫，並且確實讓有漏心與由無明（對真理的無知）引起的有漏心獲得解脫，體悟到：『我的輪迴出生已經結束，清淨的梵行已經建立，應該做的工作已經完成，自知不再經歷後世的出生（涅槃）』。這就叫做修行進展迅速（捷疾具足）。

**◈ 云何非色具足？若有問阿毘曇、律，乃至不能為決定解說，是名色不具足。**
　　為什麼說他缺乏知識（非色具足）？因為如果有人詢問他關於阿毘曇（深奧的佛法理論）或律（戒律規範），他無法給出明確且決定性的解釋，這就叫做知識不具足。

**◈ 何等非形體具足？謂非名聞大德，乃至不感湯藥、眾具，是名形體不具足，是名善男子捷疾具足、非色具足、非形體具足。**
　　什麼叫做沒有名聲（非形體具足）？是指他並非名聲顯赫的大德，因此無法吸引他人供養湯藥或各種生活必需品，這就叫做形體不具足。這就是所謂的『修行進展迅速、缺乏知識、沒有名聲』的善男子。

**◈ 「何等為善男子捷疾具足、色具足、非形體具足？**
　　「什麼樣的善男子叫做『修行進展迅速、精通知識、沒有名聲』呢？

**◈ 謂善男子此苦聖諦如實知，乃至不受後有，是名捷疾具足。**
　　是指這位善男子能如實知道苦聖諦（及其他三聖諦），直到不再經歷後世的出生，這就叫做修行進展迅速。

**◈ 云何色具足？謂若有問阿毘曇、毘尼，乃至能為決定解說，是名色具足。**
　　為什麼說他精通知識（色具足）？因為如果有人詢問他關於阿毘曇或毘尼（戒律），他能夠給出明確且決定性的解釋，這就叫做知識具足。

**◈ 何等為非形體具足？謂非名聞大德，乃至不能感湯藥、眾具，是名善男子捷疾具足、色具足、非形體具足。**
　　什麼叫做沒有名聲（非形體具足）？是指他並非名聲顯赫的大德，因此無法吸引他人供養湯藥或各種生活必需品。這就是所謂的『修行進展迅速、精通知識、沒有名聲』的善男子。

**◈ 「何等為善男子捷疾具足、色具足、形體具足？**
　　「什麼樣的善男子叫做『修行進展迅速、精通知識、名聲顯赫』呢？

**◈ 謂善男子此苦聖諦如實知，乃至不受後有，是名捷疾具足。**
　　是指這位善男子能如實知道苦聖諦（及其他三聖諦），直到不再經歷後世的出生，這就叫做修行進展迅速。

**◈ 何等為色具足？謂善男子若有問阿毘曇、毘尼，乃至能為決定解說，是名色具足。**
　　什麼叫做精通知識（色具足）？是指這位善男子若被詢問關於阿毘曇或毘尼（戒律），能夠給出明確且決定性的解釋，這就叫做知識具足。

**◈ 何等為形體具足？謂善男子名聞大德，乃至能感湯藥、眾具，是名形體具足，是名善男子捷疾具足、色具足、形體具足。」**
　　什麼叫做名聲顯赫（形體具足）？是指這位善男子被譽為名聞遐邇的大德，因此能感召他人供養湯藥或各種生活必需品，這就叫做形體具足。這就是所謂的『修行進展迅速、精通知識、名聲顯赫』的善男子。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四個真理，包含苦（生命本質的不滿足）、集（苦的根源是貪愛與無明）、滅（苦的完全止息即涅槃）、道（達成止息的八正道）。
2. **有漏 (Āsava)**：指心中仍有煩惱（如貪、嗔、痴）的狀態，如同容器有漏洞般，使眾生不斷在輪迴中流轉。
3. **阿毘曇 (Abhidhamma)**：對佛法教義的系統化、分析性的論述，旨在深入剖析心法與物質的本質。
4. **毘尼/律 (Vinaya)**：佛陀為僧團制定的戒律與生活規範，旨在維持團體和諧並輔助個人修行。
5. **梵行 (Brahmacariya)**：指清淨的修行生活，遠離世俗慾望，追求覺悟的聖潔生活。

### [現代實作指南]
本經以「良馬」比喻修行者的三種特質，給予現代人關於「能力、知識、名聲」優先順序的深刻啟示：

1. **確立核心目標（捷疾 ➔ 實踐）**：
   - 佛陀將「捷疾」（對四聖諦的體悟與解脫）放在首位。在現代生活中，這提醒我們：**「實踐的結果」遠比「理論的掌握」重要**。
   - 實作：不要陷入「閱讀大量心靈書籍」或「學習各種心理學理論」的陷阱（這僅是「色具足」），而應將重點放在觀察當下的苦、分析其原因，並嘗試放下執著。

2. **區分知識與智慧（色 ➔ 知識）**：
   - 「色具足」是指能對阿毘曇與律進行決定性的解說，這屬於「學術能力」。
   - 實作：意識到「知道什麼」不等於「成為什麼」。如果你能流暢地解釋什麼是「無我」，但面對損失時依然憤怒，那麼你僅僅是「色具足」而非「捷疾具足」。

3. **對待名聲與外部認可（形體 ➔ 名聲）**：
   - 「形體具足」是指名聲顯赫而獲得供養。佛陀將其列為最後一項，且在前兩種修行者中，即便沒有名聲，只要有「捷疾」依然是良馬。
   - 實作：在追求自我成長時，減少對「他人認可」、「社交媒體讚美」或「專業頭銜」的依賴。將外部的成功（形體）視為修行的副產品，而非目的。

4. **自我評估矩陣**：
   - 試著問自己：我現在是處於哪種狀態？
     - 僅有知識 → 缺乏實踐的學者。
     - 僅有名聲 → 缺乏內在實力的表演者。
     - 有實踐但無知識 → 樸實的修行者（依然是良馬）。
     - 三者兼具 → 理想的導師。

### [段落大意]
佛陀以三種良馬（速度、毛色、體格）比喻修行者的三種特質：對四聖諦的體悟與解脫（捷疾）、對教法知識的精通（色）、以及名聲顯赫受人供養（形體）。佛陀強調，無論是否精通理論或是否名聲顯赫，只要能如實知四聖諦並獲得解脫，即為最核心的「捷疾具足」，是修行者最優先應追求的目標。
