# 雜阿含經 第 685 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「譬如嬰兒，父母生已，付其乳母，隨時摩拭，隨時沐浴，隨時乳哺，隨時消息。若乳母不謹慎者，兒或以草、以土諸不淨物著其口中，乳母當即教令除去。能時除却者善，兒不能自却者，乳母當以左手持其頭，右手探其哽，嬰兒當時雖苦，乳母要當苦探其哽，為欲令其子長夜安樂故。」

佛告諸比丘：「若嬰兒長大，有所識別，復持草、土諸不淨物著口中不？」

比丘白佛：「不也，世尊！嬰兒長大，有所別知，尚不以脚觸諸不淨物，況著口中。」

佛告比丘：「嬰兒小時，乳母隨時料理消息，及其長大，智慧成就，乳母放捨，不勤消息，以其長大不自放逸故。如是，比丘！若諸聲聞始學，智慧未足，如來以法隨時教授而消息之；若久學智慧深固，如來放捨，不復隨時慇懃教授，以其智慧成就不放逸故。是故，聲聞五種學力，如來成就十種智力……」如上廣說。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「譬如嬰兒，父母生已，付其乳母，隨時摩拭，隨時沐浴，隨時乳哺，隨時消息。**
　　那時，世尊告訴在場的眾位比丘說：「就像一個嬰兒，父母在生下孩子後，將其交給乳母照顧，乳母隨時為他擦拭身體、隨時幫他洗澡、隨時餵他喝奶，並且隨時照顧他的起居生活。

**◈ 若乳母不謹慎者，兒或以草、以土諸不淨物著其口中，乳母當即教令除去。**
　　如果乳母不夠謹慎，嬰兒可能會把草、土或其他不潔淨的物品放進口中，乳母應該立刻教他將其除去。

**◈ 能時除却者善，兒不能自却者，乳母當以左手持其頭，右手探其哽，嬰兒當時雖苦，乳母要當苦探其哽，為欲令其子長夜安樂故。」**
　　如果孩子能及時吐出來那是最好的；但如果孩子無法自行吐出，乳母就必須用左手固定住孩子的頭，用右手探入他的喉嚨。嬰兒在當時雖然會感到痛苦，但乳母必須忍心採取這種痛苦的手段去探其喉嚨，這是為了讓孩子在長久的未來能夠獲得安樂。」

**◈ 佛告諸比丘：「若嬰兒長大，有所識別，復持草、土諸不淨物著口中不？」**
　　佛陀問諸比丘：「如果嬰兒長大了，有了分辨能力，還會把草、土等不潔淨的物品放進口中嗎？」

**◈ 比丘白佛：「不也，世尊！嬰兒長大，有所別知，尚不以脚觸諸不淨物，況著口中。」**
　　比丘對佛陀恭敬地說道：「不會的，世尊！嬰兒長大後有了分辨能力，甚至連腳都不會去觸碰不潔淨的物品，更不用說放進口中了。」

**◈ 佛告比丘：「嬰兒小時，乳母隨時料理消息，及其長大，智慧成就，乳母放捨，不勤消息，以其長大不自放逸故。**
　　佛陀告訴比丘：「嬰兒小時，乳母必須隨時料理他的起居；等到他長大、智慧成就後，乳母就可以放手，不再需要如此勤勞地照顧，因為他長大後不再會隨意放縱自己，能保持不自放逸（對生活與行為保持警覺）。

**◈ 如是，比丘！若諸聲聞始學，智慧未足，如來以法隨時教授而消息之；**
　　同樣地，比丘們！如果那些聲聞（聽聞佛法的人）剛開始學習，智慧還不足夠，如來會以法（真理）隨時地教授並照顧他們。

**◈ 若久學智慧深固，如來放捨，不復隨時慇懃教授，以其智慧成就不放逸故。**
　　但如果學習時間久了，智慧變得深厚穩固，如來就會放手，不再需要隨時如此慇懃地教授，因為他們的智慧已經成就，且能保持不放逸（對修行保持警覺）。

**◈ 是故，聲聞五種學力，如來成就十種智力……」如上廣說。**
　　因此，聲聞具備五種學習能力，而如來則成就了十種智力……」佛陀就這樣廣泛地說明。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **聲聞 (Sāvaka)**：指聽聞佛法而修行的人，主要指透過聽聞教導而進入修行路徑的弟子。
2. **不放逸 (Appamāda)**：指對修行保持高度的警覺與勤奮，不讓心隨意漂蕩於貪、嗔、癡等煩惱之中。
3. **學力 (Indriya/Bala)**：指修行者在學習與實踐法義時所具備的心理能力或根基。
4. **智力 (Ñāṇa-bala)**：指如來（佛陀）所成就的全面且深厚的覺悟能力，能精準地根據不同對象的根基給予適當的教導。

### [現代實作指南]
本經文揭示了「因材施教」與「漸進修行」的心理機制，可轉化為以下生活實踐：

1. **接納「初學者的笨拙」**：
   當我們在學習新技能或改變舊習慣時，就像經文中的嬰兒，容易將「不淨物」（錯誤的認知、舊有的慣性）放入口中。此時，不要苛責自己的失敗，而應尋找可靠的導師或建立嚴格的檢查機制（如同乳母的照顧），在錯誤發生時立即修正。

2. **理解「必要的痛苦」**：
   乳母探其哽雖令嬰兒痛苦，卻是為了長遠的安樂。在心理轉念上，當我們在修行或學習中遇到挫折、被指出錯誤而感到不適時，應將此「當下的痛苦」視為「清除錯誤認知」的必要過程，而非對個人的否定。

3. **建立「自律的轉折點」**：
   修行與成長的目標是從「依賴外部監督」轉向「內在自律」。
   ➔ **第一階段（依賴期）**：設定明確的規則，尋求外部反饋，嚴格執行。
   ➔ **第二階段（內化期）**：將外部規則轉化為內在的覺知（不放逸）。
   ➔ **第三階段（獨立期）**：當智慧（分辨力）成就後，不再需要外部鞭策，能自然地避開不淨之物。

4. **對他人的慈悲指導**：
   若處於指導者位置，應區分對方的「成熟度」。對初學者需慇懃、細膩且及時地糾錯；對成熟者則應給予空間，讓其在自律中深化智慧，避免過度干預導致對方失去獨立思考的能力。

### [段落大意]
佛陀以乳母照顧嬰兒為喻，說明教導弟子的原則：嬰兒小時缺乏分辨力，需乳母隨時照顧並在必要時採取強硬手段除去口中雜物，以確保長遠安樂；長大後則能自律。同樣地，初學聲聞智慧不足，需如來細心指導與糾錯；而久學且智慧深固者，則能保持不放逸，不再需要如此頻繁的教授。這說明了如來根據弟子根基而採取不同教法（因材施教）的智慧。
