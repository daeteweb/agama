# 雜阿含經 第 623 經

### [原文]
如是我聞：

一時，佛住波羅奈仙人住處鹿野苑中。

爾時，世尊告諸比丘：「世間言美色，世間美色者，能令多人集聚觀看者不？」

諸比丘白佛：「如是，世尊！」

佛告比丘：「若世間美色，世間美色者，又能種種歌舞伎樂，復極令多眾聚集看不？」

比丘白佛：「如是，世尊！」

佛告比丘：「若有世間美色，世間美色者，在於一處，作種種歌舞伎樂戲笑，復有大眾雲集一處，若有士夫不愚不癡，樂樂背苦，貪生畏死，有人語言：『士夫，汝當持滿油鉢，於世間美色者所及大眾中過，使一能殺人者，拔刀隨汝，若失一渧油者，輒當斬汝命。』云何？比丘！彼持油鉢士夫能不念油鉢，不念殺人者，觀彼伎女及大眾不？」

比丘白佛：「不也，世尊！所以者何？世尊！彼士夫自見其後有拔刀者，常作是念：『我若落油一渧，彼拔刀者當截我頭。』唯一其心，繫念油鉢，於世間美色及大眾中徐步而過，不敢顧眄。」

「如是，比丘！若有沙門、婆羅門正身自重，一其心念，不顧聲色，善攝一切心法，住身念處者，則是我弟子，隨我教者。云何為比丘正身自重，一其心念，不顧聲色，攝持一切心法，住身念處。如是，比丘！身身觀念，精勤方便，正智正念，調伏世間貪憂，受、心、法法觀念住亦復如是。是名比丘正身自重，一其心念，不顧聲色，善攝心法，住四念處。」

爾時，世尊即說偈言：

「專心正念，　　護持油鉢， 
　自心隨護，　　未曾至方， 
　甚難得過，　　勝妙微細， 
　諸佛所說，　　言教利劍。 
　當一其心，　　專精護持， 
　非彼凡人，　　放逸之事， 
　能入如是，　　不放逸教。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：一時，佛住波羅奈仙人住處鹿野苑中。**
　　我是這樣聽佛陀說的：在某一個時間，佛陀住在波羅奈（今瓦拉納西）仙人居住地的鹿野苑中。

**◈ 爾時，世尊告諸比丘：「世間言美色，世間美色者，能令多人集聚觀看者不？」**
　　那時，世尊告訴在場的眾位比丘說：「世間所說的美色（物質身體的吸引力），這樣的美色者，是否能讓許多人聚集在一起觀看呢？」

**◈ 諸比丘白佛：「如是，世尊！」**
　　眾位比丘對佛陀恭敬地說道：「確實如此，世尊！」

**◈ 佛告比丘：「若世間美色，世間美色者，又能種種歌舞伎樂，復極令多眾聚集看不？」**
　　佛陀告訴比丘：「如果世間的美色者，還能表演各種歌舞與音樂，是否會讓更多的人聚集觀看呢？」

**◈ 比丘白佛：「如是，世尊！」**
　　比丘對佛陀恭敬地說道：「確實如此，世尊！」

**◈ 佛告比丘：「若有世間美色，世間美色者，在於一處，作種種歌舞伎樂戲笑，復有大眾雲集一處，若有士夫不愚不癡，樂樂背苦，貪生畏死，有人語言：『士夫，汝當持滿油鉢，於世間美色者所及大眾中過，使一能殺人者，拔刀隨汝，若失一渧油者，輒當斬汝命。』云何？比丘！彼持油鉢士夫能不念油鉢，不念殺人者，觀彼伎女及大眾不？」**
　　佛陀告訴比丘：「假設有一位世間的美色者，在某個地方表演各種歌舞伎樂並嬉笑，吸引了大眾雲集。此時，若有一位不愚笨、不癡迷，懂得追求快樂並遠離痛苦，且貪生畏死的士夫，有人對他說：『士夫，你應該拿著一個裝滿油的缽，走在美色者與聚集的大眾之中，且有一名能殺人的人拔刀跟在你身後。如果你掉了一滴油，他立刻就會斬斷你的生命。』比丘！你們認為，這位拿著油缽的士夫，是否能不去在意油缽、不去在意那個殺人者，而專心觀看那位舞女與大眾呢？」

**◈ 比丘白佛：「不也，世尊！所以者何？世尊！彼士夫自見其後有拔刀者，常作是念：『我若落油一渧，彼拔刀者當截我頭。』唯一其心，繫念油鉢，於世間美色及大眾中徐步而過，不敢顧眄。」**
　　比丘對佛陀恭敬地說道：「不會的，世尊！為什麼呢？世尊！因為那位士夫看到身後有拔刀的人，心中會不斷地想：『如果我掉了一滴油，那個拔刀的人就會砍掉我的頭。』他的心會完全繫留在對油缽的關注上，在美色者與大眾之中緩緩走過，不敢向左右顧盼。」

**◈ 「如是，比丘！若有沙門、婆羅門正身自重，一其心念，不顧聲色，善攝一切心法，住身念處者，則是我弟子，隨我教者。」**
　　「確實如此，比丘！如果有沙門或婆羅門能夠端正身行且自重，將心專注於一處，不被外界的聲音與色彩（色聲香味觸法）所吸引，善於攝持（控制與觀照）一切心的法（心理現象），並安住在身念處（對身體的覺知）之中，他就是我的弟子，是遵循我教導的人。」

**◈ 「云何為比丘正身自重，一其心念，不顧聲色，攝持一切心法，住身念處。如是，比丘！身身觀念，精勤方便，正智正念，調伏世間貪憂，受、心、法法觀念住亦復如是。是名比丘正身自重，一其心念，不顧聲色，善攝心法，住四念處。」**
　　「比丘，什麼叫做端正身行、自重，並將心專注於一處，不顧聲色，攝持一切心法，安住在身念處呢？比丘！這就是對身體進行觀念（覺知），精勤地運用方便法門，以正確的智慧與正念，調伏世間的貪欲與憂慮；對於受（感受）、心（意識）、法（心理現象）的觀念與安住也是同樣的道理。這就叫做比丘端正身行、自重，將心專注於一處，不顧聲色，善於攝持心法，安住在四念處（身、受、心、法四種覺知基礎）之中。」

**◈ 爾時，世尊即說偈言：「專心正念，護持油鉢，自心隨護，未曾至方，甚難得過，勝妙微細，諸佛所說，言教利劍。當一其心，專精護持，非彼凡人，放逸之事，能入如是，不放逸教。」**
　　那時，世尊隨即說偈頌道：「專注於正念，如同護持油缽一般，同樣地護持自心，不讓心隨意漂移。這種專注極其難得，且勝妙微細。諸佛所說的教法，如同利劍一般（能斬斷煩惱）。應當將心專注於一處，精勤地護持，這不是凡夫所能做到的放逸之事，唯有不放逸（精勤不懈）的人，才能進入這樣的教導之中。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **身念處 (Kāyānupassanā)**：四念處之首，指對身體的覺知，包括呼吸、姿勢、身體的組成以及身體的活動，旨在體悟色（物質身體）的無常與無我。
2. **四念處 (Satipaṭṭhāna)**：佛教修行中建立正念的四個基礎，分別是：身（身體）、受（感受）、心（意識狀態）、法（心理現象/真理）。
3. **不放逸 (Appamāda)**：指修行者保持警覺，不懈怠，時刻維持正念，防止心念漂移至貪、嗔、癡的對象上。
4. **攝心 (Saṃvara)**：指對感官的制約與心的調伏，防止心被外界的色、聲、香、味、觸等誘惑所牽引。

### [現代實作指南]
本經以「持油缽過美色人群」為隱喻，揭示了「高度專注」與「生存危機感」之間的關係。在現代生活中，我們常被海量的資訊（美色、娛樂、社交媒體）分散注意力，導致修行或目標無法達成。

**實作步驟：**
1. **建立「心理油缽」**：
   ➔ 定義你目前生命中最核心的目標或當下的覺知對象（例如：呼吸、工作任務、或對當下情緒的觀察）。將其視為那個「裝滿油的缽」。
2. **設定「生存危機感」**：
   ➔ 意識到「放逸」的代價。如同油缽掉一滴油會被斬首，意識到每一次對雜念的隨順，都是在消耗生命、增加輪迴的苦楚。將「失去正念」視為一種精神上的損失。
3. **練習「徐步而過」**：
   ➔ 當你在嘈雜環境或面對誘惑（如手機通知、他人評價）時，提醒自己：「我正持著油缽走在人群中」。不強行對抗誘惑，而是將注意力重新錨定在「油缽」（當下的覺知）上。
4. **四念處的日常切換**：
   ➔ **身**：感覺腳掌接觸地面的壓力（走在人群中）。
   ➔ **受**：覺察到被吸引的「渴望感」或「不安感」。
   ➔ **心**：覺察到心念正在漂移。
   ➔ **法**：體悟到「吸引」與「被吸引」皆是因緣而生的現象，並非真實的我。

### 段落大意 
佛陀以一名持滿油缽、後有持刀者隨行的士夫為喻，說明在強大誘惑（美色歌舞）面前，若有強烈的危機感與目標感，人能完全專注於當下而不受干擾。以此勉勵修行者應如持油般精勤護持心念，透過四念處（身、受、心、法）的修行，調伏貪憂，不放逸地實踐正法。
