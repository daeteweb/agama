# 雜阿含經 第 610 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「我當說修四念處。諦聽，善思。云何修四念處？謂內身身觀念住，精勤方便，正智正念，調伏世間憂悲，外身、內外身觀住，精勤方便，正念正知，調伏世間憂悲。如是受、心、法，內法、外法、內外法觀念住，精勤方便，正念正知，調伏世間憂悲，是名比丘修四念處。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

過去、未來修四念處亦如是說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「我當說修四念處。諦聽，善思。**
　　那時，世尊告訴在場的眾位比丘說：「我現在要為你們說明如何修行四念處（四種正念的確立）。請仔細聽，好好思考。

**◈ 云何修四念處？**
　　什麼叫做修行四念處呢？

**◈ 謂內身身觀念住，精勤方便，正智正念，調伏世間憂悲，**
　　指的是對內在的色（物質身體）進行觀察並將正念維持在其中，透過精進的努力與適當的方法，運用正確的智慧與正念，來調伏並消除世間的憂愁與悲傷；

**◈ 外身、內外身觀住，精勤方便，正念正知，調伏世間憂悲。**
　　同樣地，對外在的色（物質身體），以及內外兩者的色（物質身體）進行觀察並維持正念，透過精進的努力與適當的方法，運用正念與正確的知覺，來調伏並消除世間的憂愁與悲傷。

**◈ 如是受、心、法，內法、外法、內外法觀念住，精勤方便，正念正知，調伏世間憂悲，**
　　同樣地，對於受（感受）、心（意識狀態）、法（心理現象），不論是內在的法、外在的法，或是內外兩者的法，都進行觀察並將正念維持在其中，透過精進的努力與適當的方法，運用正念與正確的知覺，來調伏並消除世間的憂愁與悲傷，

**◈ 是名比丘修四念處。」**
　　比丘們，這就叫做修行四念處。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 過去、未來修四念處亦如是說。**
　　無論在過去或未來，關於修行四念處的教導也都是這樣說的。

### [詞語註解]
1. **四念處 (Satipaṭṭhāna)**：指修行者確立正念的四個對象，分別是：身（物質身體）、受（感受）、心（意識狀態）、法（心理現象/萬法本質）。
2. **正念 (Sammā-sati)**：八正道之一，指對當下發生之現象保持清醒的覺知，不隨之起舞，亦不壓抑，僅僅是客觀地觀察。
3. **正知 (Sammā-sampajañña)**：在正念的基礎上，清楚地知道目前正在做什麼、為什麼做，以及該行為是否符合正法。
4. **調伏 (Damana)**：指透過覺知與智慧，將躁動、不安或染污的心態（如憂悲）平息下來，使其回歸平靜與清明。

### [現代實作指南]
本經文提供了一套完整的「身心監控系統」，可將其轉化為現代生活中的壓力管理與情緒調節技巧：

1. **身體掃描（身念處）** ➔ 當感到焦慮時，將注意力從思考轉移到身體。觀察呼吸的起伏，或感受腳掌接觸地面的壓力。意識到「身體」僅僅是色（物質身體）的組合，而非「我」，能有效降低對壓力的生理反應。
2. **標記感受（受念處）** ➔ 當強烈情緒湧現時，不要說「我很悲傷」，而是在心中標記：「現在有一個『悲傷的感受』產生了」。將自己從感受中抽離，成為觀察者，能防止情緒失控。
3. **覺察心態（心念處）** ➔ 觀察當下的心念狀態。例如：「現在心裡有貪念」、「現在心裡有憤怒」。僅僅是觀察心念的生滅，而不對其進行評判。
4. **分析模式（法念處）** ➔ 思考這個情緒是如何產生的？（例如：因為對某事的執著 ➔ 導致失望 ➔ 產生憂悲）。透過分析因緣而生的過程，體悟到所有心理現象皆是無常且非永恆的。
