# 雜阿含經 第 603 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「云何度諸流？　　云何度大海？ 　云何能捨苦？　　云何得清淨？」

爾時，世尊即說偈言：

「信能度諸流，　　不放逸度海， 　精進能除苦，　　智慧得清淨。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。**
　　那時，有一位容貌極其出色的天子，在深夜時分來到佛陀所在的地方，頂禮佛陀的足下，然後退到一邊坐下。

**◈ 身諸光明遍照祇樹給孤獨園。**
　　他的身體散發出各種光明，照亮了整個祇樹給孤獨園。

**◈ 時，彼天子說偈問佛：**
　　那時，那位天子用偈頌（詩歌形式）請問佛陀：

**◈ 「云何度諸流？**
　　「如何才能度過（跨越）種種（由貪愛與執著構成的）煩惱河流？」

**◈ 云何度大海？**
　　「如何才能度過（跨越）輪迴（Saṃsāra）的苦海？」

**◈ 云何能捨苦？**
　　「如何才能捨棄苦（Dukkha，指生命中不滿足與不安的狀態）？」

**◈ 云何得清淨？」**
　　「如何才能獲得清淨（指心靈遠離染污、恢復純淨的狀態）？」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊立刻用偈頌回答道：

**◈ 「信能度諸流，**
　　「透過信（對正法與覺悟之道的堅定信心）能度過種種煩惱河流，」

**◈ 不放逸度海，**
　　「透過不放逸（時刻警覺、不懈怠的修行）能度過輪迴苦海，」

**◈ 精進能除苦，**
　　「透過精進（持續不斷地努力修行）能除去苦（Dukkha），」

**◈ 智慧得清淨。」**
　　「透過智慧（對真理與無常的洞察力）能獲得清淨。」

**◈ 時，彼天子復說偈言：**
　　那時，那位天子再次說偈頌道：

**◈ 「久見婆羅門，**
　　「我長期以來觀察那些婆羅門（指追求真理的修行者），」

**◈ 逮得般涅槃，**
　　「直到他們達到了涅槃（超越生死、絕對寧靜的解脫境界），」

**◈ 一切怖已過，**
　　「所有的恐懼都已經過去了，」

**◈ 永超世恩愛。」**
　　「永遠超越了世間的恩愛（指對世俗情感的執著）。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，那位天子聽聞佛陀所說的教法，心中感到歡喜並隨喜（認同並共鳴），頂禮佛陀的足下，隨即消失不再出現。

### [詞語註解]
1. **諸流 (Ogha)**：指淹沒眾生的「洪水」，在原始佛教中通常指欲流、有流、慢流、見流，即貪欲、存在欲、傲慢與錯誤見解。
2. **不放逸 (Appamāda)**：指修行者對心念保持高度的警覺，不讓心在懈怠中迷失，是所有修行路徑的核心。
3. **清淨 (Visuddhi)**：指心靈不再被貪、嗔、癡等染污物（Kilesa）所遮蔽，恢復到本然的清明狀態。
4. **世恩愛 (Samsārika-pema)**：指對世間親情、愛情或物質享受的依戀。這種愛雖美，但在佛教視角中是導致輪迴與苦痛的執著之源。

### [現代實作指南]
本經文提供了一套從「信心」到「智慧」的漸進式心理轉化路徑，可用於現代壓力管理與自我成長：

1. **建立正向信心 (信 ➔ 度流)**：
   - **實作**：當面對情緒崩潰（煩惱河流）時，首先建立一個信念：「目前的痛苦是可被轉化的」。不要陷入「我永遠如此」的絕望，而是相信存在一種能讓心平靜的方法。

2. **保持覺察不懈怠 (不放逸 ➔ 度海)**：
   - **實作**：在日常生活中練習「微覺察」。例如，在感到憤怒的一瞬間，立刻意識到「我現在正在生氣」，而不是被憤怒牽著走。這種不放逸的警覺能防止我們被情緒的苦海淹沒。

3. **採取微小而持續的行動 (精進 ➔ 除苦)**：
   - **實作**：不要追求一次性的巨大改變，而要追求「持續的微小努力」。例如，每天僅花 5 分鐘練習深呼吸或正念觀察，用這種持續的精進來逐步消解長期的壓力（苦）。

4. **分析痛苦的本質 (智慧 ➔ 清淨)**：
   - **實作**：當執著於某件失去的事物時，運用智慧分析：這件事是否在變動（無常）？它是否真的屬於我（無我）？透過理性的分析，將「我的痛苦」轉化為「對痛苦現象的觀察」，從而獲得心靈的清淨與釋然。

### 段落大意 
一位天子深夜造訪佛陀，詢問如何跨越煩惱、脫離苦海並獲得清淨。佛陀開示需依序透過「信心」、「不放逸」、「精進」與「智慧」四項修行來達成。天子對此深感認同，感嘆覺悟者能超越恐懼與世俗執著，隨後歡喜消失。
