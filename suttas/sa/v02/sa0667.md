# 雜阿含經 第 667 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有四力。何等為四？謂信力、精進力、念力、慧力。復次四力——信力、念力、定力、慧力。復次四力——覺力、精進力、無罪力、攝力。」

此諸經如上三力說。差別者：「何等為覺力？於善、不善法如實知，有罪、無罪，習近、不習近，卑法、勝法，黑法、白法，有分別法、無分別法，緣起法、非緣起法如實知，是名覺力。何等為精進力？謂四正斷……」如前廣說。「何等為無罪力？謂無罪身、口、意，是名無罪力。何等為攝力？謂四攝事，惠施、愛語、行利、同利。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「有四力。何等為四？**
　　那時，世尊告訴在場的眾位比丘說：「修行中有四種力量。這四種力量是什麼呢？

**◈ 謂信力、精進力、念力、慧力。**
　　指的是：信力（對正法的堅定信心）、精進力（不斷努力修行以斷除煩惱的能量）、念力（對當下覺知且不忘失的專注力）、以及慧力（洞察無常與無我的智慧）。

**◈ 復次四力——信力、念力、定力、慧力。**
　　另外，還有另外四種力量：信力（對正法的堅定信心）、念力（對當下覺知且不忘失的專注力）、定力（心不散亂、專注於一境的安定力）、以及慧力（洞察無常與無我的智慧）。

**◈ 復次四力——覺力、精進力、無罪力、攝力。」**
　　再者，還有第三組四種力量：覺力（如實觀察現象本質的覺知力）、精進力（不斷努力修行以斷除煩惱的能量）、無罪力（身口意三業清淨、無過失的力量）、以及攝力（能攝受眾生、使其心悅誠服的感化力）。」

**◈ 此諸經如上三力說。差別者：**
　　在這些經文中，上述三組力量的定義有所不同。具體區分如下：

**◈ 「何等為覺力？於善、不善法如實知，有罪、無罪，習近、不習近，卑法、勝法，黑法、白法，有分別法、無分別法，緣起法、非緣起法如實知，是名覺力。**
　　「什麼是覺力？是指能夠如實地知道哪些法（現象）是善的、哪些是不善的；哪些是有罪的、哪些是無罪的；哪些是應該親近的、哪些是不應該親近的；哪些是卑下的法、哪些是勝上的法；哪些是黑法（染污法）、哪些是白法（清淨法）；哪些是有分別的法、哪些是無分別的法；以及哪些是緣起法（因緣而生的現象）、哪些是非緣起法。能如實地知曉這些，就稱為覺力。

**◈ 何等為精進力？謂四正斷……」如前廣說。**
　　什麼是精進力？指的是四正斷（防止未生之惡、斷除已生之惡、生起未生之善、增長已生之善）……（這部分如前文詳細說明）。

**◈ 「何等為無罪力？謂無罪身、口、意，是名無罪力。**
　　什麼是無罪力？指的是在身體行為、言語表達、以及內心意念這三方面都沒有造作過失，這就稱為無罪力。

**◈ 何等為攝力？謂四攝事，惠施、愛語、行利、同利。」**
　　什麼是攝力？指的是四種攝受眾生的方法：惠施（慷慨布施）、愛語（溫和親切的言語）、行利（採取對他人有利的行動）、以及同利（與他人共享利益）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **四正斷 (Sammappadhāna)**：正精進的具體操作，包含：(1) 止惡：防止尚未產生的不善法不再產生；(2) 斷惡：將已經產生的不善法予以根除；(3) 止善：使尚未產生的善法得以產生；(4) 增善：使已經產生的善法更加增長。
2. **緣起法 (Paṭicca-samuppāda)**：指一切現象皆是由特定條件（因緣）組合而生，沒有獨立永恆的實體。
3. **四攝法 (Sangaha-vatthu)**：佛法中用於建立良好人際關係與感化他人的四種社交技巧，旨在透過慈悲心將眾生攝受於正法之中。
4. **黑法/白法**：黑法指導致痛苦、染污心靈的惡法（如貪嗔痴）；白法指帶來清淨、導向解脫的善法。

### [現代實作指南]
本經文提供了三組「力量」模型，可用於不同階段的心理建設與行為調整：

1. **基礎心態建設（信 → 念 → 定 → 慧）**：
   - **實作**：當面對生活壓力感到迷茫時，先建立「信」（相信問題有解且正法有效），接著練習「念」（觀察當下情緒而不被捲入），透過「定」（將注意力集中在呼吸或單一任務）來穩定心神，最後產生「慧」（分析問題的根源與無常本質），從而找到解決方案。

2. **認知判別訓練（覺力實踐）**：
   - **實作**：在做決定前，進行「如實知」的掃描。問自己：這個念頭是「善」還是「不善」？這個行為是「卑法」（損人利己）還是「勝法」（利他利己）？透過這種二元對比的覺察，將直覺反應轉化為理智選擇。

3. **人際關係優化（攝力實踐）**：
   - **實作**：在職場或家庭中應用「四攝」。
     - **惠施** ➔ 分享知識或提供實質幫助。
     - **愛語** ➔ 停止指責，使用鼓勵與肯定之詞。
     - **行利** ➔ 主動承擔能為團隊帶來好處的雜事。
     - **同利** ➔ 成功後不獨佔功勞，與夥伴共同分享成果。

### [段落大意]
本經詳細說明了三組不同的「四力」修行體系。第一組強調信、精進、念、慧的基礎組合；第二組將精進替換為定力，強調心心的安定；第三組則側重於覺知（覺力）、實踐（精進力）、道德清淨（無罪力）與社會感化（攝力）。其中重點定義了「覺力」為對善惡、緣起等法相的如實辨識，以及「攝力」為利他的人際互動方法。
