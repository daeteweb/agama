# 雜阿含經 第 847 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有四種諸天天道，未淨眾生令淨，已淨者重令淨。何等為四？謂聖弟子於佛不壞淨，於法、僧不壞淨，聖戒成就，是名四種諸天天道，未淨眾生令淨，已淨者重令淨。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「有四種諸天天道，未淨眾生令淨，已淨者重令淨。**
　　「有四種通往天界的道路（天道），能夠讓尚未清淨（未脫離煩惱）的眾生變得清淨，也能讓已經清淨的人更加清淨。

**◈ 何等為四？**
　　這四種道路是什麼呢？

**◈ 謂聖弟子於佛不壞淨，於法、僧不壞淨，聖戒成就，**
　　指的是聖弟子對於佛陀保持清淨的信心而不使其毀壞，對於法（真理）與僧伽（僧團）保持清淨的信心而不使其毀壞，並且圓滿地成就聖者的戒律（聖戒）。

**◈ 是名四種諸天天道，未淨眾生令淨，已淨者重令淨。」**
　　這就稱為四種通往天界的道路，能讓尚未清淨的眾生變得清淨，也能讓已經清淨的人更加清淨。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **天道 (Deva-magga)**：在此處並非僅指死後投生天界，而是指一種高尚、清淨的生命狀態與心靈路徑。
2. **淨 (Suddhi)**：指心靈脫離貪、嗔、癡等煩惱（Kilesas）的狀態，達到純淨、無染的覺知。
3. **聖戒 (Āriya-sīla)**：聖者所持的戒律，不僅是形式上的禁條，更是基於對真理的深刻理解而自然生起的道德自律。
4. **不壞淨 (Asuddha/Pure)**：指對三寶（佛、法、僧）的信心堅定且純粹，不被懷疑、傲慢或錯誤的見解所污染。

### [現代實作指南]
本經強調的是「清淨心」的遞進式修習。在現代生活中，可將其轉化為以下實踐步驟：

1. **建立正向的信任基石（對佛、法、僧不壞淨）**：
   ➔ **實作**：在面對生活壓力或精神危機時，不再採取盲目的懷疑或憤怒，而是將注意力轉向對真理（法）的探索。練習在心中建立一個「信任錨點」，相信生命中存在著覺悟的可能性，以此穩定情緒。

2. **從「未淨」到「令淨」的轉念**：
   ➔ **實作**：當意識到自己產生貪婪或憤怒（未淨）時，不要立即自我譴責，而是將其視為「需要被淨化」的對象。透過觀察呼吸或正念，意識到煩惱的生起 ➔ 承認其存在 ➔ 讓其自然消散，這便是「令淨」的過程。

3. **追求「重令淨」的精進**：
   ➔ **實作**：即使目前生活穩定、道德良好（已淨），也不應陷入自滿。嘗試將道德標準從「不傷害他人」提升到「主動利益他人」，將基本的自律提升為對生命的深切慈悲，實現心靈品質的再次升級。

4. **將「聖戒」轉化為生活準則**：
   ➔ **實作**：設定每日的「清淨清單」。例如：今日不說謊、不批評他人、不貪圖小利。將戒律視為保護心靈清淨的「防護牆」，而非限制自由的枷鎖。

### [段落大意]
本經闡述了四種能使眾生由染污轉為清淨的「天道」，即對佛、法、僧三寶保持純淨的信心，以及圓滿聖戒。這四者能讓未清淨者獲得清淨，並讓已清淨者進一步昇華。
