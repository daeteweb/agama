# 雜阿含經 第 963 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

時，有婆蹉種出家來詣佛所，與世尊面相問訊慰勞已，退坐一面，白佛言：「瞿曇！彼云何無知故，作如是見、如是說：『世間常，此是真實，餘則虛妄，世間無常、世間常無常、世間非常非無常，世有邊、世無邊、世有邊無邊、世非有邊非無邊，命即是身、命異身異，如來有後死、無後死、有無後死、非有非無後死』？」

佛告婆蹉：「於色無知故，作如是見、如是說：『世間常，此是真實，餘則虛妄，乃至非有非無後死。』於受、想、行、識無知故，作如是見、如是說：『世間常，此是真實，餘則虛妄，乃至非有非無後死。』」

婆蹉白佛：「瞿曇！知何法故，不如是見、如是說：『世間常，此是真實，餘則虛妄，乃至非有非無後死。』」

佛告婆蹉：「知色故，不如是見、如是說：『世間常，此是真實，餘則虛妄，乃至非有非無後死。』知受、想、行、識故，不作如是見、如是說，世間常，此是真實，餘則虛妄，乃至非有非無後死，如是不知、知。如是不見、見，不識、識，不斷、斷，不觀、觀，不察、察，不覺、覺。」

佛說此經已，婆蹉種出家聞佛所說，歡喜隨喜，從坐起而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 時，有婆蹉種出家來詣佛所，與世尊面相問訊慰勞已，退坐一面，白佛言：**
　　那時，有一位婆蹉種（某種種姓或教派）的出家修行者來到佛陀這裡，與世尊面相問訊慰勞之後，退到一邊坐下，對佛陀恭敬地說道：

**◈ 「瞿曇！彼云何無知故，作如是見、如是說：『世間常，此是真實，餘則虛妄，世間無常、世間常無常、世間非常非無常，世有邊、世無邊、世有邊無邊、世非有邊非無邊，命即是身、命異身異，如來有後死、無後死、有無後死、非有非無後死』？」**
　　「瞿曇！為什麼有些人因為缺乏對真理的認知，而產生這樣的見解並這樣說：『世界是恆常不變的，這才是真實的，其餘的都是虛假的；或者說世界是無常的、世界既恆常又無常、世界既非恆常也非無常；或者說世界是有邊界的、世界沒有邊界的、世界既有邊界又無邊界、世界既非有邊界也非無邊界；或者說生命就是這個物質身體、生命與物質身體是截然不同的；甚至討論如來在死後存在、死後不存在、死後既存在又不存在、或死後既非存在也非不存在』？」

**◈ 佛告婆蹉：「於色無知故，作如是見、如是說：『世間常，此是真實，餘則虛妄，乃至非有非無後死。』**
　　佛陀告訴婆蹉：「因為他們對於色（物質身體）缺乏認知，所以才會產生這樣的見解並這樣說：『世界是恆常不變的，這才是真實的，其餘的都是虛假的，乃至於討論如來死後既非存在也非不存在。』」

**◈ 於受、想、行、識無知故，作如是見、如是說：『世間常，此是真實，餘則虛妄，乃至非有非無後死。』」**
　　「同樣地，因為他們對於受（感受）、想（想像與認知）、行（心的造作）、識（意識）缺乏認知，所以才會產生這樣的見解並這樣說：『世界是恆常不變的，這才是真實的，其餘的都是虛假的，乃至於討論如來死後既非存在也非不存在。』」

**◈ 婆蹉白佛：「瞿曇！知何法故，不如是見、如是說：『世間常，此是真實，餘則虛妄，乃至非有非無後死。』」**
　　婆蹉對佛陀恭敬地說道：「瞿曇！應當認知什麼樣的法（真理），才不會產生這樣的見解並這樣說：『世界是恆常不變的，這才是真實的，其餘的都是虛假的，乃至於討論如來死後既非存在也非不存在』？」

**◈ 佛告婆蹉：「知色故，不如是見、如是說：『世間常，此是真實，餘則虛妄，乃至非有非無後死。』**
　　佛陀告訴婆蹉：「因為認知了色（物質身體）的本質，所以才不會產生這樣的見解並這樣說：『世界是恆常不變的，這才是真實的，其餘的都是虛假的，乃至於討論如來死後既非存在也非不存在。』」

**◈ 知受、想、行、識故，不作如是見、如是說，世間常，此是真實，餘則虛妄，乃至非有非無後死，**
　　「因為認知了受（感受）、想（想像與認知）、行（心的造作）、識（意識）的本質，所以才不會產生這樣的見解並這樣說：世界是恆常不變的，這才是真實的，其餘的都是虛假的，乃至於討論如來死後既非存在也非不存在。」

**◈ 如是不知、知。如是不見、見，不識、識，不斷、斷，不觀、觀，不察、察，不覺、覺。」**
　　「就像這樣，不知與知、不見與見、不識與識、不斷與斷、不觀與觀、不察與察、不覺與覺之間，存在著決定性的差異。」

**◈ 佛說此經已，婆蹉種出家聞佛所說，歡喜隨喜，從坐起而去。**
　　佛陀講完這部經後，婆蹉種出家聽聞佛陀所說的教法，心生法喜並依照教導去實踐，從座位上站起來離開了。

### [詞語註解]
1. **五蘊 (Pañca-kkhandha)**：構成生命個體的五種要素，包括色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）。本經強調，對五蘊的無知是產生所有錯誤世界觀（如常恆或斷滅）的根源。
2. **色 (Rūpa)**：指物質性的存在，在人身上即是指肉體與生理機能。
3. **受 (Vedanā)**：對刺激產生的情緒反應，分為苦、樂、不苦不樂。
4. **想 (Saññā)**：對對象的辨識、標記與概念化認知。
5. **行 (Saṅkhāra)**：心靈的意志造作，是產生業力的心理活動。
6. **識 (Viññāṇa)**：對對象的覺知與分辨能力，是意識的基礎。
7. **如來 (Tathāgata)**：佛陀的自稱，意指「如實地來到世間」或「如實地離開世間」的覺悟者。

### [現代實作指南]
本經揭示了一個核心邏輯：**當我們對「自我」與「世界」的組成要素（五蘊）缺乏微觀的認知時，我們傾向於用「宏觀的哲學假設」來填補空白。** 例如，當我們感到不安時，會幻想有一個「恆常的靈魂」或「永恆的世界」來尋求安全感。

**實作步驟：**
1. **停止哲學爭論**：當你陷入「人生是否有意義」、「死後會如何」或「世界是否公平」等無法用邏輯證明且導致焦慮的思考時，意識到這正是經中所說的「無知故作如是見」。
2. **將焦點轉向「五蘊」觀察**：
   - **觀察色**：當身體感到疲憊或疼痛時，僅觀察「物質身體」的感覺，而不是思考「為什麼我這麼命苦」。
   - **觀察受**：當憤怒或悲傷升起時，標記為「現在有一個『受（感受）』在運作」，而不是認為「我是一個憤怒的人」。
   - **觀察想**：當大腦開始對事情做定義（例如：這件事很糟糕）時，意識到這只是「想（認知）」在標記，而非絕對事實。
   - **觀察行**：察覺內心想要反擊或逃避的衝動，這是「行（心的造作）」。
   - **觀察識**：意識到目前意識正在接收視覺或聽覺訊息，這是「識（意識）」。
3. **轉念技巧**：將「世界是怎樣的？」這個大問題，轉化為「此刻我的五蘊在如何運作？」這個小觀察。透過對組成要素的精準認知，那些關於恆常或無常的極端執著會自然消融。

### [段落大意]
本經記錄了婆蹉種出家詢問佛陀，為何人們會對世界的恆常性、邊界以及如來的死後狀態產生各種矛盾且極端的見解。佛陀指出，所有這些錯誤的見解皆源於對「五蘊」（色、受、想、行、識）的無知。反之，只要能正確認知五蘊的本質，便能超越這些虛妄的哲學爭論，獲得真正的覺悟。
