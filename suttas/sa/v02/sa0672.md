# 雜阿含經 第 672 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有四力——覺力、精進力、無罪力、攝力。何等為覺力？謂慧、大慧、深慧、難勝慧，是名覺力。何等為精進力？若於不善法不善數、黑黑數、有罪有罪數、不應親近不應親近數，離此諸法已，若諸餘善善數、白白數、無罪無罪數、應親近應親近數，如此等修習，增上精勤、欲、方便、堪能，正念正知而學，是名精進力。無罪力、攝力如上修多羅說。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有四力——覺力、精進力、無罪力、攝力。**
　　那時，世尊告訴在場的眾位比丘說：「有四種力量 ➔ 覺悟的力量（覺力）、勤奮修行的力量（精進力）、無過失的力量（無罪力），以及收攝心念的力量（攝力）。

**◈ 何等為覺力？謂慧、大慧、深慧、難勝慧，是名覺力。**
　　什麼是覺悟的力量（覺力）？指的是智慧、廣大的智慧、深奧的智慧，以及難以被超越的智慧，這就稱為覺悟的力量（覺力）。

**◈ 何等為精進力？**
　　什麼是勤奮修行的力量（精進力）？

**◈ 若於不善法不善數、黑黑數、有罪有罪數、不應親近不應親近數，離此諸法已，**
　　如果對於那些不善的法（導致痛苦與染污的心理狀態）、陰暗的心理狀態、有罪咎的心理狀態，以及不應該親近的心理狀態，能夠遠離這些法之後，

**◈ 若諸餘善善數、白白數、無罪無罪數、應親近應親近數，如此等修習，**
　　而對於其餘善的法（導致快樂與清淨的心理狀態）、光明且純淨的心理狀態、沒有罪咎的心理狀態，以及應該親近的心理狀態，能夠如此地修習，

**◈ 增上精勤、欲、方便、堪能，正念正知而學，是名精進力。**
　　並在其中增加勤奮、願力、適當的方法與能力，在正念（對當下覺知）與正知（對狀態清晰認知）的狀態下學習，這就稱為勤奮修行的力量（精進力）。

**◈ 無罪力、攝力如上修多羅說。」**
　　至於無過失的力量（無罪力）與收攝心念的力量（攝力），其修習方式也如上述的方式一樣。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **覺力 (Paññā-bala)**：指透過智慧（慧）對真理的洞察而產生的力量，能破除無明，使心靈從迷茫中覺醒。
2. **精進力 (Viriya-bala)**：指一種持續且正確的努力。其核心在於「捨惡修善」，即遠離不善法並積極培養善法。
3. **無罪力 (Anavajja-bala)**：指透過遵守戒律與正業，使身心處於無愧、無悔、無過失的狀態，從而產生的內在定力與自信。
4. **攝力 (Saṃvara-bala)**：指收攝心念的能力，能防止心靈向外散亂或被感官欲望牽引，將心專注於修行目標。
5. **正念 (Sati)**：對當下身心狀態的持續覺知，不評判、不迷失。
6. **正知 (Sampajañña)**：在覺知的基礎上，清晰地知道自己在做什麼、為什麼做，以及該行為的性質與目的。

### [現代實作指南]
本經文提供了一套關於「心理能量管理」的實踐框架，可轉化為以下生活實踐步驟：

1. **識別與切割（對應精進力之「離」）**：
   - **實作**：每日回顧，識別出哪些心理狀態是「黑黑數」（如：嫉妒、憤怒、自我懷疑）。
   - **轉念**：當這些狀態出現時，不要與其認同，而是告訴自己：「這是一個不應親近的狀態」，並有意識地將注意力移開，切斷其能量供應。

2. **選擇與培育（對應精進力之「修」）**：
   - **實作**：定義自己的「白白數」（如：感恩、耐心、專注）。
   - **轉念**：在日常小事中刻意練習這些狀態。例如，在排隊時練習「耐心」，在面對他人幫助時練習「感恩」。

3. **優化執行過程（對應「增上精勤、欲、方便、堪能」）**：
   - **精勤**：設定微小且可持續的目標，而非一次性爆發。
   - **欲**：明確知道修習善法能帶來內心的寧靜（建立正向誘因）。
   - **方便**：尋找最適合自己的方法（如：透過呼吸觀察來穩定心念）。
   - **堪能**：透過反覆練習，提高對心念的掌控能力。

4. **建立監控系統（對應「正念正知」）**：
   - **實作**：在行動前問自己：「我現在的心態是什麼？（正念）」、「這個行動是否符合我的長遠目標？（正知）」。這能防止在精進過程中陷入盲目的執著。

### [段落大意]
本經由佛陀闡述修行者應具備的四種內在力量：覺力（智慧洞察）、精進力（捨惡修善）、無罪力（清淨無過）與攝力（心念收攝）。其中詳細說明精進力之核心在於遠離不善法、修習善法，並在正念正知的指引下，透過勤奮與適當方法來強化。
