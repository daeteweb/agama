# 雜阿含經 第 835 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「轉輪王七寶具足，成就人中四種神力，王四天下，身壞命終，生於天上。雖復轉輪聖王七寶具足，成就人間神力，王四天下，身壞命終，得生天上，然猶未斷地獄、畜生、餓鬼惡趣之苦。所以者何？以轉輪王不得於佛不壞淨，法、僧不壞淨，聖戒不成就故。

「多聞聖弟子持糞掃衣，家家乞食，草蓐臥具；而彼多聞聖弟子解脫地獄、畜生、餓鬼惡趣之苦。所以者何？以彼多聞聖弟子於佛不壞淨，法、僧不壞淨，聖戒成就。是故，諸比丘！當作是學：『於佛不壞淨，法、僧不壞淨，聖戒成就。』」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「轉輪王七寶具足，成就人中四種神力，王四天下，身壞命終，生於天上。**
　　那時，世尊告訴在場的眾位比丘說：「轉輪聖王擁有具足的七種寶物，並成就了人間四種超凡的神力，統治著四方天下；當他的身體毀壞、生命結束後，會投生在天上的境界。

**◈ 雖復轉輪聖王七寶具足，成就人間神力，王四天下，身壞命終，得生天上，然猶未斷地獄、畜生、餓鬼惡趣之苦。**
　　儘管轉輪聖王擁有具足的七寶，成就了人間的神力並統治四方，在生命結束後雖然能生於天上，但仍然沒有徹底斷除地獄、畜生、餓鬼等惡趣（痛苦的輪迴境界）的苦難。

**◈ 所以者何？以轉輪王不得於佛不壞淨，法、僧不壞淨，聖戒不成就故。**
　　為什麼呢？因為轉輪聖王並沒有對佛陀保持純淨的信仰（不壞淨），對佛法與僧團保持純淨的信仰，也沒有成就聖者的戒律。

**◈ 「多聞聖弟子持糞掃衣，家家乞食，草蓐臥具；而彼多聞聖弟子解脫地獄、畜生、餓鬼惡趣之苦。**
　　「相反地，一位博學的聖弟子雖然穿著用廢棄布料縫製的糞掃衣，挨家挨戶地乞食，住在簡陋的草屋與臥具中；但這位博學的聖弟子卻能解脫地獄、畜生、餓鬼等惡趣（痛苦的輪迴境界）的苦難。

**◈ 所以者何？以彼多聞聖弟子於佛不壞淨，法、僧不壞淨，聖戒成就。**
　　為什麼呢？因為這位博學的聖弟子對佛陀保持純淨的信仰，對佛法與僧團保持純淨的信仰，並且成就了聖者的戒律。

**◈ 是故，諸比丘！當作是學：『於佛不壞淨，法、僧不壞淨，聖戒成就。』」**
　　因此，各位比丘！應該如此學習：『對佛陀保持純淨的信仰，對佛法與僧團保持純淨的信仰，並成就聖者的戒律。』」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **轉輪聖王 (Cakkavatti)**：佛教理想中的世俗統治者，以正法治理天下，擁有極大的權力與財富，但其成就僅限於世間，未能達到出世間的解脫。
2. **惡趣 (Apāya)**：指輪迴中痛苦的境界，包括地獄、餓鬼、畜生三道。
3. **不壞淨 (Asamisamsa/Faith)**：指對三寶（佛、法、僧）持有堅定、純淨且不被染污的信仰與信心。
4. **聖戒 (Sīla)**：指聖者所持守的道德律條，不僅是形式上的禁忌，更是為了斷除貪嗔痴而建立的內在清淨。

### [現代實作指南]
本經文的核心在於區分「世間的成功」與「靈性的解脫」。在現代生活中，我們可以將其轉化為以下實踐：

1. **重新定義「成功」的標準**：
   ➔ 意識到物質財富（如轉輪王的七寶）與社會地位（如統治四方）雖然能帶來暫時的舒適與享受，但無法解決生命根本的不安與輪迴的苦難。
   ➔ 練習在追求事業成就的同時，不將其視為生命的終極目標，避免陷入「成功即解脫」的錯覺。

2. **建立內在的「不壞淨」 (核心價值觀)**：
   ➔ 在紛擾的資訊時代，尋找一個能讓心靈安定且正確的導師或真理（對應於佛法僧），並對其保持誠信與純淨的信任。
   ➔ 實踐方式：每天抽出十分鐘，回顧自己的價值觀是否被貪婪或憤怒所染污，試圖恢復內心的純淨與正直。

3. **簡約生活的心理實踐**：
   ➔ 聖弟子持糞掃衣、住草蓐，象徵的是對物質依賴的降低。
   ➔ 實踐方式：嘗試「斷捨離」，減少對奢侈品的執著，體會「少即是多」的快樂，將注意力從「擁有什麼」轉移到「我是誰」以及「我如何對待他人」。

4. **將「聖戒」轉化為日常自律**：
   ➔ 聖戒在現代即是「高度的自律與誠實」。
   ➔ 實踐方式：在工作與人際關係中，堅持不欺騙、不傷害、不貪婪。將道德自律視為一種保護心靈不被煩惱侵蝕的「防護牆」，而非限制。

### [段落大意]
本經透過對比「轉輪聖王」與「多聞聖弟子」的不同處境，指出世間的權力與財富雖能帶來天上的福報，但無法斷除輪迴惡趣的苦難。真正的解脫不在於外在的富貴，而在於對三寶的純淨信仰以及聖戒的成就。世尊以此勉勵比丘應追求內在的清淨與戒律，而非世俗的榮華。
