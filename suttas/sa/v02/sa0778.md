# 雜阿含經 第 778 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「於外法中，我不見一法令未生惡不善法生，已生惡不善法重生令增廣，如說惡知識、惡伴黨、惡隨從。諸比丘！惡知識、惡伴黨、惡隨從者，能令未生邪見令生，已生邪見重生令增廣。如是未生邪志、邪語、邪業、邪命、邪方便、邪念、邪定令生，已生者重生令增廣。

「諸比丘！外法中，我不見一法令未生惡不善法不生，已生惡不善法令滅，如說善知識、善伴黨、善隨從。諸比丘！善知識、善伴黨、善隨從，能令未生邪見不生，已生邪見令滅；未生邪志、邪語、邪業、邪命、邪方便、邪念、邪定不生，已生者令滅。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「於外法中，我不見一法令未生惡不善法生，已生惡不善法重生令增廣，如說惡知識、惡伴黨、惡隨從。**
　　「在非佛法的外部教法中，我沒有看到任何一種法，能像惡知識（不良的導師）、惡伴黨（不良的同伴）、惡隨從（不良的追隨者）那樣，能讓尚未產生的惡不善法（不良的心理狀態）產生，或讓已經產生的惡不善法再次產生並擴大。

**◈ 諸比丘！惡知識、惡伴黨、惡隨從者，能令未生邪見令生，已生邪見重生令增廣。**
　　比丘們！惡知識、惡伴黨、惡隨從，能夠讓尚未產生的邪見（錯誤的見解）產生，並讓已經產生的邪見再次產生且擴大。

**◈ 如是未生邪志、邪語、邪業、邪命、邪方便、邪念、邪定令生，已生者重生令增廣。**
　　同樣地，他們能讓尚未產生的邪志（錯誤的思惟）、邪語（錯誤的言語）、邪業（錯誤的行為）、邪命（錯誤的生活方式）、邪方便（錯誤的精進）、邪念（錯誤的覺知）、邪定（錯誤的專注）產生，並讓已經產生的這些錯誤狀態再次產生且擴大。

**◈ 「諸比丘！外法中，我不見一法令未生惡不善法不生，已生惡不善法令滅，如說善知識、善伴黨、善隨從。**
　　「比丘們！在非佛法的外部教法中，我沒有看到任何一種法，能像善知識（良師）、善伴黨（良伴）、善隨從（良隨從）那樣，能讓尚未產生的惡不善法不產生，或讓已經產生的惡不善法滅除。

**◈ 諸比丘！善知識、善伴黨、善隨從，能令未生邪見不生，已生邪見令滅；**
　　比丘們！善知識、善伴黨、善隨從，能夠讓尚未產生的邪見不產生，並讓已經產生的邪見滅除；

**◈ 未生邪志、邪語、邪業、邪命、邪方便、邪念、邪定不生，已生者令滅。」**
　　同時，也能讓尚未產生的邪志、邪語、邪業、邪命、邪方便、邪念、邪定不產生，並讓已經產生的這些錯誤狀態滅除。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **惡不善法 (Akusala-dhamma)**：指那些會導致痛苦、染污心靈且產生惡業的心理狀態或行為。
2. **邪見 (Micchā-diṭṭhi)**：對真理的錯誤認知，例如否認因果業報或對自我本質的誤解，是導致所有錯誤行為的根源。
3. **八正道之反面 (Wrong Path)**：經文中提到的邪志、邪語、邪業、邪命、邪方便、邪念、邪定，分別對應八正道中的正思惟、正語、正業、正命、正精進、正念、正定。這顯示了錯誤的環境如何系統性地將人的認知與行為導向錯誤的方向。
4. **善知識 (Kalyāṇa-mittatā)**：指能引導修行者走向正見、正行，並協助其滅除煩惱的良師益友。

### [現代實作指南]
本經文強調了「環境與同伴」對心智塑造的決定性影響。在現代生活中，可採取以下實作步驟：

1. **社交圈審核 (Social Audit)**：
   ➔ 觀察身邊最常接觸的五個人。分析他們在面對衝突、利益或困難時的反應。
   ➔ 若其反應傾向於憤怒、貪婪或將責任推卸給他人（惡不善法），意識到這種能量會透過潛移默化影響你的「邪見」與「邪志」。

2. **建立「正見」的資訊屏障**：
   ➔ 現代的「惡知識」不限於真人，也包括社交媒體上的極端言論或誘導消費的資訊。
   ➔ 當發現某種資訊讓你產生強烈的嫉妒、仇恨或對現實的扭曲認知時，將其定義為「惡隨從」，並主動減少接觸。

3. **主動尋找「善知識」**：
   ➔ 尋找那些在生活中實踐誠實、慈悲且具有自律能力的人。
   ➔ 練習在與其交流時，觀察對方如何將「正念」與「正定」應用於解決問題，並嘗試模仿其思考邏輯。

4. **轉念練習**：
   ➔ 當自己陷入負面情緒（已生惡不善法）時，不要試圖強行壓抑，而是尋找一位善知識或閱讀正法典籍，利用外部的正向能量來「令其滅除」。

### [段落大意]
本經探討環境對心靈的影響。世尊指出，不良的導師與同伴（惡知識）會誘發並擴大錯誤的見解與行為（八正道的反面）；而良師益友（善知識）則能防止錯誤見解的產生，並協助滅除已有的惡不善法。強調選擇正確的社交環境是修行正法的關鍵。
