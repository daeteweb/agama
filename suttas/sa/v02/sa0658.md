# 雜阿含經 第 658 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有五根——信根、精進根、念根、定根、慧根。何等為信根？謂聖弟子於如來所起信心，根本堅固，諸天、魔、梵、沙門、婆羅門及諸世間法所不能壞，是名信根。何等為精進根？謂四正斷。何等為念根？謂四念處。何等為定根？謂四禪。何等為慧根？謂四聖諦，此諸功德，皆以慧為首。譬如堂閣，棟為其首。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有五根——信根、精進根、念根、定根、慧根。**
　　那時，世尊告訴在場的眾位比丘說：「修行中有五種根基（五根）➔ 分別是信根（信心的根基）、精進根（努力的根基）、念根（正念的根基）、定根（專注的根基）以及慧根（智慧的根基）。

**◈ 何等為信根？謂聖弟子於如來所起信心，根本堅固，諸天、魔、梵、沙門、婆羅門及諸世間法所不能壞，是名信根。**
　　什麼是信根？指的是聖弟子對如來（成就不變真理而來到世間的人）所產生的信心，這種信心根基堅固，無論是天神、魔王、梵天、沙門、婆羅門，以及世間的一切法門都無法將其摧毀，這就稱為信根。

**◈ 何等為精進根？謂四正斷。**
　　什麼是精進根？指的是四正斷（即：斷除尚未產生的惡法、止息已經產生的惡法、生起尚未產生的善法、增長已經產生的善法這四種努力）。

**◈ 何等為念根？謂四念處。**
　　什麼是念根？指的是四念處（即：對色（物質身體）、受（感受）、心（心的狀態）、法（心法現象）這四個面向的持續覺察）。

**◈ 何等為定根？謂四禪。**
　　什麼是定根？指的是四禪（即：四種由淺入深、心境極其專注且寧靜的禪定狀態）。

**◈ 何等為慧根？謂四聖諦，此諸功德，皆以慧為首。譬如堂閣，棟為其首。」**
　　什麼是慧根？指的是對四聖諦（苦、集、滅、道）的深刻體悟。在所有這些修行功德中，智慧（慧）是最核心的領袖，就像一座房屋的棟樑是整座建築最關鍵的支撐一樣。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **五根 (Indriya)**：指在修行過程中起主導作用的五種心理能力。它們如同植物的根一樣，是成長覺悟的基礎。
2. **四正斷 (Sammappadhāna)**：精進的具體操作方法，包含防止惡法、斷除惡法、生起善法、增長善法。
3. **四念處 (Satipaṭṭhāna)**：正念的四種對象，分別是身（身體）、受（感受）、心（心念）、法（心理現象/真理）。
4. **四禪 (Jhana)**：透過定力達到的一種高度專注且心靈平靜的狀態，分為四個層次。
5. **四聖諦 (Cattāri Ariyasaccāni)**：佛教的核心真理，包含苦（人生本質的苦）、集（苦的原因）、滅（苦的止息）、道（解脫之途）。

### [現代實作指南]
本經文提供了一套完整的「心理成長系統」，可將其轉化為現代生活中的心理實踐：

1. **建立正向信心 (信根)**：
   - **實作**：不要盲目相信，而是透過「試行 ➔ 驗證 ➔ 信任」的過程。例如，嘗試練習正念後發現壓力降低，進而對此方法產生堅定的信心。這種基於經驗的信心能讓我們在面對生活挫折時不輕易崩潰。

2. **有方向的努力 (精進根)**：
   - **實作**：將「努力」具體化。不要只是盲目忙碌，而要執行「四正斷」➔ 覺察並停止不良習慣（如拖延、憤怒），同時刻意培養良好習慣（如閱讀、感恩）。

3. **維持覺察狀態 (念根)**：
   - **實作**：在日常生活中練習「標記法」。當身體感到緊繃時，心中默念「身體緊繃」；當感到憤怒時，默念「憤怒升起」。這能讓我們從情緒中抽離，不再被情緒牽著走。

4. **培養深度專注 (定根)**：
   - **實作**：每天設定 10-20 分鐘的「單一任務時間」。關掉手機，只專注於呼吸或一件簡單的事，訓練心智不再跳躍，建立內心的安定感。

5. **以智慧導向 (慧根)**：
   - **實作**：在所有行動前問自己：「這件事的本質是什麼？它會帶來長久的快樂還是短暫的滿足？」將智慧視為棟樑，確保信心、努力、覺察與專注都有正確的方向，避免陷入盲信或死板的修行。

### [段落大意]
本經闡述了修行解脫的五種核心能力（五根）：信、精進、念、定、慧。佛陀詳細定義了每一項根基的內涵，並強調「智慧」是所有功德的核心與主導，如同房屋的棟樑一般至關重要。
