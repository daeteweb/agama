# 雜阿含經 第 723 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「善哉！比丘！僧人聞法，諸年少比丘供養奉事諸尊長老。所以者何？年少比丘供養奉事長老比丘者，時時得聞深妙之法，聞深法已，二正事成就，身正及心正。爾時修念覺分，修念覺分已，念覺分滿足；念覺滿足已，於法選擇，分別於法，思量於法。爾時，方便修擇法覺分，乃至捨覺分修習滿足。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「善哉！比丘！**
　　那時，世尊告訴在場的眾位比丘說：「好極了！比丘們！

**◈ 僧人聞法，諸年少比丘供養奉事諸尊長老。**
　　在僧團聽聞法（真理）的環境中，年輕的比丘們供養並侍奉那些德高望重的長老。

**◈ 所以者何？年少比丘供養奉事長老比丘者，時時得聞深妙之法，**
　　為什麼呢？因為年輕的比丘若能供養並侍奉長老比丘，就能時時地聽聞深奧微妙的法（真理）。

**◈ 聞深法已，二正事成就，身正及心正。**
　　在聽聞了深奧的法之後，就能成就兩項正確的修行事宜：一是身體的行為端正（身正），二是內心的狀態正向（心正）。

**◈ 爾時修念覺分，修念覺分已，念覺分滿足；**
　　那時便能修行念覺分（導向覺悟的七種心法要素），在修行念覺分之後，使這些覺悟要素達到圓滿滿足。

**◈ 念覺滿足已，於法選擇，分別於法，思量於法。**
　　當念覺分圓滿後，就能對法（真理）進行選擇、分辨並深入思量。

**◈ 爾時，方便修擇法覺分，乃至捨覺分修習滿足。」**
　　那時，便能方便地修行擇法覺分（能分辨真偽的覺悟要素），直到捨覺分（能放下執著、保持平等心的覺悟要素）的修習也達到圓滿滿足。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **念覺分 (Bojjhaṅga)**：指導向覺悟的七種心法要素，包括念（正念）、擇法（審視真理）、精進（努力）、喜（法喜）、輕安（身心放鬆）、定（專注）與捨（平等心）。
2. **擇法覺分 (Dhamma-vicaya)**：念覺分中的第二項，指透過觀察與分析，能分辨出哪些是導致苦因的法，哪些是導向解脫的法。
3. **捨覺分 (Upekkhā)**：念覺分中的最後一項，指在面對各種感受或現象時，能保持中立、不偏不倚且不執著的平等心。
4. **身正及心正**：指在修行中達到行為（身業）與心理狀態（心業）的端正，是進入深層禪定與覺悟的基礎。

### [現代實作指南]
本經文揭示了一個從「外在行為」到「內在覺悟」的遞進路徑：**謙卑服務 ➔ 獲取知識 ➔ 修正行為 ➔ 心理轉化 ➔ 終極覺悟**。在現代生活中可採取以下實作步驟：

1. **建立「學習者」的謙卑心**：
   - **實作**：在專業領域或精神成長中，尋找一位真正具備實踐經驗的導師（長老）。透過實際的協助或服務（如協助導師處理雜務、誠心請益），打破自我傲慢（我執），創造一個能讓智慧自然流動的環境。
   - **轉念**：將「服務他人」視為「為自己創造學習機會」的投資，而非單純的勞務。

2. **從「身正」開始修正**：
   - **實作**：在學習新知識或修行時，先從最簡單的行為規範開始（如：準時、專注、禮貌）。當身體的行為趨於穩定（身正），心靈的雜訊會減少。
   - **轉念**：意識到「行為的端正」是「心靈平靜」的物理基礎。

3. **練習「擇法」的思維**：
   - **實作**：在面對生活壓力或情緒時，不要立即反應，而是運用「擇法」技巧 ➔ 觀察這個情緒是來自於貪愛還是厭惡？這個想法會讓我更靠近平靜還是更遠離平靜？
   - **轉念**：將生活中的衝突視為「擇法」的實驗室，練習分辨「有益之法」與「有害之法」。

4. **追求「捨」的平衡**：
   - **實作**：當事情進展順利（喜）或不順利（苦）時，練習在心中對自己說：「這也是一種現象」，嘗試不被情緒牽著走，維持一種觀察者的平等心。

### [段落大意]
本經強調年輕比丘透過供養與侍奉長老，能獲得聽聞深奧真理的機會。這種外在的恭敬心能導向身心的端正，進而使修行者能圓滿地修習「念覺分」，最終透過對法的選擇與思量，達到擇法與捨覺分的圓滿，實現覺悟。
