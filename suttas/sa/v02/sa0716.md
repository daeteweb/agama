# 雜阿含經 第 716 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「於內法中，我不見一法，未生惡不善法令生，已生惡不善法重生令增廣，未生善法不生，已生則退，所謂不正思惟。諸比丘！不正思惟者，未生貪欲蓋令生，已生者重生令增廣。未生瞋恚、睡眠、掉悔、疑蓋令生，已生者重生令增廣。未生念覺分不生，已生者令退，未生擇法、精進、喜、猗、定、捨覺分令生，已生者重生令增廣。

「我不見一法能令未生惡不善法不生，已生者令斷，未生善法令生，已生者重生令增廣，正思惟。比丘！正思惟者，未生貪欲蓋令不生，已生者令斷，未生瞋恚、睡眠、掉悔、疑蓋令不生，已生者令斷，未生念覺分令生，已生者重生令增廣。未生擇法、精進、喜、猗、定、捨覺分令生，已生者重生令增廣。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「於內法中，我不見一法，未生惡不善法令生，已生惡不善法重生令增廣，未生善法不生，已生則退，所謂不正思惟。**
　　「在內在的心法（心理現象）之中，我不見有任何一種法，能像『不正思惟（錯誤的思考方向）』這樣：讓原本還沒產生的惡不善法（不健康的心理狀態）產生，讓已經產生的惡不善法再次產生並擴大；同時讓原本還沒產生的善法（健康的心理狀態）無法產生，讓已經產生的善法退化。」

**◈ 諸比丘！不正思惟者，未生貪欲蓋令生，已生者重生令增廣。**
　　「比丘們！所謂不正思惟（錯誤的思考方向），會讓尚未產生的貪欲蓋（因貪愛而遮蔽心靈的障礙）產生，並讓已經產生的貪欲蓋再次產生且更加擴大。」

**◈ 未生瞋恚、睡眠、掉悔、疑蓋令生，已生者重生令增廣。**
　　「它也會讓尚未產生的瞋恚（憤怒與厭惡）、睡眠（昏沉與懈怠）、掉悔（躁動不安與後悔）、疑蓋（因猶豫不決而遮蔽心靈的障礙）產生，並讓已經產生的這些遮蔽再次產生且更加擴大。」

**◈ 未生念覺分不生，已生者令退，未生擇法、精進、喜、猗、定、捨覺分令生，已生者重生令增廣。**
　　「（原文如此）它會讓尚未產生的念覺分（正念的覺悟要素）不產生，讓已經產生的念覺分退化；而讓尚未產生的擇法（辨別真偽）、精進（持續努力）、喜（法喜）、猗（輕安）、定（專注）、捨（平等心）等覺分（覺悟的要素）產生，並讓已經產生的這些覺分再次產生且更加擴大。」

**◈ 「我不見一法能令未生惡不善法不生，已生者令斷，未生善法令生，已生者重生令增廣，正思惟。**
　　「我不見有任何一種法，能像『正思惟（正確的思考方向）』這樣：讓尚未產生的惡不善法（不健康的心理狀態）不產生，讓已經產生的惡不善法被截斷；同時讓尚未產生的善法（健康的心理狀態）產生，並讓已經產生的善法再次產生且更加擴大。」

**◈ 比丘！正思惟者，未生貪欲蓋令不生，已生者令斷，**
　　「比丘們！所謂正思惟（正確的思考方向），會讓尚未產生的貪欲蓋（因貪愛而遮蔽心靈的障礙）不產生，並讓已經產生的貪欲蓋被截斷。」

**◈ 未生瞋恚、睡眠、掉悔、疑蓋令不生，已生者令斷，**
　　「同樣地，會讓尚未產生的瞋恚（憤怒與厭惡）、睡眠（昏沉與懈怠）、掉悔（躁動不安與後悔）、疑蓋（因猶豫不決而遮蔽心靈的障礙）不產生，並讓已經產生的這些遮蔽被截斷。」

**◈ 未生念覺分令生，已生者重生令增廣。**
　　「同時，它會讓尚未產生的念覺分（正念的覺悟要素）產生，並讓已經產生的念覺分再次產生且更加擴大。」

**◈ 未生擇法、精進、喜、猗、定、捨覺分令生，已生者重生令增廣。」**
　　「它還會讓尚未產生的擇法（辨別真偽）、精進（持續努力）、喜（法喜）、猗（輕安）、定（專注）、捨（平等心）等覺分（覺悟的要素）產生，並讓已經產生的這些覺分再次產生且更加擴大。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **不正思惟 (Micchā-saṅkappa)**：錯誤的思考方向，通常指傾向於貪欲、瞋恚或害人的思考模式，是導致心理障礙產生的根源。
2. **正思惟 (Sammā-saṅkappa)**：正確的思考方向，指離欲、無瞋、不害的思考，是消除心理障礙並培養覺悟要素的關鍵。
3. **五蓋 (Pañca Nīvaraṇā)**：遮蔽心靈、妨礙禪定的五種障礙，包括貪欲、瞋恚、睡眠（昏沉）、掉悔（躁動）與疑惑。
4. **七覺分 (Satta Bojjhaṅgā)**：導向覺悟的七種心理要素，包含：
    - **念 (Sati)** ➔ 正念。
    - **擇法 (Dhamma-vicaya)** ➔ 辨別真偽。
    - **精進 (Viriya)** ➔ 持續努力。
    - **喜 (Pīti)** ➔ 法喜。
    - **猗 (Passaddhi)** ➔ 輕安。
    - **定 (Samādhi)** ➔ 專注。
    - **捨 (Upekkhā)** ➔ 平等心。
5. **學術註記**：在「不正思惟」的段落中，原文提到「未生擇法...覺分令生」，這在邏輯上與佛法教義相悖（錯誤的思考不應導致覺悟要素產生）。根據對應的「正思惟」段落及巴利三藏之對照，此處應為文本傳抄錯誤，原意應為「令不生」且「令退」。

### [現代實作指南]
本經文揭示了「思考方向」如何決定「心理狀態」的因果關係。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **覺察思考的「方向」**：
   當你感到焦慮、憤怒或疲憊時，不要直接對抗情緒，而是回溯到「思考方向」。問自己：「我現在的思考是在強化貪欲（想要掌控）、瞋恚（厭惡現狀）還是疑惑（自我懷疑）？」

2. **識別「蓋」的出現**：
   - 當你發現自己無法專注時 ➔ 識別出這是「睡眠（昏沉）」或「掉悔（躁動）」。
   - 當你對他人產生敵意時 ➔ 識別出這是「瞋恚」。
   - 當你執著於某個結果時 ➔ 識別出這是「貪欲」。

3. **執行「思維轉向」 (Pivot to Right Thought)**：
   一旦識別出「不正思惟」，立即將思考方向調整為：
   - **離欲**：告訴自己「這件事的本質是無常的，不必強求」。
   - **無瞋**：思考「對方也是在痛苦中掙扎，我選擇不被憤怒牽引」。
   - **不害**：思考「如何以最溫和的方式解決問題」。

4. **培養「覺分」的正向循環**：
   透過正思惟，主動引入「念（覺察）」➔ 接著進行「擇法（分析這件事的因緣）」➔ 產生「精進（採取正確行動）」➔ 獲得「喜（看到進步的法喜）」➔ 進入「輕安與定（心靈平靜）」➔ 最後達到「捨（對結果的坦然）」。

### 段落大意 
本經探討了「思維方向」對心靈狀態的決定性影響。不正思惟會催生五蓋（心理障礙）並抑制七覺分（覺悟要素）；而正思惟則能截斷五蓋，並積極培養七覺分，使修行者從混亂的心理狀態轉向清明與覺悟。
