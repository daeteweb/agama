# 雜阿含經 第 644 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有五根。何等為五？謂信根、精進根、念根、定根、慧根。若比丘於此五根如實善觀察，如實善觀察者，於三結斷知，謂身見、戒取、疑，是名須陀洹，不墮惡趣法，決定正向於正覺，七有天人往生，究竟苦邊。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「有五根。何等為五？**
　　那時，世尊告訴在場的眾位比丘說：「有五種修行上的能力（五根）。這五種能力是指什麼呢？

**◈ 謂信根、精進根、念根、定根、慧根。**
　　也就是：信根（對正法的信心）、精進根（不懈怠的努力）、念根（正念的覺知）、定根（心專一的定力）以及慧根（洞察真理的智慧）。

**◈ 若比丘於此五根如實善觀察，如實善觀察者，於三結斷知，**
　　如果比丘能夠對這五種修行能力進行如實且深刻的觀察，能夠如實善於觀察的人，就能夠斷除並了知三種束縛心靈的結（三結），

**◈ 謂身見、戒取、疑，**
　　這三結是指：身見（對「我」這個實體的錯誤認知）、戒取（對形式主義的戒律或儀軌的執著）以及疑（對於正法與修行路徑的猶豫不決）。

**◈ 是名須陀洹，不墮惡趣法，決定正向於正覺，七有天人往生，究竟苦邊。」**
　　這樣的人就被稱為須陀洹（入流果者），他不再墮入惡道（如地獄、餓鬼、畜生），確定會正向地邁向最高、最圓滿的智慧與覺悟（正覺），最多在七次天人道的輪迴往生之中，就能夠徹底終結所有痛苦的邊際（達成涅槃）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **五根 (Indriya)**：指在修行過程中起主導作用的五種心理能力。信、精進、念、定、慧相互支持，能使修行者在面對煩惱時具有強大的心理韌性與洞察力。
2. **三結 (Samyojana)**：指將眾生束縛在輪迴中的心理枷鎖。其中「身見」是所有執著的根源，認為有一個恆常不變的「我」存在；「戒取」是指盲目相信某些外在的儀式或禁忌能帶來解脫；「疑」則是對佛法真理缺乏信心而產生的猶豫。
3. **須陀洹 (Sotāpanna)**：字面意思是「入流者」，指修行達到第一階段果位的人。此階段的人已證悟了無我與緣起，因此在邏輯上不再可能退轉回未證果的狀態。
4. **惡趣 (Apāya)**：指由於業力牽引而墮入的痛苦境界，通常指地獄、餓鬼、畜生三道。

### [現代實作指南]
本經強調的是「平衡」與「觀察」五種心理能力的運作。在現代生活中，我們可以將其轉化為以下心理實踐：

1. **建立正向信心（信根）**：
   ➔ 實作：不要盲信，而是透過「試驗 ➔ 驗證 ➔ 信任」的過程。例如，嘗試練習正念呼吸，觀察其對壓力緩解的實際效果，從而建立對方法的信心。

2. **維持穩定努力（精進根）**：
   ➔ 轉念：將「強迫性的努力」轉化為「持續性的習慣」。避免短時間的過度衝刺導致倦怠，而是在日常生活中設定微小且可持續的修行目標。

3. **培養即時覺知（念根）**：
   ➔ 實作：練習「標記法」。當情緒升起時，在心中輕輕標記「生氣」、「焦慮」或「期待」，將自己從情緒的漩渦中抽離，成為觀察者。

4. **訓練專注深度（定根）**：
   ➔ 實作：在碎片化資訊時代，每天設定 10-20 分鐘的「單一任務時間」，不使用手機，僅專注於一件事情（如行走或呼吸），訓練心念不散亂。

5. **運用理智分析（慧根）**：
   ➔ 轉念：當面對痛苦時，運用「因緣分析法」。問自己：「這個痛苦是由哪些條件構成的？（例如：對結果的期待 ➔ 現實不符 ➔ 產生失望）」，透過分析打破對痛苦的盲目執著。

**【平衡技巧】**：
- 當信心過強而缺乏智慧時 ➔ 容易陷入迷信 ➔ **需增加「慧根」的分析**。
- 當智慧過強而缺乏信心時 ➔ 容易陷入枯燥的理論爭論 ➔ **需增加「信根」的實踐**。
- 當定力過強而缺乏覺知時 ➔ 容易陷入昏沉或僵化 ➔ **需增加「念根」的靈活**。

---

### 段落大意 
本經闡述了修行中五種核心能力（信、精進、念、定、慧）的重要性。佛陀指出，若能如實觀察並平衡這五根，修行者可斷除身見、戒取、疑這三種根本束縛，成就須陀洹果位。一旦進入此境界，便能確保不再墮入惡道，且在最多七次輪迴內達成最終的解脫（涅槃）。
