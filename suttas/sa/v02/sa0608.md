# 雜阿含經 第 608 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若比丘離四念處者，則離如實聖法；離如實聖法者，則離聖道；離聖道者，則離甘露法；離甘露法者，不得脫生、老、病、死、憂、悲、惱、苦，我說彼於苦不得解脫。

「若比丘不離四念處者，得不離聖如實法；不離聖如實者，則不離聖道；不離聖道者，則不離甘露法；不離甘露法者，得脫生、老、病、死、憂、悲、惱、苦，我說彼人解脫眾苦。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若比丘離四念處者，則離如實聖法；**
　　「如果比丘離開了四念處（對身體、感受、心靈、法性的四種正念覺察），就等於離開了如實聖法（符合真實情況的聖妙真理）；

**◈ 離如實聖法者，則離聖道；**
　　離開了如實聖法的人，就等於離開了聖道（通往覺悟的解脫之路）；

**◈ 離聖道者，則離甘露法；**
　　離開了聖道的人，就等於離開了甘露法（能消除生死之苦、帶來永恆寧靜的法）；

**◈ 離甘露法者，不得脫生、老、病、死、憂、悲、惱、苦，我說彼於苦不得解脫。**
　　離開了甘露法的人，就無法脫離出生、衰老、疾病、死亡，以及憂慮、悲傷、煩惱與痛苦，我說這樣的人無法從苦中獲得解脫。

**◈ 「若比丘不離四念處者，得不離聖如實法；**
　　「如果比丘不離開四念處（對身體、感受、心靈、法性的四種正念覺察），就能不離開聖如實法（符合真實情況的聖妙真理）；

**◈ 不離聖如實者，則不離聖道；**
　　不離開聖如實法的人，就不會離開聖道（通往覺悟的解脫之路）；

**◈ 不離聖道者，則不離甘露法；**
　　不離開聖道的人，就不會離開甘露法（能消除生死之苦、帶來永恆寧靜的法）；

**◈ 不離甘露法者，得脫生、老、病、死、憂、悲、惱、苦，我說彼人解脫眾苦。」**
　　不離開甘露法的人，就能脫離出生、衰老、疾病、死亡，以及憂慮、悲傷、煩惱與痛苦，我說這樣的人能從所有苦難中解脫。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四念處 (Satipaṭṭhāna)**：佛教修行中最重要的正念觀察法，分為四個對象：身念處（觀察物質身體）、受念處（觀察感受）、心念處（觀察心靈狀態）、法念處（觀察心法運作的規律）。
2. **如實 (Yathā-bhūta)**：指事物呈現的真實面貌，不被主觀的偏見、妄想或情緒所遮蔽。
3. **甘露法 (Amata-dhamma)**：原意為「不死之法」，比喻涅槃的境界。如同甘露能使人獲救，此法能使修行者脫離輪迴的生死苦海。
4. **聖道 (Ariya-magga)**：指能導向解脫的正確路徑，通常指八正道。

### [現代實作指南]
本經文揭示了一個「因果鏈條」：**正念 ➔ 真相 ➔ 正確路徑 ➔ 解脫**。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **建立「微正念」觸發點**：
   當你感到壓力（憂、悲、惱、苦）時，立即停止對事件的邏輯分析，將注意力轉向「四念處」：
   ➔ **身**：感覺呼吸的起伏，或腳掌接觸地面的壓力。
   ➔ **受**：覺察心中現在是「不舒服」的感受，僅僅標記它，而不試圖推開它。
   ➔ **心**：觀察心現在是「焦慮」或「憤怒」的狀態。
   ➔ **法**：意識到「這種焦慮感也是無常的，它在升起，然後會消失」。

2. **從「反應」轉向「觀察」**：
   大多數人的痛苦來自於對感受的「自動反應」（例如：感到被冒犯 ➔ 立即反擊）。實作指南是：在「受」與「行（心的造作）」之間插入一個正念的間隙。當你能「如實」觀察到憤怒的升起，你就不再被憤怒控制，這便是進入了「聖道」的起點。

3. **邏輯轉念**：
   告訴自己：「如果我現在失去了覺察（離四念處），我就會陷入慣性的情緒反應（離如實聖法），進而做出錯誤的決定（離聖道），最終導致更多的壓力與痛苦（不得解脫）。」透過這個邏輯，將「保持正念」視為一種最高效率的自我保護機制。

---

### [段落大意]
本經強調「四念處」是解脫苦難的唯一入口。佛陀指出，若失去正念覺察，將導致無法認知真理、偏離修行正道，最終無法脫離生老病死及種種心理痛苦；反之，持守四念處則能循序漸進地達成如實知見，最終獲得徹底的解脫。
