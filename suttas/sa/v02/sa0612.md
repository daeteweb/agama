# 雜阿含經 第 612 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「如人執持四種強弓，大力方便射多羅樹影，疾過無閡。如是如來四種聲聞，增上方便，利根智慧，盡百年壽，於如來所百年說法教授，唯除食息、[示*甫]寫、睡眠，中間常說、常聽；智慧明利，於如來所說，盡底受持，無諸障閡，於如來所不加再問。如來說法無有終極，聽法盡壽，百歲命終，如來說法猶不能盡。當知如來所說無量無邊，名、句、味身亦復無量，無有終極，所謂四念處。何等為四？謂身念處，受、心、法念處。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

一切四念處經，皆以此總句，所謂「是故，比丘！於四念處修習，起增上欲，精勤方便，正念正智，應當學。」

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「如人執持四種強弓，大力方便射多羅樹影，疾過無閡。**
　　「就像一個人拿著四種強力的弓，運用強大的力量與技巧射向多羅樹的影子，箭速極快且毫無阻礙。

**◈ 如是如來四種聲聞，增上方便，利根智慧，盡百年壽，於如來所百年說法教授，唯除食息、[示*甫]寫、睡眠，中間常說、常聽；**
　　同樣地，如來（成就不變真理而來到世間的人）有四種聲聞弟子，他們擁有卓越的修行方法與敏銳的智慧根基，即便花盡一百年的壽命，在如來這裡接受百年的法義教授，除了吃飯、呼吸、排泄與睡眠之外，其餘時間都在不斷地討論與聆聽；

**◈ 智慧明利，於如來所說，盡底受持，無諸障閡，於如來所不加再問。**
　　他們智慧明亮敏銳，對於如來所說的法（真理）能徹底地領悟並持守，沒有任何障礙，因此不需要對如來重複詢問。

**◈ 如來說法無有終極，聽法盡壽，百歲命終，如來說法猶不能盡。**
　　如來所說的法（真理）是沒有窮盡的，即便聽法直到壽命結束，即便活到一百歲去世，如來所說的法依然無法全部講完。

**◈ 當知如來所說無量無邊，名、句、味身亦復無量，無有終極，所謂四念處。**
　　應當了解，如來所說的教法是無量無邊的，其名稱、句子以及深層的義理（味身）也同樣無量且沒有終結，其中就包括所謂的四念處（四種正念的確立）。

**◈ 何等為四？謂身念處，受、心、法念處。」**
　　什麼是這四種？指的是：對色（物質身體）的念處、對受（感受）的念處、對心（意識狀態）的念處，以及對法（心理現象與真理）的念處。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 一切四念處經，皆以此總句，所謂「是故，比丘！於四念處修習，起增上欲，精勤方便，正念正智，應當學。」**
　　所有關於四念處的經文，都以此總結句作為核心，也就是說：「因此，比丘們！應當在四念處中進行修行（Bhāvanā），生起強烈的解脫願望，精進地運用方便方法，保持正念與正知，應當如此學習。」

### [詞語註解（解釋重要佛教概念）]
1. **四念處 (Satipaṭṭhāna)**：原始佛教的核心修行法門，指建立四種正念的觀察對象，分別是身體（身）、感受（受）、心靈狀態（心）以及心理現象/法義（法），旨在透過覺知打破對「我」的執著。
2. **名、句、味 (Nama, Pada, Rasa)**：指理解教法的三個層次。名 ➔ 術語的名稱；句 ➔ 句子的結構與字面含義；味 ➔ 經文背後深層的真理義理（實相）。
3. **聲聞 (Sāvaka)**：指透過聆聽佛陀或聖者的教導而進入修行路徑的人。
4. **增上欲 (Chanda)**：一種正向的、強烈的願望，指對解脫與覺悟的強烈追求，而非世俗的貪欲。

### [現代實作指南]
本經強調了「四念處」作為修行核心的重要性。在現代生活中，我們可以將其轉化為一套「全方位覺察系統」：

1. **身念處 ➔ 身體掃描**：
   - 當感到壓力時，不要立刻分析原因，而是先將注意力移至身體。觀察肩膀是否僵硬？呼吸是否淺快？僅僅是「觀察」物質身體的狀態，而不試圖立即改變它，這能迅速將心從混亂的思考中拉回現實。

2. **受念處 ➔ 感受標記**：
   - 當情緒升起時，在心中給它一個標記。例如：「現在有『不舒服』的感受」、「現在有『愉悅』的感受」。將「我很生氣」轉化為「觀察到有『憤怒』的受（感受）在運作」，從而創造出觀察者與情緒之間的空間。

3. **心念處 ➔ 意識狀態監控**：
   - 覺察當下的心念狀態。例如：「現在心是散亂的」、「現在心是專注的」、「現在心是有貪念的」。像監控螢幕一樣觀察心的起伏，而不對其進行道德評判。

4. **法念處 ➔ 模式識別**：
   - 觀察上述過程中的規律。例如：「每當我感到被否定（受），我的心就會變得憤怒（心），接著身體會發熱（身）」。意識到這是一個「因緣而生」的連鎖反應，而非「我」在生氣，進而體會到無我（Anattā）的真理。

### 段落大意 
本經以強弓射箭比喻優秀弟子的學習能力，強調如來所說之法無量無邊，即便百歲亦不能盡。其中最核心的修行路徑為「四念處」（身、受、心、法），並勉勵修行者應生起強烈的願望，以正念正知精進修習。
