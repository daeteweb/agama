# 雜阿含經 第 848 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有四種諸天天道。何等為四？謂聖弟子念如來事，如是：如來、應、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊。於此如來事生隨喜心，隨喜已，心歡悅，心歡悅已，身猗息，身猗息已，覺受樂，覺受樂已，三昧定，三昧定已，聖弟子作如是學：『何等為諸天天道？』復作是念：『我聞無恚為上諸天天道。』作是念：『我從今日，於世間若怖若安，不起瞋恚，我但當自受純一滿淨諸天天道。』是名第一諸天天道，未淨眾生令淨，已淨者重令淨。

「復次，比丘！聖弟子念於法事，謂如來說正法、律，現法離諸熾然，不待時節，通達涅槃，即身觀察，緣自覺知。如是知法事已，心生隨喜；隨喜已，身猗息，身猗息已，覺受樂，覺受樂已，三昧定，三昧定已，聖弟子作如是學：『何等為諸天天道？』復作是念：『我聞無恚為上諸天天道，我從今日，於此世間若怖若安，不起瞋恚，我當受持純一滿淨諸天天道。』是名第二諸天天道。

「復次，比丘！若於僧事起於正念，謂世尊弟子僧正直等向，所應恭敬、尊重、供養，無上福田。彼如是於諸僧事正憶念已，心生隨喜；心隨喜已，得歡悅，歡悅已，身猗息，身猗息已，覺受樂，覺受樂已，三昧定，三昧定已，彼聖弟子作如是學：『何等諸天天道？』復作是念：『我聞諸天無恚為上諸天天道，我從今日，於諸世間若怖若安，不起瞋恚，我但當受持純一滿淨諸天天道。』是名第三諸天天道。

「復次，比丘！謂聖弟子自念所有戒事，隨憶念言：『我於此不缺戒、不污戒、不雜戒、明智所歎戒、智者不厭戒。』於如是等戒事正憶念已，心生隨喜；隨喜已，歡悅，歡悅已；身猗息，身猗息已，覺受樂，覺受樂已，三昧定，三昧定已，聖弟子作是念：『何等為諸天天道？』復作是念：『我聞諸天無恚為上，我從今日，於諸世間若怖若安，不起瞋恚，我當受持純一滿淨諸天天道。』是名第四諸天天道。未淨眾生令淨，已淨者重令淨。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「有四種諸天天道。何等為四？**
　　那時，世尊告訴在場的眾位比丘說：「有四種能導向天界般喜悅心境的修行路徑（天道）。這四種路徑是什麼呢？

**◈ 謂聖弟子念如來事，如是：如來、應、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊。**
　　首先，是指聖弟子憶念關於如來（成就不變真理而來到世間的人）的特質，例如：如來、應供（應受世間供養的人）、正遍知（對真理有全面覺知的人）、明行足（智慧與實踐皆圓滿）、善逝（已走上解脫之路）、世間解（徹底洞悉世間本質）、無上士（最卓越的覺悟者）、調御丈夫（能調伏身心之能者）、天人師（天與人的導師）、佛世尊。

**◈ 於此如來事生隨喜心，隨喜已，心歡悅，心歡悅已，身猗息，身猗息已，覺受樂，覺受樂已，三昧定，三昧定已，**
　　在憶念如來的這些特質時，心中生起隨喜（對正面的特質感到欣喜）的心；隨喜之後，內心感到歡悅；內心歡悅之後，身體感到放鬆舒暢；身體放鬆之後，感受到快樂的覺受（感受）；感受到快樂之後，進入三昧定（心意識高度專注且寧靜的狀態）。

**◈ 聖弟子作如是學：『何等為諸天天道？』復作是念：『我聞無恚為上諸天天道。』**
　　進入定境後，聖弟子如此思惟學習：「什麼才是最殊勝的天道心境？」接著又想到：「我聽說『無恚』（沒有瞋恨與憤怒）才是最上乘的天道心境。」

**◈ 作是念：『我從今日，於世間若怖若安，不起瞋恚，我但當自受純一滿淨諸天天道。』**
　　於是決定：「我從今天起，在世間無論遇到恐懼或安穩的情況，都不起瞋恨心，我只要讓自己受持這種純淨且圓滿的無恚天道心境。」

**◈ 是名第一諸天天道，未淨眾生令淨，已淨者重令淨。**
　　這就稱為第一種天道。這種修行能讓尚未純淨的眾生變得純淨，讓已經純淨的人更加純淨。

**◈ 「復次，比丘！聖弟子念於法事，謂如來說正法、律，現法離諸熾然，不待時節，通達涅槃，即身觀察，緣自覺知。**
　　另外，比丘！聖弟子憶念關於法（真理）的事項，也就是如來所宣說的正法與律條：這些法能讓人現前就脫離各種熾熱的煩惱（如貪、瞋、癡），不需要等待漫長的時節，就能通達涅槃（超越生死、絕對寧靜的解脫境界），在當下就能親自觀察並自覺體悟。

**◈ 如是知法事已，心生隨喜；隨喜已，身猗息，身猗息已，覺受樂，覺受樂已，三昧定，三昧定已，**
　　如此了解法的事項後，心中生起隨喜；隨喜之後，身體感到放鬆舒暢；身體放鬆之後，感受到快樂的覺受（感受）；感受到快樂之後，進入三昧定（心意識高度專注且寧靜的狀態）。

**◈ 聖弟子作如是學：『何等為諸天天道？』復作是念：『我聞無恚為上諸天天道，我從今日，於此世間若怖若安，不起瞋恚，我當受持純一滿淨諸天天道。』是名第二諸天天道。**
　　進入定境後，聖弟子如此思惟學習：「什麼才是最殊勝的天道心境？」接著又想到：「我聽說『無恚』才是最上乘的天道心境，我從今天起，在世間無論遇到恐懼或安穩的情況，都不起瞋恨心，我應當受持這種純淨且圓滿的無恚天道心境。」這就稱為第二種天道。

**◈ 「復次，比丘！若於僧事起於正念，謂世尊弟子僧正直等向，所應恭敬、尊重、供養，無上福田。**
　　另外，比丘！如果對僧團的事項起正念，也就是憶念世尊的弟子僧團是正直且志向一致的，是應該被恭敬、尊重與供養的，是世間最無上的福田（種植善因能獲得最大果報的對象）。

**◈ 彼如是於諸僧事正憶念已，心生隨喜；心隨喜已，得歡悅，歡悅已，身猗息，身猗息已，覺受樂，覺受樂已，三昧定，三昧定已，**
　　他在對僧團的事項正確認憶念後，心中生起隨喜；隨喜之後，感到歡悅；歡悅之後，身體感到放鬆舒暢；身體放鬆之後，感受到快樂的覺受（感受）；感受到快樂之後，進入三昧定（心意識高度專注且寧靜的狀態）。

**◈ 彼聖弟子作如是學：『何等諸天天道？』復作是念：『我聞諸天無恚為上諸天天道，我從今日，於諸世間若怖若安，不起瞋恚，我但當受持純一滿淨諸天天道。』是名第三諸天天道。**
　　進入定境後，聖弟子如此思惟學習：「什麼才是最殊勝的天道心境？」接著又想到：「我聽說諸天之中，『無恚』才是最上乘的天道心境，我從今天起，在世間無論遇到恐懼或安穩的情況，都不起瞋恨心，我只要讓自己受持這種純淨且圓滿的無恚天道心境。」這就稱為第三種天道。

**◈ 「復次，比丘！謂聖弟子自念所有戒事，隨憶念言：『我於此不缺戒、不污戒、不雜戒、明智所歎戒、智者不厭戒。』**
　　另外，比丘！是指聖弟子回顧自己所持守的戒律事項，在憶念時對自己說：「我在這些戒律上沒有缺失、沒有污染、沒有雜染，我的持戒是能讓明智之人讚嘆的，也是讓智者不厭倦的。」

**◈ 於如是等戒事正憶念已，心生隨喜；隨喜已，歡悅，歡悅已；身猗息，身猗息已，覺受樂，覺受樂已，三昧定，三昧定已，**
　　在對這些持戒事項正確認憶念後，心中生起隨喜；隨喜之後，感到歡悅；歡悅之後，身體感到放鬆舒暢；身體放鬆之後，感受到快樂的覺受（感受）；感受到快樂之後，進入三昧定（心意識高度專注且寧靜的狀態）。

**◈ 聖弟子作是念：『何等為諸天天道？』復作是念：『我聞諸天無恚為上，我從今日，於諸世間若怖若安，不起瞋恚，我當受持純一滿淨諸天天道。』**
　　進入定境後，聖弟子如此思惟：「什麼才是最殊勝的天道心境？」接著又想到：「我聽說諸天之中，『無恚』才是最上乘的，我從今天起，在世間無論遇到恐懼或安穩的情況，都不起瞋恨心，我應當受持這種純淨且圓滿的無恚天道心境。」

**◈ 是名第四諸天天道。未淨眾生令淨，已淨者重令淨。」**
　　這就稱為第四種天道。這種修行能讓尚未純淨的眾生變得純淨，讓已經純淨的人更加純淨。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **隨喜 (Muditā)**：對他人或正面的特質、善行感到共同的喜悅，是一種能消除嫉妒與瞋恨的心理狀態。
2. **三昧定 (Samādhi)**：心意識高度集中於單一對象，排除雜念，達到極度寧靜與穩定的心理狀態。
3. **無恚 (Avihiṃsā/Adosa)**：指心中沒有瞋恨、憤怒或傷害他人的意圖。在本經中，這被視為最上乘的心理狀態（天道）。
4. **福田 (Puññakkhetta)**：比喻能讓修行者透過供養或服務而獲得巨大善果的對象，如具足德行的僧團。
5. **現法離諸熾然 (Diṭṭhadhamma-virāga)**：指在當下的生命中，就能透過修行脫離由貪、瞋、癡所構成的「熾熱」煩惱。

### [現代實作指南]
本經提供了一套從「認知 ➔ 情緒 ➔ 生理 ➔ 定力 ➔ 轉念」的心理調適流程，可用於現代壓力管理與情緒調節：

1. **建立正向錨點（憶念）**：
   - 當你感到焦慮或憤怒時，不要強行壓抑，而是將注意力轉向一個「正向的對象」。
   - **對象選擇**：可以是你崇拜的智者（如佛陀）、一個深刻改變你的真理（如法）、一個正直的團隊（如僧團），或是回顧自己曾經堅持的原則與善良（如戒）。

2. **觸發隨喜機制**：
   - 專注於該對象的「優點」而非「缺陷」。例如：回想某人如何寬容地對待他人，並對這種寬容感感到欣喜。

3. **覺察生理放鬆（身猗息）**：
   - 意識到心靈歡悅時，觀察身體的反應。嘗試將這種心理的喜悅轉化為肌肉的放鬆（如肩膀下沉、呼吸深長），讓身心同步進入放鬆狀態。

4. **進入短暫的寧靜（三昧定）**：
   - 在身體放鬆與心情愉悅的交匯點，保持片刻的靜默，感受當下的平靜，這能有效中斷負面情緒的循環。

5. **設定「無恚」的行為準則**：
   - 在心境平穩時，對自己下達一個明確的指令：「無論接下來遇到什麼情況（恐懼或安穩），我選擇不讓瞋恨心主導我的反應。」這是一種預先的心理建設，將「不憤怒」視為一種最高級的心理享受（天道）。

### [段落大意]
本經闡述了四種能導向喜悅心境（天道）的修行路徑，分別是憶念佛陀的特質、憶念正法的殊勝、憶念僧團的純淨以及憶念自身的持戒。這四種憶念能引發「隨喜 ➔ 歡悅 ➔ 身心放鬆 ➔ 定境」的心理鏈條，最終導向「無恚」（無瞋恨）的最高心境，使修行者在任何環境下都能保持純淨與平靜。
