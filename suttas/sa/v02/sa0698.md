# 雜阿含經 第 698 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有九力。何等為九？謂信力、精進力、慚力、愧力、念力、定力、慧力、數力、修力。何等為信力？於如來所起正信心，深入堅固……」如上說。「何等為精進力？謂四正斷……」如上說。「何等為慚力？……」如上說。「何等為愧力？……」如上說。「何等為念力？謂內身身觀住……」如上說。「何等為定力？謂四禪。何等為慧力？謂四聖諦。何等為數力？謂聖弟子若於閑房、樹下作如是學：『身、口惡行者於現法後世當受惡報……』」如上廣說。「何等為修力？謂修四念處……」如前說。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有九力。何等為九？謂信力、精進力、慚力、愧力、念力、定力、慧力、數力、修力。」**
　　那時，世尊告訴在場的眾位比丘說：「有九種心靈的力量。這九種力量是什麼？指的是：信力（對真理的信心）、精進力（持續努力的能量）、慚力（對自身過錯的羞愧）、愧力（對他人評價的恐懼）、念力（覺知當下的專注）、定力（心不散亂的安定）、慧力（洞察實相的智慧）、數力（對因果規律的計算與省思）、以及修力（實踐修行的能力）。」

**◈ 「何等為信力？於如來所起正信心，深入堅固……」如上說。**
　　「什麼是信力？是指對如來（成就不變真理而來到世間的人）產生正確的信心，且這種信心深入心底且堅固不可動搖……」佛陀就這樣詳細說明。

**◈ 「何等為精進力？謂四正斷……」如上說。**
　　「什麼是精進力？是指四正斷（即：斷除已生惡法、防止未生惡法、生起未生善法、增長已生善法）……」佛陀就這樣詳細說明。

**◈ 「何等為慚力？……」如上說。**
　　「什麼是慚力（指對自己所造惡業而感到羞愧的力量）？……」佛陀就這樣詳細說明。

**◈ 「何等為愧力？……」如上說。**
　　「什麼是愧力（指因擔心他人知曉自己的惡行而感到恐懼的力量）？……」佛陀就這樣詳細說明。

**◈ 「何等為念力？謂內身身觀住……」如上說。**
　　「什麼是念力？是指內身身觀住（在心中對自己的身體狀態保持覺知與觀察）……」佛陀就這樣詳細說明。

**◈ 「何等為定力？謂四禪。」**
　　「什麼是定力？是指四禪（四種由淺入深、心進入絕對寧靜的禪定狀態）。」

**◈ 「何等為慧力？謂四聖諦。」**
　　「什麼是慧力？是指對四聖諦（苦、集、滅、道這四個關於生命真相的聖諦）的深刻洞察。」

**◈ 「何等為數力？謂聖弟子若於閑房、樹下作如是學：『身、口惡行者於現法後世當受惡報……』」如上廣說。**
　　「什麼是數力？是指聖弟子如果在安靜的房間或樹下進行如此的思考與計算：『那些在身體或言語上造下惡行的人，在現世或來世必然會承受相應的惡報（業報）……』」佛陀就這樣廣泛地說明。

**◈ 「何等為修力？謂修四念處……」如前說。**
　　「什麼是修力？是指修習四念處（對色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）以及識（意識）等身心現象進行持續的覺知觀察）……」佛陀就如前所述。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四正斷 (Sammappadhāna)**：精進的四種正確方向 ➔ 1. 斷除已生的惡 2. 防止未生的惡 3. 生起未生的善 4. 增長已生的善。
2. **慚 (Hiri) 與 愧 (Ottappa)**：慚是指內在的自省，對自己的錯誤感到羞愧；愧是指對外在評價的顧慮，恐懼惡行被揭露或受譴責。兩者合稱「慚愧」，是防止造惡的心理防線。
3. **四禪 (Jhana)**：透過專注（定）而達到的四個層次的心靈狀態，從初步的喜樂到最終的絕對寧靜與平等心。
4. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四個真理 ➔ 苦（生命的本質是苦）、集（苦的原因是貪愛）、滅（苦的止息即是涅槃）、道（達成止息的八正道）。
5. **四念處 (Satipaṭṭhāna)**：四種覺知觀察的對象 ➔ 1. 身念處 2. 受念處 3. 心念處 4. 法念處。

### [現代實作指南]
本經提到的「九力」可視為一套完整的「心靈強化系統」，可用於現代生活中的壓力管理與人格成長：

1. **建立正向基調（信力）**：在面對困難時，先建立對「因果規律」的信任，相信只要採取正確的行動，必然會產生正向的結果，避免陷入絕望。
2. **建立道德底線（慚愧力）**：在沒有人監督時，利用「慚」來自我審視，在面對社會規範時，利用「愧」來約束行為，將外部壓力轉化為內在的自律。
3. **實踐正向行動（精進力）**：不要試圖一次改變所有習慣，而應採取「四正斷」策略 ➔ 意識到一個壞習慣（如拖延）並嘗試切斷它，同時刻意培養一個好習慣（如早起）並將其強化。
4. **訓練專注力（念力與定力）**：在碎片化資訊時代，練習「身觀住」 ➔ 每天花5分鐘僅觀察呼吸或身體感覺，將散亂的心重新聚焦，提升工作與學習的效率。
5. **邏輯分析與省思（數力與慧力）**：當面對衝突或失敗時，不要情緒化反應，而應像「數力」一樣進行邏輯推演 ➔ 「我現在的憤怒（因） ➔ 會導致關係破裂（果）」，透過這種因果計算，運用「慧力」洞察問題核心，選擇最理性的應對方式。
6. **將覺知生活化（修力）**：將「四念處」融入日常 ➔ 走路時覺知腳步（身）、感到不適時覺知疼痛（受）、生氣時覺知憤怒（心）、思考時覺知念頭的起伏（法）。
