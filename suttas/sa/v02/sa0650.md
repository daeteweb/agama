# 雜阿含經 第 650 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘……如上說。差別者：「諸比丘！若我於此信根、信根集、信根滅、信根滅道跡不如實知者，我終不得於諸天、魔、梵、沙門、婆羅門中，為出為離，心離顛倒，亦不得成阿耨多羅三藐三菩提。如信根，精進根、念根、定根、慧根亦如是說。

「諸比丘！我於此信根正智如實觀察故，信根集、信根滅、信根滅道跡正智如實觀察故，我於諸天、魔、梵、沙門、婆羅門眾中，為出為離，心離顛倒，成阿耨多羅三藐三菩提。如信根，精進、念、定、慧根亦如是說。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘……如上說。**
　　那時，世尊告訴在場的眾位比丘說……（內容如前所述）。

**◈ 差別者：「諸比丘！若我於此信根、信根集、信根滅、信根滅道跡不如實知者，**
　　不同的部分是：「諸比丘！如果我對於『信根（信仰的力量）』、以及『信根集（導致信仰產生的原因）』、『信根滅（信仰之苦的止息）』、『信根滅道跡（導向信仰之苦止息的解脫之道）』不能如實地認知，

**◈ 我終不得於諸天、魔、梵、沙門、婆羅門中，為出為離，心離顛倒，亦不得成阿耨多羅三藐三菩提。**
　　我最終將無法在諸天、魔、梵、沙門、婆羅門這些種種存在之中，獲得出離與解脫，無法讓心脫離顛倒的錯誤認知，也無法成就最高、最圓滿的智慧與覺悟 (阿耨多羅三藐三菩提)。

**◈ 如信根，精進根、念根、定根、慧根亦如是說。**
　　正如關於信根的說明，關於精進根（努力的力量）、念根（正念的力量）、定根（專注的力量）、慧根（智慧的力量）也是同樣的道理。

**◈ 「諸比丘！我於此信根正智如實觀察故，信根集、信根滅、信根滅道跡正智如實觀察故，**
　　「諸比丘！因為我以正確的智慧如實地觀察了信根（信仰的力量），且同樣以正確的智慧如實觀察了信根集（導致信仰產生的原因）、信根滅（信仰之苦的止息）以及信根滅道跡（導向信仰之苦止息的解脫之道），

**◈ 我於諸天、魔、梵、沙門、婆羅門眾中，為出為離，心離顛倒，成阿耨多羅三藐三菩提。**
　　所以我才能在諸天、魔、梵、沙門、婆羅門這些眾生之中，獲得出離與解脫，使心脫離顛倒的錯誤認知，並成就最高、最圓滿的智慧與覺悟 (阿耨多羅三藐三菩提)。

**◈ 如信根，精進、念、定、慧根亦如是說。」**
　　正如關於信根的說明，關於精進、念、定、慧根也是同樣的道理。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **五根 (Indriya)**：指在修行中起主導作用的五種心理能力，包括信、精進、念、定、慧。
2. **集 (Samudaya) ➔ 滅 (Nirodha) ➔ 道跡 (Magga)**：此處將「四聖諦」的邏輯應用於五根。集是指該心理狀態產生的原因；滅是指該狀態所帶來的束縛或苦痛的止息；道跡則是達成止息的具體修行路徑。
3. **顛倒 (Viparyāsa)**：指對現實的錯誤認知，例如將無常視為常恆、將苦視為樂、將無我視為我、將不淨視為淨。
4. **阿耨多羅三藐三菩提 (Anuttara-samyak-sambodhi)**：指最高、最圓滿的智慧與覺悟，是修行者的最終目標。

### [現代實作指南]
本經文的核心在於將「四聖諦」的分析法應用到自身的心理功能（五根）上。在現代生活中，我們可以將其轉化為一種「心理機制分析法」：

1. **識別功能 (信/精進/念/定/慧)**：
   當你感到缺乏動力（精進不足）或無法專注（定力不足）時，不要僅僅將其視為「我的缺點」，而將其視為一種「根（功能）」的運作狀態。

2. **分析原因 (集)**：
   問自己：「是什麼原因導致我現在缺乏信心或專注力？」➔ 找出背後的執著、恐懼或外界干擾（即「集」）。

3. **設定目標 (滅)**：
   明確定義什麼是「止息」的狀態。例如：不再被焦慮牽引，而是能平靜地面對當下。

4. **制定路徑 (道跡)**：
   採取具體的行動來修正。例如：若發現缺乏專注（定根不足）是因為手機干擾（集），則實作「在工作時將手機關機」的具體行動（道跡）。

透過這種「如實觀察」的過程，我們能將情緒化的反應轉化為理性的認知，從而脫離對自我狀態的「顛倒」認知（如：我就是個失敗的人 ➔ 只是目前的精進根受阻）。

### [段落大意]
本段經文強調，修行者必須以正確的智慧，將四聖諦（苦、集、滅、道）的分析框架應用於五根（信、精進、念、定、慧）的觀察上。唯有如實了解這些心理功能的產生原因、止息狀態及其達成路徑，才能脫離各種存在層級的束縛與認知顛倒，最終成就最高圓滿的覺悟。
