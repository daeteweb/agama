# 雜阿含經 第 745 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若比丘修空入處，多修習已，得大果大福利。云何比丘修空入處，多修習已，得大果大福利？是比丘心與空入處俱，修念覺分，依遠離、依無欲、依滅、向於捨；乃至修捨覺分，依遠離、依無欲、依滅、向於捨。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如修空入處，如是識入處、無所有入處、非想非非想入處三經，亦如上說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若比丘修空入處，多修習已，得大果大福利。**
　　「如果比丘修行『空入處』（一種體悟空間無限、空無一物的禪定境界），並且經過大量的修習之後，將會獲得巨大的果報與福德利益。

**◈ 云何比丘修空入處，多修習已，得大果大福利？**
　　那麼，比丘是如何修行空入處並經過大量修習，進而獲得巨大果報與福德利益的呢？

**◈ 是比丘心與空入處俱，修念覺分，依遠離、依無欲、依滅、向於捨；**
　　這位比丘在心與空入處（空無一物的禪定境界）相結合的狀態下，修行『念覺分』（覺悟的要素之一：正念），這種修行是依據遠離世俗、依據無欲、依據滅除煩惱，並趨向於捨（不執著的平等心）；

**◈ 乃至修捨覺分，依遠離、依無欲、依滅、向於捨。」**
　　直到修行到『捨覺分』（覺悟的要素之一：捨），同樣也是依據遠離、依據無欲、依據滅除煩惱，並趨向於捨（不執著的平等心）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如修空入處，如是識入處、無所有入處、非想非非想入處三經，亦如上說。**
　　正如修行空入處的方法一樣，關於『識入處』（意識無限的境界）、『無所有入處』（什麼都沒有的境界）以及『非想非非想入處』（超越想與非想的極微細境界）這三部經中的修行方法，也同樣如上述般說明。

### [詞語註解]
1. **空入處 (Ākāsānañcāyatana)**：無色界四禪的第一層，修行者將意識集中於「無限的空間」，捨棄對物質形態（色）的執著，進入一種空靈的定境。
2. **覺分 (Bojjhaṅga)**：指導向覺悟的七種心理要素（念、擇法、精進、喜、輕安、定、捨）。本經強調將這些覺悟要素與深層定境結合。
3. **捨 (Upekkhā)**：一種不偏不倚、不貪不憎的心理平衡狀態，是最高層次的心理穩定，能讓修行者在面對任何現象時保持中立。
4. **識入處 (Viññāṇañcāyatana)**：無色界四禪之第二層，意識不再專注於空間，而專注於能感知空間的「意識」本身，體悟意識的無限。
5. **無所有入處 (Ākiñcaññāyatana)**：無色界四禪之第三層，體悟到「這裡什麼都沒有」的狀態，超越了對意識的執著。
6. **非想非非想入處 (Nevasaññānāsaññāyatana)**：無色界四禪之最高層，意識微細到不能說有「想」（認知），但又不能說完全沒有「想」的極致定境。

### [現代實作指南]
本經揭示了「定（集中）」與「慧（覺察）」結合的關鍵。許多人在追求內心寧靜（如冥想、放空）時，容易陷入單純的昏沉或空白，而缺乏轉化能力。

**實作步驟：**
1. **建立基礎寧靜（對應空入處）**：
   在生活中練習「空間化」思考。當壓力襲來時，想像將壓力放置在一個無限寬廣的空間中，讓壓力在巨大的空間感中變得微小，以此獲得初步的心理空間。
2. **注入覺察要素（對應覺分）**：
   在寧靜的狀態中，不要僅僅是「放空」，而要啟動「念（Mindfulness）」。觀察這個寧靜的狀態，意識到「我現在正處於寧靜中」，將意識從單純的感受提升到對感受的覺察。
3. **執行「遠離 ➔ 無欲 ➔ 滅 ➔ 捨」的轉念路徑**：
   ➔ **遠離**：意識到當前的煩惱源於對某件事的過度關注，主動將注意力移開。
   ➔ **無欲**：體認到「想要改變現狀」的欲望正是痛苦的來源，嘗試放下「必須如何」的執著。
   ➔ **滅**：觀察欲望在失去支持後如何逐漸熄滅。
   ➔ **捨**：最終達到一種「無論結果如何，我都能平靜面對」的平等心。

### [段落大意]
本經說明修行無色界定（如空入處）若要獲得真正的解脫果報，不能僅止於定境，必須在定中結合「七覺分」的修行，透過遠離、無欲、滅除煩惱，最終趨向於「捨」的平等心。此方法同樣適用於識入處、無所有入處及非想非非想入處。
