# 雜阿含經 第 916 經

### [原文]
如是我聞：

一時，佛住那羅聚落好衣菴羅園中。

時，有刀師氏聚落主，尼揵弟子，來詣佛所，稽首禮足，退坐一面。

爾時，世尊告聚落主：「欲何所論？尼揵若提子為何所說？」

聚落主言：「彼尼揵若提子說：『殺生者，一切皆墮泥犁中，以多行故，則將至彼；如是盜、邪婬、妄語皆墮泥犁中，以多行故，則將至彼。』」

佛告聚落主：「若如尼犍若提子說殺生者墮泥犁中，以多行故，而往生彼者，則無有眾生墮泥犁中。聚落主！於意云何？何等眾生於一切時有心殺生？復於何時有心不殺生，乃至何時有心妄語，何時有心不妄語？」

聚落主白佛言：「世尊！人於晝夜，少時有心殺生，乃至少時有心妄語，而多時不有心殺生，乃至妄語。」

佛告聚落主：「若如是者，豈非無有人墮於泥犁中耶？如尼揵所說：『有人殺生者，一切墮泥犁中，多習行者將往生彼，乃至妄語亦復如是。』聚落主！彼大師出興于世，覺想籌量，入覺想地住。於凡夫地自辯所說，隨意籌量，為諸弟子作如是說法，言殺生者，一切皆墮泥犁中，多習行將往生彼，乃至妄語亦復如是。彼諸弟子若信其說，言：『我大師知其所知，見其所見，能為弟子作如是說：「若殺生者，一切皆墮泥犁中，多習行故，將往生彼。」我本有心殺生、偷盜、邪婬、妄語，當墮泥犁中。』得如是見，乃至不捨此見，不厭彼業，不覺彼悔，於未來世，不捨殺生，乃至不捨妄語，彼意解脫不滿足，慧解脫亦不滿足。意解脫不滿足、慧解脫不滿足故，則為謗聖邪見；邪見因緣故，身壞命終，生惡趣泥犁中。如是，聚落主！有因、有緣眾生煩惱，有因、有緣眾生業煩惱。

「聚落主！如來、應、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊出興於世，常為眾生呵責殺生，讚歎不殺；呵責偷盜、邪婬、妄語，讚歎不盜、不婬、不妄語。常以此法，化諸聲聞，令念、樂、信、重，言：『我大師知其所知，見其所見，呵責殺生，讚歎不殺，乃至呵責妄語，讚歎不妄語，我從昔來，以愚癡無慧，有心殺生，我緣是故，今自悔責。』雖不能令彼業不為，且因此悔責故，於未來世，得離殺生，乃至得離盜、婬、妄語，亦得滿足正意解脫，滿足慧解脫，意解脫、慧解脫滿足已，得不謗賢聖，正見成就；正見因緣故，得生善趣天上。如是，聚落主！有因、有緣眾生業煩惱清淨。

「聚落主！彼多聞聖弟子作如是學：『隨時晝夜觀察所起少有心殺生、多有心不殺生。』若於有心殺生，當自悔責，不是不類。若不有心殺生，無怨無憎，心生隨喜；隨喜已，歡喜生；歡喜已，心猗息；心猗息已，心受樂；受樂已，則心定。心定已，聖弟子心與慈俱，無怨無嫉，無有瞋恚，廣大無量，滿於一方，正受住；二方、三方，乃至四方、四維、上下、一切世間，心與慈俱，無怨無嫉，無有瞋恚，廣大無量，善修習，充滿諸方，具足正受住。」

爾時，世尊以爪甲抄少土，語刀師氏聚落主言：「云何？聚落主！我爪甲土多？大地為多？」

聚落主白佛言：「世尊！爪甲土少少耳，大地土無量無數。」

佛告聚落主：「如甲上之土甚少，大地之土其數無量。如是心與慈俱，修習多修習，諸有量業者，如甲上土，不能將去，不能令住。如是偷盜對以悲心，邪婬對以喜心，妄語對以捨心，不得為比。」

說是語時，刀師氏聚落主遠塵離垢，得法眼淨，聚落主見法、得法、覺法、知法、深入於法，離諸狐疑不由於他、不隨於他，於正法、律得無所畏。從坐起，整衣服，右膝著地，合掌白佛：「我已度。世尊！已越。世尊！我今歸佛、歸法、歸比丘僧，盡其壽命為優婆塞。世尊！譬如士夫欲求燈明，取其馬尾，以為燈炷，欲吹令然，終不得明，徒自疲勞，燈竟不然。我亦如是，欲求明智，於諸愚癡尼揵子所，愚癡習近，愚癡和合，愚癡奉事，徒自勞苦，不得明智，是故我今重歸依佛、歸法、歸僧。從今以去，於彼尼揵愚癡不善不辯者所，少信、少敬、少愛、少念，於今遠離。是故，我今第三歸佛、歸法、歸僧，乃至盡壽，為優婆塞，自淨其心。」

時，刀師氏聚落主聞佛所說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住那羅聚落好衣菴羅園中。**
　　在某一個時間，佛陀住在那羅聚落中一座環境優美、種滿菴羅樹（芒果樹）的園林裡。

**◈ 時，有刀師氏聚落主，尼揵弟子，來詣佛所，稽首禮足，退坐一面。**
　　那時，有一位姓氏為刀師氏的村長，他是尼揵（耆那教）的追隨者，來到佛陀這裡，恭敬地頂禮佛陀的足部，然後退到一邊坐下。

**◈ 爾時，世尊告聚落主：「欲何所論？尼揵若提子為何所說？」**
　　那時，世尊告訴這位村長：「你想要討論什麼話題？尼揵若提子（耆那教導師）是如何教導你們的？」

**◈ 聚落主言：「彼尼揵若提子說：『殺生者，一切皆墮泥犁中，以多行故，則將至彼；如是盜、邪婬、妄語皆墮泥犁中，以多行故，則將至彼。』」**
　　村長回答：「那位尼揵若提子說：『凡是殺生的人，全部都會墮入泥犁（地獄）之中；如果殺生行為很多，就必然會去那裡。同樣地，偷盜、邪婬、妄語的人也全部都會墮入泥犁之中，因為這些惡行做多了，就必然會去那裡。』」

**◈ 佛告聚落主：「若如尼犍若提子說殺生者墮泥犁中，以多行故，而往生彼者，則無有眾生墮泥犁中。**
　　佛陀告訴村長：「如果像尼揵若提子所說的那樣，只要殺生就會墮入泥犁，且因為行為多而往生彼處，那麼（事實上）就沒有任何眾生能逃脫墮入泥犁的命運了（因為幾乎所有眾生都曾殺生）。

**◈ 聚落主！於意云何？何等眾生於一切時有心殺生？復於何時有心不殺生，乃至何時有心妄語，何時有心不妄語？」**
　　村長！你的看法如何？有什麼樣的眾生在所有時間裡都一直存有殺生的心？又有什麼時候存有不殺生的心？甚至什麼時候存有妄語的心，什麼時候存有不妄語的心？」

**◈ 聚落主白佛言：「世尊！人於晝夜，少時有心殺生，乃至少時有心妄語，而多時不有心殺生，乃至妄語。」**
　　村長對佛陀恭敬地說道：「世尊！人在白天黑夜之中，只有少部分時間會存有殺生的心，或少部分時間存有妄語的心，而大部分時間並沒有存有殺生或妄語的心。」

**◈ 佛告聚落主：「若如是者，豈非無有人墮於泥犁中耶？**
　　佛陀告訴村長：「如果真是這樣，那豈不是沒有人會墮入泥犁之中嗎？（這證明了尼揵若提子的說法在邏輯上是不成立的）。

**◈ 如尼揵所說：『有人殺生者，一切墮泥犁中，多習行者將往生彼，乃至妄語亦復如是。』**
　　就像尼揵所說的：『只要有人殺生，全部都會墮入泥犁，經常這樣做的人必然往生彼處，妄語也是同樣道理。』

**◈ 聚落主！彼大師出興于世，覺想籌量，入覺想地住。於凡夫地自辯所說，隨意籌量，為諸弟子作如是說法，言殺生者，一切皆墮泥犁中，多習行將往生彼，乃至妄語亦復如是。**
　　村長！那位大師出現在世上，是憑藉自己的思考與推論，進入一種自以為覺悟的狀態中。他在凡夫的層次上，根據自己的辯才，隨意地推論，為弟子們作這樣的說法，宣稱殺生者全部墮泥犁，多習行者必往生彼處，妄語亦然。

**◈ 彼諸弟子若信其說，言：『我大師知其所知，見其所見，能為弟子作如是說：「若殺生者，一切皆墮泥犁中，多習行故，將往生彼。」我本有心殺生、偷盜、邪婬、妄語，當墮泥犁中。』**
　　那些弟子如果盲目相信他的說法，會想：「我的大師知道他所知道的，看到他所看到的，能為弟子這樣說：『殺生者全部墮泥犁，因為多習行而往生彼處。』既然我過去曾有心殺生、偷盜、邪婬、妄語，那我必然會墮入泥犁之中。」

**◈ 得如是見，乃至不捨此見，不厭彼業，不覺彼悔，於未來世，不捨殺生，乃至不捨妄語，彼意解脫不滿足，慧解脫亦不滿足。**
　　一旦產生這種見解，如果不能捨棄這種（絕望的）見解，就不會對過去的惡業產生厭離心，也不會真正覺悟並悔改。在未來世中，他依然不會捨棄殺生或妄語。這樣的人，其意解脫（心靈從執著中解脫）無法滿足，慧解脫（透過智慧而解脫）也無法滿足。

**◈ 意解脫不滿足、慧解脫不滿足故，則為謗聖邪見；邪見因緣故，身壞命終，生惡趣泥犁中。如是，聚落主！有因、有緣眾生煩惱，有因、有緣眾生業煩惱。**
　　因為意解脫與慧解脫都無法滿足，這就變成了誹謗聖者、持有邪見（錯誤的認知）。因為持有邪見作為因緣，當身體毀壞、生命終結時，就會生在惡趣的泥犁之中。就像這樣，村長！這就是因為（錯誤的教導）作為因緣而產生的眾生煩惱，以及因為（錯誤認知導致的惡行）作為因緣而產生的業煩惱。

**◈ 「聚落主！如來、應、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊出興於世，常為眾生呵責殺生，讚歎不殺；呵責偷盜、邪婬、妄語，讚歎不盜、不婬、不妄語。**
　　村長！如來、應供、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊出現在世上，總是為眾生呵責殺生之行，讚歎不殺之德；呵責偷盜、邪婬、妄語，讚歎不盜、不婬、不妄語。

**◈ 常以此法，化諸聲聞，令念、樂、信、重，言：『我大師知其所知，見其所見，呵責殺生，讚歎不殺，乃至呵責妄語，讚歎不妄語，我從昔來，以愚癡無慧，有心殺生，我緣是故，今自悔責。』**
　　佛陀經常以此法教化諸位聲聞，令他們能憶念、喜愛、信仰並尊重，讓他們能說：「我的大師知道他所知道的，看到他所看到的，他呵責殺生，讚歎不殺，乃至呵責妄語，讚歎不妄語。我從過去以來，因為愚癡缺乏智慧，曾有心殺生，因此我現在對此深感悔責。」

**◈ 雖不能令彼業不為，且因此悔責故，於未來世，得離殺生，乃至得離盜、婬、妄語，亦得滿足正意解脫，滿足慧解脫，意解脫、慧解脫滿足已，得不謗賢聖，正見成就；正見因緣故，得生善趣天上。如是，聚落主！有因、有緣眾生業煩惱清淨。**
　　雖然悔責不能讓過去已經造的業消失，但因為這種真正的悔責，在未來世中，能使他遠離殺生，乃至遠離偷盜、邪婬、妄語，並且能獲得滿足的正意解脫與滿足的慧解脫。當意解脫與慧解脫滿足後，就能不再誹謗賢聖，成就正見（正確的認知）。因為正見作為因緣，就能生在善趣的天上。就像這樣，村長！這就是因為（正確的教導與悔改）作為因緣而使眾生的業煩惱得到清淨。

**◈ 「聚落主！彼多聞聖弟子作如是學：『隨時晝夜觀察所起少有心殺生、多有心不殺生。』**
　　村長！那些博學的聖弟子會這樣修行：『在晝夜之中隨時觀察自己的心，發現自己只有少部分時間存有殺心，而大部分時間都沒有殺心。』

**◈ 若於有心殺生，當自悔責，不是不類。若不有心殺生，無怨無憎，心生隨喜；隨喜已，歡喜生；歡喜已，心猗息；心猗息已，心受樂；受樂已，則心定。**
　　如果發現自己曾有殺心，就應該自我悔責，不讓這種心態持續。如果發現自己沒有殺心，對眾生沒有怨恨與憎惡，心中就生起隨喜（對善行的認同）；隨喜之後，生起歡喜；歡喜之後，心靈感到安息；安息之後，心靈感受到快樂；感受到快樂之後，心就安定下來。

**◈ 心定已，聖弟子心與慈俱，無怨無嫉，無有瞋恚，廣大無量，滿於一方，正受住；二方、三方，乃至四方、四維、上下、一切世間，心與慈俱，無怨無嫉，無有瞋恚，廣大無量，善修習，充滿諸方，具足正受住。」**
　　心安定之後，聖弟子的心便與慈心（願眾生得樂）相隨，沒有怨恨與嫉妒，沒有瞋恚（憤怒），這種慈心廣大無量，充滿在一個方向，正念地維持；接著擴展到二方、三方，乃至四方、四維、上下、整個世間，心與慈心相隨，沒有怨恨嫉妒與瞋恚，廣大無量，經過善巧的修習，充滿所有方向，完整地維持這種正受狀態。

**◈ 爾時，世尊以爪甲抄少土，語刀師氏聚落主言：「云何？聚落主！我爪甲土多？大地為多？」**
　　那時，世尊用指甲輕輕地撥起一點土壤，對刀師氏村長說：「你覺得如何？村長！是我指甲上的土多，還是大地上的土多？」

**◈ 聚落主白佛言：「世尊！爪甲土少少耳，大地土無量無數。」**
　　村長對佛陀恭敬地說道：「世尊！指甲上的土極其微小，而大地的土則是無量無數的。」

**◈ 佛告聚落主：「如甲上之土甚少，大地之土其數無量。如是心與慈俱，修習多修習，諸有量業者，如甲上土，不能將去，不能令住。如是偷盜對以悲心，邪婬對以喜心，妄語對以捨心，不得為比。」**
　　佛陀告訴村長：「就像指甲上的土極少，而大地的土無量一樣。當一個人修習慈心，且大量地修習時，那些有量（有限）的惡業，就像指甲上的土一樣微小，無法將人拉走（墮入惡道），也無法讓惡業長久停留。同樣地，用悲心來對治偷盜，用喜心來對治邪婬，用捨心（平等心）來對治妄語，這些惡業在巨大的正向心力面前，根本無法相比。」

**◈ 說是語時，刀師氏聚落主遠塵離垢，得法眼淨，聚落主見法、得法、覺法、知法、深入於法，離諸狐疑不由於他、不隨於他，於正法、律得無所畏。**
　　佛陀說完這段話時，刀師氏村長遠離了煩惱的塵垢，獲得了法眼淨（能洞察真理的清淨眼光）。村長見到了法、得到了法、覺悟了法、知道了法，並深入地體會法，除去了所有狐疑，不再依賴他人或隨波逐流，對於正法與律儀感到無所畏懼。

**◈ 從坐起，整衣服，右膝著地，合掌白佛：「我已度。世尊！已越。世尊！我今歸佛、歸法、歸比丘僧，盡其壽命為優婆塞。**
　　他從座位上站起來，整理衣服，右膝跪在地上，合掌對佛陀說：「我已經度過了苦海。世尊！我已經越過了障礙。世尊！我現在歸依佛、歸依法、歸依比丘僧，在盡其壽命之前都願意成為一名優婆塞（在家男弟子）。

**◈ 世尊！譬如士夫欲求燈明，取其馬尾，以為燈炷，欲吹令然，終不得明，徒自疲勞，燈竟不然。**
　　世尊！就像一個男人想要獲得燈光，卻拿馬尾巴來當作燈炷，試圖吹氣點燃它，結果終究得不到光明，只是徒然疲勞，燈始終沒有燃起。

**◈ 我亦如是，欲求明智，於諸愚癡尼揵子所，愚癡習近，愚癡和合，愚癡奉事，徒自勞苦，不得明智，是故我今重歸依佛、歸法、歸僧。**
　　我也像這樣，想要追求明智，卻在愚癡的尼揵弟子身邊，愚癡地親近、愚癡地和合、愚癡地奉事，徒然勞苦，卻得不到明智。因此，我現在重新歸依佛、歸依法、歸依僧。

**◈ 從今以去，於彼尼揵愚癡不善不辯者所，少信、少敬、少愛、少念，於今遠離。是故，我今第三歸佛、歸法、歸僧，乃至盡壽，為優婆塞，自淨其心。」**
　　從今以後，對於那些愚癡、不善、不辯的尼揵追隨者，我將減少信仰、減少尊敬、減少喜愛、減少憶念，從此遠離。因此，我現在第三次歸依佛、歸依法、歸依僧，直到壽命結束，作為一名優婆塞，自淨其心。」

**◈ 時，刀師氏聚落主聞佛所說，歡喜隨喜，作禮而去。**
　　那時，刀師氏村長聽聞佛陀所說的教法，心生法喜並依照教導去實踐，向佛陀行禮後才離開。

### [詞語註解]
1. **泥犁 (Niraya)**：梵語意為「無光處」，即地獄。在原始佛教中，它是由於強烈的瞋心與惡業而導致的極端痛苦之重生處。
2. **意解脫 (Ceto-vimutti)**：指心靈透過定力（Samādhi）與禪修，從貪欲與瞋恚等煩惱中獲得的自由與解脫。
3. **慧解脫 (Paññā-vimutti)**：指透過對四聖諦、無常、無我等真理的深刻洞察（智慧），徹底斷除無明而獲得的解脫。
4. **正見 (Sammā-diṭṭhi)**：八正道之首，指正確的認知與見解，特別是指對因果律（業報）與四聖諦的正確理解。
5. **慈 (Mettā)**：一種對所有眾生不分彼此，願其獲得快樂、平安的無條件之愛與善意。
6. **法眼淨 (Dhammacakkhu)**：指修行者在禪定中首次直接體悟到「凡所有相皆是虛妄」或「緣起法」的真理，不再僅僅依賴邏輯推論。

### [現代實作指南]
本經文提供了一套從「罪惡感」轉化為「正向能量」的心理實踐路徑，可用於處理現代人的愧疚感與自我否定：

1. **從「絕對定論」轉向「覺察觀察」**：
   - **轉念技巧**：不要陷入「我做錯了某事，所以我是一個糟糕的人/我注定失敗」的絕對定論（如尼揵的教法）。
   - **實作**：練習觀察心念。記錄一天中，有多少時間在產生負面念頭（如憤怒、嫉妒），又有多少時間是平靜或善良的。你會發現，負面念頭僅佔少數，這能幫助你從自我厭惡中解脫。

2. **正確的悔責 (Regret vs. Guilt)**：
   - **區分**：將「病態的罪惡感」（導致絕望、放棄改變）轉化為「建設性的悔責」（承認錯誤 ➔ 產生厭離心 ➔ 決定不再重複）。
   - **實作**：當意識到錯誤時，對自己說：「我過去因為缺乏智慧而這樣做，我現在對此感到遺憾，並決定從此刻起採取不同的行動。」

3. **以「量級對沖」法清除心理陰影**：
   - **核心邏輯**：用巨大的正向心力（慈心）來稀釋少量的負面業力（惡行）。
   - **實作步驟**：
     - **第一步（隨喜）**：看到他人做善事時，真心感到高興，將其正能量內化。
     - **第二步（擴展慈心）**：每天花 5-10 分鐘，想像將善意從自己開始，擴展到家人 ➔ 朋友 ➔ 陌生人 ➔ 討厭的人 ➔ 全世界。
     - **第三步（對治法）**：
       - 感到貪婪/偷盜心 ➔ 練習「悲心」（體會他人失去的痛苦）。
       - 感到不滿足/邪婬心 ➔ 練習「喜心」（為他人的幸福感到快樂）。
       - 感到想妄語/評判 ➔ 練習「捨心」（接納事實，不加評判）。

4. **警惕「馬尾燈炷」的盲從**：
   - **反思**：檢查自己是否在追求心靈成長時，追隨了那些只提供恐懼（如：不這樣做就會下地獄）或簡單答案的導師。
   - **實作**：真正的智慧應帶來內心的安定與慈悲，而非恐懼與壓力。

### [段落大意]
本經記述佛陀與一名尼揵弟子（村長）的對話。佛陀反駁了尼揵教「只要殺生必墮地獄」的極端論點，指出這種教法會導致追隨者陷入絕望與邪見，反而增加墮落的風險。佛陀提出正確的修行路徑：透過覺察心念、真誠悔責，並大量修習廣大的「慈心」來對治過去的惡業。佛陀以「指甲土」比喻有限的惡業，以「大地土」比喻無量的慈心，說明正向心力能淨化業障。最終，村長體悟到盲從愚癡導師的徒勞，轉而歸依佛法，獲得心靈的清淨與解脫。
