# 雜阿含經 第 553 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。爾時，尊者摩訶迦旃延在釋氏訶梨聚落。

聚落主長者詣尊者摩訶迦旃延所，稽首禮足，退坐一面，問尊者摩訶迦旃延：「如世尊界隔山石窟中為天帝釋說言：『憍尸迦！若沙門、婆羅門無上愛盡解脫，心善解脫，邊際究竟，究竟無垢，究竟梵行，畢竟清淨。』云何於此法、律究竟邊際，究竟無垢，究竟梵行，畢竟清淨？」

尊者迦旃延語長者言：「若比丘眼界取，心法境界繫著使，彼若盡、無欲、息、沒，於此法、律究竟邊際，究竟無垢，究竟梵行，究竟清淨。耳、鼻、舌、身、意、意界取，心法境界繫著使，若盡、離、滅、息、沒，於此法、律究竟邊際，究竟無垢，究竟梵行，畢竟清淨。」

時，訶梨聚落主長者聞尊者摩訶迦旃延所說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛住在舍衛國的祇樹給孤獨園。

**◈ 爾時，尊者摩訶迦旃延在釋氏訶梨聚落。**
　　那時，尊者摩訶迦旃延住在釋迦族的訶梨聚落。

**◈ 聚落主長者詣尊者摩訶迦旃延所，稽首禮足，退坐一面，**
　　聚落的首領長者前往尊者摩訶迦旃延的地方，頂禮他的雙足，然後退到一邊坐下，

**◈ 問尊者摩訶迦旃延：「如世尊界隔山石窟中為天帝釋說言：『憍尸迦！若沙門、婆羅門無上愛盡解脫，心善解脫，邊際究竟，究竟無垢，究竟梵行，畢竟清淨。』**
　　他請問尊者摩訶迦旃延：「聽說世尊在山石窟中對天帝釋說道：『憍尸迦！如果沙門或婆羅門能達到徹底斷除貪愛而獲得的最高解脫，心靈得到良好的解脫，達到終極的境界，完全沒有染污，實踐徹底清淨的修行生活，並達到最終的純淨。』

**◈ 云何於此法、律究竟邊際，究竟無垢，究竟梵行，畢竟清淨？」**
　　請問關於這套法（真理）與律（戒律），如何才能達到終極的境界、完全沒有染污、徹底清淨的修行生活以及最終的純淨？」

**◈ 尊者迦旃延語長者言：「若比丘眼界取，心法境界繫著使，**
　　尊者迦旃延告訴長者說：「如果一位比丘在眼界（視覺感官領域）產生執取，被心法境界（心靈所認知的現象）作為束縛的枷鎖，

**◈ 彼若盡、無欲、息、沒，於此法、律究竟邊際，究竟無垢，究竟梵行，究竟清淨。**
　　而他能將這些執取徹底斷除、不再有欲望、使其平息並消失，那麼他對於這套法與律就達到了終極的境界、完全沒有染污、徹底清淨的修行生活以及最終的純淨。

**◈ 耳、鼻、舌、身、意、意界取，心法境界繫著使，若盡、離、滅、息、沒，於此法、律究竟邊際，究竟無垢，究竟梵行，畢竟清淨。」**
　　同樣地，如果對於耳、鼻、舌、身、意以及意界（意識領域）的執取，以及被心法境界（心靈所認知的現象）作為束縛的枷鎖，都能夠徹底斷除、遠離、滅盡、平息並消失，那麼他對於這套法與律就達到了終極的境界、完全沒有染污、徹底清淨的修行生活以及最終的純淨。」

**◈ 時，訶梨聚落主長者聞尊者摩訶迦旃延所說，歡喜隨喜，作禮而去。**
　　那時，訶梨聚落的首領長者聽完尊者摩訶迦旃延的解說後，心中感到歡喜並隨喜贊嘆，向尊者行禮後才離開。

### [詞語註解]
1. **愛盡 (Virāga)**：指徹底斷除對五蘊（物質身體、感受、想像與認知、心的造作、意識）的貪愛，這是達到解脫的核心條件。
2. **界 (Dhātu)**：指構成經驗的基本元素或領域。如「眼界」指視覺感官及其運作的領域。
3. **繫著使 (Samyojana)**：指將眾生束縛在輪迴（Saṃsāra）之中的心理枷鎖，如貪欲、嗔恨、無明等。
4. **梵行 (Brahmacariya)**：原意為清淨的生活方式，在佛教中指為了追求涅槃而實踐的聖潔修行生活。
5. **取 (Upādāna)**：指對感官對象的執著與抓取，是導致苦（Dukkha）產生的直接原因。

### [現代實作指南]
本經文揭示了「束縛」是如何產生的：它始於感官（眼、耳、鼻、舌、身、意）與對象的接觸，隨後產生「取」（執著），最終形成「繫著使」（心理枷鎖）。

**實作步驟：**
1. **覺察感官觸發**：在日常生活中，當你看到一件喜歡的物品（眼界取）或聽到一句讚美的話（耳界取）時，立即標記這個動作：「現在有接觸發生了」。
2. **識別「取」的瞬間**：觀察心中是否升起「我想擁有它」或「我希望這種感覺持續」的渴望。這個「想要」的瞬間就是「取」。
3. **切斷繫著使**：意識到這種渴望會將你束縛在情緒的波動中。嘗試用「無常 (Anicca)」的觀念提醒自己：這個視覺對象或聽覺感受在下一秒就會改變。
4. **練習「息」與「沒」**：不強行壓抑欲望，而是透過觀察欲望的升起與消逝，讓它自然平息（息）並消失（沒）。

**轉念技巧：**
將「我想要這個」轉化為「我的心現在產生了執取的傾向」。透過將自己從「參與者」變為「觀察者」，即可將感官的束縛轉化為修行的契機。

### [段落大意]
本經記述聚落長者請教尊者摩訶迦旃延關於「最高解脫與純淨」的定義。尊者解釋，真正的純淨與解脫，在於徹底斷除六根（眼耳鼻舌身意）對境界的執取，並消除心靈被現象所束縛的枷鎖，使欲望平息消失，從而達到法與律的終極境界。
