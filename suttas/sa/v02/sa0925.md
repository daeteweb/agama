# 雜阿含經 第 925 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，世尊告諸比丘：「世間良馬有八種德成就者，隨人所欲，取道多少。何等為八？生於良馬鄉，是名良馬第一之德。復次，體性溫良，不驚恐人，是名良馬第二之德。復次，良馬不擇飲食，是名良馬第三之德。復次，良馬厭惡不淨，擇地而臥，是名良馬第四之德。復次，良馬諸情態速為調馬者現，馬師調習，速捨其態，是名良馬第五之德。復次，良馬安於駕乘，不顧餘馬，隨其輕重，能盡其力，是名良馬第六之德。復次，良馬常隨正路，不隨非道，是名良馬第七之德。復次，良馬若病、若老，勉力駕乘，不厭不倦，是名良馬第八之德。

「如是，丈夫於正法、律八德成就，當知是賢士夫。何等為八？謂賢士夫住於正戒波羅提木叉律儀，威儀行處具足，見微細罪，能生怖畏，受持學戒，是名丈夫於正法、律第一之德。復次，丈夫性自賢善，善調善住，不惱、不怖諸梵行者，是名丈夫第二之德。復次，丈夫次行乞食，隨其所得，若麁、若細，其心平等，不嫌、不著，是名丈夫第三之德。復次，丈夫心生厭離於身惡業，口、意惡業，惡不善法及諸煩惱，重受諸有熾然苦報，於未來世生、老、病、死、憂、悲、惱、苦，增其厭離，是名丈夫第四之德。復次，丈夫若有沙門過，諂曲不實，速告大師及善知識，大師說法則時除斷，是名丈夫第五之德。復次，丈夫學心具足，作如是念：『設使餘人學以不學，我悉當學。』是名丈夫第六之德。復次，丈夫行八正道，不行非道，是名丈夫第七之德。復次，丈夫乃至盡壽，精勤方便，不厭不倦，是名丈夫第八之德。如是丈夫八德成就，隨其行地，能速昇進。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，世尊告諸比丘：「世間良馬有八種德成就者，隨人所欲，取道多少。何等為八？**
　　那時，世尊告訴在場的眾位比丘說：「世間上優秀的良馬，若能成就八種優良特質，就能隨主人意願，無論路途遠近都能勝任。這八種特質是什麼呢？

**◈ 生於良馬鄉，是名良馬第一之德。**
　　第一，出生在良馬的產地，這是良馬的第一種優點。

**◈ 復次，體性溫良，不驚恐人，是名良馬第二之德。**
　　另外，天性溫順善良，不會讓他人感到驚恐，這是良馬的第二種優點。

**◈ 復次，良馬不擇飲食，是名良馬第三之德。**
　　另外，良馬對於飲食不挑剔，這是良馬的第三種優點。

**◈ 復次，良馬厭惡不淨，擇地而臥，是名良馬第四之德。**
　　另外，良馬厭惡污穢之地，會選擇乾淨的地方休息，這是良馬的第四種優點。

**◈ 復次，良馬諸情態速為調馬者現，馬師調習，速捨其態，是名良馬第五之德。**
　　另外，良馬會迅速將自己的習性展現給調馬師看，且在馬師調教後能快速改掉壞習慣，這是良馬的第五種優點。

**◈ 復次，良馬安於駕乘，不顧餘馬，隨其輕重，能盡其力，是名良馬第六之德。**
　　另外，良馬在被騎乘時感到安穩，不被其他馬匹分心，無論負荷輕重都能竭盡全力，這是良馬的第六種優點。

**◈ 復次，良馬常隨正路，不隨非道，是名良馬第七之德。**
　　另外，良馬總是行走在正確的道路上，不走歧路，這是良馬的第七種優點。

**◈ 復次，良馬若病、若老，勉力駕乘，不厭不倦，是名良馬第八之德。**
　　另外，良馬即使在生病或年老時，依然勉力承載主人，不厭倦也不懈怠，這是良馬的第八種優點。

**◈ 「如是，丈夫於正法、律八德成就，當知是賢士夫。何等為八？**
　　世尊接著說：「同樣地，一個修行者若在正法（正確的真理教法）與律儀上成就八種優點，應當知道他是一位真正的賢士。這八種優點是什麼呢？

**◈ 謂賢士夫住於正戒波羅提木叉律儀，威儀行處具足，見微細罪，能生怖畏，受持學戒，是名丈夫於正法、律第一之德。**
　　第一，這位賢士遵循波羅提木叉（比丘應遵守的戒律清單）的律儀，在言行舉止上端正具足，對於微小的過錯也能產生警覺與畏懼，並嚴格受持學戒，這是修行者在正法與律儀上的第一種優點。

**◈ 復次，丈夫性自賢善，善調善住，不惱、不怖諸梵行者，是名丈夫第二之德。**
　　另外，修行者天性賢良，能良好地調伏心念並安定處世，不會讓其他追求清淨生活的梵行者感到惱怒或恐懼，這是第二種優點。

**◈ 復次，丈夫次行乞食，隨其所得，若麁、若細，其心平等，不嫌、不著，是名丈夫第三之德。**
　　另外，修行者在乞食時，對於所得的食物無論是粗劣或精細，內心都能保持平等，不嫌棄也不執著，這是第三種優點。

**◈ 復次，丈夫心生厭離於身惡業，口、意惡業，惡不善法及諸煩惱，重受諸有熾然苦報，於未來世生、老、病、死、憂、悲、惱、苦，增其厭離，是名丈夫第四之德。**
　　另外，修行者內心厭離身體、言語與意識所造的惡業，厭惡不善的法以及諸多煩惱（內心的染污與執著），深知這些會導致熾熱的苦報，對於未來世中必然經歷的生、老、病、死以及憂、悲、惱、苦而增加厭離心，這是第四種優點。

**◈ 復次，丈夫若有沙門過，諂曲不實，速告大師及善知識，大師說法則時除斷，是名丈夫第五之德。**
　　另外，修行者若發現沙門有諂媚、曲折或不真實的過錯，會迅速告知導師或善知識，讓導師透過說法及時予以除斷，這是第五種優點。

**◈ 復次，丈夫學心具足，作如是念：『設使餘人學以不學，我悉當學。』是名丈夫第六之德。**
　　另外，修行者具足強烈的學習心，心想著：「即使其他人不學習，我也要全部學習」，這是第六種優點。

**◈ 復次，丈夫行八正道，不行非道，是名丈夫第七之德。**
　　另外，修行者實踐八正道（通往解脫的八種正確修行路徑），而不走錯誤的道路，這是第七種優點。

**◈ 復次，丈夫乃至盡壽，精勤方便，不厭不倦，是名丈夫第八之德。**
　　另外，修行者直到生命結束，依然精進勤勉地尋找修行方便，不厭倦也不懈怠，這是第八種優點。

**◈ 如是丈夫八德成就，隨其行地，能速昇進。」**
　　世尊總結道：「如此成就八種優點的修行者，無論處於什麼修行階段，都能迅速昇進。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **波羅提木叉 (Pātimokkha)**：指比丘與比丘尼必須共同誦持並遵守的戒律清單，是僧團維持純淨與和諧的基礎。
2. **梵行者 (Brahmacārī)**：指實踐清淨生活、追求解脫之聖道的人。
3. **煩惱 (Kilesa)**：指染污心靈的心理狀態，如貪、嗔、痴，使人無法看清真理而產生執著。
4. **八正道 (Ariya Aṭṭhaṅgika Magga)**：包含正見、正思惟、正語、正業、正命、正精進、正念、正定，是消除苦因、達成涅槃的唯一路徑。
5. **熾然苦報 (Ādittapariyāya)**：比喻輪迴中的生命如同被火燒灼一般，充滿著貪、嗔、痴的煎熬。

### [現代實作指南]
本經以「良馬」比喻「良才」，將動物的訓練特質轉化為心靈的修煉指標。現代人可將此轉化為以下實作步驟：

1. **建立底線（對應第一、七德）**：
   ➔ 確立自己的價值觀與行為準則（如誠實、不傷害他人）。在面對誘惑時，提醒自己「不隨非道」，將行為規範化為一種習慣，而非強迫的壓力。

2. **培養情緒穩定度（對應第二、三德）**：
   ➔ 練習「平等心」。在生活中面對不理想的環境（如糟糕的食物、不舒服的通勤）時，嘗試觀察內心的「嫌棄」之情，並對其保持中立，不讓外在條件決定內心的情緒狀態。

3. **正視缺陷與開放心態（對應第五、六德）**：
   ➔ 建立「反饋機制」。主動尋找能給予誠實批評的導師或朋友，將被指正視為「除斷煩惱」的機會，而非對自尊的攻擊。保持終身學習的謙卑，不與他人比較學習進度，僅關注自身的成長。

4. **深化對生命本質的覺知（對應第四德）**：
   ➔ 定期進行「無常觀」。意識到身體的衰老與生命的有限，這並非為了製造悲觀，而是為了將注意力從瑣碎的物質追求，轉移到更有意義的精神成長與利他行為上。

5. **維持恆心（對應第八德）**：
   ➔ 將修行視為「馬拉松」而非「短跑」。不追求瞬間的頓悟，而是在日常瑣事中保持微小的精進，直到成為生命的一部分。

### [段落大意]
佛陀以良馬的八種優良特質（如天性溫良、不擇飲食、循正路、不倦怠等）為喻，對比出一位優秀修行者應具備的八種德行：包括嚴守戒律、性格溫和、對物質平等、厭離惡業與苦報、勇於面對過錯、具足學習心、實踐八正道以及終身精進。佛陀強調，具備這些特質的人能迅速在修行之路上獲得進步。
