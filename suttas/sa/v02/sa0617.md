# 雜阿含經 第 617 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「過去世時有一鳥，名曰羅婆，為鷹所捉，飛騰虛空，於空鳴喚言：『我不自覺，忽遭此難，我坐捨離父母境界而遊他處，故遭此難。如何今日為他所困，不得自在。』鷹語羅婆：『汝當何處自有境界而得自在？』羅婆答言：『我於田耕壠中自有境界，足免諸難，是為我家父母境界。』

「鷹於羅婆起憍慢言：『放汝令去，還耕壠中，能得脫以不？』於是羅婆得脫鷹爪，還到耕壠大塊之下，安住止處，然後於塊上欲與鷹鬪。

「鷹則大怒，彼是小鳥，敢與我鬪，瞋恚極盛，駿飛直搏，於是羅婆入於塊下，鷹鳥飛勢，臆衝堅塊，碎身即死。

「時，羅婆鳥深伏塊下，仰說偈言：

「『鷹鳥用力來，　　羅婆依自界， 
 　　乘瞋猛盛力，　　致禍碎其身。 
 　　我具足通達，　　依於自境界， 
 　　伏怨心隨喜，　　自觀欣其力。 
 　　設汝有兇愚，　　百千龍象力， 
 　　不如我智慧，　　十六分之一。 
 　　觀我智殊勝，　　摧滅於蒼鷹。』

「如是，比丘！如彼鷹鳥，愚癡自捨所親父母境界，遊於他處，致斯災患。汝等比丘亦應如是，於自境界所行之處，應善守持，離他境界，應當學。

「比丘！他處他境界者，謂五欲境界，眼見可意、受、念妙色，欲心染著；耳識聲、鼻識香、舌識味、身識觸，可意、受、念妙觸，欲心染著。是名比丘他處他境界。比丘！自處父母境界者，謂四念處。云何為四？謂身身觀念處，受、心、法法觀念處。是故，比丘！於自行處父母境界而自遊行，遠離他處他境界，應當學。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「過去世時有一鳥，名曰羅婆，為鷹所捉，飛騰虛空，於空鳴喚言：『我不自覺，忽遭此難，我坐捨離父母境界而遊他處，故遭此難。如何今日為他所困，不得自在。』**
　　那時，世尊告訴在場的眾位比丘說：「在過去的世間，有一隻名叫羅婆的小鳥，被一隻老鷹捉住，在空中被飛騰地帶著，這時小鳥在空中大聲呼喊：『我之前沒有覺察，突然遭遇這樣的困難，是因為我捨棄了原本安全、能保護我的父母境界（原生棲息地）而遊蕩到其他地方，所以才遭遇這次災難。為什麼我今天會被他人困住，而無法獲得自在？』」

**◈ 鷹語羅婆：『汝當何處自有境界而得自在？』**
　　老鷹對羅婆說：「你說你原本在什麼樣的地方才有自己的境界，能夠獲得自在？」

**◈ 羅婆答言：『我於田耕壠中自有境界，足免諸難，是為我家父母境界。』**
　　羅婆回答說：「我在農田耕作後留下的土塊縫隙中，有我自己的安全境界，足以避開所有危險，那才是我的家，是我父母留給我的安全境界。」

**◈ 「鷹於羅婆起憍慢言：『放汝令去，還耕壠中，能得脫以不？』**
　　老鷹對羅婆產生了傲慢心，說道：「我就放你走，讓你回到那耕地的土塊中，看看你是否還能逃脫？」

**◈ 於是羅婆得脫鷹爪，還到耕壠大塊之下，安住止處，然後於塊上欲與鷹鬪。**
　　於是羅婆從老鷹的爪中脫身，飛回耕地的一塊巨大的土塊之下，安穩地待在那個處所，然後在土塊上方挑釁，想要與老鷹戰鬥。

**◈ 「鷹則大怒，彼是小鳥，敢與我鬪，瞋恚極盛，駿飛直搏，於是羅婆入於塊下，鷹鳥飛勢，臆衝堅塊，碎身即死。**
　　老鷹聽了勃然大怒，心想：「它只是一隻小鳥，竟然敢跟我戰鬥！」瞋恚（憤怒）的情緒極其強烈，於是猛力俯衝直擊。就在這時，羅婆迅速鑽入土塊下方，老鷹由於衝擊速度太快，胸口直接撞在堅硬的土塊上，身體破碎而死。

**◈ 「時，羅婆鳥深伏塊下，仰說偈言：**
　　這時，羅婆鳥深伏在土塊之下，仰頭說出偈頌：

**◈ 『鷹鳥用力來，羅婆依自界，乘瞋猛盛力，致禍碎其身。**
　　老鷹憑著強大的力量衝來，而羅婆依託於自己的安全境界；老鷹乘著極其猛烈的瞋心，反而導致災禍，撞碎了自己的身體。

**◈ 我具足通達，依於自境界，伏怨心隨喜，自觀欣其力。**
　　我具足對真理的通達，依託於自己的安全境界，讓對方在憤怒中自毀，我心中隨喜，觀察到智慧的力量是如此令人欣喜。

**◈ 設汝有兇愚，百千龍象力，不如我智慧，十六分之一。**
　　就算你擁有凶猛且愚癡的百千頭龍象般的力量，也比不上我十六分之一的智慧。

**◈ 觀我智殊勝，摧滅於蒼鷹。』**
　　請看我的智慧是如此殊勝，竟然能摧毀這隻強大的蒼鷹。」

**◈ 「如是，比丘！如彼鷹鳥，愚癡自捨所親父母境界，遊於他處，致斯災患。**
　　比丘們！就像那隻老鷹一樣，愚癡地捨棄了自己親近且安全的父母境界，遊蕩在其他地方，導致了這樣的災患。

**◈ 汝等比丘亦應如是，於自境界所行之處，應善守持，離他境界，應當學。**
　　你們這些比丘也應該如此，對於自己修行應在的境界（正確的修行方向），應該善於守持，遠離他人的境界（錯誤的追求），這才是應該學習的。

**◈ 「比丘！他處他境界者，謂五欲境界，眼見可意、受、念妙色，欲心染著；耳識聲、鼻識香，舌識味、身識觸，可意、受、念妙觸，欲心染著。是名比丘他處他境界。**
　　比丘們！所謂的「他處他境界」，是指五欲（眼、耳、鼻、舌、身五種感官的欲望）境界：眼睛看到令人滿意的色相，產生受（感受）並在心中念著那美妙的色彩，導致欲心染著；耳朵聽到、鼻子聞到、舌頭嚐到、身體觸碰到令人滿意的對象，產生受（感受）並在心中念著那美妙的觸感，導致欲心染著。這就叫做比丘的「他處他境界」。

**◈ 比丘！自處父母境界者，謂四念處。云何為四？謂身身觀念處，受、心、法法觀念處。**
　　比丘們！所謂的「自處父母境界」，是指四念處（對身、受、心、法四個面向的正念觀察）。什麼是四個面向？指的是對色（物質身體）的觀念處、對受（感受）的觀念處、對心（心靈狀態）的觀念處，以及對法（心理現象）的觀念處。

**◈ 是故，比丘！於自行處父母境界而自遊行，遠離他處他境界，應當學。」**
　　因此，比丘們！你們應該在自己的父母境界（四念處）中修行遊行，遠離他處的他境界（五欲），這才是應該學習的。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **五欲 (Pañca-kāma-gunā)**：指透過五根（眼、耳、鼻、舌、身）所接觸到的感官享受，包括色、聲、香、味、觸。在佛教中，對這些對象的貪著被視為導致輪迴與痛苦的根源。
2. **四念處 (Satipaṭṭhāna)**：正念修行的四個基礎。
    *   **身念處**：觀察身體的呼吸、姿勢、動作及物質組成。
    *   **受念處**：覺察當下產生的苦、樂、不苦不樂的感受。
    *   **心念處**：觀察心的狀態（如：有貪或無貪、有瞋或無瞋）。
    *   **法念處**：觀察心法運作的規律與現象（如：五蘊的生滅）。
3. **憍慢 (Māna)**：一種心理染污，表現為自大、傲慢或與他人比較而產生的優越感，會遮蔽智慧並導致錯誤的判斷。
4. **隨喜 (Muditā)**：對他人或正向結果感到高興，在此處指羅婆鳥在智慧獲勝後，對真理運作的欣喜。

### [現代實作指南]
本經以「羅婆鳥與老鷹」的寓言，揭示了「力量 $\neq$ 智慧」以及「定位 $\neq$ 迷失」的核心道理。在現代生活中，我們可以將其轉化為以下實踐：

1. **識別你的「他處境界」（誘因分析）**：
    *   **實作**：當你感到焦慮、空虛或被壓力追趕時，問自己：「我現在是否正處於『他處境界』？」例如：過度追求社交媒體的讚美（色/聲）、沉溺於物質消費的快感（味/觸）。
    *   **轉念**：意識到這些快感就像老鷹的爪子，雖然看似強大或吸引人，但實際上會讓你失去「自在」。

2. **建立你的「父母境界」（心理安全區）**：
    *   **實作**：將「四念處」簡化為日常的「覺察練習」。
        *   **身**：感覺腳掌接觸地面的觸感 → 回到當下。
        *   **受**：意識到心中升起的一絲不耐煩 → 僅僅觀察它，不被它牽著走。
        *   **心**：發現自己正在對某人產生評判 → 標記「評判心」。
        *   **法**：意識到「這份不快感也是無常的，會隨時消失」 → 體會空性。
    *   **目標**：將正念覺察視為你的「土塊」，當外界的壓力（老鷹）衝擊時，你不是用憤怒對抗，而是退回到覺察之中，讓壓力在自己的慣性中瓦解。

3. **以智慧取代對抗（非暴力化解）**：
    *   **技巧**：面對衝突時，不要試圖用「更強的力量」去壓制對方（如老鷹的猛衝），而要尋找對方的「盲點」或利用「因緣」。
    *   **實踐**：當他人對你憤怒時，保持內心的「自界」（平靜與覺察），不被對方的情緒捲入。對方的憤怒（瞋心）往往是其最大的弱點，只要你不接招，憤怒的力量最終會反噬其自身。

### [段落大意]
本經透過羅婆鳥利用地形與智慧擊敗強大老鷹的寓言，說明智慧遠勝於蠻力。佛陀以此比喻，告誡比丘應遠離由五欲（感官欲望）構成的「他處境界」，而應守持以四念處（對身、受、心、法的覺察）為核心的「自處父母境界」，透過正念修行獲得真正的自在與解脫。
