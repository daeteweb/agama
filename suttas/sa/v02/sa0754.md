# 雜阿含經 第 754 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，尊者舍利弗詣佛所，稽首佛足，退坐一面，白佛言：「世尊！所謂賢聖等三昧根本眾具。云何為賢聖等三昧根本眾具？」

佛告舍利弗：「謂七正道分為賢聖等三昧，為根本，為眾具。何等為七？謂正見、正志、正語、正業、正命、正方便、正念。舍利弗！於此七道分為基業已，得一其心，是名賢聖等三昧根本眾具。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，尊者舍利弗詣佛所，稽首佛足，退坐一面，白佛言：**
　　那時，尊者舍利弗來到佛陀所在之處，恭敬地頂禮佛陀的足部，退到一邊坐下，對佛陀恭敬地說道：

**◈ 「世尊！所謂賢聖等三昧根本眾具。云何為賢聖等三昧根本眾具？」**
　　「世尊！關於所謂的賢聖（覺悟的聖者）之三昧（禪定專注）的根本基礎與必要條件。請問什麼才是賢聖之三昧的根本基礎與必要條件？」

**◈ 佛告舍利弗：「謂七正道分為賢聖等三昧，為根本，為眾具。」**
　　佛陀告訴舍利弗：「所謂的七種正道分（正確的修行路徑）就是賢聖之三昧的根本基礎與必要條件。」

**◈ 「何等為七？謂正見、正志、正語、正業、正命、正方便、正念。」**
　　「這七種是什麼呢？分別是正見（正確的見解）、正志（正確的思惟與志向）、正語（正確的言語）、正業（正確的行為）、正命（正確的生活方式）、正方便（正確的精進努力）以及正念（正確的覺知）。」

**◈ 「舍利弗！於此七道分為基業已，得一其心，是名賢聖等三昧根本眾具。」**
　　「舍利弗！當一個人以這七種正道分作為修行的基礎後，使心能夠專一統一，這就叫做賢聖之三昧的根本基礎與必要條件。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **三昧 (Samādhi)**：指心意識高度集中、統一且不散亂的狀態，是進入深層禪定與獲得智慧的必要前提。
2. **賢聖 (Āriya)**：指已證得聖果、擺脫部分或全部煩惱的覺悟者。
3. **眾具 (Parikkhāra)**：原意為隨身用品或器具，在此指達成某種目標（如三昧）所必須具備的條件、要素或準備工作。
4. **正道分 (Magga-ṅga)**：指通往解脫之道的組成要素。本經中列出七項，是因為「正定」本身即是三昧，因此前七項被視為達成三昧的「基礎」。
5. **正志 (Sammā-saṅkappa)**：即八正道中的「正思惟」，指離貪、離嗔、不害的正確思考方向。
6. **正方便 (Sammā-vāyāma)**：即八正道中的「正精進」，指防止未生惡法生起、斷除已生惡法，並使未生善法生起、增長已生善法。

### [現代實作指南]
本經揭示了一個核心邏輯：**「專注力（三昧）並非憑空而來，而是建立在正確的生活方式與認知之上。」** 若想在現代生活中提升專注力或心靈寧靜，可採取以下實作步驟：

1. **建立道德底線（正語、正業、正命）**：
   ➔ 減少說謊、爭吵或傷害他人的行為。當一個人的行為與言語與他人和諧時，內心的愧疚感與不安會降低，從而減少心靈的雜訊。
2. **校準認知方向（正見、正志）**：
   ➔ 練習觀察事物的無常與無我，減少對特定結果的執著。當不再強求「必須如何」時，心靈的壓力會減輕，更容易進入專注狀態。
3. **動態調整與覺察（正方便、正念）**：
   ➔ **正方便**：當發現心念轉向憤怒或焦慮時，及時地將其導向平靜。
   ➔ **正念**：在做任何事情時，將意識留在當下，不對過去後悔，不對未來憂慮。
4. **最終目標 ➔ 統一心念**：
   ➔ 當上述七項基礎穩固後，嘗試將心專注於單一對象（如呼吸），此時心會自然地「得一」（統一），而非強行壓制雜念。

### 段落大意 
本經由舍利弗尊者請益關於達成聖者禪定（三昧）的必要條件。佛陀回答指出，正定（三昧）並非獨立存在，而是必須以正見、正志、正語、正業、正命、正方便、正念這七種正道分作為基礎。當修行者在生活中實踐這七項正確的路徑後，心才能夠專一統一，進而成就三昧。
