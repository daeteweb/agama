# 雜阿含經 第 535 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。爾時，尊者阿那律住松林精舍。時，尊者大目犍連住跋祇聚落失收摩羅山恐怖稠林禽獸之處。

時，尊者阿那律獨一靜處，禪思思惟，作是念：「有一乘道，淨眾生，離憂、悲、惱、苦，得真如法，所謂四念處。何等為四？身身觀念處，受、心、法法觀念處。若於四念處遠離者，於賢聖法遠離；於賢聖法遠離者，於聖道遠離；聖道遠離者，於甘露法遠離；甘露法遠離者，則不能脫生、老、病、死、憂、悲、惱、苦。若於四念處信樂者，於聖法信樂；聖法信樂者，於聖道信樂；聖道信樂者，於甘露法信樂；甘露法信樂者，得脫生、老、病、死、憂、悲、惱、苦。」

爾時，尊者大目揵連知尊者阿那律心之所念，如力士屈伸臂頃，以神通力於跋祇聚落失收摩羅山恐怖稠林禽獸之處沒，至舍衛城松林精舍尊者阿那律前現，語阿那律言：「汝獨一靜處，禪思思惟，作是念：『有一乘道，令眾生清淨，離生、老、病、死、憂、悲、惱、苦，得真如法，所謂四念處。何等為四？身身觀念處，受、心、法觀念處。若於四念處不樂者，於賢聖法不樂；聖法不樂者，於聖道不樂；不樂聖道者，於甘露法亦不樂；不樂甘露法者，則不能脫生、老、病、死、憂、悲、惱、苦。若於四念處信樂者，樂賢聖法；樂賢聖法者，樂於聖道；樂聖道者，得甘露法；得甘露法者，得脫生、老、病、死、憂、悲、惱、苦。』耶？」

尊者阿那律語尊者大目揵連言：「如是，如是。尊者！」

大目犍連語尊者阿那律言：「云何名為樂四念處？」

尊者大目犍連：「若比丘身身觀念處，心緣身正念住調伏、止息、寂靜，一心增進。如是，受、心、法念處，正念住調伏、止息、寂靜，一心增進，尊者大目犍連，是名比丘樂四念處。」

時，尊者大目犍連即如其像三昧正受，從舍衛國松林精舍門，還至跋祇聚落失收摩羅山恐怖稠林禽獸之處。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，尊者阿那律住松林精舍。**
　　那時，尊者阿那律住在松林精舍。

**◈ 時，尊者大目犍連住跋祇聚落失收摩羅山恐怖稠林禽獸之處。**
　　同時，尊者大目犍連住在跋祇聚落失收摩羅山那處充滿恐怖稠密森林與禽獸的地方。

**◈ 時，尊者阿那律獨一靜處，禪思思惟，作是念：**
　　那時，尊者阿那律獨自處於一個安靜的地方，在禪定中思惟，心中這樣想：

**◈ 「有一乘道，淨眾生，離憂、悲、惱、苦，得真如法，所謂四念處。**
　　「有一條唯一的解脫之道，能使眾生清淨，遠離憂愁、悲傷、煩惱與痛苦，並獲得真實的法，這就是所謂的『四念處』（四種正念的確立）。

**◈ 何等為四？身身觀念處，受、心、法法觀念處。**
　　這四種是什麼呢？分別是觀照色（物質身體）的念處、觀照受（感受）的念處、觀照心（意識狀態）的念處，以及觀照法（心理現象與真理）的念處。

**◈ 若於四念處遠離者，於賢聖法遠離；於賢聖法遠離者，於聖道遠離；聖道遠離者，於甘露法遠離；甘露法遠離者，則不能脫生、老、病、死、憂、悲、惱、苦。**
　　如果一個人遠離了這四種正念的確立，就等於遠離了賢聖的法；遠離了賢聖的法，就等於遠離了通往解脫的聖道；遠離了聖道，就等於遠離了甘露法（指不死、不退的涅槃境界）；而遠離了甘露法，就無法從出生、衰老、疾病、死亡以及憂愁、悲傷、煩惱與痛苦中解脫。

**◈ 若於四念處信樂者，於聖法信樂；聖法信樂者，於聖道信樂；聖道信樂者，於甘露法信樂；甘露法信樂者，得脫生、老、病、死、憂、悲、惱、苦。」**
　　如果一個人對四念處產生信心並感到喜悅，就等於對聖法產生信心並感到喜悅；對聖法信樂的人，也會對聖道產生信心並感到喜悅；對聖道信樂的人，也會對甘露法產生信心並感到喜悅；而對甘露法信樂的人，最終能從出生、衰老、疾病、死亡以及憂愁、悲傷、煩惱與痛苦中獲得解脫。」

**◈ 爾時，尊者大目揵連知尊者阿那律心之所念，**
　　那時，尊者大目犍連以神通知道了尊者阿那律心中所思惟的內容。

**◈ 如力士屈伸臂頃，以神通力於跋祇聚落失收摩羅山恐怖稠林禽獸之處沒，**
　　他如同力士屈伸手臂般自在，運用神通力從跋祇聚落失收摩羅山那處恐怖的稠林禽獸之地消失。

**◈ 至舍衛城松林精舍尊者阿那律前現，語阿那律言：**
　　隨即出現在舍衛城松林精舍的尊者阿那律面前，對阿那律說：

**◈ 「汝獨一靜處，禪思思惟，作是念：『有一乘道，令眾生清淨，離生、老、病、死、憂、悲、惱、苦，得真如法，所謂四念處。**
　　「你是否獨自處於安靜之處，在禪思中這樣想：『有一條能令眾生清淨、遠離生老病死與憂悲惱苦、獲得真如法的唯一道路，即是四念處。

**◈ 何等為四？身身觀念處，受、心、法觀念處。**
　　這四種是：觀照色（物質身體）、受（感受）、心（意識狀態）、法（心理現象）的念處。

**◈ 若於四念處不樂者，於賢聖法不樂；聖法不樂者，於聖道不樂；不樂聖道者，於甘露法亦不樂；不樂甘露法者，則不能脫生、老、病、死、憂、悲、惱、苦。**
　　若不喜愛四念處，就不喜愛聖法；不喜愛聖法，就不喜愛聖道；不喜愛聖道，就不喜愛甘露法；不喜愛甘露法，就不能脫離生老病死與憂悲惱苦。

**◈ 若於四念處信樂者，樂賢聖法；樂賢聖法者，樂於聖道；樂聖道者，得甘露法；得甘露法者，得脫生、老、病、死、憂、悲、惱、苦。』耶？」**
　　若信樂四念處，就喜愛聖法；喜愛聖法，就喜愛聖道；喜愛聖道，就獲得甘露法；獲得甘露法，就脫離生老病死與憂悲惱苦。』是這樣嗎？」

**◈ 尊者阿那律語尊者大目揵連言：「如是，如是。尊者！」**
　　尊者阿那律對尊者大目犍連說：「確實如此，就是這樣，尊者！」

**◈ 大目犍連語尊者阿那律言：「云何名為樂四念處？」**
　　大目犍連問尊者阿那律：「什麼叫做『喜愛（樂）四念處』呢？」

**◈ 尊者大目犍連：「若比丘身身觀念處，心緣身正念住調伏、止息、寂靜，一心增進。**
　　尊者阿那律（原文誤植為大目犍連）回答：「如果比丘在觀照色（物質身體）的念處時，心以身體為對象，保持正念的安住，使心靈得到調伏、止息與寂靜，並讓專注力不斷增進。

**◈ 如是，受、心、法念處，正念住調伏、止息、寂靜，一心增進，尊者大目犍連，是名比丘樂四念處。」**
　　同樣地，在觀照受（感受）、心（意識狀態）、法（心理現象）的念處時，也能保持正念的安住，使心靈得到調伏、止息與寂靜，並讓專注力不斷增進。尊者大目犍連，這就叫做比丘喜愛四念處。」

**◈ 時，尊者大目犍連即如其像三昧正受，從舍衛國松林精舍門，還至跋祇聚落失收摩羅山恐怖稠林禽獸之處。**
　　那時，尊者大目犍連立即進入如其原像的三昧（定境），從舍衛國松林精舍的大門離開，回到了跋祇聚落失收摩羅山那處恐怖的稠林禽獸之地。

### [詞語註解]
1. **四念處 (Satipaṭṭhāna)**：佛教修行中確立正念的四種對象，包括觀照身體（色）、感受（受）、心靈狀態（心）以及心理現象/真理（法）。這是達成覺悟的唯一路徑。
2. **甘露法 (Amata-dhamma)**：字面意為「不死的法」，指涅槃（Nibbāna）的境界。因為涅槃超越了生死的輪迴，不再受苦，故稱為甘露。
3. **聖道 (Ariya-magga)**：指通往解脫的八正道，是聖者所行的路徑，能直接導向苦的滅盡。
4. **三昧 (Samādhi)**：指心意識的高度集中與統一狀態，即定力。文中指大目犍連利用定力進行空間上的移動（神通）。

### [現代實作指南]
本經文強調了「正念」是解脫痛苦的唯一路徑。在現代生活中，我們可以將「四念處」轉化為以下心理實踐：

1. **色念處（身體掃描）**：
   - **實作**：在感到壓力時，將注意力從思考轉移到身體。感受呼吸的起伏，或意識到腳掌接觸地面的感覺。
   - **轉念**：意識到身體僅僅是物質的組合，不將身體的疼痛或不適等同於「我」的痛苦。

2. **受念處（感受標記）**：
   - **實作**：當情緒升起時（如憤怒、悲傷），在心中輕輕標記：「現在有『不快』的感受升起了」。
   - **轉念**：將感受視為暫時的心理現象，而非事實。從「我很生氣」轉變為「我觀察到心中有一股憤怒的感受」。

3. **心念處（意識觀察）**：
   - **實作**：觀察心靈的狀態。例如，意識到現在心是「散亂的」或「專注的」、「貪婪的」或「平靜的」。
   - **轉念**：不對心的狀態做評判，僅僅是客觀地觀察心的起伏，如同觀察天空中的雲朵。

4. **法念處（模式分析）**：
   - **實作**：分析痛苦產生的因緣。例如，思考「是因為我對某件事有執著（取），所以才產生了這份痛苦」。
   - **轉念**：體悟到所有心理現象都是因緣而生、因緣而滅，沒有永恆不變的自我。

### [段落大意]
本經記述尊者阿那律與大目犍連關於「四念處」的對話。阿那律體悟到觀照色、受、心、法是遠離生老病死、獲得涅槃（甘露法）的唯一路徑。大目犍連以神通確認此念，並請教何謂「樂四念處」，阿那律解釋為心在觀照四念處時能達到調伏、寂靜且專注增進的狀態。
