# 雜阿含經 第 638 經

### [原文]
如是我聞：

一時，佛在王舍城迦蘭陀竹園。爾時，尊者舍利弗住摩竭提那羅聚落，疾病涅槃，純陀沙彌瞻視供養。

爾時，尊者舍利弗因病涅槃。

時，純陀沙彌供養尊者舍利弗已，取餘舍利，擔持衣鉢，到王舍城，舉衣鉢，洗足已，詣尊者阿難所。禮尊者阿難足已，却住一面，白尊者阿難：「尊者當知，我和上尊者舍利弗已涅槃，我持舍利及衣鉢來。」

於是尊者阿難聞純陀沙彌語已，往詣佛所，白佛言：「世尊！我今舉體離解，四方易韻，持辯閉塞，純陀沙彌來語我言：『和上舍利弗已涅槃，持餘舍利及衣鉢來。』」

佛言：「云何？阿難！彼舍利弗持所受戒身涅槃耶？定身、慧身、解脫身、解脫知見身涅槃耶？」

阿難白佛言：「不也，世尊！」

佛告阿難：「若法我自知，成等正覺所說，謂四念處、四正斷、四如意足、五根、五力、七覺支、八道支涅槃耶？」

阿難白佛：「不也，世尊！雖不持所受戒身乃至道品法而涅槃，然尊者舍利弗持戒多聞，少欲知足，常行遠離，精勤方便，攝念安住，一心正受捷疾智慧、深利智慧、超出智慧、分別智慧、大智慧、廣智慧、甚深智慧、無等智慧，智寶成就，能示、能教、能照、能喜，善能讚歎，為眾說法。是故，世尊！我為法故，為受法者故，愁憂苦惱。」

佛告阿難：「汝莫愁憂苦惱。所以者何？若坐、若起、若作，有為敗壞之法，何得不壞，欲令不壞者，無有是處，我先已說，一切所愛念種種諸物、適意之事，一切皆是乖離之法，不可常保。譬如大樹，根、莖、枝、葉、華、果茂盛，大枝先折，如大寶山，大巖先崩。如是，如來大眾眷屬，其大聲聞先般涅槃，若彼方有舍利弗住者，於彼方我則無事，然其彼方，我則不空，以有舍利弗故，我先已說故。汝今，阿難！如我先說，所可愛念種種適意之事，皆是別離之法，是故汝今莫大愁毒。阿難！當知，如來不久亦當過去。是故，阿難！當作自洲而自依，當作法洲而法依，當作不異洲不異依。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛在王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園中。

**◈ 爾時，尊者舍利弗住摩竭提那羅聚落，疾病涅槃，純陀沙彌瞻視供養。**
　　那時，尊者舍利弗住在摩竭提國的那羅聚落，因病而進入涅槃（超越生死、絕對寧靜的解脫境界），由純陀沙彌在旁照顧並供養。

**◈ 爾時，尊者舍利弗因病涅槃。**
　　那時，尊者舍利弗因為疾病而進入了涅槃（超越生死、絕對寧靜的解脫境界）。

**◈ 時，純陀沙彌供養尊者舍利弗已，取餘舍利，擔持衣鉢，到王舍城，舉衣鉢，洗足已，詣尊者阿難所。**
　　這時，純陀沙彌在供養完尊者舍利弗後，取走遺留的舍利（骨灰遺物），擔著衣缽前往王舍城。到達後，他舉起衣缽，洗淨雙足，前往尊者阿難處。

**◈ 禮尊者阿難足已，却住一面，白尊者阿難：「尊者當知，我和上尊者舍利弗已涅槃，我持舍利及衣鉢來。」**
　　他向尊者阿難的足部行禮後，退至一側站立，對尊者阿難恭敬地說道：「尊者應當了解，和上（尊者）舍利弗已經進入涅槃（超越生死、絕對寧靜的解脫境界）了，我現在持著舍利以及衣缽而來。」

**◈ 於是尊者阿難聞純陀沙彌語已，往詣佛所，白佛言：「世尊！我今舉體離解，四方易韻，持辯閉塞，純陀沙彌來語我言：『和上舍利弗已涅槃，持餘舍利及衣鉢來。』」**
　　於是尊者阿難聽完純陀沙彌的話後，前往佛陀處，對佛陀恭敬地說道：「世尊！我現在全身感到沉重且不自在，呼吸急促且不順暢，言語表達也變得阻塞。純陀沙彌來告訴我：『和上（尊者）舍利弗已經進入涅槃（超越生死、絕對寧靜的解脫境界）了，他持著遺留的舍利以及衣鉢而來。』」

**◈ 佛言：「云何？阿難！彼舍利弗持所受戒身涅槃耶？定身、慧身、解脫身、解脫知見身涅槃耶？」**
　　佛陀問道：「怎麼回事，阿難？舍利弗是因為持守所受之戒律的身心狀態而進入涅槃（超越生死、絕對寧靜的解脫境界）的嗎？還是因為定力之身、智慧之身、解脫之身、或是對解脫的知見之身而進入涅槃的？」

**◈ 阿難白佛言：「不也，世尊！」**
　　阿難對佛陀恭敬地說道：「不是的，世尊！」

**◈ 佛告阿難：「若法我自知，成等正覺所說，謂四念處、四正斷、四如意足、五根、五力、七覺支、八道支涅槃耶？」**
　　佛陀告訴阿難：「那麼，他是因為我自覺悟並由等正覺（最高、最圓滿的智慧與覺悟）所說的法，也就是四念處（四種正念觀察）、四正斷（四種止息煩惱的方法）、四如意足（四種成就定力的基礎）、五根（五種修行能力）、五力（五種修行力量）、七覺支（七種覺悟的因素）、八道支（八正道）而進入涅槃的嗎？」

**◈ 阿難白佛：「不也，世尊！雖不持所受戒身乃至道品法而涅槃，然尊者舍利弗持戒多聞，少欲知足，常行遠離，精勤方便，攝念安住，一心正受捷疾智慧、深利智慧、超出智慧、分別智慧、大智慧、廣智慧、甚深智慧、無等智慧，智寶成就，能示、能教、能照、能喜，善能讚歎，為眾說法。是故，世尊！我為法故，為受法者故，愁憂苦惱。」**
　　阿難對佛陀說：「不是的，世尊！雖然他不是單純持守戒律之身或道品法而進入涅槃，但尊者舍利弗持戒嚴謹且博學多聞，少欲知足，經常遠離喧囂，精勤於修行方便，能攝心正念而安住。他擁有一心正受的捷疾智慧、深利智慧、超出世俗的智慧、能分辨真偽的智慧、廣大的智慧、深邃的智慧、以及無與倫比的智慧。他成就了智慧之寶，能示導、能教化、能照破迷妄、能令眾生歡喜，擅長讚歎，為眾生說法。因此，世尊！我為了這份法，以及為了能受法的人而感到愁憂苦惱。」

**◈ 佛告阿難：「汝莫愁憂苦惱。所以者何？若坐、若起、若作，有為敗壞之法，何得不壞，欲令不壞者，無有是處，我先已說，一切所愛念種種諸物、適意之事，一切皆是乖離之法，不可常保。**
　　佛陀告訴阿難：「你不要愁憂苦惱。為什麼呢？無論是坐著、站著或在行動，所有由因緣而生的有為法（有為敗壞之法）必然會敗壞，沒有任何東西能不壞，想要讓它不壞是不可能的。我先前已經說過，一切被愛念的種種事物、令心適意之事，全部都是必然會乖離（分離）的法，無法永遠保持。

**◈ 譬如大樹，根、莖、枝、葉、華、果茂盛，大枝先折，如大寶山，大巖先崩。**
　　就像大樹一樣，根、莖、枝、葉、花、果雖然茂盛，但巨大的枝幹會先折斷；如同巨大的寶山，巨大的岩石會先崩塌。

**◈ 如是，如來大眾眷屬，其大聲聞先般涅槃，若彼方有舍利弗住者，於彼方我則無事，然其彼方，我則不空，以有舍利弗故，我先已說故。**
　　同樣地，如來的大眾眷屬中，那些大聲聞（頂級弟子）會先進入涅槃（超越生死、絕對寧靜的解脫境界）。如果那個方向有舍利弗住在其中，對我來說那個方向就沒有缺失；但實際上那個方向並不空虛，因為有舍利弗（的法）存在。我先前已經說過了。

**◈ 汝今，阿難！如我先說，所可愛念種種適意之事，皆是別離之法，是故汝今莫大愁毒。**
　　阿難！正如我先前所說，所有可愛念的種種適意之事，皆是必然別離的法，因此你現在不要陷入巨大的愁苦之中。

**◈ 阿難！當知，如來不久亦當過去。**
　　阿難！應當了解，如來不久之後也將會離去。

**◈ 是故，阿難！當作自洲而自依，當作法洲而法依，當作不異洲不異依。**
　　因此，阿難！你應當將自己視為一座島嶼而依止自己，將法（真理）視為一座島嶼而依止法，應當作不異於自我的島嶼與不異於法的依止。」

### [詞語註解]
1. **涅槃 (Nibbāna)**：原意為「熄滅」，在學術上指熄滅貪、嗔、痴三毒，從而超越生死的輪迴，達到絕對寧靜與解脫的境界。
2. **等正覺 (Samma-sambuddha)**：指佛陀以自身力量完全覺悟宇宙真理，且能教導他人同樣覺悟的最高智慧。
3. **有為法 (Sankhata)**：指所有由因緣（條件）組合而成的現象。因為是組合而成的，所以必然會隨著條件的改變而生滅、敗壞。
4. **四念處 (Satipaṭṭhāna)**：正念修行的四個對象：身（物質身體）、受（感受）、心（心法）、法（心理現象與真理）。
5. **自洲/法洲 (Atta-dipa / Dhamma-dipa)**：比喻在苦海（輪迴）中，唯一的安全避風港是「自覺」與「真理」。

---

### [現代實作指南]
本經文處理的是面對「重大喪失」時的心理轉化。當我們面對親人、好友或依賴對象的離去而感到極度痛苦（如阿難之愁）時，可採取以下實作步驟：

1. **承認並觀察情緒 (正念觀察)**：
   - 當悲傷襲來時，不要壓抑，而是像阿難向佛陀描述身體反應（呼吸阻塞、言語閉塞）一樣，客觀地觀察悲傷在身體上的顯現（如胸口悶、眼眶熱）。
   - 告訴自己：「現在有『悲傷』這種感受在發生」，而非「我很悲傷」。

2. **理智分析「有為法」的本質 (轉念技巧)**：
   - 思考：這個人、這段關係、這個身體，是否是由因緣構成的？
   - 既然是因緣構成（有為法），那麼「變遷」與「分離」就是其內在的必然屬性。
   - 將「分離」視為自然法則的運作，而非針對個人的不幸。

3. **從「外在依止」轉向「內在依止」 (建立自洲)**：
   - 意識到我們對他人的依戀，本質上是將自己的安全感（島嶼）建立在他人身上。
   - 練習將重心移回自身：問自己「在沒有這個人的情況下，我能依止什麼真理來生活？」。
   - 實踐「法依」：將對人的依戀，轉化為對該人所代表的「美德」或「教導」的繼承。

---

### [段落大意]
本經記述尊者舍利弗涅槃後，阿難因深感失去智慧導師而極度悲慟。佛陀透過詢問舍利弗涅槃的層次，引導阿難體悟到：所有因緣和合的有為法必然敗壞，分離是世間常態。佛陀以此提醒阿難，包括如來在內的所有存在皆將離去，因此修行者不應依止外在的人事物，而應將自己與正法視為唯一的依止之洲。

### [詞語註解]
1. **觀念處 (Satipaṭṭhāna)**：指「四念處」的修行，即對色（身體）、受（感受）、心（意識）、法（現象）這四個面向建立正念的觀察，是解脫苦難的核心修行路徑。
2. **自洲 (Atta-dīpa)**：比喻將修行者自身的覺知與正念視為一座穩固的島嶼，在生死輪迴的苦海中不被情緒與欲望沖走，實現精神上的獨立。
3. **法洲 (Dhamma-dīpa)**：比喻將佛陀所揭示的真理（法）視為唯一的依止對象，透過對因緣與無常的體悟來獲得安全感與解脫。
4. **不異洲 (Non-different Island)**：指修行達到高度圓滿，不再將「修行者」、「修行方法（法）」與「解脫目標」視為分離的個體，三者合一的境界。

### [現代實作指南]
本經文提供了從「依賴他人」轉向「內在自足」的具體操作路徑，可轉化為以下心理實踐：

1. **建立「身體錨點」（色觀）**：
   當感到焦慮或情緒失控時，不要試圖用思考去解決，而是將注意力轉向物質身體（色）。觀察呼吸的起伏、足底接觸地面的感覺。將身體視為當下的「島嶼」，讓意識回歸物理現實，切斷對未來的憂慮。

2. **標記「感受標籤」（受觀）**：
   當強烈的情緒（如憤怒、悲傷）出現時，在心中輕輕標記：「現在有『不舒服的感受』產生了」。不要認同這個感受，而是像觀察路人一樣觀察它。這就是將「受」視為觀察對象，而非將自己等同於感受。

3. **覺察「心念流動」（心觀）**：
   練習觀察自己的心念。例如：「現在心裡產生了貪婪的念頭」或「現在心裡產生了散亂的念頭」。透過這種「元認知」的觀察，你不再被念頭牽著走，而是成為觀察念頭的人。

4. **分析「因緣模式」（法觀）**：
   當遇到挫折時，分析這件事發生的因緣（例如：因為 A 條件加上 B 因素，所以產生了 C 結果）。透過理性的邏輯分析，將情緒化的事件轉化為對「法」（現象規律）的認知，從而消除不必要的痛苦。
