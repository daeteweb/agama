# 雜阿含經 第 287 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「我憶宿命未成正覺時，獨一靜處，專精禪思，作是念：『何法有故老死有？何法緣故老死有？』即正思惟，生如實無間等，生有故老死有，生緣故老死有。如是有、取、愛、受、觸、六入處、名色。何法有故名色有？何法緣故名色有？即正思惟，如實無間等生，識有故名色有，識緣故有名色有。我作是思惟時，齊識而還不能過彼，謂緣識名色，緣名色六入處，緣六入處觸，緣觸受，緣受愛，緣愛取，緣取有，緣有生，緣生老、病、死、憂、悲、惱、苦。如是如是純大苦聚集。

「我時作是念：『何法無故則老死無？何法滅故老死滅？』即正思惟，生如實無間等，生無故老死無，生滅故老死滅，如是生、有、取、愛、受、觸、六入處、名色、識、行……」廣說。

「我復作是思惟：『何法無故行無？何法滅故行滅？』即正思惟，如實無間等，無明無故行無，無明滅故行滅，行滅故識滅，識滅故名色滅，名色滅故六入處滅，六入處滅故觸滅，觸滅故受滅，受滅故愛滅，愛滅故取滅，取滅故有滅，有滅故生滅，生滅故老、病、死、憂、悲、惱、苦滅，如是如是純大苦聚滅。

「我時作是念：『我得古仙人道、古仙人逕、古仙人道跡，古仙人從此跡去，我今隨去。』譬如有人遊於曠野，披荒覓路，忽遇故道古人行處，彼則隨行，漸漸前進，見故城邑、故王宮殿、園觀浴池、林木清淨。彼作是念：『我今當往白王令知。』即往白王：『大王當知，我遊曠野，披荒求路，忽見故道古人行處，我即隨行；我隨行已，見故城邑、故王宮殿、園觀浴池、林流清淨，大王可往居止其中。』王即往彼，止住其中，豐樂安隱，人民熾盛。

「今我如是，得古仙人道、古仙人逕、古仙人跡，古仙人去處，我得隨去，謂八聖道，正見、正志、正語、正業、正命、正方便、正念、正定。

### [白話翻譯]
**◈ 如是我聞：一時，佛住舍衛國祇樹給孤獨園。**
　　我是這樣聽佛陀說的：那時，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「我憶宿命未成正覺時，獨一靜處，專精禪思，作是念：『何法有故老死有？何法緣故老死有？』**
　　那時，世尊告訴在場的眾比丘說：「我回憶起在尚未成就正覺（圓滿覺悟）之前的過去生中，我獨自處於安靜之處，專注地進行禪修思維，心中產生這樣的念頭：『是什麼樣的條件導致了老死（衰老與死亡）的存在？又是因為什麼因緣導致了老死的發生？』」

**◈ 即正思惟，生如實無間等，生有故老死有，生緣故老死有。如是有、取、愛、受、觸、六入處、名色。**
　　接著我進行正思惟（正確的思考），如實且連續不斷地觀察到：因為有了『有（存在）』，所以產生了老死；因為有了『有』作為因緣，才導致了老死的發生。這其中的環節包含了：有（存在）、取（執取）、愛（貪愛）、受（感受）、觸（接觸）、六入處（感官與對象）、名色（心理與物質現象）。

**◈ 何法有故名色有？何法緣故名色有？即正思惟，如實無間等生，識有故名色有，識緣故有名色有。**
　　我進一步思考：是什麼樣的條件導致了名色（心理與物質現象）的存在？又是因為什麼因緣導致了名色的發生？透過正思惟，我如實且連續不斷地觀察到：因為有了『識（意識）』，所以產生了名色；因為有了『識』作為因緣，才導致了名色的發生。

**◈ 我作是思惟時，齊識而還不能過彼，謂緣識名色，緣名色六入處，緣六入處觸，緣觸受，緣受愛，緣愛取，緣取有，緣有生，緣生老、病、死、憂、悲、惱、苦。如是如是純大苦聚集。**
　　當我思考到『識（意識）』的時候，發現因果鏈條就此展開，無法超越：因為緣起於識（意識），所以產生了名色（心理與物質現象）；因為緣起於名色，所以產生了六入處（感官與對象）；因為緣起於六入處，所以產生了觸（接觸）；因為緣起於觸，所以產生了受（感受）；因為緣起於受，所以產生了愛（貪愛）；因為緣起於愛，所以產生了取（執取）；因為緣起於取，所以產生了有（存在）；因為緣起於有，所以產生了生（出生）；因為緣起於生，所以導致了老、病、死、憂、悲、惱、苦。這就是如此不斷累積而成的巨大痛苦集合。

**◈ 「我時作是念：『何法無故則老死無？何法滅故老死滅？』即正思惟，生如實無間等，生無故老死無，生滅故老死滅，如是生、有、取、愛、受、觸、六入處、名色、識、行……」廣說。**
　　那時我心中想著：「是什麼樣的條件不存在，則老死就不會存在？又是因為什麼因緣滅除，老死才會滅除？」接著我進行正思惟，如實且連續不斷地觀察到：因為『生（出生）』的消失，老死便不再發生；因為『生』的滅除，老死也隨之滅除。這包含了從生、有、取、愛、受、觸、六入處、名色、識，直到行（心的造作）……的連鎖反應。」佛陀對此進行了廣泛的說明。

**◈ 「我復作是思惟：『何法無故行無？何法滅故行滅？』即正思惟，如實無間等，無明無故行無，無明滅故行滅，行滅故識滅，識滅故名色滅，名色滅故六入處滅，六入處滅故觸滅，觸滅故受滅，受滅故愛滅，愛滅故取滅，取滅故有滅，有滅故生滅，生滅故老、病、死、憂、悲、惱、苦滅，如是如是純大苦聚滅。**
　　「我再次思考：『是什麼樣的條件不存在，則行（心的造作）就不會存在？又是因為什麼因緣滅除，行才會滅除？』透過正思惟，我如實且連續不斷地觀察到：因為無明（對真理的無知）的不存在，所以行（心的造作）不再發生；因為無明滅除了，行也隨之滅除。當行滅除了，識（意識）便滅除；當識滅除了，名色（心理與物質現象）便滅除；當名色滅除了，六入處（感官與對象）便滅除；當六入處滅除了，觸（接觸）便滅除；當觸滅除了，受（感受）便滅除；當受滅除了，愛（貪愛）便滅除；當愛滅除了，取（執取）便滅除；當取滅除了，有（存在）便滅除；當有滅除了，生（出生）便滅除；當生滅除了，老、病、死、憂、悲、惱、苦也隨之滅除。如此一來，這巨大的痛苦集合也隨之滅除了。」

**◈ 「我時作是念：『我得古仙人道、古仙人逕、古仙人道跡，古仙人從此跡去，我今隨去。』譬如有人遊於曠野，披荒覓路，忽遇故道古人行處，彼則隨行，漸漸前進，見故城邑、故王宮殿、園觀浴池、林木清淨。**
　　「那時我心想：『我發現了古仙人所走過的道路、路徑與足跡，古仙人正是從這裡走去的，我現在要跟隨這條路前進。』好比有一個人在荒野中行走，在雜草叢中尋找路徑，突然發現了古人曾經走過的老路，於是便跟著那條路前進，漸漸地，他看到了舊有的城邑、王宮殿堂、庭園池畔以及清淨的林木。

**◈ 彼作是念：『我今當往白王令知。』即往白王：『大王當知，我遊曠野，披荒求路，忽見故道古人行處，我即隨行；我隨行已，見故城邑、故王宮殿、園觀浴池、林流清淨，大王可往居止其中。』王即往彼，止住其中，豐樂安隱，人民熾盛。**
　　那人想著：『我現在應該去告訴國王。』於是前往向國王禀報說：『大王請知悉，我在荒野中尋路時，偶然發現了古人的舊道，我便跟隨其後；沿途我看到了城邑、宮殿與清淨的林園，大王可以帶領眾人前往那裡居住。』國王隨即前往該處定居，那裡生活豐饒、安穩舒適，人民也繁榮興盛。

**◈ 「今我如是，得古仙人道、古仙人逕、古仙人跡，古仙人去處，我得隨去，謂八聖道，正見、正志、正語、正業、正命、正方便、正念、正定。**
　　「現在的我亦是如此，我找到了古仙人的道路、路徑與足跡，並跟隨著他們的去處，這條路就是八聖道（聖者的八種修行途徑），即：正見（正確的見解）、正志（正確的意圖/思惟）、正語（正確的言語）、正業（正確的行為）、正命（正確的生活方式）、正方便（正確的精進）、正念（正確的覺察）、正定（正確的禪定）。」

### [詞語註解（解釋重要佛教概念）]
1. **十二因緣 (Twelve Nidanas)**：描述生命如何因條件而生、因條件滅的連鎖機制，從無明開始，經過行、識、名色、六入處、觸、受、愛、取、有、生，最終導致老死苦。
2. **正思惟 (Sammā-saṅkappa)**：八正道之一，指排除貪欲、瞋恚與傷害性的正確思考方式。
3. **無明 (Avijjā)**：對四聖諦與因緣法則的根本無知，是所有苦難的起點。
4. **八聖道 (Ariya-aṭṭhaṅgika-magga)**：通往解脫的八種正確修行途徑，包含智慧、道德與定力三個面向。

### [現代實作指南]
本經文提供了「因果溯源」與「路徑跟隨」兩種極具價值的心理實踐方法：

1. **情緒與痛苦的溯源練習（拆解十二因緣）**：
   當你感到強烈的負面情緒（如憤怒、焦慮或憂鬱）時，不要只停留在「我好痛苦」的表面。試著像佛陀一樣進行「正思惟」，將情緒拆解為連鎖反應：
   * **受（感受）** $\rightarrow$ 我現在感覺到一種不舒服的壓力。
   * **愛（貪愛/瞋愛）** $\rightarrow$ 我為什麼想要排斥這種感覺？或為什麼我想要抓住某種特定的狀態？
   * **取（執取）** $\rightarrow$ 我是否在心裡對這個感受進行了「這是不對的」、「我不該這樣」的過度標籤與抓取？
   * **有（存在/造作）** $\rightarrow$ 這種抓取如何形塑了我現在「一個受害者」或「一個憤怒者」的身分感？
   透過這種拆解，你會發現痛苦並非一個整體，而是一系列可以被觀察與切斷的因果鏈。

2. **尋找「古人足跡」的學習法（借鑒智慧）**：
   當你在生活中感到迷惘、像在荒野中覓路時，不要試圖憑空創造全新的生存法則。去尋找那些已經被證明有效的「老路」（如經典教導、前人的成功經驗或成熟的心靈實踐方法）。一旦找到這條路（如八正道），就專注於「隨行」，即一步步落實正見與正念，而非頻繁更換目標。

### [段落大意]
佛陀回憶在成道前，透過觀察十二因緣的生滅規律，發現了痛苦（老死）是如何由無明、行、識等環節連鎖產生，以及如何透過滅除無明來終止苦難。佛陀將此發現比喻為在荒野中發現古人的舊路，最終引導眾人走向能帶來安穩與解脫的「八聖道」。

# 雜阿含經 第 N 經

### [原文]
我從彼道見老病死、老病死集、老病死滅、老病死滅道跡，

見生、有、取、愛、受、觸、六入處、名色、識、行、行集、行滅、行滅道跡。

我於此法自知自覺，成等正覺，

為比丘、比丘尼、優婆塞、優婆夷，及餘外道沙門、婆羅門、在家、出家，彼諸四眾聞法正向、信樂，知法善，梵行增廣，多所饒益，開示顯發。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 我從彼道見老病死、老病死集、老病死滅、老病死滅道跡**
　　我透過那條路徑，看見了老、病、死的苦難，也看見了導致老、病、死的苦因（集），以及老、病、死苦止息的狀態（滅），並找到了通往老、病、死苦止息的方法與路徑（道跡）。

**◈ 見生、有、取、愛、受、觸、六入處、名色、識、行、行集、行滅、行滅道跡**
　　我觀察到出生、存在（有）、執取（取）、貪愛（愛）、感受（受）、接觸（觸）、六種感官進入世界的通道（六入處）、身心現象（名色）、識（意識）、行（心的造作），以及導致心的造作產生的原因（行集）、使心的造作止息的狀態（行滅）與通往其止息的路徑（行滅道跡）。

**◈ 我於此法自知自覺，成等正覺**
　　我對這套因緣法理進行了親身的體證與覺悟，成就了圓滿的正等正覺。

**◈ 為比丘、比丘尼、優婆塞、優婆夷，及餘外道沙門、婆羅門、在家、出家，彼諸四眾聞法正向、信樂，知法善，梵行增廣，多所饒益，開示顯發**
　　我向在場的比丘（男修行者）、比丘尼（女修行者）、優婆塞（在家男信徒）、優婆夷（在家女信徒），以及其他的沙門（修行者）、婆羅門（祭司階級）、出家與在家的眾生，進行教導。這些大眾聽聞了佛法後，心念轉向正確、生起信賴與喜悅，體認到這套法理的良善，增長了清淨的修行，並從中獲得極大的利益，使智慧得以開顯。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行**
　　佛陀講完後，比丘們聽聞教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **八聖道 (Ariyaṭṭhaṅgika-magga)**：指通往解脫的八種正確修行路徑，包含正見、正思惟、正語、正業、正命、正精進、正念與正定。
2. **十二因緣 (Paṭicca-samuppāda)**：描述生命苦難如何由一系列環環相扣的條件（如無明、行、識等）而產生的因果鏈條。
3. **四眾 (Cattāri upāsaka/bhikkhus etc.)**：指佛教社群中的四大核心群體，即比丘、比丘尼、優婆塞與優婆夷。
4. **等正覺 (Sammā-sambodhi)**：指修行者透過完全的智慧，對宇宙真理（法）達到無誤且圓滿的覺悟。

### [現代實作指南]
本段經文揭示了「觀察因果鏈條」的重要性，可轉化為以下生活心理實踐技巧：

1. **因果溯源練習**：當你感到強烈的負面情緒（如憤怒或焦慮）時，不要只停留在「受（感受）」的層次。嘗試往回追溯：這個情緒是因為什麼樣的「觸（接觸/刺激）」引起的？是否因為我對某件事產生了過度的「取（執取/偏見）」？透過觀察「行（心的造作）」，你可以發現情緒是如何被製造出來的。
2. **打破自動反應**：經文提到從「行集」到「行滅」。在日常生活中，當你察覺到自己正陷入一種慣性的負面思考模式（行）時，刻意停頓，觀察那個導致這種思考產生的念頭（行集），並利用「正念」來切斷這個鏈條，尋找讓情緒平息的路徑（道跡）。
3. **建立正確的價值觀**：運用「八聖道」中的正見與正思惟，在面對複雜資訊時，不急於做出判斷，而是先審視自己的認知是否符合事實，避免因錯誤的認知而導致後續錯誤的行為。
