# 雜阿含經 第 71 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「我今當說有身、有身集、有身滅、有身滅道跡。諦聽，善思，當為汝說。云何有身？謂五受陰。云何為五？色受陰，受、想、行、識受陰，是名有身。云何有身集？當來有愛，貪、喜俱，彼彼染著，是名有身集。云何有身滅？當來有愛，貪、喜俱，彼彼樂著無餘斷、吐、盡、離欲、滅，是名有身滅。

「云何有身滅道跡？謂八聖道，正見、正志、正語、正業、正命、正方便、正念、正定，是名有身滅道跡。是名當說有身、有身集、有身滅、有身滅道跡。」

佛說是經已，諸比丘聞佛所說，歡喜奉行。

餘如是說。差別者：「當知有身，當知斷有身集，當知證有身滅，當知修斷有身道跡。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如當說，有及當知，亦如是說。又復差別者：「比丘知有身，斷有身集，證有身滅，修斷有身道，是名比丘斷愛欲縛諸結等法，修無間等，究竟苦邊。」

又復差別者：「是名比丘究竟邊際，究竟離垢，究竟梵行，純淨上士。」

又復差別者：「是名比丘阿羅漢盡諸有漏，所作已作，已捨重擔，逮得己利，盡諸有結，正智心解脫。」

又復差別者：「是名比丘斷關、度塹，超越境界，脫諸防邏，建聖法幢。」又復差別者：「云何斷關？謂斷五下分結。云何度塹？謂度無明深塹。云何超越境界？謂究竟無始生死。云何脫諸防邏？謂有愛盡。云何建聖法幢？謂我慢盡。」

又復差別者：「是名比丘斷五枝，成六枝，守護一，依四種，棄捨諸諦，離諸求，淨諸覺，身行息，心善解脫，慧善解脫，純一立梵行，無上士。」

　其道有三種　　實覺亦三種 　有身四種說　　羅漢有六種

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「我今當說有身、有身集、有身滅、有身滅道跡。**
　　那時，世尊告訴在場的眾位比丘說：「我現在要為你們解說關於『存在（有身）』、『存在的苦因（有身集）』、『存在的苦止（有身滅）』以及『導向存在苦止的解脫之道（有身滅道跡）』。

**◈ 諦聽，善思，當為汝說。**
　　請你們仔細聽，好好思考，我現在為你們解說。

**◈ 云何有身？謂五受陰。**
　　什麼是『存在（有身）』？指的是五種執著的蘊（五受陰）。

**◈ 云何為五？色受陰，受、想、行、識受陰，是名有身。**
　　這五種是什麼？分別是色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）這五種執著的蘊，這就稱為『存在（有身）』。

**◈ 云何有身集？當來有愛，貪、喜俱，彼彼染著，是名有身集。**
　　什麼是『存在的苦因（有身集）』？是指對未來存在的貪愛（當來有愛），伴隨著貪婪與喜悅，對這些對象產生染著，這就稱為『存在的苦因（有身集）』。

**◈ 云何有身滅？當來有愛，貪、喜俱，彼彼樂著無餘斷、吐、盡、離欲、滅，是名有身滅。**
　　什麼是『存在的苦止（有身滅）』？是指對未來存在的貪愛，以及隨之而來的貪婪與喜悅，對這些對象的樂著被完全地斷除、吐棄、盡除、離欲並滅盡，這就稱為『存在的苦止（有身滅）』。

**◈ 云何有身滅道跡？謂八聖道，正見、正志、正語、正業、正命、正方便、正念、正定，是名有身滅道跡。**
　　什麼是『導向存在苦止的解脫之道（有身滅道跡）』？指的是八正道：正見、正思惟（正志）、正語、正業、正命、正精進（正方便）、正念、正定，這就稱為『導向存在苦止的解脫之道（有身滅道跡）』。

**◈ 是名當說有身、有身集、有身滅、有身滅道跡。」**
　　這就是我所要說的關於存在、存在的苦因、存在的苦止以及導向存在苦止的解脫之道。」

**◈ 佛說是經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 餘如是說。差別者：「當知有身，當知斷有身集，當知證有身滅，當知修斷有身道跡。」**
　　其餘的部分也是這樣說的。僅在措辭上有所不同，例如：「應當了解存在，應當了解如何斷除存在的苦因，應當了解如何證悟存在的苦止，應當了解如何修行以斷除存在的解脫之道。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如當說，有及當知，亦如是說。**
　　關於『應當說』以及『應當了解』的表述，也是同樣這樣說的。

**◈ 又復差別者：「比丘知有身，斷有身集，證有身滅，修斷有身道，是名比丘斷愛欲縛諸結等法，修無間等，究竟苦邊。」**
　　另外在措辭上又有所不同：「比丘了解存在，斷除存在的苦因，證悟存在的苦止，修行斷除存在的道路，這就稱為比丘斷除了由愛欲所束縛的種種結使（煩惱），修行無間斷的定力，最終到達苦的邊際（結束苦輪迴）。」

**◈ 又復差別者：「是名比丘究竟邊際，究竟離垢，究竟梵行，純淨上士。」**
　　另外在措辭上又有所不同：「這就稱為比丘到達了最終的邊際，徹底遠離了垢染，實踐了究竟的梵行（清淨生活），成為純淨的高尚之人。」

**◈ 又復差別者：「是名比丘阿羅漢盡諸有漏，所作已作，已捨重擔，逮得己利，盡諸有結，正智心解脫。」**
　　另外在措辭上又有所不同：「這就稱為比丘已成為阿羅漢，盡除了所有有漏（受煩惱染污）的狀態，完成了應做的事業，捨棄了沉重的重擔，獲得了自身的利益，斷盡了所有存在的結使，以正智獲得了心的解脫。」

**◈ 又復差別者：「是名比丘斷關、度塹，超越境界，脫諸防邏，建聖法幢。」**
　　另外在措辭上又有所不同：「這就稱為比丘斷開了關隘、跨越了深塹，超越了生死境界，脫離了種種防禦的柵欄，建立了聖法的旗幟。」

**◈ 又復差別者：「云何斷關？謂斷五下分結。云何度塹？謂度無明深塹。云何超越境界？謂究竟無始生死。云何脫諸防邏？謂有愛盡。云何建聖法幢？謂我慢盡。」**
　　另外在措辭上又有所不同：「什麼是斷關？是指斷除五種下分結（較淺層的煩惱）。什麼是度塹？是指跨越無明（缺乏真理覺知）的深塹。什麼是超越境界？是指徹底終結無始以來的生死輪迴。什麼是脫諸防邏？是指對存在的貪愛盡除。什麼是建聖法幢？是指我慢（對自我的傲慢執著）盡除。」

**◈ 又復差別者：「是名比丘斷五枝，成六枝，守護一，依四種，棄捨諸諦，離諸求，淨諸覺，身行息，心善解脫，慧善解脫，純一立梵行，無上士。」**
　　另外在措辭上又有所不同：「這就稱為比丘斷除了五種（煩惱的）枝條，成就了六種（覺悟的）枝條，守護唯一（的真理），依止四種（正道），捨棄對諸諦的執著，遠離種種追求，淨化種種覺知，身體的行止趨於寧靜，心獲得了善解脫，慧獲得了善解脫，純粹地建立梵行，成為無上的高尚之人。」

**◈ 其道有三種 實覺亦三種 有身四種說 羅漢有六種**
　　修行道路有三種，真實的覺悟亦有三種，關於存在的說法有四種，關於阿羅漢的描述有六種。

### [詞語註解（解釋重要佛教概念）]
1. **有身 (Bhava)**：在此處並非指物質身體，而是指「存在」或「成為（Becoming）」。它指的是在輪迴中不斷生起、維持生命狀態的過程。
2. **五受陰 (Upādānakkhandha)**：指被執著的五蘊。當色、受、想、行、識這五種要素被視為「我」而產生執著時，便形成了輪迴的基礎。
3. **當來有愛 (Bhavataṇhā)**：對未來存在、對繼續生存或對更高層次存在狀態的貪愛。這是驅動輪迴（有身集）的核心動力。
4. **五下分結 (Orambhagiya-samyojana)**：指將眾生束縛在欲界中的五種較淺層煩惱，包括：身見（執著有我）、疑（對正法懷疑）、戒禁見（執著錯誤的儀軌）、貪欲、瞋恚。
5. **我慢 (Māna)**：對「我」的傲慢或執著，認為自己比他人優、劣或相等，是深層的自我認同感。
6. **有漏 (Āsava)**：指被煩惱（如貪、瞋、痴）所染污的狀態，如同容器漏水一樣，心靈不斷流出煩惱而無法安定。

### [現代實作指南]
本經將「四聖諦」的結構應用在「存在（有身）」的分析上，我們可以將其轉化為以下生活實踐：

1. **識別「存在」的組成（觀察五蘊）**：
   - 當你感到壓力或焦慮時，試著將其拆解：這是一個身體的反應（色）、一種不舒服的感覺（受）、一個對未來的擔憂認知（想）、一種想要逃避的衝動（行）、以及對此現象的覺知（識）。
   - **轉念技巧**：告訴自己「這不是『我』在痛苦，而是『五蘊』在運作」。

2. **覺察「對未來的貪愛」（分析有身集）**：
   - 觀察心中對「未來必須如何」的強烈執著（例如：我必須在明年升職、我必須被某人認可）。這種對未來特定狀態的渴望，正是導致當下不安的「有身集」。
   - **實作方法**：每當產生強烈期待時，標記它為「對未來的貪愛」，並將注意力拉回當下的呼吸。

3. **練習「斷除樂著」（實踐有身滅）**：
   - 意識到所有的快感（喜）與貪婪（貪）都是暫時的。當你對某種物質或名聲產生強烈依戀時，提醒自己這僅僅是「染著」。
   - **轉念技巧**：練習「吐棄」心靈的沉重感，意識到放下執著後，心靈會獲得真正的輕盈與寧靜。

4. **建立「聖法幢」（克服我慢）**：
   - 在日常互動中，當你感到被冒犯或想要證明自己正確時，意識到這是「我慢」在作祟。
   - **實作方法**：嘗試以「無我」的視角看待問題，將關注點從「我的面子」轉移到「如何解決問題」或「如何幫助他人」。

### [段落大意]
本經以四聖諦的框架分析「存在（有身）」，指出存在即是五蘊的執著，其成因是對未來存在的貪愛，而解脫之道則是透過八正道斷除此貪愛。經文後半段詳細描述了阿羅漢在斷除煩惱（如五下分結、我慢）後，超越生死輪迴、獲得心慧解脫的純淨狀態。
