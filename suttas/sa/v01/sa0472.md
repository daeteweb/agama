# 雜阿含經 第 472 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，世尊告諸比丘：「譬如客舍種種人住，若剎利、婆羅門、長者居士、野人獵師、持戒犯戒、在家出家，悉於中住。此身亦復如是，種種受生——苦受、樂受、不苦不樂受，樂身受、苦身受、不苦不樂身受，樂心受、苦心受、不苦不樂心受，樂食受、苦食受、不苦不樂食受，樂無食受、苦無食受、不苦不樂無食受，樂貪著受、苦貪著受、不苦不樂貪著受，樂出要受、苦出要受、不苦不樂出要受。」

爾時，世尊即說偈言：

「譬如客舍中，　　種種人住止， 　剎利婆羅門，　　長者居士等， 　旃陀羅野人，　　持戒犯戒者， 　在家出家人，　　如是等種種， 　此身亦如是。　　種種諸受生， 　若樂若苦受，　　及不苦不樂， 　有食與無食，　　貪著不貪著， 　比丘勤方便，　　正智不傾動， 　於此一切受，　　黠慧能了知， 　了知諸受故，　　現法盡諸漏， 　身死不墮數，　　永處般涅槃。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「譬如客舍種種人住，若剎利、婆羅門、長者居士、野人獵師、持戒犯戒、在家出家，悉於中住。**
　　「就好比一座客棧，有各種不同的人居住在裡面，無論是剎利（王族）、婆羅門（祭司）、長者、居士、野人、獵師，或是持戒的人、犯戒的人，以及在家的人或出家的人，全部都住在其中。

**◈ 此身亦復如是，種種受生——苦受、樂受、不苦不樂受，**
　　這個物質身體（色）也是如此，會產生各種不同的受（感受） $\rightarrow$ 包括苦受、樂受，以及不苦不樂的受。

**◈ 樂身受、苦身受、不苦不樂身受，樂心受、苦心受、不苦不樂心受，**
　　具體分為：身體上的樂受、苦受、不苦不樂受；以及心靈上的樂受、苦受、不苦不樂受。

**◈ 樂食受、苦食受、不苦不樂食受，樂無食受、苦無食受、不苦不樂無食受，**
　　還有飲食充足時的樂受、苦受、不苦不樂受；以及缺乏飲食時的樂受、苦受、不苦不樂受。

**◈ 樂貪著受、苦貪著受、不苦不樂貪著受，樂出要受、苦出要受、不苦不樂出要受。」**
　　此外，還包含對貪著（執著）產生的樂受、苦受、不苦不樂受；以及對出要（追求解脫）產生的樂受、苦受、不苦不樂受。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即說了偈頌（詩偈）道：

**◈ 「譬如客舍中，種種人住止，剎利婆羅門，長者居士等，旃陀羅野人，持戒犯戒者，在家出家人，如是等種種，**
　　「就像客棧之中，各種不同的人在此停留，無論是王族、婆羅門，或是長者與居士，甚至是賤民、野人，持戒者或犯戒者，在家者或出家人，就像這樣種種不同的人。

**◈ 此身亦如是。種種諸受生，若樂若苦受，及不苦不樂，**
　　這個物質身體（色）也是如此，會生起各種不同的受（感受），有快樂的、痛苦的，以及不苦不樂的。

**◈ 有食與無食，貪著不貪著，**
　　無論是飲食滿足或匱乏，或是產生貪著執著或不貪著。

**◈ 比丘勤方便，正智不傾動，於此一切受，黠慧能了知，**
　　比丘應當勤於運用修行方法，以正智（正確的智慧）保持心不動搖，對於這一切的受（感受），能以敏銳的智慧去透徹了解。

**◈ 了知諸受故，現法盡諸漏，身死不墮數，永處般涅槃。」**
　　因為能透徹了解所有的受（感受），所以能在現法（當下的生命）中盡除所有的漏（煩惱與缺陷），在身體死亡後不再墮入輪迴的數量（生死循環）中，永遠處於超越生死、絕對寧靜的解脫境界（涅槃）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **受 (Vedanā)**：指意識對刺激產生的初步情感反應，分為苦、樂、不苦不樂（捨受）三種。本經強調「受」僅是暫時造訪身體的「客人」，而非恆常的自我。
2. **漏 (Āsava)**：原意為「滲漏」，指潛在於心底、不斷流出的煩惱，主要包括欲漏（對感官慾望的執著）、有漏（對存在的執著）與無明漏（對真理的無知）。
3. **現法 (Samyutta/Present)**：指在目前的生命週期中，或在當下的覺知狀態中。
4. **不墮數 (Not falling into number)**：指不再進入輪迴（Saṃsāra）的次數計算中，即不再投生於六道之中。
5. **出要**：指渴望脫離輪迴、追求解脫（Vimokkha）的志向。

### [現代實作指南]
本經提供了一個極其強大的心理轉念模型：「客舍模型」。當我們在生活中面對強烈的情緒或身體感受時，可以採取以下實作步驟：

1. **標記感受（Labeling）**：
   當強烈的情緒（如憤怒、焦慮）或身體不適（如疼痛、飢餓）出現時，不要說「我很痛苦」或「我很生氣」，而是在心中標記：「現在有一個『苦受』的客人來訪了」。

2. **建立觀察者視角（The Observer）**：
   想像你的身體與心靈是一座「客棧」。感受（受）就像是入住的客人。客人的特點是：**會入住，也必然會離開**。
   $\rightarrow$ 提醒自己：我是客棧的主人（觀察者），而不是那個入住的客人（感受）。

3. **中立化處理（Neutralization）**：
   無論來訪的是「樂受」（愉快的客人）還是「苦受」（不愉快的客人），都以同樣的態度接待。
   $\rightarrow$ 對於樂受：不因其美好而產生執著（貪著），知道它終將離去。
   $\rightarrow$ 對於苦受：不因其痛苦而產生排斥，知道它也終將離去。

4. **正智不傾動（Equanimity）**：
   在感受起伏時，練習保持一種「不傾動」的狀態。意識到感受的生起 $\rightarrow$ 觀察感受的變化 $\rightarrow$ 觀察感受的消失。透過這種練習，能切斷「受 $\rightarrow$ 愛（貪愛） $\rightarrow$ 取（執取）」的十二因緣鏈條，從而減少煩惱的產生。

### [段落大意]
佛陀以「客舍」比喻物質身體（色），說明各種受（感受）如同不同身分的人入住客棧一樣，僅是暫時的造訪。無論是身體或心理的苦、樂、不苦不樂，以及飲食或貪著所引起的感受，本質上都是無常且非自我的。比丘若能以正智敏銳地觀察並透徹了解這些感受的本質，便能在當下盡除煩惱，脫離輪迴，最終證得涅槃。
