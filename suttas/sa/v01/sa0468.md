# 雜阿含經 第 468 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，尊者羅睺羅往詣佛所，稽首佛足，退坐一面，白佛言：「世尊！云何知、云何見我此識身及外境界一切相，得無有我、我所見、我慢繫著使？」

佛告羅睺羅：「有三受——苦受、樂受、不苦不樂受。觀於樂受，為斷樂受貪使故，於我所修梵行；斷苦受瞋恚使故，於我所修梵行；斷不苦不樂受癡使故，於我所修梵行。

「羅睺羅！若比丘樂受貪使已斷、已知，苦受恚使已斷、已知，不苦不樂受癡使已斷、已知者，是名比丘斷除愛欲縛、去諸結、慢無間等、究竟苦邊。」

爾時，世尊即說偈言：

「樂受所受時，　　則不知樂受， 　貪使之所使，　　不見出要道。 　苦受所受時，　　則不知苦受， 　瞋恚使所使，　　不見出要道。 　不苦不樂受，　　正覺之所說， 　不善觀察者，　　終不度彼岸。 　比丘勤精進，　　正知不動轉， 　如此一切受，　　慧者能覺知。 　覺知諸受者，　　現法盡諸漏， 　明智者命終，　　不墮於眾數。 　眾數既已斷，　　永處般涅槃。」

佛說此經已，尊者羅睺羅聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，尊者羅睺羅往詣佛所，稽首佛足，退坐一面，白佛言：**
　　那時，尊者羅睺羅走到佛陀面前，恭敬地頂禮佛陀的足部，退到一邊坐下，對佛陀恭敬地說道：

**◈ 「世尊！云何知、云何見我此識身及外境界一切相，得無有我、我所見、我慢繫著使？」**
　　「世尊！我應該如何去認知、如何去觀察我這個識（意識）的身體，以及外部環境的所有現象，才能讓心中不再產生『我是』、『這是我的』這種執著，以及不再被傲慢的自我意識所束縛？」

**◈ 佛告羅睺羅：「有三受——苦受、樂受、不苦不樂受。」**
　　佛陀告訴羅睺羅：「感受（受）分為三種 → 苦受（痛苦的感受）、樂受（快樂的感受）、以及不苦不樂受（中性的感受）。」

**◈ 「觀於樂受，為斷樂受貪使故，於我所修梵行；斷苦受瞋恚使故，於我所修梵行；斷不苦不樂受癡使故，於我所修梵行。」**
　　「觀察快樂的感受，為了斷除對樂受的貪愛（愛）之驅使，我才修行清淨的梵行；為了斷除對苦受的瞋恨之驅使，我才修行清淨的梵行；為了斷除對不苦不樂受的愚癡之驅使，我才修行清淨的梵行。」

**◈ 「羅睺羅！若比丘樂受貪使已斷、已知，苦受恚使已斷、已知，不苦不樂受癡使已斷、已知者，是名比丘斷除愛欲縛、去諸結、慢無間等、究竟苦邊。」**
　　「羅睺羅！如果一位比丘能夠斷除並覺知樂受中的貪愛、斷除並覺知苦受中的瞋恨、以及斷除並覺知不苦不樂受中的愚癡，這就叫做比丘斷除了愛（貪愛）與欲望的束縛，除去了所有心理的結結（執著），消除了傲慢的隔閡，最終到達了苦的盡頭。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即說了偈頌：

**◈ 「樂受所受時，則不知樂受，貪使之所使，不見出要道。」**
　　「當處於快樂的感受中時，若不能覺知這僅僅是樂受，就會被貪愛所驅使，而看不見解脫的道路。」

**◈ 「苦受所受時，則不知苦受，瞋恚使所使，不見出要道。」**
　　「當處於痛苦的感受中時，若不能覺知這僅僅是苦受，就會被瞋恨所驅使，而看不見解脫的道路。」

**◈ 「不苦不樂受，正覺之所說，不善觀察者，終不度彼岸。」**
　　「至於不苦不樂的中性感受，這是正覺者（佛陀）所說的關鍵，若不能善於觀察它，最終將無法渡到解脫的彼岸。」

**◈ 「比丘勤精進，正知不動轉，如此一切受，慧者能覺知。」**
　　「比丘應勤於精進，保持正知且心不動搖，如此觀察所有感受，具備智慧的人才能真正覺知。」

**◈ 「覺知諸受者，現法盡諸漏，明智者命終，不墮於眾數。」**
　　「能覺知所有感受的人，能在現法（當下的生命）中盡除所有煩惱的漏失；這樣明智的人在生命結束時，不會再墮入輪迴的眾數之中。」

**◈ 「眾數既已斷，永處般涅槃。」**
　　「當輪迴的因緣被斷除後，將永遠處於超越生死、絕對寧靜的解脫境界（涅槃）中。」

**◈ 佛說此經已，尊者羅睺羅聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，尊者羅睺羅聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **受 (Vedanā)**：指意識對刺激產生的初步情感反應，分為苦、樂、不苦不樂三種。它是十二因緣中觸之後的環節，是產生貪愛（愛）的直接誘因。
2. **梵行 (Brahmacariya)**：原意為「梵天般的行徑」，在原始佛教中指為了解脫而實踐的清淨生活與修行，包括戒律的遵守與心的調伏。
3. **漏 (Āsava)**：指像滲漏一樣不斷流出的煩惱，主要包括欲漏（對感官欲望的執著）、有漏（對存在與生存的執著）、無明漏（對真理的不覺知）。
4. **眾數 (Sāmañña/Common group)**：在此指普通眾生在輪迴（Saṃsāra）中共同的處境，即不斷在生死中流轉的狀態。

### [現代實作指南]
本經文提供了一套極其精準的「情緒管理與解脫」操作流程，可轉化為以下心理實踐步驟：

1. **即時標記法 (Labeling)**：
   當情緒升起時，不要進入情緒的劇情（例如：不要想「他為什麼這樣對我」），而是立即在心中將其標記為：
   - 「這是樂受」 $\rightarrow$ 覺知到愉悅感。
   - 「這是苦受」 $\rightarrow$ 覺知到不適感。
   - 「這是不苦不樂受」 $\rightarrow$ 覺知到平淡或麻木感。

2. **觀察驅動力 (Observing the Driver)**：
   在標記之後，觀察隨之而來的心理傾向：
   - 樂受 $\rightarrow$ 是否產生了「想要更多」、「不想失去」的**貪愛**？
   - 苦受 $\rightarrow$ 是否產生了「想要排斥」、「感到憤怒」的**瞋恨**？
   - 中性受 $\rightarrow$ 是否陷入了「沒感覺」、「心不在焉」的**愚癡**？

3. **去我化觀察 (De-identification)**：
   將「我很痛苦」轉變為「現在有一個苦受在發生」。透過將「受」視為一種客觀的心理現象，而非「我」的屬性，從而切斷「我慢」與「我所」的繫著。

4. **維持正知 (Maintaining Mindfulness)**：
   練習在情緒波動時，像觀察天氣一樣觀察感受的生起與滅去，不隨之起舞，不試圖強行改變它，僅僅是「覺知」。

### [段落大意]
尊者羅睺羅請教佛陀如何消除對自我與外境的執著。佛陀指導他應觀察三種感受（苦、樂、不苦不樂），並分別斷除其對應的貪、瞋、癡。透過精進覺知所有感受的本質，比丘能盡除煩惱漏失，脫離輪迴，最終證得涅槃。
