# 雜阿含經 第 343 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。爾時，尊者浮彌比丘住耆闍崛山。

時，有眾多外道出家詣尊者浮彌所，共相問訊慶慰，共相問訊慶慰已，退坐一面，語尊者浮彌言：「欲有所問，寧有閑暇見答與不？」

尊者浮彌語諸外道出家：「隨汝所問，當為汝說。」

時，諸外道出家問尊者浮彌：「苦樂自作耶？」

尊者浮彌答言：「諸外道出家說苦樂自作者，世尊說言：『此是無記。』」

復問：「苦樂他作耶？」

答言：「苦樂他作者，世尊說言：『此是無記。』」

復問：「苦樂自他作耶？」

答言：「苦樂自他作者，世尊說言：『此是無記。』」

復問：「苦樂非自非他無因作耶？」

答言：「苦樂非自非他無因作者，世尊說言：『此是無記。』」

諸外道出家復問：「云何？尊者浮彌苦樂自作耶？說言無記。苦樂他作耶？說言無記。苦樂自他作耶？說言無記。苦樂非自非他無因作耶？說言無記。今沙門瞿曇說苦樂云何生？」

尊者浮彌答言：「諸外道出家！世尊說苦樂從緣起生。」

時，諸外道出家聞尊者浮彌所說，心不歡喜，呵責而去。

爾時，尊者舍利弗去尊者浮彌不遠，坐一樹下。

爾時，尊者浮彌知諸外道出家去已，往詣尊者舍利弗所，到已，與舍利弗面相慶慰。慶慰已，以彼諸外道出家所問事，具白尊者舍利弗：「我作此答，得不謗毀世尊！如說說、不如法說、不為是隨順法行法，得無為餘因法論者來難詰呵責不？」

尊者舍利弗言：「尊者浮彌！汝之所說，實如佛說，不謗如來，如說說、如法說、法行法說，不為餘因論義者來難詰呵責。所以者何？世尊說苦樂從緣起生故。尊者浮彌！彼諸沙門、婆羅門所問苦樂自作者，彼亦從因起生；言不從緣起生者，無有是處。苦樂他作、自他作、非自非他無因作說者，彼亦從緣起生；若言不從緣生者，無有是處。尊者浮彌！彼沙門、婆羅門所說苦樂自作者，亦緣觸生；若言不從觸生者，無有是處。

苦樂他作、自他作、非自非他無因作者，彼亦緣觸生，若言不緣觸生者，無有是處。」

爾時，尊者阿難去舍利弗不遠，坐一樹下，聞尊者舍利弗與尊者浮彌所論說事。聞已，從座起，往詣佛所，稽首佛足，退住一面。以尊者浮彌與尊者舍利弗共論說一一具白世尊。

佛告阿難：「善哉！善哉！阿難！尊者舍利弗有來問者，能隨時答。善哉！舍利弗！有應時智故，有來問者，能隨時答。若我聲聞，有隨時問者，應隨時答，如舍利弗所說。

「阿難！我昔時住王舍城山中仙人住處，有諸外道出家以如是義、如是句、如是味來問於我，我為斯等以如是義、如是句、如是味而為記說，如尊者舍利弗所說。

「阿難！若諸沙門、婆羅門苦樂自作，我即往彼問言：『汝實作是說苦樂自作耶？』彼答我言：『如是。』我即問言：『汝能堅執持此義，言是真實，餘則愚者，我所不許。所以者何？我說苦樂所起異於此。』彼若問我：『云何瞿曇所說，苦樂所起異者？』我當答言：『從其緣起而生苦樂。如是說苦他作、自他作、非自非他無因作者，我亦往彼所說如上。』」

阿難白佛：「如世尊所說義，我已解知，有生故有老死，非緣餘；有生故有老死，乃至無明故有行，非緣餘；有無明故有行，無明滅則行滅，乃至生滅則老、病、死、憂、悲、惱、苦滅，如是純大苦聚滅。」

佛說此經已，尊者阿難聞佛所說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。爾時，尊者浮彌比丘住耆闍崛山。**
　　在某一個時間，佛住在王舍城的迦蘭陀竹園。那時，尊者浮彌比丘住在耆闍崛山。

**◈ 時，有眾多外道出家詣尊者浮彌所，共相問訊慶慰，共相問訊慶慰已，退坐一面，語尊者浮彌言：「欲有所問，寧有閑暇見答與不？」**
　　那時，有許多外道出家的人前往尊者浮彌比丘那裡，彼此互相問候關心。問候完畢後，他們退到一邊坐下，對尊者浮彌比丘說：「我們想請問一些問題，請問您現在是否有空閒時間為我們解答？」

**◈ 尊者浮彌語諸外道出家：「隨汝所問，當為汝說。」**
　　尊者浮彌比丘告訴那些外道出家的人：「請隨意提問，我將為你們解答。」

**◈ 時，諸外道出家問尊者浮彌：「苦樂自作耶？」**
　　那時，那些外道出家的人問尊者浮彌比丘：「苦（痛苦）與樂（快樂）是由自己創造的嗎？」

**◈ 尊者浮彌答言：「諸外道出家說苦樂自作者，世尊說言：『此是無記。』」**
　　尊者浮彌比丘回答說：「對於那些主張苦（痛苦）與樂（快樂）是由自己創造的外道出家者，世尊說這屬於『無記』（指不採取肯定或否定的立場，因為該問題基於錯誤的假設，對解脫沒有幫助）。」

**◈ 復問：「苦樂他作耶？」**
　　他們又問：「苦（痛苦）與樂（快樂）是由他人或外在力量創造的嗎？」

**◈ 答言：「苦樂他作者，世尊說言：『此是無記。』」**
　　尊者浮彌比丘回答：「對於主張苦樂是由他人創造的說法，世尊說這同樣屬於『無記』。」

**◈ 復問：「苦樂自他作耶？」**
　　他們再次問：「苦（痛苦）與樂（快樂）是由自己與他人共同創造的嗎？」

**◈ 答言：「苦樂自他作者，世尊說言：『此是無記。』」**
　　尊者浮彌比丘回答：「對於主張苦樂是由自他共同創造的說法，世尊說這同樣屬於『無記』。」

**◈ 復問：「苦樂非自非他無因作耶？」**
　　他們又問：「苦（痛苦）與樂（快樂）既非自己創造，也非他人創造，而是無因地而然產生的嗎？」

**◈ 答言：「苦樂非自非他無因作者，世尊說言：『此是無記。』」**
　　尊者浮彌比丘回答：「對於主張苦樂是非自非他且無因而生的說法，世尊說這同樣屬於『無記』。」

**◈ 諸外道出家復問：「云何？尊者浮彌苦樂自作耶？說言無記。苦樂他作耶？說言無記。苦樂自他作耶？說言無記。苦樂非自非他無因作耶？說言無記。今沙門瞿曇說苦樂云何生？」**
　　那些外道出家的人再次追問：「怎麼回事？關於苦樂是自作、他作、自他共同作，或是非自非他無因作，您都回答是『無記』。那麼，沙門瞿曇（佛陀）究竟認為苦（痛苦）與樂（快樂）是如何產生的？」

**◈ 尊者浮彌答言：「諸外道出家！世尊說苦樂從緣起生。」**
　　尊者浮彌比丘回答說：「外道出家的人們！世尊說苦（痛苦）與樂（快樂）是從緣起（因緣而生，即依賴條件的組合而產生）而生的。」

**◈ 時，諸外道出家聞尊者浮彌所說，心不歡喜，呵責而去。**
　　那時，那些外道出家的人聽完尊者浮彌比丘的回答後，內心並不歡喜，於是呵斥著離開了。

**◈ 爾時，尊者舍利弗去尊者浮彌不遠，坐一樹下。**
　　那時，尊者舍利弗就在離尊者浮彌比丘不遠的地方，坐在一棵樹下。

**◈ 爾時，尊者浮彌知諸外道出家去已，往詣尊者舍利弗所，到已，與舍利弗面相慶慰。慶慰已，以彼諸外道出家所問事，具白尊者舍利弗：「我作此答，得不謗毀世尊！如說說、不如法說、不為是隨順法行法，得無為餘因法論者來難詰呵責不？」**
　　那時，尊者浮彌比丘知道那些外道出家的人已經離開，便前往尊者舍利弗那裡。到達後，與舍利弗互相問候。問候完畢後，他將那些外道出家者所詢問的事情詳細地對尊者舍利弗說道：「我作了這樣的回答，是否沒有誹謗世尊？我的說法是否符合佛陀的教導、是否如法地說明，且是否隨順正法而行？這樣回答是否能讓其他持有不同因果論的人無法對我進行詰問或呵責？」

**◈ 尊者舍利弗言：「尊者浮彌！汝之所說，實如佛說，不謗如來，如說說、如法說、法行法說，不為餘因論義者來難詰呵責。所以者何？世尊說苦樂從緣起生故。尊者浮彌！彼諸沙門、婆羅門所問苦樂自作者，彼亦從因起生；言不從緣起生者，無有是處。苦樂他作、自他作、非自非他無因作說者，彼亦從緣起生；若言不從緣生者，無有是處。尊者浮彌！彼沙門、婆羅門所說苦樂自作者，亦緣觸生；若言不從觸生者，無有是處。**
　　尊者舍利弗說：「尊者浮彌！你的說法確實與佛陀所說的一致，沒有誹謗如來，是如實的說法、如法的說法，且是依照正法而行的說法，其他持有不同因果論的人無法對此進行詰問或呵責。為什麼呢？因為世尊說苦（痛苦）與樂（快樂）是從緣起（因緣而生）而產生的。尊者浮彌！那些沙門或婆羅門所主張的『苦樂自作』，其實也是從因緣而起生的；如果說它不是從緣起而生，那是沒有這回事的。同樣地，主張『苦樂他作』、『自他共同作』或『非自非他無因作』的說法，也同樣是從緣起（因緣而生）而產生的；如果說這些說法不是從緣起而生，那是沒有這回事的。尊者浮彌！那些沙門或婆羅門所說的『苦樂自作』，也是緣於觸（感官與對象的接觸）而產生的；如果說它不是從觸（接觸）而生，那是沒有這回事的。」

**◈ 苦樂他作、自他作、非自非他無因作者，彼亦緣觸生，若言不緣觸生者，無有是處。」**
　　那些主張苦與樂是由他人創造、由自他共同創造，或是既非自作也非他作且沒有原因而產生的觀點，其實也都是依賴於觸（感官與對象的接觸）而產生的；如果有人說這不是依賴觸（接觸）而產生的，那是沒有這回事的。

**◈ 爾時，尊者阿難去舍利弗不遠，坐一樹下，聞尊者舍利弗與尊者浮彌所論說事。**
　　那時，尊者阿難在距離舍利弗不遠的地方，坐在一棵樹下，聽到了尊者舍利弗與尊者浮彌討論的事情。

**◈ 聞已，從座起，往詣佛所，稽首佛足，退住一面。**
　　聽完之後，他從座位上站起來，前往佛陀所在之處，向佛陀的足部頂禮，然後退到一邊站著。

**◈ 以尊者浮彌與尊者舍利弗共論說一一具白世尊。**
　　他將尊者浮彌與尊者舍利弗討論的內容，一一詳細地報告給世尊。

**◈ 佛告阿難：「善哉！善哉！阿難！尊者舍利弗有來問者，能隨時答。**
　　佛陀告訴阿難：「好極了！好極了！阿難！尊者舍利弗面對來詢問的人，能夠隨時給出正確的回答。

**◈ 善哉！舍利弗！有應時智故，有來問者，能隨時答。**
　　好極了，舍利弗！因為擁有應時智（在適當時間給予適當教導的智慧），所以面對來詢問的人，能隨時回答。

**◈ 若我聲聞，有隨時問者，應隨時答，如舍利弗所說。**
　　如果我的聲聞弟子中，有能隨時提問的人，就應該像舍利弗所說的那樣隨時回答。」

**◈ 「阿難！我昔時住王舍城山中仙人住處，有諸外道出家以如是義、如是句、如是味來問於我，**
　　「阿難！我以前住在王舍城的山中仙人住處時，有許多出家的外道以同樣的義理、同樣的句子、同樣的觀點來詢問我，

**◈ 我為斯等以如是義、如是句、如是味而為記說，如尊者舍利弗所說。**
　　而我對他們所給出的回答，其義理、句子與觀點，就如同尊者舍利弗所說的一樣。」

**◈ 「阿難！若諸沙門、婆羅門苦樂自作，我即往彼問言：『汝實作是說苦樂自作耶？』**
　　「阿難！如果那些沙門或婆羅門主張苦與樂是自作的，我就會走到他們面前問道：『你確實主張苦與樂是自作的嗎？』

**◈ 彼答我言：『如是。』**
　　對方回答我：『是的。』

**◈ 我即問言：『汝能堅執持此義，言是真實，餘則愚者，我所不許。所以者何？我說苦樂所起異於此。』**
　　我就會問：『你能堅定地持有這個觀點，認為這是真實的，而其他人的看法都是愚笨的且我不認同嗎？為什麼呢？因為我所說的苦與樂的產生方式與此不同。』

**◈ 彼若問我：『云何瞿曇所說，苦樂所起異者？』**
　　如果對方問我：『瞿曇（佛陀）所說的苦樂產生方式有何不同？』

**◈ 我當答言：『從其緣起而生苦樂。**
　　我會回答說：『苦與樂是從其緣起（因緣而生）而產生的。

**◈ 如是說苦他作、自他作、非自非他無因作者，我亦往彼所說如上。』」**
　　對於那些主張苦樂是由他人創造、自他共同創造，或是既非自作也非他作且無因而產生的觀點，我也會像上面這樣對他們說明。」

**◈ 阿難白佛：「如世尊所說義，我已解知，有生故有老死，非緣餘；**
　　阿難對佛陀說：「如世尊所說的義理，我已經理解並知道了：因為有生（出生），所以才有老死（衰老與死亡），而不是依賴其他原因；

**◈ 有生故有老死，乃至無明故有行，非緣餘；**
　　因為有生（出生），所以才有老死（衰老與死亡），一直到因為無明（對真理的無知）所以有行（心的造作），而不是依賴其他原因；

**◈ 有無明故有行，無明滅則行滅，乃至生滅則老、病、死、憂、悲、惱、苦滅，如是純大苦聚滅。」**
　　因為有無明（對真理的無知）所以有行（心的造作），如果無明（對真理的無知）滅除，則行（心的造作）也隨之滅除，一直到生（出生）滅除，則老、病、死、憂、悲、惱、苦也隨之滅除，如此便能讓這純粹的巨大苦聚滅除。」

**◈ 佛說此經已，尊者阿難聞佛所說，歡喜隨喜，作禮而去。**
　　佛陀講完這部經後，尊者阿難聽聞佛陀所說的教法，心生法喜並依照教導去實踐，向佛陀行禮後才離開。

### [詞語註解]
1.  **無記 (Avyākata)**：指佛陀對於某些問題不予肯定或否定的回答。這通常發生在問題本身基於錯誤的假設（例如假設有一個永恆不變的「我」），或者該問題對修行解脫沒有實質幫助時。
2.  **緣起 (Paṭicca-samuppāda)**：指一切現象（法）都沒有獨立的永恆本質或偶然產生，而是依賴於特定的條件（因緣）組合而生起。當條件具足時，現象產生；條件消失時，現象滅除。
3.  **觸 (Phassa)**：指感官（眼、耳、鼻、舌、身、意）與其對應的對象（色、聲、香、味、觸、法）以及意識三者相遇的過程。例如：眼（感官）＋ 顏色（對象）＋ 眼識（意識）＝ 觸。這是產生「受（感受）」的直接前提。
4.  **應時智 (Sāmāyika-ñāṇa)**：佛陀或大弟子在面對不同根基、不同問題的對象時，能根據對方當下的狀態，選擇最合適的時機與方法來開示的智慧。
5.  **十二因緣 (Twelve Links of Dependent Origination)**：描述生命如何從無明（對真理的無知）開始，經過行（心的造作）、識（意識）、名色、六入、觸、受（感受）、愛、取、有、生，最終導致老死與苦集之循環過程。

### [現代實作指南]
本經文深入探討了苦樂的本質與來源，從根本上糾正了將情緒歸咎於單一主體或無因的常見誤解。在現代生活中，我們可以將其轉化為以下系統化的實踐步驟：

1.  **轉變歸因視角：從「誰」到「如何」**
    *   **停止「歸因陷阱」**：當我們感到痛苦或快樂時，直覺常將其歸類為「自作」（自我批判或傲慢）、「他作」（受害者心態與怨恨）、「自他共同作」（停留在對主體的執著）或「無因作」（對無常的恐懼或迷茫）。這些都是基於錯誤假設的「無記」問題。
    *   **實作技巧**：當強烈情緒升起時，提醒自己：「這不是誰創造的，而是條件組合的結果。」將注意力從「誰的責任」轉向「如何產生」的機制。

2.  **追溯「觸」的條件：情緒的源頭觀察**
    *   既然苦樂緣於「觸（感官與對象的接觸）」，我們可以練習在情緒爆發前，回溯觸發點。
    *   **觀察步驟**：
        1.  **覺察受（感受）**：我現在感覺到「苦（不舒服）」或「樂（舒服）」。
        2.  **回溯觸（接觸）**：剛才發生了什麼接觸？（例如：聽到主管的一句批評 $\rightarrow$ 耳根＋聲音＋意識 $\rightarrow$ 觸）。
        3.  **分析緣（因緣）**：除了直接的接觸，還有什麼條件促成了這個感受？（例如：我昨晚沒睡好、我對主管有先入為主的恐懼、我對這項工作有過高的期待）。
    *   **轉念效果**：將「他讓我痛苦」轉化為「在疲憊、恐懼和期待的條件下，聽到這句話產生了痛苦的感受」。這樣能將注意力從「對人的衝突」移向「對條件的觀察」，從而降低情緒強度，並提供改變條件的可能性。

3.  **接納「無記」的智慧：放下無益的追問**
    *   學習接受某些問題（如：我命中注定要受苦嗎？）在邏輯上是無解的，因為它們基於錯誤的假設，對解脫沒有實質幫助。
    *   **實作方法**：不要在無意義的爭論或鑽牛角尖中耗費心神，而應將注意力轉向「現在如何改變條件」以停止苦的產生，或「如何培養正見」以超越苦樂。

4.  **實踐十二因緣滅路：斷除苦聚的根本方法**
    *   阿難所理解的十二因緣滅路，提供了一個精準的「除根」指南，從根本上斷除苦的循環。
    *   **操作步驟**：
        1.  **覺察無明（對真理的無知）**：意識到自己對某件事的執著、貪愛或厭惡，是因為對其本質（無常、無我、苦）缺乏正確認知。
        2.  **停止造作（行）**：當不舒服的感受（受）產生時，不要立刻用憤怒、悲傷、執取或逃避去回應（心的造作），而是僅僅觀察它，不加以判斷或強化。
        3.  **切斷貪愛與執取**：不希望痛苦消失（貪愛），也不將痛苦視為「我的」痛苦（執取），而是視為無常、無我的現象。
        4.  **結果**：當不再對感受進行心理造作，並從無明中解脫，後續的壓力、精神折磨（老死、憂悲惱苦）以及輪迴的動力便失去了生長的土壤，最終能讓純粹的巨大苦聚徹底滅除。
