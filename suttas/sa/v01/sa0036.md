# 雜阿含經 第 36 經

### [原文]
如是我聞：

一時，佛住摩偷羅國跋提河側傘蓋菴羅樹園。

爾時，世尊告諸比丘：「住於自洲，住於自依；住於法洲，住於法依；不異洲不異依。比丘！當正觀察，住自洲自依，法洲法依，不異洲不異依。何因生憂悲惱苦？云何有四？何故何繫著？云何自觀察未生憂悲惱苦而生，已生憂悲惱苦生長增廣？」

諸比丘白佛：「世尊法根、法眼、法依，唯願為說！諸比丘聞已，當如說奉行。」

佛告比丘：「諦聽，善思，當為汝說。比丘！有色、因色、繫著色，自觀察未生憂悲惱苦而生，已生而復增長廣大；受、想、行、識亦復如是。比丘！頗有色常、恒、不變易、正住耶？」

答言：「不也，世尊！」

佛告比丘：「善哉！善哉！比丘！色是無常。若善男子知色是無常、苦、變易，離欲、滅、寂靜、沒，從本以來，一切色無常、苦、變易法知已，若色因緣生憂悲惱苦斷，彼斷已無所著，不著故安隱樂住，安隱樂住已，名為涅槃。受、想、行、識亦復如是。」

佛說此經時，十六比丘不生諸漏，心得解脫。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

　竹園．毘舍離　　清淨．正觀察 　無常．苦．非我　　五．三．與十六

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住摩偷羅國跋提河側傘蓋菴羅樹園。**
　　在某一個時間，佛陀住在摩偷羅國的跋提河邊，一座像傘蓋一樣茂密的菴羅樹園中。

**◈ 爾時，世尊告諸比丘：「住於自洲，住於自依；住於法洲，住於法依；不異洲不異依。**
　　那時，世尊告訴在場的眾位比丘說：「要住在自己的心靈之島，依託於自己的覺醒；要住在正法（真理）的島嶼，依託於正法（真理）的教導；這兩者並不分離，也不相異。

**◈ 比丘！當正觀察，住自洲自依，法洲法依，不異洲不異依。**
　　比丘！你們應當正觀察（正確地觀察），意識到住在自我的覺醒與正法（真理）的依託之中，這兩者是同一回事。

**◈ 何因生憂悲惱苦？云何有四？何故何繫著？云何自觀察未生憂悲惱苦而生，已生憂悲惱苦生長增廣？」**
　　是什麼原因導致憂慮、悲傷、惱怒與苦（不滿足與壓力）的產生？這四種痛苦是如何運作的？是因為什麼原因而產生繫縛與執著？應當如何觀察這些憂悲惱苦在尚未產生時是如何產生的，以及在產生之後又是如何生長並擴大的？」

**◈ 諸比丘白佛：「世尊法根、法依，唯願為說！諸比丘聞已，當如說奉行。」**
　　眾比丘對佛陀恭敬地說道：「世尊，請您為我們開示關於真理的根源、真理的洞察力以及真理的依託！我們聽聞之後，定會虔誠地接受並依照教導去實踐。」

**◈ 佛告比丘：「諦聽，善思，當為汝說。**
　　佛陀告訴比丘：「你們仔細聽，好好思考，我現在為你們解說。

**◈ 比丘！有色、因色、繫著色，自觀察未生憂悲惱苦而生，已生而復增長廣大；受、想、行、識亦復如是。**
　　比丘！關於色（物質身體）、因為色（物質身體）而產生的因緣、以及對色（物質身體）的執著繫縛，請你們觀察憂悲惱苦是如何從未生到產生的，以及產生後如何增長廣大的；受（感受）、想（想像與認知）、行（心的造作）、識（意識）這四種要素也是同樣的情況。

**◈ 比丘！頗有色常、恒、不變易、正住耶？」**
　　比丘！請問色（物質身體）是否存在著恆常、永恆、不變易且能永遠固定不變的情況嗎？」

**◈ 答言：「不也，世尊！」**
　　比丘們回答：「不，世尊！」

**◈ 佛告比丘：「善哉！善哉！比丘！色是無常。**
　　佛陀告訴比丘：「好極了！做得好！比丘！色（物質身體）確實是無常（變遷不恆久）的。

**◈ 若善男子知色是無常、苦、變易，離欲、滅、寂靜、沒，從本以來，一切色無常、苦、變易法知已，**
　　如果修行者知道色（物質身體）是無常（變遷不恆久）、是苦（不滿足與壓力）、且是變易（不斷改變）的，進而遠離貪欲、使苦因滅盡、達到寂靜與消融的狀態；從根本上了解一切色（物質身體）都屬於無常、苦、變易的法（現象）之後，

**◈ 若色因緣生憂悲惱苦斷，彼斷已無所著，不著故安隱樂住，安隱樂住已，名為涅槃。受、想、行、識亦復如是。」**
　　如果能斷除由色（物質身體）之因緣所產生的憂悲惱苦，一旦斷除就再也沒有執著，因為不再執著，所以能安穩快樂地生活，這種安穩快樂的狀態，就稱為涅槃（超越生死、絕對寧靜的解脫境界）。受（感受）、想（想像與認知）、行（心的造作）、識（意識）也是同樣的道理。」

**◈ 佛說此經時，十六比丘不生諸漏，心得解脫。**
　　佛陀講這部經的時候，有十六位比丘不再產生諸漏（心理的染污與煩惱），獲得了解脫。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **法洲 (Dhamma-dvīpa)**：將「正法」比喻為大海中的島嶼。在充滿苦海（輪迴）的世間，唯有依託真理才能獲得安全與安定。
2. **五蘊 (Pañca-khandha)**：構成個體身心的五種要素，包括色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）。
3. **漏 (Āsava)**：原意為「滲漏」，指潛在於心底、不斷流出的煩惱（如貪欲、存在欲、無明），是導致輪迴的根本原因。
4. **無常 (Anicca)**：指一切條件合成的現象（行法）都在不斷地生起與滅盡，沒有永恆不變的實體。
5. **涅槃 (Nibbāna)**：指貪、嗔、痴三毒完全熄滅後的狀態，是超越了生死輪迴、絕對寧靜的解脫境界。

### [現代實作指南]
本經文提供了一套從「觀察身體」到「心靈解脫」的實作路徑，可轉化為以下生活實踐：

1. **建立「心靈之島」 (自洲/法洲)**：
   ➔ **實作**：在面對外界混亂或情緒衝擊時，練習將注意力從外部事件移回內在的覺知。告訴自己：「我現在回到我的覺知之島」，不讓外界的風暴將自己捲走。

2. **追蹤「苦的生長過程」 (觀察憂悲惱苦)**：
   ➔ **實作**：當感到煩惱（如憤怒或焦慮）時，不要立即反應，而是像科學家一樣觀察：
     - **未生而生**：觀察這個情緒是從哪個「觸點」開始的？（例如：聽到一句批評 → 產生不快感）。
     - **生長增廣**：觀察情緒如何擴大？（例如：不快感 → 想起過去的委屈 → 變成強烈的憤怒）。

3. **拆解「執著的對象」 (五蘊分析)**：
   ➔ **實作**：將痛苦對象化。當你對身體衰老或疾病感到痛苦時，提醒自己：「這只是『色（物質身體）』在運作其無常的本質」，而非「我」在受苦。將「我的痛苦」轉化為「觀察到一種痛苦的感受（受）正在發生」。

4. **體認「變易」以達成「離欲」**：
   ➔ **實作**：每天花五分鐘觀察身體的微小變化（如呼吸的起伏、皮膚的觸感），體會沒有任何一秒是完全靜止不變的。當意識到「一切都在變」時，對「永恆擁有」的執著會自然降低，從而獲得內心的安穩（安隱樂住）。

### [段落大意]
本經強調修行者應依託自覺與正法，如同住在安全的島嶼中。佛陀指導比丘觀察憂悲惱苦如何從五蘊（色、受、想、行、識）中產生並擴大。透過體悟五蘊的無常與變易本質，斷除對身心的執著，最終能達到不再繫縛、安穩快樂的涅槃境界。
