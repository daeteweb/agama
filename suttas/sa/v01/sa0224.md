# 雜阿含經 第 224 經

### [原文]
（二二四）
如是我聞：
一時，佛住舍衛國祇樹給孤獨園。
爾時，世尊告諸比丘：「一切欲法應當斷。云何一切欲法應當斷？謂眼是一切欲法應當斷，若色、眼識、眼觸、眼觸因緣生受，內覺若苦、若樂、不苦不樂，彼一切欲法應當斷。耳、鼻、舌、身、意亦復如是。」
佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽聞的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　有一次，佛住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「一切欲法應當斷。**
　　那時候，世尊告訴各位比丘：「所有引起貪愛與執著的欲望現象（欲法）都應該被斷除。

**◈ 云何一切欲法應當斷？**
　　那麼，該如何斷除這一切欲望現象呢？

**◈ 謂眼是一切欲法應當斷，若色、眼識、眼觸、眼觸因緣生受，內覺若苦、若樂、不苦不樂，彼一切欲法應當斷。**
　　指的是從視覺感官開始，所有欲望現象都應被斷除；例如透過物質對象（色）、視覺意識（眼識）與感官接觸（眼觸），並由這些條件所引發出的感受（受）——無論內在覺知到的是痛苦、快樂，還是不苦不樂的中性感受——這些所有的欲望現象都應該被斷除。

**◈ 耳、鼻、舌、身、意亦復如是。」**
　　耳朵、鼻子、舌頭、身體以及意識感官，也同樣如此。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀說完這部經典後，各位比丘聽聞了佛陀的教導，感到非常歡喜並遵從修行。

### [詞語註解]
1. **欲法 (Desire-phenomena)**：指一切引發貪愛、渴求與執著的心理與感官現象。
2. **色 (Rūpa)**：在此脈絡下指感官所對應的物質對象（如視覺對應的顏色、形狀）。
3. **眼識 (Eye-consciousness)**：指透過眼睛接觸對象時，所產生的視覺意識功能。
4. **觸 (Contact/Paṭisandhi)**：指感官、對象與意識三者結合而產生的接觸作用。
5. **受 (Feeling/Vedanā)**：指接觸後在心中升起的感受，分為苦（痛苦）、樂（快樂）、不苦不樂（中性）。

### [現代實作指南]
本經的核心在於「截斷感官刺激與情緒反應之間的連鎖反應」。在日常生活中，我們可以透過以下步驟進行練習：

1. **覺察接觸點**：當你看到一件漂亮的衣服、聽到一段動聽的音樂或吃到美食時，立刻意識到「這是一個接觸（觸）發生的時刻」，而不是直接陷入對該事物的渴望中。
2. **標籤化感受**：在感覺升起時，迅速在心中為其貼上標籤。例如：「這是快樂的感受」、「這是痛苦的感受」或「這是平淡的感受」。透過這種客觀的觀察（標籤化），你可以將自己從「我正在享受/受苦」的主觀狀態中抽離，轉變為「這僅是一個正在產生的感受（受）」。
3. **阻斷執取**：當你發現自己因為某種感受（特別是快樂或不苦不樂的平淡感）而想要去抓取、留住它時，提醒自己這就是經文所說的「欲法」。練習觀察這種「想留住」的衝動，但不隨之起舞。
4. **全感官監控**：不要只針對視覺，也要將此練習擴展到聽覺（聽到批評或讚美）、嗅覺、味覺、觸覺以及內在的思想（意）上，建立一個全面的感官防禦機制。

### [段落大意]
佛陀教導比丘，應當斷除所有引發貪愛的欲望現象。其原理在於：當感官（眼耳鼻舌身意）與物質對象接觸並產生意識時，會隨之引發各種苦、樂或中性的感受；這些由感官接觸所產生的連鎖反應與感受，皆是應當斷除的欲望來源。
