# 雜阿含經 第 46 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有五受陰。云何為五？色、受、想、行、識受陰。若沙門、婆羅門以宿命智自識種種宿命，已識、當識、今識，皆於此五受陰；已識、當識、今識，我過去所經。如是色、如是受、如是想、如是行、如是識。若可閡可分，是名色受陰。指所閡，若手、若石、若杖、若刀、若冷、若暖、若渴、若飢、若蚊、虻、諸毒虫、風、雨觸，是名觸閡，是故閡是色受陰。復以此色受陰無常、苦、變易。諸覺相是受受陰，何所覺？覺苦、覺樂、覺不苦不樂，是故名覺相是受受陰。復以此受受陰是無常、苦、變易。諸想是想受陰，何所想？少想、多想、無量想、都無所有作無所有想，是故名想受陰。復以此想受陰是無常、苦、變易法。為作相是行受陰，何所為作？於色為作，於受、想、行、識為作，是故為作相是行受陰。復以此行受陰是無常、苦、變易法。別知相是識受陰，何所識？識色，識聲、香、味、觸、法，是故名識受陰。復以此識受陰是無常、苦、變易法。

「諸比丘！彼多聞聖弟子於此色受陰作如是學：『我今為現在色所食，過去世已曾為彼色所食，如今現在。』復作是念：『我今為現在色所食，我若復樂著未來色者，當復為彼色所食，如今現在。』作如是知已，不顧過去色，不樂著未來色，於現在色生厭、離欲、滅患、向滅。多聞聖弟子於此受、想、行、識受陰學：『我今現在為現在識所食，於過去世已曾為識所食，如今現在。我今已為現在識所食，若復樂著未來識者，亦當復為彼識所食，如今現在。』如是知已，不顧過去識，不樂未來識，於現在識生厭、離欲、滅患、向滅，滅而不增，退而不進，滅而不起，捨而不取。

「於何滅而不增？色滅而不增，受、想、行、識滅而不增。於何退而不進？色退而不進，受、想、行、識退而不進。於何滅而不起？色滅而不起，受、想、行、識滅而不起。於何捨而不取？色捨而不取，受、想、行、識捨而不取。

「滅而不增，寂滅而住；退而不進，寂退而住；滅而不起，寂滅而住；捨而不取，不生繫著；不繫著已，自覺涅槃：『我生已盡，梵行已立，所作已作，自知不受後有。』」

佛說此經時，眾多比丘不起諸漏，心得解脫。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「有五受陰。云何為五？色、受、想、行、識受陰。**
　　那時，世尊告訴在場的眾位比丘說：「有五種構成經驗的蘊（受陰）。什麼是這五種？即是色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）這五種蘊。」

**◈ 若沙門、婆羅門以宿命智自識種種宿命，已識、當識、今識，皆於此五受陰；已識、當識、今識，我過去所經。**
　　如果沙門或婆羅門利用能想起前世的宿命智，去認知各種過去的生命，無論是已經認知的、將要認知的、或是現在認知的，全部都集中在這五種蘊上；無論是過去、未來或現在所經歷的，都離不開這五種蘊。

**◈ 如是色、如是受、如是想、如是行、如是識。**
　　（過去所經歷的）正是這樣的色（物質身體）、這樣的受（感受）、這樣的想（想像與認知）、這樣的行（心的造作）以及這樣的識（意識）。

**◈ 若可閡可分，是名色受陰。**
　　凡是能夠被觸碰、能被分割的，就稱為色（物質身體）蘊。

**◈ 指所閡，若手、若石、若杖、若刀、若冷、若暖、若渴、若飢、若蚊、虻、諸毒虫、風、雨觸，是名觸閡，是故閡是色受陰。**
　　舉例來說，被觸碰到的對象，像是手、石頭、手杖、刀刃，或是感受到寒冷、溫暖、口渴、飢餓，以及被蚊子、虻蠅、各種毒蟲叮咬，或是被風、雨觸碰，這些都稱為觸碰，因此這種能被觸碰的性質就是色（物質身體）蘊。

**◈ 復以此色受陰無常、苦、變易。**
　　而這個色（物質身體）蘊具有無常（遷變）、苦（不滿足）與變易（不穩定）的特性。

**◈ 諸覺相是受受陰，何所覺？覺苦、覺樂、覺不苦不樂，是故名覺相是受受陰。**
　　所有覺知感受的相狀就是受（感受）蘊。覺知什麼呢？覺知到苦、覺知到樂、或是覺知到不苦不樂，所以稱這種覺知相狀為受（感受）蘊。

**◈ 復以此受受陰是無常、苦、變易。**
　　而這個受（感受）蘊同樣是無常（遷變）、苦（不滿足）與變易（不穩定）的。

**◈ 諸想是想受陰，何所想？少想、多想、無量想、都無所有作無所有想，是故名想受陰。**
　　所有的認知與想像就是想（想像與認知）蘊。想像什麼呢？有少量的想像、大量的想像、無量的想像，甚至是將「完全沒有」認知為「完全沒有」的想像，所以稱之為想（想像與認知）蘊。

**◈ 復以此想受陰是無常、苦、變易法。**
　　而這個想（想像與認知）蘊也是無常（遷變）、苦（不滿足）與變易（不穩定）的法。

**◈ 為作相是行受陰，何所為作？於色為作，於受、想、行、識為作，是故為作相是行受陰。**
　　具有造作相狀的就是行（心的造作）蘊。造作什麼呢？針對色（物質身體）產生造作，針對受（感受）、想（想像與認知）、行（心的造作）、識（意識）產生造作，所以稱這種造作相狀為行（心的造作）蘊。

**◈ 復以此行受陰是無常、苦、變易法。**
　　而這個行（心的造作）蘊同樣是無常（遷變）、苦（不滿足）與變易（不穩定）的法。

**◈ 別知相是識受陰，何所識？識色，識聲、香、味、觸、法，是故名識受陰。**
　　具有分別認知相狀的就是識（意識）蘊。認知什麼呢？認知色（物質身體）、認知聲音、氣味、味道、觸感以及心法，所以稱之為識（意識）蘊。

**◈ 復以此識受陰是無常、苦、變易法。**
　　而這個識（意識）蘊也是無常（遷變）、苦（不滿足）與變易（不穩定）的法。

**◈ 「諸比丘！彼多聞聖弟子於此色受陰作如是學：『我今為現在色所食，過去世已曾為彼色所食，如今現在。』**
　　「諸比丘！那些博學的聖弟子針對這個色（物質身體）蘊這樣學習：『我現在正被現在的色（物質身體）所驅使吞噬，在過去世我也曾被那樣的色（物質身體）所驅使吞噬，直到現在依然如此。』」

**◈ 復作是念：『我今為現在色所食，我若復樂著未來色者，當復為彼色所食，如今現在。』**
　　接著又這樣思考：『我現在正被現在的色（物質身體）所驅使吞噬，如果我對未來的色（物質身體）產生貪愛執著，將來必然會再次被那樣的色（物質身體）所驅使吞噬，就像現在一樣。』

**◈ 作如是知已，不顧過去色，不樂著未來色，於現在色生厭、離欲、滅患、向滅。**
　　有了這樣的認知後，不再眷戀過去的色（物質身體），不再貪著未來的色（物質身體），對現在的色（物質身體）產生厭離心、遠離欲望、熄滅苦患，並趨向於滅（苦止）。

**◈ 多聞聖弟子於此受、想、行、識受陰學：『我今現在為現在識所食，於過去世已曾為識所食，如今現在。我今已為現在識所食，若復樂著未來識者，亦當復為彼識所食，如今現在。』**
　　博學的聖弟子針對受（感受）、想（想像與認知）、行（心的造作）、識（意識）蘊也這樣學習：『我現在正被現在的識（意識）所驅使吞噬，在過去世我也曾被識（意識）所驅使吞噬，直到現在依然如此。我現在已被現在的識（意識）所驅使吞噬，如果我對未來的識（意識）產生貪愛執著，將來必然會再次被那樣的識（意識）所驅使吞噬，就像現在一樣。』

**◈ 如是知已，不顧過去識，不樂未來識，於現在識生厭、離欲、滅患、向滅，滅而不增，退而不進，滅而不起，捨而不取。**
　　有了這樣的認知後，不再眷戀過去的識（意識），不再貪著未來的識（意識），對現在的識（意識）產生厭離心、遠離欲望、熄滅苦患，並趨向於滅（苦止）；使煩惱滅而不增加，退而不再前進，滅而不再興起，捨棄而不再執取。

**◈ 「於何滅而不增？色滅而不增，受、想、行、識滅而不增。**
　　「什麼是滅而不增？是指色（物質身體）滅而不增加，受（感受）、想（想像與認知）、行（心的造作）、識（意識）滅而不增加。」

**◈ 於何退而不進？色退而不進，受、想、行、識退而不進。**
　　什麼是退而不進？是指色（物質身體）退而不再前進，受（感受）、想（想像與認知）、行（心的造作）、識（意識）退而不再前進。

**◈ 於何滅而不起？色滅而不起，受、想、行、識滅而不起。**
　　什麼是滅而不起？是指色（物質身體）滅而不再興起，受（感受）、想（想像與認知）、行（心的造作）、識（意識）滅而不再興起。

**◈ 於何捨而不取？色捨而不取，受、想、行、識捨而不取。**
　　什麼是捨而不取？是指色（物質身體）捨棄而不再執取，受（感受）、想（想像與認知）、行（心的造作）、識（意識）捨棄而不再執取。

**◈ 「滅而不增，寂滅而住；退而不進，寂退而住；滅而不起，寂滅而住；捨而不取，不生繫著；**
　　「滅而不增加，便在寂滅中安住；退而不前進，便在寂退中安住；滅而不再興起，便在寂滅中安住；捨而不執取，便不再產生繫著（執著）。」

**◈ 不繫著已，自覺涅槃：『我生已盡，梵行已立，所作已作，自知不受後有。』」**
　　不再繫著之後，便自覺地進入涅槃（超越生死、絕對寧靜的解脫境界），體悟到：『我的出生已盡，聖潔的修行已建立，應做的事已完成，自知不再受後世的輪迴。』」

**◈ 佛說此經時，眾多比丘不起諸漏，心得解脫。**
　　佛陀在說這部經時，許多比丘不再產生煩惱（漏），獲得了解脫。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **五受陰 (Pañca-khandhā)**：此處「受陰」即指「蘊」。指構成生命個體的五種要素：色（物質）、受（感受）、想（認知）、行（意志造作）、識（意識）。
2. **宿命智 (Pubbenivāsānussati-ñāṇa)**：一種修行達到的能力，能回憶起過去世的所有生命經歷。
3. **無常 (Anicca)**：指一切有為法（由因緣而生）都在不斷地變遷，沒有永恆不變的實體。
4. **變易**：與無常相近，強調現象在時間流逝中不斷地改變、不穩定。
5. **所食**：比喻被五蘊的運作所驅使、吞噬或控制。指人被物質欲望、感受、認知、造作與意識所牽引，導致在輪迴中受苦。
6. **厭 (Nibbidā)**：並非世俗的討厭，而是一種基於智慧的「厭離」，意識到五蘊的無常與苦後，自然產生的不再執著的心態。
7. **漏 (Āsava)**：指內心的染污、煩惱或精神上的「滲漏」，是導致輪迴的主要原因。
8. **梵行 (Brahmacariya)**：指聖潔的修行生活，旨在消除貪嗔痴，達成解脫。

### [現代實作指南]
本經文提供了一套將「自我」拆解為五個功能模組，並觀察其運作模式的實踐方法：

1. **功能拆解法 (Deconstruction)**：
   - 當你感到壓力或痛苦時，不要說「我很痛苦」，而要將其拆解：
     - **色**：身體是否有緊繃感？（如肩膀僵硬、心跳加快）
     - **受**：現在的感受是苦、樂，還是不苦不樂？
     - **想**：大腦正在如何定義這個情況？（例如：認定這是「失敗」或「被背叛」）
     - **行**：心中是否產生了反擊、逃避或憤怒的衝動？
     - **識**：意識如何捕捉這些資訊並將其整合為一個「我」的經驗？

2. **「被吞噬」的覺察 (Awareness of Consumption)**：
   - 觀察自己如何被這些模組「驅使」。例如：因為「受（不舒服）」 → 導致「行（想抱怨）」 → 導致「識（認定對方很壞）」。
   - 意識到：「我現在正被這個『憤怒的造作（行）』所驅使」，而非「我在憤怒」。

3. **時間軸的斷捨離 (Temporal Detachment)**：
   - **過去**：意識到過去的痛苦模式（如：我以前總是這樣反應）已成過去，不必再回頭眷戀或悔恨。
   - **未來**：意識到如果現在繼續執著於某種感受，未來將會重複同樣的痛苦循環。
   - **現在**：對當下的自動化反應產生「厭離」，選擇不跟隨該反應，從而切斷輪迴的鏈條。

4. **實踐目標**：
   - 練習「捨而不取」：看到情緒升起 → 認出它是「受」或「行」 → 觀察它 → 讓它自然消散，而不將其抓取為「我的」情緒。

### [段落大意]
本經詳細分析了構成身心的五蘊（色、受、想、行、識），指出每一蘊都具有無常、苦與變易的特性。佛陀教導聖弟子應觀察自己如何被五蘊所驅使（所食），透過對過去、現在、未來五蘊運作的洞察，產生厭離心，進而停止對五蘊的執著。最終透過「滅而不增、退而不進、滅而不起、捨而不取」的過程，消除所有繫著，達成寂滅的涅槃境界，終結輪迴之苦。
