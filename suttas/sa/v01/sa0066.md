# 雜阿含經 第 66 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「常當修習方便禪思，內寂其心。所以者何？修習方便禪思，內寂其心已，如實觀察。云何如實觀察？如實觀察此色、此色集、此色滅，此受、想、行、識，此識集、此識滅。

「云何色集？云何受、想、行、識集？比丘！愚癡無聞凡夫不如實觀察色集、色味、色患、色離故，樂彼色，讚歎愛著，於未來世色復生；受、想、行、識……」亦如是廣說。「彼色生，受、想、行、識生已，不解脫於色，不解脫於受、想、行、識。我說彼不解脫生、老、病、死、憂、悲、惱、苦，純大苦聚，是名色集，受、想、行、識集。

「云何色滅，受、想、行、識滅？多聞聖弟子如實觀察色集、色滅、色味、色患、色離，如實知。如實知故，不樂於色，不讚歎色，不樂著色，亦不生未來色。受、想、行、識……」亦如是廣說。「色不生，受、想、行、識不生故，於色得解脫，於受、想、行、識得解脫。我說彼解脫生、老、病、死、憂、悲、惱、苦聚，是名色滅，受、想、行、識滅。

「是故，比丘！常當修習方便禪思，內寂其心，精勤方便，如實觀察。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如觀察，如是乃至作證十二經，亦應廣說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「常當修習方便禪思，內寂其心。**
　　那時，世尊告訴在場的眾位比丘說：「你們應該經常修習方便的禪定思惟，使內心達到寂靜狀態。

**◈ 所以者何？修習方便禪思，內寂其心已，如實觀察。**
　　為什麼呢？因為在修習方便的禪定思惟並使內心寂靜之後，才能如實地觀察真相。

**◈ 云何如實觀察？如實觀察此色、此色集、此色滅，此受、想、行、識，此識集、此識滅。**
　　如何才算是如實觀察？就是如實地觀察這個色（物質身體）、這個色（物質身體）的集（苦因/產生）、這個色（物質身體）的滅（苦止/消滅）；以及觀察這個受（感受）、想（想像與認知）、行（心的造作）、識（意識），以及這個識（意識）的集（苦因/產生）與這個識（意識）的滅（苦止/消滅）。

**◈ 云何色集？云何受、想、行、識集？**
　　什麼是色（物質身體）的集（苦因/產生）？什麼是受（感受）、想（想像與認知）、行（心的造作）、識（意識）的集（苦因/產生）？

**◈ 比丘！愚癡無聞凡夫不如實觀察色集、色味、色患、色離故，樂彼色，讚歎愛著，於未來世色復生；受、想、行、識……」亦如是廣說。**
　　比丘們！那些愚癡且缺乏聞法機會的凡夫，因為不能如實地觀察色（物質身體）的集（苦因/產生）、色（物質身體）的滋味、色（物質身體）的過患以及如何脫離色（物質身體），所以對物質身體感到快樂，讚歎並產生貪愛執著，導致在未來世中再次生出色（物質身體）；至於受（感受）、想（想像與認知）、行（心的造作）、識（意識）……也是同樣地詳細說明。

**◈ 「彼色生，受、想、行、識生已，不解脫於色，不解脫於受、想、行、識。**
　　當物質身體產生，且受（感受）、想（想像與認知）、行（心的造作）、識（意識）隨之產生後，若不能從色（物質身體）中解脫，也不能從受、想、行、識中解脫。

**◈ 我說彼不解脫生、老、病、死、憂、悲、惱、苦，純大苦聚，是名色集，受、想、行、識集。**
　　我說這種不解脫的狀態會帶來出生、衰老、疾病、死亡，以及憂愁、悲傷、煩惱與痛苦，這完全是大苦的聚集，這就叫做色（物質身體）的集（苦因/產生），以及受、想、行、識的集（苦因/產生）。

**◈ 「云何色滅，受、想、行、識滅？**
　　那麼，什麼是色（物質身體）的滅（苦止/消滅），以及受（感受）、想（想像與認知）、行（心的造作）、識（意識）的滅（苦止/消滅）？

**◈ 多聞聖弟子如實觀察色集、色滅、色味、色患、色離，如實知。**
　　聽聞廣博的聖弟子如實地觀察色（物質身體）的集（苦因/產生）、色（物質身體）的滅（苦止/消滅）、色（物質身體）的滋味、色（物質身體）的過患以及如何脫離色（物質身體），並對此有如實的認知。

**◈ 如實知故，不樂於色，不讚歎色，不樂著色，亦不生未來色。受、想、行、識……」亦如是廣說。**
　　因為有了如實的認知，所以不再對色（物質身體）感到快樂，不再讚歎色（物質身體），不再執著於色（物質身體），因此在未來世也不再生出色（物質身體）；至於受（感受）、想（想像與認知）、行（心的造作）、識（意識）……也是同樣地詳細說明。

**◈ 「色不生，受、想、行、識不生故，於色得解脫，於受、想、行、識得解脫。**
　　因為色（物質身體）不再產生，受（感受）、想（想像與認知）、行（心的造作）、識（意識）也不再產生，所以對色（物質身體）獲得了解脫，對受、想、行、識也獲得了解脫。

**◈ 我說彼解脫生、老、病、死、憂、悲、惱、苦聚，是名色滅，受、想、行、識滅。**
　　我說這種解脫狀態讓一個人遠離了出生、衰老、疾病、死亡，以及憂愁、悲傷、煩惱與痛苦的聚集，這就叫做色（物質身體）的滅（苦止/消滅），以及受、想、行、識的滅（苦止/消滅）。

**◈ 「是故，比丘！常當修習方便禪思，內寂其心，精勤方便，如實觀察。」**
　　因此，比丘們！你們應該經常修習方便的禪定思惟，使內心寂靜，精勤地運用方便法門，來進行如實的觀察。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如觀察，如是乃至作證十二經，亦應廣說。**
　　像這樣觀察，直到證悟十二經（指十二因緣）的內容，也應如此詳細地說明。

### [詞語註解（解釋重要佛教概念）]
1. **五蘊 (Pañca-khandha)**：構成生命個體的五種要素，包含色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）。
2. **集 (Samudaya)**：指苦的產生原因，在本文中指因對五蘊的貪愛與執著，導致五蘊再次生起，進而產生輪迴之苦。
3. **滅 (Nirodha)**：指苦的止息，透過如實觀察五蘊的過患而斷除貪愛，使五蘊不再生起，達成解脫。
4. **十二經 (Paṭicca-samuppāda)**：指十二因緣，描述無明 ➔ 行 ➔ 識 ➔ 名色 ➔ 六入 ➔ 觸 ➔ 受 ➔ 愛 ➔ 取 ➔ 有 ➔ 生 ➔ 老死 的連鎖反應。
5. **方便禪思 (Upāya-bhāvanā)**：指為了進入洞察（Vipassanā）而先修習的定心方法，使心靈平靜且敏銳，足以觀察法義。

### [現代實作指南]
本經文提供了一套從「心靈平靜」到「認知轉化」再到「行為解脫」的實作流程：

1. **建立平靜的觀察基底（內寂其心）**：
   - 在面對強烈情緒或欲望時，不要立即反應。先透過深呼吸或專注於當下，將心靈從混亂狀態調整為「寂靜」狀態。只有在心不躁動時，才能看到事物的真相而非被情緒扭曲的幻象。

2. **執行「四步觀察法」**：
   - **觀察集（產生）**：意識到這個欲望或痛苦是如何產生的？（例如：看到美食 ➔ 產生渴望 ➔ 執著於口感）。
   - **觀察味（快感）**：承認該事物帶來的暫時快感（例如：美食確實好吃）。
   - **觀察患（過患）**：思考快感後的代價（例如：過量飲食導致身體沉重、對食物的依賴、失去快感後的空虛）。
   - **觀察離（脫離）**：體會當我不再執著於此快感時，內心反而獲得的自由與輕鬆。

3. **轉念實踐**：
   - 當我們發現「快感」與「痛苦」是綑綁在一起的（色集 ➔ 苦聚），我們對該事物的「讚歎」與「愛著」會自然降低。
   - 將注意力從「我想得到它」轉向「它帶來的連鎖反應是什麼」，從而將執著轉化為如實的認知。

### [段落大意]
世尊教導比丘應先修習禪思使內心寂靜，以便如實觀察五蘊的產生（集）與消滅（滅）。凡夫因不見五蘊的過患而執著，導致輪迴與苦聚；聖弟子則透過觀察五蘊的滋味與患處，斷除貪愛，從而獲得解脫，止息生老病死之苦。
