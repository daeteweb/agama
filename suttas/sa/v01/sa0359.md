# 雜阿含經 第 359 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若思量、若妄想生，彼使攀緣識住；有攀緣識住故，有未來世生、老、病、死、憂、悲、惱、苦。如是純大苦聚集。若不思量、不妄想，無使無攀緣識住；無攀緣識住故，於未來世生、老、病、死、憂、悲、惱、苦滅，如是純大苦聚滅。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若思量、若妄想生，彼使攀緣識住；**
　　「如果產生了思量（對對象的思考）或妄想（不真實的想像），這會使得識（意識）產生攀緣（對外在對象的執著與依附）而停留；

**◈ 有攀緣識住故，有未來世生、老、病、死、憂、悲、惱、苦。**
　　因為有了這種攀緣（執著）的識（意識）停留，所以會在未來世產生生（出生）、老、病、死、憂、悲、惱、苦。

**◈ 如是純大苦聚集。**
　　如此一來，純粹巨大的苦（Dukkha）就聚集在一起了。

**◈ 若不思量、不妄想，無使無攀緣識住；**
　　如果不再產生思量（思考）與妄想（想像），就不會使得識（意識）產生攀緣（執著）而停留；

**◈ 無攀緣識住故，於未來世生、老、病、死、憂、悲、惱、苦滅，**
　　因為沒有攀緣（執著）的識（意識）停留，在未來世中，生（出生）、老、病、死、憂、悲、惱、苦都會滅除，

**◈ 如是純大苦聚滅。」**
　　如此一來，純粹巨大的苦（Dukkha）聚集也就隨之滅除了。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **識 (Viññāṇa)**：意識，指對對象的覺知與分辨能力。在本經中，重點在於識如何因執著而「住」在對象上。
2. **攀緣 (Upādāna/Samyoga)**：指心意識像攀爬藤蔓一樣，緊緊地依附、執著於某個對象（如欲望、觀念或自我），不願放手。
3. **苦 (Dukkha)**：指一切不穩定、不滿足且令人不安的狀態，包含生理的痛苦與心理的壓抑。
4. **生、老、病、死**：十二因緣中的最後環節，描述受業力驅使而陷入的輪迴循環。

### [現代實作指南]
本經揭示了「思考 $\rightarrow$ 執著 $\rightarrow$ 痛苦」的心理機制。在現代生活中，我們可以透過以下步驟實踐：

1. **覺察「思量」與「妄想」的起點**：
   當你開始對一件事情反覆思考（思量），或對尚未發生的事感到焦慮/幻想（妄想）時，立即在心中標記：「現在有思考在發生」或「現在有想像在發生」。

2. **識別「攀緣識住」的瞬間**：
   注意你的意識是否「停留」在那個念頭上。例如，當你生氣時，意識會不斷地在「對方錯了」這個念頭上盤旋（住），這就是攀緣。意識到「住」的狀態，是解脫的開始。

3. **練習「不住」的轉念技巧**：
   - **觀察法**：將念頭視為天空中的雲，雲會經過，但不需要抓住它。
   - **切斷鏈條**：當發現意識開始攀緣時，將注意力迅速移回呼吸（正念），讓意識從對象上脫離，打破「思量 $\rightarrow$ 攀緣 $\rightarrow$ 苦」的連鎖反應。

4. **理解因果邏輯**：
   明白目前的憂、悲、惱、苦，並非來自外部事件本身，而是來自於我們讓意識「住」在對該事件的妄想與思量之中。

### 段落大意 
本經闡述了痛苦產生的心理機制：當人產生思量與妄想時，會導致意識對對象產生攀緣與執著（識住），進而導致未來世的生老病死及種種憂悲苦惱。反之，若能止息思量與妄想，使意識不再攀緣執著，則能滅除所有巨大的苦聚集。
