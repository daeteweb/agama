# 雜阿含經 第 490 經

### [原文]
如是我聞：

一時，佛住摩竭提國那羅聚落。

爾時，尊者舍利弗亦在摩竭提國那羅聚落。

時，有外道出家名閻浮車，是舍利弗舊善知識，來詣舍利弗，問訊、共相慰勞已，退坐一面，問舍利弗言：「賢聖法、律中，有何難事？」

舍利弗告閻浮車：「唯出家難。」

「云何出家難？」

答言：「愛樂者難。」

「云何愛樂難？」

答言：「樂常修善法難。」

復問：「舍利弗！有道有向，修習多修習，常修善法增長耶？」

答言：「有。謂八正道。謂正見、正志、正語、正業、正命、正方便、正念、正定。」

閻浮車言：「舍利弗！此則善道，此則善向，修習多修習，於諸善法常修習增長。舍利弗！出家常修習此道，不久疾得盡諸有漏。」

時，二正士共論議已，各從座起而去。

如是比閻浮車所問序四十經。

閻浮車問舍利弗：「云何名善說法者？為世間正向。云何名為世間善逝？」

舍利弗言：「若說法調伏欲貪，調伏瞋恚，調伏愚癡，是名世間說法者。若向調伏欲貪，向調伏瞋恚，向調伏愚癡，是名正向。若貪欲已盡，無餘斷知，瞋恚、愚癡已盡，無餘斷知，是名善斷。」

復問：「舍利弗！有道有向，修習多修習，能起善斷？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「謂涅槃者，云何為涅槃？」

舍利弗言：「涅槃者，貪欲永盡，瞋恚永盡，愚癡永盡，一切諸煩惱永盡，是名涅槃。」

復問：「舍利弗！有道有向，修習多修習，得涅槃耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「何故於沙門瞿曇所出家修梵行？」

舍利弗言：「為斷貪欲故，斷瞋恚故，斷愚癡故，於沙門瞿曇所出家修梵行。」

復問：「舍利弗！有道有向，修習多修習，得斷貪欲、瞋恚、愚癡耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「謂有漏盡。云何名為有漏盡？」

舍利弗言：「有漏者，三有漏，謂欲有漏、有有漏、無明有漏。此三有漏，欲盡無餘，名有漏盡。」

復問：「舍利弗！有道有向，修習多修習，得漏盡耶？」

舍利弗答言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂阿羅漢者，云何名阿羅漢？」

舍利弗言：「貪欲已斷無餘，瞋恚、愚癡已斷無餘，是名阿羅漢。」

復問：「舍利弗！有道有向，修習多修習，得阿羅漢耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂阿羅漢者，云何名阿羅漢者？」

舍利弗言：「貪欲永盡無餘，瞋恚、愚癡永盡無餘，是名阿羅漢者。」

復問：「有道有向，修習多修習，得阿羅漢者耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂無明者，云何為無明？」

舍利弗言：「所謂無明者，於前際無知，後際無知，前、後、中際無知；佛、法、僧寶無知，苦、集、滅、道無知；善、不善、無記無知，內無知、外無知，若於彼彼事無知闇障，是名無明。」

閻浮車語舍利弗：「此是大闇積聚。」復問：「舍利弗！有道有向，修習多修習，斷無明耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車復問尊者舍利弗：「所謂有漏，云何有漏？」如前說。

閻浮車問舍利弗：「所謂有，云何為有？」

舍利弗言：「有謂三有——欲有、色有、無色有。」

復問：「舍利弗！有道有向，修習多修習，斷此有耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂有身，云何有身？」

舍利弗言：「有身者，五受陰。云何五受陰？謂色受陰，受、想、行、識受陰。」

復問：「舍利弗！有道有向，斷此有身耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂苦者。云何為苦？」

舍利弗言：「苦者，謂生苦、老苦、病苦、死苦、恩愛別離苦、怨憎會苦、所求不得苦。略說五受陰苦，是名為苦。」

復問：「舍利弗！有道有向，斷此苦耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂流者，云何為流？」

舍利弗言：「流者，謂欲流、有流、見流、無明流。」

復問：「舍利弗！有道有向，修習多修習，斷此流耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂扼者。云何為扼？」扼如流說。

閻浮車問舍利弗：「所謂取者，云何為取？」

舍利弗言：「取者，四取，謂欲取、我取、見取、戒取。」

復問：「舍利弗！有道有向，修習多修習，斷此取耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂縛者。云何為縛？」

舍利弗言：「縛者，四縛，謂貪欲縛、瞋恚縛、戒取縛、我見縛。」

復問：「舍利弗！有道有向，修習多修習，斷此縛耶？」

舍利弗言：「有。謂八正道。正見，乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂結者。云何為結？」

舍利弗言：「結者，九結，謂愛結、恚結、慢結、無明結、見結、他取結、疑結、嫉結、慳結。」

復問：「舍利弗！有道有向，修習多修習，斷此結耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂使者。云何為使？」

舍利弗言：「使者，七使，謂貪欲使、瞋恚使、有愛使、慢使、無明使、見使、疑使。」

復問：「舍利弗！有道有向，修習多修習，斷此使耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂欲者。云何為欲？」

舍利弗言：「欲者，謂眼所識色可愛、樂、念，染著色。耳聲、鼻香、舌味、身所識觸可愛、樂、念，染著觸。閻浮車！此功德非欲，但覺想思惟者。」是時，舍利弗即說偈言：

「非彼愛欲使，　　世間種種色， 　唯有覺想者，　　是則士夫欲。 　彼諸種種色，　　常在於世間， 　調伏愛欲心，　　是則黠慧者。」

復問：「舍利弗！有道有向，修習多修習，斷此欲耶？」

舍利弗答言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗言：「所謂養者，云何為養？」

舍利弗言：「養者有五養，謂貪欲養、瞋恚養、睡眠養、掉悔養、疑養。」

復問：「舍利弗！有道有向，修習多修習，斷此五養耶？」

舍利弗答言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「謂穌息者。云何為穌息？」

舍利弗言：「穌息者，謂斷三結。」

復問：「舍利弗！有道有向，修習多修習，斷三結耶？」

舍利弗答言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「謂得穌息者。云何為得穌息者？」

舍利弗言：「得穌息者，謂三結已盡、已知。」

復問：「有道有向，斷此結耶？」

舍利弗答言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「謂得上穌息。云何為得上穌息？」

舍利弗言：「得上穌息者，謂貪欲永盡，瞋恚、愚癡永盡，是名得上穌息。」

復問：「舍利弗！有道有向，修習多修習，得上穌息耶？」

舍利弗答言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「謂得上穌息處。云何為得上穌息處？」

舍利弗言：「得上穌息處者，謂貪欲已斷、已知，永盡無餘，瞋恚、愚癡已斷、已知，永盡無餘，是為得上穌息處。」

復問：「舍利弗！有道有向，修習多修習，得上穌息處耶？」

舍利弗答言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂清涼。云何為清涼？」

舍利弗言：「清涼者，五下分結盡，謂身見、戒取、疑、貪欲、瞋恚。」

復問：「有道有向，修習多修習，斷此五下分結，得清涼耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「謂得清涼。云何為得清涼？」

舍利弗言：「五下分已盡、已知，是名得清涼。」

復問：「舍利弗！有道有向，修習多修習，得清涼耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂上清涼者。云何為上清涼？」

舍利弗言：「上清涼者，謂貪欲永盡無餘，瞋恚、愚癡永盡無餘，一切煩惱永盡無餘，是名上清涼。」

復問：「有道有向，得此上清涼耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂得上清涼。云何名得上清涼？」

舍利弗言：「得上清涼者，謂貪欲永盡無餘，已斷、已知，瞋恚、愚癡永盡無餘，已斷、已知，是名得上清涼。」

復問：「舍利弗！有道有向，得此上清涼耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「所謂愛。云何為愛？」

舍利弗言：「有三愛，謂欲愛、色愛、無色愛。」

復問：「有道有向，斷此三愛耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗：「謂業跡。云何為業跡？」

舍利弗言：「業跡者，十不善業跡，謂殺生、偷盜、邪婬、妄語、兩舌、惡口、綺語、貪欲、瞋恚、邪見。」

復問：「舍利弗！有道有向，斷此十業跡耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

閻浮車問舍利弗，「所謂穢者。云何為穢？」

舍利弗言：「穢者，謂三穢，貪欲穢、瞋恚穢、愚癡穢。」

復問：「舍利弗！有道有向，斷此三穢耶？」

舍利弗言：「有。謂八正道——正見乃至正定。」

時，二正士共論議已，各從座起而去。

如穢，如是垢、膩、刺、戀、縛亦爾。

### [白話翻譯]

**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住摩竭提國那羅聚落。**
　　在某一個時間，佛陀住在摩竭提國的那羅聚落。

**◈ 爾時，尊者舍利弗亦在摩竭提國那羅聚落。**
　　那時，尊者舍利弗也同樣住在摩竭提國的那羅聚落。

**◈ 時，有外道出家名閻浮車，是舍利弗舊善知識，來詣舍利弗，問訊、共相慰勞已，退坐一面，問舍利弗言：「賢聖法、律中，有何難事？」**
　　當時，有一位名為閻浮車的非佛教出家修行者，他曾是舍利弗的舊識好友。他來到舍利弗面前，在互相問候、關心彼此近況之後，退到一旁坐下，詢問舍利弗：「在你們那些賢聖的法（真理）與律（戒律）之中，最困難的事情是什麼？」

**◈ 舍利弗告閻浮車：「唯出家難。」**
　　舍利弗告訴閻浮車：「唯有出家（捨棄世俗生活以追求覺悟）是最困難的。」

**◈ 「云何出家難？」**
　　「為什麼說出家很困難？」

**◈ 答言：「愛樂者難。」**
　　舍利弗回答：「因為對世俗的愛著與樂趣（貪愛）使其變得困難。」

**◈ 「云何愛樂難？」**
　　「為什麼愛著與樂趣會讓出家變得困難？」

**◈ 答言：「樂常修善法難。」**
　　舍利弗回答：「因為要讓自己樂於恆常地修行善法（正向的修行）是非常困難的。」

**◈ 復問：「舍利弗！有道有向，修習多修習，常修善法增長耶？」**
　　閻浮車又問：「舍利弗！是否有一條道路或一個方向，只要多加練習與修習，就能讓善法恆常地增長呢？」

**◈ 答言：「有。謂八正道。謂正見、正志、正語、正業、正命、正方便、正念、正定。」**
　　舍利弗回答：「有的。那就是八正道（通往解脫的八種正確修行路徑）：也就是正見（正確的見地）、正思惟（正確的思考）、正語（正確的言語）、正業（正確的行為）、正命（正確的生活方式）、正精進（正確的努力）、正念（正確的覺知）以及正定（正確的專注）。」

**◈ 閻浮車言：「舍利弗！此則善道，此則善向，修習多修習，於諸善法常修習增長。舍利弗！出家常修習此道，不久疾得盡諸有漏。」**
　　閻浮車說：「舍利弗！這確實是良善的道路與正確的方向，只要多加修習，就能讓各種善法恆常增長。舍利弗！出家者若恆常修習這條道路，不久就能快速地消除所有有漏（被煩惱染污的狀態）。」

**◈ 時，二正士共論議已，各從座起而去。**
　　那時，這兩位正理之士討論完畢後，各自從座位上起身離開。

**◈ 如是比閻浮車所問序四十經。**
　　像這樣由閻浮車發問而開始的經文共有四十篇。

**◈ 閻浮車問舍利弗：「云何名善說法者？為世間正向。云何名為世間善逝？」**
　　閻浮車問舍利弗：「什麼樣的人才稱得上是善於說法的人？什麼樣的人才是世間的正向引導者？而什麼樣的人才稱得上是世間的善逝（圓滿地解脫而離去的人）？」

**◈ 舍利弗言：「若說法調伏欲貪，調伏瞋恚，調伏愚癡，是名世間說法者。若向調伏欲貪，向調伏瞋恚，向調伏愚癡，是名正向。若貪欲已盡，無餘斷知，瞋恚、愚癡已盡，無餘斷知，是名善斷。」**
　　舍利弗說：「如果說法能讓聽者調伏欲貪（對物質的貪慾）、調伏瞋恚（憤怒與厭惡）、調伏愚癡（對真理的無知），這就稱作世間的說法者。如果能引導他人調伏欲貪、調伏瞋恚、調伏愚癡，這就稱作正向。如果貪欲已經完全消失且被徹底斷除，瞋恚與愚癡也完全消失且被徹底斷除，這就稱作善斷（圓滿的斷除）。」

**◈ 復問：「舍利弗！有道有向，修習多修習，能起善斷？」**

### [詞語註解]

1. **出家 (Pabbajjā)**：原意為「走出家門」，指捨棄世俗的家庭生活與社會角色，全心投入對真理的追求與心靈的解脫。
2. **有漏 (Āsava)**：指像滲漏一樣不斷流出的心理染污，使心靈的平靜流失，導致眾生在輪迴中受苦。分為三種：欲有漏（對感官欲望的執著）、有有漏（對生存狀態的執著）、無明有漏（對真理的無知）。
3. **八正道 (Ariya Aṭṭhaṅgika Magga)**：佛法中消除苦因的核心修行路徑，分為智慧（正見、正思惟）、戒律（正語、正業、正命）與心靈訓練（正精進、正念、正定）三個部分。
4. **三毒 (Kilesa)**：貪欲、瞋恚、愚癡，是一切煩惱與痛苦的根源。貪是強烈的渴求，瞋是強烈的排斥，癡是對事物本質（無常、無我）的誤認。
5. **涅槃 (Nibbāna)**：字面意思為「熄滅」，指貪、瞋、癡三把火完全熄滅後的狀態，是超越所有痛苦與輪迴的絕對寧靜境界。
6. **梵行 (Brahmacariya)**：指清淨的修行生活，強調在道德與心靈上的純潔，不被世俗欲望所牽引。

### [現代實作指南]

本經文透過閻浮車與舍利弗的層層問答，揭示了從「識別問題」到「找到方法」再到「達成解脫」的完整修行地圖。在現代生活中，我們可以將其轉化為以下系統化的實踐步驟：

**第一步：識別心靈的「障礙」**

1. **覺察「愛樂」的阻礙**：當我們想要改變生活（如戒除壞習慣、學習新技能）卻遲遲無法行動時，誠實地問自己：「我現在正愛著什麼（舒適區、短暫的快感）而讓我無法前進？」意識到「困難」並非來自外部，而是來自於對舊有快感的依戀。

2. **分類「苦」的具體形式**：當感到不快樂時，不要只說「我很痛苦」，而要將其分類。是「所求不得」（目標未達成）？還是「怨憎會」（遇到討厭的人）？將模糊的情緒具體化，能降低情緒的壓迫感。

3. **標記「驅使者」**：當你感到情緒起伏或陷入壓力時，試著標記出目前的驅使因素。例如：我是因為「慢」（覺得自己比別人優秀而受挫）還是因為「見」（對某件事有固執的錯誤看法）而痛苦？

**第二步：運用「八正道」進行系統性對治**

1. **智慧層面（正見、正思惟）**：
   - 建立正確的因果觀：意識到貪婪導致痛苦，憤怒破壞關係，無明造成困惑。
   - 每天提醒自己「一切現象皆是因緣而生，且無常」。
   - 當憤怒升起時，思考「這個憤怒能解決問題嗎？」，將情緒轉化為理性的分析。

2. **戒律層面（正語、正業、正命）**：
   - **正語**：今天嘗試不說任何抱怨或傷害他人的話。在社交媒體或對話中，練習在發言前自問：這句話是否包含「兩舌」或「惡口」？
   - **正業**：不採取損人利己的行動。
   - **正命**：確保生活方式不產生新的染污。

3. **心靈訓練層面（正精進、正念、正定）**：
   - **正精進**：持續努力，不讓惡法生起，已生惡法令斷，未生善法令生，已生善法令增長。
   - **正念**：在洗碗或走路時，將意識完全放在當下的動作上，不讓心隨意飄移。觀察心中升起的「受（感受）」，僅僅觀察它，而不被它捲走。
   - **正定**：每天抽出10分鐘，僅僅觀察呼吸，將心從對外界的「渴愛」中收回，體驗短暫的「清涼」。

**第三步：針對「三毒」進行精準調伏**

1. **對治貪**：練習「知足」，在想要購買非必要物品時，停頓三秒，觀察內心那股「渴求」的感覺，直到它自然消退。區分「覺知」與「欲」——看到物品是覺知，但「想要擁有它」才是欲。

2. **對治瞋**：在感到憤怒時，深呼吸並意識到「瞋恚正在傷害我的心」，將注意力從對方的錯誤轉移到自己的心態上。練習「多維度觀察」：嘗試從對方的視角、第三者的視角以及長遠的時間軸來觀察當下。

3. **對治癡**：保持好奇心與批判性思考，不盲從於直覺或偏見，透過學習與觀察來了解事物的因果關係。當你對某人產生強烈偏見時，提醒自己：「我現在可能處於無明狀態，我所看到的並非全貌。」

**第四步：追求「穌息」與「清涼」**

1. **分層清理心理枷鎖**：
   - **對抗身見**：當你覺得「我被冒犯了」時，提醒自己這個「我」是由不斷變化的情緒與想法組成，並非恆常實體。
   - **對抗疑與戒取**：不要盲從於某種特定的生活儀式或對未來產生過度擔憂，而是專注於當下的正念觀察。
   - **對抗貪與瞋**：練習「觀察欲望的生滅」。當貪欲升起時，觀察它如何產生，又如何隨時間淡化，而非立即滿足它。

2. **覺察「三愛」的陷阱**：意識到我們對快樂的追求往往在升級：從追求物質（欲愛）➔ 追求精神上的優越感或寧靜感（色愛/無色愛）。真正的清涼不是尋找更高級的快樂，而是斷除對「必須獲得快樂」的執著。

3. **系統性清理（心靈大掃除）**：意識到「垢、膩、刺、戀、縛」都是心靈的負擔。每天睡前進行簡單的檢視：今日是否有製造不善業跡？透過「正定（專注）」的練習，讓心靈在寧靜中洗滌三穢，而非在忙碌中不斷累積污垢。
