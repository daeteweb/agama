# 雜阿含經 第 85 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「比丘！於何所不見我、異我、相在？」

比丘白佛：「世尊為法根、法眼、法依，唯願為說！諸比丘聞已，如說奉行。」

佛告比丘：「諦聽，善思，當為汝說。於色不見有我、異我、相在不？於受、想、行、識亦復如是。比丘！色為是常、無常耶？」

比丘白佛：「無常。世尊！」

佛言：「比丘！若無常者，是苦不？」

比丘白佛：「是苦。世尊！」

「比丘！若無常、苦，是變易法，多聞聖弟子寧於中見我、異我、相在不？」

比丘白佛：「不也，世尊！」

「受、想、行、識亦復如是。是故，比丘！諸所有色，若過去、若未來、若現在，若內、若外，若麤、若細，若好、若醜，若遠、若近，彼一切非我、不異我、不相在。受、想、行、識亦復如是。比丘！多聞聖弟子觀察五受陰非我、非我所。如是觀察者，於諸世間都無所取，無所取者無所著，無所著故自覺涅槃：『我生已盡，梵行已立，所作已作，自知不受後有。』」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「比丘！於何所不見我、異我、相在？」**
　　那時，世尊告訴在場的眾位比丘說：「比丘們！在什麼地方，你們不會看到『我』、『與我不同的我』或是『我的特徵』存在？」

**◈ 比丘白佛：「世尊為法根、法眼、法依，唯願為說！諸比丘聞已，如說奉行。」**
　　比丘們對佛陀恭敬地說道：「世尊您是法之根源、法之眼光與法的依託，唯願您為我們開示！我們眾比丘聽聞之後，定會虔誠地接受並依照教導去實踐。」

**◈ 佛告比丘：「諦聽，善思，當為汝說。於色不見有我、異我、相在不？於受、想、行、識亦復如是。比丘！色為是常、無常耶？」**
　　佛陀告訴比丘：「請仔細聽，好好思考，我現在為你們解說。在色（物質身體）之中，難道沒有看到『我』、『與我不同的我』或『我的特徵』存在嗎？在受（感受）、想（想像與認知）、行（心的造作）、識（意識）之中也是同樣的情況。比丘們！色（物質身體）是恆常的，還是無常的呢？」

**◈ 比丘白佛：「無常。世尊！」**
　　比丘們對佛陀恭敬地說道：「是無常的，世尊！」

**◈ 佛言：「比丘！若無常者，是苦不？」**
　　佛陀說：「比丘們！如果它是無常的，那它是否就是苦（不滿足與痛苦）呢？」

**◈ 比丘白佛：「是苦。世尊！」**
　　比丘們對佛陀恭敬地說道：「是苦的，世尊！」

**◈ 「比丘！若無常、苦，是變易法，多聞聖弟子寧於中見我、異我、相在不？」**
　　「比丘們！既然它是無常且是苦的，屬於一種不斷變遷的法（現象），那麼博學的聖弟子難道會在其中看到『我』、『與我不同的我』或『我的特徵』存在嗎？」

**◈ 比丘白佛：「不也，世尊！」**
　　比丘們對佛陀恭敬地說道：「不會的，世尊！」

**◈ 「受、想、行、識亦復如是。是故，比丘！諸所有色，若過去、若未來、若現在，若內、若外，若麤、若細，若好、若醜，若遠、若近，彼一切非我、不異我、不相在。受、想、行、識亦復如是。」**
　　「受（感受）、想（想像與認知）、行（心的造作）、識（意識）也是同樣的情況。因此，比丘們！所有屬於色（物質身體）的現象，無論是過去、未來或現在，無論是內在或外在，無論是粗糙或微細，無論是美好或醜陋，無論是遙遠或接近，這一切都不是我，也不與我不同，更沒有我的特徵存在。受（感受）、想（想像與認知）、行（心的造作）、識（意識）也是同樣的情況。」

**◈ 「比丘！多聞聖弟子觀察五受陰非我、非我所。如是觀察者，於諸世間都無所取，無所取者無所著，無所著故自覺涅槃：『我生已盡，梵行已立，所作已作，自知不受後有。』」**
　　「比丘們！博學的聖弟子觀察這五蘊（構成身心的五種要素）並非是我，也不是我所擁有的。如此觀察的人，對於世間的一切都沒有執取（貪愛而抓取），沒有執取就沒有執著，因為沒有執著，所以能自覺地進入涅槃（超越生死、絕對寧靜的解脫境界），並體悟到：『我的輪迴出生已經結束，清淨的修行已經建立，應該完成的任務已經完成，自知不再受後世的出生。』」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **五蘊 (Pañca-khandha)**：指構成生命個體的五種要素，包含色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）。
2. **無常 (Anicca)**：指一切有為法（因緣而生的現象）都在不斷變遷，沒有永恆不變的實體。
3. **苦 (Dukkha)**：在學術上指「不滿足」或「不穩定」的狀態。因為事物無常，所以任何對無常之物的依附必然導致痛苦。
4. **涅槃 (Nibbāna)**：指熄滅貪、嗔、痴三毒，從輪迴的苦海中獲得絕對寧靜與自由的解脫境界。
5. **梵行 (Brahmacariya)**：指清淨的修行生活，旨在消除煩惱並達到覺悟。

### [現代實作指南]
本經文提供了一套邏輯嚴密的「去認同」練習法，可用於緩解現代人的焦慮與自我執著：

1. **觀察現象的無常性**：
   當你感到強烈的情緒（如憤怒或悲傷）時，不要說「我很憤怒」，而要觀察到「現在有一個『受（感受）』在發生」。意識到這個感受在這一秒出現，下一秒就在改變，它是不恆常的。

2. **邏輯推演苦因**：
   意識到：因為這個感受是無常的 → 如果我試圖抓住它或要求它不變 → 我必然會感到痛苦（苦）。

3. **切斷「我」的連結**：
   問自己：「這個不斷變動、且會帶來痛苦的感受，真的是『我』嗎？」
   ➔ 如果它是「我」，它應該由我掌控且永恆不變。
   ➔ 但事實上它在自行變遷，因此它「非我」。

4. **實踐「無所取」**：
   當你意識到身體的疼痛、心靈的壓力或外界的評價都只是「五蘊」的暫時組合，而非你的本質時，你會發現自己不再需要強行「抓取」某種狀態來定義自己。這種「不抓取」的狀態，就是心靈平靜（微型涅槃）的開始。

### [段落大意]
佛陀透過問答法，引導比丘體悟五蘊（色、受、想、行、識）的本質。由於五蘊具有「無常」與「苦」的特性，因此它們是不可能成為恆常的「我」或「我的特徵」。聖弟子若能觀察到五蘊非我，便能消除對世間的執取與執著，最終達成涅槃的解脫，終結輪迴之苦。
