# 雜阿含經 第 436 經

### [原文]
如須達長者所問，有異比丘問，亦如是說，唯譬有差別：「如有四登階道，昇於殿堂，若有說言不登初階，而登第二、第三、第四階昇堂殿者，無有是處。

所以者何？要由初階，然後次登第二、第三、第四階得昇殿堂。

如是，比丘！於苦聖諦未無間等，而欲於苦集聖諦、苦滅聖諦、苦滅道跡聖諦無間等者，無有是處。

「譬如，比丘！若有人言：『以四階道昇於殿堂，要由初階，然後次登第二、第三、第四階得昇殿堂，應作是說。』

所以者何？要由初階，然後次登第二、第三、第四階昇於殿堂，有是處故。

如是，比丘！若言於苦聖諦無間等已，然後次第於苦集聖諦、苦滅聖諦、苦滅道跡聖諦無間等者，應作是說。

所以者何？若於苦聖諦無間等已，然後次第於苦集聖諦、苦滅聖諦、苦滅道跡聖諦無間等者，有是處故。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如須達長者所問，有異比丘問，亦如是說，唯譬有差別**
　　就像先前回答須達長者的提問一樣，現在有另一位比丘（出家修行者）來詢問，佛陀也以同樣的道理來回答，僅僅是在使用的比喻上有所不同。

**◈ 「如有四登階道，昇於殿堂，若有說言不登初階，而登第二、第三、第四階昇堂殿者，無有是處。**
　　「比丘們，假設有四級階梯可以走上殿堂，如果有人聲稱不需要踏上第一級階梯，就能直接踏上第二、第三、第四級階梯而進入殿堂，這在現實中是不可能的。

**◈ 所以者何？要由初階，然後次登第二、第三、第四階得昇殿堂。**
　　為什麼呢？因為必須先從第一級階梯開始，接著才依序踏上第二、第三、第四級階梯，才能成功進入殿堂。

**◈ 如是，比丘！於苦聖諦未無間等，而欲於苦集聖諦、苦滅聖諦、苦滅道跡聖諦無間等者，無有是處。**
　　同樣地，比丘們！如果一個人還沒有徹底領悟且毫無間隔地體會苦聖諦（關於苦的真理），卻想要直接徹底領悟苦集聖諦（關於苦之原因的真理）、苦滅聖諦（關於苦之止息的真理）以及苦滅道跡聖諦（關於解脫之道路的真理），這也是不可能實現的。

**◈ 「譬如，比丘！若有人言：『以四階道昇於殿堂，要由初階，然後次登第二、第三、第四階得昇殿堂，應作是說。』**
　　「比丘們，譬如有人說：『若要透過四級階梯進入殿堂，必須先從第一級開始，然後依序踏上第二、第三、第四級才能進入』，這樣說才是正確的。

**◈ 所以者何？要由初階，然後次登第二、第三、第四階昇於殿堂，有是處故。**
　　為什麼呢？因為在現實中，確實必須由第一級階梯開始，依序踏上後續階梯才能進入殿堂。

**◈ 如是，比丘！若言於苦聖諦無間等已，然後次第於苦集聖諦、苦滅聖諦、苦滅道跡聖諦無間等者，應作是說。**
　　同樣地，比丘們！如果有人說：『必須先徹底領悟且毫無間隔地體會苦聖諦（關於苦的真理），接著才依序領悟苦集聖諦（關於苦之原因的真理）、苦滅聖諦（關於苦之止息的真理）以及苦滅道跡聖諦（關於解脫之道路的真理）』，這樣說才是正確的。

**◈ 所以者何？若於苦聖諦無間等已，然後次第於苦集聖諦、苦滅聖諦、苦滅道跡聖諦無間等者，有是處故。」**
　　為什麼呢？因為在修行實踐中，確實是先徹底領悟苦聖諦，然後才依序領悟苦集、苦滅與苦滅道跡聖諦，這才是正確且可行的方法。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四項真理，包含苦（生命的本質是苦）、集（苦的根源是貪愛與無明）、滅（苦的止息即是涅槃）、道（達成止息的八正道）。
2. **無間等**：在此指一種直接的、不被妄想或錯誤認知所中斷的深刻體悟，即對真理的完全覺知。
3. **苦聖諦 (Dukkha-Sacca)**：對生命中不可避免的苦（如生老病死、求不得、怨憎會、愛別離）及其不穩定本質的正確認知。
4. **苦集聖諦 (Samudaya-Sacca)**：領悟導致苦產生的根本原因，即對感官欲望的貪愛（Taṇhā）與對真理的無明。
5. **苦滅聖諦 (Nirodha-Sacca)**：體悟當貪愛與無明完全熄滅時，苦也隨之消失的寂靜狀態（涅槃）。
6. **苦滅道跡聖諦 (Magga-Sacca)**：實踐能導向苦之止息的具體路徑，即八正道。

### [現代實作指南]
本經文強調的是「認知順序」的重要性。在現代生活中，我們常試圖跳過「面對問題」直接尋找「解決方案」，這就像是不登初階而欲昇殿堂。

**實作步驟：**
1. **第一階：正視苦（苦聖諦）**
   - **轉念技巧**：當壓力或痛苦發生時，不要急著逃避或用娛樂掩蓋。練習對自己說：「現在這種不舒服的感覺就是『苦』，它是生命的一部分。」
   - **操作**：記錄下目前最令你不安的事，不加評判地觀察它，承認它存在，而非試圖立刻消除它。

2. **第二階：追溯因（苦集聖諦）**
   - **轉念技巧**：問自己：「我對這個情況的什麼『執著』導致了我的痛苦？」
   - **操作**：分析痛苦背後的貪愛（想要某物）或厭惡（不想要某物）。例如：是因為「我認為事情應該按我的意願發展」而痛苦嗎？

3. **第三階：確認可能性（苦滅聖諦）**
   - **轉念技巧**：意識到「如果執著消失，痛苦也會隨之消失」。
   - **操作**：想像一種狀態，如果你不再強求結果，當下的心境會變得如何？感受那種暫時的鬆弛感。

4. **第四階：採取行動（苦滅道跡聖諦）**
   - **轉念技巧**：將注意力從「結果」轉移到「正確的行為」上。
   - **操作**：根據八正道（如正念、正精進），採取一個能減少貪執的具體行動，而非採取一個基於恐懼或貪婪的反應。

### [段落大意]
佛陀以「登四級階梯進入殿堂」為喻，說明領悟四聖諦必須遵循嚴格的次第。修行者必須先徹底領悟「苦聖諦」，才能依序領悟「苦集」、「苦滅」與「苦滅道跡」聖諦。若試圖跳過對苦的認知而直接追求解脫，在邏輯與實踐上均不可能達成。
