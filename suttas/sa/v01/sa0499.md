# 雜阿含經 第 499 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。爾時，尊者舍利弗在耆闍崛山中。

時，有月子比丘！是提婆達多弟子，詣尊者舍利弗，共相問訊慰勞已，退住一面。退住一面已，尊者舍利弗問月子比丘言：「提婆達多比丘為諸比丘說法不？」

月子比丘答言：「說法。」

尊者舍利弗問月子比丘言：「提婆達多云何說法？」

月子比丘語尊者舍利弗言：「彼提婆達多如是說法言：『比丘心法修心，是比丘能自記說：「我已離欲，解脫五欲功德。」』」

舍利弗語月子比丘言：「汝提婆達多何以不說法言：『比丘心法善修心，離欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，不轉還欲有、色有、無色有法，彼比丘能自記說言：「我生已盡，梵行已立，所作已作，自知不受後有。」耶？』」

月子比丘言：「彼不能也。尊者舍利弗！」

爾時，尊者舍利弗語月子比丘言：「若有比丘心法善修心者，能離貪欲心、瞋恚、愚癡心，得無貪法、無恚、無癡法，是比丘能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』

「譬如村邑，近有大石山，不斷、不壞、不穿、厚密，正使東方風來，不能令動，亦復不能過至西方。如是南、西、北方、四維風來，不能傾動，亦不能過。如是，比丘心法善修心者，離貪欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，是比丘能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』

「譬如因陀銅鐵及銅柱深入地中，築令堅密，四方風吹不能傾動。如是，比丘心法善修心已，離貪欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，是比丘能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』

「譬如石柱長十六肘，八肘入地，四方風吹不能傾動。如是，比丘心法善修心已，悉離貪欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』

「譬如火燒未燒者，燒已不復更燒。

如是，比丘心法修心已，離貪欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』

舍利弗說此經已，諸比丘聞其所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。爾時，尊者舍利弗在耆闍崛山中。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。那時，尊者舍利弗則在耆闍崛山中。

**◈ 時，有月子比丘！是提婆達多弟子，詣尊者舍利弗，共相問訊慰勞已，退住一面。**
　　當時，有一位名叫月子的比丘，他是提婆達多的弟子，來到尊者舍利弗這裡，雙方在互相問候慰勞之後，月子比丘便退到一旁恭敬地站著。

**◈ 退住一面已，尊者舍利弗問月子比丘言：「提婆達多比丘為諸比丘說法不？」**
　　在月子比丘退到一旁後，尊者舍利弗問他：「提婆達多比丘是否有為其他比丘們宣說法（真理）？」

**◈ 月子比丘答言：「說法。」**
　　月子比丘回答：「他有說法。」

**◈ 尊者舍利弗問月子比丘言：「提婆達多云何說法？」**
　　尊者舍利弗問月子比丘：「提婆達多是如何說法的？」

**◈ 月子比丘語尊者舍利弗言：「彼提婆達多如是說法言：『比丘心法修心，是比丘能自記說：「我已離欲，解脫五欲功德。」』」**
　　月子比丘告訴尊者舍利弗：「提婆達多是這樣說法的：『比丘只要在心中修習心法（修行），這位比丘就能自我宣告：我已經遠離了貪欲，從五種感官欲望（五欲）的束縛中解脫。』」

**◈ 舍利弗語月子比丘言：「汝提婆達多何以不說法言：『比丘心法善修心，離欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，不轉還欲有、色有、無色有法，彼比丘能自記說言：「我生已盡，梵行已立，所作已作，自知不受後有。」耶？』」**
　　舍利弗對月子比丘說：「你們的提婆達多為什麼不這樣說法：『比丘若能善於修習心法，遠離貪愛的心、遠離憤怒瞋恨的心、遠離愚昧無知的癡心，進而獲得無貪、無瞋、無癡的境界，不再輪迴於欲界（充滿欲望的世界）、色界（具有物質形式但無欲望的世界）以及無色界（完全沒有物質形式的世界），這位比丘才能自我宣告：我的出生（輪迴）已經結束，清淨的修行（梵行）已經完成，應該做的事已經全部做到，自知不再有後世的出生。』為什麼他不這樣說呢？」

**◈ 月子比丘言：「彼不能也。尊者舍利弗！」**
　　月子比丘回答：「尊者舍利弗，他不能這樣說。」

**◈ 爾時，尊者舍利弗語月子比丘言：「若有比丘心法善修心者，能離貪欲心、瞋恚、愚癡心，得無貪法、無恚、無癡法，是比丘能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』**
　　那時，尊者舍利弗告訴月子比丘：「如果有一位比丘能善於修習心法，能夠遠離貪欲心、瞋恚心與愚癡心，並獲得無貪、無瞋、無癡的法（狀態），這位比丘才能自我宣告：我的出生（輪迴）已經結束，清淨的修行（梵行）已經完成，應該做的事已經全部做到，自知不再有後世的出生。」

**◈ 「譬如村邑，近有大石山，不斷、不壞、不穿、厚密，正使東方風來，不能令動，亦復不能過至西方。如是南、西、北方、四維風來，不能傾動，亦不能過。如是，比丘心法善修心者，離貪欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，是比丘能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』**
　　「就如同在村落附近有一座巨大的石山，它完整且堅固，沒有裂縫或破損，厚實且緻密。即便東方的風吹來，也不能使其搖動，更不能讓風直接穿過到達西方。同樣地，無論南、西、北或四方八方的風吹來，都不能使其傾斜或撼動。同樣地，比丘若能善於修習心法，遠離貪欲心、瞋恚心與愚癡心，獲得無貪、無瞋、無癡的法，這位比丘才能自我宣告：我的出生（輪迴）已經結束，清淨的修行（梵行）已經完成，應該做的事已經全部做到，自知不再有後世的出生。」

**◈ 「譬如因陀銅鐵及銅柱深入地中，築令堅密，四方風吹不能傾動。如是，比丘心法善修心已，離貪欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，是比丘能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』**
　　「就如同銅鐵鑄成的柱子深深地植入地中，築造得極其堅固緻密，四方之風吹襲也不能使其搖動。同樣地，比丘在善於修習心法後，遠離了貪欲心、瞋恚心與愚癡心，獲得無貪、無瞋、無癡的法，這位比丘才能自我宣告：我的出生（輪迴）已經結束，清淨的修行（梵行）已經完成，應該做的事已經全部做到，自知不再有後世的出生。」

**◈ 「譬如石柱長十六肘，八肘入地，四方風吹不能傾動。如是，比丘心法善修心已，悉離貪欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』**
　　「就如同石柱長十六肘（一肘約 45 公分），其中八肘深埋於地底，四方之風吹襲也不能使其搖動。同樣地，比丘在善於修習心法後，悉數遠離了貪欲心、瞋恚心與愚癡心，獲得無貪、無瞋、無癡的法，才能自我宣告：我的出生（輪迴）已經結束，清淨的修行（梵行）已經完成，應該做的事已經全部做到，自知不再有後世的出生。」

**◈ 「譬如火燒未燒者，燒已不復更燒。**
　　「就如同火燒掉尚未被燒過的東西，一旦燒完了，就不能再次被燒毀。」

**◈ 如是，比丘心法修心已，離貪欲心，離瞋恚心，離愚癡心，得無貪法、無恚法、無癡法，能自記說：『我生已盡，梵行已立，所作已作，自知不受後有。』**
　　就像這樣，當一位比丘透過修行圓滿地訓練心靈之後，遠離了貪欲之心，遠離了瞋恚之心，遠離了愚癡之心，進而獲得了無貪、無瞋、無癡的心靈狀態，這位比丘便能自信地宣告：「我的輪迴出生已經結束，清淨的修行生活已經圓滿建立，應該完成的修行任務已經全部完成，我清楚地知道自己不再會經歷後世的輪迴出生。」

**◈ 舍利弗說此經已，諸比丘聞其所說，歡喜奉行。**
　　舍利弗講完這段教法後，在場的眾位比丘聽聞了他所說的內容，內心生起法喜並依照教導去實踐。

### [詞語註解]
1. **五欲 (Pañca-kāma)**：指透過五種感官（眼、耳、鼻、舌、身）所產生的對物質與快感的貪愛與追求。提婆達多的教法僅止於此層次的解脫。
2. **貪、瞋、癡 (Lobha, Dosa, Moha)**：佛教中的「三毒」。貪是指對對象的執著與渴求；瞋是指對不滿意對象的憤怒與排斥；癡是指對真理（如無常、無我）的無知與迷茫。這是導致眾生在輪迴中受苦的核心心理染污。
3. **欲有、色有、無色有 (Kāma-bhava, Rūpa-bhava, Arūpa-bhava)**：輪迴的三個層次。欲有（欲界）是充滿欲望的層級；色有（色界）是擺脫粗糙欲望但仍有物質形態的層級；無色有（無色界）是完全沒有物質形態，僅有意識狀態的層級。
4. **梵行 (Brahmacariya)**：原意為「清淨的行持」，在解脫脈絡中指為了達到涅槃而實踐的聖潔生活與修行路徑，特指遠離感官慾望、致力於解脫的聖潔生活。
5. **不受後有 (Anupādāna / Nibbāna)**：指由於斷除了執取與渴愛，不再產生新的業力，因此在肉身死亡後不再投生於六道輪迴，徹底終結生死流轉。

### [現代實作指南]
本經文揭示了「表面心靈修煉」與「徹底根除煩惱」的區別。提婆達多的教法僅強調「修心」與「離欲」，這在現代心理學中類似於「壓抑」或「暫時的放空」，而舍利弗尊者強調的是對三毒（貪、瞋、癡）的徹底根除。以下是系統化的實踐步驟：

**步驟一：識別三毒的即時反應**
- 當你感到想要佔有某物時，標記為「貪」。
- 當你感到憤怒或厭惡時，標記為「瞋」。
- 當你感到迷茫、執著於錯誤認知或不願面對真相時，標記為「癡」。

**步驟二：從「壓抑」轉向「洞察」**
- 不要試圖用意志力強行「壓制」欲望（如提婆達多的方法），而應觀察欲望產生的因緣。
- 思考：這個欲望是基於什麼樣的錯誤認知（癡）？如果得到它，是否能帶來永恆的滿足（無常）？
- 每次意識到貪、瞋、癡升起時，告訴自己：「這只是心的造作（行），並非我本身。」

**步驟三：建立「心靈之山」的穩定性**
- 練習在情緒風暴（如被批評、失去財產）來襲時，像經文中的「大石山」一樣，不被情緒牽動。
- 技巧：將注意力集中在呼吸或當下的身體感受，意識到「情緒只是經過的風」，而「覺知」才是不動的石山。
- 練習在情緒起伏時，不立即做出反應，如同深埋地底的石柱，將「覺知」深植於心中。

**步驟四：從「遠離」到「獲得」的轉念**
- 不要強行壓抑情緒，而是透過觀察其「無常」的本質，讓其自然消退。
- 當憤怒消退後，感受那種「無恚」的寧靜狀態，刻意停留在此狀態，讓心靈習慣於這種清淨的法。

**步驟五：驗證修行的深度**
- 真正的進步不在於能進入多深的精神狀態，而是在於面對同樣的誘惑或衝突時，內心的貪、瞋、癡是否減少了。
- 如同火燒盡後不再復燃，徹底斷除煩惱後，心靈將不再被同樣的煩惱所困擾。

### [段落大意]
本段記錄了尊者舍利弗與提婆達多弟子月子比丘的對話。舍利弗指出提婆達多的教法過於淺層，僅強調修心與離欲；真正的解脫必須徹底根除貪、瞋、癡三毒，才能斷除在欲界、色界、無色界的輪迴，達到不再出生的涅槃境界。文中以大石山、深埋的銅鐵柱與石柱為喻，強調徹底斷除煩惱後的心靈將如同堅固之山石，不再受外界干擾與輪迴之風的搖撼。最後，眾比丘對此教法深感歡喜並致力實踐。
