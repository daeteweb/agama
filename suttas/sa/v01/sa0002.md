# 雜阿含經 第 2 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「於色當正思惟，色無常如實知。所以者何？比丘！於色正思惟，觀色無常如實知者，於色欲貪斷；欲貪斷者，說心解脫。

「如是受、想、行、識當正思惟，觀識無常如實知。所以者何？於識正思惟，觀識無常者，則於識欲貪斷；欲貪斷者，說心解脫。

「如是心解脫者，若欲自證，則能自證：『我生已盡，梵行已立，所作已作，自知不受後有。』如是正思惟無常，苦、空、非我亦復如是。」

時，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「於色當正思惟，色無常如實知。**
　　「對於色（物質身體），應當運用正思惟（正確的思考方式），如實地了解色（物質身體）是無常（遷變不定、不恆久）的。

**◈ 所以者何？比丘！**
　　為什麼呢？比丘們！

**◈ 於色正思惟，觀色無常如實知者，於色欲貪斷；**
　　當對色（物質身體）進行正思惟，並如實地觀察到色（物質身體）的無常時，對色（物質身體）的欲貪（渴望與執著）就會被截斷。

**◈ 欲貪斷者，說心解脫。**
　　當欲貪被截斷後，心靈便能獲得解脫（從束縛中釋放）。

**◈ 「如是受、想、行、識當正思惟，觀識無常如實知。**
　　「同樣地，對於受（感受）、想（想像與認知）、行（心的造作）、識（意識），也應當運用正思惟，如實地了解識（意識）是無常（遷變不定、不恆久）的。

**◈ 所以者何？於識正思惟，觀識無常者，則於識欲貪斷；**
　　為什麼呢？因為對識（意識）進行正思惟，並觀察到識（意識）的無常，對識（意識）的欲貪（渴望與執著）就會被截斷。

**◈ 欲貪斷者，說心解脫。**
　　當欲貪被截斷後，心靈便能獲得解脫（從束縛中釋放）。

**◈ 「如是心解脫者，若欲自證，則能自證：**
　　「像這樣獲得心靈解脫的人，如果想要自我證悟，就能夠自我證悟：

**◈ 『我生已盡，梵行已立，所作已作，自知不受後有。』**
　　『我的輪迴出生已經結束，清淨的修行（梵行）已經建立，應做的事已經完成，我自知不再受後世的出生（不再輪迴）。』

**◈ 如是正思惟無常，苦、空、非我亦復如是。」**
　　像這樣正思惟地觀察無常，對於苦（不滿足的狀態）、空（缺乏恆常實體）、非我（無我，非永恆不變的自我）也是同樣的道理。」

**◈ 時，諸比丘聞佛所說，歡喜奉行。**
　　那時，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **五蘊 (Pañca-kkhandha)**：構成個體身心的五種要素，包含色（物質）、受（感受）、想（認知）、行（意志造作）、識（意識）。
2. **正思惟 (Samma-sankappa)**：八正道之一，指正確的思考方向，在此指將心念導向觀察無常、離欲與不害。
3. **無常 (Anicca)**：指一切有為法（因緣和合之物）都在不斷地生滅變遷，沒有永恆不變的實體。
4. **梵行 (Brahmacariya)**：原指禁欲生活，在學術上指清淨的修行生活，旨在消除貪嗔痴以達到解脫。
5. **非我 (Anattā)**：指五蘊中沒有一個恆常、獨立、主宰的「自我」存在。

### [現代實作指南]
本經文提供了一套從「認知」到「解脫」的邏輯路徑：**正思惟 ➔ 觀察無常 ➔ 斷除貪愛 ➔ 心靈解脫**。在現代生活中可嘗試以下實作步驟：

1. **物質層面的無常觀察（色）**：
   - 當你對某件物質（如手機、名車）或自己的外貌產生強烈執著時，試著思考：這件東西在十年前不存在，十年後必然毀壞。
   - 意識到身體的衰老與病痛是必然過程 ➔ 降低對物質恆久不變的幻想 ➔ 減輕失去時的痛苦。

2. **心理層面的無常觀察（受、想、行、識）**：
   - **觀察感受（受）**：當強烈的憤怒或悲傷襲來時，告訴自己：「這是一個『受』的現象，它正在升起，也必然會消逝。」
   - **觀察認知（想）**：注意到自己對某人的刻板印象（想）會隨著新資訊而改變 ➔ 意識到認知並非絕對真理。
   - **觀察意識（識）**：觀察念頭的跳躍，發現意識像流水般不斷變換，沒有一個固定不變的「我」在思考。

3. **轉念技巧**：
   - 每當產生「我想要（欲貪）」的衝動時，立即將其轉化為「觀察對象」。
   - 將焦點從「我想要這個」 ➔ 轉向「這個被我想要的對象是無常的」 ➔ 最終達到「既然無常，執著便無意義」的心理狀態。

### [段落大意]
本段經文闡述了透過「正思惟」觀察五蘊（色、受、想、行、識）之無常，能有效截斷對身心的欲貪，進而達成心靈解脫。當修行者能自證輪迴結束、修行圓滿時，即達到了不再受後有的境界。此邏輯同樣適用於對苦、空、無我的體悟。
