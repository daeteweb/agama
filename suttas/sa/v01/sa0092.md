# 雜阿含經 第 92 經

### [原文]
如是我聞：

一時，佛在拘薩羅人間遊行，至舍衛國祇樹給孤獨園。

爾時，舍衛國有憍慢婆羅門止住，父母種姓俱淨，無瑕點能說者，七世相承悉皆清淨；為婆羅門師，言論通達，諸論記典悉了萬名，解法優劣，分明諸，句句記說，容貌端正。或生志高、族姓志高、容色志高、聰明志高、財富志高，不敬父母、諸尊、師長。聞沙門瞿曇在拘薩羅國人間遊行，至舍衛國祇樹給孤獨園。聞已，作是念：「我當往彼沙門瞿曇所，若有所說，我當共論；無所說者，默然而還。」

時，憍慢婆羅門乘白馬車，諸年少婆羅門前後導從，持金柄傘蓋，手執金瓶，往見世尊。至於園門，下車步進。

爾時，世尊與諸大眾圍遶說法，不時顧念憍慢婆羅門。

時，憍慢婆羅門作是念：「沙門瞿曇不顧念我，且當還去。」

爾時，世尊知憍慢婆羅門心念，而說偈言：

「憍慢既來此，　　不善更增慢， 　向以義故來，　　應轉增其義。」

時，憍慢婆羅門作是念：「沙門瞿曇已知我心。」欲修敬禮。

爾時，世尊告憍慢婆羅門：「止！止！不須作禮，心淨已足。」

時，諸大眾咸各高聲唱言：「奇哉！世尊！大德大力。今此憍慢婆羅門恃生憍慢、族姓憍慢、容色憍慢、聰明憍慢、財富憍慢，不敬父母、諸尊、師長，今於沙門瞿曇所謙卑下下，欲接足禮。」

時，憍慢婆羅門於大眾前唱令靜默，而說偈言：

「云何不起慢？　　云何起恭敬？ 　云何善慰諭？　　云何善供養？」

爾時，世尊說偈答言：

「父母及長兄，　　和尚諸師長， 　及諸尊重者，　　所不應生慢。 　應當善恭敬，　　謙下而問訊， 　盡心而奉事，　　兼設諸供養。 　離貪恚癡心，　　漏盡阿羅漢， 　正智善解脫，　　伏諸憍慢心。 　於此賢聖等，　　合掌稽首禮。」

爾時，世尊為憍慢婆羅門種種說法，示、教、照、喜。如佛世尊次第說法，說布施、持戒、生天功德，愛欲味、患，煩惱清淨，出要遠離，諸清淨分……如是廣說，如白淨衣無諸黑惡，速受染色。憍慢婆羅門即於座上解四聖諦——苦、習、滅、道，得無間等。

時，憍慢婆羅門見法、得法、知法、入法，度諸疑惑，不由他度，於正法中得無所畏。即從座起，整衣服，為佛作禮，合掌白佛：「我今可得於正法中出家、受具足不？」

佛告憍慢婆羅門：「汝今可得於正法中出家、受具足。」

彼即出家，獨正思惟：「所以善男子剃除鬚髮，著袈裟衣，正信非家，出家學道……。」得阿羅漢，心善解脫。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛在拘薩羅人間遊行，至舍衛國祇樹給孤獨園。**
　　在某一個時間，佛在拘薩羅國的人間遊行，來到舍衛國的祇樹給孤獨園。

**◈ 爾時，舍衛國有憍慢婆羅門止住，父母種姓俱淨，無瑕點能說者，七世相承悉皆清淨；為婆羅門師，言論通達，諸論記典悉了萬名，解法優劣，分明諸，句句記說，容貌端正。**
　　那時，舍衛國有一位傲慢的婆羅門居住在那裡。他的父母與種姓都非常純淨，沒有任何瑕疵可被指責，且這種清淨的血統已連續承襲了七代；他是一位婆羅門教師，言談通暢，對各種論著典籍的名稱瞭若指掌，能分辨法義的優劣，對每一句論述都能分明地解釋，且容貌端正。

**◈ 或生志高、族姓志高、容色志高、聰明志高、財富志高，不敬父母、諸尊、師長。**
　　然而他因為自己的出生、族姓、容貌、聰明才智以及財富而心生傲慢，因此不尊敬父母、尊長與師長。

**◈ 聞沙門瞿曇在拘薩羅國人間遊行，至舍衛國祇樹給孤獨園。聞已，作是念：「我當往彼沙門瞿曇所，若有所說，我當共論；無所說者，默然而還。」**
　　他聽說沙門瞿曇在拘薩羅國人間遊行，並來到了舍衛國的祇樹給孤獨園。聽聞之後，他心想：「我應該去那位沙門瞿曇那裡，如果他有值得討論的法義，我就與他共同論辯；如果他沒有什麼可說的，我就默默地回去。」

**◈ 時，憍慢婆羅門乘白馬車，諸年少婆羅門前後導從，持金柄傘蓋，手執金瓶，往見世尊。至於園門，下車步進。**
　　於是，這位傲慢的婆羅門乘坐著白馬車，由許多年輕的婆羅門在前後隨從導引，隨從們持有金柄的傘蓋，手中拿著金瓶，前去見世尊。到達園門後，他才下車步行進入。

**◈ 爾時，世尊與諸大眾圍遶說法，不時顧念憍慢婆羅門。**
　　那時，世尊正被眾多大眾圍繞著說法，並沒有立刻注意到這位傲慢的婆羅門。

**◈ 時，憍慢婆羅門作是念：「沙門瞿曇不顧念我，且當還去。」**
　　這時，傲慢的婆羅門心想：「沙門瞿曇竟然不理會我，我還是回去吧。」

**◈ 爾時，世尊知憍慢婆羅門心念，而說偈言：「憍慢既來此，不善更增慢，向以義故來，應轉增其義。」**
　　那時，世尊知道這位傲慢婆羅門的心念，於是說了一首偈頌：「既然傲慢的人已經來到這裡，就不應該再增加不善的傲慢心；既然當初是為了追求真理的義理而來，就應該讓這份對真理的領悟增加。」

**◈ 時，憍慢婆羅門作是念：「沙門瞿曇已知我心。」欲修敬禮。**
　　這時，傲慢的婆羅門心想：「沙門瞿曇已經知道我的心思了。」於是想要行敬禮之禮。

**◈ 爾時，世尊告憍慢婆羅門：「止！止！不須作禮，心淨已足。」**
　　那時，世尊告訴這位傲慢的婆羅門：「停止！停止！不需要行禮，只要內心清淨就足夠了。」

**◈ 時，諸大眾咸各高聲唱言：「奇哉！世尊！大德大力。今此憍慢婆羅門恃生憍慢、族姓憍慢、容色憍慢、聰明憍慢、財富憍慢，不敬父母、諸尊、師長，今於沙門瞿曇所謙卑下下，欲接足禮。」**
　　這時，在場的大眾都高聲讚嘆道：「太奇妙了！世尊！您真是大德大力。這位婆羅門先前憑藉著出生、族姓、容貌、聰明與財富而心生傲慢，不尊敬父母、尊長與師長，如今在沙門瞿曇面前卻如此謙卑，甚至想要行接足禮（最恭敬的禮節）。」

**◈ 時，憍慢婆羅門於大眾前唱令靜默，而說偈言：「云何不起慢？云何起恭敬？云何善慰諭？云何善供養？」**
　　這時，傲慢的婆羅門在眾人面前請大家靜默，然後說偈詢問：「如何才能不起傲慢心？如何才能生起恭敬心？如何才能善於安慰勸諭他人？如何才能善於供養？」

**◈ 爾時，世尊說偈答言：「父母及長兄，和尚諸師長，及諸尊重者，所不應生慢。應當善恭敬，謙下而問訊，盡心而奉事，兼設諸供養。」**
　　那時，世尊用偈頌回答道：「對於父母、長兄、導師以及所有值得尊敬的人，都不應該生起傲慢心。應該對他們保持善意的恭敬，謙卑地問候，盡心盡力地侍奉，並提供適當的供養。」

**◈ 「離貪恚癡心，漏盡阿羅漢，正智善解脫，伏諸憍慢心。於此賢聖等，合掌稽首禮。」**
　　「而對於那些已經遠離貪愛、瞋恚與愚癡之心，將煩惱之漏完全除盡的阿羅漢（證得解脫的聖者），他們擁有正確的智慧並獲得了善的解脫，完全伏住了所有傲慢心。對於這些賢聖之人，應該合掌頂禮。」

**◈ 爾時，世尊為憍慢婆羅門種種說法，示、教、照、喜。**
　　那時，世尊為這位婆羅門說了種種法義，給予啟示、教導、照耀其心並令其歡喜。

**◈ 如佛世尊次第說法，說布施、持戒、生天功德，愛欲味、患，煩惱清淨，出要遠離，諸清淨分……如是廣說，如白淨衣無諸黑惡，速受染色。**
　　佛世尊循序漸進地說法，先說布施、持戒以及由此獲得生在天界的功德，接著說明愛欲的快感及其隱藏的患害，以及如何清除煩惱（內心的染污與執著）以達到清淨，進而說明出離世間的必要性與遠離之法，以及各種清淨的修行階段……如此廣泛地開示，就像一件沒有污垢的純白衣服，能夠迅速地染上顏色。

**◈ 憍慢婆羅門即於座上解四聖諦——苦、習、滅、道，得無間等。**
　　這位婆羅門就在座上領悟了四聖諦，即：苦（生命中不可避免的苦難）、習（導致苦難的貪愛之因）、滅（苦難完全止息的狀態）、道（通往苦止的解脫之道），並獲得了不間斷的禪定等覺悟。

**◈ 時，憍慢婆羅門見法、得法、知法、入法，度諸疑惑，不由他度，於正法中得無所畏。**
　　這時，這位婆羅門親見法義、獲得法義、了知法義並進入法義之中，超越了所有疑惑，且不需要他人引导而自行覺悟，在正法（正確的真理）中獲得了無所畏懼的自信。

**◈ 即從座起，整衣服，為佛作禮，合掌白佛：「我今可得於正法中出家、受具足不？」**
　　他立即從座位上站起，整理衣服，向佛陀行禮，合掌對佛陀恭敬地說道：「我現在可以在正法中出家，受具足戒（成為正式的比丘）嗎？」

**◈ 佛告憍慢婆羅門：「汝今可得於正法中出家、受具足。」**
　　佛告訴這位婆羅門：「你現在可以在正法中出家，受具足戒。」

**◈ 彼即出家，獨正思惟：「所以善男子剃除鬚髮，著袈裟衣，正信非家，出家學道……。」得阿羅漢，心善解脫。**
　　他隨即出家，獨自正思惟：「為什麼善男子要剃除鬚髮，穿著袈裟，以正信之心離開家庭，出家學習道業……」最終他證得阿羅漢果，心靈獲得了善的解脫。

### [詞語註解（解釋重要佛教概念）]
1. **憍慢 (Māna)**：指一種自大、傲慢的心態，認為自己比他人優越。在佛教中，慢被視為一種煩惱，會阻礙對真理的領悟。
2. **四聖諦 (Cattāri Ariyasaccāni)**：佛教的核心教義。
    - **苦 (Dukkha)**：指生命本質上的不滿足與痛苦。
    - **集 (Samudaya)**：指苦的根源，即對感官欲望的貪愛。
    - **滅 (Nirodha)**：指苦的止息，即涅槃境界。
    - **道 (Magga)**：指消除苦的具體路徑，即八正道。
3. **漏盡 (Āsavakkhaya)**：指將「漏」（Asava，指滲出的煩惱，如欲漏、有漏、無明漏）完全清除。漏盡是證得阿羅漢果的標誌。
4. **阿羅漢 (Arahant)**：指已完全斷除煩惱、不再輪迴、證得解脫的聖者。
5. **正法 (Saddhamma)**：指佛陀所開示的正確、真實且能導向解脫的真理。

### [現代實作指南]
本經文揭示了「傲慢」如何成為學習的障礙，以及如何透過「謙卑」與「正確的對象」來轉化自我。

**1. 識別「身分認同」的陷阱：**
婆羅門的傲慢來自於五個維度：出生、血統、容貌、才智、財富。在現代生活中，這對應於：學歷、職位、外貌、專業技能、經濟能力。
➔ **實作技巧**：當你感到自己比他人優越時，試著提醒自己，這些條件（色、想、行）皆是「緣起」而生，是暫時的組合，而非永恆的「我」。

**2. 練習「心淨」而非「形式禮儀」：**
佛陀告訴婆羅門「心淨已足」，強調內在的謙卑高於外在的儀式。
➔ **實作技巧**：在與他人互動時，不追求表面的客套，而是在心中先建立「對方也是我的老師」的覺知，將對立的競爭心轉化為學習心。

**3. 建立正確的恭敬對象：**
經文區分了兩種恭敬：一種是對長輩、師長的社會倫理恭敬（基於感恩與秩序）；另一種是對覺悟者（阿羅漢）的真理恭敬（基於對解脫的嚮往）。
➔ **實作技巧**：在生活中區分「權威」與「智慧」。對權威保持禮貌，但對真正的智慧（能讓自己內心平靜、減少貪嗔的人）保持深切的敬意與學習態度。

**4. 循序漸進的轉念過程：**
佛陀對婆羅門的教導是「次第」的：從布施 → 持戒 → 功德 → 覺察欲望 → 遠離煩惱 → 領悟四聖諦。
➔ **實作技巧**：不要強求立即達到最高境界。先從簡單的「給予」（布施）和「自律」（持戒）開始，讓心靈逐漸純淨，才能在面對複雜問題時，清晰地看到問題的根源（集）與解決之道（道）。

### [段落大意]
本經描述一位擁有極高社會地位與才華但極其傲慢的婆羅門，在與佛陀相遇後，被佛陀以智慧洞察其心，使其由傲慢轉為謙卑。佛陀教導他應對長輩及聖者保持恭敬，並循序漸進地引導他從布施、持戒到領悟四聖諦。最終，該婆羅門徹底除盡煩惱，出家證得阿羅漢果，實現了心靈的完全解脫。
