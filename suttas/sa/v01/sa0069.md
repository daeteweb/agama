# 雜阿含經 第 69 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「我今當說有身集趣道及有身集滅道。云何有身集趣道？愚癡無聞凡夫，見不如實知色集、色滅、色味、色患、色離。不如實知故，樂色、歎色、著色、住色；樂色、歎色、著色、住色故，愛樂取；緣取有，緣有生，緣生、老、病、死、憂、悲、苦、惱。如是純大苦聚生。如是受、想、行、識廣說，是名有身集趣道。比丘！有身集趣道，當知即是苦集趣道。

「云何有身集滅道？多聞聖弟子如實知色、色集、色滅、色味、色患、色離。如實知故，於色不樂、不歎、不著、不住；不樂、不歎、不著、不住故，彼色愛樂滅；愛樂滅則取滅，取滅則有滅，有滅則生滅，生滅則老、病、死、憂、悲、苦、惱，純大苦聚滅。如色，受、想、行、識亦如是，是名有身滅道跡。有身滅道跡，則是苦滅道跡，是故說有身滅道跡。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如當說，有及當知，亦如是說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「我今當說有身集趣道及有身集滅道。**
　　那時，世尊告訴在場的眾位比丘說：「我現在要為你們解說導致身心存在而聚集的道路（有身集趣道），以及使身心存在而滅盡的道路（有身集滅道）。

**◈ 云何有身集趣道？**
　　什麼是導致身心存在而聚集的道路呢？

**◈ 愚癡無聞凡夫，見不如實知色集、色滅、色味、色患、色離。**
　　那些愚癡且缺乏聞法機會的凡夫，在觀察時不能如實地認知色（物質身體）是如何聚集、如何滅除、其吸引之處（味）、其缺陷（患）以及如何遠離。

**◈ 不如實知故，樂色、歎色、著色、住色；**
　　因為不能如實認知，所以會喜愛色（物質身體）、讚歎色、執著於色、並停留於色之中。

**◈ 樂色、歎色、著色、住色故，愛樂取；**
　　因為喜愛、讚歎、執著並停留於色（物質身體），所以產生了貪愛與執取。

**◈ 緣取有，緣有生，緣生、老、病、死、憂、悲、苦、惱。**
　　以執取為緣而產生存在（有），以存在為緣而產生出生（生），進而以出生為緣，產生老、病、死、憂、悲、苦、惱。

**◈ 如是純大苦聚生。**
　　就這樣，純粹且巨大的苦之聚集便產生了。

**◈ 如是受、想、行、識廣說，是名有身集趣道。**
　　對於受（感受）、想（想像與認知）、行（心的造作）、識（意識）也同樣地廣泛說明，這就稱為導致身心存在而聚集的道路。

**◈ 比丘！有身集趣道，當知即是苦集趣道。**
　　比丘們！應當了解，導致身心存在而聚集的道路，也就是導致苦之聚集的道路。

**◈ 云何有身集滅道？**
　　什麼是使身心存在而滅盡的道路呢？

**◈ 多聞聖弟子如實知色、色集、色滅、色味、色患、色離。**
　　聽聞教法多且具備智慧的聖弟子，能如實地認知色（物質身體）是如何聚集、如何滅除、其吸引之處（味）、其缺陷（患）以及如何遠離。

**◈ 如實知故，於色不樂、不歎、不著、不住；**
　　因為能如實認知，所以對於色（物質身體）不產生喜愛、不讚歎、不執著、也不停留。

**◈ 不樂、不歎、不著、不住故，彼色愛樂滅；**
　　因為不喜愛、不讚歎、不執著、不停留，所以對於色（物質身體）的貪愛與喜樂隨之滅除。

**◈ 愛樂滅則取滅，取滅則有滅，有滅則生滅，生滅則老、病、死、憂、悲、苦、惱，純大苦聚滅。**
　　貪愛滅除 ➔ 執取滅除 ➔ 存在（有）滅除 ➔ 出生（生）滅除 ➔ 老、病、死、憂、悲、苦、惱以及純粹巨大的苦聚全部滅除。

**◈ 如色，受、想、行、識亦如是，是名有身滅道跡。**
　　對於受（感受）、想（想像與認知）、行（心的造作）、識（意識）也同樣如此，這就稱為使身心存在而滅盡的道路跡象。

**◈ 有身滅道跡，則是苦滅道跡，是故說有身滅道跡。」**
　　使身心存在而滅盡的道路跡象，就是苦之滅除的道路跡象，因此我才說明使身心存在而滅盡的道路跡象。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如當說，有及當知，亦如是說。**
　　如同應當說的、應當知道的，確實就是這樣說的。

### [詞語註解（解釋重要佛教概念）]
1. **色 (Rūpa)**：指物質身體及其構成的物質現象。
2. **受 (Vedanā)**：指對刺激產生的感受（苦、樂、不苦不樂）。
3. **想 (Saññā)**：指對對象的辨識、標記與想像認知。
4. **行 (Saṅkhāra)**：指心靈的造作、意志驅動的心理活動。
5. **識 (Viññāṇa)**：指對對象的覺知或意識。
6. **十二因緣 (Paṭicca-samuppāda)**：描述生命如何因無明而輪迴的連鎖反應。本經重點在於「取 ➔ 有 ➔ 生 ➔ 老死」的因果鏈條。
7. **如實知**：指不被主觀偏見、貪愛或厭惡所遮蔽，客觀地觀察現象的本質（無常、苦、無我）。

### [現代實作指南]
本經提供了一套極其精準的「認知解構法」，可用於處理現代生活中的執著與情緒痛苦：

1. **實作步驟：五階段分析法**
   當你對某個對象（如：名聲、財富、一段關係、甚至自己的身體外貌）產生強烈執著時，請依序進行以下思考：
   - **分析聚集 (集)**：思考這個對象是如何由各種條件組合而成的？（例如：美貌是由基因、保養、光線、他人認知組合而成）。
   - **分析滅除 (滅)**：意識到這個對象必然會隨著時間或條件改變而消失（無常）。
   - **分析吸引力 (味)**：誠實面對它帶給你的快感是什麼？（例如：被讚美時的虛榮心）。
   - **分析缺陷 (患)**：思考維持這個對象需要付出什麼代價？失去它會帶來什麼痛苦？（例如：為了維持美貌而產生的焦慮）。
   - **分析遠離 (離)**：思考如何從對這個對象的依賴中解脫，獲得真正的寧靜。

2. **轉念技巧：從「我擁有」到「現象在運作」**
   - **錯誤認知**：「我的身體很老了，我很痛苦。」（執著於色 → 產生取 → 產生苦）。
   - **如實認知**：「色（物質身體）正在經歷衰老這個自然過程 → 這是色之患 → 我不需要對這個過程產生執著 → 痛苦隨之減輕。」

3. **操作目標**：
   透過將「喜愛 → 讚歎 → 執著 → 停留」這個心理慣性，轉化為「觀察 → 理解 → 放下 → 自由」，從而切斷苦的產生鏈條。

### [段落大意]
本經對比了凡夫與聖弟子對「五蘊」的不同認知路徑。凡夫因不能如實認知物質與精神現象的本質，產生貪愛與執取，導致輪迴生死與巨大的苦聚；而聖弟子則能如實觀察五蘊的聚集、滅除、吸引力、缺陷與遠離，從而滅除貪愛與執取，最終達成苦的滅盡與解脫。
