# 雜阿含經 第 209 經

### [原文]
如是我聞：
一時，佛住毘舍離耆婆拘摩羅藥師菴羅園。
爾時，世尊告諸比丘：「有六觸入處。云何為六？眼觸入處，耳、鼻、舌、身、意觸入處。沙門、婆羅門於此六觸入處集、滅、味、患、離不如實知，當知是沙門、婆羅門去我法、律遠，如虛空與地。」
時，有異比丘從坐起，整衣服，為佛作禮，合掌白佛言：「我具足如實知此六觸入處集、滅、味、患、離。」
佛告比丘：「我今問汝，汝隨問答我。比丘！汝見眼觸入處是我、異我、相在不？」
答言：「不也，世尊！」
佛告比丘：「善哉！善哉！於此眼觸入處非我、非異我、不相在，如實知見者，不起諸漏、心不染著、心得解脫，是名初觸入處已斷、已知，斷其根本，如截多羅樹頭，於未來法永不復起，所謂眼識及色。汝見耳、鼻、舌、身、意觸入處是我、異我、相在不？」
答言：「不也，世尊！」
佛告比丘：「善哉！善哉！於耳、鼻、舌、身、意觸入處非我、非異我、不相在，作如是如實知見者，不起諸漏、心不染著，以得解脫，是名比丘六觸入處已斷、已知，斷其根本，如截多羅樹頭，於未來世欲不復生，謂意識法。」
佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我如此聽聞：

**◈ 一時，佛住毘舍離耆婆拘摩羅藥師菴羅園。**
　　一時，佛住在毘舍離城的拘摩羅藥師菴羅園。

**◈ 爾時，世尊告諸比丘：「有六觸入處。云何為六？眼觸入處，耳、鼻、舌、身、意觸入處。**
　　那時，世尊告訴諸位比丘：「存在著六種感官接觸的入口（六觸入處）。所謂六種是什麼呢？分別是：眼睛的接觸入口、耳朵的接觸入口、鼻子的接觸入口、舌頭的接觸入口、身體的接觸入口，以及心智的接觸入口。」

**◈ 沙門、婆羅門於此六觸入處集、滅、味、患、離不如實知，當知是沙門、婆羅門去我法、律遠，如虛空與地。」**
　　「若沙門或婆羅門對於這六種接觸入口的生起（集）、熄滅（滅）、滋味（味）、危害（患）以及捨離（離）不能如實知見，那麼可以知道這些沙門或婆羅門正遠離我的法與律，猶如虛空與大地般遙不可及。」

**◈ 時，有異比丘從坐起，整衣服，為佛作禮，合掌白佛言：「我具足如實知此六觸入處集、滅、味、患、離。」**
　　當時，有一位比丘從座位上站起，整理好衣服，向佛行禮，合掌對佛說道：「我已具備了對於這六種接觸入口的生起、熄滅、滋味、危害與捨離之如實知見。」

**◈ 佛告比丘：「我今問汝，汝隨問答我。比丘！汝見眼觸入處是我、異我、相在不？」**
　　佛告訴這位比丘：「我現在要詢問你，請你隨我的問題作答。比丘！你認為眼睛的接觸入口是『我（自我）』、『非我（與我不同）』或是『與我相同（與自我重疊）』嗎？」

**◈ 答言：「不也，世尊！」**
　　回答說：「不是，世尊！」

**◈ 佛告比丘：「善哉！善哉！於此眼觸入處非我、非異我、不相在，如實知見者，不起諸漏、心不染著、心得解脫，是名初觸入處已斷、已知，斷其根本，如截多羅樹頭，於未來法永不復起，所謂眼識及色。**
　　佛告訴比丘：「善哉！善哉！對於這眼睛的接觸入口，若能如實知見其『非我（不是自我）』、『非異我（與自我無關聯）』且『不相在（不與自我重疊）』，便不會產生諸漏（煩惱流），心不再染著，並獲得解脫。這就稱為初步的接觸入口已經被斷除且被透徹認知，從根本上切斷了它，就像砍斷多羅樹的頂端一樣，在未來的法中永不再次生起，也就是所謂的眼識（意識）以及色（物質對象）。

**◈ 汝見耳、鼻、舌、身、意觸入處是我、異我、相在不？」**
　　「那麼，你認為耳朵、鼻子、舌頭、身體、心智的接觸入口是『我』、『非我』或是『與我相同』嗎？」

**◈ 答言：「不也，世尊！」**
　　回答說：「不是，世尊！」

**◈ 佛告比丘：「善哉！善哉！於耳、鼻、舌、身、意觸入處非我、非異我、不相在，作如是如實知見者，不起諸漏、心不染著，以得解脫，是名比丘六觸入處已斷、已知，斷其根本，如截多羅樹頭，於未來世欲不復生，謂意識法。」**
　　佛告訴比丘：「善哉！善哉！對於耳朵、鼻子、舌頭、身體、心智的接觸入口，若能作如是『非我、非異我、不相在』的如實知見，便不會產生諸漏（煩惱流），心不再染著並獲得解脫。這就稱為比丘已斷除並透徹認知了六種接觸入口，從根本上切斷了它們，如同砍斷多羅樹的頂端一樣，在未來的世間不再重新出生，即是指意識法（意識的運作）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛說完這部經後，諸位比丘聽聞了佛陀的教誨，皆感到歡喜並遵從修行。

### [詞語註解（解釋重要佛教概念）]
1. **六觸入處 (Saḷāyatana)**：指感官與外界接觸的六個通道（眼、耳、鼻、舌、身、意），是經驗生起的基礎。
2. **集、滅、味、患、離 (Samudaya, Nirodha, Rasa, Dukkha, Virāga)**：對感官經驗的五種深度認知：其生起過程（集）、其熄滅狀態（滅）、其帶來的快感或體驗（味）、其隱含的痛苦與不安（患）、以及如何從中脫離（離）。
3. **諸漏 (Āsava)**：指使心靈不斷流向輪迴、束縛生命的煩惱流（如貪愛、嗔恚、無明等）。
4. **非我、非異我、不相在 (Anattā)**：這是對「無我」的精確描述。指現象既不是一個獨立的自我，也不是與自我完全分離的外在物，更不是兩者重疊的混合體，強調其緣起性。
5. **多羅樹 (Dāradu tree)**：一種比喻，形容斷除煩惱如同砍斷樹木頂端，使其不再向上生長，象徵從根本切斷因果鏈結。

### [現代實作指南]
本經的核心在於「解構感官經驗與自我認同的連結」。當我們看到、聽到或感覺到某種事物時，大腦會自動產生「這是我的」、「我喜歡這個」或「這是我感受到的」這種強烈的自我中心化反應。

**實踐技巧：**
1. **觀察間隙（The Gap）**：當你遇到一個強烈的感官刺激（例如看到一張令你憤怒的臉，或聽到一段悅耳的音樂）時，試著在「感覺生起」與「產生情緒/認同」之間製造一個微小的停頓。
2. **去標籤化練習**：
   - 不要說：「我感到很痛苦。」
   - 改為觀察：「身體（色）正在產生一種不適的觸碰感，而意識（識）正在對此進行反應。」
3. **三維度檢驗法**：當你覺得某種情緒或感覺「就是我」時，運用經文中的邏輯進行反思：
   - 「這個感覺是我的核心嗎？」（非我）
   - 「這個感覺與我完全無關嗎？」（非異我）
   - 「這個感覺是否等於我？」（不相在）
4. **目標**：透過這種觀察，將感官經驗視為「純粹的物理/心理現象」，而非「自我的延伸」，從而減少因執著於感官體驗（味、患）而產生的情緒波動。

### [段落大意]
佛陀指出，若不能透徹理解六種感官接觸入口（眼耳鼻舌身意）的生起、熄滅、滋味、危害與捨離，便無法接近正法。透過比丘與佛陀的問答，確立了感官經驗具有「非我、非異我、不相在」的特質。若能如實知見此理，即可斷除煩惱流（諸漏），不再受感官驅使而產生執著，從根本上切斷輪迴中意識與物質對象的連鎖反應，獲得解脫。
