# 雜阿含經 第 81 經

### [原文]
如是我聞：

一時，佛住毘耶離獼猴池側重閣講堂。

爾時，有離車名摩訶男，日日遊行，往詣佛所。時，彼離車作是念：「若我早詣世尊所者，世尊及我知識比丘皆悉禪思，我今當詣七菴羅樹阿耆毘外道所。」即往詣彼富蘭那迦葉住處。

時，富蘭那迦葉——外道眾主，與五百外道前後圍遶，高聲嬉戲，論說俗事。時，富蘭那迦葉遙見離車摩訶男來，告其眷屬，令寂靜住：「汝等默然！是離車摩訶男是沙門瞿曇弟子，此是沙門瞿曇白衣弟子，毘耶離中最為上首，常樂靜寂，讚歎寂靜，彼所之詣寂靜之眾，是故汝等應當寂靜。」

時，摩訶男詣彼眾富蘭那所，與富蘭那共相問訊，相慰勞已，却坐一面。時，摩訶男語富蘭那言：「我聞富蘭那為諸弟子說法：『無因、無緣眾生有垢，無因、無緣眾生清淨。』世有此論，汝為審有此，為是外人相毀之言？世人所撰，為是法、為非法，頗有世人共論、難問、嫌責以不？」

富蘭那迦葉言：「實有此論，非世妄傳。我立此論，是如法論，我說此法，皆是順法，無有世人來共難問而呵責者。所以者何？摩訶男！我如是見、如是說：『無因、無緣眾生有垢，無因、無緣眾生清淨。』」

時，摩訶男聞富蘭那所說，心不喜樂，呵罵已，從坐起去，向世尊所，頭面禮足，却坐一面，以向與富蘭那所論事，向佛廣說。

佛告離車摩訶男：「彼富蘭那為出意語，不足記也。如是富蘭那愚癡，不辨、不善、非因而作是說：『無因、無緣眾生有垢，無因、無緣眾生清淨。』所以者何？有因、有緣眾生有垢，有因、有緣眾生清淨。

「摩訶男！何因、何緣眾生有垢，何因、何緣眾生清淨？摩訶男！若色一向是苦、非樂、非隨樂、非樂長養、離樂者，眾生不應因此而生樂著。摩訶男！以色非一向是苦、非樂，隨樂、樂所長養、不離樂，是故眾生於色染著；染著故繫，繫故有惱。摩訶男！若受、想、行、識，一向是苦、非樂、非隨樂、非樂長養、離樂者，眾生不應因此而生樂著。摩訶男！以識非一向是苦、非樂，隨樂、樂所長養、不離樂，是故眾生於識染著；染著故繫，繫故生惱。摩訶男！是名有因、有緣眾生有垢。

「摩訶男！何因、何緣眾生清淨？摩訶男！若色一向是樂、非苦、非隨苦、非憂苦長養、離苦者，眾生不應因色而生厭離。摩訶男！以色非一向樂，是苦、隨苦、憂苦長養、不離苦，是故眾生厭離於色；厭故不樂，不樂故解脫。摩訶男！若受、想、行、識，一向是樂、非苦、非隨苦、非憂苦長養、離苦者，眾生不應因識而生厭離。摩訶男！以受、想、行、識，非一向樂，是苦、隨苦、憂苦長養、不離苦，是故眾生厭離於識；厭故不樂，不樂故解脫。摩訶男！是名有因、有緣眾生清淨。」

時，摩訶男聞佛所說，歡喜隨喜，禮佛而退。

　知法及重擔　　往詣．觀．欲貪 　生及與略說　　法印．富蘭那

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住毘耶離獼猴池側重閣講堂。**
　　在某一個時間，佛陀住在毘耶離城的獼猴池旁的重閣講堂中。

**◈ 爾時，有離車名摩訶男，日日遊行，往詣佛所。**
　　那時，有一位名叫摩訶男的離車族人，每天在城中遊行，並前往佛陀那裡。

**◈ 時，彼離車作是念：「若我早詣世尊所者，世尊及我知識比丘皆悉禪思，我今當詣七菴羅樹阿耆毘外道所。」即往詣彼富蘭那迦葉住處。**
　　當時，這位離車族人心裡想：「如果我太早去世尊那裡，世尊以及我認識的比丘們可能都在禪定思惟中，我現在應該先去七棵菴羅樹那邊，拜訪阿耆毘外道。」於是，他前往富蘭那迦葉的住處。

**◈ 時，富蘭那迦葉——外道眾主，與五百外道前後圍遶，高聲嬉戲，論說俗事。**
　　那時，作為外道領袖的富蘭那迦葉，被五百名外道弟子簇擁在前後，正大聲地嬉戲，討論著世俗的瑣事。

**◈ 時，富蘭那迦葉遙見離車摩訶男來，告其眷屬，令寂靜住：「汝等默然！是離車摩訶男是沙門瞿曇弟子，此是沙門瞿曇白衣弟子，毘耶離中最為上首，常樂靜寂，讚歎寂靜，彼所之詣寂靜之眾，是故汝等應當寂靜。」**
　　那時，富蘭那迦葉從遠處看到離車族摩訶男走來，便告訴他的隨從們，要求他們保持安靜：「你們都安靜！這位離車族摩訶男是沙門瞿曇的弟子，他是沙門瞿曇最優秀的在家白衣弟子，在毘耶離城中地位最高。他平時喜好靜寂，讚歎靜寂，他所拜訪的通常也是追求靜寂的人，所以你們現在應該保持安靜。」

**◈ 時，摩訶男詣彼眾富蘭那所，與富蘭那共相問訊，相慰勞已，却坐一面。**
　　於是，摩訶男走到富蘭那及其眾人面前，與富蘭那互相問候、慰問彼此的近況後，便坐在一旁。

**◈ 時，摩訶男語富蘭那言：「我聞富蘭那為諸弟子說法：『無因、無緣眾生有垢，無因、無緣眾生清淨。』**
　　這時，摩訶男對富蘭那說：「我聽說富蘭那你對弟子們宣說這樣的法義：『眾生之所以產生染污（垢），或是變得清淨，並非因為任何原因或條件（無因無緣）而隨機發生的。』」

**◈ 世有此論，汝為審有此，為是外人相毀之言？世人所撰，為是法、為非法，頗有世人共論、難問、嫌責以不？」**
　　「世上確實有這種論調嗎？你是否確實持有這個觀點，還是這只是外人毀謗你的話？這是世人編造的，究竟是正確的法，還是錯誤的法？是否有人對此共同討論、提出質詢或予以譴責？」

**◈ 富蘭那迦葉言：「實有此論，非世妄傳。我立此論，是如法論，我說此法，皆是順法，無有世人來共難問而呵責者。**
　　富蘭那迦葉回答：「確實有這種論調，並非世人妄傳。這是我建立的論點，是符合法理的論述，我所說的法都順應真理，從來沒有世人對此提出質疑或呵責。」

**◈ 所以者何？摩訶男！我如是見、如是說：『無因、無緣眾生有垢，無因、無緣眾生清淨。』」**
　　「為什麼呢？摩訶男！因為我的見解與說法就是：『眾生產生染污或獲得清淨，完全不需要任何原因或條件（無因無緣）。』」

**◈ 時，摩訶男聞富蘭那所說，心不喜樂，呵罵已，從坐起去，向世尊所，頭面禮足，却坐一面，以向與富蘭那所論事，向佛廣說。**
　　這時，摩訶男聽了富蘭那的說法，內心感到不悅，在呵斥對方後便起身離開。他前往世尊那裡，俯身頂禮佛陀的足部，然後坐在一旁，將剛才與富蘭那討論的事情詳細地告訴佛陀。

**◈ 佛告離車摩訶男：「彼富蘭那為出意語，不足記也。**
　　佛陀告訴離車族摩訶男：「那個富蘭那說的是出格且荒謬的話，不值得記錄或在意。」

**◈ 如是富蘭那愚癡，不辨、不善、非因而作是說：『無因、無緣眾生有垢，無因、無緣眾生清淨。』**
　　「富蘭那如此愚癡，無法辨別、不善思考，且在沒有正確根據的情況下說出：『眾生產生染污或獲得清淨，是無因無緣的』這種話。」

**◈ 所以者何？有因、有緣眾生有垢，有因、有緣眾生清淨。**
　　「為什麼這麼說？因為眾生之所以產生染污，以及之所以變得清淨，都是有其特定的原因與條件（因緣而生）的。」

**◈ 「摩訶男！何因、何緣眾生有垢，何因、何緣眾生清淨？**
　　「摩訶男！究竟是什麼原因與條件導致眾生有染污？又是什麼原因與條件導致眾生清淨？」

**◈ 摩訶男！若色一向是苦、非樂、非隨樂、非樂長養、離樂者，眾生不應因此而生樂著。**
　　「摩訶男！如果色（物質身體）完全只有痛苦，沒有任何快樂，不伴隨快樂，沒有快樂的滋養，且完全脫離快樂，那麼眾生就不會因為色（物質身體）而產生對快樂的執著。」

**◈ 摩訶男！以色非一向是苦、非樂，隨樂、樂所長養、不離樂，是故眾生於色染著；染著故繫，繫故有惱。**
　　「但摩訶男！因為色（物質身體）並非完全只有痛苦，它包含快樂，伴隨著快樂，被快樂所滋養，且不脫離快樂，所以眾生才會對色（物質身體）產生染著；因為染著而受到束縛，因為束縛而產生煩惱。」

**◈ 摩訶男！若受、想、行、識，一向是苦、非樂、非隨樂、非樂長養、離樂者，眾生不應因此而生樂著。**
　　「摩訶男！如果受（感受）、想（想像與認知）、行（心的造作）、識（意識）完全只有痛苦，沒有快樂，不伴隨快樂，沒有快樂的滋養，且完全脫離快樂，那麼眾生就不會因此而產生對快樂的執著。」

**◈ 摩訶男！以識非一向是苦、非樂，隨樂、樂所長養、不離樂，是故眾生於識染著；染著故繫，繫故生惱。摩訶男！是名有因、有緣眾生有垢。**
　　「但因為識（意識）並非完全只有痛苦，它包含快樂，伴隨著快樂，被快樂所滋養，且不脫離快樂，所以眾生才會對識（意識）產生染著；因為染著而受到束縛，因為束縛而產生煩惱。摩訶男！這就叫做眾生產生染污是有其原因與條件的。」

**◈ 「摩訶男！何因、何緣眾生清淨？**
　　「摩訶男！那麼，是什麼原因與條件導致眾生清淨？」

**◈ 摩訶男！若色一向是樂、非苦、非隨苦、非憂苦長養、離苦者，眾生不應因色而生厭離。**
　　「摩訶男！如果色（物質身體）完全只有快樂，沒有痛苦，不伴隨痛苦，沒有憂苦的滋養，且完全脫離痛苦，那麼眾生就不會因為色（物質身體）而產生厭離心。」

**◈ 摩訶男！以色非一向樂，是苦、隨苦、憂苦長養、不離苦，是故眾生厭離於色；厭故不樂，不樂故解脫。**
　　「但摩訶男！因為色（物質身體）並非完全只有快樂，它包含痛苦，伴隨著痛苦，被憂苦所滋養，且不脫離痛苦，所以眾生才會對色（物質身體）產生厭離；因為厭離而不再追求其中的樂趣，因為不再追求樂趣而獲得解脫。」

**◈ 摩訶男！若受、想、行、識，一向是樂、非苦、非隨苦、非憂苦長養、離苦者，眾生不應因識而生厭離。**
　　「摩訶男！如果受（感受）、想（想像與認知）、行（心的造作）、識（意識）完全只有快樂，沒有痛苦，不伴隨痛苦，沒有憂苦的滋養，且完全脫離痛苦，那麼眾生就不會因為識（意識）而產生厭離心。」

**◈ 摩訶男！以受、想、行、識，非一向樂，是苦、隨苦、憂苦長養、不離苦，是故眾生厭離於識；厭故不樂，不樂故解脫。摩訶男！是名有因、有緣眾生清淨。」**
　　「但因為受（感受）、想（想像與認知）、行（心的造作）、識（意識）並非完全只有快樂，它們包含痛苦，伴隨著痛苦，被憂苦所滋養，且不脫離痛苦，所以眾生才會對識（意識）產生厭離；因為厭離而不再追求其中的樂趣，因為不再追求樂趣而獲得解脫。摩訶男！這就叫做眾生獲得清淨是有其原因與條件的。」

**◈ 時，摩訶男聞佛所說，歡喜隨喜，禮佛而退。**
　　這時，摩訶男聽了佛陀的教導，內心感到歡喜並隨喜，向佛陀行禮後才離開。

### [詞語註解（解釋重要佛教概念）]
1. **垢 (Kilesa)**：指染污心靈的煩惱，如貪、嗔、痴，使眾生無法看清真理而陷入輪迴。
2. **清淨 (Visuddhi)**：指心靈除去染污、恢復純淨的狀態，最終導向涅槃的解脫。
3. **無因、無緣 (Ahetu-apaccaya)**：一種錯誤的見解，認為事物的發生（如痛苦或解脫）是隨機的、無因果律的，否定緣起法。
4. **染著 (Upādāna)**：對五蘊（色、受、想、行、識）產生的強烈執著與抓取，是導致痛苦的核心原因。
5. **厭離 (Nibbidā)**：在體悟到現象的無常與苦後，產生的不再執著、不再追求的心理狀態，是解脫的必要過程。

### [現代實作指南]
本經文揭示了一個深刻的心理機制：**「快樂是陷阱，痛苦是階梯」**。我們可以將其轉化為以下生活實踐：

1. **覺察「快樂的誘餌」**：
   - 當我們對某件事物（如名聲、物質、一段關係）感到極度愉悅時，提醒自己：正是因為其中夾雜著「樂」，我才會產生「染著」。
   - 實作技巧：在感到快樂時，輕聲對自己說：「這個快樂是暫時的，它正吸引我產生執著，而執著最終會帶來束縛。」

2. **重新定義「痛苦的價值」**：
   - 當我們遭遇挫折或身體不適時，不要僅僅將其視為不幸，而要將其視為「厭離」的契機。
   - 實作技巧：在痛苦發生時，觀察這種不舒服的感覺，並體悟到：「既然這件事（或這個身體）不能提供恆久的快樂，反而帶來痛苦，那麼我不需要如此執著於它。」

3. **打破「隨機論」的迷思**：
   - 避免將生活中的好壞歸結為「運氣」或「命中注定」（即無因無緣）。
   - 實作技巧：練習追溯因果。當心中產生負面情緒（垢）時，分析其背後的「因」（如：對某事的期待）與「緣」（如：對方的反應），透過理性的分析來消解情緒。

### [段落大意]
本經記述離車族摩訶男質詢外道富蘭那關於「眾生染污與清淨是否無因無緣」的觀點。富蘭那主張兩者皆是隨機發生，佛陀則予以反駁，指出一切皆由因緣而生。佛陀解釋：眾生因五蘊中夾雜著「樂」而產生染著，導致煩惱（有因有緣之垢）；而因五蘊中夾雜著「苦」而產生厭離，最終導向解脫（有因有緣之清淨）。
