# 雜阿含經 第 79 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「過去、未來色尚無常，況復現在色！多聞聖弟子如是觀察已，不顧過去色，不欣未來色，於現在色厭、離欲、滅寂靜；受、想、行、識亦復如是。

「比丘！若無過去色者，多聞聖弟子無不顧過去色；以有過去色故，多聞聖弟子不顧過去色。若無未來色者，多聞聖弟子無不欣未來色；以有未來色故，多聞聖弟子不欣未來色。若無現在色者，多聞聖弟子不於現在色生厭、離欲、滅盡向；以欲現在色故，多聞聖弟子於現在色生厭、離欲、滅盡向。受、想、行、識亦如是說。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如無常，苦、空、非我三經，亦如是說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「過去、未來色尚無常，況復現在色！**
　　「過去的色（物質身體）與未來的色（物質身體）尚且是無常（變遷不定的），更不用說現在的色（物質身體）了！

**◈ 多聞聖弟子如是觀察已，不顧過去色，不欣未來色，於現在色厭、離欲、滅寂靜；**
　　博學的聖弟子在這樣觀察之後，不再眷戀過去的色（物質身體），不再期待未來的色（物質身體），對於現在的色（物質身體）產生厭離心，遠離貪欲，進入滅盡的寂靜狀態；

**◈ 受、想、行、識亦復如是。**
　　受（感受）、想（想像與認知）、行（心的造作）、識（意識）也同樣如此。

**◈ 「比丘！若無過去色者，多聞聖弟子無不顧過去色；**
　　「比丘們！如果沒有過去的色（物質身體）存在，博學的聖弟子就不會產生『不眷戀過去色』這種心態；

**◈ 以有過去色故，多聞聖弟子不顧過去色。**
　　正因為有過去的色（物質身體）存在，博學的聖弟子才會選擇不眷戀過去的色（物質身體）。

**◈ 若無未來色者，多聞聖弟子無不欣未來色；**
　　如果沒有未來的色（物質身體）存在，博學的聖弟子就不會產生『不期待未來色』這種心態；

**◈ 以有未來色故，多聞聖弟子不欣未來色。**
　　正因為有未來的色（物質身體）存在，博學的聖弟子才會選擇不期待未來的色（物質身體）。

**◈ 若無現在色者，多聞聖弟子不於現在色生厭、離欲、滅盡向；**
　　如果沒有現在的色（物質身體）存在，博學的聖弟子就不會對現在的色（物質身體）產生厭離、遠離貪欲以及追求滅盡的傾向；

**◈ 以欲現在色故，多聞聖弟子於現在色生厭、離欲、滅盡向。**
　　正因為對現在的色（物質身體）有貪欲，博學的聖弟子才會對現在的色（物質身體）生起厭離、遠離貪欲以及追求滅盡的傾向。

**◈ 受、想、行、識亦如是說。」**
　　受（感受）、想（想像與認知）、行（心的造作）、識（意識）也是這樣說的。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如無常，苦、空、非我三經，亦如是說。**
　　關於無常、苦、空、非我這三類經文，也是這樣說的。

### [詞語註解（解釋重要佛教概念）]
1. **色 (Rūpa)**：指物質身體及所有物質現象。在本經中強調其隨時間變遷而不可得的特性。
2. **五蘊 (Pañca-khandha)**：構成身心的五種要素，包括色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）。
3. **無常 (Anicca)**：指一切有為法（因緣而生之現象）都在不斷地變遷，沒有永恆不變的實體。
4. **厭 (Nirviddha)**：並非指世俗的厭惡或憎恨，而是一種基於智慧的「覺醒」，意識到對無常之物的執著必然導致痛苦，進而產生的冷靜抽離感。
5. **離欲 (Virāga)**：指透過對真理的體悟，使內心不再被貪愛所驅使，從而擺脫對物質與心理現象的執著。

### [現代實作指南]
本經提供了一套處理「時間軸執著」的邏輯訓練，可用於緩解對過去的遺憾與對未來的焦慮：

1. **打破時間幻象**：
   ➔ 當你陷入「如果當年...（過去）」或「如果以後能...（未來）」的情緒時，立刻提醒自己：當時的身體（色）與心靈狀態（受想行識）早已消失，未來的狀態尚未產生。
   ➔ 意識到：你所眷戀的「過去之我」或期待的「未來之我」，其實都是無常的幻影。

2. **利用「存在」來「放下」**：
   ➔ 經文提到一個深刻的邏輯：正因為有「過去」的存在，我們才有機會練習「不顧過去」。
   ➔ 實作技巧：將「遺憾」視為一種練習素材。每當遺憾升起，告訴自己：「正因為這段記憶存在，我現在可以練習對它產生厭離心，從而獲得自由。」

3. **對現在的「覺察與抽離」**：
   ➔ 觀察現在身體的不適或心理的壓力（現在色、受）。
   ➔ 不要試圖強行消除它，而是觀察它如何隨時間變動。當你發現現在的狀態也在不斷變遷時，自然會生起「離欲」之心，不再強求它必須維持在某種理想狀態。

### [段落大意]
本經論述五蘊（色、受、想、行、識）在過去、現在、未來三時皆為無常。聖弟子透過觀察這種無常性，不再眷戀過去、不期待未來，並對現在的執著產生厭離心以達成寂靜。佛陀強調，正是因為這些現象的存在，修行者才有機會透過觀察它們來實踐放下與解脫。
