# 雜阿含經 第 392 經

### [原文]
如是我聞：

一時，佛住波羅㮈國仙人住處鹿野苑中。

爾時，世尊告諸比丘：「若沙門、婆羅門於苦聖諦不如實知，苦集聖諦不如實知，苦滅聖諦不如實知，苦滅道跡聖諦不如實知，當知是沙門、婆羅門不得脫苦。若沙門、婆羅門於苦聖諦如實知，於苦集聖諦如實知，於苦滅聖諦如實知，於苦滅道跡聖諦如實知，當知是沙門、婆羅門解脫於苦。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

「如於苦不解脫、解脫。如是捨惡趣不解脫、解脫，堪能捨戒退減，不捨戒退減，能自說得過人法自證，不能自說得過人法作證，能於此外求良福田，不能於此外求良福田。

「能於此外求大師，不能於此外求大師，不能越苦，堪能越苦，不堪能脫苦，堪能脫苦。」

如是上諸經重說悉繼以偈：

「若不知苦者，　　及彼眾苦因， 　一切諸苦法，　　寂滅永無餘。 　若不知道跡，　　能思一切苦， 　心解脫於苦，　　慧解脫亦然。 　不能越眾苦，　　令苦究竟脫， 　若如實知苦，　　亦知眾苦因。 　及一切諸苦，　　永滅盡無餘， 　若復如實知，　　息苦之道跡。 　意解脫具足，　　慧解脫亦然， 　堪能越眾苦，　　究竟得解脫。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住波羅㮈國仙人住處鹿野苑中。**
　　在某一個時間，佛陀住在波羅㮈國（今印度瓦拉納西附近）仙人居住的鹿野苑中。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若沙門、婆羅門於苦聖諦不如實知，苦集聖諦不如實知，苦滅聖諦不如實知，苦滅道跡聖諦不如實知，當知是沙門、婆羅門不得脫苦。**
　　「如果沙門或婆羅門對於苦（人生本質的苦）聖諦不能如實地認知，對於集（苦之因）聖諦不能如實地認知，對於滅（苦之止）聖諦不能如實地認知，以及對於道（解脫之道的路徑）聖諦不能如實地認知，應當了解，這樣的沙門或婆羅門無法從苦中解脫。

**◈ 若沙門、婆羅門於苦聖諦如實知，於苦集聖諦如實知，於苦滅聖諦如實知，於苦滅道跡聖諦如實知，當知是沙門、婆羅門解脫於苦。」**
　　如果沙門或婆羅門能如實地認知苦（人生本質的苦）聖諦，如實地認知集（苦之因）聖諦，如實地認知滅（苦之止）聖諦，以及如實地認知道（解脫之道的路徑）聖諦，應當了解，這樣的沙門或婆羅門能從苦中獲得解脫。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 「如於苦不解脫、解脫。**
　　「正如在苦中無法解脫或能解脫一樣，

**◈ 如是捨惡趣不解脫、解脫，**
　　同樣地，能否捨棄惡趣（低劣的輪迴境界）而獲得解脫，

**◈ 堪能捨戒退減，不捨戒退減，**
　　能否避免戒律的退減（修行退步），

**◈ 能自說得過人法自證，不能自說得過人法作證，**
　　能否自稱獲得超越常人的法（神通或境界）並自我證明，

**◈ 能於此外求良福田，不能於此外求良福田。**
　　能否在佛法之外尋求更好的福田（能帶來功德的對象），

**◈ 能於此外求大師，不能於此外求大師，**
　　能否在佛法之外尋求大師，

**◈ 不能越苦，堪能越苦，不堪能脫苦，堪能脫苦。」**
　　能否超越苦，能否從苦中徹底解脫，（這一切都取決於是否如實認知四聖諦）。」

**◈ 如是上諸經重說悉繼以偈：**
　　上述諸經的內容再次重複說明，並接續以偈頌形式表達：

**◈ 「若不知苦者，及彼眾苦因，一切諸苦法，寂滅永無餘。**
　　「如果不知道苦（人生本質的苦）以及眾多苦的因（集），就無法讓一切苦的現象達到永恆寂滅且毫無餘留的狀態。

**◈ 若不知道跡，能思一切苦，心解脫於苦，慧解脫亦然。**
　　如果不知道解脫的路徑（道），即便思考一切苦，心也無法從苦中解脫，智慧的解脫也是如此。

**◈ 不能越眾苦，令苦究竟脫，**
　　將無法超越眾多苦，無法讓苦獲得究竟的解脫。

**◈ 若如實知苦，亦知眾苦因。及一切諸苦，永滅盡無餘，**
　　如果能如實地認知苦，且知道眾苦的因，並了解一切苦如何永恆滅盡且毫無餘留，

**◈ 若復如實知，息苦之道跡。**
　　如果能進一步如實地認知止息苦的路徑（道），

**◈ 意解脫具足，慧解脫亦然，堪能越眾苦，究竟得解脫。」**
　　則意根能獲得具足的解脫，智慧也能獲得解脫，進而能超越眾苦，最終獲得徹底的解脫。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四個真理，包括苦（人生本質的不滿足與痛苦）、集（導致苦的貪愛與無明）、滅（苦的完全止息）、道（通往止息苦的八正道）。
2. **惡趣 (Apāya)**：指輪迴中較為痛苦的境界，如地獄、餓鬼、畜生道。
3. **福田 (Puññakkhetta)**：比喻能讓修行者種植善因、獲得功德的對象（如佛、法、僧）。
4. **慧解脫 (Paññā-vimokkha)**：透過對真理（如無常、無我）的深刻洞察與智慧，而使心靈從執著中解脫的狀態。

### [現代實作指南]
本經文強調「認知」是解脫的前提。在現代生活中，我們可以將其轉化為以下心理實踐步驟：

1. **如實認知「苦」 (正視壓力)**：
   - 當感到焦慮或痛苦時，不要試圖立即壓抑或逃避，而是採取「觀察者」視角，承認：「現在有一種不舒服的感受正在發生」。這就是「如實知苦」，將痛苦客觀化，而非將其等同於「我」。

2. **分析「集」 (追溯根源)**：
   - 詢問自己：「這個痛苦背後的『貪愛』或『執著』是什麼？」例如，因為希望他人認同（貪）而導致被批評時的痛苦。找出導致痛苦的心理造作（行），即是認知苦因。

3. **確認「滅」 (建立希望)**：
   - 意識到只要「執著」消失，痛苦也會隨之消失。體驗在短暫放下期待或掌控欲時，心中產生的那份寧靜，以此確認「苦是可以止息的」。

4. **實踐「道」 (採取行動)**：
   - 運用「正念」與「正思惟」。在面對壓力時，不再採取慣性的憤怒或逃避，而是選擇一種理智且不傷害他人的應對方式（正業、正語），將解脫的路徑融入日常行為中。

### [段落大意]
本經強調如實認知「四聖諦」（苦、集、滅、道）是解脫痛苦的唯一途徑。若不能正確認知這四項真理，無論是沙門或婆羅門都無法脫離苦海，亦無法在修行上有所進展（如捨惡趣、持戒、證悟）。經文最後以偈頌總結：唯有同時如實了解苦、苦因、苦之滅盡以及解脫的路徑，才能使心與智慧獲得徹底的解脫，超越一切苦難。
