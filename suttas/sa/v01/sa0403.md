# 雜阿含經 第 403 經

### [原文]
如是我聞：

一時，佛在摩竭國人間遊行，於王舍城波羅利弗是中間竹林聚落，國王於中造福德舍。爾時，世尊與諸大眾於中宿止。

爾時，世尊告諸比丘：「我與汝等於四聖諦無知、無見、無隨順覺、無隨順受者，應當長夜驅馳生死。何等為四？謂苦聖諦、苦集聖諦、苦滅聖諦、苦滅道跡聖諦。我與汝等於四聖諦無知、無見、無隨順覺、無隨順受者，應當長夜驅馳生死。以我及汝於此苦聖諦順知、順入，斷諸有流，盡諸生死，不受後有。於苦集聖諦、苦滅聖諦、苦滅道跡聖諦順知、順入，斷諸有流，盡諸生死，不受後有。是故，比丘！於四聖諦未無間等者，當勤方便，起增上欲，修無間等。」

爾時，世尊即說偈言：

「我常與汝等，　　長夜涉生死， 　不見聖諦故，　　大苦日增長。 　若見四聖諦，　　斷有大流海， 　生死永已除，　　不復受後生。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛在摩竭國人間遊行，於王舍城波羅利弗是中間竹林聚落，國王於中造福德舍。**
　　在某一個時間，佛陀在摩竭國中遊行，來到王舍城的波羅利弗竹林聚落，國王在那裡建造了一座福德舍。

**◈ 爾時，世尊與諸大眾於中宿止。**
　　那時，世尊與眾多隨行弟子在其中暫時住宿。

**◈ 爾時，世尊告諸比丘：「我與汝等於四聖諦無知、無見、無隨順覺、無隨順受者，應當長夜驅馳生死。**
　　那時，世尊告訴在場的眾位比丘說：「如果我與你們對於四聖諦（四種關於苦與解脫的真理）沒有認知、沒有見地、沒有隨順的覺悟與領受，就必須在漫長的輪迴之夜中，不斷地在生死（輪迴）中奔波驅馳。

**◈ 何等為四？謂苦聖諦、苦集聖諦、苦滅聖諦、苦滅道跡聖諦。**
　　這四種真理是什麼呢？指的是：苦聖諦（關於苦的真理）、苦集聖諦（關於苦之原因的真理）、苦滅聖諦（關於苦之止息的真理）、以及苦滅道跡聖諦（關於解脫之道的真理）。

**◈ 我與汝等於四聖諦無知、無見、無隨順覺、無隨順受者，應當長夜驅馳生死。**
　　我再次強調，如果我與你們對於這四聖諦沒有認知、沒有見地、沒有隨順的覺悟與領受，就必須在漫長的輪迴之夜中，不斷地在生死（輪迴）中奔波驅馳。

**◈ 以我及汝於此苦聖諦順知、順入，斷諸有流，盡諸生死，不受後有。**
　　正因為我以及你們能夠對這苦聖諦（關於苦的真理）產生正確的認知並進入其中，才能斷除所有對存在的執著之流（有流），終結所有的生死（輪迴），不再承受未來的出生（後有）。

**◈ 於苦集聖諦、苦滅聖諦、苦滅道跡聖諦順知、順入，斷諸有流，盡諸生死，不受後有。**
　　同樣地，對於苦集聖諦（關於苦因的真理）、苦滅聖諦（關於苦止的真理）、以及苦滅道跡聖諦（關於解脫之道的真理）產生正確的認知並進入其中，也能斷除所有對存在的執著之流，終結所有的生死（輪迴），不再承受未來的出生。

**◈ 是故，比丘！於四聖諦未無間等者，當勤方便，起增上欲，修無間等。」**
　　因此，比丘們！對於那些尚未對四聖諦達到完全領悟且無間隙契合（無間等）的人，應該勤奮地尋找方法，激發強烈的修行願望（增上欲），去實踐修行以達到那種完全領悟的狀態。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即說了偈頌：

**◈ 「我常與汝等，長夜涉生死，不見聖諦故，大苦日增長。**
　　「我與你們長期以來，在漫長的輪迴之夜中涉足於生死（輪迴），正是因為沒有見到聖諦，導致巨大的痛苦日益增長。

**◈ 若見四聖諦，斷有大流海，生死永已除，不復受後生。」**
　　「如果能見到四聖諦，就能斷除如大海般深廣的對存在之執著（有流），生死（輪迴）將永遠被消除，不再承受未來的出生。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **四聖諦 (Cattāri Ariyasaccāni)**：佛教的核心教義，分為苦（認清生命本質的苦）、集（找出苦的根源是貪愛與無明）、滅（體悟苦的止息即涅槃）、道（實踐導致止息的八正道）。
2. **有流 (Bhava-sota)**：指眾生對「存在」的執著，像河流一樣不斷流轉，使意識在生死輪迴中無法停止。
3. **後有 (Punabbhava)**：指在當前生命結束後，由於業力的牽引而再次出生於六道中的現象。
4. **無間等 (Equanimity/Realization)**：在此指心靈與真理之間沒有任何隔閡、完全契合的領悟狀態。
5. **增上欲 (Chanda)**：一種正向的、強烈的願望，指修行者渴望解脫、追求覺悟的積極動力。

### [現代實作指南]
本經文強調「認知」與「實踐」的對接。在現代生活中，我們可以將四聖諦轉化為一套「問題解決邏輯」來實踐：

1. **正視問題（苦聖諦）**：
   - **實作**：當壓力或痛苦發生時，不要試圖逃避或壓抑，而是練習「客觀觀察」。告訴自己：「現在有一種痛苦的感受正在發生」，將其視為一種現象而非「我的」痛苦。

2. **分析根源（苦集聖諦）**：
   - **實作**：追問痛苦背後的「執著」。例如：我感到憤怒，是因為我執著於「對方必須按照我的意願行動」嗎？找出那個「想要控制」或「不願接受」的貪愛點。

3. **設定目標（苦滅聖諦）**：
   - **實作**：意識到「痛苦是可以停止的」。當你發現執著點時，嘗試在心中放下那個控制欲，體驗那一瞬間的放鬆與寧靜，這就是微小的「滅」之體驗。

4. **執行路徑（苦滅道跡聖諦）**：
   - **實作**：將八正道簡化為日常習慣。例如：在說話前思考是否誠實且友善（正語），在工作時保持專注於當下（正念），在思考時不被偏見牽引（正見）。

**轉念技巧**：
將「為什麼我這麼痛苦？」轉化為「這個痛苦在提醒我，我對什麼產生了執著？」。將痛苦視為覺悟的「信號燈」，而非需要被消除的敵人。

### [段落大意]
本經由佛陀在王舍城向比丘說明，若缺乏對四聖諦（苦、集、滅、道）的正確認知與領悟，眾生將在生死輪迴中長久受苦。唯有正確地進入並體悟這四種真理，才能斷除對存在的執著，終結輪迴，不再受後生之苦。佛陀勉勵修行者應激發強烈的願望，勤於實踐以達到與真理完全契合的境界。
