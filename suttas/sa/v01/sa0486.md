# 雜阿含經 第 486 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，世尊告諸比丘：「若於一法，生正厭離、不樂、背捨，得盡諸漏，所謂一切眾生由食而存。復有二法，名及色。復有三法，謂三受。復有四法，謂四食。復有五法，謂五受陰。復有六法，謂六內外入處。復有七法，謂七識住。復有八法，謂世八法。復有九法，謂九眾生居。復有十法，謂十業跡。於此十法，生厭、不樂、背捨，得盡諸漏。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若於一法，生正厭離、不樂、背捨，得盡諸漏，所謂一切眾生由食而存。**
　　「如果對於一種法（現象），能生起正確的厭離（不再貪著）、不樂（不再追求快感）、背捨（徹底放下），就能使所有的漏（煩惱的流出）盡除。這所謂的一法，就是『一切眾生都依賴食物而生存』這個事實。

**◈ 復有二法，名及色。**
　　另外有兩種法，分別是名（精神組成）與色（物質身體）。

**◈ 復有三法，謂三受。**
　　另外有三種法，是指三種受（感受：苦、樂、捨）。

**◈ 復有四法，謂四食。**
　　另外有四種法，是指四種食（維持生存的四種營養來源）。

**◈ 復有五法，謂五受陰。**
　　另外有五種法，是指五受陰（即五蘊：色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識））。

**◈ 復有六法，謂六內外入處。**
　　另外有六種法，是指六個內外入處（六種感官及其對應的外部對象）。

**◈ 復有七法，謂七識住。**
　　另外有七種法，是指七個識住（意識停留的七個領域）。

**◈ 復有八法，謂世八法。**
　　另外有八種法，是指世間的八種法（利、衰、毀、譽、苦、樂、得、失）。

**◈ 復有九法，謂九眾生居。**
　　另外有九種法，是指眾生居住的九種處所。

**◈ 復有十法，謂十業跡。**
　　另外有十種法，是指十種業跡（十種行為的痕跡，即十不善業）。

**◈ 於此十法，生厭、不樂、背捨，得盡諸漏。」**
　　對於這十組法，若能生起厭離、不樂與背捨，就能使所有的漏（煩惱的流出）盡除。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **漏 (Āsava)**：指從心靈中流出的煩惱，如欲漏、有漏、無明漏。這些「滲漏」使眾生被束縛在輪迴之中，無法獲得解脫。
2. **名色 (Nāma-rūpa)**：名指精神層面的組成（受、想、行、識），色指物質身體。兩者共同構成了個體的生命形態。
3. **四食 (Āhāra)**：指維持生命與存在所需的四種營養：段食（物質食物）、觸食（感官接觸）、意欲食（心的造作/渴望）、識食（意識的延續）。
4. **世八法 (Loka-dhamma)**：指世間人普遍在意的八種對立現象：獲利與損失、名聲與毀譽、稱讚與批評、快樂與痛苦。
5. **十業跡 (Kamma-patha)**：指導致惡果的十種行為路徑，包括身業（殺、盜、淫）、口業（妄語、兩舌、惡口、兩舌）、意業（貪、嗔、Wrong view/邪見）。

### [現代實作指南]
本經的核心在於透過「對條件化現象的覺察」來產生「厭離」與「背捨」。這並非指對生活產生厭惡感，而是對「依賴外部條件而產生之不穩定快感」的清醒認知。

**實作步驟：**
1. **識別當下的「法」**：當你感到情緒波動時，試著將其歸類到經文提到的法中。例如：
   - 當你因為被批評而難受 $\rightarrow$ 識別為「世八法」中的「毀譽」。
   - 當你對某種食物或物質有強烈渴望 $\rightarrow$ 識別為「由食而存」或「四食」。
   - 當你陷入對自我的定義 $\rightarrow$ 識別為「五受陰」的運作。
2. **練習「正厭離」**：觀察該現象的無常性。告訴自己：「這個快感/痛苦是由條件組合而成的，它必然會消失，因此它不能提供永恆的滿足。」
3. **執行「背捨」**：在意識到「它不能提供永恆滿足」後，有意識地將注意力從該對象移開，不再試圖控制或抓住它。
4. **轉念技巧**：將「我好痛苦」轉化為「現在有『苦受（感受）』在運作」，將主體從「我」移至「法（現象）」，從而創造出觀察的空間，減少煩惱的流出。

### [段落大意]
本經由佛陀列舉了從一到十組不同的現象（法），涵蓋了生存條件、身心組成、感受、感官、意識領域、世間得失及行為業跡。佛陀教導，若修行者能對這些所有條件構成的現象生起正確的厭離心，不再追求其中的快感並徹底放下執著，即可消除所有內心的煩惱滲漏（漏），最終達成解脫。
