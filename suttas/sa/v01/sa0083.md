# 雜阿含經 第 83 經

### [原文]
如是我聞：

一時，佛住毘耶離獼猴池側重閣講堂。

爾時，世尊告諸比丘：「多聞聖弟子於何所見非我、不異我、不相在。如是平等正觀，如實知見？」

比丘白佛：「世尊為法根、法眼、法依，唯願為說！諸比丘聞已，如說奉行。」

佛告比丘：「諦聽，善思，當為汝說。多聞聖弟子於色見非我、不異我、不相在，是名如實正觀。受、想、行、識亦復如是。」

佛告諸比丘：「色為是常、為無常耶？」

比丘白佛：「無常。世尊！」

又告比丘：「若無常者，是苦不？」

比丘白佛：「是苦。世尊！」

「比丘！若無常、苦，是變易法，多聞聖弟子於中寧見有我、異我、相在不？」

比丘白佛：「不也，世尊！」

「受、想、行、識亦復如是。是故，比丘！所有諸色，若過去、若未來、若現在，若內、若外，若麤、若細，若好、若醜，若遠、若近，彼一切皆非我、不異我、不相在，是名如實正觀。受、想、行、識亦復如是。多聞聖弟子如是觀察，於色得解脫，於受、想、行、識得解脫。我說彼解脫生、老、病、死、憂、悲、惱、苦，純大苦聚。」

佛說此經時，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住毘耶離獼猴池側重閣講堂。**
　　在某一個時間，佛陀住在毘耶離（Vesālī）獼猴池旁的重閣講堂中。

**◈ 爾時，世尊告諸比丘：「多聞聖弟子於何所見非我、不異我、不相在。如是平等正觀，如實知見？」**
　　那時，世尊告訴在場的眾位比丘說：「一名博學的聖弟子，應該在什麼地方觀察到：這不是我，也不與我不同，更不是存在於我之中的（非我、不異我、不相在）？如此平等地正觀，才能如實地知見真理？」

**◈ 比丘白佛：「世尊為法根、法眼、法依，唯願為說！諸比丘聞已，如說奉行。」**
　　比丘們對佛陀恭敬地說道：「世尊您是真理的根源、真理的洞察者以及真理的依託，唯願您為我們解說！我們眾比丘聽聞之後，定會依照教導去實踐。」

**◈ 佛告比丘：「諦聽，善思，當為汝說。多聞聖弟子於色見非我、不異我、不相在，是名如實正觀。受、想、行、識亦復如是。」**
　　佛陀告訴比丘們：「請仔細聽，好好思考，我現在為你們解說。博學的聖弟子對於色（物質身體）觀察到：這不是我，也不與我不同，更不是存在於我之中的，這就叫做如實的正觀。對於受（感受）、想（想像與認知）、行（心的造作）、識（意識）也同樣如此。」

**◈ 佛告諸比丘：「色為是常、為無常耶？」**
　　佛陀問諸比丘：「色（物質身體）是恆常不變的，還是無常的呢？」

**◈ 比丘白佛：「無常。世尊！」**
　　比丘們對佛陀回答：「是無常的，世尊！」

**◈ 又告比丘：「若無常者，是苦不？」**
　　佛陀又問比丘們：「如果它是無常的，那它是不是苦（不滿足、不安穩）的？」

**◈ 比丘白佛：「是苦。世尊！」**
　　比丘們對佛陀回答：「是苦的，世尊！」

**◈ 「比丘！若無常、苦，是變易法，多聞聖弟子於中寧見有我、異我、相在不？」**
　　「比丘們！既然它是無常且是苦的，這就是一種不斷變遷的法（變易法），那麼博學的聖弟子在其中，怎麼可能看到有一個恆定的『我』、一個與我不同的『我』，或者一個存在於其中的『我』呢？」

**◈ 比丘白佛：「不也，世尊！」**
　　比丘們對佛陀回答：「不可能，世尊！」

**◈ 「受、想、行、識亦復如是。**
　　「受（感受）、想（想像與認知）、行（心的造作）、識（意識）也同樣如此。

**◈ 是故，比丘！所有諸色，若過去、若未來、若現在，若內、若外，若麤、若細，若好、若醜，若遠、若近，彼一切皆非我、不異我、不相在，是名如實正觀。受、想、行、識亦復如是。**
　　因此，比丘們！所有屬於色（物質身體）的現象，無論是過去、未來或現在，無論是內在的或外在的，無論是粗糙的或細微的，無論是美好的或醜陋的，無論是遙遠的或近在咫尺的，這一切全部都不是我，也不與我不同，更不是存在於我之中的，這才叫做如實的正觀。受（感受）、想（想像與認知）、行（心的造作）、識（意識）也同樣如此。

**◈ 多聞聖弟子如是觀察，於色得解脫，於受、想、行、識得解脫。**
　　博學的聖弟子如此觀察，就能在色（物質身體）中獲得解脫，並在受（感受）、想（想像與認知）、行（心的造作）、識（意識）中獲得解脫。

**◈ 我說彼解脫生、老、病、死、憂、悲、惱、苦，純大苦聚。」**
　　我所說的這種解脫，是從出生、衰老、疾病、死亡，以及憂慮、悲傷、煩惱、痛苦這整個純粹且巨大的苦之聚集體中解脫出來。

**◈ 佛說此經時，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **非我、不異我、不相在 (Anattā)**：這是對「無我」的三種邏輯否定。
    - **非我**：指現象本身並非一個恆常、獨立的自我實體。
    - **不異我**：指不能將自我視為一個獨立於現象之外的靈魂或主體。
    - **不相在**：指自我並非寄宿或存在於這些現象之中。
2. **如實正觀 (Yathā-bhūta-ñāṇa-dassana)**：指不經過主觀偏見、不被貪愛或厭惡所遮蔽，直接觀察事物本然的樣子。
3. **變易法**：指所有處於因緣生滅之中、不斷變動且無法維持原狀的現象。
4. **苦聚 (Dukkha-kkhandha)**：指生命中所有痛苦的總和，包括生理的病痛與心理的憂悲，其根源在於對無常之物的執著。

### [現代實作指南]
本經提供了一套邏輯嚴密的「去自我化」練習法，可用於緩解現代人的焦慮與對身份認同的執著：

1. **標記現象（對應五蘊）**：
   - 當你感到壓力時，不要說「我很痛苦」，而要將其拆解為：
     - **色** ➔ 身體感到胸悶、肩膀僵硬。
     - **受** ➔ 感覺到一種不舒服的壓迫感。
     - **想** ➔ 大腦在認知這是一個「失敗」的場景。
     - **行** ➔ 心中產生了想要逃避或憤怒的衝動。
     - **識** ➔ 意識到目前處於壓力狀態。

2. **邏輯推演（無常 ➔ 苦 ➔ 非我）**：
   - 觀察上述任何一項（例如：胸悶的感覺）。
   - **問**：這個感覺是恆常的嗎？ ➔ **答**：不是，它會隨時間改變（無常）。
   - **問**：因為它會改變，所以它能給我永恆的滿足嗎？ ➔ **答**：不能，它帶來不安（苦）。
   - **問**：既然它在變動且不穩定，我能把它當作「我」或「我的」嗎？ ➔ **答**：不能（非我）。

3. **擴展觀察（全方位解構）**：
   - 將此方法應用於不同維度：
     - **時間**：過去的成就與未來的擔憂同樣是變易法。
     - **性質**：無論是美好的讚美（好）或醜陋的批評（醜），都只是暫時的現象。
   - **目標**：透過不斷地將「我」與「現象」分離，減少對情緒的認同感，從而獲得心理上的解脫。

### [段落大意]
本經透過佛陀與比丘的問答，論證了五蘊（色、受、想、行、識）皆為無常且屬於苦的變易法，因此不能被視為「我」。佛陀教導聖弟子應對所有時間、空間與性質的物質與精神現象進行「非我、不異我、不相在」的如實正觀，以此打破對自我的執著，最終從生老病死等巨大的苦聚中獲得徹底的解脫。
