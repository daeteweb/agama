# 雜阿含經 第 9 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「色無常，無常即苦，苦即非我，非我者亦非我所。如是觀者，名真實正觀。如是受、想、行、識無常，無常即苦，苦即非我，非我者亦非我所。如是觀者，名真實觀。

「聖弟子！如是觀者，厭於色，厭受、想、行、識，厭故不樂，不樂故得解脫。解脫者真實智生：『我生已盡，梵行已立，所作已作，自知不受後有。』」

時，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「色無常，無常即苦，苦即非我，非我者亦非我所。**
　　「色（物質身體）是無常的（不斷變遷的），而這種無常的本質即是苦（因為無法恆常而導致的不安與痛苦），既然是苦，就不是永恆不變的自我（非我），而既然不是自我，也就不是我所擁有的東西（非我所）。

**◈ 如是觀者，名真實正觀。**
　　能夠如此觀察的人，就稱為真實的正觀（正確的洞察）。

**◈ 如是受、想、行、識無常，無常即苦，苦即非我，非我者亦非我所。**
　　同樣地，受（感受）、想（想像與認知）、行（心的造作）、識（意識）也都是無常的，無常即是苦，苦即是非我，非我者亦非我所。

**◈ 如是觀者，名真實觀。**
　　能夠如此觀察的人，就稱為真實的觀察。

**◈ 「聖弟子！如是觀者，厭於色，厭受、想、行、識，**
　　「聖弟子！能夠如此觀察的人，會對色（物質身體）感到厭離，對受（感受）、想（想像與認知）、行（心的造作）、識（意識）感到厭離，

**◈ 厭故不樂，不樂故得解脫。**
　　因為產生了厭離心，所以不再執著於感官的快樂，因為不再執著於樂，而獲得了解脫（Vimokkha）。

**◈ 解脫者真實智生：『我生已盡，梵行已立，所作已作，自知不受後有。』」**
　　獲得解脫的人，真實的智慧會生起，並體悟到：『我的輪迴出生已經結束，聖潔的修行已經建立，應做的事已經全部完成，自知不再受後世的輪迴出生。』」

**◈ 時，諸比丘聞佛所說，歡喜奉行。**
　　那時，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **五蘊 (Pañca-khandha)**：構成個體身心的五種要素，包含色（物質）、受（感受）、想（認知）、行（意志造作）、識（意識）。本經強調五蘊皆具備無常、苦、無我的特性。
2. **無常 (Anicca)**：指一切有為法（因緣和合而生之物）都在不斷地變遷，沒有永恆不變的實體。
3. **非我 (Anattā)**：指沒有一個獨立、恆常、主宰的「自我」存在於五蘊之中。
4. **厭 (Nibbidā)**：並非指世俗的厭惡或反感，而是一種基於智慧的「厭離」，即看清事物的本質後，不再產生貪著與執取。
5. **梵行 (Brahmacariya)**：原指禁欲生活，在佛教中指聖潔的修行生活，即完全遠離染污、趨向覺悟的修行路徑。

### [現代實作指南]
本經提供了一套從「觀察」到「解脫」的邏輯鏈條，可用於處理現代生活中的情緒壓力與執著：

1. **標記現象（正觀）**：
   當你感到強烈的情緒（如憤怒或焦慮）時，不要將其視為「我很憤怒」，而將其拆解為五蘊。例如：
   ➔ 身體的緊繃 ➔ 色（物質身體）
   ➔ 不舒服的感覺 ➔ 受（感受）
   ➔ 「他怎麼能這樣對我」的念頭 ➔ 想（認知）
   ➔ 想反擊的衝動 ➔ 行（心的造作）

2. **分析屬性（無常 ➔ 苦 ➔ 非我）**：
   觀察這個憤怒的過程：
   ➔ 它是恆常的嗎？（不，它在起伏，所以是**無常**）
   ➔ 試圖抓住這個感覺會快樂嗎？（不會，因為它在變，所以是**苦**）
   ➔ 我能命令這個憤怒立刻消失嗎？（不能，它隨因緣而生，所以**非我**）

3. **建立距離（厭離 ➔ 不樂）**：
   當你意識到這個情緒只是「一套無常的程序」而非「我的本質」時，你會對這種反覆的痛苦產生一種「厭離感」。這種厭離會讓你不再渴望透過壓抑或宣洩來尋找暫時的快感（不樂）。

4. **獲得心理空間（解脫）**：
   不再執著於情緒的起伏，心靈便獲得了空間，這就是微觀層面的「解脫」。

### [段落大意]
本經闡述了對五蘊（色、受、想、行、識）進行正確觀察的路徑：體悟其無常 ➔ 認清其為苦 ➔ 洞察其非我且非我所。透過這種真實的正觀，修行者能對五蘊產生厭離心，不再執著於感官之樂，進而獲得解脫，終結輪迴之苦，成就聖潔的修行。
