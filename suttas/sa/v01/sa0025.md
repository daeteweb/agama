# 雜阿含經 第 25 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有異比丘來詣佛所，為佛作禮，却住一面，白佛言：「如世尊說多聞，云何為多聞？」

佛告比丘：「善哉！善哉！汝今問我多聞義耶？」

比丘白佛：「唯然，世尊！」

佛告比丘：「諦聽，善思，當為汝說。比丘當知，若聞色是生厭、離欲、滅盡、寂靜法，是名多聞；如是聞受、想、行、識，是生厭、離欲、滅盡、寂靜法，是名多聞比丘，是名如來所說多聞。」

時，彼比丘聞佛所說，踊躍歡喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有異比丘來詣佛所，為佛作禮，却住一面，白佛言：「如世尊說多聞，云何為多聞？」**
　　那時，有一位特殊的比丘來到佛陀這裡，向佛陀行禮，然後站在一旁，對佛陀恭敬地說道：「如世尊所說的『多聞』（博學），究竟什麼才算是真正的多聞？」

**◈ 佛告比丘：「善哉！善哉！汝今問我多聞義耶？」**
　　佛陀告訴這位比丘：「好極了！做得好！你現在是在問我關於『多聞』的義理嗎？」

**◈ 比丘白佛：「唯然，世尊！」**
　　比丘對佛陀恭敬地說道：「是的，世尊！」

**◈ 佛告比丘：「諦聽，善思，當為汝說。**
　　佛陀告訴比丘：「仔細聽，好好思考，我現在為你們解說。

**◈ 比丘當知，若聞色是生厭、離欲、滅盡、寂靜法，是名多聞；**
　　比丘應當了解，如果聽到色（物質身體）是會讓人產生厭離心、遠離貪欲、最終滅盡煩惱並達到寂靜的法，這才叫做多聞；

**◈ 如是聞受、想、行、識，是生厭、離欲、滅盡、寂靜法，是名多聞比丘，是名如來所說多聞。」**
　　同樣地，如果聽到受（感受）、想（想像與認知）、行（心的造作）、識（意識）也是會讓人產生厭離心、遠離貪欲、最終滅盡煩惱並達到寂靜的法，這才叫做多聞的比丘，這才叫做如來（成就不變真理而來到世間的人）所說的多聞。」

**◈ 時，彼比丘聞佛所說，踊躍歡喜，作禮而去。**
　　那時，那位比丘聽聞佛陀所說的教法，心中感到非常歡喜，向佛陀行禮後才離開。

### [詞語註解（解釋重要佛教概念）]
1. **多聞 (Bahu-ssuta)**：字面上指「聽聞得多」，在一般脈絡中指博學。但在本經中，佛陀將其重新定義為：能透過聽聞教法，深刻體悟到五蘊的本質，進而產生厭離心並趨向解脫的智慧。
2. **生厭 (Nibbida)**：並非指世俗的厭惡或反感，而是一種基於智慧的「厭離」。是指在觀察到現象的無常與苦後，對執著於此類現象而產生的冷淡與不再渴求。
3. **離欲 (Viraga)**：指對感官對象或自我認同的貪愛（Taṇhā）逐漸消失，心不再被欲望所牽引的狀態。
4. **滅盡 (Nirodha)**：指苦因（貪、嗔、癡）的停止，進而使苦果不再產生。
5. **寂靜 (Santi)**：指心不再被煩惱攪動，達到絕對寧靜、不再輪迴的狀態。

### [現代實作指南]
本經將「知識的累積」轉化為「覺知的轉化」。在現代生活中，我們常追求「多聞」（閱讀大量書籍、獲取資訊），但若僅是資訊的堆疊，無法帶來真正的平靜。可採取以下實作步驟：

1. **定義對象 ➔ 標記五蘊**：
   當你感到壓力或情緒波動時，嘗試將其拆解。例如：身體的緊繃是「色（物質身體）」，心中的憤怒是「受（感受）」，對對方的刻板印象是「想（想像與認知）」。

2. **觀察本質 ➔ 體悟無常**：
   觀察這個「憤怒的感受」是否在持續變化？它是否能被你完全掌控？當你發現它是不穩定且會帶來痛苦的，這就是「生厭」的開始 ➔ 意識到執著於此並無益處。

3. **練習抽離 ➔ 實踐離欲**：
   告訴自己：「這只是一個『受』的現象，並非『我』的憤怒。」透過這種認知，將自己從情緒中抽離，不再試圖抓住或推開它，從而達到「離欲」。

4. **體驗寂靜 ➔ 進入滅盡**：
   當你不再對五蘊的變動產生反應（不貪、不嗔），心會自然進入一個短暫的寧靜狀態。這就是將「多聞」轉化為實踐，從資訊獲取轉向心靈解脫。

### [段落大意]
本經重新定義了「多聞」的真義。佛陀指出，真正的博學並非單純地獲取大量知識，而是能透過聽聞教法，體悟到構成身心的五蘊（色、受、想、行、識）皆是無常且令人厭離的，進而遠離貪欲，達到滅盡煩惱與內心寂靜的境界。
