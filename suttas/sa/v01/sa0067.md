# 雜阿含經 第 67 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「常當修習方便禪思，內寂其心。所以者何？比丘！修習方便禪思，內寂其心已，如實觀察。云何如實觀察？如實知此色、此色集、此色滅；此受、想、行、識，此識集、此識滅。

「云何色集，受、想、行、識集？愚癡無聞凡夫不如實知色集、色滅、色味、色患、色離。不如實知故，樂著彼色，讚歎於色；樂著於色，讚歎色故取；取緣有，有緣生，生緣老、死、憂、悲、惱、苦。如是純大苦聚生，是名色集，受、想、行、識集。

「云何色滅，受、想、行、識滅？多聞聖弟子如實知色集、色滅、色味、色患、色離。如實知故，不樂著色，不讚歎色；不樂著、讚歎色故，愛樂滅；愛樂滅故取滅，取滅故有滅，有滅故生滅，生滅故老、病、死、憂、悲、惱、苦滅，如是純大苦聚滅。云何多聞聖弟子如實知受、想、行、識？識集、識滅、識味、識患、識離如實知？知彼故不樂著彼識，不讚歎於識；不樂著、讚歎識故，樂愛滅；樂愛滅故取滅，取滅故有滅，有滅故生滅，生滅故老、病、死、憂、悲、惱、苦滅，如是純大苦聚滅，皆悉得滅。比丘！是名色滅，受、想、行、識滅。

「比丘！常當修習方便禪思，內寂其心。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如觀察，乃至作證十二經，亦如是廣說。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「常當修習方便禪思，內寂其心。**
　　那時，世尊告訴在場的眾位比丘說：「你們應該經常修習方便的禪思修行，使內心達到寂靜狀態。

**◈ 所以者何？比丘！修習方便禪思，內寂其心已，如實觀察。**
　　為什麼呢？比丘！在修習方便的禪思修行並使內心寂靜之後，就能夠如實地觀察現象。

**◈ 云何如實觀察？如實知此色、此色集、此色滅；此受、想、行、識，此識集、此識滅。**
　　如何才算是如實觀察？就是如實地知道這個色（物質身體）、這個色（物質身體）的集（苦因/產生過程）、這個色（物質身體）的滅（苦止/消失過程）；以及這個受（感受）、想（想像與認知）、行（心的造作）、識（意識），以及這個識（意識）的集（苦因/產生過程）與這個識（意識）的滅（苦止/消失過程）。

**◈ 云何色集，受、想、行、識集？**
　　什麼是色（物質身體）的集（苦因），以及受（感受）、想（想像與認知）、行（心的造作）、識（意識）的集（苦因）？

**◈ 愚癡無聞凡夫不如實知色集、色滅、色味、色患、色離。**
　　愚癡且缺乏聞法機會的凡夫，不能如實地知道色（物質身體）的集（苦因）、色（物質身體）的滅（苦止）、色（物質身體）的味（快感）、色（物質身體）的患（缺陷與危險）以及色（物質身體）的離（脫離）。

**◈ 不如實知故，樂著彼色，讚歎於色；**
　　因為不能如實地知道，所以對那個色（物質身體）產生樂著，並對色（物質身體）讚歎。

**◈ 樂著於色，讚歎色故取；**
　　因為樂著於色（物質身體）且讚歎色（物質身體），所以產生了取（執取）。

**◈ 取緣有，有緣生，生緣老、死、憂、悲、惱、苦。**
　　取（執取）作為緣而產生有（存在），有（存在）作為緣而產生生（出生），生（出生）作為緣而產生老、死、憂、悲、惱、苦。

**◈ 如是純大苦聚生，是名色集，受、想、行、識集。**
　　就這樣，純粹且巨大的苦之聚集產生了，這就叫做色（物質身體）的集（苦因），以及受（感受）、想（想像與認知）、行（心的造作）、識（意識）的集（苦因）。

**◈ 云何色滅，受、想、行、識滅？**
　　什麼是色（物質身體）的滅（苦止），以及受（感受）、想（想像與認知）、行（心的造作）、識（意識）的滅（苦止）？

**◈ 多聞聖弟子如實知色集、色滅、色味、色患、色離。**
　　聽聞多聞的聖弟子，能如實地知道色（物質身體）的集（苦因）、色（物質身體）的滅（苦止）、色（物質身體）的味（快感）、色（物質身體）的患（缺陷與危險）以及色（物質身體）的離（脫離）。

**◈ 如實知故，不樂著色，不讚歎色；**
　　因為如實地知道，所以不再樂著於色（物質身體），也不再讚歎色（物質身體）。

**◈ 不樂著、讚歎色故，愛樂滅；**
　　因為不再樂著且不再讚歎色（物質身體），所以愛（貪愛）滅除了。

**◈ 愛樂滅故取滅，取滅故有滅，有滅故生滅，生滅故老、病、死、憂、悲、惱、苦滅，如是純大苦聚滅。**
　　因為愛（貪愛）滅除了，所以取（執取）滅除了；取（執取）滅除了，所以有（存在）滅除了；有（存在）滅除了，所以生（出生）滅除了；生（出生）滅除了，所以老、病、死、憂、悲、惱、苦也隨之滅除了，就這樣，純粹且巨大的苦之聚集滅除了。

**◈ 云何多聞聖弟子如實知受、想、行、識？識集、識滅、識味、識患、識離如實知？**
　　多聞聖弟子如何如實地知道受（感受）、想（想像與認知）、行（心的造作）、識（意識）？如何如實地知道識（意識）的集（苦因）、識（意識）的滅（苦止）、識（意識）的味（快感）、識（意識）的患（缺陷與危險）以及識（意識）的離（脫離）？

**◈ 知彼故不樂著彼識，不讚歎於識；**
　　因為知道了這些，所以不再樂著於那個識（意識），也不再讚歎那個識（意識）。

**◈ 不樂著、讚歎識故，樂愛滅；**
　　因為不再樂著且不再讚歎識（意識），所以樂愛滅除了。

**◈ 樂愛滅故取滅，取滅故有滅，有滅故生滅，生滅故老、病、死、憂、悲、惱、苦滅，如是純大苦聚滅，皆悉得滅。**
　　因為樂愛滅除了，所以取（執取）滅除了；取（執取）滅除了，所以有（存在）滅除了；有（存在）滅除了，所以生（出生）滅除了；生（出生）滅除了，所以老、病、死、憂、悲、惱、苦也隨之滅除了，就這樣，純粹且巨大的苦之聚集滅除了，一切都得到了滅除。

**◈ 比丘！是名色滅，受、想、行、識滅。**
　　比丘！這就叫做色（物質身體）的滅（苦止），以及受（感受）、想（想像與認知）、行（心的造作）、識（意識）的滅（苦止）。

**◈ 比丘！常當修習方便禪思，內寂其心。」**
　　比丘！你們應該經常修習方便的禪思修行，使內心達到寂靜狀態。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如觀察，乃至作證十二經，亦如是廣說。**
　　像這樣觀察，直到證悟十二經（十二因緣），也是如此廣泛地解說。

### [詞語註解（解釋重要佛教概念）]
1. **五蘊 (Pañca-kkhandha)**：構成生命個體的五種要素，包含色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）。
2. **集 (Samudaya)**：在四聖諦中指苦之原因，在此經文中特指導致五蘊產生並陷入輪迴的貪愛與執取過程。
3. **滅 (Nirodha)**：指苦的止息，即透過斷除貪愛與執取，使五蘊不再作為輪迴的動力，最終達到涅槃。
4. **十二因緣 (Paṭicca-samuppāda)**：描述生命如何因無明而產生，並在愛、取、有等環節中不斷輪迴的十二個遞進環節。
5. **色味、色患、色離**：
    - 味 (Rasa) ➔ 指對物質對象產生的快感或吸引力。
    - 患 (Ādīnava) ➔ 指物質對象必然會變質、毀壞且帶來痛苦的缺陷。
    - 離 ( Nissāraṇa) ➔ 指從對物質對象的執著中解脫出來。

### [現代實作指南]
本經文提供了一套從「心靈寂靜」到「認知解構」的實踐路徑，可用於處理現代生活中的情緒內耗與物質執著：

1. **建立寂靜空間 (內寂其心)**：
   在面對強烈情緒或慾望時，不要立即反應。先透過深呼吸或短暫的靜坐，將心從混亂的思緒中抽離，創造一個「觀察者」的空間。

2. **執行「味 → 患 → 離」的認知轉念**：
   當你強烈渴望某件物品或某種認同（色/識）時，進行以下三步驟分析：
   - **分析「味」**：承認它帶來的快感（例如：買新手機的興奮感）。
   - **分析「患」**：思考其必然的缺陷（例如：手機會過時、損壞，且追求新機帶來的焦慮感）。
   - **實踐「離」**：體會到快感是暫時的，而執著才是痛苦的根源，從而降低對該對象的依賴。

3. **觀察因果鏈條 (斷除取緣)**：
   意識到「讚歎 → 樂著 → 取 → 苦」的心理路徑。當你發現自己在心中不斷讚美某物時，提醒自己這正是「取（執取）」的開始。透過覺察，在「樂著」與「取」之間切斷連結，防止其演變成長期的精神壓力（憂、悲、惱、苦）。

### [段落大意]
本經強調透過禪思使內心寂靜，進而如實觀察五蘊的產生（集）與滅除（滅）。凡夫因不識五蘊的缺陷而產生貪愛與執取，導致輪迴之苦；聖弟子則能洞察五蘊的味、患與離，從而滅除愛與取，最終止息所有苦聚。
