# 雜阿含經 第 94 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有年少婆羅門名僧迦羅，來詣佛所，與世尊面相問訊慰勞已，退坐一面，白佛言：「瞿曇！不善男子云何可知？」

佛告婆羅門：「譬猶如月。」

婆羅門復問：「善男子云何可知？」

佛告婆羅門：「譬猶如月。」

婆羅門白佛：「云何不善男子如月？」

佛告婆羅門：「如月黑分，光明亦失，色亦失，所係亦失，日夜消滅，乃至不現。如是，有人於如來所，得信家心，受持淨戒，善學多聞，損己布施，正見真實。於如來所淨信、持戒、惠施、多聞、正見真直已，然後退失，於戒、聞、施、正見悉皆忘失，日夜消滅，乃至須臾，一切忘失。

「復次，婆羅門！若善男子不習近善知識，不數聞法，不正思惟，身行惡行，口行惡行，意行惡行。行惡因緣故，身壞命終，墮惡趣泥梨中。如是，婆羅門！不善男子其譬如月。」

婆羅門白佛：「云何善男子其譬如月？」

佛告婆羅門：「譬如明月淨分光明，色澤日夜增明，乃至月滿，一切圓淨。如是，善男子於如來法、律得淨信心，乃至正見真淨增明，戒增、施增、聞增、慧增，日夜增長；復於餘時親近善知識，聞說正法，內正思惟，行身善行，行口善行，行意善行故，以是因緣，身壞命終，化生天上。婆羅門！是故善男子譬如月。」

爾時，世尊而說偈言：

「譬如月無垢，　　周行於虛空， 　一切小星中，　　其光最盛明。 　淨信亦如是，　　戒聞離慳施， 　於諸慳世間，　　其施特明顯。」

佛說此經已，僧迦羅婆羅門聞佛所說，歡喜隨喜，從座起而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 時，有年少婆羅門名僧迦羅，來詣佛所，與世尊面相問訊慰勞已，退坐一面，白佛言：「瞿曇！不善男子云何可知？」**
　　那時，有一位名叫僧迦羅的年輕婆羅門來到佛陀這裡，在與世尊互相問候慰勞之後，退到一邊坐下，對佛陀恭敬地說道：「瞿曇！不善的男子該如何辨識呢？」

**◈ 佛告婆羅門：「譬猶如月。」**
　　佛陀告訴這位婆羅門：「就譬如月亮一樣。」

**◈ 婆羅門復問：「善男子云何可知？」**
　　婆羅門再次請問：「那麼，善的男子又該如何辨識呢？」

**◈ 佛告婆羅門：「譬猶如月。」**
　　佛陀告訴婆羅門：「也譬如月亮一樣。」

**◈ 婆羅門白佛：「云何不善男子如月？」**
　　婆羅門對佛陀恭敬地說：「為什麼說不善的男子像月亮呢？」

**◈ 佛告婆羅門：「如月黑分，光明亦失，色亦失，所係亦失，日夜消滅，乃至不現。**
　　佛陀告訴婆羅門：「就像月亮在虧損的階段，光明消失了，色澤失去了，原本依附的圓滿也失去了，日夜之間逐漸消滅，直到完全不顯現。

**◈ 如是，有人於如來所，得信家心，受持淨戒，善學多聞，損己布施，正見真實。**
　　同樣地，有人在如來這裡獲得了對佛法的信心，受持淨戒（遵守清淨的道德規範），善於學習並廣博多聞，損己布施（捨棄自己的財物來幫助他人），且擁有正見（對真理的正確認知）且真實。

**◈ 於如來所淨信、持戒、惠施、多聞、正見真直已，然後退失，於戒、聞、施、正見悉皆忘失，日夜消滅，乃至須臾，一切忘失。**
　　在如來這裡擁有淨信、持戒、惠施、多聞、正見且真直之後，隨後卻退轉失去了，將戒律、聞法、布施、正見全部忘卻，日夜消滅，直到短時間內一切忘盡。

**◈ 「復次，婆羅門！若善男子不習近善知識，不數聞法，不正思惟，身行惡行，口行惡行，意行惡行。**
　　另外，婆羅門！如果一個人不親近善知識（能導向正道的良師益友），不經常聽聞法（真理），不進行正思惟（正確的思考），在身體上行惡業，在言語上行惡業，在意識上行惡業。

**◈ 行惡因緣故，身壞命終，墮惡趣泥梨中。如是，婆羅門！不善男子其譬如月。」**
　　因為造作惡業的因緣（條件的組合），在身體毀壞、生命終結後，會墮入惡趣的泥梨（地獄）之中。所以，婆羅門！不善的男子就譬如月亮（的虧損）。」

**◈ 婆羅門白佛：「云何善男子其譬如月？」**
　　婆羅門對佛陀恭敬地說：「那麼，為什麼說善的男子也譬如月亮呢？」

**◈ 佛告婆羅門：「譬如明月淨分光明，色澤日夜增明，乃至月滿，一切圓淨。**
　　佛陀告訴婆羅門：「就像明月在盈滿的階段，光明與色澤日夜增加，直到月亮圓滿，一切圓融清淨。

**◈ 如是，善男子於如來法、律得淨信心，乃至正見真淨增明，戒增、施增、聞增、慧增，日夜增長；**
　　同樣地，善男子對於如來所說的法（真理）與律（戒律）產生了清淨的信心，直到正見（對真理的正確認知）變得真淨且增長，戒律增加、布施增加、多聞增加、智慧增加，日夜不斷增長；

**◈ 復於餘時親近善知識，聞說正法，內正思惟，行身善行，行口善行，行意善行故，以是因緣，身壞命終，化生天上。婆羅門！是故善男子譬如月。」**
　　此外，在其餘時間親近善知識，聽聞正法（正確的教法），在內心進行正思惟（正確的思考），在身體上行善，在言語上行善，在意識上行善。因為這些因緣（條件的組合），在身體毀壞、生命終結後，能化生在天上。婆羅門！所以說善男子譬如月亮（的盈滿）。」

**◈ 爾時，世尊而說偈言：**
　　那時，世尊說了一首偈頌：

**◈ 「譬如月無垢，周行於虛空，一切小星中，其光最盛明。**
　　就像月亮沒有污垢，周行於虛空之中，在所有的小星星之中，它的光芒最為盛大明亮。

**◈ 淨信亦如是，戒聞離慳施，於諸慳世間，其施特明顯。」**
　　清淨的信心也是如此，具備戒律、多聞、遠離吝嗇且能布施的人，在充滿吝嗇的世間中，其布施之光格外明顯。」

**◈ 佛說此經已，僧迦羅婆羅門聞佛所說，歡喜隨喜，從座起而去。**
　　佛陀講完這部經後，僧迦羅婆羅門聽聞佛陀所說的教法，心生法喜並依照教導去實踐，從座位上站起來離開了。

### [詞語註解（解釋重要佛教概念）]
1. **正見 (Sammā-diṭṭhi)**：八正道之首，指對四聖諦、因果業報及緣起法有正確的認知與理解，是修行解脫的導航。
2. **泥梨 (Niraya)**：巴利語 Niraya 的音譯，指地獄，是六道輪迴中最痛苦的惡趣，由強烈的嗔心與惡業導致。
3. **善知識 (Kalyāṇa-mitta)**：指能引導修行者走向正道、提供正確指導且具備德行的良師益友。
4. **因緣 (Paṭicca-samuppāda/Paccaya)**：指事物產生的條件。因（Hetu）是主因，緣（Paccaya）是助緣，兩者結合促成結果（Vipāka）。
5. **化生 (Opapāta)**：指在天界或地獄等非人類世界中，不經過胎生或卵生，而是直接以完整形態出現的出生方式。

### [現代實作指南]
本經以月亮的「盈」與「虧」比喻心靈狀態的增長與退轉，可轉化為以下生活實踐：

1. **建立「心靈盈虧」的覺察機制**：
   ➔ 每日回顧：檢查自己今日的「正見」是否增加（是否更理智地看待問題）、「布施」是否增加（是否對他人寬容或幫助），還是處於「虧損」狀態（變得吝嗇、憤怒或偏執）。
   ➔ 警惕「退轉」：意識到即便曾經有過良好的習慣或正向的認知，若缺乏維護，也會像月亮虧損一樣逐漸消失。

2. **優化「外部輸入」 (親近善知識與聞法)**：
   ➔ 篩選社交圈：減少與傾向於抱怨、毀謗或誘導惡行的人接觸，主動尋找能激發正向思考、鼓勵自我成長的對象（現代的善知識）。
   ➔ 定期學習：將「聞法」轉化為閱讀經典或聆聽理性的智慧教導，避免被碎片化的情緒資訊填滿。

3. **執行「三業」的同步修正**：
   ➔ 身行 ➔ 實踐一件具體的小善事（如幫忙他人）。
   ➔ 口行 ➔ 練習「正語」，停止批評，嘗試讚美或保持沈默。
   ➔ 意行 ➔ 練習「正思惟」，當負面念頭升起時，用「因緣」的角度分析，而非陷入情緒。

### [段落大意]
佛陀以月亮的盈虧比喻人的精神狀態。不善之人如同虧損的月亮，即便曾有信心與德行，但因遠離善友、不正思惟且造惡業，最終導致心靈枯竭並墮入惡趣。善之人如同盈滿的月亮，透過親近善知識、聽聞正法、持戒布施與正向思考，使智慧與德行日夜增長，最終獲得天上的果報。
