# 雜阿含經 第 31 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，尊者舍利弗在耆闍崛山。

時，有長者子名輸屢那，日日遊行，到耆闍崛山，詣舍利弗所，頭面禮足，却坐一面。

時，舍利弗謂：「輸屢那！若沙門、婆羅門於色不如實知，色集不如實知，色滅不如實知，色滅道跡不如實知故，輸屢那！當知此沙門、婆羅門不堪能斷色。如是沙門、婆羅門於受、想、行、識不如實知，識集不如實知，識滅不如實知，識滅道跡不如實知故，不堪能斷識。

「輸屢那！若沙門、婆羅門於色如實知，色集如實知，色滅如實知，色滅道跡如實知故，輸呂那！當知此沙門、婆羅門堪能斷色。如是，輸呂那！若沙門、婆羅門於受、想、行、識如實知，識集如實知，識滅如實知，識滅道跡如實知故，輸呂那！當知此沙門、婆羅門堪能斷識。

「輸屢那！於意云何？色為常、為無常耶？」

答言：「無常。」

又問：「若無常者，是苦耶？」

答言：「是苦。」

舍利弗言：「若色無常、苦者，是變易法，聖弟子寧於中見色是我、異我、相在不？」

答言：「不也。」

「輸屢那！如是受、想、行、識為常、為無常耶？」

答言：「無常。」

又問：「若無常者，是苦耶？」

答言：「是苦。」

又問：「若無常、苦者，是變易法，聖弟子寧於中見識是我、異我、相在不？」

答曰：「不也。」

「輸屢那！當知色，若過去、若未來、若現在，若內、若外，若麁、若細、若好、若醜，若遠、若近，於一切色不是我、不異我、不相在，是名如實知。輸屢那！聖弟子於色生厭、離欲、解脫，解脫生、老、病、死、憂、悲、苦、惱。

「如是受、想、行、識，若過去、若未來、若現在，若內、若外，若麁、若細，若好、若醜，若遠、若近，彼一切識不是我、不異我、不相在，是名如實知。輸屢那！聖弟子於識生厭、離欲、解脫，解脫生、老、病、死、憂、悲、苦、惱。」

時，輸屢那聞舍利弗所說，歡喜踊躍，作禮已，去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園中。

**◈ 爾時，尊者舍利弗在耆闍崛山。**
　　那時，尊者舍利弗住在耆闍崛山。

**◈ 時，有長者子名輸屢那，日日遊行，到耆闍崛山，詣舍利弗所，頭面禮足，却坐一面。**
　　當時，有一位名叫輸屢那的長者之子，每天遊歷到耆闍崛山，前往舍利弗尊者那裡，恭敬地低頭頂禮其足，然後在一旁坐下。

**◈ 時，舍利弗謂：「輸屢那！若沙門、婆羅門於色不如實知，色集不如實知，色滅不如實知，色滅道跡不如實知故，輸屢那！當知此沙門、婆羅門不堪能斷色。**
　　那時，舍利弗告訴他：「輸屢那！如果沙門或婆羅門對於色（物質身體）不能如實地認知，對於色（物質身體）如何聚集而生（苦因）不能如實地認知，對於色（物質身體）如何熄滅（苦止）不能如實地認知，對於導致色（物質身體）熄滅的道路（解脫之道）不能如實地認知，輸屢那！應當了解，這樣的沙門或婆羅門無法斷除對色（物質身體）的執著。

**◈ 如是沙門、婆羅門於受、想、行、識不如實知，識集不如實知，識滅不如實知，識滅道跡不如實知故，不堪能斷識。**
　　同樣地，如果沙門或婆羅門對於受（感受）、想（想像與認知）、行（心的造作）、識（意識）不能如實地認知，對於識（意識）如何聚集而生不能如實地認知，對於識（意識）如何熄滅不能如實地認知，對於導致識（意識）熄滅的道路不能如實地認知，就無法斷除對識（意識）的執著。

**◈ 「輸屢那！若沙門、婆羅門於色如實知，色集如實知，色滅如實知，色滅道跡如實知故，輸呂那！當知此沙門、婆羅門堪能斷色。**
　　輸屢那！如果沙門或婆羅門能如實地認知色（物質身體），如實地認知色（物質身體）的聚集原因，如實地認知色（物質身體）的熄滅，以及如實地認知導致色（物質身體）熄滅的道路，輸屢那！應當了解，這樣的沙門或婆羅門能夠斷除對色（物質身體）的執著。

**◈ 如是，輸呂那！若沙門、婆羅門於受、想、行、識如實知，識集如實知，識滅如實知，識滅道跡如實知故，輸呂那！當知此沙門、婆羅門堪能斷識。**
　　同樣地，輸呂那！如果沙門或婆羅門能如實地認知受（感受）、想（想像與認知）、行（心的造作）、識（意識），如實地認知識（意識）的聚集原因，如實地認知識（意識）的熄滅，以及如實地認知導致識（意識）熄滅的道路，輸呂那！應當了解，這樣的沙門或婆羅門能夠斷除對識（意識）的執著。

**◈ 「輸屢那！於意云何？色為常、為無常耶？」**
　　輸呂那！你的看法如何？色（物質身體）是恆常不變的，還是無常（隨時在變遷）的？

**◈ 答言：「無常。」**
　　輸屢那回答：「是無常的。」

**◈ 又問：「若無常者，是苦耶？」**
　　舍利弗又問：「如果它是無常的，那它是否就是苦（不圓滿且令人不安）的？」

**◈ 答言：「是苦。」**
　　輸呂那回答：「是苦的。」

**◈ 舍利弗言：「若色無常、苦者，是變易法，聖弟子寧於中見色是我、異我、相在不？」**
　　舍利弗說：「既然色（物質身體）是無常且是苦的，它就是一種不斷變易的現象，那麼聖弟子能在其中看到色（物質身體）就是『我』，或是『與我不同』，或是『我擁有它』的特徵嗎？」

**◈ 答言：「不也。」**
　　輸呂那回答：「不能。」

**◈ 「輸屢那！如是受、想、行、識為常、為無常耶？」**
　　輸呂那！同樣地，受（感受）、想（想像與認知）、行（心的造作）、識（意識）是恆常不變的，還是無常的？

**◈ 答言：「無常。」**
　　輸呂那回答：「是無常的。」

**◈ 又問：「若無常者，是苦耶？」**
　　又問：「如果它是無常的，那它是否就是苦的？」

**◈ 答言：「是苦。」**
　　輸呂那回答：「是苦的。」

**◈ 又問：「若無常、苦者，是變易法，聖弟子寧於中見識是我、異我、相在不？」**
　　又問：「既然它是無常且是苦的，它就是一種不斷變易的現象，那麼聖弟子能在其中看到識（意識）就是『我』，或是『與我不同』，或是『我擁有它』的特徵嗎？」

**◈ 答曰：「不也。」**
　　輸呂那回答：「不能。」

**◈ 「輸呂那！當知色，若過去、若未來、若現在，若內、若外，若麁、若細、若好、若醜，若遠、若近，於一切色不是我、不異我、不相在，是名如實知。**
　　輸呂那！應當了解，無論是過去、未來或現在的色（物質身體）；無論是內在的或外在的；無論是粗大的或微細的；無論是美好的或醜陋的；無論是遙遠的或近在咫尺的，對於所有這些色（物質身體）都能體悟到：它不是我，也不異於我，更沒有一個『我』存在其中，這就叫做如實地認知。

**◈ 輸呂那！聖弟子於色生厭、離欲、解脫，解脫生、老、病、死、憂、悲、苦、惱。**
　　輸呂那！聖弟子對色（物質身體）產生厭離心，脫離了貪欲，從而獲得解脫，進而從出生、衰老、疾病、死亡以及憂愁、悲傷、痛苦、煩惱中解脫出來。

**◈ 「如是受、想、行、識，若過去、若未來、若現在，若內、若外，若麁、若細，若好、若醜，若遠、若近，彼一切識不是我、不異我、不相在，是名如實知。**
　　同樣地，對於受（感受）、想（想像與認知）、行（心的造作）、識（意識），無論是過去、未來或現在；無論是內在的或外在的；無論是粗大的或微細的；無論是美好的或醜陋的；無論是遙遠的或近在咫尺的，對所有這些識（意識）都能體悟到：它不是我，也不異於我，更沒有一個『我』存在其中，這就叫做如實地認知。

**◈ 輸呂那！聖弟子於識生厭、離欲、解脫，解脫生、老、病、死、憂、悲、苦、惱。」**
　　輸呂那！聖弟子對識（意識）產生厭離心，脫離了貪欲，從而獲得解脫，進而從出生、衰老、疾病、死亡以及憂愁、悲傷、痛苦、煩惱中解脫出來。」

**◈ 時，輸呂那聞舍利弗所說，歡喜踊躍，作禮已，去。**
　　那時，輸呂那聽完舍利弗所說的教法，心中感到無比歡喜，行禮之後便離開了。

### [詞語註解（解釋重要佛教概念）]
1. **五蘊 (Khandhas)**：構成生命個體的五種要素，包括色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識）。本經強調這五者皆非恆常且非自我。
2. **如實知 (Yathā-bhūta-ñāṇa)**：對事物真實本質的正確覺知，不被主觀偏見或幻想所遮蔽。
3. **色集、色滅、色滅道跡**：將「四聖諦」應用於五蘊。色集 ➔ 導致物質身體執著的因（貪愛與無明）；色滅 ➔ 執著的熄滅；色滅道跡 ➔ 達成熄滅的修行路徑（八正道）。
4. **不是我、不異我、不相在 (Anattā)**：這是對「無我」的詳細定義。不是我 ➔ 身體/意識本身不是永恆的自我；不異我 ➔ 並非有一個獨立的自我與身體/意識分開；不相在 ➔ 並非自我擁有身體/意識。
5. **厭 (Nibbidā)**：並非指世俗的討厭，而是指在洞察事物無常與苦後，產生的冷靜、不再執著的心理狀態。

### [現代實作指南]
本經提供了一套邏輯嚴密的「去中心化」心理練習，可用於緩解對身體老化、疾病或情緒波動的焦慮：

1. **標記現象 (Labeling)**：
   當強烈的情緒（受）或固執的念頭（想）出現時，不要說「我很痛苦」或「我在生氣」，而是在心中標記：「現在有一個『受（感受）』在運作」或「現在有一個『想（認知）』在產生」。將「我」與「現象」分離。

2. **邏輯推演 (Logical Inquiry)**：
   對著當下的壓力源問自己三個問題：
   - ➔ 「這個感覺/想法是恆常不變的嗎？」(發現其無常)
   - ➔ 「既然它在變，它能給我絕對的滿足與安全感嗎？」(發現其苦)
   - ➔ 「如果它在變且不穩定，它能被定義為『我』或『我的』嗎？」(體悟無我)

3. **擴展觀察 (Expanding Observation)**：
   練習將觀察對象從「內在」擴展到「外在」。例如，觀察身體的疼痛（內在色）與觀察窗外的枯葉（外在色），體悟兩者同樣遵循「無常 ➔ 苦 ➔ 非我」的規律，從而降低對特定痛苦的執著。

4. **轉化為厭離 (Cultivating Detachment)**：
   當意識到所有身心現象（五蘊）都像流水一樣不停變換且無法掌控時，自然會對「試圖掌控不可控之物」的行為感到疲倦（厭），進而放手（離欲），獲得內心的寧靜（解脫）。

### [段落大意]
本經記述舍利弗尊者指導輸呂那，說明若要斷除對身心五蘊的執著，必須如實認知五蘊的生起、熄滅及其路徑。透過對五蘊「無常」與「苦」的邏輯推演，證悟五蘊皆非自我（無我），進而產生厭離心，最終從生老病死與憂悲苦惱中獲得徹底的解脫。
