# 雜阿含經 第 457 經

### [原文]
如是我聞：

一時，佛住舍衛國東園鹿子母講堂。

爾時，世尊晡時從禪覺，於講堂陰中敷座，於大眾前坐，說優檀那句，告諸比丘：「緣界故生說，非不界，緣界故生見，非不界，緣界故生想，非不界。緣下界，我說生下說、下見、下想、下思、下欲、下願、下士夫、下所作、下施設、下建立、下部分、下顯示、下受生。如是中。如是勝界，緣勝界，我說彼生勝說、勝見、勝想、勝思、勝願、勝士夫、勝所作、勝施設、勝建立、勝部分、勝顯示、勝受生勝。」

時，有婆迦利比丘在佛後執扇扇佛，白佛言：「世尊！若於三藐三佛陀起非三藐三佛陀見，彼見亦緣界而生耶？」

佛告比丘：「於三藐三佛陀起非三藐三佛陀見，亦緣界而生，非不界。所以者何？凡夫界者，是無明界，如我先說，緣下界生下說、下見，乃至下受生，中勝界生勝說、勝見，乃至勝受生。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國東園鹿子母講堂。**
　　在某一個時間，佛陀住在舍衛國東園的鹿子母講堂中。

**◈ 爾時，世尊晡時從禪覺，於講堂陰中敷座，於大眾前坐，說優檀那句，告諸比丘：**
　　那時，世尊在傍晚時分從禪定中覺醒，在講堂的陰涼處鋪好座位，坐在大眾面前，說了一段偈頌（優檀那句），然後告訴在場的眾位比丘：

**◈ 「緣界故生說，非不界，緣界故生見，非不界，緣界故生想，非不界。**
　　「因為依緣於界（構成現象的基礎元素或領域）而產生了說法（言語表達），而非不依緣於界而產生；因為依緣於界而產生了見（對事物的看法），而非不依緣於界而產生；因為依緣於界而產生了想（想像與認知），而非不依緣於界而產生。

**◈ 緣下界，我說生下說、下見、下想、下思、下欲、下願、下士夫、下所作、下施設、下建立、下部分、下顯示、下受生。**
　　依緣於下界（低劣的心靈領域），我說會產生低劣的說法、低劣的見解、低劣的想（想像與認知）、低劣的思維、低劣的欲求、低劣的願望、低劣的人格特質、低劣的行為、低劣的制度、低劣的建立、低劣的組成部分、低劣的表現以及低劣的出生狀態。

**◈ 如是中。**
　　中界（中等的心靈領域）也是同樣的道理。

**◈ 如是勝界，緣勝界，我說彼生勝說、勝見、勝想、勝思、勝願、勝士夫、勝所作、勝施設、勝建立、勝部分、勝顯示、勝受生勝。」**
　　同樣地，對於勝界（優越的心靈領域），依緣於勝界，我說會產生優越的說法、優越的見解、優越的想（想像與認知）、優越的思維、優越的願望、優越的人格特質、優越的行為、優越的制度、優越的建立、優越的組成部分、優越的表現以及優越的出生狀態。」

**◈ 時，有婆迦利比丘在佛後執扇扇佛，白佛言：「世尊！若於三藐三佛陀起非三藐三佛陀見，彼見亦緣界而生耶？」**
　　那時，有一位名叫婆迦利的比丘在佛陀身後執扇為佛陀扇風，他對佛陀恭敬地說道：「世尊！如果一個人對於最高、最圓滿的智慧與覺悟者（三藐三佛陀）產生了『他不是三藐三佛陀』的錯誤見解，這種見解也是依緣於界而產生的嗎？」

**◈ 佛告比丘：「於三藐三佛陀起非三藐三佛陀見，亦緣界而生，非不界。**
　　佛陀告訴比丘：「對於最高、最圓滿的智慧與覺悟者（三藐三佛陀）產生『他不是三藐三佛陀』的見解，同樣也是依緣於界而產生，而非不依緣於界。

**◈ 所以者何？凡夫界者，是無明界，如我先說，緣下界生下說、下見，乃至下受生，中勝界生勝說、勝見，乃至勝受生。」**
　　為什麼呢？因為凡夫（未覺悟之人）的界，就是無明界（缺乏真理覺知、被蒙蔽的基礎領域）。正如我先前所說，依緣於下界會產生低劣的說法、低劣的見解，乃至於低劣的出生狀態；依緣於中界或勝界，則會產生中等或優越的說法、見解，乃至於優越的出生狀態。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **界 (Dhātu)**：指構成現象的基礎元素、領域或本質。在本文中，指心靈的底層狀態或認知框架，決定了個體如何看待世界並產生相應的反應。
2. **三藐三佛陀 (Samyak-Sambuddha)**：指完全覺悟的佛陀，擁有最高、最圓滿的智慧，能獨立發現真理並教導他人。
3. **無明 (Avijjā)**：指對四聖諦（苦、集、滅、道）的無知，是導致輪迴與痛苦的根本原因。
4. **凡夫 (Putujjana)**：指尚未證得聖果、仍被煩惱與無明所束縛的普通人。

### [現代實作指南]
本經文揭示了一個核心邏輯：**「認知框架（界） $\text{→}$ 認知結果（見/想/說）」**。我們對世界的看法並非客觀事實，而是由我們內在的「界」（心靈底層的傾向、偏見、知識儲備或情緒狀態）所決定。

**實作步驟：**
1. **覺察認知偏差**：當你對某人產生強烈的負面評價（如：認為對方不可理喻）時，試著問自己：「我現在是處在什麼樣的『界』之中？」（例如：我是否正處於疲憊、憤怒或先入為主的偏見之界？）。
2. **分析因果鏈條**：意識到你的「見解」 $\text{→}$ 「言語」 $\text{→}$ 「行為」全部源自於底層的「界」。如果你不滿意目前的反應，不要試圖強行改變結果，而要試著改變底層的「界」。
3. **提升心靈領域**：
    *   **從下界 $\text{→}$ 中界**：透過學習、閱讀與反思，將「無明」轉化為「知曉」，減少盲目的偏見。
    *   **從中界 $\text{→}$ 勝界**：透過修行（如正念、慈悲心），將自私的視角轉化為利他的視角，從而產生優越的見解與行為。
4. **對待分歧的轉念**：當他人對你產生誤解時，理解對方是「緣其界而生其見」。對方的誤解並非針對你的本質，而是受限於他目前的認知領域（界）。這能幫助我們放下憤怒，生起慈悲心。

### [段落大意]
本經論述了「界」（認知基礎）與心靈產物（說、見、想等）之間的因果關係。佛陀指出，所有言語、見解與認知皆依緣於不同的心靈領域（下、中、勝界）而生。即便對佛陀產生錯誤的見解，也是因為該個體處於「無明」的凡夫界之中。因此，個體的認知品質取決於其心靈底層的狀態。
