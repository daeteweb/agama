# 雜阿含經 第 231 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有比丘名三彌離提，往詣佛所，稽首佛足，退坐一面，白佛言：「世尊！所謂世間者，云何名世間？」

佛告三彌離提：「危脆敗壞，是名世間。云何危脆敗壞？三彌離提！眼是危脆敗壞法，若色、眼識、眼觸、眼觸因緣生受，內覺若苦、若樂、不苦不樂，彼一切亦是危脆敗壞。耳、鼻、舌、身、意亦復如是。是說危脆敗壞法，名為世間。」

佛說此經已，三彌離提比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　有一次，佛住在舍衛國的祇樹給孤獨園。

**◈ 時，有比丘名三彌離提，往詣佛所，稽首佛足，退坐一面，白佛言：「世尊！所謂世間者，云何名世間？」**
　　那時，有一位名叫三彌離提的比丘，前往佛那裡，頂禮佛的足，退到一旁坐下，對佛說：「世尊！所謂的『世間』，該如何稱之為世間呢？」

**◈ 佛告三彌離提：「危脆敗壞，是名世間。云何危脆敗壞？三彌離提！眼是危脆敗壞法，若色、眼識、眼觸、眼觸因緣生受，內覺若苦、若樂、不苦不樂，彼一切亦是危脆敗壞。耳、鼻、舌、身、意亦復如是。是說危脆敗壞法，名為世間。」**
　　佛告訴三彌離提：「那種具有危險、脆弱且易於敗壞的特性，就是所謂的世間。三彌離提！眼睛（感官器官）就是一種危脆且易於敗壞的現象；當色（物質身體/對象）、眼識（視覺意識）、眼觸（感官與對象的接觸），以及由這些因緣所產生的受（感受）——無論是痛苦的、快樂的，或是非苦非樂的中性感受時，這一切也都是危脆且易於敗壞的。耳朵、鼻子、舌頭、身體、意念也是同樣的道理。說這些具有危脆敗壞特性的現象，就是所謂的世間。」

**◈ 佛說此經已，三彌離提比丘聞佛所說，歡喜奉行。**
　　佛說完這部經後，三彌離提比丘聽聞了佛所說的話，感到非常歡喜並遵從修行。

### [詞語註解]
1. **世間 (Loka/Saṃsāra)**：指一切有為法、受因緣牽引而存在的現象界，其本質是不穩定且處於不斷變遷中的狀態。
2. **危脆敗壞 (Viparinama-dukkha)**：指事物因無常而必然經歷改變、崩解與毀滅的特性，是苦（Dukkha）的一種表現形式。
3. **眼觸 (Cakkhu-phassa)**：指視覺感官、視覺對象與視覺意識三者結合產生的接觸作用。
4. **因緣 (Paṭicca-samuppāda)**：指事物並非獨立存在，而是由各種條件（因）與環境（緣）相互依存而生起的規律。

### [現代實作指南]
本經教導我們透過「感官解構」來觀察世界的本質，以對抗過度的情緒起伏或執著：

1. **感官拆解練習**：當你感到強烈的情緒（如極度憤怒或狂喜）時，不要直接認同這個情緒。試著將其拆解為：
   → 「我看到了什麼顏色/形狀」（色） 
   → 「我的眼睛接收到了資訊」（眼識） 
   → 「視覺與對象接觸了」（觸） 
   → 「產生了不舒服的感覺」（受）。
2. **觀察「脆弱性」**：在享受快樂時，提醒自己這種感受是「危脆敗壞」的，它依賴於特定的條件（如環境、身體狀態），一旦條件改變，感受必然消失。這不是為了消極，而是為了建立心理韌性，避免在失去時產生巨大的痛苦。
3. **轉念技巧**：當面對負面刺激時，將其視為一種「正在發生且即將消逝的物理過程」，而非「我」所擁有的痛苦。
