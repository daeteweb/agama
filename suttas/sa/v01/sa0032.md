# 雜阿含經 第 32 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。爾時，尊者舍利弗在耆闍崛山。

時，有長者子名輸屢那，日日遊行，到耆闍崛山，詣舍利弗所，頭面禮足，却坐一面。

時，舍利弗告輸屢那：「若沙門、婆羅門於色不如實知，色集不如實知，色滅不如實知，色味不如實知，色患不如實知，色離不如實知故，不堪能超越色。若沙門、婆羅門於受、想、行、識不如實知，識集不如實知，識滅不如實知，識味不如實知，識患不如實知，識離不如實知故，此沙門、婆羅門不堪能超越識。

「若沙門、婆羅門於色、色集、色滅、色味、色患、色離如實知，此沙門、婆羅門堪能超越色。若沙門、婆羅門於受、想、行、識、識集、識滅、識味、識患、識離如實知，此沙門、婆羅門堪能超越識。

「輸屢那！於汝意云何？色為常、為無常耶？」

答言：「無常。」

「無常者，為苦耶？」

答言：「是苦。」

「輸屢那！若色無常、苦，是變易法，聖弟子於中寧有是我、異我、相在不？」

答言：「不也。」

「輸屢那！於汝意云何？如是受、想、行、識為常、為無常？」

答言：「無常。」

「若無常者，是苦耶？」

答言：「是苦。」

「輸屢那！若無常、苦，是變易法，聖弟子於中寧有是我、異我、相在不？」答言：「不也。」

「輸屢那！當知色，若過去、若未來、若現在，若內、若外，若麁、若細，若好、若醜，若遠、若近，於一切色不是我、不異我、不相在，是名如實知。

「輸屢那！受、想、行、識，若過去、若未來、若現在，若內、若外，若麁、若細，若好、若醜，若遠、若近，於一切識不是我、不異我、不相在，是名如實知。

「輸屢那！聖弟子於此五受陰正觀非我、非我所。如是正觀，於諸世間無所攝受；無攝受者，則無所著；無所著者，自得涅槃：『我生已盡，梵行已立，所作已作，自知不受後有。』」

時，長者子輸屢那聞舍利弗所說，歡喜踊躍，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。爾時，尊者舍利弗在耆闍崛山。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。那時，舍利弗尊者則在耆闍崛山。

**◈ 時，有長者子名輸屢那，日日遊行，到耆闍崛山，詣舍利弗所，頭面禮足，却坐一面。**
　　當時，有一位名叫輸屢那的長者之子，每天在附近遊行，來到耆闍崛山拜訪舍利弗尊者，他恭敬地低頭禮拜尊者的足部，然後在一旁坐下。

**◈ 時，舍利弗告輸屢那：「若沙門、婆羅門於色不如實知，色集不如實知，色滅不如實知，色味不如實知，色患不如實知，色離不如實知故，不堪能超越色。**
　　那時，舍利弗告訴輸屢那：「如果沙門或婆羅門對於色（物質身體）不能如實地知曉，不能如實地知曉色（物質身體）是如何聚集而成的（色集），不能如實地知曉色（物質身體）是如何滅去的（色滅），不能如實地知曉色（物質身體）所帶來的快感（色味），不能如實地知曉色（物質身體）所隱含的缺陷與危險（色患），以及不能如實地知曉如何脫離色（物質身體）的束縛（色離），那麼，這位沙門或婆羅門就無法超越色（物質身體）的限制。

**◈ 若沙門、婆羅門於受、想、行、識不如實知，識集不如實知，識滅不如實知，識味不如實知，識患不如實知，識離不如實知故，此沙門、婆羅門不堪能超越識。**
　　同樣地，如果沙門或婆羅門對於受（感受）、想（想像與認知）、行（心的造作）、識（意識）不能如實地知曉，且不能如實地知曉識（意識）的聚集、滅去、快感、缺陷與危險以及脫離的方法，那麼，這位沙門或婆羅門就無法超越識（意識）的限制。

**◈ 若沙門、婆羅門於色、色集、色滅、色味、色患、色離如實知，此沙門、婆羅門堪能超越色。**
　　反之，如果沙門或婆羅門能如實地知曉色（物質身體）本身及其聚集、滅去、快感、缺陷與脫離的方法，這位沙門或婆羅門就能夠超越色（物質身體）。

**◈ 若沙門、婆羅門於受、想、行、識、識集、識滅、識味、識患、識離如實知，此沙門、婆羅門堪能超越識。**
　　如果沙門或婆羅門能如實地知曉受（感受）、想（想像與認知）、行（心的造作）、識（意識）本身，以及識（意識）的聚集、滅去、快感、缺陷與脫離的方法，這位沙門或婆羅門就能夠超越識（意識）。

**◈ 輸屢那！於汝意云何？色為常、為無常耶？」**
　　輸屢那！你的看法如何？你認為色（物質身體）是永恆不變的，還是無常（隨時在變遷）的？

**◈ 答言：「無常。」**
　　輸屢那回答：「是無常的。」

**◈ 「無常者，為苦耶？」**
　　舍利弗問：「既然是無常的，那它是否就是苦（不滿足且令人不安）的？」

**◈ 答言：「是苦。」**
　　輸屢那回答：「是的，是苦。」

**◈ 「輸屢那！若色無常、苦，是變易法，聖弟子於中寧有是我、異我、相在不？」**
　　舍利弗問：「輸屢那！既然色（物質身體）是無常且是苦的，是一種不斷變易的現象，那麼聖弟子在其中，是否還能認為『這是我』、『這不是我但與我有關』，或者認為有一個『我的特徵』存在？」

**◈ 答言：「不也。」**
　　輸呂那回答：「不，不能。」

**◈ 「輸屢那！於汝意云何？如是受、想、行、識為常、為無常？」**
　　舍利弗問：「輸屢那！你的看法如何？同樣地，受（感受）、想（想像與認知）、行（心的造作）、識（意識）是永恆的，還是無常的？」

**◈ 答言：「無常。」**
　　輸呂那回答：「是無常的。」

**◈ 「若無常者，是苦耶？」**
　　舍利弗問：「既然無常，是否就是苦？」

**◈ 答言：「是苦。」**
　　輸呂那回答：「是的，是苦。」

**◈ 「輸呂那！若無常、苦，是變易法，聖弟子於中寧有是我、異我、相在不？」答言：「不也。」**
　　舍利弗問：「輸呂那！既然它們是無常且是苦的，是一種不斷變易的現象，那麼聖弟子在其中，是否還能認為『這是我』、『這不是我但與我有關』，或者認為有一個『我的特徵』存在？」輸呂那回答：「不，不能。」

**◈ 「輸呂那！當知色，若過去、若未來、若現在，若內、若外，若麁、若細，若好、若醜，若遠、若近，於一切色不是我、不異我、不相在，是名如實知。**
　　舍利弗說：「輸呂那！你應當了解，無論是過去、未來或現在的色（物質身體）；無論是內在的還是外在的；無論是粗糙的還是細微的；無論是美好的還是醜陋的；無論是遙遠的還是近在咫尺的，對於所有這些色（物質身體），都應體悟到它們『不是我』、『不異於我』且『沒有我的特徵』，這才叫做如實地知曉。

**◈ 輸呂那！受、想、行、識，若過去、若未來、若現在，若內、若外，若麁、若細，若好、若醜，若遠、若近，於一切識不是我、不異我、不相在，是名如實知。**
　　輸呂那！對於受（感受）、想（想像與認知）、行（心的造作）、識（意識），無論是過去、未來或現在；無論是內在還是外在；無論是粗糙還是細微；無論是美好還是醜陋；無論是遙遠還是近在咫尺，對於所有這些識（意識），都應體悟到它們『不是我』、『不異於我』且『沒有我的特徵』，這才叫做如實地知曉。

**◈ 輸呂那！聖弟子於此五受陰正觀非我、非我所。**
　　輸呂那！聖弟子對於這構成身心的五種要素（五蘊），正確地觀察到它們既不是「我」，也不是「我所（我擁有的東西）」。

**◈ 如是正觀，於諸世間無所攝受；無攝受者，則無所著；無所著者，自得涅槃：『我生已盡，梵行已立，所作已作，自知不受後有。』」**
　　透過這樣正確的觀察，對世間的一切現象不再產生執著與認同（無所攝受）；不再認同執著的人，就沒有了攀緣與掛礙（無所著）；沒有了攀緣與掛礙的人，自然能進入超越生死、絕對寧靜的解脫境界 (涅槃)，並能體悟到：「我的輪迴出生已盡，清淨的修行已成就，應該做的事已完成，深知自己不再受後世的出生。」

**◈ 時，長者子輸呂那聞舍利弗所說，歡喜踊躍，作禮而去。**
　　那時，長者之子輸呂那聽完舍利弗的教導，心中法喜充滿，恭敬地行禮後離開了。

### [詞語註解（解釋重要佛教概念）]
1. **如實知 (Yathā-bhūta-ñāṇa-dassana)**：指不經過主觀偏見、不被貪愛或厭惡所遮蔽，直接看到事物本質（如無常、苦、無我）的洞察力。
2. **色集、色滅 (Samudaya / Nirodha of Rūpa)**：指物質身體產生的因緣（如業力、營養、溫度等）以及其消亡的過程。
3. **色味、色患、色離 (Rasa / Ādīnava / Nissaraṇa)**：這是一種分析法。味 ➔ 指物質身體帶來的感官快感；患 ➔ 指物質身體必然面臨的衰老、疾病與死亡之苦；離 ➔ 指透過智慧脫離對物質身體的執著。
4. **五受陰 (Pañca-upādānakkhandha)**：即五蘊（色、受、想、行、識）在被貪愛與執取（Upādāna）之後的狀態，強調人們如何將這五種要素誤認為「我」。
5. **非我、非我所 (Anattā / Anattā-vāda)**：非我 ➔ 這些要素本身不是永恆的自我；非我所 ➔ 這些要素也不是屬於我的財產或所有物。

### [現代實作指南]
本經文提供了一套極其理性的「去認同化」操作流程，可用於處理現代人的焦慮與壓力：

1. **分析「快感」與「缺陷」 (味與患)**：
   當你極度渴望某樣物質（如名車、名聲）或執著於某種感受（如被讚美）時，先分析其「味」（它能帶給我什麼短暫快感？），接著立刻分析其「患」（維持這個快感需要付出什麼代價？它消失時會帶來什麼痛苦？）。透過對比，降低對該對象的盲目貪愛。

2. **執行「無常 → 苦 → 無我」的邏輯推演**：
   - **觀察變遷**：意識到此刻的情緒（受）或想法（想）正在快速變換 → 結論：它是無常的。
   - **體會不安**：意識到因為它在變，所以我無法完全掌控它，導致內心不安 → 結論：它是苦的。
   - **撤回認同**：既然它是不斷變動且不可掌控的，它怎麼可能是那個永恆、穩定、主宰一切的「我」？ → 結論：它不是我。

3. **全方位掃描 (內外、粗細、好醜)**：
   在面對強烈情緒時，嘗試將其客觀化。告訴自己：「這是一個『現在』產生的、『內在』的、一種『細微』的、令人『不快』的意識現象」，而不是說「我很痛苦」。將「我」從情緒中抽離，轉變為「觀察者」視角。

### [段落大意]
舍利弗尊者指導輸呂那，若要超越物質（色）與意識（識）的束縛，必須如實知曉其產生、滅去、快感、缺陷與脫離之法。透過邏輯推演證明五蘊皆是無常且是苦的，因此不能被視為「我」。聖弟子藉由正觀五蘊非我，消除對世間的執著，最終達成涅槃的解脫。
