# 雜阿含經 第 64 經

### [原文]
如是我聞：

一時，佛住舍衛國東園鹿子母講堂。

爾時，世尊晡時從禪起，出講堂，於堂陰中大眾前，敷座而坐。爾時，世尊歎優陀那偈：

「法無有吾我，　　亦復無我所； 　我既非當有，　　我所何由生？ 　比丘解脫此，　　則斷下分結。」

時，有一比丘從座起，偏袒右肩，右膝著地，合掌白佛言：「世尊！云何『無吾我，亦無有我所；我既非當有，我所何由生？比丘解脫此，則斷下分結。』？」

佛告比丘：「愚癡無聞凡夫計色是我、異我、相在；受、想、行、識，是我、異我、相在。多聞聖弟子不見色是我、異我、相在，不見受、想、行、識，是我、異我、相在；亦非知者，亦非見者。此色是無常，受、想、行、識是無常；色是苦，受、想、行、識是苦；色是無我，受、想、行、識是無我；此色非當有，受、想、行、識非當有；此色壞有，受、想、行、識壞有；故非我、非我所，我、我所非當有。如是解脫者，則斷五下分結。」

時，彼比丘白佛言：「世尊！斷五下分結已，云何漏盡，無漏心解脫、慧解脫，現法自知作證具足住：『我生已盡，梵行已立，所作已作，自知不受後有。』？」

佛告比丘：「愚癡凡夫、無聞眾生於無畏處而生恐畏。愚癡凡夫、無聞眾生怖畏，無我、無我所，二俱非當有。

「攀緣四識住。何等為四？謂色識住、色攀緣、色愛樂、增進廣大生長；於受、想、行，識住、攀緣、愛樂、增進廣大生長。比丘！識於此處，若來、若去、若住、若起、若滅，增進廣大生長。若作是說：『更有異法，識若來、若去、若住、若起、若滅、若增進廣大生長。』者，但有言說，問已不知，增益生癡，以非境界故。所以者何？比丘！離色界貪已，於色意生縛亦斷，於色意生縛斷已，識攀緣亦斷；識不復住，無復增進廣大生長，受、想、行界離貪已，於受、想、行意生縛亦斷。受、想、行意生縛斷已，攀緣亦斷，識無所住，無復增進廣大生長。識無所住故不增長，不增長故無所為作，無所為作故則住，住故知足，知足故解脫，解脫故於諸世間都無所取，無所取故無所著，無所著故自覺涅槃：『我生已盡，梵行已立，所作已作，自知不受後有。』比丘！我說識不住東方、南、西、北方、四維、上、下，除欲見法，涅槃、滅盡、寂靜、清涼。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

　生滅以不樂　　及三種分別 　貪著等觀察　　是名優陀那

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國東園鹿子母講堂。**
　　在某一個時間，佛陀住在舍衛國東園的鹿子母講堂。

**◈ 爾時，世尊晡時從禪起，出講堂，於堂陰中大眾前，敷座而坐。爾時，世尊歎優陀那偈：**
　　那時，世尊在傍晚時分從禪定中醒來，走出講堂，在講堂陰涼處的大眾面前鋪好座位坐下。接著，世尊感嘆地說出了一首優陀那偈（隨興而發的覺悟詩偈）：

**◈ 「法無有吾我，亦復無我所；我既非當有，我所何由生？比丘解脫此，則斷下分結。」**
　　「法（一切現象）之中並沒有一個永恆不變的『我』，也沒有任何屬於『我』的所有物；既然一個實體性的『我』根本不會存在，那麼所謂『我的東西』又怎麼會產生呢？比丘若能從此觀念中解脫，就能斷除下分結（束縛凡夫的五種低階煩惱）。」

**◈ 時，有一比丘從座起，偏袒右肩，右膝著地，合掌白佛言：「世尊！云何『無吾我，亦無有我所；我既非當有，我所何由生？比丘解脫此，則斷下分結。』？」**
　　這時，有一位比丘從座位上站起來，整理衣服並露出右肩，右膝跪在地上，雙手合十向著佛陀恭敬地說道：「世尊！請問您所說的『沒有我，也沒有我所；我既然不存在，我所如何產生？比丘解脫此，則斷下分結』是什麼意思？」

**◈ 佛告比丘：「愚癡無聞凡夫計色是我、異我、相在；受、想、行、識，是我、異我、相在。」**
　　佛陀告訴比丘：「愚癡且缺乏聞法機會的凡夫，會誤以為色（物質身體）就是『我』，或者認為色（物質身體）與『我』不同但彼此相關，或者認為色（物質身體）具有恆常的自性；對於受（感受）、想（想像與認知）、行（心的造作）、識（意識），也同樣誤以為它們就是『我』、與『我』不同，或具有恆常自性。」

**◈ 「多聞聖弟子不見色是我、異我、相在，不見受、想、行、識，是我、異我、相在；亦非知者，亦非見者。」**
　　「但博學的聖弟子則不會將色（物質身體）視為『我』、異於『我』或具有自性；也不會將受（感受）、想（想像與認知）、行（心的造作）、識（意識）視為『我』、異於『我』或具有自性；且明白這些要素既不是覺知的主體，也不是觀察的主體。」

**◈ 「此色是無常，受、想、行、識是無常；色是苦，受、想、行、識是苦；色是無我，受、想、行、識是無我；此色非當有，受、想、行、識非當有；此色壞有，受、想、行、識壞有；故非我、非我所，我、我所非當有。如是解脫者，則斷五下分結。」**
　　「色（物質身體）是無常（Anicca）的，受（感受）、想（想像與認知）、行（心的造作）、識（意識）也是無常的；色（物質身體）是苦（Dukkha）的，受、想、行、識也是苦的；色（物質身體）是無我（Anattā）的，受、想、行、識也是無我的；色（物質身體）並非實有，受、想、行、識亦非實有；色（物質身體）處於崩壞的過程，受、想、行、識亦處於崩壞的過程；因此，它們既不是『我』，也不是『我的』，而『我』與『我的』根本不可能存在。能如此解脫的人，就能斷除五種下分結。」

**◈ 時，彼比丘白佛言：「世尊！斷五下分結已，云何漏盡，無漏心解脫、慧解脫，現法自知作證具足住：『我生已盡，梵行已立，所作已作，自知不受後有。』？」**
　　這時，那位比丘對佛陀恭敬地說道：「世尊！在斷除五種下分結之後，如何才能達到漏盡（煩惱完全消失），實現心靈的解脫與智慧的解脫，並在現法（當下生命）中自證並圓滿地處於：『我的輪迴出生已盡，清淨的梵行已建立，應做的事已完成，自知不再受後世出生』的狀態？」

**◈ 佛告比丘：「愚癡凡夫、無聞眾生於無畏處而生恐畏。愚癡凡夫、無聞眾生怖畏，無我、無我所，二俱非當生。」**
　　佛陀告訴比丘：「愚癡且缺乏聞法機會的眾生，即便處在無畏之處（空性或涅槃境界）也會產生恐懼。他們之所以恐懼，是因為執著於『我』與『我所』，但這兩者實際上根本不會產生。」

**◈ 「攀緣四識住。何等為四？謂色識住、色攀緣、色愛樂、增進廣大生長；於受、想、行，識住、攀緣、愛樂、增進廣大生長。」**
　　「他們依止於四種意識的運作。什麼是四種？第一是識（意識）停留在色（物質身體）上，對色（物質身體）產生攀緣、貪愛與愛樂，導致執著心增進並廣大生長；第二、三、四則是識（意識）同樣停留在受（感受）、想（想像與認知）、行（心的造作）之上，對它們產生攀緣、愛樂，使其執著心增進並廣大生長。」

**◈ 「比丘！識於此處，若來、若去、若住、若起、若滅，增進廣大生長。」**
　　「比丘！識（意識）在這些對象中，無論是產生、消失、停留、興起或滅盡，都會隨著貪愛而增進並廣大生長。」

**◈ 「若作是說：『更有異法，識若來、若去、若住、若起、若滅、若增進廣大生長。』者，但有言說，問已不知，增益生癡，以非境界故。」**
　　「如果有人主張：『除了這些五蘊之外，還有另一種不同的法，讓識（意識）能在那裡來去、停留、興起、滅盡或增長』，那僅僅是空談，被追問後也無法回答，反而會增加愚癡，因為那並非真實存在的境界。」

**◈ 「所以者何？比丘！離色界貪已，於色意生縛亦斷，於色意生縛斷已，識攀緣亦斷；識不復住，無復增進廣大生長，受、想、行界離貪已，於受、想、行意生縛亦斷。」**
　　「為什麼呢？比丘！一旦對色（物質身體）界的貪愛消失，對色的心理束縛也就斷了；束縛斷了，識（意識）的攀緣也就斷了；識（意識）不再停留，也就不再增進廣大生長。同樣地，一旦對受（感受）、想（想像與認知）、行（心的造作）界的貪愛消失，對它們的心理束縛也就斷了。」

**◈ 「受、想、行意生縛斷已，攀緣亦斷，識無所住，無復增進廣大生長。」**
　　「當受、想、行的心理束縛斷除後，攀緣也隨之斷除，識（意識）再也沒有依附之處，因此不再增進廣大生長。」

**◈ 「識無所住故不增長，不增長故無所為作，無所為作故則住，住故知足，知足故解脫，解脫故於諸世間都無所取，無所取故無所著，無所著故自覺涅槃：『我生已盡，梵行已立，所作已作，自知不受後有。』」**
　　「因為識（意識）無所住，所以不再增長；不再增長，所以不再造業（無所為作）；不再造業，心便能安定（住）；安定後便能知足；知足後便能解脫；解脫後對世間一切現象都無所執取；無所執取，便無所著染；無所著染，便能自覺地進入涅槃（Nibbāna）：『我的輪迴出生已盡，清淨的梵行已建立，應做的事已完成，自知不再受後世出生』。」

**◈ 「比丘！我說識不住東方、南、西、北方、四維、上、下，除欲見法，涅槃、滅盡、寂靜、清涼。」**
　　「比丘！我告訴你們，識（意識）並不住在東、南、西、北四方，也不住在四維、上方或下方。只要除掉對欲望的見解（欲見法），就能達到涅槃、滅盡、寂靜與清涼的境界。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 生滅以不樂　及三種分別 　貪著等觀察　　是名優陀那**
　　（結語：觀察生滅之不快樂，以及對三種分別與貪著的觀察，這就叫做優陀那。）

### [詞語註解（解釋重要佛教概念）]
1. **下分結 (Orambhagiya-samyojana)**：指束縛凡夫在欲界輪迴的五種低階煩惱，包括：身見（認為有恆常的我）、疑（對正法產生懷疑）、戒禁取見（執著於錯誤的儀式或禁忌）、貪欲、瞋恚。
2. **漏盡 (Asavakkhaya)**：指完全消除「漏」（Asava），即心靈中滲漏出的煩惱（如欲漏、有漏、無明漏），達到不再輪迴的狀態。
3. **梵行 (Brahmacariya)**：原指禁欲生活，在佛教中指清淨的修行生活，即依循八正道而行的聖潔生活。
4. **優陀那 (Udana)**：指佛陀在進入某種深層禪定或體悟真理後，隨興而發、充滿感觸的覺悟詩偈。
5. **欲見法 (Tanha-ditthi)**：指基於貪愛而產生的錯誤見解，認為世界或自我具有實體且可透過欲望滿足而獲得永恆。

### [現代實作指南]
本經文的核心在於「解構自我」與「停止意識的攀緣」。在現代生活中，我們可以透過以下步驟實踐：

1. **五蘊拆解練習 (Deconstruction of Self)**：
   - 當你感到強烈的情緒（如憤怒或焦慮）時，嘗試將其拆解：
     - **色** ➔ 觀察身體的反應（如心跳加快、肌肉緊繃）。
     - **受** ➔ 標記這是一種「不舒服」的感受。
     - **想** ➔ 觀察腦中出現的標籤（如「他竟然這樣對我」、「這太不公平了」）。
     - **行** ➔ 覺察內心想要反擊或逃避的衝動（心的造作）。
     - **識** ➔ 意識到這整個過程只是意識在對象間跳轉。
   - **轉念**：告訴自己「這只是五蘊的運作，其中沒有一個永恆的『我』在受苦」。

2. **觀察「識」的來去 (Observing the Flow of Consciousness)**：
   - 練習在日常工作中，觀察意識如何「攀緣」：
     - 意識 ➔ 看到手機（色） ➔ 產生想滑手機的欲望（愛樂） ➔ 意識停留在此（住） ➔ 執著心增長。
   - **實作**：在意識「住」在對象上的一瞬間，輕輕地提醒自己：「識正在攀緣」。一旦覺察到攀緣，執著的鏈條就會暫時中斷。

3. **追求「無所住」的心理狀態 (Non-attachment)**：
   - 練習「不定義」的觀察。當一件好事或壞事發生時，試著不將其定義為「我的成功」或「我的失敗」。
   - 意識到所有定義（我、我的）都是心理的虛構，當你不再試圖將經驗「私有化」時，內心的壓力會顯著降低，體驗到經文中所說的「寂靜與清涼」。

### [段落大意]
本經透過佛陀的優陀那偈，闡明「我」與「我所」皆為虛構。佛陀指出凡夫將五蘊（色、受、想、行、識）誤認為自我，而聖弟子則體悟五蘊之無常、苦、無我。修行之關鍵在於斷除對五蘊的貪愛與攀緣，使意識（識）不再依附於任何對象而增長。當意識達到「無所住」的狀態，便能停止造業，進而證悟涅槃，斷除下分結，終結輪迴之苦。
