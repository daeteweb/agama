# 雜阿含經 第 28 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，有異比丘來詣佛所，頭面禮足，却住一面，白佛言：「世尊！如世尊所說，得見法涅槃，云何比丘得見法涅槃？」

佛告比丘：「善哉！善哉！汝今欲知見法涅槃耶？」

比丘白佛：「唯然，世尊！」

佛告比丘：「諦聽，善思，當為汝說。」

佛告比丘：「於色生厭、離欲、滅盡，不起諸漏，心正解脫，是名比丘見法涅槃；如是受、想、行、識，於識生厭、離欲、滅盡，不起諸漏，心正解脫，是名比丘見法涅槃。」

時，彼比丘聞佛所說，踊躍歡喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，有異比丘來詣佛所，頭面禮足，却住一面，白佛言：**
　　那時，有一位特殊的比丘（出家修行者）來到佛陀這裡，頭面觸地禮拜佛陀的足部，然後站在一旁，對佛陀恭敬地說道：

**◈ 「世尊！如世尊所說，得見法涅槃，云何比丘得見法涅槃？」**
　　「世尊！正如世尊所教導的，能夠體悟到法涅槃（超越生死、絕對寧靜的解脫境界），請問比丘應該如何做才能體悟到法涅槃？」

**◈ 佛告比丘：「善哉！善哉！汝今欲知見法涅槃耶？」**
　　佛陀告訴那位比丘說：「好極了！做得好！你現在想要知道如何體悟法涅槃嗎？」

**◈ 比丘白佛：「唯然，世尊！」**
　　比丘對佛陀恭敬地說道：「是的，世尊！」

**◈ 佛告比丘：「諦聽，善思，當為汝說。」**
　　佛陀告訴比丘：「仔細聽，好好思考，我現在為你解說。」

**◈ 佛告比丘：「於色生厭、離欲、滅盡，不起諸漏，心正解脫，是名比丘見法涅槃；**
　　佛陀告訴比丘：「對於色（物質身體）產生厭離心、遠離對它的貪欲並使其執著滅盡，不再產生諸漏（內心的染污與煩惱），心靈達到真正的解脫，這就叫做比丘體悟到了法涅槃；

**◈ 如是受、想、行、識，於識生厭、離欲、滅盡，不起諸漏，心正解脫，是名比丘見法涅槃。」**
　　同樣地，對於受（感受）、想（想像與認知）、行（心的造作）以及識（意識），在識（意識）上產生厭離心、遠離貪欲並使其執著滅盡，不再產生諸漏（內心的染污與煩惱），心靈達到真正的解脫，這就叫做比丘體悟到了法涅槃。」

**◈ 時，彼比丘聞佛所說，踊躍歡喜，作禮而去。**
　　那時，那位比丘聽完佛陀所說的教法，心中感到極其歡喜，向佛陀行禮後才離開。

### [詞語註解（解釋重要佛教概念）]
1. **法涅槃 (Dhamma-Nibbāna)**：指透過對「法」（真理）的徹悟，而達到的熄滅煩惱、超越輪迴的絕對寧靜狀態。
2. **厭 (Nibbidā)**：並非指情緒上的厭惡或仇恨，而是一種基於智慧的「厭離」，即看清現象的無常與苦後，自然產生的不再執著的心理狀態。
3. **漏 (Āsava)**：原意為「滲漏」，比喻潛伏在心底、不斷滲出並導致輪迴的染污，包含欲漏（對感官慾望的執著）、有漏（對存在的執著）、見漏（對錯誤見解的執著）及無明漏（對真理的無知）。
4. **解脫 (Vimokkha)**：心靈從貪、嗔、痴等束縛中完全釋放，不再受業力牽引的狀態。

### [現代實作指南]
本經文提供了一套針對「五蘊」進行心理拆解與轉念的實踐路徑，可用於處理現代人的精神內耗與執著：

1. **觀察物質與身體的侷限（色）**：
   當我們對外貌、健康或物質財產產生強烈執著或焦慮時，練習觀察這些物質（色）的無常性 ➔ 意識到它們無法提供永恆的滿足 ➔ 產生「厭離心」（不再盲目追求） ➔ 降低對物質的依賴感。

2. **解構情緒與認知（受、想、行、識）**：
   - **受（感受）**：當強烈的情緒（如憤怒、悲傷）升起時，僅將其標記為「這是一個感受」，而非「我是悲傷的」。
   - **想（認知）**：覺察大腦如何將經驗標籤化（例如：將對方的沉默定義為「他討厭我」），意識到這只是大腦的想像而非事實。
   - **行（造作）**：觀察內心衝動想做出反應的傾向（例如：想反擊、想逃避）。
   - **識（意識）**：覺察意識如何將上述過程整合為一個「我」的經驗。

3. **實作步驟**：
   - **覺察** ➔ **看破**（發現其無常且不可掌控） ➔ **放下**（離欲） ➔ **清淨**（不起漏）。
   - 透過這種方式，將原本「與情緒認同」的狀態，轉化為「觀察情緒」的狀態，從而達到心靈的暫時解脫與寧靜。

### [段落大意]
本經描述一名比丘請問如何體悟法涅槃，佛陀教導其對構成身心的五蘊（色、受、想、行、識）產生智慧上的厭離，遠離貪欲並滅盡執著，使內心不再產生染污（漏），進而達到真正的解脫。
