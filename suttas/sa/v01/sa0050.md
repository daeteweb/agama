# 雜阿含經 第 50 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告尊者阿難曰：「若有諸外道出家來問汝言：『阿難！世尊何故教人修諸梵行？』如是問者，云何答乎？」

阿難白佛：「世尊！若外道出家來問我言：『阿難！世尊何故教人修諸梵行？』者，我當答言：『為於色修厭、離欲、滅盡、解脫、不生故，世尊教人修諸梵行；為於受、想、行、識，修厭、離欲、滅盡、解脫、不生故，教人修諸梵行。』世尊！若有外道出家作如是問者，我當作如是答。」

佛告阿難：「善哉！善哉！應如是答。所以者何？我實為於色修厭、離欲、滅盡、解脫、不生故，教人修諸梵行；於受、想、行、識，修厭、離欲、滅盡、解脫、不生故，教人修諸梵行。」

佛說此經已，尊者阿難聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告尊者阿難曰：「若有諸外道出家來問汝言：『阿難！世尊何故教人修諸梵行？』如是問者，云何答乎？」**
　　那時，世尊告訴尊者阿難說：「如果有其他宗派的出家修行者來問你說：『阿難！世尊為什麼教導人們修行清淨的梵行？』如果被這樣問，你該如何回答？」

**◈ 阿難白佛：「世尊！若外道出家來問我言：『阿難！世尊何故教人修諸梵行？』者，我當答言：**
　　阿難對佛陀恭敬地說道：「世尊！如果有其他宗派的出家修行者來問我說：『阿難！世尊為什麼教導人們修行清淨的梵行？』我應該回答說：

**◈ 『為於色修厭、離欲、滅盡、解脫、不生故，世尊教人修諸梵行；**
　　『為了對色（物質身體）修習厭離、遠離貪欲、使其滅盡、獲得解脫且不再輪迴出生，所以世尊教人修行清淨的梵行；

**◈ 為於受、想、行、識，修厭、離欲、滅盡、解脫、不生故，教人修諸梵行。』**
　　為了對受（感受）、想（想像與認知）、行（心的造作）、識（意識）修習厭離、遠離貪欲、使其滅盡、獲得解脫且不再輪迴出生，所以教人修行清淨的梵行。』

**◈ 世尊！若有外道出家作如是問者，我當作如是答。」**
　　世尊！如果有其他宗派的出家修行者這樣問我，我就會這樣回答。」

**◈ 佛告阿難：「善哉！善哉！應如是答。**
　　佛陀告訴阿難：「好極了！做得好！應該這樣回答。

**◈ 所以者何？我實為於色修厭、離欲、滅盡、解脫、不生故，教人修諸梵行；**
　　為什麼呢？我確實是為了讓人們對色（物質身體）修習厭離、遠離貪欲、使其滅盡、獲得解脫且不再輪迴出生，而教導修行清淨的梵行；

**◈ 於受、想、行、識，修厭、離欲、滅盡、解脫、不生故，教人修諸梵行。」**
　　並且為了讓人們對受（感受）、想（想像與認知）、行（心的造作）、識（意識）修習厭離、遠離貪欲、使其滅盡、獲得解脫且不再輪迴出生，而教導修行清淨的梵行。」

**◈ 佛說此經已，尊者阿難聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，尊者阿難聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **梵行 (Brahmacariya)**：原指婆羅門的禁欲生活，在佛法中指清淨的修行生活，旨在透過戒律與禪定，消除貪欲以達到解脫。
2. **厭 (Nibbida)**：並非指情緒上的厭惡，而是一種基於智慧的「厭離」。指在洞察現象的無常與苦後，心不再對其產生執著與渴求。
3. **離欲 (Virāga)**：指心靈脫離對感官對象的貪愛與執著，達到一種不被欲望牽引的冷靜狀態。
4. **不生 (Non-arising)**：指因斷除了輪迴的根本原因（無明與貪愛），使得未來不再產生新的業力，從而停止在生死輪迴中出生。
5. **五蘊 (Pañca-kkhandha)**：構成個體身心的五種要素，包括色（物質）、受（感受）、想（認知）、行（意志造作）、識（意識）。

### [現代實作指南]
本經文的核心在於「對五蘊的解構與脫鉤」。在現代生活中，我們可以將其轉化為以下心理實踐步驟：

1. **覺察物質身體（色）的局限** ➔ 當身體感到疲憊、病痛或對外貌產生過度執著時，提醒自己：身體是物質的組合，會隨時間衰老與改變。練習將「我」與「身體」分開，觀察身體的感覺而非認同它。
2. **標記感受（受）的暫時性** ➔ 當強烈的情緒（如憤怒、悲傷或狂喜）湧現時，在心中標記：「現在有一個『受』在發生」，而非說「我很生氣」。透過這種方式，將感受視為一種流動的現象，而非自我的本質。
3. **質疑認知的正確性（想）** ➔ 當我們對某人或某事產生強烈偏見或定義時，意識到這只是「想（認知）」在運作。問自己：「這個認知是絕對的事實，還是我根據過去經驗所建構的想像？」
4. **觀察衝動的起心動念（行）** ➔ 在採取行動前，觀察內心產生的「想要做某事」的衝動。意識到這種造作（行）是由先前的感受與認知驅動的，在衝動與行動之間創造一個空間，選擇不被其牽引。
5. **覺知意識的流轉（識）** ➔ 觀察意識如何在不同的對象之間跳轉。意識到意識本身也是一種條件反射的過程，而非一個永恆不變的靈魂。

**轉念技巧**：將「我想獲得」轉化為「我觀察到一個渴望在升起」。透過將「執著」轉化為「觀察」，即可實踐經文中所述的「厭離」與「解脫」。

### [段落大意]
本經透過佛陀與阿難的對話，闡明修行清淨梵行的目的：即透過對構成身心的五蘊（色、受、想、行、識）修習厭離與遠離貪欲，使其滅盡，最終達到解脫且不再輪迴出生的境界。
