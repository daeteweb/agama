# 雜阿含經 第 342 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有六常行。云何為六？若比丘眼見色，不苦、不樂，捨心住正念正智。耳聲、鼻香、舌味、身觸、意識法，不苦、不樂，捨心住正念正智。若比丘成就此六常行者，當知是舍利弗等。舍利弗比丘眼見色已，不苦、不樂，捨心住正念正智。耳聲、鼻香、舌味、身觸、意識法，不苦、不樂，捨心住正念正智。舍利弗比丘成就此六常行故，世間難得，所應承事、恭敬、供養，則為世間無上福田。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有六常行。云何為六？**
　　那時，世尊告訴在場的眾位比丘說：「有六種恆常的修行方式。什麼是這六種方式呢？

**◈ 若比丘眼見色，不苦、不樂，捨心住正念正智。**
　　如果比丘的眼睛看見色（物質身體與現象），心中不產生苦（不快）或樂（愉快）的感受，而是保持一種不偏不倚的捨心，並維持著正念（對當下的覺知）與正智（對真理的正確認知）。

**◈ 耳聲、鼻香、舌味、身觸、意識法，不苦、不樂，捨心住正念正智。**
　　當耳朵聽到聲音、鼻子聞到香氣、舌頭品嚐味道、身體感受到觸碰、以及意識（識）接觸到意識法時，心中同樣不產生苦（不快）或樂（愉快）的感受，而是保持一種不偏不倚的捨心，並維持著正念（對當下的覺知）與正智（對真理的正確認知）。

**◈ 若比丘成就此六常行者，當知是舍利弗等。**
　　如果比丘能成就這六種恆常的修行方式，應當了解這就是像舍利弗比丘那樣的境界。

**◈ 舍利弗比丘眼見色已，不苦、不樂，捨心住正念正智。**
　　舍利弗比丘在眼睛看見色（物質身體與現象）後，心中不產生苦（不快）或樂（愉快）的感受，而是保持一種不偏不倚的捨心，並維持著正念（對當下的覺知）與正智（對真理的正確認知）。

**◈ 耳聲、鼻香、舌味、身觸、意識法，不苦、不樂，捨心住正念正智。**
　　當他耳朵聽到聲音、鼻子聞到香氣、舌頭品嚐味道、身體感受到觸碰、以及意識（識）接觸到意識法時，心中同樣不產生苦（不快）或樂（愉快）的感受，而是保持一種不偏不倚的捨心，並維持著正念（對當下的覺知）與正智（對真理的正確認知）。

**◈ 舍利弗比丘成就此六常行故，世間難得，所應承事、恭敬、供養，則為世間無上福田。」**
　　因為舍利弗比丘成就了這六種恆常的修行方式，他在世間是非常罕見的，是應該被承事、恭敬與供養的人，他就是世間最無上的福田（能讓供養者獲得巨大功德的對象）。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **六常行 (Six Constant Practices)**：指在面對六根（眼、耳、鼻、舌、身、意）與六塵（色、聲、香、味、觸、法）接觸時，恆常保持不被情緒牽引的心理狀態。
2. **捨心 (Upekkhā)**：一種中立、平衡的心理狀態，指在面對對立的感受（如苦與樂）時，不產生貪著或厭惡，保持平靜。
3. **正念 (Sati)**：對當下經驗的清晰覺知，不隨之漂移，能及時意識到心念的運作。
4. **正智 (Sampajañña)**：清楚地知道當下正在做什麼、為什麼做，以及該行為是否符合真理的正確認知。
5. **福田 (Puññakkhetta)**：比喻如良田般能讓種植種子（供養或善行）的人獲得豐碩果報（功德）的聖者。

### [現代實作指南]
本經文提供了一套極其精準的「感官管理系統」，可用於現代生活中緩解情緒壓力與衝動反應：

1. **建立「感官緩衝區」**：
   - 當你接收到外界刺激（如：聽到批評、看到討厭的人、品嚐到難吃的食物）時，練習在「刺激」與「反應」之間創造一個空隙。
   - 意識到：現在是「眼見色」或「耳聞聲」。

2. **標記感受，但不認同**：
   - 當心中升起「苦（不快）」或「樂（愉快）」時，僅僅將其標記為「這是一個不快感」或「這是一個愉快感」，而不是說「我很痛苦」或「我很開心」。
   - 透過這種標記，將自己從情緒中抽離，轉化為觀察者。

3. **實踐「捨心」的中立觀察**：
   - 告訴自己：「這只是一個聲音/影像，它本身沒有好壞，好壞是我心中產生的反應。」
   - 嘗試維持一種「不評判」的狀態，讓情緒像雲一樣飄過，而不被其牽引。

4. **維持正念與正智**：
   - **正念**：持續覺察「我現在正在感到不快」。
   - **正智**：提醒自己「這種不快感是無常的，它是因緣而生，不代表我的本質」。

### [段落大意]
本經由佛陀闡述「六常行」的修行法門，即在面對六種感官接觸（色聲香味觸法）時，能不被苦樂感受所左右，恆常保持平靜的捨心，並維持正念與正智。佛陀以舍利弗比丘作為實踐此法的典範，稱其為世間罕見且能帶來無上功德的福田。
