# 雜阿含經 第 341 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有六常行。云何為六？若比丘眼見色，不苦、不樂，捨心住正念正智。耳聲、鼻香、舌味、身觸、意識法，不苦、不樂，捨心住正念正智。若比丘成就此六常行者，世間難得，所應承事、恭敬、供養，則為世間無上福田。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有六常行。云何為六？**
　　那時，世尊告訴在場的眾位比丘說：「有六種恆常的修行方法。什麼是這六種方法呢？

**◈ 若比丘眼見色，不苦、不樂，捨心住正念正智。**
　　如果比丘用眼睛看到色（物質身體與現象），心中不感到苦（不悅），也不感到樂（喜悅），而是保持一種不偏不倚的捨心，並維持著正念（對當下的正確覺知）與正智（對真理的正確理解）。

**◈ 耳聲、鼻香、舌味、身觸、意識法，不苦、不樂，捨心住正念正智。**
　　當耳朵聽到聲音、鼻子聞到香氣、舌頭品嚐味道、身體感受到觸（接觸）、意識（識）接觸到意識法時，同樣不感到苦，也不感到樂，而是保持捨心，維持著正念與正智。

**◈ 若比丘成就此六常行者，世間難得，所應承事、恭敬、供養，則為世間無上福田。」**
　　如果比丘能成就這六種恆常的修行方法，在世間是非常罕見的。這樣的人應該被承事、恭敬與供養，因為他就是世間最無上的福田（能讓供養者獲得巨大福德的對象）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **六常行 (Six Constant Practices)**：指在面對六根（眼、耳、鼻、舌、身、意）與六塵（色、聲、香、味、觸、法）接觸時，恆常保持中立、不被情緒牽引的修行方式。
2. **捨心 (Upekkha)**：一種心理上的平衡狀態，指面對對象時既不產生貪愛（樂），也不產生厭惡（苦），維持客觀的平靜。
3. **正念 (Sati)**：八正道之一，指清晰地覺知當下正在發生的現象，而不對其進行評判或反應。
4. **正智 (Samma-ñāṇa)**：對事物本質（如無常、苦、無我）的正確認知與理解，使心不被幻象所欺。
5. **福田 (Puññakkhetta)**：比喻修行圓滿的人如同肥沃的田地，他人對其進行恭敬或供養，能如同種種好種子般，獲得巨大的善業果報。

### [現代實作指南]
本經文提供了一套處理「感官刺激」的心理操作流程，可用於現代生活中的情緒管理與壓力緩解：

1. **覺察感官觸發**：當你接收到外部資訊（例如：看到討厭的人、聽到批評的話、感受到身體的不適）時，立即標記這個行為。例如：「現在眼睛看到了色」、「現在耳朵聽到了聲」。
2. **識別情緒反應**：觀察心中是否立即升起「苦（不悅）」或「樂（喜悅）」的反應。僅僅是觀察，不要試圖壓抑或強迫改變它。
3. **練習「捨心」轉念**：告訴自己：「這只是一個感官現象，不需要被定義為好或壞。」將注意力從「對對象的評價」轉移到「對現象的觀察」。
4. **維持正念正智**：
    * **正念** $\rightarrow$ 保持在當下，不讓心隨之陷入對過去的憤怒或對未來的擔憂。
    * **正智** $\rightarrow$ 提醒自己，這個感受（受）是暫時的，它隨緣而起，必然隨緣而滅（無常）。
5. **實作目標**：將此流程從「刻意練習」轉化為「恆常習慣」（常行），使心在面對外界劇烈波動時，依然能保持內在的寧靜與清明。

### [段落大意]
世尊教導比丘在面對六種感官對象（色、聲、香、味、觸、法）時，應練習不產生苦樂反應，以捨心維持正念與正智。能成就此六種恆常修行的人在世間極其罕見，是值得恭敬供養的無上福田。
