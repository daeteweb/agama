# 雜阿含經 第 207 經

### [原文]
（二○七）

如是我聞：

一時，佛住毘舍離耆婆拘摩羅藥師菴羅園。

爾時，世尊告諸比丘：「當修無量三摩提，精勤繫念。所以者何？修無量三摩提，精勤繫念已，則如實顯現。於何如實顯現？於眼如實顯現……」如是廣說，乃至「此諸法無常、有為，此如實顯現。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我（弟子）是這樣聽聞的：

**◈ 一時，佛住毘舍離耆婆拘摩羅藥師菴羅園。**
　　那時，佛居住在毘舍離城的耆婆拘摩羅藥師所提供的阿菴羅園中。

**◈ 爾時，世尊告諸比丘：「當修無量三摩提，精勤繫念。」**
　　當時，世尊對眾多比丘（出家修行者）說道：「你們應該修行廣大的三摩提（禪定狀態），並在修行中精勤地繫念（持續且專注地維持覺知）。」

**◈ 「所以者何？修無量三摩提，精勤繫念已，則如實顯現。」**
　　「為什麼要這樣做呢？因為當你們修行了廣大的三摩提（禪定）並持續保持專注的覺知後，真理就會如實地顯現出來。」

**◈ 「於何如實顯現？於眼如實顯現……」**
　　「究竟是如何如實顯現呢？例如在對待眼睛（視覺感官）時，會如實顯現出事物的真相……」

**◈ 「如是廣說，乃至『此諸法無常、有為，此如實顯現。』」**
　　「佛以此類推進行了廣泛的說明，直到最後得出結論：『這些現象（諸法）都是無常（不斷變化且不穩定的）與有為（由因緣條件所構成）的，這就是真理如實顯現的樣子。』」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　當佛說完這部經後，眾多比丘聽聞了佛陀的教導，感到十分歡喜並遵從實踐。

### [詞語註解]
1. **三摩提 (Samādhi)**：指心念高度集中、不散亂的定力狀態，是觀察真理的心理基礎。
2. **精勤繫念**：指以精進的心力，將覺知緊緊繫於當下的對象上，不讓意識隨意流轉。
3. **如實顯現 (Yathābhūta-ñāṇadassana)**：指透過智慧直接觀察到事物本來的真實樣貌，而非受主觀偏見、情緒或幻覺所干擾。
4. **無常 (Anicca)**：指一切現象皆處於不斷生滅、變化的狀態中，沒有任何事物具有永恆不變的本質。
5. **有為 (Saṅkhata)**：指所有依賴因緣條件（如時間、空間、各種組成要素）而產生的現象，必然會隨著條件的改變而消亡。

### [現代實作指南]
本經的核心在於「定」與「觀」的結合。若要將此教法轉化為生活中的心理實踐，可遵循以下步驟：

1. **建立專注點（修三摩提）**：
   在日常生活中，選擇一個簡單的對象作為覺知的錨點（例如呼吸、走路的腳步感或當下的聲音）。練習將注意力集中於該單一對象上，減少心智的游移。

2. **持續性的覺察（精勤繫念）**：
   當發現心念開始飄向過去的回憶或未來的焦慮時，不要責備自己，而是溫和地、精進地將注意力帶回原本的對象。這種「帶回」的過程就是精勤繫念。

3. **觀察現象的本質（如實顯現）**：
   當你的心變得相對安定後，嘗試以「觀察者」的身分去觀察感官接收到的資訊。例如，觀察一個情緒（如憤怒）或一個生理感覺（如疼痛）：
   → 觀察它如何升起？
   → 觀察它在過程中如何變化？
   → 觀察它如何消逝？

4. **轉念技巧**：
   當你陷入對某種狀態的執著時，提醒自己：「這是一個『有為』的現象，它是因緣湊巧產生的，且必然是『無常』的。」透過這種理性的認知，可以有效降低情緒的強度，達到心理上的解脫。
