# 雜阿含經 第 659 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有五根。何等為五？謂信根、精進根、念根、定根、慧根。何等為信根？若聖弟子於如來發菩提心所得淨信心，是名信根。何等為精進根？於如來發菩提心所起精進方便，是名精進根。何等念根？於如來初發菩提心所起念根。何等為定根？於如來初發菩提心所起三昧，是名定根。何等為慧根？於如來初發菩提心所起智慧，是名慧根。所餘堂閣，譬如上說。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有五根。何等為五？謂信根、精進根、念根、定根、慧根。**
　　那時，世尊告訴在場的眾位比丘說：「有五種修行根基（五根）。這五種是什麼？指的是信根（信仰的根基）、精進根（努力的根基）、念根（正念的根基）、定根（專注的根基）、以及慧根（智慧的根基）。」

**◈ 何等為信根？若聖弟子於如來發菩提心所得淨信心，是名信根。**
　　什麼是信根？如果聖弟子對著如來（成就不變真理而來到世間的人）發起追求最高、最圓滿的智慧與覺悟（菩提心）而獲得的純淨信心，這就稱為信根。

**◈ 何等為精進根？於如來發菩提心所起精進方便，是名精進根。**
　　什麼是精進根？對著如來發起追求最高、最圓滿的智慧與覺悟（菩提心）而產生的精進修行方法，這就稱為精進根。

**◈ 何等念根？於如來初發菩提心所起念根。**
　　什麼是念根？對著如來在最初發起追求最高、最圓滿的智慧與覺悟（菩提心）時，所產生的正念根基。

**◈ 何等為定根？於如來初發菩提心所起三昧，是名定根。**
　　什麼是定根？對著如來在最初發起追求最高、最圓滿的智慧與覺悟（菩提心）時，所產生的三昧（心不動搖的專注狀態），這就稱為定根。

**◈ 何等為慧根？於如來初發菩提心所起智慧，是名慧根。**
　　什麼是慧根？對著如來在最初發起追求最高、最圓滿的智慧與覺悟（菩提心）時，所產生的智慧，這就稱為慧根。

**◈ 所餘堂閣，譬如上說。」**
　　其餘的細節與結構，就如同上述的說明一樣。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **五根 (Indriya)**：指在修行過程中起主導作用的五種心靈能力，如同植物的根一樣，是成長與成就解脫的基礎。
2. **菩提心 (Bodhicitta)**：在此處指對覺悟真理、追求最高智慧與解脫的強烈願望與志向。
3. **三昧 (Samādhi)**：指心念專注於單一對象而不再散亂的狀態，是達到智慧洞察的前提。
4. **信 (Saddhā)**：並非盲信，而是基於對真理的初步認知而產生的信任與信心。

### [現代實作指南]
本經提到的「五根」可視為一套心智成長的平衡系統，在現代生活中可採取以下實作步驟：

1. **建立正向信心 (信根)** ➔ 找出生活中能讓你感到平靜且真實的真理（例如：觀察萬物皆在變遷的無常），對此建立信任，作為行動的起點。
2. **採取持續行動 (精進根)** ➔ 將目標拆解為微小習慣。不要追求瞬間的爆發，而是在日常中維持穩定且不懈的努力（例如：每天固定冥想5分鐘）。
3. **維持當下覺知 (念根)** ➔ 練習「正念」。在洗碗、走路或工作時，將注意力拉回當下的身體感受或呼吸，防止心智陷入對過去的後悔或對未來的焦慮。
4. **培養深度專注 (定根)** ➔ 減少數位分心（如關閉通知）。練習在單一任務上保持長時間的專注，讓心靈從碎片化狀態恢復到整合狀態。
5. **發展批判思考 (慧根)** ➔ 運用觀察到的事實來分析問題，而非依賴情緒反應。思考「這件事的因緣是什麼？」，從而產生對事物本質的洞察。

**平衡技巧**：
- 若感到過於焦慮 ➔ 增加「定根」（專注呼吸）。
- 若感到懶散懈怠 ➔ 增加「精進根」（設定小目標）。
- 若感到盲目跟隨 ➔ 增加「慧根」（邏輯分析與觀察）。

### 段落大意 
本經詳細定義了修行所需的五種根基（五根）：信、精進、念、定、慧。佛陀指出，這五種能力皆源於對如來發起追求覺悟（菩提心）的志向，是達成解脫不可或缺的心靈支柱。
