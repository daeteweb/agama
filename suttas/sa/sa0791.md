# 雜阿含經 第 791 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有邪、有邪道，有正、有正道。諦聽，善思，當為汝說。何等為邪？謂地獄、畜生、餓鬼。何等為邪道？謂殺、盜、邪婬、妄語、兩舌、惡口、綺語、貪、恚、邪見。何等為正？謂人、天、涅槃。何等為正道？謂不殺、不盜、不邪婬、不妄語、不兩舌、不惡口、不綺語、無貪、無恚、正見。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有邪、有邪道，有正、有正道。**
　　那時，世尊告訴在場的眾位比丘說：「世間存在著錯誤的境界與錯誤的行為道路，也存在著正確的境界與正確的行為道路。

**◈ 諦聽，善思，當為汝說。**
　　請你們仔細聽，好好思考，我現在為你們解說。

**◈ 何等為邪？謂地獄、畜生、餓鬼。**
　　什麼是錯誤的境界？指的是地獄、畜生、餓鬼（這三種充滿痛苦的惡道）。

**◈ 何等為邪道？謂殺、盜、邪婬、妄語、兩舌、惡口、綺語、貪、恚、邪見。**
　　什麼是錯誤的行為道路？指的是殺生、偷盜、不正當的性行為、說謊、挑撥離間、惡劣的言語、花言巧語、貪婪、嗔恨以及錯誤的見解（與真理相悖的認知）。

**◈ 何等為正？謂人、天、涅槃。**
　　什麼是正確的境界？指的是人類、天人以及涅槃（超越生死、絕對寧靜的解脫境界）。

**◈ 何等為正道？謂不殺、不盜、不邪婬、不妄語、不兩舌、不惡口、不綺語、無貪、無恚、正見。」**
　　什麼是正確的行為道路？指的是不殺生、不偷盜、不從事不正當的性行為、不說謊、不挑撥離間、不使用惡劣言語、不說花言巧語、沒有貪婪、沒有嗔恨以及持有正見（符合四聖諦的正確見解）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **邪 (Micchā)**：指偏離真理、導致痛苦或惡果的狀態或方向。在此經文中，分為「邪（境界）」與「邪道（行為）」。
2. **正 (Sammā)**：指符合真理、導向幸福或解脫的方向。對應於八正道中的「正」字。
3. **邪見 (Micchā-diṭṭhi)**：對因果律的否定或對真理的錯誤認知，是導致錯誤行為（邪道）的根源。
4. **正見 (Sammā-diṭṭhi)**：正確的見解，指能洞察四聖諦（苦、集、滅、道）以及緣起（Paṭicca-samuppāda/因緣而生）的真理。
5. **涅槃 (Nibbāna)**：超越生死輪迴、熄滅一切貪嗔痴煩惱的絕對寧靜與解脫境界。

### [現代實作指南]
本經文提供了一個簡單的「因果對照表」，將人生狀態（境界）與日常行為（道路）直接掛鉤。在現代生活中，可採取以下實踐步驟：

1. **行為審核（行為 ➔ 境界）**：
   - 每日回顧自己的言行。當發現自己產生「貪（渴望佔有）」或「恚（憤怒厭惡）」時，意識到這正是在走「邪道」。
   - 意識到目前的負面情緒或困境（如壓力、衝突）是過去「邪道」行為的業報（Vipāka），而非外界的偶然。

2. **語言淨化實作**：
   - 針對經文提到的四種口業（妄語、兩舌、惡口、綺語），練習「正語」：在說話前詢問自己：這句話是否真實？是否和諧？是否必要？是否溫柔？

3. **認知轉念（正見實作）**：
   - 當面對不順遂時，將其視為「無常 (Anicca)」的體現，而非對個人的針對。
   - 將目標從追求短暫的感官滿足（邪道之貪），轉向追求內心的平靜與對他人的慈悲（正道之無貪、無恚）。

### [段落大意]
本經由佛陀明確區分了「邪」與「正」的境界及其對應的行為道路。錯誤的行為（如殺盜、妄語、貪嗔等）將導致痛苦的惡道境界（地獄、畜生、餓鬼）；而正確的行為（如持戒、無貪、正見等）則導向較好的生存狀態（人、天）以及最終的解脫境界（涅槃）。
