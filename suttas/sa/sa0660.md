# 雜阿含經 第 660 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有五根。何等為五？謂信根、精進根、念根、定根、慧根。於此五根修習多修習，過去、未來、現在一切苦斷。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如苦斷，如是究竟苦邊、苦盡、苦息、苦沒，度苦流，於縛得解，害諸色，過去、未來、現在一切漏盡，亦如是說。

---

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有五根。何等為五？**
　　那時，世尊告訴在場的眾位比丘說：「有五種修行根基（五根）。什麼是這五種根基呢？

**◈ 謂信根、精進根、念根、定根、慧根。**
　　指的是：信根（對正法的信心）、精進根（不懈怠的努力）、念根（正念的覺知）、定根（心不散亂的專注）、以及慧根（洞察真理的智慧）。

**◈ 於此五根修習多修習，過去、未來、現在一切苦斷。」**
　　對於這五種修行根基進行深入且大量的修行，就能斷除過去、未來以及現在的所有苦（Dukkha）。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如苦斷，如是究竟苦邊、苦盡、苦息、苦沒，度苦流，於縛得解，害諸色，過去、未來、現在一切漏盡，亦如是說。**
　　正如斷除苦（Dukkha）一樣，對於達到苦的邊際（終結）、苦的盡除、苦的息滅、苦的消亡，以及度過苦難的激流、從束縛中獲得解脫、破除對色（物質身體）的執著，使過去、未來、現在的所有漏（煩惱的滲漏）全部盡除，也是同樣地這麼說的。

### [詞語註解（解釋重要佛教概念）]
1. **五根 (Pañca Indriya)**：指修行者在解脫道路上必須培養的五種心靈能力。信、精進、念、定、慧五者相輔相成，能驅使心靈趨向覺悟。
2. **苦 (Dukkha)**：指一切不穩定、不滿足的狀態，包含出生、衰老、疾病、死亡以及與不愛之人相會、與所愛之人分離等所有不安之感。
3. **漏 (Āsava)**：原意為「滲漏」，比喻煩惱如水般滲漏出心靈，使人陷入輪迴。主要分為欲漏（對感官欲望的執著）、有漏（對存在與生存的執著）、無明漏（對真理的無知）。
4. **色 (Rūpa)**：指物質身體或所有物質現象。在此處指破除對物質身體是「我」的執著。

### [現代實作指南]
本經強調的「五根」可轉化為一套心智訓練系統，用於處理現代生活中的壓力與焦慮：

1. **建立信根（正向信念）**：
   ➔ 實作：在面對困境時，先建立「問題是可以被解決」且「心靈是可以被訓練」的基礎信念，避免陷入絕望的習氣。
2. **啟動精進根（持續行動）**：
   ➔ 實作：將大目標拆解為微小習慣。不追求瞬間的爆發，而追求「每天進步一點」的穩定性，將修行轉化為生活規律。
3. **維持念根（覺知當下）**：
   ➔ 實作：練習「標記法」。當憤怒升起時，在心中輕輕標記「憤怒、憤怒」，將自己從情緒中抽離，成為觀察者而非參與者。
4. **深化定根（專注單一）**：
   ➔ 實作：練習單一任務處理（Single-tasking）。在吃飯時僅吃飯，在走路時僅走路，訓練心靈不再被碎片化資訊牽引。
5. **開發慧根（理性洞察）**：
   ➔ 實作：運用「無常」視角分析痛苦。問自己：「這個令我痛苦的情況，在一年後、十年後是否依然如此？」透過洞察現象的變遷，降低對現狀的執著。

### [段落大意]
本段經文闡述了修行解脫的五種核心能力（信、精進、念、定、慧）。佛陀指出，透過對這五根的深入修習，能徹底斷除三世的一切苦，並最終達到煩惱盡除、從物質與精神束縛中獲得完全解脫的境界。
