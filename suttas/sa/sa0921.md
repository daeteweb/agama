# 雜阿含經 第 921 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，世尊告諸比丘：「世有良馬，四能具足，當知是良馬，王所服乘。何等為四？所謂賢善、捷疾、堪能、調柔。如是善男子四德成就，世所宗重，承事供養，為無上福田。何等為四？謂善男子成就無學戒身、無學定身、無學慧身、無學解脫身。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「世有良馬，四能具足，當知是良馬，王所服乘。**
　　「世上如果有良馬，具足四種能力，應當了解這就是良馬，是國王所駕乘的。

**◈ 何等為四？所謂賢善、捷疾、堪能、調柔。**
　　這四種能力是什麼？也就是指：品格賢善、速度捷疾、能力堪能、性格調柔。

**◈ 如是善男子四德成就，世所宗重，承事供養，為無上福田。**
　　同樣地，如果一位善男子（修學佛法的男子）成就了四種德行，會被世人尊崇重視，受到侍奉供養，成為最殊勝的福田（能讓他人種植善因而獲福報的對象）。

**◈ 何等為四？謂善男子成就無學戒身、無學定身、無學慧身、無學解脫身。」**
　　這四種德行是什麼？是指善男子成就了：無學的戒身（不再需要學習、已圓滿的戒律）、無學的定身（不再需要學習、已圓滿的禪定）、無學的慧身（不再需要學習、已圓滿的智慧）、以及無學的解脫身（不再需要學習、已圓滿的解脫境界）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **無學 (Asekha)**：指已經完成所有修行階段，不再需要進一步學習的境界，通常指達到阿羅漢果位的聖者。
2. **福田 (Puññakkhetta)**：比喻像田地一樣，當他人對具足德行的人進行供養或行善時，能產生巨大的善業果報。
3. **戒、定、慧 (Sīla, Samādhi, Paññā)**：佛教修行的核心三學。戒是指規範行為以止惡；定是指集中心念以除亂；慧是指洞察真理以斷除煩惱。
4. **解脫 (Vimokkha)**：指心靈從貪、嗔、癡等煩惱的束縛中完全釋放，不再受輪迴之苦的狀態。

### [現代實作指南]
本經以「良馬」比喻「聖者」，強調內在品質的圓滿會自然帶來外在的尊重。在現代生活中，可將其轉化為以下實踐路徑：

1. **建立「戒身」的自律 ➔ 穩定基石**：
   不追求形式上的禁欲，而是在日常生活中建立明確的底線（如：不欺騙、不傷害他人）。當一個人行為可預測且誠實時，便建立了內在的安定感，如同良馬的「賢善」。

2. **培養「定身」的專注 ➔ 效能提升**：
   在資訊碎片化的時代，練習在單一任務上保持專注（如：正念呼吸或深度工作）。這種心力的集中能將雜亂的思緒轉化為強大的執行力，如同良馬的「捷疾」與「堪能」。

3. **開發「慧身」的覺察 ➔ 轉念技巧**：
   面對衝突時，不立即反應，而是觀察自己的情緒（受）與認知（想）。意識到「這份憤怒是暫時的」或「對方的行為是由其因緣而生」，以此化解執著，如同良馬的「調柔」。

4. **追求「解脫身」的自在 ➔ 心理韌性**：
   練習「不執取」的態度。當我們不再強求結果必須符合預期，而是全然接納當下的實相，心靈便能從壓力與焦慮中解脫，達到一種不被外境左右的自在狀態。

### [段落大意]
佛陀以良馬具備賢善、捷疾、堪能、調柔四種特質而受國王青睞為喻，說明修學佛法的善男子若能圓滿戒、定、慧、解脫這四種「無學」的德行，將成為世人尊崇的殊勝福田。
