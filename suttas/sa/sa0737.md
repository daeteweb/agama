# 雜阿含經 第 737 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「所謂覺分。何等為覺分？」

諸比丘白佛：「世尊是法根、法眼、法依，唯願為說。諸比丘聞已，當受奉行。」

佛告諸比丘、比丘尼：「七覺分者，謂七道品法。諸比丘！此七覺分漸次起，漸次起已，修習滿足。」

諸比丘白佛：「云何七覺分漸次起，漸次起已，修習滿足？

「若比丘身身觀念住，彼身身觀念住已，專心繫念不忘，當於爾時方便修念覺分；方便修念覺分已，修習滿足，謂修念覺分已，於法選擇，當於爾時修擇法覺分方便，修擇法覺分方便已，修習滿足。如是精進、喜、猗、定、捨覺分亦如是說。如內身，如是外身、內外身，受、心、法法觀念住，專心繫念不忘，當於爾時方便修念覺分，方便修念覺分已，修習滿足。乃至捨覺分亦如是說，是名比丘七覺分漸次起，漸次起已，修習滿足。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「所謂覺分。何等為覺分？」**
　　那時，世尊告訴在場的眾位比丘說：「所謂的覺分（覺悟的要素），究竟什麼才是覺分？」

**◈ 諸比丘白佛：「世尊是法根、法眼、法依，唯願為說。諸比丘聞已，當受奉行。」**
　　眾比丘對佛陀恭敬地說道：「世尊您是正法的根源、真理的眼目以及修行的依據，唯願您為我們解說。我們比丘聽聞之後，定會虔誠地接受並依照教導去實踐。」

**◈ 佛告諸比丘、比丘尼：「七覺分者，謂七道品法。諸比丘！此七覺分漸次起，漸次起已，修習滿足。」**
　　佛陀告訴在場的比丘與比丘尼：「所謂的七覺分，是指七種通往覺悟的道品法。比丘們！這七種覺悟要素是循序漸進地生起的，在漸次生起之後，透過修行使之圓滿。」

**◈ 諸比丘白佛：「云何七覺分漸次起，漸次起已，修習滿足？」**
　　眾比丘對佛陀恭敬地說道：「這七覺分是如何循序漸進地生起，並在生起之後透過修行而圓滿的呢？」

**◈ 若比丘身身觀念住，彼身身觀念住已，專心繫念不忘，當於爾時方便修念覺分；**
　　如果比丘對著色（物質身體）進行觀照並安住其中，在對身體觀照安住之後，能專注地繫住正念而不遺忘，在那時就能方便地修習「念覺分」（正念的覺悟要素）；

**◈ 方便修念覺分已，修習滿足，謂修念覺分已，於法選擇，當於爾時修擇法覺分方便，修擇法覺分方便已，修習滿足。**
　　在方便地修習念覺分並使其圓滿之後，接著對法（現象）進行選擇與辨析，在那時就能方便地修習「擇法覺分」（辨別真理的覺悟要素），並在修習擇法覺分後使其圓滿。

**◈ 如是精進、喜、猗、定、捨覺分亦如是說。**
　　同樣地，關於精進覺分、喜（法喜）覺分、猗（輕安）覺分、定（專注）覺分以及捨（平等心）覺分，也是如此循序漸進地修習圓滿。

**◈ 如內身，如是外身、內外身，受、心、法法觀念住，專心繫念不忘，當於爾時方便修念覺分，方便修念覺分已，修習滿足。**
　　如同觀照內在的身體一樣，對外在的身體、內外身體，以及受（感受）、心（行與識的造作）、法（心法現象）進行觀照安住，並專注地繫住正念而不遺忘，在那時就能方便地修習念覺分，並使其圓滿。

**◈ 乃至捨覺分亦如是說，是名比丘七覺分漸次起，漸次起已，修習滿足。**
　　直到最後的捨覺分也是如此說明。這就叫做比丘的七覺分循序漸進地生起，並在生起之後透過修行而圓滿。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **七覺分 (Satta Bojjhaṅga)**：指通往覺悟的七種心理要素，包括：念（正念）、擇法（辨別真理）、精進（持續努力）、喜（法喜）、猗（輕安/身心放鬆）、定（專注）、捨（平等心）。
2. **法根/法眼/法依**：比丘對佛陀的讚嘆，指佛陀是真理的源頭（根）、能洞察一切真理的智慧（眼）以及修行者可以依止的最高準則（依）。
3. **身身觀念住**：指「身念住」，即對色（物質身體）的持續覺知，是七覺分修行的起點。
4. **擇法 (Dhamma-vicaya)**：在正念的基礎上，分析現象的本質，區分何為善、何為不善，何為導致解脫的法，何為導致束縛的法。
5. **猗 (Passaddhi)**：指輕安，即在修行中消除身心的緊張與躁動，達到一種極其放鬆且清明的狀態。

### [現代實作指南]
本經文提供了一套從「覺知」到「解脫」的階梯式心理訓練法，可轉化為以下實踐步驟：

1. **建立基礎覺知（念覺分）** ➔ 
   從最簡單的物質身體（色）開始。例如：在行走或呼吸時，將注意力完全放在身體的感覺上（如腳底觸地、氣息進出），不加評判，僅僅是「繫住念頭」不讓其飄走。

2. **啟動分析功能（擇法覺分）** ➔ 
   當注意力穩定後，開始觀察這些感覺的特性。例如：觀察呼吸的感覺是如何生起又消失的，體會其「無常」的本質。這就是將單純的覺知轉化為對真理的辨析。

3. **維持正向動力（精進 → 喜 → 輕安）** ➔ 
   在辨析中發現真理時，會產生一種自然的喜悅（喜），這種喜悅會讓身心放鬆（輕安），進而讓修行不再感到吃力，而是變成一種享受。

4. **深化專注與平衡（定 → 捨）** ➔ 
   在輕安的狀態下，心更容易進入深度專注（定）。最後，練習對所有出現的現象（無論是痛苦或快樂）保持一種不偏不倚的平等心（捨），不再被情緒牽著走。

**轉念技巧**：當你感到壓力大或心亂時，不要試圖直接強迫自己「平靜」（捨），而應從最底層的「身體覺知」（念）開始，循序漸進地往上走。

### [段落大意]
本經詳細說明了「七覺分」的修習路徑。佛陀教導比丘，覺悟並非一蹴而就，而是從對身體、感受、心法等現象的「正念」開始，進而發展出「擇法」的辨析力，隨後依序觸發精進、喜、輕安、定與捨。這種循序漸進的修習過程，能使修行者從基礎的覺知逐步走向最終的圓滿解脫。
