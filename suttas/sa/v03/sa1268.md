# 雜阿含經 第 1268 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。

時，彼天子白佛言：「比丘知一切眾生所著、所集，決定解脫、廣解脫、極廣解脫耶？」

佛告天子：「我悉知一切眾生所著、所集，決定解脫、廣解脫、極廣解脫。」天子白佛：「比丘云何知一切眾生所著、所集，決定解脫、廣解脫、極廣解脫？」

佛告天子：「愛喜滅盡，我心解脫；心解脫已，故知一切眾生所著、所集，決定解脫、廣解脫、極廣解脫。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。**
　　那時，有一位天子，容貌極其出眾，在深夜時分來到佛陀這裡，頂禮佛陀的足部，然後退到一邊坐下。

**◈ 其身光明遍照祇樹給孤獨園。**
　　他的身體發出的光明照亮了整個祇樹給孤獨園。

**◈ 時，彼天子白佛言：「比丘知一切眾生所著、所集，決定解脫、廣解脫、極廣解脫耶？」**
　　那時，這位天子對佛陀恭敬地說道：「比丘（出家修行者）是否知道一切眾生所執著、所聚集的（貪愛之因），以及如何達成決定性的解脫、廣大的解脫以及極其廣大的解脫呢？」

**◈ 佛告天子：「我悉知一切眾生所著、所集，決定解脫、廣解脫、極廣解脫。」**
　　佛陀告訴天子：「我完全知道一切眾生所執著、所聚集的（貪愛之因），以及決定性的解脫、廣大的解脫與極其廣大的解脫。」

**◈ 天子白佛：「比丘云何知一切眾生所著、所集，決定解脫、廣解脫、極廣解脫？」**
　　天子對佛陀恭敬地說道：「比丘（出家修行者）是如何知道一切眾生所執著、所聚集的（貪愛之因），以及決定性的解脫、廣大的解脫與極其廣大的解脫的？」

**◈ 佛告天子：「愛喜滅盡，我心解脫；心解脫已，故知一切眾生所著、所集，決定解脫、廣解脫、極廣解脫。」**
　　佛陀告訴天子：「當愛（貪愛）與喜（對感官快感的喜悅）完全滅盡時，我的心就獲得了解脫（從束縛中釋放）；在心解脫之後，因此我能知道一切眾生所執著、所聚集的（貪愛之因），以及決定性的解脫、廣大的解脫與極其廣大的解脫。」

**◈ 時，彼天子復說偈言：**
　　那時，這位天子又說了偈頌：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「我曾見過一位婆羅門，最終達到了涅槃（超越生死、絕對寧靜的解脫境界），所有恐懼都已過去，永遠超越了世間的恩愛（對世俗情感的執著）。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，這位天子聽聞佛陀所說的教法後，心生法喜並隨喜讚嘆，頂禮佛陀的足部，隨即消失不見。

### [詞語註解]
1. **所著、所集 (Upādāna / Saṃyojana)**：指眾生對物質或精神對象的「執取」與「結縛」。著是指心中對某物的黏著，集是指因貪愛而聚集的煩惱與業力。
2. **解脫 (Vimokkha)**：指心靈從貪、嗔、癡三毒以及輪迴的束縛中完全釋放，不再受苦。
3. **愛喜 (Taṇhā / Nandi)**：愛是指對感官滿足的渴求（貪愛），喜是指在滿足渴求時產生的愉悅感。這兩者是導致輪迴與痛苦的核心動力。
4. **涅槃 (Nibbāna)**：超越生死、絕對寧靜的解脫境界，是貪、嗔、癡完全熄滅的狀態。

### [現代實作指南]
本經揭示了「解脫」的關鍵在於識別並熄滅「愛喜」。在現代生活中，我們可以透過以下步驟實踐：

1. **識別「所著」 (識別執著點)**：
   ➔ 當你感到焦慮、憤怒或強烈渴望時，停下來問自己：「我現在正『著』在什麼上面？」
   ➔ 例如：著在對他人評價的在意、著在對金錢的掌控感、著在對某種身份的認同。

2. **觀察「愛喜」的運作 (覺察快感陷阱)**：
   ➔ 意識到當我們追求某個目標（愛）並在達成時感到快感（喜）時，這種快感其實是一種「鉤子」，會讓我們在未來更深地被該對象束縛。
   ➔ 練習在感到「喜悅」的瞬間，同步覺察到這份喜悅的「無常」性質，避免將其轉化為長期的執著。

3. **練習「心解脫」的微小步驟 (練習放下)**：
   ➔ 嘗試在日常小事中練習「不執著」。例如：當你期待某件事發生卻未如願時，觀察內心產生的不適感，並告訴自己：「這只是心的造作（行），不需要被它牽著走。」
   ➔ 透過不斷地觀察「渴望 ➔ 獲取 ➔ 喜悅 ➔ 恐懼失去」的循環，心會逐漸對這種模式感到厭離，進而趨向寧靜。

### [段落大意]
一位天子請問佛陀如何得知眾生的執著及其解脫之道。佛陀回答，唯有當內心的貪愛與喜悅（愛喜）完全滅盡，心才能獲得解脫；而一旦心解脫，便能洞悉一切眾生的束縛與解脫之理。天子聞後讚嘆並頂禮消失。
