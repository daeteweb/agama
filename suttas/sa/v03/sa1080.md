# 雜阿含經 第 1080 經

### [原文]
如是我聞：

一時，佛住波羅[木*奈]國仙人住處鹿野苑中。

爾時，世尊晨朝著衣持鉢，入波羅[木*奈]城乞食。

時，有異比丘以不住心，其心惑亂，不攝諸根，晨朝著衣持鉢，入波羅[木*奈]城乞食。是比丘遙見世尊，見已，攝持諸根，端視而行。世尊見是比丘攝持諸根，端視而行，見已入城。乞食畢，還精舍，舉衣鉢，洗足已，入室坐禪。晡時從禪覺，入僧中，敷坐具，於大眾前坐，告諸比丘：「我今晨朝著衣持鉢，入波羅[木*奈]城乞食，見有比丘！以不住心、惑亂心，諸根放散，亦持衣鉢入城乞食，彼遙見我，即自斂攝竟，為是誰？」

時，彼比丘從座起，整衣服，到於佛前，偏袒右肩，合掌白佛：「世尊！我於晨朝入城乞食，其心惑亂，不攝諸根行，遙見世尊，即自斂心，攝持諸根。」

佛告比丘：「善哉！善哉！汝見我已，能自斂心，攝持諸根。比丘！是法應當如是。若見比丘，亦應自攝持。若復見比丘尼、優婆塞、優婆夷，亦當如是攝持諸根，當得長夜以義饒益，安隱快樂。」

爾時，眾中復有異比丘說偈歎曰：

「以其心迷亂，　　不專繫念住， 　晨朝持衣鉢，　　入城邑乞食。 　中路見大師，　　威德容儀備， 　欣悅生慚愧，　　即攝持諸根。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住波羅[木*奈]國仙人住處鹿野苑中。**
　　在某一個時間，佛住在波羅奈國（今印度瓦拉納西）仙人居住的鹿野苑中。

**◈ 爾時，世尊晨朝著衣持鉢，入波羅[木*奈]城乞食。**
　　那時，世尊在早晨穿好袈裟、拿著缽，進入波羅奈城乞食。

**◈ 時，有異比丘以不住心，其心惑亂，不攝諸根，晨朝著衣持鉢，入波羅[木*奈]城乞食。**
　　那時，有一位比丘因為心念不專注（不住心），內心感到困惑混亂，沒有攝持諸根（指未能控制感官的覺知，使其不隨外境散亂），也在早晨穿好袈裟、拿著缽，進入波羅奈城乞食。

**◈ 是比丘遙見世尊，見已，攝持諸根，端視而行。**
　　這位比丘從遠處看見了世尊，看見之後，立刻攝持諸根（收斂感官，將注意力集中於當下），端正地注視著前方行走。

**◈ 世尊見是比丘攝持諸根，端視而行，見已入城。**
　　世尊看見這位比丘攝持諸根、端正行走，看見之後便進入城內。

**◈ 乞食畢，還精舍，舉衣鉢，洗足已，入室坐禪。**
　　乞食結束後，回到精舍，放下衣鉢，洗淨雙足，進入房間坐禪修行。

**◈ 晡時從禪覺，入僧中，敷坐具，於大眾前坐，告諸比丘：**
　　到了傍晚從禪定中醒來，進入比丘僧團中，鋪好坐具，坐在大眾面前，告訴在場的眾位比丘說：

**◈ 「我今晨朝著衣持鉢，入波羅[木*奈]城乞食，見有比丘！以不住心、惑亂心，諸根放散，亦持衣鉢入城乞食，彼遙見我，即自斂攝竟，為是誰？」**
　　「我今天早晨穿好袈裟、拿著缽，進入波羅奈城乞食時，見到了一位比丘！他當時心念不專注、內心惑亂，感官放散（不攝諸根），同樣拿著衣鉢入城乞食，但他在遠處看見我時，立刻將心收斂攝持，請問這位比丘是誰？」

**◈ 時，彼比丘從座起，整衣服，到於佛前，偏袒右肩，合掌白佛：**
　　那時，那位比丘從座位上站起來，整理衣服，走到佛陀面前，整理衣服並露出右肩，雙手合十對佛陀恭敬地說道：

**◈ 「世尊！我於晨朝入城乞食，其心惑亂，不攝諸根行，遙見世尊，即自斂心，攝持諸根。」**
　　「世尊！我今天早晨入城乞食時，內心確實惑亂，沒有攝持諸根而行走，但在遠處看見世尊後，立刻將心收斂，攝持諸根。」

**◈ 佛告比丘：「善哉！善哉！汝見我已，能自斂心，攝持諸根。**
　　佛告訴比丘：「好極了！好極了！你在看見我之後，能夠自我收斂心念，攝持諸根。

**◈ 比丘！是法應當如是。若見比丘，亦應自攝持。**
　　比丘！這種法（修行方式）應當如此。如果看見其他比丘，也應當自我攝持。

**◈ 若復見比丘尼、優婆塞、優婆夷，亦當如是攝持諸根，當得長夜以義饒益，安隱快樂。」**
　　如果看見比丘尼、優婆塞（男信徒）、優婆夷（女信徒），也應當如此攝持諸根，這樣才能在漫長的生命之夜（輪迴）中獲得意義上的饒益，得到安穩與快樂。」

**◈ 爾時，眾中復有異比丘說偈歎曰：**
　　那時，眾人之中又有一位比丘用偈頌讚歎說道：

**◈ 「以其心迷亂，不專繫念住，晨朝持衣鉢，入城邑乞食。**
　　「因為心念迷亂，沒有專注地繫住正念，早晨拿著衣鉢，進入城邑乞食。

**◈ 中路見大師，威德容儀備，欣悅生慚愧，即攝持諸根。」**
　　在路途中看見大師（佛陀），其威德與容儀如此完備，心中生起欣悅與慚愧，立刻攝持諸根。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **攝持諸根 (Indriya-saṃvara)**：指對感官（眼、耳、鼻、舌、身、意）的覺知進行管理。當感官接觸到外界對象時，不隨之產生貪著或厭惡的反應，使心保持在正念狀態，防止心念散亂。
2. **不住心**：指心念缺乏定力，隨境而轉，無法專注於當下的對象，處於一種散亂（Uddhacca）的狀態。
3. **慚愧 (Hiri-Ottappa)**：慚（Hiri）是指對自己不端正行為的內在羞愧；愧（Ottappa）是指對違背道德或法義行為的恐懼。在文中，比丘見佛後意識到自己的散亂，因而產生慚愧心，這成為了轉化心念的動力。
4. **長夜**：佛教比喻輪迴（Saṃsāra）的漫長且充滿無明黑暗的過程。

### [現代實作指南]
本經文提供了一個關於「利用外部觸發點恢復正念」的實踐模型，可用於現代生活中的壓力管理與情緒調節：

1. **識別「散亂模式」**：
   意識到自己在行走、工作或社交時，是否處於「不住心」的狀態（例如：身體在走，心在想昨天的爭執或明天的焦慮）。這種狀態會導致對當下的感知下降，增加錯誤率與壓力。

2. **建立「正念觸發點」 (Mindfulness Trigger)**：
   經中比丘將「看見佛陀」作為恢復正念的觸發點。在現代生活中，我們可以設定自己的觸發點：
   ➔ **環境觸發**：例如，每當走過一扇門、看到紅燈、或聽到手機通知聲時，立刻進行一次深呼吸。
   ➔ **人物觸發**：將某些具有正向影響力的人（如導師、長輩或溫柔的朋友）視為「提醒者」。每當與他們接觸時，提醒自己收斂心念，回到當下。

3. **執行「攝持諸根」的微練習**：
   當觸發點出現時，執行以下步驟：
   ➔ **覺察**：意識到「我現在心很亂」。
   ➔ **收斂**：將注意力從外部的雜念，迅速拉回到身體的感覺（如腳掌接觸地面的感覺，或呼吸的起伏）。
   ➔ **端正**：調整身體姿態（如挺直背脊），透過身體的端正來誘導心靈的安定。

4. **將「慚愧」轉化為「覺醒」**：
   當發現自己散亂時，不要陷入自我責備，而應像經中比丘一樣，將這種「不對勁」的感覺轉化為一種「欣悅的覺醒」 ➔ 「太好了，我發現自己散亂了，現在我可以回來了」。

### [段落大意]
本經敘述一名比丘在乞食途中心念散亂，但在遠見世尊後，受其威德感召，立刻收斂心念並攝持諸根。佛陀以此鼓勵眾人，無論面對比丘、比丘尼或在家信徒，都應保持攝持諸根的覺知，如此方能在輪迴長夜中獲得安穩快樂。
