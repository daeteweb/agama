# 雜阿含經 第 1333 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有異比丘在拘薩羅住林中，入晝正受，心起不善覺，依於惡貪。

時，彼林中住止天神作是念：「非比丘法，止住林中，入晝正受，心生不善覺，依於惡貪，我今當往開悟之。」

時，彼天神即說偈言：

「其心欲遠離，　　正於空閑林， 　放心隨外緣，　　亂想而流馳。 　調伏樂世心，　　常樂心解脫， 　當捨不樂心，　　執受安樂住。 　思非於正念，　　莫著我我所， 　如以塵頭染，　　是著極難遣。 　莫令染樂著，　　欲心所濁亂， 　如釋君馳象，　　奮迅去塵穢。 　比丘於自身，　　正念除塵垢， 　塵者謂貪欲，　　非世間塵土。 　黠慧明智者，　　當悟彼諸塵， 　於如來法律，　　持心莫放逸。 　塵垢謂瞋恚，　　非世間塵土， 　黠慧明智者，　　當悟彼諸塵。 　於如來法律，　　持心莫放逸， 　塵垢謂愚癡，　　非世間塵土。 　明智黠慧者，　　當捨彼諸塵， 　於如來法律，　　持心莫放逸。」

時，彼天神說是偈已，彼比丘聞其所說，專精思惟，斷諸煩惱心，得阿羅漢。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有異比丘在拘薩羅住林中，入晝正受，心起不善覺，依於惡貪。**
　　那時，有一位比丘住在拘薩羅國的森林中，在白天的禪定狀態（正受）中，心中升起了不善的覺知，依止於惡劣的貪愛。

**◈ 時，彼林中住止天神作是念：「非比丘法，止住林中，入晝正受，心生不善覺，依於惡貪，我今當往開悟之。」**
　　那時，住在該森林中的天神心想：「這不符合比丘的修行法門。住在森林中，在白天的禪定狀態中，心中竟然生起不善的覺知並依止於惡劣的貪愛，我現在應該前往使他覺悟。」

**◈ 時，彼天神即說偈言：**
　　於是，那位天神立即說了以下偈頌：

**◈ 「其心欲遠離，正於空閑林，放心隨外緣，亂想而流馳。」**
　　「他的心本應追求遠離煩惱，且正處於空閑的森林之中，卻讓心隨外在的緣分而漂移，陷入混亂的妄想中奔馳。」

**◈ 「調伏樂世心，常樂心解脫，當捨不樂心，執受安樂住。」**
　　「應當調伏追求世俗快樂的心，使心常在解脫（超越生死、絕對寧靜的境界）中安樂；應捨棄那令人不安的貪欲心，而安住在真正的安樂之中。」

**◈ 「思非於正念，莫著我我所，如以塵頭染，是著極難遣。」**
　　「思惟時若缺乏正念（正確的覺知），就容易執著於『我』以及『屬於我的』這種無我（Anattā）的錯覺；這種執著就像被微小的塵埃染污一樣，極其難以清除。」

**◈ 「莫令染樂著，欲心所濁亂，如釋君馳象，奮迅去塵穢。」**
　　「不要讓心中染上對快樂的執著，導致欲念使心變得濁亂；應像馴象師驅使大象一樣，奮力地遠離內心的塵穢。」

**◈ 「比丘於自身，正念除塵垢，塵者謂貪欲，非世間塵土。」**
　　「比丘應對自己的內心運用正念（正確的覺知）來清除塵垢；這裡所說的『塵』是指貪欲（愛/貪愛），而不是指世間物質的塵土。」

**◈ 「黠慧明智者，當悟彼諸塵，於如來法律，持心莫放逸。」**
　　「聰明且具備智慧的人，應當覺察到這些內心的塵垢，依照如來（成就不變真理而來到世間的人）的法（真理）與律（戒律），持守心念，不要放逸。」

**◈ 「塵垢謂瞋恚，非世間塵土，黠慧明智者，當悟彼諸塵。於如來法律，持心莫放逸，」**
　　「這裡的塵垢是指瞋恚（憤怒），而不是指世間物質的塵土。聰明且具備智慧的人，應當覺察到這些內心的塵垢，依照如來（成就不變真理而來到世間的人）的法與律，持守心念，不要放逸。」

**◈ 「塵垢謂愚癡，非世間塵土。明智黠慧者，當捨彼諸塵，於如來法律，持心莫放逸。」**
　　「這裡的塵垢是指愚癡（無明），而不是指世間物質的塵土。明智且聰明的人，應當捨棄這些內心的塵垢，依照如來（成就不變真理而來到世間的人）的法與律，持守心念，不要放逸。」

**◈ 時，彼天神說是偈已，彼比丘聞其所說，專精思惟，斷諸煩惱心，得阿羅漢。**
　　那時，天神說完偈頌後，那位比丘聽聞其教導，專注且深入地思考，斷除了所有煩惱（內心的染污與執著），證得了阿羅漢（已斷除所有煩惱的聖者）果位。

### [詞語註解]
1. **正受 (Sama-apatti)**：指進入禪定（Jhana）的狀態，心意識集中於單一對象而達到平靜與統一。
2. **正念 (Sati)**：對當下身心狀態的正確覺知，不隨之起舞，也不予以壓抑，僅僅是清晰地觀察。
3. **我我所 (Ahankāra-mamankāra)**：一種錯誤的認知，認為有一個永恆不變的「我」存在，並將現象標記為「我的」。
4. **放逸 (Pamāda)**：心不在焉、缺乏警覺，任由心意識隨欲望或妄想漂流而未加調伏。
5. **貪、瞋、癡 (Lobha, Dosa, Moha)**：佛教定義的三毒，分別為貪愛、憤怒與無明（愚癡），是導致輪迴（Saṃsāra）的核心原因。

### [現代實作指南]
本經文提供了一套將「心理染污」視覺化為「塵垢」的清理機制，可用於現代壓力管理與情緒調節：

1. **識別「心塵」 (Labeling)**：
   當強烈的情緒升起時，不要將其視為「我」的一部分（例如：不要說「我很生氣」），而將其標記為「塵垢」。
   ➔ 貪欲 ➔ 「貪塵」
   ➔ 憤怒 ➔ 「瞋塵」
   ➔ 迷茫/偏見 ➔ 「癡塵」

2. **建立「觀察者」視角 (Mindfulness)**：
   效法經中天神的提醒，意識到心正在「隨外緣流馳」。當你發現自己在社交媒體上不斷刷屏或在心中反覆咀嚼憤怒時，立刻對自己說：「心現在正在流馳，塵垢正在聚集。」

3. **執行「心靈除塵」 (Cleansing)**：
   - **面對貪欲**：意識到追求的快感如同塵埃，雖暫時顯現但本質空虛，透過「捨」的練習將其清除。
   - **面對瞋恚**：意識到憤怒會使心變得濁亂，如同塵土遮蔽鏡面，透過深呼吸與正念觀察，讓憤怒自然沉澱。
   - **面對愚癡**：透過閱讀正法或邏輯分析，打破對「我」的執著，用智慧之光照亮無明。

4. **持守不放逸 (Consistency)**：
   將此練習習慣化。每天設定 5-10 分鐘的「除塵時間」，回顧今日心中升起的染污，並在心中將其輕輕拭去，保持心靈的清明。
