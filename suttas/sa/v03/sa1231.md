# 雜阿含經 第 1231 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，波斯匿王於正殿上自觀察王事，見勝剎利大姓、見勝婆羅門大姓、見勝長者大姓因貪欲故，欺詐妄語，即作是念：「止此斷事！息此斷事！我更不復親臨斷事；我有賢子，當令斷事。云何自見此勝剎利大姓、婆羅門大姓、長者大姓為貪欲故，欺詐妄語？」

時，波斯匿王作是念已，往詣佛所，稽首佛足，退坐一面，白佛言：「世尊！我於殿上自斷王事，見諸勝剎利大姓、婆羅門大姓，長者大姓為貪利故，欺詐妄語。世尊！我見是事已，作是念：『我從今日，止此斷事，息此斷事；我有賢子，當令其斷。不親自見此勝剎利大姓、婆羅門大姓、長者大姓緣貪利故，欺詐妄語。』」

佛告波斯匿王：「如是，大王！如是，大王！彼勝剎利大姓、婆羅門大姓、長者大姓因貪利故，欺詐妄語，彼愚癡人長夜當得不饒益苦！大王當知，譬如漁師、漁師弟子，於河溪谷截流張網，殘殺眾生，令遭大苦。如是，大王！彼勝剎利大姓、婆羅門大姓、長者大姓因貪利故，欺詐妄語，長夜當得不饒益苦！」爾時，世尊復說偈言：

「於財起貪欲，　　貪欲所迷醉， 　狂亂不自覺，　　猶如漁捕者， 　緣斯惡業故，　　當受劇苦報。」

佛說此經已，波斯匿王聞佛所說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，波斯匿王於正殿上自觀察王事，見勝剎利大姓、見勝婆羅門大姓、見勝長者大姓因貪欲故，欺詐妄語，即作是念：「止此斷事！息此斷事！我更不復親臨斷事；我有賢子，當令斷事。云何自見此勝剎利大姓、婆羅門大姓、長者大姓為貪欲故，欺詐妄語？」**
　　那時，波斯匿王在正殿上親自處理王室的司法審判事務，看到那些顯赫的剎利大姓、婆羅門大姓以及富有的長者大姓，因為貪欲（對物質的強烈渴求）的緣故，而說謊欺騙。他立刻心裡這樣想：「停止處理這些審判事務吧！結束這些審判事務吧！我不再親自出席審判；我有聰明賢能的兒子，應該讓他來處理審判。為什麼我會親眼看到這些顯赫的剎利大姓、婆羅門大姓、長者大姓，竟然因為貪欲而說謊欺騙？」

**◈ 時，波斯匿王作是念已，往詣佛所，稽首佛足，退坐一面，白佛言：「世尊！我於殿上自斷王事，見諸勝剎利大姓、婆羅門大姓，長者大姓為貪利故，欺詐妄語。世尊！我見是事已，作是念：『我從今日，止此斷事，息此斷事；我有賢子，當令其斷。不親自見此勝剎利大姓、婆羅門大姓、長者大姓緣貪利故，欺詐妄語。』」**
　　那時，波斯匿王在心中這樣想過後，前往佛陀所在之處，向佛陀的足下頂禮，退到一旁坐下，對佛陀恭敬地說道：「世尊！我在殿上親自審理王事，看到許多顯赫的剎利大姓、婆羅門大姓、長者大姓，為了貪利而說謊欺騙。世尊！我看到這件事後，心裡這樣想：『我從今天起，停止處理這些審判事務，結束這些審判事務；我有賢能的兒子，將令他來處理。我不再親自看到這些顯赫的剎利大姓、婆羅門大姓、長者大姓，因為貪利而說謊欺騙。』」

**◈ 佛告波斯匿王：「如是，大王！如是，大王！彼勝剎利大姓、婆羅門大姓、長者大姓因貪利故，欺詐妄語，彼愚癡人長夜當得不饒益苦！**
　　佛陀告訴波斯匿王：「確實如此，大王！確實如此，大王！那些顯赫的剎利大姓、婆羅門大姓、長者大姓，因為貪利而說謊欺騙，那些被愚癡（缺乏對真理的覺知）所蒙蔽的人，在漫長的輪迴（Saṃsāra）中，將會承受得不到任何利益的痛苦！

**◈ 大王當知，譬如漁師、漁師弟子，於河溪谷截流張網，殘殺眾生，令遭大苦。如是，大王！彼勝剎利大姓、婆羅門大姓、長者大姓因貪利故，欺詐妄語，長夜當得不饒益苦！」**
　　大王應當了解，譬如漁夫及其弟子，在河川溪谷中截斷水流並張開漁網，殘殺眾生，使眾生遭受巨大的痛苦。同樣地，大王！那些顯赫的剎利大姓、婆羅門大姓、長者大姓，因為貪利而說謊欺騙，在漫長的輪迴中，將會承受得不到任何利益的痛苦！」

**◈ 爾時，世尊復說偈言：「於財起貪欲，貪欲所迷醉，狂亂不自覺，猶如漁捕者，緣斯惡業故，當受劇苦報。」**
　　那時，世尊又說了一首偈頌：「對財富產生貪欲，被貪欲所迷醉，導致心智狂亂而不能自覺，就像那些捕魚的人一樣，因為造了這種惡業（Kamma），將會承受劇烈的業報（Vipāka）。」

**◈ 佛說此經已，波斯匿王聞佛所說，歡喜隨喜，作禮而去。**
　　佛陀講完這部經後，波斯匿王聽聞佛陀所說的教法，心生法喜並依照教導去實踐，向佛陀行禮後才離開。

### [詞語註解]
1. **貪欲 (Lobha/Taṇhā)**：對感官對象或物質財富的強烈渴求與執著，是導致苦受的主要原因。
2. **愚癡 (Moha)**：對四聖諦與緣起真理的無知，無法分辨善惡與真偽的心理狀態。
3. **輪迴 (Saṃsāra)**：受業力驅使，在六道中不斷出生、老死、再出生的循環過程。
4. **業 (Kamma)**：由意圖（思）驅使的身體、言語或心理行為。
5. **業報 (Vipāka)**：行為之後所產生的結果，善業導致樂報，惡業導致苦報。

### [現代實作指南]
本經文揭示了「貪欲 ➔ 愚癡 ➔ 妄語 ➔ 苦報」的心理連鎖反應。在現代生活中，我們可以透過以下步驟實踐轉念：

1. **覺察貪欲的萌芽**：當我們在工作中或人際關係中，為了獲取某種利益（金錢、名聲、權力）而產生強烈渴望時，立即在心中標記：「現在貪欲が生起了」。
2. **觀察「妄語」的誘因**：當我們發現自己為了掩蓋錯誤或獲取利益而想要說謊時，意識到這正是「愚癡」的表現 ➔ 即將短期的利益誤認為長期的幸福。
3. **建立「業報」的邏輯連結**：提醒自己，任何基於欺詐的獲益，都會在心理上造成不安（壓力、恐懼被揭穿），這便是即時的「苦報」。而長遠來看，誠信的崩潰將導致更深重的孤立與痛苦。
4. **實踐「誠實」的定力**：練習在利益衝突時，選擇誠實面對。即使短期內看似損失，但能獲得內心的寧靜與長久的信用，這才是真正的「饒益」。

### [段落大意]
波斯匿王在審理案件時，發現許多社會高層人士因貪婪而說謊，深感厭惡而決定不再親自審判。佛陀對此表示認同，並指出這些人因愚癡而造惡業，如同漁夫殘殺眾生般，將在漫長的輪迴中承受劇烈的苦報。佛陀以此警示貪欲會使人迷醉狂亂，最終導致痛苦的業報。
