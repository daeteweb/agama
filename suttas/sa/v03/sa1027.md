# 雜阿含經 第 1027 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園……如上說。差別者，乃至佛告病比丘：「汝不自犯戒耶？」

比丘白佛言：「世尊！我不以持淨戒故，於世尊所修梵行。」

佛告比丘：「汝以何等法故，於我所修梵行？」

比丘白佛：「為離貪欲故，於世尊所修梵行；為離瞋恚、愚癡故，於世尊所修梵行。」

佛告比丘：「如是，如是。汝正應為離貪欲故，於我所修梵行；離瞋恚、愚癡故，於我所修梵行。比丘！貪欲纏故，不得離欲；無明纏故，慧不清淨。是故，比丘！於欲離欲心解脫，離無明故慧解脫。若比丘於欲離欲心解脫身作證，離無明故慧解脫，是名比丘斷諸愛欲，轉結縛，止慢無間等，究竟苦邊。是故，比丘！於此法善思惟……」如前廣說，乃至「受第一記。」

佛說此經已，諸比丘聞佛所說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園……如上說。差別者，乃至佛告病比丘：「汝不自犯戒耶？」**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園……（內容）如同前面所說的。不同之處在於，直到佛陀告訴一位生病的比丘說：「你沒有違反戒律嗎？」

**◈ 比丘白佛言：「世尊！我不以持淨戒故，於世尊所修梵行。」**
　　比丘對佛陀恭敬地說道：「世尊！我並不是為了維持淨潔的戒律，才在世尊您這裡修行梵行（清淨的生活）。」

**◈ 佛告比丘：「汝以何等法故，於我所修梵行？」**
　　佛陀告訴比丘：「你是基於什麼樣的法（目的與真理），才在我這裡修行梵行（清淨的生活）？」

**◈ 比丘白佛：「為離貪欲故，於世尊所修梵行；為離瞋恚、愚癡故，於世尊所修梵行。」**
　　比丘對佛陀說：「為了遠離貪欲（對感官對象的渴求），我才在世尊您這裡修行梵行；為了遠離瞋恚（憤怒）與愚癡（對真理的無知），我才在世尊您這裡修行梵行。」

**◈ 佛告比丘：「如是，如是。汝正應為離貪欲故，於我所修梵行；離瞋恚、愚癡故，於我所修梵行。」**
　　佛陀告訴比丘：「確實如此，就是這樣。你正應該為了遠離貪欲（對感官對象的渴求），而在我這裡修行梵行；為了遠離瞋恚（憤怒）與愚癡（對真理的無知），而在我這裡修行梵行。」

**◈ 「比丘！貪欲纏故，不得離欲；無明纏故，慧不清淨。」**
　　「比丘！因為被貪欲（對感官對象的渴求）所纏繞，所以無法遠離欲求；因為被無明（對四聖諦的無知）所纏繞，所以智慧（對實相的洞察力）就不清淨。」

**◈ 「是故，比丘！於欲離欲心解脫，離無明故慧解脫。」**
　　「因此，比丘！在欲求中遠離欲求，心就能獲得解脫；因為遠離了無明（對真理的無知），智慧就能獲得解脫。」

**◈ 「若比丘於欲離欲心解脫身作證，離無明故慧解脫，是名比丘斷諸愛欲，轉結縛，止慢無間等，究竟苦邊。」**
　　「如果比丘在欲求中遠離欲求，使心獲得解脫並親身證悟，且因為遠離無明而使智慧解脫，這就稱為比丘斷除了所有愛欲（對生存與對象的執著），轉化並消除了結縛（束縛心靈的煩惱），停止了傲慢以及其他不間斷的煩惱，最終到達苦的邊緣（即終結苦）。」

**◈ 「是故，比丘！於此法善思惟……」如前廣說，乃至「受第一記。」**
　　「因此，比丘！你應當對這個法好好思考……」如同前面詳細說過的，直到「獲得了佛陀對其將成正果的第一預言」。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜隨喜，作禮而去。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並隨喜讚嘆，向佛陀行禮後才離開。

### [詞語註解]
1. **梵行 (Brahmacariya)**：原意為「梵天般的生活」，在原始佛教中指清淨的修行生活，旨在遠離感官欲望，追求精神的絕對純淨。
2. **無明 (Avijjā)**：對四聖諦（苦、集、滅、道）的無知，是十二因緣中的首環，也是所有痛苦與輪迴的根本原因。
3. **結縛 (Samyojana)**：指將眾生束縛在輪迴（Saṃsāra）之中的心理枷鎖，如色欲結、有欲結、見結、戒結等。
4. **慢 (Māna)**：一種對自我價值的錯誤認知，表現為傲慢、自大或與他人比較而產生的優越感/自卑感。
5. **第一記 (First Prediction)**：佛陀對某位修行者在未來定會達成阿羅漢果或佛果的正式預言。

### [現代實作指南]
本經文揭示了修行動機的關鍵：**「目的論」優於「形式論」**。許多人在生活中追求「正確的行為」（如持戒、遵守規範），但若僅是為了形式上的「淨潔」，則容易陷入僵化或自我滿足。

**實作步驟：**
1. **審視動機 (Motivation Audit)**：
   當你在執行某項正向行為（如冥想、閱讀、幫助他人）時，問自己：「我這樣做是為了維持一個『我很善良/很虔誠』的形象（形式上的淨潔），還是為了真正地遠離內心的貪婪與憤怒（實質的解脫）？」
2. **識別「纏繞」 (Identifying Entanglements)**：
   - 當你感到強烈渴望某物時 ➔ 標記為「貪欲纏」，意識到這種渴求正在限制你的自由。
   - 當你對某事感到憤怒或困惑時 ➔ 標記為「無明纏」，意識到你目前缺乏對該事件實相的洞察。
3. **從「對抗」轉為「遠離」 (From Fighting to Detaching)**：
   不要試圖用暴力地壓抑欲望，而是透過「善思惟」觀察欲望產生的過程。當你發現欲望是因緣而生且無常時，心會自然地從欲求中「解脫」，而非強行「禁止」。
4. **轉化傲慢 (Transforming Conceit)**：
   當你發現自己比他人做得更好而產生優越感時，提醒自己這正是「慢」的結縛。將注意力從「我比誰好」轉向「我還在哪些地方被無明纏繞」。

### [段落大意]
本段記錄佛陀與一名病比丘的對話。佛陀指出，修行的目的不在於形式上的持戒，而是在於遠離貪、瞋、癡。貪欲與無明如同枷鎖纏繞心靈，導致無法離欲且智慧不清淨。唯有透過遠離欲求與無明，才能獲得心與慧的解脫，進而斷除愛欲、消除結縛與傲慢，最終終結痛苦。
