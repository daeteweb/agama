# 雜阿含經 第 1042 經

### [原文]
如是我聞：

一時，佛在拘薩羅國人間遊行，住鞞羅磨聚落北身恕林中。

鞞羅聚落婆羅門長者聞世尊住聚落北身恕林中，聞已，共相招集，往詣佛所，稽首佛足，退坐一面，白佛言：「世尊！何因、何緣有眾生身壞命終，生地獄中？」

佛告諸婆羅門長者：「行非法行、行危嶮行因緣故，身壞命終，生地獄中。」諸婆羅門長者白佛：「行何等非法行、危嶮行，身壞命終，生地獄中？」

佛告婆羅門長者：「殺生，乃至邪見，具足十不善業因緣故。婆羅門！是非法行、危嶮行，身壞命終，生地獄中。」

婆羅門白佛：「何因緣諸眾生身壞命終，得生天上？」

佛告婆羅門長者：「行法行、行正行，以是因緣故，身壞命終，得生天上。」

復問：「世尊！行何等法行、何等正行，身壞命終，得生天上？」

佛告婆羅門長者：「謂離殺生，乃至正見，十善業跡因緣故，身壞命終，得生天上。婆羅門長者！若有行此法行、行此正行者，欲求剎利大性家、婆羅門大性家、居士大性家，悉得往生。所以者何？以法行、正行因緣故。

「若復欲求生四王、三十三天，乃至他化自在天，悉得往生。所以者何？以法行、正行故，行淨戒者，其心所願，悉自然得。

「若復如是法行、正行者，欲求生梵天，亦得往生。所以者何？以行正行、法行故，持戒清淨，心離愛欲，所願必得。

「若復欲求往生光音、遍淨，乃至阿伽尼吒，亦復如是。所以者何？以彼持戒清淨，心離欲故。

「若復欲求離欲、惡不善法，有覺有觀，乃至第四禪具足住，悉得成就。所以者何？以彼法行、正行故，持戒清淨，心離愛欲，所願必得。

「欲求慈、悲、喜、捨，空入處、識入處、無所有入處、非想非非想入處皆悉得。所以者何？以法行、正行故，持戒清淨，心離愛欲，所願必得。

「欲求斷三結，得須陀洹、斯陀含、阿那含果，無量神通，天耳、他心智、宿命智、生死智、漏盡智皆悉得。所以者何？以法行、正行故，持戒、離欲，所願必得。」

時，婆羅門長者聞佛所說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛在拘薩羅國人間遊行，住鞞羅磨聚落北身恕林中。**
　　在某一個時間，佛在拘薩羅國的民眾之間遊行，住在鞞羅磨聚落北邊的身恕林中。

**◈ 鞞羅聚落婆羅門長者聞世尊住聚落北身恕林中，聞已，共相招集，往詣佛所，稽首佛足，退坐一面，白佛言：「世尊！何因、何緣有眾生身壞命終，生地獄中？」**
　　鞞羅聚落的婆羅門長者們聽說世尊住在聚落北邊的身恕林中，聽聞之後，他們共同招集在一起，前往佛陀所在之處，向佛陀的足下頂禮，然後退到一邊坐下，對佛陀恭敬地說道：「世尊！是因為什麼原因、什麼樣的因緣，會導致眾生在色（物質身體）毀壞、生命終結之後，出生在痛苦的地獄之中？」

**◈ 佛告諸婆羅門長者：「行非法行、行危嶮行因緣故，身壞命終，生地獄中。」**
　　佛告訴眾位婆羅門長者：「是因為實踐了不符合正法的行為、實踐了危險且險峻的行為（指不善業）之因緣，導致色（物質身體）毀壞、生命終結後，出生在地獄之中。」

**◈ 諸婆羅門長者白佛：「行何等非法行、危嶮行，身壞命終，生地獄中？」**
　　眾位婆羅門長者對佛陀恭敬地說道：「實踐什麼樣的非法行為與危險行為，會導致色（物質身體）毀壞、生命終結後，出生在地獄之中？」

**◈ 佛告婆羅門長者：「殺生，乃至邪見，具足十不善業因緣故。婆羅門！是非法行、危嶮行，身壞命終，生地獄中。」**
　　佛告訴婆羅門長者：「從殺生開始，直到持有錯誤的認知（邪見），具足了這十種不善業的因緣。婆羅門！這就是所謂的非法行為與危險行為，會導致色（物質身體）毀壞、生命終結後，出生在地獄之中。」

**◈ 婆羅門白佛：「何因緣諸眾生身壞命終，得生天上？」**
　　婆羅門對佛陀恭敬地說道：「是什麼樣的因緣，能讓眾生在色（物質身體）毀壞、生命終結後，能夠出生在天界？」

**◈ 佛告婆羅門長者：「行法行、行正行，以是因緣故，身壞命終，得生天上。」**
　　佛告訴婆羅門長者：「實踐符合正法的行為、實踐正確的行為，因為這樣的因緣，在色（物質身體）毀壞、生命終結後，能夠出生在天界。」

**◈ 復問：「世尊！行何等法行、何等正行，身壞命終，得生天上？」**
　　他們另外詢問：「世尊！實踐什麼樣的法行與正行，能讓人在色（物質身體）毀壞、生命終結後，出生在天界？」

**◈ 佛告婆羅門長者：「謂離殺生，乃至正見，十善業跡因緣故，身壞命終，得生天上。婆羅門長者！若有行此法行、行此正行者，欲求剎利大性家、婆羅門大性家、居士大性家，悉得往生。所以者何？以法行、正行因緣故。」**
　　佛告訴婆羅門長者：「也就是指遠離殺生，直到持有正確的認知（正見），具足了十種善業的痕跡之因緣，在色（物質身體）毀壞、生命終結後，能出生在天界。婆羅門長者！如果有人實踐這種法行與正行，若希望出生在剎利大性家、婆羅門大性家或居士大性家等富貴家庭，都能夠往生。為什麼呢？是因為實踐了法行與正行的因緣。」

**◈ 「若復欲求生四王、三十三天，乃至他化自在天，悉得往生。所以者何？以法行、正行故，行淨戒者，其心所願，悉自然得。」**
　　「如果進一步希望出生在四王天、三十三天，乃至他化自在天，也都能夠往生。為什麼呢？是因為實踐了法行與正行，且持守清淨戒律的人，其內心所願望的，都能夠自然地獲得。」

**◈ 「若復如是法行、正行者，欲求生梵天，亦得往生。所以者何？以行正行、法行故，持戒清淨，心離愛欲，所願必得。」**
　　「如果能如此實踐法行與正行，若希望出生在梵天，也能夠往生。為什麼呢？是因為實踐了正行與法行，持守清淨的戒律，且內心遠離了貪愛與欲望，所願望的必定能達成。」

**◈ 「若復欲求往生光音、遍淨，乃至阿伽尼吒，亦復如是。所以者何？以彼持戒清淨，心離欲故。」**
　　「如果進一步希望往生到光音天、遍淨天，乃至最高層的阿伽尼吒天，情況也是一樣的。為什麼呢？是因為他們持守清淨的戒律，且內心遠離了欲望。」

**◈ 「若復欲求離欲、惡不善法，有覺有觀，乃至第四禪具足住，悉得成就。所以者何？以彼法行、正行故，持戒清淨，心離愛欲，所願必得。」**
　　「如果進一步希望遠離欲望、消除不善的法，透過覺知（正念）與觀察（正定），乃至於完全進入第四禪的定境，都能夠成就。為什麼呢？是因為實踐了法行與正行，持守清淨的戒律，且內心遠離了貪愛與欲望，所願望的必定能達成。」

**◈ 「欲求慈、悲、喜、捨，空入處、識入處、無所有入處、非想非非想入處皆悉得。所以者何？以法行、正行故，持戒清淨，心離愛欲，所願必得。」**
　　「若希望獲得慈、悲、喜、捨這四種無量心，以及進入空入處、識入處、無所有入處、非想非非想入處等深層定境，全部都能夠獲得。為什麼呢？是因為實踐了法行與正行，持守清淨的戒律，且內心遠離了貪愛與欲望，所願望的必定能達成。」

**◈ 「欲求斷三結，得須陀洹、斯陀含、阿那含果，無量神通，天耳、他心智、宿命智、生死智、漏盡智皆悉得。所以者何？以法行、正行故，持戒、離欲，所願必得。」**
　　「若希望斷除三種束縛（三結），獲得須陀洹（初果）、斯陀含（二果）、阿那含（三果）的聖果，以及無量的神通，如天耳、知道他人心意、知道過去生命、知道生死輪迴、以及消除所有煩惱的漏盡智，全部都能夠獲得。為什麼呢？是因為實踐了法行與正行，持守戒律並遠離欲望，所願望的必定能達成。」

**◈ 時，婆羅門長者聞佛所說，歡喜隨喜，作禮而去。**
　　那時，婆羅門長者們聽聞佛陀所說的教法，內心感到歡喜並隨喜，向佛陀行禮後才離開。

### [詞語註解]
1. **十不善業 (Akusala-kamma)**：指十種導致痛苦果報的惡行，包括身業（殺生、偷盜、邪淫）、口業（妄語、兩舌、惡口、綺語）以及意業（貪欲、瞋恚、邪見）。
2. **十善業 (Kusala-kamma)**：與十不善業相對，指十種帶來幸福果報的善行，如不殺生、不偷盜、正見等。
3. **四禪 (Jhana)**：一種深度的禪定狀態，透過排除五蓋（貪欲、瞋恚、睡眠、掉舉、疑），使心進入極度的寧靜與專注。
4. **四無量心 (Brahmavihara)**：慈（願眾生得樂）、悲（願眾生離苦）、喜（隨他人的成功而歡喜）、捨（對所有眾生平等且不執著的心）。
5. **四無色定 (Arupa-jhana)**：超越物質形態的四種最高定境，包括空無邊處、識無邊處、無所有處、非想非非想處。
6. **三結 (Samyojana)**：指將眾生束縛在輪迴中的三種基本煩惱（通常指欲結、有結、慢結，或依上下文指初果需斷除的三結：身見、疑、戒禁苟請）。
7. **漏盡智 (Asavakkhaya-ñana)**：指完全消除所有精神染污（漏）的智慧，是達成阿羅漢果的最終標誌。

### [現代實作指南]
本經文揭示了一個清晰的「因果階梯」，我們可以將其轉化為現代生活的心理實踐路徑：

1. **基礎層：行為修正（止損）**
   ➔ **實作**：審視自己的日常行為是否包含「十不善業」。例如，在社交媒體上避免「惡口」與「兩舌」（挑撥離間），在工作中避免「偷盜」（包括偷時間或剽竊創意）。
   ➔ **轉念**：將「不作惡」視為一種心理上的「風險管理」，避免創造未來不必要的壓力與衝突。

2. **進階層：積極善行（積累）**
   ➔ **實作**：刻意練習「十善業」。每天設定一個小目標，例如「今日練習正語」，對他人說鼓勵而非批評的話。
   ➔ **轉念**：意識到善行不僅是為了來世的福報，更是為了在當下建立一個正向、安定且無愧的心理環境。

3. **深化層：心念淨化（升級）**
   ➔ **實作**：練習「四無量心」。當面對討厭的人時，嘗試用「悲心」思考（他之所以如此，是因為他內心在受苦），用「捨心」看待衝突（不將對方的情緒視為對我的攻擊）。
   ➔ **轉念**：從「我想要什麼」轉向「我能為他人帶來什麼」，透過降低對「我」的執著（離欲），來獲得真正的心理自由。

4. **終極層：覺知與觀察（覺悟）**
   ➔ **實作**：在日常生活中練習「有覺有觀」。當情緒升起時，不立刻反應，而是像觀察者一樣看著這個情緒的生起與滅去。
   ➔ **轉念**：理解到所有痛苦都源於對無常現象的執取，透過正見（正確的認知）來解構煩惱。

### [段落大意]
本經記述佛陀回答婆羅門長者關於生死去向的疑問。佛陀指出，實踐十不善業會導致出生在地獄；而實踐十善業則能令人生於天界或富貴人家。進一步地，若能持戒清淨並遠離愛欲，修行者可依其願力成就四禪、四無量心、四無色定，乃至斷除煩惱、獲得聖果與神通。全經強調了從道德律條（戒）到心靈定力（定），最後達到智慧解脫（慧）的漸進修行路徑。
