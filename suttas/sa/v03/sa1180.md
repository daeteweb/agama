# 雜阿含經 第 1180 經

### [原文]
如是我聞：

一時，佛住娑羅樹林婆羅門聚落。爾時，世尊晨朝著衣持鉢，入婆羅門聚落乞食，有非時雲起。

爾時，世尊作是念：「我今當往婆羅門聚落婆羅長者大會堂中。」作是念已，即往向彼大會堂所。

時，婆羅門長者悉集堂上，遙見世尊，共相謂言：「彼剃頭沙門竟知何法？」

爾時，世尊告彼婆羅聚落婆羅門長者言：「諸婆羅門有知法者，有不知者；剎利長者亦有知法者，有不知法者。」

爾時，世尊即說偈言：

「非朋欲勝朋，　　王不伏難伏， 　妻不求勝夫，　　無子不恭父。 　無會無智者，　　無智不法言， 　貪恚癡悉斷，　　是則名智者。」

時，彼婆羅門長者白佛言：「善士瞿曇！善士夫可入此堂，就座而坐。」世尊坐已。即白言：「瞿曇說法，我等樂聽。」

爾時，世尊為彼大會婆羅門長者種種說法，示、教、照、喜已，復說偈言：

「愚智群聚會，　　非說孰知明？ 　能說寂靜道，　　因說智則辯。 　說者顯正法，　　建立大仙幢， 　善說為仙幢，　　法為羅漢幢。」

爾時，世尊為婆羅聚落婆羅門長者建立正法，示、教、照、喜；示、教、照、喜已，從座起而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住娑羅樹林婆羅門聚落。**
　　在某一個時間，佛住在娑羅樹林的婆羅門聚落中。

**◈ 爾時，世尊晨朝著衣持鉢，入婆羅門聚落乞食，有非時雲起。**
　　那時，世尊在早晨穿好衣服並拿著缽，進入婆羅門聚落乞食，此時突然起了不合時節的雲朵。

**◈ 爾時，世尊作是念：「我今當往婆羅門聚落婆羅長者大會堂中。」作是念已，即往向彼大會堂所。**
　　那時，世尊心裡這麼想：「我現在應該前往婆羅門聚落中，進入婆羅門長者的大會堂。」想完之後，立即向那座大會堂走去。

**◈ 時，婆羅門長者悉集堂上，遙見世尊，共相謂言：「彼剃頭沙門竟知何法？」**
　　那時，婆羅門長者們全部聚集在堂上，遠遠看見世尊，便共同議論說：「那個剃掉頭髮的沙門（出家修行者），究竟知道什麼樣的法（真理）？」

**◈ 爾時，世尊告彼婆羅聚落婆羅門長者言：「諸婆羅門有知法者，有不知者；剎利長者亦有知法者，有不知法者。」**
　　那時，世尊告訴那群婆羅門聚落的長者說：「在婆羅門之中，有人了解法（真理），也有人不了解；同樣地，在剎利（種姓）長者之中，也有了解法的人，以及不了解法的人。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊立即說了偈頌道：

**◈ 「非朋欲勝朋，王不伏難伏，妻不求勝夫，無子不恭父。」**
　　「真正的朋友不會想要勝過朋友，強大的國王也無法征服那些難以征服的人，妻子不追求勝過丈夫，孩子不會對父親不恭敬。」

**◈ 「無會無智者，無智不法言，貪恚癡悉斷，是則名智者。」**
　　「在集會中，若沒有智慧的人，就不會發表不符合法（真理）的言論；能將貪（貪愛）、恚（瞋恨）、癡（無明）這三種煩惱悉數斷除的人，這才被稱為智者。」

**◈ 時，彼婆羅門長者白佛言：「善士瞿曇！善士夫可入此堂，就座而坐。」**
　　那時，那些婆羅門長者對佛陀恭敬地說道：「善士瞿曇！請您進入這座大堂，就座而坐。」

**◈ 世尊坐已。即白言：「瞿曇說法，我等樂聽。」**
　　世尊坐下後，他們隨即對佛陀說：「請瞿曇為我們說法，我們很樂意聆聽。」

**◈ 爾時，世尊為彼大會婆羅門長者種種說法，示、教、照、喜已，復說偈言：**
　　那時，世尊為大會中的婆羅門長者們說了各種法，在示導、教導、照亮心靈並令他們法喜充滿之後，再次說偈頌道：

**◈ 「愚智群聚會，非說孰知明？」**
　　「愚笨的人與智慧的人聚集在一起，如果不說出來，誰能知道誰是明智的？」

**◈ 「能說寂靜道，因說智則辯。」**
　　「能說明導向涅槃的寂靜道（解脫之道），因為能說明這點，所以才顯得智慧且辯才卓越。」

**◈ 「說者顯正法，建立大仙幢，善說為仙幢，法為羅漢幢。」**
　　「說法的人顯現了正法（正確的真理），就像建立起一面巨大的仙人旗幟；能善於說法的人本身就是仙人旗幟，而法（真理）則是阿羅漢（證果聖者）的旗幟。」

**◈ 爾時，世尊為婆羅聚落婆羅門長者建立正法，示、教、照、喜；示、教、照、喜已，從座起而去。**
　　那時，世尊為婆羅門聚落的長者們建立正法（正確的真理），進行示導、教導、照亮心靈並令他們法喜充滿；在完成這些後，世尊從座位上起身離開了。

### [詞語註解]
1. **沙門 (Samaṇa)**：原指捨棄家居生活、追求真理的出家修行者，不拘泥於種姓制度。
2. **貪恚癡 (Lobha, Dosa, Moha)**：佛教稱之為「三毒」。貪是指對對象的強烈渴求（貪愛）；恚是指對不滿意之物的排斥與憤怒（瞋恨）；癡是指對真理的無知與錯認（無明）。
3. **正法 (Saddhamma)**：指符合宇宙真理、能導向解脫的正確教法。
4. **寂靜道 (Santi-magga)**：指透過修行消除煩惱，最終達到心靈絕對寧靜、不再受苦的涅槃之路。
5. **阿羅漢 (Arahant)**：指已完全斷除煩惱、證得解脫且不再輪迴的聖者。

### [現代實作指南]
本經文的核心在於區分「世俗的勝負心」與「真正的智慧」。在現代生活中，我們可以透過以下方式實踐：

1. **覺察「勝負心」的運作** ➔ 在與伴侶、朋友或同事溝通時，當你感覺到想要「贏過對方」或「證明對方錯了」的衝動時，請立即標記這個念頭為「貪」或「恚」。意識到這種勝負心並不會帶來真正的和諧，反而會增加對立。
2. **重新定義「智者」** ➔ 將對智慧的定義從「博學」或「能辯」轉向「能斷除煩惱」。練習在面對衝突時，不以邏輯擊敗對方，而以「不貪、不瞋、不癡」的平靜心來回應。
3. **實踐「不法言」的自律** ➔ 在社交場合或網路討論中，練習在發言前自問：「我的話是為了顯現真理（正法），還是為了顯現我的聰明？」若後者，則選擇保持寂靜。
4. **建立心靈的「旗幟」** ➔ 學習將生活中的正向價值（如慈悲、誠實、耐心地聆聽）作為自己的行為準則（旗幟），讓自己的行為成為他人學習正法的榜樣，而非僅用言語說教。

### [段落大意]
佛陀在婆羅門聚落中，面對長者們對其身分與知識的質疑，以偈頌說明真正的智者並非追求勝負，而是能斷除貪、瞋、癡三毒之人。佛陀進一步教導，真正的智慧在於能說明導向解脫的寂靜道，並將正法比作引領眾生的旗幟，最終令在場長者法喜充滿。
