# 雜阿含經 第 1170 經

### [原文]
如是我聞：

一時，佛住拘睒彌國瞿師羅園。

爾時，世尊告諸比丘：「如癩病人，四體瘡壞，入茅荻中，為諸刺葉針刺所傷，倍增苦痛；如是愚癡凡夫六觸入處受諸苦痛亦復如是。如彼癩人，為草葉針刺所傷，膿血流出；如是愚癡凡夫，其性弊暴，六觸入處所觸則起瞋恚，惡聲流出，如彼癩人。所以者何？愚癡無聞凡夫心如癩瘡。

「我今當說律儀、不律儀。云何律儀？云何不律儀？愚癡無聞凡夫眼見色已，於可念色而起貪著，不可念色而起瞋恚，於彼次第隨生眾多覺想相續，不見過患；復見過患，不能除滅。耳、鼻、舌、身、意亦復如是。比丘！是名不律儀。云何律儀？多聞聖弟子若眼見色，於可念色不起欲想，不可念色不起恚想，次第不起眾多覺想相續住，見色過患；見過患已，能捨離。耳、鼻、舌、身、意亦復如是。是名律儀。」

佛說此經已，諸比丘聞佛所說，歡喜奉行！

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住拘睒彌國瞿師羅園。**
　　在某一個時間，佛陀住在拘睒彌國的瞿師羅園中。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「如癩病人，四體瘡壞，入茅荻中，為諸刺葉針刺所傷，倍增苦痛；**
　　「就像一個患有癩病的人，全身皮膚潰爛破損，如果進入茅荻草叢中，被那些尖銳的葉片刺傷，痛苦會倍增；

**◈ 如是愚癡凡夫六觸入處受諸苦痛亦復如是。**
　　那些愚癡的凡夫在六觸入處（眼、耳、鼻、舌、身、意與外界接觸）所感受到的種種苦痛，也是同樣的情況。

**◈ 如彼癩人，為草葉針刺所傷，膿血流出；**
　　正如那位癩病人被草葉刺傷後，會流出膿血；

**◈ 如是愚癡凡夫，其性弊暴，六觸入處所觸則起瞋恚，惡聲流出，如彼癩人。**
　　愚癡的凡夫其本性暴躁，當六觸入處（感官接觸）發生接觸時，便會立刻生起瞋恚（憤怒），並說出惡劣的話語，就像那位癩病人流膿血一樣。

**◈ 所以者何？愚癡無聞凡夫心如癩瘡。**
　　為什麼會這樣呢？因為愚癡且缺乏聞法（學習正法）的凡夫，其心靈就像患了癩瘡一樣潰爛。

**◈ 「我今當說律儀、不律儀。**
　　「我現在要為你們說明什麼是律儀（對感官的節制與防護），以及什麼是不律儀。

**◈ 云何律儀？云何不律儀？**
　　什麼是律儀？什麼是不律儀？

**◈ 愚癡無聞凡夫眼見色已，於可念色而起貪著，不可念色而起瞋恚，**
　　愚癡且缺乏聞法的凡夫，當眼睛看見色（物質身體/現象）後，對於令其心生歡喜的色（可念色）產生貪著，而對於令其不悅的色（不可念色）則產生瞋恚；

**◈ 於彼次第隨生眾多覺想相續，不見過患；**
　　接著在這種情緒中，次第生起許多覺想（認知與想像）的連續反應，卻看不出其中的過患（危險與缺陷）；

**◈ 復見過患，不能除滅。**
　　即使後來發現了過患，也無法將其除滅。

**◈ 耳、鼻、舌、身、意亦復如是。**
　　耳朵聽聲、鼻子聞香、舌頭嚐味、身體觸感、意根思惟，也同樣如此。

**◈ 比丘！是名不律儀。**
　　比丘們！這就叫做不律儀。

**◈ 云何律儀？**
　　那麼，什麼是律儀？

**◈ 多聞聖弟子若眼見色，於可念色不起欲想，不可念色不起恚想，**
　　學習廣博的聖弟子，如果眼睛看見色（物質身體/現象），對於令其心生歡喜的色不起貪欲之想，對於令其不悅的色也不起憤怒之想；

**◈ 次第不起眾多覺想相續住，見色過患；**
　　並且不會次第生起許多覺想（認知與想像）的連續反應，而是能看見色（物質現象）的過患；

**◈ 見過患已，能捨離。**
　　在看見過患之後，能夠將其捨離。

**◈ 耳、鼻、舌、身、意亦復如是。**
　　耳朵聽聲、鼻子聞香、舌頭嚐味、身體觸感、意根思惟，也同樣如此。

**◈ 是名律儀。」**
　　這就叫做律儀。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行！**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐！

### [詞語註解]
1. **六觸入處 (Saḷāyatana/Phassa)**：指六種感官（眼、耳、鼻、舌、身、意）與其對應的外部對象（色、聲、香、味、觸、法）接觸的過程。
2. **律儀 (Indriya-saṃvara)**：字面意思是「感官的防護」。指在感官接觸對象時，能保持覺知，不讓貪愛或瞋恚等煩惱隨之生起，防止心靈被外界牽著走。
3. **覺想相續 (Papañca)**：指在初步的感受之後，心靈不斷地進行妄想、推論、編織故事的過程（即「戲論」），使簡單的接觸演變成複雜的情緒反應或執著。
4. **過患 (Ādīnava)**：指事物中潛在的缺陷、危險或導致苦受的性質。在本文中指色法（物質現象）的無常與不可得，若執著其中必將導致痛苦。

### [現代實作指南]
本經將「情緒失控」比喻為「皮膚潰爛」，揭示了心靈在缺乏覺知時，對外界刺激的過度反應。我們可以將其轉化為以下心理實踐步驟：

1. **識別「刺痛感」 (覺察接觸)**：
   當你感到憤怒或強烈渴望時，意識到這就像「癩病人被草葉刺到」。告訴自己：「現在我的六觸入處（例如耳朵聽到了批評）產生了接觸，而我的心目前處於敏感（潰爛）狀態。」

2. **截斷「覺想相續」 (停止內心戲)**：
   凡夫在被刺後會開始想：「他怎麼敢這樣對我？」、「我以前對他那麼好」 → 這就是「覺想相續」。
   **實作技巧**：在情緒升起的一瞬間，使用「標記法」。心中默念：「這是貪」、「這是瞋」，將注意力從「故事內容」拉回到「情緒本身」，阻止妄想的擴散。

3. **分析「過患」 (理智轉念)**：
   問自己：「如果我繼續執著於這個憤怒/渴望，會帶來什麼後果？」
   意識到對方的言語（色/聲）是無常的，而自己的憤怒（行/心造作）才是真正造成痛苦的根源。

4. **練習「律儀」 (建立緩衝區)**：
   在「刺激」與「反應」之間創造一個空間。
   - **不律儀**：刺激 → 反應（立刻爆發或沉溺）。
   - **律儀**：刺激 → **覺知 → 觀察過患** → 選擇反應（或不反應）。

### [段落大意]
佛陀以癩病人被草葉刺傷而倍感痛苦且流膿血為喻，說明愚癡凡夫在感官接觸外界時，易因心靈潰爛而生起強烈的貪瞋反應並說惡言。佛陀進而定義「不律儀」為對感官對象產生貪瞋，並陷入妄想相續而無法捨離；而「律儀」則是聖弟子在感官接觸時能不起貪瞋，不生妄想，洞察現象的過患並能將其捨離。
