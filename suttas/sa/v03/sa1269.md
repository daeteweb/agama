# 雜阿含經 第 1269 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「誰度於諸流？　　晝夜勤精進， 　不攀亦不住，　　何染而不著？」

爾時，世尊說偈答言：

「一切戒具足，　　智慧善正受， 　內思惟繫念，　　度難度諸流。 　不樂於欲想，　　超越於色結， 　不繫亦不住，　　於染亦不著。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出色的天子，在深夜時分來到佛陀這裡，恭敬地頂禮佛陀的足部，他的身體散發出光明，照亮了整個祇樹給孤獨園。

**◈ 時，彼天子說偈問佛：**
　　那時，這位天子用偈頌的形式請問佛陀：

**◈ 「誰度於諸流？**
　　「誰能夠度過那些如洪水般洶湧的煩惱之流？

**◈ 晝夜勤精進，**
　　誰能晝夜不懈地勤奮修行（正精進），

**◈ 不攀亦不住，**
　　既不貪攀追求，也不執著停留，

**◈ 何染而不著？」**
　　面對什麼樣的染污（煩惱）才不會產生執著？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「一切戒具足，**
　　「具足一切的戒律（道德規範），

**◈ 智慧善正受，**
　　以智慧正確地接納與體驗（正見與正受），

**◈ 內思惟繫念，**
　　在內心持續地思惟並繫住正念（正念），

**◈ 度難度諸流。**
　　就能度過那些難以度過的煩惱之流。

**◈ 不樂於欲想，**
　　對於欲望的想像（欲想）不感到快樂，

**◈ 超越於色結，**
　　超越了對色（物質身體與物質世界）的執著結縛，

**◈ 不繫亦不住，**
　　不被繫縛，也不執著停留，

**◈ 於染亦不著。」**
　　面對染污（煩惱）時也不會產生執著。」

**◈ 時，彼天子復說偈言：**
　　那時，這位天子再次說偈頌道：

**◈ 「久見婆羅門，**
　　「我曾見到許多婆羅門，

**◈ 逮得般涅槃，**
　　最終達到了超越生死、絕對寧靜的解脫境界（涅槃）。

**◈ 一切怖已過，**
　　已經度過了所有的恐懼，

**◈ 永超世恩愛。」**
　　永遠超越了世間的情愛執著。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，這位天子聽聞佛陀所說的教法，心中感到法喜並隨喜讚嘆，頂禮佛陀的足部後，立即消失不再出現。

### [詞語註解]
1. **諸流 (Ogha)**：指五種洪水（五蘊之染污），包括欲愛流、有愛流、見流、行流、無明流，比喻煩惱如洪水般將眾生淹沒並推向輪迴。
2. **色結 (Rūpa-bandhana)**：對色（物質身體或物質現象）的執著，將物質形式視為「我」或「我的」，從而產生束縛。
3. **般涅槃 (Parinibbāna)**：完全的熄滅，指徹底斷除一切煩惱與渴愛，不再受生於輪迴之中。
4. **隨喜 (Anumodana)**：對他人的善行或正法之聞法感到歡喜，是一種消除嫉妒心、增加功德的心理狀態。

### [現代實作指南]
本經文提供了一套從「行為 ➔ 認知 ➔ 心理狀態」的解脫路徑，可轉化為以下生活實踐：

1. **建立底線（一切戒具足）**：
   在生活中設定明確的道德底線。當我們不再因違背原則而感到不安（後悔或恐懼）時，心靈才能在平靜的基礎上進行更高層次的觀察。

2. **正向接納（智慧善正受）**：
   當面對壓力或不快時，練習「正受」。不對感受進行好壞的評判，而是以智慧觀察：「現在有一個不舒服的感受產生了」，而非「我很不舒服」。將「我」與「感受」分離。

3. **繫住正念（內思惟繫念）**：
   練習「錨定法」。在繁忙的日常中，隨時將注意力拉回到呼吸或當下的動作上。當心念開始飄向過去的遺憾或未來的焦慮時，溫柔地將其「繫」回當下。

4. **觀察「欲想」的運作（不樂於欲想）**：
   當心中升起「如果我擁有某物會很快樂」的想像時，意識到這僅僅是一個「想（Saññā）」的造作。觀察這個想像如何引起身體的興奮或渴求，但不要跳進這個想像中去享受它，而是觀察它如何升起又如何消失。

5. **練習「不攀不住」（不攀亦不住）**：
   ➔ **不攀**：面對成功或讚美時，不試圖將其擴大或執著於維持這種狀態。
   ➔ **不住**：面對失敗或痛苦時，不反覆咀嚼，不讓心靈停留在負面情緒中。

### [段落大意]
一位天子請問佛陀如何才能度過煩惱之流且不被染污。佛陀回答：需具足戒律、以智慧正向接納、內在繫住正念，且對欲望與物質身體不生執著，方能解脫。天子聞後讚嘆涅槃之境界，隨即消失。
