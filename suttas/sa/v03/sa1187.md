# 雜阿含經 第 1187 經

### [原文]
如是我聞：

一時，佛住迦毘羅衛國尼拘律園中。

時，有縈髻波羅豆婆遮婆羅門來詣佛所，而前問訊，相慰勞已，退坐一面。而說偈言：

「身外縈髻者，　　是但名縈髻； 　內心縈髻者，　　是結縛眾生。 　我今問瞿曇，　　如此縈髻者， 　云何作方便？　　於何斷縈髻？」

爾時，世尊說偈答言：

「眼耳及與鼻，　　舌身意入處， 　於彼名及色，　　滅盡令無餘。 　諸識永滅者，　　於彼斷縈髻。」

佛說此經已，縈髻婆羅豆婆遮婆羅門聞佛所說，歡喜隨喜，從座起去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住迦毘羅衛國尼拘律園中。**
　　在某一個時間，佛陀住在迦毘羅衛國的尼拘律園中。

**◈ 時，有縈髻波羅豆婆遮婆羅門來詣佛所，而前問訊，相慰勞已，退坐一面。**
　　那時，有一位名叫「縈髻」的波羅豆婆遮婆羅門來到佛陀這裡，在佛陀面前問候並互相慰問之後，退到一邊坐下。

**◈ 而說偈言：「身外縈髻者，是但名縈髻；內心縈髻者，是結縛眾生。」**
　　然後他用偈頌說道：「身體外在的盤髮（縈髻），僅僅是名稱上的盤髮；而內心深處的糾結（縈髻），才是真正將眾生束縛在輪迴中的根源。」

**◈ 「我今問瞿曇，如此縈髻者，云何作方便？於何斷縈髻？」**
　　「我現在請問瞿曇（佛陀），面對這種內心的糾結，應該採取什麼樣的修行方法（方便）？要在什麼地方才能切斷這些糾結？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「眼耳及與鼻，舌身意入處，」**
　　「眼、耳、鼻、舌、身、意這六個感官的入處（六入），」

**◈ 「於彼名及色，滅盡令無餘。」**
　　「對於這些感官所接觸到的名（心理現象）以及色（物質身體），要將其徹底滅盡，使其不再殘留。」

**◈ 「諸識永滅者，於彼斷縈髻。」**
　　「當所有的識（意識）永遠滅盡時，才能在那個地方切斷內心的糾結。」

**◈ 佛說此經已，縈髻婆羅豆婆遮婆羅門聞佛所說，歡喜隨喜，從座起去。**
　　佛陀講完這部經後，縈髻婆羅豆婆遮婆羅門聽聞佛陀所說的教法，心生法喜並依照教導去實踐，隨後從座位上起身離開。

### [詞語註解（解釋重要佛教概念）]
1. **入處 (Ayatana)**：指感官與外界接觸的通道。分為內入處（眼、耳、鼻、舌、身、意）與外入處（色、聲、香、味、觸、法）。
2. **名色 (Nāma-rūpa)**：佛教對生命構成的總稱。「色」指物質身體，「名」指心理現象（如受、想、行、識）。
3. **識 (Viññāṇa)**：意識，指對對象的覺知與分辨能力。在十二因緣中，識是導致輪迴的核心環節之一。
4. **方便 (Upāya)**：指為了達到解脫目的而採取的適當方法或修行手段。

### [現代實作指南]
本經以「盤髮（縈髻）」比喻內心的執著與糾結。現代人常感到精神內耗或情緒打結，正是因為內心的「縈髻」在作祟。可採取以下實作步驟：

1. **覺察感官入口 (Input Awareness)**：
   當你感到焦慮或憤怒時，立刻觀察你的「入處」。問自己：我是透過什麼感官接收到刺激的？（例如：眼睛看到了不順眼的訊息 ➔ 眼入處；耳朵聽到了批評 ➔ 耳入處）。

2. **拆解名色反應 (Deconstructing Reaction)**：
   意識到刺激進入後，觀察隨之而來的「名色」反應。
   ➔ **色（物質反應）**：心跳加快、肌肉緊繃。
   ➔ **名（心理反應）**：產生「他在針對我」的認知（想）以及「不舒服」的感受（受）。

3. **切斷意識的連鎖反應 (Breaking the Chain)**：
   佛陀提到「諸識永滅」，在現代實踐中可轉化為「停止對該意識流的認同」。當你意識到「這只是一個意識的運作過程」而非「我」在受苦時，你就從那個糾結的點（縈髻）抽離了。

4. **轉念技巧**：
   將「我好煩」轉化為「現在有一個『煩』的意識在運作」。透過這種客觀化觀察，將內心的糾結（縈髻）由「我的問題」轉變為「觀察的對象」，從而達到心靈的鬆綁。

### 段落大意 
一名婆羅門以盤髮比喻內心的糾結，詢問佛陀如何解脫。佛陀教導應從六入處（感官）入手，滅盡對名色（身心現象）的執著與意識的運作，如此方能徹底切斷內心的束縛。
