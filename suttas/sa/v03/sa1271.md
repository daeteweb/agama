# 雜阿含經 第 1271 經

### [原文]
如是我聞：

一時，佛住王舍城山谷精舍。

爾時，尊者阿難告諸比丘：「我今當說四句法經。諦聽，善思，當為汝說何等為四句法經。」

爾時，尊者阿難即說偈言：

「其心不為惡，　　及身口世間， 　五欲悉虛空，　　正智正繫念。 　不習近眾苦，　　非義和合者。

「諸比丘！是名四句法經。」

爾時，有一異婆羅門，去尊者阿難不遠，為諸年少婆羅門受誦經。時，彼婆羅門作是念：「若沙門阿難所說偈，於我所說經，便是非人所說。」

時，彼婆羅門即往詣佛所，與世尊面相問訊慰勞已，退坐一面，白佛言：「瞿曇！沙門阿難所說偈言：

「其心不為惡，　　及身口世間， 　五欲悉虛空，　　正智正繫念。 　不習近眾苦，　　非義和合者。

「如是等所說，則是非人語，非為人語。」

佛告婆羅門：「如是，如是，婆羅門！是非人語，非為人語也。時，有拘迦尼天女來詣我所，稽首我足，退坐一面而說偈言：

「其心不為惡，　　及身口世間， 　五欲悉虛空，　　正智正繫念。 　不習近眾苦，　　非義和合者。」

「我時答言：『如是，如是，如天女所言：

「『「其心不為惡，　　及身口世間，　　　五欲悉虛空，　　正智正繫念。 　　　不習近眾苦，　　非義和合者。」』

「是故，婆羅門！當知此所說偈，是非人所說，非是人所說也。」

佛說此經已，彼婆羅門聞佛所說，歡喜隨喜，禮佛足而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城山谷精舍。**
　　在某一個時間，佛陀住在王舍城的山谷精舍中。

**◈ 爾時，尊者阿難告諸比丘：「我今當說四句法經。諦聽，善思，當為汝說何等為四句法經。」**
　　那時，尊者阿難告訴在場的眾位比丘說：「我現在要為你們宣說一部由四句偈構成的法經。請你們仔細聽，好好思考，我現在為你們解說什麼才是真正的四句法經。」

**◈ 爾時，尊者阿難即說偈言：**
　　那時，尊者阿難隨即說出偈頌：

**◈ 「其心不為惡，及身口世間，五欲悉虛空，正智正繫念。不習近眾苦，非義和合者。」**
　　「內心不生起惡念，且在世間的身體行為與言語表達上亦不造惡；對於五欲（對財富、色欲、飲食、睡眠、名聲的貪愛）都能視為虛空般不執著；具備正智（正確的智慧/正見）並保持正繫念（正確的覺知與專注）。不親近會導致各種苦（Dukkha）的環境與對象，也不與那些不合正義、不合道義的人結交。」

**◈ 「諸比丘！是名四句法經。」**
　　「各位比丘！這就是所謂的四句法經。」

**◈ 爾時，有一異婆羅門，去尊者阿難不遠，為諸年少婆羅門受誦經。**
　　那時，有一位特殊的婆羅門，就在尊者阿難不遠處，正指導年輕的婆羅門們誦讀經文。

**◈ 時，彼婆羅門作是念：「若沙門阿難所說偈，於我所說經，便是非人所說。」**
　　當時，那位婆羅門心裡想：「如果沙門阿難所說的偈頌與我所教的經文相比，那阿難所說的內容簡直不像是由人類所能說出的（意指其境界超越常人或不符合常理）。」

**◈ 時，彼婆羅門即往詣佛所，與世尊面相問訊慰勞已，退坐一面，白佛言：「瞿曇！沙門阿難所說偈言：**
　　於是，那位婆羅門立刻前往佛陀所在之處，在與世尊面對面問候慰勞之後，退到一邊坐下，對佛陀恭敬地說道：「瞿曇！沙門阿難所說的偈頌是：

**◈ 「其心不為惡，及身口世間，五欲悉虛空，正智正繫念。不習近眾苦，非義和合者。」**
　　『內心不生惡念，身口行為不造惡，五欲視為虛空，具備正智與正念，不親近苦因，不與非義之人和合。』

**◈ 「如是等所說，則是非人語，非為人語。」**
　　「像這樣的話語，根本不是人類能說出的話，不是人的語言。」

**◈ 佛告婆羅門：「如是，如是，婆羅門！是非人語，非為人語也。**
　　佛陀告訴那位婆羅門：「確實如此，就是這樣，婆羅門！這確實是非人語，不是人類的語言。

**◈ 時，有拘迦尼天女來詣我所，稽首我足，退坐一面而說偈言：**
　　當時，有一位名叫拘迦尼的天女來到我這裡，向我的足下頂禮，然後退到一邊坐下，說出這樣的偈頌：

**◈ 「其心不為惡，及身口世間，五欲悉虛空，正智正繫念。不習近眾苦，非義和合者。」**
　　『內心不生惡念，身口行為不造惡，五欲視為虛空，具備正智與正念，不親近苦因，不與非義之人和合。』

**◈ 「我時答言：『如是，如是，如天女所言：**
　　「我當時回答說：『確實如此，就是這樣，正如天女所言：

**◈ 『「其心不為惡，及身口世間，五欲悉虛空，正智正繫念。不習近眾苦，非義和合者。」』**
　　『內心不生惡念，身口行為不造惡，五欲視為虛空，具備正智與正念，不親近苦因，不與非義之人和合。』」

**◈ 「是故，婆羅門！當知此所說偈，是非人所說，非是人所說也。」**
　　「因此，婆羅門！應當了解這首偈頌確實是非人（天人）所說，而不是人類所說的。」

**◈ 佛說此經已，彼婆羅門聞佛所說，歡喜隨喜，禮佛足而去。**
　　佛陀講完這部經後，那位婆羅羅門聽聞佛陀所說的教法，心生法喜並隨喜讚嘆，向佛陀的足下行禮後才離開。

### [詞語註解]
1. **五欲 (Pañca-kāma)**：指對五種感官對象的貪愛，包括財富（財欲）、色情（色欲）、飲食（食欲）、睡眠（眠欲）以及名聲（名欲）。在經文中，將其視為「虛空」意味著不再對其產生執著。
2. **正智 (Sammā-ñāṇa)**：正確的知識與智慧，對事物本質（如無常、苦、無我）的正確認知，即八正道中的「正見」。
3. **正繫念 (Sammā-sati)**：此處指正確的覺知與專注力，能將心繫於正法而不散亂，對應八正道中的「正念」。
4. **非義和合 (A-dhamma-samāgama)**：指基於不正當的目的（如利益、拍馬屁、共同作惡）而與他人建立的關係，而非基於正法與真理的共修。
5. **非人語 (A-manussa-bhāsita)**：在原始佛教文獻中，「非人」通常指天人（Deva）或其他非人類的覺悟存在。此處是指該教法的境界超越了普通凡夫的認知水準。

### [現代實作指南]
本經提供了一套從「內心 ➔ 行為 ➔ 欲望 ➔ 認知 ➔ 環境」的全方位自我管理模型，可轉化為以下實踐步驟：

1. **意圖監控（其心不為惡）**：
   - **實作**：在採取任何行動前，先停頓三秒，詢問自己：「我現在的動機是否包含傷害他人或自私的企圖？」若有，立即將其標記為「惡念」並放下。

2. **行為對齊（及身口世間）**：
   - **實作**：確保你的身體行動（業）與言語（語）與你的正向意圖一致。例如，若心中想幫助人，但言語卻在諷刺，這就是「身口」與「心」的脫節。練習「言行一致」的誠實生活。

3. **欲望去中心化（五欲悉虛空）**：
   - **實作**：當強烈的物慾或名譽欲升起時，不要壓抑，而是觀察它。告訴自己：「這個欲望像雲一樣，雖然存在，但本質是空虛且暫時的。」將關注點從「擁有」轉移到「覺知」。

4. **建立正向錨點（正智正繫念）**：
   - **實作**：每天設定一個「正念錨點」（如呼吸或行走）。每當意識到自己陷入焦慮或憤怒時，立即將意識「繫」回呼吸上，用「這一切都是無常的」這一正智來對沖情緒。

5. **社交環境篩選（不習近眾苦，非義和合）**：
   - **實作**：審視社交圈。減少與那些不斷抱怨、散播負能量或鼓勵不道德行為的人共處（非義和合）。主動尋找能激發你成長、討論真理與正向價值觀的社群。

本經記載尊者阿難宣說一部關於內心純淨、遠離五欲、具備正智正念且謹慎選擇社交環境的四句偈頌。一名婆羅門認為此教法境界太高，非人類之語，遂向佛陀求證。佛陀確認此偈頌原是由天女拘迦尼所說，證明其超凡的境界，婆羅門聞後深信隨喜。
