# 雜阿含經 第 1019 經

### [原文]
如是我聞：

一時，佛在舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「何名為非道？　　云何日夜遷？ 　云何垢梵行？　　云何累世間？」

爾時，世尊說偈答言：

「貪欲名非道，　　壽命日夜遷， 　女人梵行垢，　　女則累世間。 　熾然修梵行，　　已洗諸非小。」

時彼天子復說偈言： 「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛在舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出眾的天子，在深夜時分來到佛陀面前，頂禮佛陀的足下，他身體散發出的光芒遍照了整個祇樹給孤獨園。

**◈ 時，彼天子說偈問佛：**
　　當時，那位天子用偈頌的形式請問佛陀：

**◈ 「何名為非道？　　云何日夜遷？ 　云何垢梵行？　　云何累世間？」**
　　「什麼被稱為非道（不通往解脫的錯誤道路）？生命是如何在日夜之間流逝遷轉的？什麼會玷污梵行（清淨的修行生活）？什麼會讓人被牽絆在世間的輪迴之中？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「貪欲名非道，　　壽命日夜遷， 　女人梵行垢，　　女則累世間。」**
　　「貪欲（對感官與存在的渴求）被稱為非道。壽命在日夜之間不斷流逝。對女性的執著會玷污梵行（清淨的修行生活），而女性（所代表的愛欲牽絆）則會讓人累繫於世間。」

**◈ 「熾然修梵行，　　已洗諸非小。」**
　　「若能熾熱地精進修行梵行（清淨的修行生活），就能洗淨所有微小的染污。」

**◈ 時彼天子復說偈言：**
　　當時，那位天子再次說偈頌道：

**◈ 「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」**
　　「我曾久久觀察那些婆羅門，直到他們達到了涅槃（超越生死、絕對寧靜的解脫境界），所有的恐懼都已過去，永遠超越了世間的愛欲執著。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，那位天子聽聞佛陀所說的教法，心中感到法喜並隨喜，頂禮佛陀的足下後，隨即消失不見。

### [詞語註解]
1. **非道 (Wrong Path)**：指不符合八正道、不能導向解脫的行為或心態，尤其是由貪、嗔、癡驅動的生命方向。
2. **梵行 (Brahmacarya)**：原意為「梵天之行」，在原始佛教中指清淨的修行生活，包含禁欲、持戒以及遠離感官欲望的心理狀態。
3. **涅槃 (Nibbāna)**：超越生死、絕對寧靜的解脫境界，指貪、嗔、癡三毒完全熄滅後的狀態。
4. **世恩愛 (Worldly Attachment)**：指對世間親情、情愛及物質對象的執著與依戀，這是導致輪迴（Saṃsāra）的主要原因。

### [現代實作指南]
本經文核心在於辨識「牽絆」與「清淨」的區別。在現代生活中，我們可以將其轉化為以下實踐：

1. **辨識「非道」的訊號** ➔ 當我們在追求某個目標時，觀察內心是否充滿了「貪欲」（即：如果不得到它，我就無法快樂）。如果快樂建立在「佔有」之上，這便是「非道」的心理模式。
2. **覺察「時間的遷轉」** ➔ 練習在日常中意識到「壽命日夜遷」。每當陷入情緒糾結時，提醒自己時間正在流逝，將注意力從「對過去的執著」或「對未來的焦慮」轉移到當下的覺知。
3. **定義現代的「梵行」** ➔ 梵行不一定是指出家禁欲，而是一種「心理上的清淨」。實作方法是：在面對誘惑或強烈的情緒牽絆時，練習「暫停 ➔ 觀察 ➔ 放下」。不讓感官的衝動直接主導行為，即是在生活中修習梵行。
4. **超越「恩愛」的執著** ➔ 學習「無條件的愛」而非「佔有式的愛」。將對他人的依戀（執著於對方必須如何對我）轉化為慈悲心（希望對方獲得快樂），從而減輕被世間牽絆的沉重感。

### [段落大意]
一位天子請問佛陀關於錯誤道路、生命流逝以及修行障礙的問題。佛陀回答指出，貪欲是導致錯誤方向的主因，而對愛欲的執著會玷污清淨修行並使人受困於世間；唯有精進修行清淨行才能洗滌染污。天子隨後讚嘆涅槃境界能超越恐懼與愛欲，最後在法喜中消失。
