# 雜阿含經 第 1009 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「誰持世間去，　　誰拘牽世間？ 　何等為一法，　　制御於世間？」

爾時，世尊說偈答言：

「心持世間去，　　心拘引世間， 　其心為一法，　　能制御世間。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。**
　　那時，有一位相貌極其出眾的天子，在深夜時分來到佛陀面前，恭敬地向佛陀的足下頂禮，然後退到一邊坐下。

**◈ 身諸光明遍照祇樹給孤獨園。**
　　他身體散發出的各種光明，將整個祇樹給孤獨園照得通亮。

**◈ 時，彼天子說偈問佛：**
　　那時，這位天子用詩偈的形式請問佛陀：

**◈ 「誰持世間去，誰拘牽世間？何等為一法，制御於世間？」**
　　「究竟是誰帶著世間的眾生遠離真理？又是誰將世間的眾生拘禁並牽引在輪迴之中？究竟是什麼樣的一種法（現象或法則），能夠控制整個世間？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用詩偈的形式回答說：

**◈ 「心持世間去，心拘引世間，其心為一法，能制御世間。」**
　　「正是『心（意識的主導力量）』帶著世間遠離，也是『心』拘禁並牽引著世間；這『心』就是那唯一的一法，能夠控制整個世間。」

**◈ 時，彼天子復說偈言：**
　　那時，這位天子再次說偈道：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「我曾見到許多長久的修行者（婆羅門），最終達到了涅槃（超越生死、絕對寧靜的解脫境界），他們已經超越了所有恐懼，永遠超脫了世間的愛欲與執著。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，這位天子聽聞佛陀的教導，內心感到法喜並隨喜讚嘆，再次向佛陀的足下頂禮，隨即消失不見。

### [詞語註解（解釋重要佛教概念）]
1. **心 (Citta)**：在此指主導意識的心理機能。它不僅是思考的工具，更是產生貪、嗔、痴等煩惱的根源，透過「愛（貪愛）」與「取（執取）」將眾生牽引至輪迴的業報之中。
2. **世間 (Loka)**：在原始佛教中，世間不僅指物理空間，更指由「色（物質身體）」、「受（感受）」、「想（想像與認知）」、「行（心的造作）」、「識（意識）」這五蘊構成的經驗世界，以及受苦的輪迴狀態。
3. **涅槃 (Nibbāna)**：超越生死、絕對寧靜的解脫境界。指熄滅了貪、嗔、痴三毒，從而終結了因緣而生的苦輪迴。
4. **恩愛 (Pema/Snehā)**：指對世間對象的情感依戀與執著。這種依戀被視為拘禁眾生在輪迴中的主要心理紐帶。

### [現代實作指南]
本經文揭示了「心」是所有生命經驗的操縱者。在現代生活中，我們可以透過以下步驟實踐：

1. **覺察「牽引力」**：
   當你感到強烈的渴望（如：極度想要某件物品）或強烈的厭惡（如：無法忍受某人的行為）時，立即在心中標記：「這是『心』在牽引我」。意識到這種衝動並非「我」本身，而是一種心理機能的運作。

2. **觀察心的「控制模式」**：
   記錄自己在一天中被哪些情緒「控制」了。例如：因為一句批評而憤怒一整天 ➔ 意識到是「心」在持世間去（將意識帶向負面方向）。

3. **練習「不隨之轉」**：
   當心產生「拘牽」的傾向（例如：對過去的執著或對未來的焦慮）時，練習將注意力重新拉回到當下的呼吸。透過正念（Sati），將控制權從「自動反應的心」移交給「覺知的心」。

4. **轉念技巧**：
   將「我好痛苦」轉化為「心現在產生了痛苦的感受」。這種語言上的微小改變，能創造出一個觀察者的空間，減少被心控制的程度。

一位天子請問佛陀，究竟是什麼力量在控制世間並使眾生輪迴。佛陀回答指出，正是「心」主導了世間的遠離與拘禁。天子隨即感嘆修行者若能證得涅槃，即可超越恐懼與愛欲的束縛。
