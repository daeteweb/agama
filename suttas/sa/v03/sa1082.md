# 雜阿含經 第 1082 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊晨朝著衣持鉢，入舍衛城乞食。食畢，還精舍，洗足已，入安陀林坐禪。

時，有異比丘亦復晨朝著衣持鉢，入舍衛城乞食。食畢，還精舍，洗足已，入安陀林，坐一樹下，入晝正受。是比丘入晝正受時，有惡不善覺起，依貪嗜心。

時，有天神，依安陀林住止者，作是念：「此比丘不善不類，於安陀林坐禪而起不善覺，心依惡貪，我當往呵責。」作是念已，往語比丘言：「比丘！比丘！作瘡疣耶？」

比丘答言：「當治令愈！」

天神語比丘：「瘡如鐵鑊，云何可復？」

比丘答言：「正念正智，足能令復。」

天神白言：「善哉！善哉！此是真賢治瘡，如是治瘡，究竟能愈，無有發時。」

爾時，世尊晡時從禪覺，還祇樹給孤獨園。入僧中，於大眾前敷座而坐，告諸比丘：「我今晨朝著衣持鉢，入舍衛城乞食。乞食還，至安陀林坐禪，入晝正受。有一比丘亦乞食還至安陀林，坐一樹下，入晝正受，而彼比丘起不善覺，心依惡貪。有天神依安陀林住，語比丘言：『比丘！比丘！作瘡疣耶？』……」如上廣說，乃至「『如是，比丘！善哉！善哉！此治眾賢。』」爾時，世尊即說偈言：

「士夫作瘡疣，　　自生於苦患， 
　　願求世間欲，　　心依於惡貪。 
　　以生瘡疣故，　　蛆蠅競來集， 
　　愛欲為瘡疣，　　蛆蠅諸惡覺。 
　　及諸貪嗜心，　　皆悉從意生， 
　　鑽鑿士夫心，　　以求華名利。 
　　欲火轉熾然，　　妄想不善覺， 
　　身心日夜羸，　　遠離寂靜道。 
　　若內心寂靜，　　決定智明了， 
　　無有斯瘡疣，　　見佛安隱路。 
　　正士所遊跡，　　賢聖善宣說， 
　　明智所知道，　　不復受諸有。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊晨朝著衣持鉢，入舍衛城乞食。食畢，還精舍，洗足已，入安陀林坐禪。**
　　那時，世尊在早晨穿好袈裟、拿著缽，進入舍衛城乞食。在進食完畢後，回到精舍，洗淨雙足，隨後進入安陀林中坐禪。

**◈ 時，有異比丘亦復晨朝著衣持鉢，入舍衛城乞食。食畢，還精舍，洗足已，入安陀林，坐一樹下，入晝正受。**
　　當時，有另一位比丘同樣在早晨穿好袈裟、拿著缽，進入舍衛城乞食。進食完畢後，回到精舍，洗淨雙足，進入安陀林，坐在的一棵樹下，進入白天的禪定狀態（晝正受）。

**◈ 是比丘入晝正受時，有惡不善覺起，依貪嗜心。**
　　這位比丘在進入白天禪定時，心中生起了惡劣且不善的覺知（負面思考），而這些思考是依止於貪愛與渴求之心的。

**◈ 時，有天神，依安陀林住止者，作是念：「此比丘不善不類，於安陀林坐禪而起不善覺，心依惡貪，我當往呵責。」**
　　那時，有一位住在安陀林的天神心想：「這位比丘的行為不端且不合規範，竟然在安陀林坐禪時生起不善的覺知，心依止於惡劣的貪欲，我應該過去呵責他。」

**◈ 作是念已，往語比丘言：「比丘！比丘！作瘡疣耶？」**
　　天神如此想過後，走過去對比丘說：「比丘！比丘！你是在製造瘡疣（比喻內心的貪欲之傷）嗎？」

**◈ 比丘答言：「當治令愈！」**
　　比丘回答說：「我會治療它，使其痊癒！」

**◈ 天神語比丘：「瘡如鐵鑊，云何可復？」**
　　天神對比丘說：「這傷口大得像個鐵鍋一樣，怎麼可能恢復得好？」

**◈ 比丘答言：「正念正智，足能令復。」**
　　比丘回答說：「透過正念（對當下覺知的清明）與正智（正確的洞察智慧），足以讓這傷口癒合。」

**◈ 天神白言：「善哉！善哉！此是真賢治瘡，如是治瘡，究竟能愈，無有發時。」**
　　天神對他說：「好極了！好極了！這才是真正賢明地治療傷口，如此治療，最終能徹底痊癒，再也不會復發。」

**◈ 爾時，世尊晡時從禪覺，還祇樹給孤獨園。入僧中，於大眾前敷座而坐，告諸比丘：**
　　那時，世尊在傍晚從禪定中覺醒，回到祇樹給孤獨園。進入僧團之中，在大眾面前鋪好座墊坐下，告訴在場的眾位比丘說：

**◈ 「我今晨朝著衣持鉢，入舍衛城乞食。乞食還，至安陀林坐禪，入晝正受。有一比丘亦乞食還至安陀林，坐一樹下，入晝正受，而彼比丘起不善覺，心依惡貪。有天神依安陀林住，語比丘言：『比丘！比丘！作瘡疣耶？』……」**
　　「我今天早晨穿好袈裟、拿著缽，進入舍衛城乞食。乞食回來後，到安陀林坐禪，進入白天禪定。當時有一位比丘也乞食回來來到安陀林，坐在樹下進入白天禪定，但那位比丘生起了不善的覺知，心依止於惡劣的貪欲。有一位住在安陀林的天神對比丘說：『比丘！比丘！你是在製造瘡疣嗎？』……」

**◈ 如上廣說，乃至「『如是，比丘！善哉！善哉！此治眾賢。』」**
　　佛陀如此詳細地敘述，直到天神說：「確實如此，比丘！好極了！好極了！這是賢者治療傷口的方法。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即說了偈頌：

**◈ 「士夫作瘡疣，自生於苦患，願求世間欲，心依於惡貪。」**
　　世人製造了瘡疣（比喻貪欲），讓自己陷入苦患之中；這是因為渴望世間的欲望，心依止於惡劣的貪愛。

**◈ 「以生瘡疣故，蛆蠅競來集，愛欲為瘡疣，蛆蠅諸惡覺。」**
　　因為生了瘡疣，蛆蟲與蒼蠅便爭相聚集；愛欲就像是瘡疣，而各種不善的覺知（負面思考）就像是聚集其上的蛆蠅。

**◈ 「及諸貪嗜心，皆悉從意生，鑽鑿士夫心，以求華名利。」**
　　以及各種貪婪渴求的心，全部由意根（心的造作）而生，它們像鑽頭一樣挖掘世人的心，只為了追求虛華的名利。

**◈ 「欲火轉熾然，妄想不善覺，身心日夜羸，遠離寂靜道。」**
　　欲望之火轉而熾熱燃燒，產生妄想與不善的覺知，使得身心日夜羸弱，遠離了寂靜的解脫之道。

**◈ 「若內心寂靜，決定智明了，無有斯瘡疣，見佛安隱路。」**
　　如果內心能保持寂靜，且擁有決定性的智慧明了，就不會產生這種瘡疣（貪欲），能看見佛陀所指引的安穩道路。

**◈ 「正士所遊跡，賢聖善宣說，明智所知道，不復受諸有。」**
　　這是正道修行者所行走的足跡，由賢聖者善於宣說，是明智者所知道的真理，如此便不再承受輪迴中各種存在狀態（諸有）的苦。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **晝正受 (Daytime Absorption)**：指在白天進行的深度禪定狀態，心意識高度集中於單一對象，暫時脫離對外界感官的依賴。
2. **不善覺 (Akusala Vitakka)**：指不善的思考或覺知。在原始佛教中，「覺」指心的初步思考方向，若方向導向貪、嗔、痴，則稱為不善覺。
3. **正念 (Samma-sati)**：對當下身心狀態保持清明、不評判的覺知，是八正道的核心，用於及時察覺不善覺的升起。
4. **正智 (Samma-ñāṇa)**：正確的洞察智慧，能看穿現象的無常、苦、無我本質，從而根除貪欲的根源。
5. **諸有 (Bhava)**：指輪迴（Saṃsāra）中各種不同的存在狀態（如欲界、色界、無色界），受業力驅使而持續生滅。

### [現代實作指南]
本經以「瘡疣」比喻「貪欲」，將心理狀態具象化，提供以下實作步驟：

1. **識別「心理瘡疣」**：
   ➔ 當心中升起強烈的渴望（如：對名利的執著、對感官享樂的渴求）時，不要將其視為「我的願望」，而要將其標記為「一個瘡疣」。
   ➔ 意識到：貪欲本身並非快感，而是一種心靈的「傷口」，會吸引更多負面情緒（蛆蠅）聚集。

2. **觀察「蛆蠅」的聚集**：
   ➔ 當貪欲（瘡疣）升起後，隨之而來的焦慮、比較、妄想、不安，即是「蛆蠅」。
   ➔ 練習觀察：當我想要某物時，隨後產生了哪些不善的思考？（例如：如果我沒有它，我就比別人差 ➔ 這就是一隻蛆蠅）。

3. **運用「正念」與「正智」治療**：
   ➔ **正念實作**：在不善覺升起的瞬間，立即覺察「現在有貪欲升起了」，不壓抑也不隨之起舞，僅僅是觀察它。
   ➔ **正智轉念**：思考該欲望的本質 ➔ 「這個欲望能帶來永恆的滿足嗎？」➔ 「它是否會導致更多的煩惱？」➔ 透過洞察其「無常」與「不滿足」的本質，讓貪欲自然消退。

4. **建立「寂靜道」的習慣**：
   ➔ 每天撥出短時間，練習將心從「追求」轉向「覺知」。
   ➔ 當心不再向外尋求填補（不製造瘡疣），內在的安穩（寂靜）會自然顯現。

### [段落大意]
本經透過一位比丘在禪定中起貪欲，被天神以「製造瘡疣」比喻呵責的故事，說明貪欲如同傷口，會吸引不善的思考（如蛆蠅）聚集，導致身心羸弱並遠離解脫。佛陀強調，唯有透過正念與正智的治療，才能根除貪欲之傷，回歸內心寂靜，最終終結輪迴之苦。
