# 雜阿含經 第 1108 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊晨朝著衣持鉢，入舍衛城乞食。乞食已，還精舍，舉衣鉢，洗足已，持尼師壇，著右肩上，至安陀林，布尼師壇，坐一樹下，入晝正受。

爾時，祇桓中有兩比丘諍起，一人罵詈，一人默然。其罵詈者，即便改悔懺謝於彼，而彼比丘不受其懺，以不受懺故。時，精舍中眾多比丘共相勸諫，高聲閙亂。

爾時，世尊以淨天耳過於人耳，聞祇桓中高聲閙亂。聞已，從禪覺，還精舍，於大眾前敷座而坐，告諸比丘：「我今晨朝乞食，還至安陀林，坐禪人晝正受，聞精舍中高聲大聲，紛紜閙亂，竟為是誰？」

比丘白佛：「此精舍中有二比丘諍起，一比丘罵，一者默然。時，罵比丘尋向悔謝，而彼不受。緣不受故，多人勸諫，故致大聲，高聲閙亂。」

佛告比丘：「云何？比丘！愚癡之人，人向悔謝，不受其懺，若人懺而不受者，是愚癡人，長夜當得不饒益苦。諸比丘！過去世時，釋提桓因有三十三天共諍，說偈教誡言：

「『於他無害心，　　瞋亦不纏結， 
　　懷恨不經久，　　於瞋以不住。 
　　雖復瞋恚盛，　　不發於麁言， 
　　不求彼開節，　　揚人之虛短。 
　　常當自防護，　　以義內省察， 
　　不怒亦不害，　　常與賢聖俱。 
　　若與惡人俱，　　剛彊猶山石， 
　　盛恚能自持，　　如制逸馬車。 
　　我說為善御，　　非謂執繩者。』」

佛告諸比丘：「釋提桓因於三十三天為自在王，常行忍辱，亦復讚歎行忍者。汝等比丘正信非家，出家學道，當行忍辱，讚歎忍者，應當學！」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊晨朝著衣持鉢，入舍衛城乞食。**
　　那時，世尊在早晨穿好衣服，拿著缽，進入舍衛城乞食。

**◈ 乞食已，還精舍，舉衣鉢，洗足已，持尼師壇，著右肩上，至安陀林，布尼師壇，坐一樹下，入晝正受。**
　　乞食完畢後，回到精舍，放下衣鉢，洗淨雙腳，帶著尼師壇（僧侶用坐墊），披在右肩上，走到安陀林，鋪開坐墊，坐在樹下，進入正午的禪定（晝正受）。

**◈ 爾時，祇桓中有兩比丘諍起，一人罵詈，一人默然。**
　　那時，在祇桓園中有兩位比丘發生爭執，一人辱罵，另一人保持沉默。

**◈ 其罵詈者，即便改悔懺謝於彼，而彼比丘不受其懺，以不受懺故。**
　　那個辱罵的人隨即感到後悔，向對方懺悔道歉，但對方比丘不接受他的懺悔。因為不接受懺悔的緣故...

**◈ 時，精舍中眾多比丘共相勸諫，高聲閙亂。**
　　這時，精舍中許多比丘共同勸說，導致現場聲音嘈雜，陷入混亂。

**◈ 爾時，世尊以淨天耳過於人耳，聞祇桓中高聲閙亂。**
　　那時，世尊擁有超越凡人的淨天耳（天耳通），聽到了祇桓園中的喧鬧聲。

**◈ 聞已，從禪覺，還精舍，於大眾前敷座而坐，告諸比丘：**
　　聽聞後，從禪定中覺醒，回到精舍，在眾人面前鋪好座位坐下，告訴諸位比丘說：

**◈ 「我今晨朝乞食，還至安陀林，坐禪人晝正受，聞精舍中高聲大聲，紛紜閙亂，竟為是誰？」**
　　「我今天早晨乞食後，來到安陀林坐禪進入正午的禪定，卻聽到精舍中傳來大聲喧嘩、紛亂嘈雜的聲音，到底是誰在鬧事？」

**◈ 比丘白佛：「此精舍中有二比丘諍起，一比丘罵，一者默然。時，罵比丘尋向悔謝，而彼不受。緣不受故，多人勸諫，故致大聲，高聲閙亂。」**
　　比丘對佛陀恭敬地說道：「這精舍中有兩位比丘發生爭執，一位辱罵，另一位沉默。隨後辱罵的比丘向對方悔謝，但對方不接受。因為不接受的緣故，許多人上前勸諫，才導致大聲喧嘩，陷入混亂。」

**◈ 佛告比丘：「云何？比丘！愚癡之人，人向悔謝，不受其懺，若人懺而不受者，是愚癡人，長夜當得不饒益苦。**
　　佛陀告訴比丘：「怎麼回事？比丘！愚癡的人，在他人向其悔謝時，卻不接受對方的懺悔。如果有人懺悔而對方不接受，這個人就是愚癡的人，在漫長的輪迴（長夜）中，將會得到沒有利益且痛苦的業報（Vipāka）。

**◈ 諸比丘！過去世時，釋提桓因有三十三天共諍，說偈教誡言：**
　　諸位比丘！在過去世時，釋提桓因（天主）在三十三天中遇到共同爭執，便說偈頌來教誡說：

**◈ 「『於他無害心，瞋亦不纏結，懷恨不經久，於瞋以不住。**
　　『對他人抱持無害之心，即使生起瞋心也不讓它纏結心靈，懷恨的時間不要太久，不要停留在瞋心之中。

**◈ 雖復瞋恚盛，不發於麁言，不求彼開節，揚人之虛短。**
　　即使瞋怒之情強烈，也不發出粗魯的言語，不去揭露對方的弱點，不去宣揚他人的缺點。

**◈ 常當自防護，以義內省察，不怒亦不害，常與賢聖俱。**
　　應當經常自我防護（覺察），以正義與真理在內心省察，既不憤怒也不傷害，經常與賢聖之人在一起。

**◈ 若與惡人俱，剛彊猶山石，盛恚能自持，如制逸馬車。**
　　如果必須與惡人在一起，就要像山石一樣剛強堅定（不被動搖），在強烈的憤怒中能自我克制，就像控制一輛奔馳的馬車一樣。

**◈ 我說為善御，非謂執繩者。』」**
　　我所說的『善於駕馭』，並非是指那些僅僅抓著繩子（強行控制）的人。』」

**◈ 佛告諸比丘：「釋提桓因於三十三天為自在王，常行忍辱，亦復讚歎行忍者。**
　　佛陀告訴諸位比丘：「釋提桓因在三十三天中作為自在天王，經常實踐忍辱（忍耐與包容），並且讚歎實踐忍辱的人。

**◈ 汝等比丘正信非家，出家學道，當行忍辱，讚歎忍者，應當學！」**
　　你們這些比丘虔誠地信奉並離開家庭，出家學道，應當實踐忍辱，並讚歎那些忍辱的人，應當學習這種精神！」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **尼師壇 (Nisīdhana)**：原指僧侶在禪修或聽法時所坐的墊子或單衣，象徵著修行的莊嚴與安定。
2. **晝正受**：指在正午時分進入深層的禪定狀態，是一種特定的修行時間安排。
3. **長夜 (Samsāra)**：佛教的比喻，將漫長的輪迴生死視為一個黑暗且無盡的夜晚，直到覺悟（如日出）才能終結。
4. **釋提桓因 (Śakra/Indra)**：印度神話中的天王，在佛教典籍中常被描述為護法神，並在佛陀的教導下學習正法。
5. **忍辱 (Khanti)**：並非單純的忍耐或壓抑，而是一種基於對「無我」與「緣起」理解後的包容心，能讓心在面對逆境時保持不動搖。

### [現代實作指南]
本經文探討的是「憤怒的處理」與「寬恕的智慧」。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **區分「忍耐」與「忍辱」**：
   - **忍耐**：是強行壓抑憤怒（如同經文末尾提到的「執繩者」），這種方式會讓壓力積累，最終爆發。
   - **忍辱**：是以「山石」般的定力觀察憤怒。當對方辱罵時，意識到這只是對方的「行（心的造作）」在運作，而非針對真正的「我」。

2. **建立「不纏結」的心理機制**：
   - 當憤怒升起時，練習「不讓它纏結」。告訴自己：「憤怒已經升起了」，僅僅是觀察它，而不是與它認同（我 $\neq$ 憤怒）。
   - 縮短「懷恨」的時間。意識到執著於仇恨會產生負面的業報，讓自己成為受害者，而非寬恕的掌控者。

3. **面對衝突的「山石策略」**：
   - 當身處有毒的環境或面對惡意攻擊時，採取「剛強猶山石」的態度 ➔ 保持情緒的穩定，不被對方的情緒頻率牽引，不以粗魯言語回擊。
   - 實踐「不求彼開節」，即不試圖在爭吵中揭露對方的短處來獲勝，因為這只會加深對立。

4. **寬恕的邏輯轉念**：
   - 意識到「不接受懺悔」是一種愚癡。寬恕對方並非為了對方，而是為了讓自己的心脫離「長夜」的痛苦。接受對方的道歉，是給予自己解脫的機會。

### [段落大意]
本經記述兩位比丘因爭執而導致精舍混亂，其中一人雖道歉但另一人拒絕寬恕。佛陀聞訊後，以釋提桓因在三十三天的教誡為例，強調面對瞋心應保持無害、不執著且不揭短的態度。佛陀指出，拒絕寬恕是愚癡之舉，會導致長期的痛苦業報，並勉勵出家弟子應學習忍辱與讚歎忍辱的精神。
