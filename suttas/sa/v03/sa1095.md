# 雜阿含經 第 1095 經

### [原文]
如是我聞：

一時，佛住娑羅婆羅門聚落。

爾時，世尊晨朝著衣持鉢，入婆羅聚落乞食。時，魔波旬作是念：「今沙門瞿曇晨朝著衣持鉢，入婆羅聚落乞食，我今當往，先入其舍，語諸信心婆羅門長者，令沙門瞿曇空鉢而出。」

時，魔波旬隨逐佛後，作是唱言：「沙門！沙門！都不得食耶？」

爾時，世尊作是念：「惡魔波旬欲作嬈亂。」即說偈言：

「汝新於如來，　　獲得無量罪， 　汝謂呼如來，　　受諸苦惱耶？」

時，魔波旬作是言：「瞿曇！更入聚落，當令得食。」

爾時，世尊而說偈言：

「正使無所有，　　安樂而自活， 　如彼光音天，　　常以欣悅食。 　正使無所有，　　安樂而自活， 　常以欣悅食，　　不依於有身。」

時，魔波旬作是念：「沙門瞿曇已知我心。」內懷憂慼，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住娑羅婆羅門聚落。**
　　在某一個時間，佛陀住在娑羅婆羅門的村落中。

**◈ 爾時，世尊晨朝著衣持鉢，入婆羅聚落乞食。**
　　那時，世尊在早晨穿好袈裟並拿著缽，進入婆羅門的村落中乞食。

**◈ 時，魔波旬作是念：「今沙門瞿曇晨朝著衣持鉢，入婆羅聚落乞食，我今當往，先入其舍，語諸信心婆羅門長者，令沙門瞿曇空鉢而出。」**
　　那時，魔波旬心裡這樣想：「現在沙門瞿曇在早晨穿好袈裟拿著缽，進入婆羅門村落乞食，我現在應該前往，先進入那些住家，告訴那些有信心的婆羅門長者，讓沙門瞿曇最後只能拿著空缽離開。」

**◈ 時，魔波旬隨逐佛後，作是唱言：「沙門！沙門！都不得食耶？」**
　　那時，魔波旬跟在佛陀後面，大聲喊道：「沙門！沙門！你完全沒有得到食物嗎？」

**◈ 爾時，世尊作是念：「惡魔波旬欲作嬈亂。」即說偈言：**
　　那時，世尊心裡想：「惡魔波旬想要製造攪亂。」於是說了偈頌：

**◈ 「汝新於如來，獲得無量罪，汝謂呼如來，受諸苦惱耶？」**
　　「你剛才對如來（成就不變真理而來到世間的人）造下了無量的罪業（Kamma），你以為呼喚如來，會讓如來感到苦惱嗎？」

**◈ 時，魔波旬作是言：「瞿曇！更入聚落，當令得食。」**
　　那時，魔波旬說道：「瞿曇！再進入村落一次，我會讓你得到食物。」

**◈ 爾時，世尊而說偈言：**
　　那時，世尊說了偈頌：

**◈ 「正使無所有，安樂而自活，如彼光音天，常以欣悅食。正使無所有，安樂而自活，常以欣悅食，不依於有身。」**
　　「即便真正地一無所有，也能安樂地生存；就像那光音天的天人，恆常以欣悅的心情攝取養分。即便真正地一無所有，也能安樂地生存，恆常以欣悅的心情攝取養分，而不依賴於色（物質身體）的存在。」

**◈ 時，魔波旬作是念：「沙門瞿曇已知我心。」內懷憂慼，即沒不現。**
　　那時，魔波旬心裡想：「沙門瞿曇已經知道我的心意了。」內心感到憂愁苦惱，隨即消失不再出現。

### [詞語註解]
1. **如來 (Tathāgata)**：指成就不變真理而來到世間的人，強調佛陀覺悟了宇宙真理且能自在出入。
2. **業 (Kamma)**：指有意識的行為及其產生的潛在力量，會導致相應的業報（Vipāka）。
3. **色 (Rūpa)**：指物質身體或一切物質現象，是五蘊（構成身心的五種要素）中的第一項。
4. **光音天 (Ābhāsvara)**：佛教宇宙觀中的一種淨天，其居民不依賴物質食物，而是以喜悅和精神能量為食。

### [現代實作指南]
本經文描述了佛陀面對外界惡意干擾（魔波旬）時的心理狀態，可轉化為以下生活實踐：

1. **識別「內在魔波旬」**：
   當我們在追求目標或維持正向生活（如佛陀乞食）時，內心常會出現自我懷疑、焦慮或外界的嘲諷（如魔波旬的挑釁）。實作時，請將這些負面聲音標記為「攪亂」，而非將其視為事實。

2. **建立「不依賴」的滿足感**：
   佛陀提到「正使無所有，安樂而自活」，這是一種心理上的「去物質化」練習。
   ➔ **實作技巧**：練習在缺乏外部認可（如讚美、金錢、名聲）的情況下，依然能感受到內在的平靜。試著問自己：「如果我現在失去這個外部條件，我的核心價值是否依然存在？」

3. **以「欣悅」取代「匱乏」**：
   魔波旬試圖用「空缽（匱乏）」來威脅佛陀，但佛陀以「欣悅（豐盛）」回應。
   ➔ **轉念技巧**：將注意力從「我缺少什麼」轉移到「我現在擁有什麼」。即便物質簡陋，只要心念處於欣悅狀態，就能產生強大的心理韌性，使外界的干擾失去作用。

### [段落大意]
魔波旬企圖透過操縱信眾使佛陀乞食失敗，並以嘲諷之詞試圖動搖佛陀。佛陀以如來的定力化解干擾，並指出真正的安樂來自於不依賴物質身體（色）的執著，而是像光音天一樣以內在欣悅自足。魔波旬意識到佛陀已洞悉其心，最終悻悻離去。
