# 雜阿含經 第 1222 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「過去世時，天、阿修羅對陣鬪戰，阿修羅勝，諸天不如。時，天帝釋軍壞退散，極生恐怖，乘車北馳，還歸天宮。

「須彌山下道逕叢林，林下有金翅鳥巢，多有金翅鳥子。爾時，帝釋恐車馬過，踐殺鳥子，告御者言：『可回車還，勿殺鳥子。』御者白王：『阿修羅軍後來逐人，若回還者，為彼所困。』帝釋告言：『寧當回還為阿修羅殺，不以軍眾蹈殺眾生。』於道，御者轉乘南向。阿修羅軍遙見帝釋轉乘而還，謂為戰策，即還退走，眾大恐怖，壞陣流散，歸阿修羅宮。」

佛告諸比丘：「彼天帝釋於三十三天為自在王，以慈心故，威力摧伏阿修羅軍，亦常讚嘆慈心功德。汝等比丘正信非家，出家學道，當修慈心，亦應讚嘆慈心功德。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「過去世時，天、阿修羅對陣鬪戰，阿修羅勝，諸天不如。**
　　那時，世尊告訴在場的眾位比丘說：「在過去的世間，天神與阿修羅（非天之類的神靈）對陣交戰，結果阿修羅獲勝，諸天神則落敗。

**◈ 時，天帝釋軍壞陣退散，極生恐怖，乘車北馳，還歸天宮。**
　　當時，天帝釋的軍隊陣型崩潰並潰敗散開，心中極其恐懼，於是乘車向北疾馳，準備返回天宮。

**◈ 須彌山下道逕叢林，林下有金翅鳥巢，多有金翅鳥子。**
　　在須彌山下的道路旁有一片叢林，林下有金翅鳥的巢穴，裡面有許多金翅鳥的小鳥。

**◈ 爾時，帝釋恐車馬過，踐殺鳥子，告御者言：『可回車還，勿殺鳥子。』**
　　那時，帝釋擔心車馬經過時，會將鳥子踐踏殺死，於是告訴駕車的人說：『請將車轉回，不要殺死這些鳥子。』

**◈ 御者白王：『阿修羅軍後來逐人，若回還者，為彼所困。』**
　　駕車者對天王說：『阿修羅軍在後面追趕，如果現在折返，會被他們圍困。』

**◈ 帝釋告言：『寧當回還為阿修羅殺，不以軍眾蹈殺眾生。』**
　　帝釋回答說：『我寧願折返而被阿修羅殺死，也不願讓軍隊踐踏殺死眾生（所有具有意識的生命）。』

**◈ 於道，御者轉乘南向。**
　　於是，駕車者在道路上將車轉向南面折返。

**◈ 阿修羅軍遙見帝釋轉乘而還，謂為戰策，即還退走，眾大恐怖，壞陣流散，歸阿修羅宮。」**
　　阿修羅軍從遠處看到帝釋轉車折返，誤以為這是某種戰術策略，於是立即退走，眾軍陷入巨大的恐懼中，陣型崩潰流散，回到了阿修羅宮。」

**◈ 佛告諸比丘：「彼天帝釋於三十三天為自在王，以慈心故，威力摧伏阿修羅軍，亦常讚嘆慈心功德。**
　　佛陀告訴比丘們：「那位天帝釋在三十三天中作為自在王，正是因為具備慈心（對所有眾生生起無條件的關懷與愛心），其威力才能摧毀並擊敗阿修羅軍，且他經常讚嘆慈心所帶來的功德。

**◈ 汝等比丘正信非家，出家學道，當修慈心，亦應讚嘆慈心功德。」**
　　你們這些比丘既然虔誠地選擇離開家庭，出家修行佛法，就應當修行慈心（對所有眾生生起無條件的關懷與愛心），也應當讚嘆慈心的功德。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **慈心 (Mettā)**：一種對所有眾生不分彼此、希望他人獲得快樂且遠離痛苦的純淨願望，不含私欲或執著。
2. **眾生 (Satta)**：指所有具有意識、在輪迴中生存的生命體。
3. **功德 (Puñña)**：指透過正向的行為（如慈悲、布施、持戒）而積累的清淨精神能量與正向果報。
4. **出家 (Pabbajjā)**：指離開世俗家庭生活，捨棄對物質與親緣的執著，專注於修行解脫之道的行為。

### [現代實作指南]
本經文的核心在於「慈心」能轉化危機並產生強大的心理能量。在現代生活中，可採取以下實作步驟：

1. **微小生命的覺察（意識練習）**：
   ➔ 練習在行走或行動時，留意腳下或周圍的微小生命（如昆蟲、植物）。
   ➔ 當意識到自己的行動可能對他人造成傷害時，即便這對自己造成微小的不便（如繞路），也選擇採取保護生命的行動。這種「不忍心」的覺察即是慈心的萌芽。

2. **優先順序的轉念（價值重構）**：
   ➔ 當面臨壓力或恐懼（如經文中的阿修羅追擊）時，我們傾向於採取自保的自私行為。
   ➔ 試著將「不傷害他人」置於「自我利益」之上。當我們能放下對自我生存的極端恐懼，轉而關懷他人時，內心的恐懼反而會降低，從而產生冷靜應對危機的「威力」。

3. **慈心冥想（日常實踐）**：
   ➔ 每日花五分鐘，在心中默念：「願我平安快樂，願我的親友平安快樂，願我不喜歡的人平安快樂，願所有眾生平安快樂。」
   ➔ 透過這種擴展，將關懷從自我 ➔ 親近者 ➔ 中立者 ➔ 敵對者逐步擴大，打破對立的心理牆壁。

### 段落大意 
本經透過天帝釋在撤退途中，寧願冒著被阿修羅軍圍困的風險也要避開金翅鳥子的故事，闡述「慈心」的強大力量。帝釋因不忍殺生而折返，反而使敵軍誤以為是戰術而潰散。佛陀以此勉勵比丘，出家修行者應修習慈心並讚嘆其功德，以慈悲轉化對立與恐懼。
