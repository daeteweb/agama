# 雜阿含經 第 1172 經

### [原文]
如是我聞：

一時，佛住拘睒彌國瞿師羅園。

爾時，世尊告諸比丘：「譬如有四蚖蛇，凶惡毒虐，盛一篋中。時，有士夫聰明不愚，有智慧，求樂厭苦，求生厭死。時，有一士夫語向士夫言：『汝今取此篋盛毒蛇，摩拭洗浴，恩親養食，出內以時。若四毒蛇脫有惱者，或能殺汝，或令近死，汝當防護。』爾時，士夫恐怖馳走。『忽有五怨，拔刀隨逐，要求欲殺，汝當防護。』爾時，士夫畏四毒蛇及五拔刀怨，驅馳而走。

「人復語言：『士夫！內有六賊，隨逐伺汝，得便當殺，汝當防護。』爾時，士夫畏四毒蛇、五拔刀怨及內六賊，恐怖馳走，還入空村，見彼空舍，危朽腐毀，有諸惡物，捉皆危脆，無有堅固。

「人復語言：『士夫！是空聚落當有群賊，來必奄害汝。』爾時，士夫畏四毒蛇、五拔刀賊、內六惡賊、空村群賊，而復馳走。忽爾道路臨一大河，其水浚急，但見此岸有諸怖畏，而見彼岸安隱快樂，清涼無畏，無橋船可渡得至彼岸，作是思惟：『我取諸草木，縛束成栰，手足方便，渡至彼岸。』作是念已，即拾草木，依於岸傍，縛束成栰，手足方便，截流橫渡。

「如是士夫免四毒蛇、五拔刀怨、六內惡賊，復得脫於空村群賊，渡於浚流，離於此岸種種怖畏，得至彼岸安隱快樂。我說此譬，當解其義。比丘！篋者，譬此身色麁四大，四大所造精血之體，穢食長養，沐浴衣服，無常變壞危脆之法。毒蛇者，譬四大——地界、水界、火界、風界。地界若諍，能令身死，及以近死；水、火、風諍亦復如是。五拔刀怨者，譬五受陰。六內賊者，譬六愛喜。空村者，譬六內入。善男子！觀察眼入處，是無常變壞，執持眼者，亦是無常虛偽之法；耳、鼻、舌、身、意入處亦復如是。空村群賊者，譬外六入處。眼為可意、不可意色所害；耳聲、鼻香、舌味、身觸、意，為可意、不可意法所害。浚流者，譬四流——欲流、有流、見流、無明流。河者，譬三愛——欲愛、色愛、無色愛。此岸多恐怖者，譬有身。彼岸清涼安樂者，譬無餘涅槃。栰者，譬八正道。手足方便截流渡者，譬精進勇猛到彼岸。婆羅門住處者，譬如來、應、等正覺。

「如是，比丘！大師慈悲安慰弟子，為其所作；我今已作，汝今亦當作其所作，於空閑樹下，房舍清淨，敷草為座，露地、塚間，遠離邊坐，精勤禪思，慎莫放逸，令後悔恨！此則是我教授之法。」

佛說此經已，諸比丘聞佛所說，歡喜奉行！

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住拘睒彌國瞿師羅園。**
　　在某一個時間，佛陀住在拘睒彌國的瞿師羅園。

**◈ 爾時，世尊告諸比丘：「譬如有四蚖蛇，凶惡毒虐，盛一篋中。**
　　那時，世尊告訴在場的眾位比丘說：「譬如有人將四條凶惡毒辣的毒蛇，裝在一個盒子裡。

**◈ 時，有士夫聰明不愚，有智慧，求樂厭苦，求生厭死。**
　　當時，有一位聰明且不愚鈍的士夫，他擁有智慧，追求快樂而厭惡痛苦，追求生存而厭惡死亡。

**◈ 時，有一士夫語向士夫言：『汝今取此篋盛毒蛇，摩拭洗浴，恩親養食，出內以時。**
　　這時，另一位士夫對他說：『你現在接手這個裝著毒蛇的盒子，要為牠們擦拭、洗浴，親切地餵養，並按時照顧牠們出入。

**◈ 若四毒蛇脫有惱者，或能殺汝，或令近死，汝當防護。』**
　　如果這四條毒蛇在脫殼或不安時，可能會殺死你，或讓你瀕臨死亡，你必須小心防護。』

**◈ 爾時，士夫恐怖馳走。『忽有五怨，拔刀隨逐，要求欲殺，汝當防護。』**
　　這時，那位士夫感到恐懼而奔逃。接著又有人告訴他：『突然有五個仇家拔出刀隨後追趕，想要殺死你，你必須小心防護。』

**◈ 爾時，士夫畏四毒蛇及五拔刀怨，驅馳而走。**
　　於是，那位士夫既害怕四條毒蛇，又害怕五個拔刀的仇家，驚惶地奔跑逃命。

**◈ 「人復語言：『士夫！內有六賊，隨逐伺汝，得便當殺，汝當防護。』**
　　又有人對他說：『士夫！你的內心有六個小偷在伺機追隨，一旦得便就會殺死你，你必須小心防護。』

**◈ 爾時，士夫畏四毒蛇、五拔刀怨及內六賊，恐怖馳走，還入空村，見彼空舍，危朽腐毀，有諸惡物，捉皆危脆，無有堅固。**
　　這時，那位士夫害怕四條毒蛇、五個拔刀仇家以及內在的六個小偷，恐懼地奔逃，進入了一個空無一人的村落。他看到那裡的房屋危險、朽爛且毀壞，裡面充滿了惡心的東西，觸碰起來都危險脆弱，沒有任何堅固之處。

**◈ 「人復語言：『士夫！是空聚落當有群賊，來必奄害汝。』**
　　又有人對他說：『士夫！這個空聚落將會有一群強盜來，必定會突然傷害你。』

**◈ 爾時，士夫畏四毒蛇、五拔刀賊、內六惡賊、空村群賊，而復馳走。**
　　於是，那位士夫害怕四條毒蛇、五個拔刀仇家、內在的六個惡賊以及空村的強盜群，再次奔逃。

**◈ 忽爾道路臨一大河，其水浚急，但見此岸有諸怖畏，而見彼岸安隱快樂，清涼無畏，無橋船可渡得至彼岸，**
　　突然，道路來到一座大河邊，河水深且湍急。他看到此岸充滿了種種恐怖，而彼岸則是安穩快樂、清涼無畏的。但河上沒有橋樑或船隻可以渡到彼岸，

**◈ 作是思惟：『我取諸草木，縛束成栰，手足方便，渡至彼岸。』**
　　於是他思考：『我要採取一些草木，將它們綑紮成筏子，運用手腳想辦法渡到彼岸。』

**◈ 作是念已，即拾草木，依於岸傍，縛束成栰，手足方便，截流橫渡。**
　　思考完畢後，他立即拾取草木，在岸邊將其綑紮成筏子，運用手足的方便，橫渡湍急的河流。

**◈ 「如是士夫免四毒蛇、五拔刀怨、六內惡賊，復得脫於空村群賊，渡於浚流，離於此岸種種怖畏，得至彼岸安隱快樂。**
　　就這樣，這位士夫擺脫了四條毒蛇、五個拔刀仇家、六個內在惡賊，又脫離了空村的強盜群，渡過了深流，離開了此岸的種種恐怖，到達了彼岸的安穩快樂。

**◈ 我說此譬，當解其義。**
　　我說這個比喻，現在為你們解釋其中的義理。

**◈ 比丘！篋者，譬此身色麁四大，四大所造精血之體，穢食長養，沐浴衣服，無常變壞危脆之法。**
　　比丘！那個盒子，譬喻為這個色（物質身體）的粗大四大元素，以及由四大元素構成的精血之體；它是靠穢食長養，經過沐浴衣服維持，但本質上是無常、變壞且危脆的法。

**◈ 毒蛇者，譬四大——地界、水界、火界、風界。**
　　毒蛇，譬喻為四大元素 ➔ 地界、水界、火界、風界。

**◈ 地界若諍，能令身死，及以近死；水、火、風諍亦復如是。**
　　如果地界失調（諍），能令身體死亡或瀕臨死亡；水、火、風界失調也是同樣的情況。

**◈ 五拔刀怨者，譬五受陰。**
　　五個拔刀的仇家，譬喻為五受陰（五種感受）。

**◈ 六內賊者，譬六愛喜。**
　　六個內在小偷，譬喻為六種愛喜（貪愛）。

**◈ 空村者，譬六內入。**
　　空村，譬喻為六內入（六種內在感官）。

**◈ 善男子！觀察眼入處，是無常變壞，執持眼者，亦是無常虛偽之法；耳、鼻、舌、身、意入處亦復如是。**
　　善男子！觀察眼入處（視覺感官），它是無常且變壞的；執持眼睛的人（主體），也是無常且虛偽的法；耳、鼻、舌、身、意入處也是同樣的情況。

**◈ 空村群賊者，譬外六入處。**
　　空村的強盜群，譬喻為外六入處（六種外部感官對象）。

**◈ 眼為可意、不可意色所害；耳聲、鼻香、舌味、身觸、意，為可意、不可意法所害。**
　　眼睛會被令人滿意或不滿意的色（物質形態）所傷害；耳朵、鼻子、舌頭、身體、意識，也會被令人滿意或不滿意的法（現象）所傷害。

**◈ 浚流者，譬四流——欲流、有流、見流、無明流。**
　　深流，譬喻為四流 ➔ 欲流、有流、見流、無明流。

**◈ 河者，譬三愛——欲愛、色愛、無色愛。**
　　河流，譬喻為三愛 ➔ 欲愛、色愛、無色愛。

**◈ 此岸多恐怖者，譬有身。**
　　此岸充滿恐怖，譬喻為擁有身體（處於輪迴之中）。

**◈ 彼岸清涼安樂者，譬無餘涅槃。**
　　彼岸清涼安樂，譬喻為無餘涅槃（超越生死、絕對寧靜的解脫境界）。

**◈ 栰者，譬八正道。**
　　筏子，譬喻為八正道。

**◈ 手足方便截流渡者，譬精進勇猛到彼岸。**
　　運用手足方便橫渡河流，譬喻為精進勇猛地達到彼岸。

**◈ 婆羅門住處者，譬如來、應、等正覺。**
　　婆羅門的住處，譬喻為如來、應供、等正覺者。

**◈ 「如是，比丘！大師慈悲安慰弟子，為其所作；我今已作，汝今亦當作其所作，**
　　就這樣，比丘！大師慈悲地安慰弟子，為他們做該做的事；我現在已經為你們做了，你們現在也應當去做該做的事：

**◈ 於空閑樹下，房舍清淨，敷草為座，露地、塚間，遠離邊坐，精勤禪思，慎莫放逸，令後悔恨！**
　　在空閑的樹下，或在清淨的房舍中，鋪草為座，或在露地、塚間，遠離他人而坐，精勤地進行禪思，慎重地不要放逸，以免日後後悔！

**◈ 此則是我教授之法。」**
　　這就是我教授你們的法。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行！**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐！

### [詞語註解（解釋重要佛教概念）]
1. **四大 (Mahābhūta)**：構成物質世界的四種基本元素，即地（堅實）、水（凝聚）、火（溫熱）、風（流動）。
2. **五受陰 (Pañca Vedanā-kkhandha)**：指對五種感官接觸後產生的感受（苦、樂、捨），在此被比喻為「拔刀仇家」，意指感受若被執著，將成為傷害心靈的來源。
3. **六內入 (Saḷāyatana)**：指六種內在感官（眼、耳、鼻、舌、身、意）。
4. **四流 (Catu-ota)**：指將眾生牽引在輪迴中的四種強大衝動 ➔ 欲流（對感官快感的追求）、有流（對生存與存在的執著）、見流（對錯誤見解的執著）、無明流（對真理的無知）。
5. **三愛 (Taṇhā)**：指三種貪愛 ➔ 欲愛（對感官慾望的渴求）、色愛（對有色物質形態的渴求）、無色愛（對無色精神境界的渴求）。
6. **無餘涅槃 (Anupādisesa-nibbāna)**：指徹底熄滅所有煩惱且身體死亡後，不再受生於輪迴的最終解脫狀態。

### [現代實作指南]
本經以極其生動的「逃亡」比喻，揭示了生命處於輪迴中的危險狀態。我們可以將其轉化為以下心理實踐：

1. **覺察「身體的脆弱性」 (盒子與毒蛇)**：
   - **實作**：在日常生活中，當身體感到不適（如發燒、疼痛）時，不要僅僅將其視為「我的病」，而應觀察到這是「四大元素」的失調。意識到身體如同一個暫時存放元素的盒子，本質上是無常且危脆的，藉此降低對物質身體的過度執著。

2. **識別「感受的陷阱」 (拔刀仇家)**：
   - **轉念**：當強烈的情緒（如憤怒、極度悲傷或狂喜）襲來時，將其視為「拔刀的仇家」。意識到這些感受若被我們「認同」或「執取」，就會像刀一樣傷害我們。練習對感受採取「觀察者」視角，而非「參與者」視角。

3. **警覺「感官的誘惑」 (空村強盜)**：
   - **實作**：當我們被外界的讚美、美色或物質誘惑吸引時，提醒自己這就像是「空村中的強盜」。這些外部對象（外六入）會利用我們的感官（內六入）來奪走我們的內心平靜。

4. **構建「心靈的筏子」 (八正道)**：
   - **實作**：在面對生活壓力（深流）時，不要試圖用情緒化的反應去對抗，而應運用「八正道」作為工具。例如：以「正見」分析問題本質，以「正念」覺察當下，以「正精進」採取正確的行動。

### [段落大意]
佛陀以一名士夫逃離毒蛇、仇家、內賊與強盜，最終橫渡深河到達彼岸的比喻，詳盡地對照出修行者如何從物質身體的無常（四大）、感受的折磨（五受）、貪愛的牽引（六愛）、感官的虛幻（六入）以及輪迴的衝動（四流與三愛）中解脫。佛陀強調，唯有透過八正道（筏子）與勇猛精進，才能從充滿恐怖的此岸（有身）到達清涼安樂的彼岸（無餘涅槃），並勉勵比丘遠離喧囂，精勤禪思。
