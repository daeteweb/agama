# 雜阿含經 第 1223 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

時，王舍城中有一士夫，貧窮辛苦，而住佛、法、僧，受持禁戒，多聞廣學，力行惠施，正見成就。彼身壞命終，得生天上，生三十三天，有三事勝於餘三十三天。何等為三？一者天壽，二者天色，三者天名稱。諸三十三天見是天子三事特勝，天壽、天色、天名稱勝。

餘諸天見已，往詣天帝釋所，作如是言：「憍尸迦當知，有一天子始生此天，於先諸天三事特勝，天壽、天色及天名稱。」

時，天帝釋告彼天子：「諸仁者！我見此人於王舍城作一士夫，貧窮辛苦，於如來法、律得信向心，乃至正見成就。身壞命終，來生此天，於諸三十三天三事特勝，天壽、天色及天名稱。」時，天帝釋即說偈言：

「正信於如來，　　決定不傾動， 　受持真實戒，　　聖戒無厭者。 　於佛心清淨，　　成就於正見， 　當知非貧苦，　　不空而自活。 　故於佛法僧，　　當生清淨信， 　智慧力增明，　　思念佛正教。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 時，王舍城中有一士夫，貧窮辛苦，而住佛、法、僧，受持禁戒，多聞廣學，力行惠施，正見成就。**
　　那時，王舍城中有一位在家男性（士夫），雖然生活貧窮辛苦，但依然信仰佛、法、僧三寶，承接並實踐戒律（受持禁戒），廣泛聽聞並學習教法（多聞廣學），盡力實踐布施（力行惠施），最終達成了正見（正確的見解）。

**◈ 彼身壞命終，得生天上，生三十三天，有三事勝於餘三十三天。**
　　他在身體毀壞、生命結束（身壞命終）後，得以轉生到天界，出生在三十三天（忉利天），且在三件事上優於該天界的其他眾生。

**◈ 何等為三？一者天壽，二者天色，三者天名稱。**
　　這三件事是什麼呢？第一是天壽（壽命），第二是天色（相貌），第三是天名稱（名聲）。

**◈ 諸三十三天見是天子三事特勝，天壽、天色、天名稱勝。**
　　三十三天的所有眾生看到這位天子在天壽、天色與天名稱這三件事上特別卓越。

**◈ 餘諸天見已，往詣天帝釋所，作如是言：「憍尸迦當知，有一天子始生此天，於先諸天三事特勝，天壽、天色及天名稱。」**
　　其他天人們看到後，前往帝釋天（天帝釋）那裡，這樣說道：「憍尸迦（天人名）請知道，現在有一位天子剛出生在這個天界，他在天壽、天色以及天名稱這三件事上，比之前的天人們都要卓越。」

**◈ 時，天帝釋告彼天子：「諸仁者！我見此人於王舍城作一士夫，貧窮辛苦，於如來法、律得信向心，乃至正見成就。**
　　那時，帝釋天告訴那些天人們說：「諸位仁者！我曾看見這個人在王舍城時是一位在家男性（士夫），雖然貧窮辛苦，但對如來（成就不變真理而來到世間的人）的法（真理）與律（戒律）產生了信仰之心，直到達成了正見（正確的見解）。」

**◈ 身壞命終，來生此天，於諸三十三天三事特勝，天壽、天色及天名稱。」**
　　因此在身體毀壞、生命結束（身壞命終）後，轉生到這個天界，在天壽、天色以及天名稱這三件事上，優於三十三天的所有眾生。」

**◈ 時，天帝釋即說偈言：**
　　那時，帝釋天隨即說了偈頌：

**◈ 「正信於如來，決定不傾動，受持真實戒，聖戒無厭者。」**
　　「對如來（成就不變真理而來到世間的人）擁有正信，且堅定不移，承接並實踐真實的戒律，對聖者的戒律永不厭倦的人。」

**◈ 「於佛心清淨，成就於正見，當知非貧苦，不空而自活。」**
　　「對佛陀心懷清淨，達成了正見（正確的見解），應當了解這樣的人並非貧苦，也不是空虛地生存。」

**◈ 「故於佛法僧，當生清淨信，智慧力增明，思念佛正教。」**
　　「所以對於佛、法、僧三寶，應當生起清淨的信仰，如此智慧的力量會增加光明，並時刻思念佛陀的正教。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **正見 (Sammā-diṭṭhi)**：八正道之首，指對四聖諦（苦、集、滅、道）的正確認知與理解，是修行解脫的基礎。
2. **三十三天 (Tavatimsa)**：佛教宇宙觀中的忉利天，由帝釋天統治，是欲界四天之一。
3. **惠施 (Dāna)**：一種修行方法，透過慷慨地給予他人（物質或法施）來對治貪婪，積累善業。
4. **聖戒 (Sīla)**：指符合真理、能導向解脫的道德規範，非僅是世俗的禮儀，而是為了止息煩惱而設的禁戒。

### [現代實作指南]
本經文強調「內在精神資產」與「外在物質條件」的脫鉤。在現代生活中，我們可以將其轉化為以下實踐：

1. **定義「真正的貧窮」**：
   - 轉念技巧 ➔ 將「貧窮」從「金錢不足」重新定義為「缺乏正見與信仰」。當一個人擁有正確的價值觀（正見）並能實踐善良（惠施）時，其心靈狀態是富足的，而非匱乏。
2. **建立「精神定錨」**：
   - 操作方法 ➔ 在面對生活壓力（如經濟困難、環境惡劣）時，練習將注意力從「缺乏什麼」轉移到「能實踐什麼」。例如：即便沒有錢財，依然可以實踐「正語」或「耐心的傾聽」（這即是現代的惠施）。
3. **培養「不傾動」的定力**：
   - 實踐步驟 ➔ 選擇一個自己認同的真理或道德準則（如：誠實、慈悲），在面對誘惑或困難時，練習堅持該準則而不隨環境搖擺。這種「決定不傾動」的心理韌性，能有效降低焦慮感。

### [段落大意]
本段敘述一名王舍城的貧苦士夫，雖物質匱乏但深信三寶、持戒、布施並成就正見，死後轉生至三十三天，且在壽命、相貌與名聲三方面優於其他天人。帝釋天藉此說明，真正的富足來自於對如來的正信與正見的成就，鼓勵眾生生起清淨信並思念佛陀教法。
