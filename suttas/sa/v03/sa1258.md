# 雜阿含經 第 1258 經

### [原文]
如是我聞：

一時，佛住波羅[木*奈]國仙人住處鹿野苑中。

爾時，世尊告諸比丘：「過去世時，有一人名陀舍羅訶，彼陀舍羅訶有鼓名阿能訶，好聲、美聲、深聲，徹四十里。彼鼓既久，處處裂壞。爾時，鼓士裁割牛皮，周匝纏縛；雖復纏縛，鼓猶無復高聲、美聲、深聲。彼於後時，轉復朽壞，皮大剝落，唯有聚木。

「如是，比丘！修身、修戒、修心、修慧，以彼修身、修戒、修心、修慧故，於如來所說修多羅甚深明照，難見難覺，不可思量，微密決定，明智所知，彼則頓受、周備受，聞其所說，歡喜崇習，出離饒益。當來比丘不修身、不修戒、不修心、不修慧，聞如來所說修多羅甚深明照空相應隨順緣起法，彼不頓受持，不至到受。聞彼說者，不歡喜崇習，而於世間眾雜異論、文辭綺飾、世俗雜句，專心頂受，聞彼說者，歡喜崇習，不得出離饒益。於彼如來所說甚深明照空相要法隨順緣起者，於此則滅，猶如彼鼓，朽故壞裂，唯有聚木。是故，諸比丘！當勤方便修身、修戒、修心、修慧，於如來所說甚深明照空相要法隨順緣起，頓受、遍受。聞彼說者，歡喜崇習，出離饒益。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住波羅[木*奈]國仙人住處鹿野苑中。**
　　在某一個時間，佛陀住在波羅奈國（今印度瓦拉納西）仙人居住的鹿野苑中。

**◈ 爾時，世尊告諸比丘：「過去世時，有一人名陀舍羅訶，彼陀舍羅訶有鼓名阿能訶，好聲、美聲、深聲，徹四十里。**
　　那時，世尊告訴在場的眾位比丘說：「在過去的世間，有一個人名叫陀舍羅訶，他擁有一面名為『阿能訶』的鼓，這面鼓的聲音非常好、優美且深沉，聲音能傳播到四十里之外。

**◈ 彼鼓既久，處處裂壞。**
　　然而這面鼓經過長時間的使用，各處都開始裂開損壞。

**◈ 爾時，鼓士裁割牛皮，周匝纏縛；雖復纏縛，鼓猶無復高聲、美聲、深聲。**
　　那時，擊鼓的人裁切了新的牛皮，將鼓面周圍重新纏繞縛緊；儘管重新修補，這面鼓卻再也無法發出像以前那樣高亢、優美且深沉的聲音。

**◈ 彼於後時，轉復朽壞，皮大剝落，唯有聚木。」**
　　在那之後，這面鼓進一步朽爛，皮面大面積地剝落，最後只剩下一塊聚在一起的木頭。」

**◈ 「如是，比丘！修身、修戒、修心、修慧，以彼修身、修戒、修心、修慧故，於如來所說修多羅甚深明照，難見難覺，不可思量，微密決定，明智所知，彼則頓受、周備受，聞其所說，歡喜崇習，出離饒益。**
　　「比丘們，就像這樣！如果能修行身體、修行戒律、修行心靈、修行智慧，正因為有了這些基礎的修行，對於如來所說的、關於修多羅（佛陀的教法）中那些深奧明徹、難以見到且難以覺察、無法用常理思量、極其微細且確定的真理，以及只有具備明智的人才能領悟的法義，修行者就能夠立刻領悟並全面地接受。當他們聽聞這些教法時，會感到歡喜並虔誠地學習，從而獲得出離輪迴的利益。

**◈ 當來比丘不修身、不修戒、不修心、不修慧，聞如來所說修多羅甚深明照空相應隨順緣起法，彼不頓受持，不至到受。**
　　反之，如果比丘不修行身體、不修行戒律、不修行心靈、不修行智慧，即便聽聞如來所說的、關於空相（一切現象的本質都是空性的）且隨順緣起（因緣而生）的深奧明徹之法，他們也無法立刻領悟並持守，甚至無法真正接收到這些法義。

**◈ 聞彼說者，不歡喜崇習，而於世間眾雜異論、文辭綺飾、世俗雜句，專心頂受，聞彼說者，歡喜崇習，不得出離饒益。**
　　他們聽聞佛法時，不會感到歡喜或想要學習，反而對於世間各種雜亂的異端論調、華麗的辭藻修飾以及世俗的瑣碎句子，專心地頂禮接受；聽聞這些世俗之論時，他們反而歡喜地學習，但這樣做無法獲得出離輪迴的利益。

**◈ 於彼如來所說甚深明照空相要法隨順緣起者，於此則滅，猶如彼鼓，朽故壞裂，唯有聚木。**
　　對於如來所說的、關於空相（一切現象的本質都是空性的）且隨順緣起（因緣而生）的深奧明徹之要法，他們的領悟力將會消失，就像那面鼓因為朽爛而裂開，最後只剩下一塊聚在一起的木頭。

**◈ 是故，諸比丘！當勤方便修身、修戒、修心、修慧，於如來所說甚深明照空相要法隨順緣起，頓受、遍受。聞彼說者，歡喜崇習，出離饒益。」**
　　因此，比丘們！應當勤奮地尋找方法來修行身體、修行戒律、修行心靈、修行智慧，以便能立刻且全面地領悟如來所說的、關於空相（一切現象的本質都是空性的）且隨順緣起（因緣而生）的深奧明徹之要法。如此聽聞教法，才能心生歡喜並虔誠學習，獲得出離輪迴的利益。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **修多羅 (Sutta)**：指佛陀的經文或教說，是修行者領悟真理的指南。
2. **空相 (Suññatā)**：指一切現象（法）都沒有永恆不變的獨立自性，其本質是空性的。
3. **緣起 (Paṭicca-samuppāda)**：指一切現象都是依據特定的條件（因緣）而生起，沒有無因之果。
4. **出離 (Viveka/Mutti)**：指從對世俗欲望的執著以及輪迴的苦海中解脫出來。

### [現代實作指南]
本經以「朽壞之鼓」比喻「缺乏基礎的心靈」。若想在生活中實踐此教導，可採取以下步驟：

1. **建立「接收器」的基礎（修身與修戒）**：
   ➔ **身體層面**：保持適度的健康與規律生活，避免因極端疲憊或病痛導致心神不寧。
   ➔ **行為層面**：設定簡單的道德底線（如：不欺騙、不傷害他人）。當行為端正時，內心的愧疚感與焦慮會減少，為學習深奧真理創造平靜的環境。

2. **優化「處理能力」（修心與修慧）**：
   ➔ **心靈層面**：練習正念，觀察情緒的起伏而不被其牽引。這就像是在修補鼓面，讓心靈恢復彈性與穩定。
   ➔ **智慧層面**：練習邏輯思考與反思，不盲從於華麗的言辭或流行觀點，而是分析事物的因果關係（緣起）。

3. **辨別「有效資訊」與「噪音」**：
   ➔ 意識到現代社會充滿了「文辭綺飾」的碎片化資訊（如社群媒體上的成功學、情緒化言論）。
   ➔ 當發現自己對這些資訊感到興奮，而對深刻的真理感到枯燥時，應警覺自己的「心靈鼓面」是否已朽壞，進而回歸到基礎的修行（戒、定、慧）。

### [段落大意]
佛陀以一面因朽壞而無法發聲的鼓為喻，說明若修行者不先在身體、戒律、心靈與智慧上打好基礎，即便聽聞最深奧的空相與緣起法，也無法領悟或獲益，反而容易被世俗的華麗言論吸引。因此，修行者必須勤於基礎修習，才能真正接收並實踐佛法，達成出離之果。
