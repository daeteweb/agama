# 雜阿含經 第 1207 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，尸利沙遮羅比丘尼亦住舍衛國王園比丘尼眾中，晨朝著衣持鉢，入舍衛城乞食。食已，還精舍，舉衣鉢，洗足畢，持尼師壇，著肩上，入安陀林，坐一樹下，入晝正受。

時，魔波旬作是念：「今沙門瞿曇住舍衛國祇樹給孤獨園。時，尸利沙遮羅比丘尼亦住舍衛國王園比丘尼眾中，晨朝著衣持鉢，入舍衛城乞食，食已，還精舍，舉衣鉢，洗足畢，持尼師壇，著肩上，入安陀林，坐一樹下，入晝正受。我當往彼，為作留難。」化作年少，容貌端正，往到尸利沙遮羅比丘尼所而作是言：「阿姨！汝樂何等諸道？」

比丘尼答言：「我都無所樂！」

時，魔波旬即說偈言：

「汝何所諮受，　　剃頭作沙門？ 　身著袈裟衣，　　而作出家相， 　不樂於諸道，　　而守愚癡住。」

時，尸利沙遮羅比丘尼作是念：「此何等人？欲恐怖我。為人？為非人？為姦狡人？」如是思惟已，即自知覺：「惡魔波旬欲作嬈亂。」即說偈言：

「此法外諸道，　　諸見所纏縛， 　縛於諸見已，　　常隨魔自在。 　若生釋種家，　　稟無比大師， 　能伏諸魔怨，　　不為彼所伏。 　清淨一切脫，　　道眼普觀察， 　一切智悉知，　　最勝離諸漏。 　彼則我大師，　　我唯樂彼法， 　我入彼法已，　　得遠離寂滅。 　離一切愛喜，　　捨一切闇冥， 　寂滅以作證，　　安住諸漏盡。 　已知汝惡魔，　　如是自滅去。」

時，魔波旬作是念：「尸利沙遮羅比丘尼已知我心。」內懷憂慼，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛住在舍衛國的祇樹給孤獨園。

**◈ 時，尸利沙遮羅比丘尼亦住舍衛國王園比丘尼眾中，晨朝著衣持鉢，入舍衛城乞食。**
　　那時，尸利沙遮羅比丘尼也住在舍衛國王園的比丘尼群落中。她每天早晨穿好袈裟、拿著缽，進入舍衛城乞食。

**◈ 食已，還精舍，舉衣鉢，洗足畢，持尼師壇，著肩上，入安陀林，坐一樹下，入晝正受。**
　　乞食完畢後，她回到精舍，收拾好衣鉢，洗淨雙足，將比丘尼的坐墊（尼師壇）披在肩上，進入安陀林，在樹下坐定，進入白天的正定禪修狀態。

**◈ 時，魔波旬作是念：「今沙門瞿曇住舍衛國祇樹給孤獨園。時，尸利沙遮羅比丘尼亦住舍衛國王園比丘尼眾中，晨朝著衣持鉢，入舍衛城乞食，食已，還精舍，舉衣鉢，洗足畢，持尼師壇，著肩上，入安陀林，坐一樹下，入晝正受。我當往彼，為作留難。」**
　　這時，魔波旬心裡想：「現在沙門瞿曇住在舍衛國的祇樹給孤獨園。而尸利沙遮羅比丘尼也住在舍衛國王園的比丘尼群落中，她早晨乞食，隨後回到精舍收拾衣鉢、洗足，拿著坐墊進入安陀林，在樹下進入白天的正定禪修。我應該前往那裡，為她製造困難以阻礙她的修行。」

**◈ 化作年少，容貌端正，往到尸利沙遮羅比丘尼所而作是言：「阿姨！汝樂何等諸道？」**
　　於是魔波旬幻化成一名容貌端正的年輕男子，走到尸利沙遮羅比丘尼面前說：「阿姨！您在追求或喜愛什麼樣的修行道路？」

**◈ 比丘尼答言：「我都無所樂！」**
　　比丘尼回答說：「我對任何世俗的追求都沒有喜愛。」

**◈ 時，魔波旬即說偈言：「汝何所諮受，剃頭作沙門？身著袈裟衣，而作出家相，不樂於諸道，而守愚癡住。」**
　　這時，魔波旬立即說道：「妳究竟在追求什麼，才要剃除頭髮成為沙門？身上穿著袈裟，擺出出家的樣子，卻說對任何道路都沒有喜愛，這不是在愚癡中生活嗎？」

**◈ 時，尸利沙遮羅比丘尼作是念：「此何等人？欲恐怖我。為人？為非人？為姦狡人？」**
　　這時，尸利沙遮羅比丘尼心裡想：「這個人是誰？想要用恐嚇來動搖我。他是人類？還是非人類？或者是個狡詐的人？」

**◈ 如是思惟已，即自知覺：「惡魔波旬欲作嬈亂。」即說偈言：**
　　這樣思考之後，她立刻覺察到：「這是惡魔波旬想要製造混亂。」於是她說道：

**◈ 「此法外諸道，諸見所纏縛，縛於諸見已，常隨魔自在。」**
　　「那些在佛法之外的修行道路，都被各種錯誤的見解（想）所纏縛；一旦被這些見解所束縛，就只能隨波逐流地被魔波旬所掌控。」

**◈ 「若生釋種家，稟無比大師，能伏諸魔怨，不為彼所伏。」**
　　「但如果能生在釋迦族中，隨侍於那位無比的大師（佛陀），就能降伏所有魔之怨敵，而不會被他們所掌控。」

**◈ 「清淨一切脫，道眼普觀察，一切智悉知，最勝離諸漏。」**
　　「大師清淨地脫離了一切束縛，用道眼普普觀察，擁有全知的智慧，最卓越地遠離了所有使心染污的漏（煩惱）。」

**◈ 「彼則我大師，我唯樂彼法，我入彼法已，得遠離寂滅。」**
　　「那位佛陀就是我的大師，我唯獨喜愛他所教導的法。在我進入這法門之後，獲得了遠離煩惱的寂滅境界。」

**◈ 「離一切愛喜，捨一切闇冥，寂滅以作證，安住諸漏盡。」**
　　「我遠離了所有貪愛與喜好，捨棄了所有無明（闇冥），以寂滅為證，安住在所有煩惱（漏）完全盡除的狀態中。」

**◈ 「已知汝惡魔，如是自滅去。」**
　　「我已經看穿了你這個惡魔，請你立刻消失吧。」

**◈ 時，魔波旬作是念：「尸利沙遮羅比丘尼已知我心。」內懷憂慼，即沒不現。**
　　這時，魔波旬心裡想：「尸利沙遮羅比丘尼已經看穿了我的意圖。」他內心感到憂慮與沮喪，隨即消失不見。

### [詞語註解]
1. **正受 (Samaāpatti)**：指進入禪定或特定的心理狀態，在此指比丘尼在白天的禪修定境。
2. **留難 (Obstruction)**：指魔波旬透過誘惑或恐嚇，使修行者產生雜念，從而中斷定力或退轉信心的行為。
3. **見 (Diṭṭhi)**：指對現實的認知或看法。在文中指「錯誤的見解」，即未能洞察緣起真理而產生的執著。
4. **漏 (Āsava)**：字面意思是「滲漏」，指深藏在心靈深處、不斷流出的染污煩惱（如煩惱、渴愛、無明），是導致輪迴的根本原因。
5. **寂滅 (Nirodha/Nibbāna)**：指煩惱熄滅、痛苦停止的狀態，即超越生死、絕對寧靜的解脫境界（涅槃）。

### [現代實作指南]
本經文描述了修行者在面對「內在誘惑」與「外在質疑」時的心理應對機制，可轉化為以下實踐步驟：

1. **覺察「魔」的形態 ➔ 識別負面情緒**：
   魔波旬化身為「容貌端正的年少」，象徵誘惑往往以吸引人的形式出現；而他的質疑（如「妳在守愚癡住」）則象徵內心的自我懷疑或外界的評判。當你在專注目標時感到焦慮或被質疑，請將其標記為「魔波旬（干擾項）」，而非事實。

2. **運用「不樂」的心理防禦 ➔ 練習捨心**：
   比丘尼回答「我都無所樂」，並非指生活痛苦，而是練習「捨（Upekkhā）」。在面對誘惑或挑釁時，不採取激烈的對抗，也不陷入喜悅或厭惡，而是保持中立，使干擾項找不到切入點。

3. **邏輯分析與正見對照 ➔ 轉念技巧**：
   比丘尼在面對恐嚇時，先進行分析（是人？非人？狡詐人？），隨後將其對照到「法」的標準（法外諸道 → 纏縛 → 隨魔）。
   **實作方法**：當你被負面情緒纏繞時，問自己：「這個想法是基於對現實的正確認知（正見），還是基於一種對特定結果的執著（見）？」

4. **回歸核心價值 ➔ 建立心理錨點**：
   比丘尼透過回想大師的特質（清淨、全知、離漏）來強化自己的信心。
   **實作方法**：在混亂時，迅速將注意力轉移到你的核心價值或最終目標（如：內心的平靜、對他人的慈悲），用「確定性」來抵消「不確定性」帶來的恐懼。
