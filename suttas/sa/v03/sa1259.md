# 雜阿含經 第 1259 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「譬如鐵丸投著火中，與火同色，盛著劫貝綿中。云何？比丘！當速燃不？」

比丘白佛：「如是，世尊！」

佛告比丘：「愚癡之人依聚落住，晨朝著衣持鉢，入村乞食，不善護身，不守根門，心不繫念，若見年少女人，不正思惟，取其色相，起貪欲心；欲燒其心，欲燒其身；身心燒已，捨戒退減。是愚癡人長夜當得非義饒益。是故，比丘！當如是學：『善護其身，守護根門，繫念，入村乞食。』當如是學。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「譬如鐵丸投著火中，與火同色，盛著劫貝綿中。云何？比丘！當速燃不？」**
　　那時，世尊告訴在場的眾位比丘說：「比喻就像一顆鐵球被投入火中，變得與火一樣紅熱，接著被放入貝殼棉花之中。比丘們，你們認為這顆鐵球會迅速燃燒起來嗎？」

**◈ 比丘白佛：「如是，世尊！」**
　　比丘們對佛陀恭敬地說道：「是的，世尊！」

**◈ 佛告比丘：「愚癡之人依聚落住，晨朝著衣持鉢，入村乞食，不善護身，不守根門，心不繫念，若見年少女人，不正思惟，取其色相，起貪欲心；**
　　佛陀告訴比丘：「那些愚癡的人住在村落附近，每天早晨穿上袈裟、拿著缽，進入村莊乞食，但他們沒有好好地保護自己的身體行為，沒有守護根門（眼、耳、鼻、舌、身、意這六個感官的入口），心中也沒有繫住正念（對當下的覺知）。如果他們看到年輕女性，產生不正的思惟（不正確的思考方向），執著於對方的色（物質身體）相貌，進而起了貪愛之心；

**◈ 欲燒其心，欲燒其身；身心燒已，捨戒退減。**
　　這種貪欲就像火一樣，會燒灼他們的心，也會燒灼他們的身；當身心被貪欲之火燒盡後，他們便會捨棄戒律，修行退步。

**◈ 是愚癡人長夜當得非義饒益。**
　　這樣的愚癡之人，在輪迴（長夜）的過程中，將會得到不正當且有害的果報。

**◈ 是故，比丘！當如是學：『善護其身，守護根門，繫念，入村乞食。』當如是學。」**
　　因此，比丘們！應該這樣學習：『在進入村莊乞食時，要善於保護自己的身體行為，守護根門（感官入口），並繫住正念（對當下的覺知）。』應該這樣學習。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **根門 (Indriya-dvāra)**：指六根（眼、耳、鼻、舌、身、意）與外界接觸的入口。守護根門是指在感官接觸外界對象時，不讓貪欲或嗔心隨之生起。
2. **正念 (Sati)**：對當下身心狀態的清晰覺知，不被妄想或情緒牽引。
3. **色 (Rūpa)**：在此指物質身體及其外在相貌。
4. **長夜 (Samsāra)**：比喻輪迴的漫長過程，充滿了無明與苦難。
5. **非義饒益**：指不符合正法、不道德或會導致未來痛苦的獲益或結果。

### [現代實作指南]
本經文的核心在於「感官管理」與「預防機制」。在現代生活中，我們雖不需入村乞食，但每天面對海量的視覺（社群媒體、廣告）與聽覺資訊，極易陷入「鐵球入棉花」般的快速燃燒狀態。

**實作步驟：**
1. **識別「觸」的瞬間** ➔ 當你看到令人心動的人或物（色相）時，立刻意識到這是一個「觸」的過程。
2. **建立「根門」緩衝區** ➔ 在「看到」與「起貪欲」之間建立一個微小的停頓。告訴自己：「我現在看到了某個對象，我的心正在產生反應。」
3. **防止「不正思惟」的擴散** ➔ 貪欲像火，想像力則是棉花。當你開始想像對方的美好或幻想擁有時，就是在增加「棉花」。此時應立即將注意力移回呼吸或當下的腳步（繫念）。
4. **覺察「燒灼感」** ➔ 體會貪欲生起時，內心那種不安、焦慮或急迫的感覺，這就是「燒心」的實相。透過觀察這種不適感，讓貪欲自然冷卻。

佛陀以紅熱鐵球放入棉花會迅速燃燒為喻，告誡比丘在乞食過程中若不守護感官入口（根門）且缺乏正念，一旦對年輕女性產生貪欲，將如火燒身心般導致戒律崩潰，在輪迴中受苦。因此，強調修行者必須在日常活動中善護身心並保持覺知。
