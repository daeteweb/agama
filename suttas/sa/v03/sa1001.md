# 雜阿含經 第 1001 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。

時，彼天子而說偈言：

「冥運持命去，　　故令人短壽， 　為老所侵迫，　　而無救護者。 　覩斯老病死，　　令人大恐怖， 　唯作諸功德，　　樂住至樂所。」

爾時，世尊說偈答言：

「冥運持命去，　　故令人短壽， 　為老所侵迫，　　而無救護者。 　觀此有餘過，　　令人大恐怖， 　當斷世貪愛，　　無餘涅槃樂。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。**
　　那時，有一位相貌極其端正美麗的天子，在深夜時分來到佛陀所在之處，頂禮佛陀的足下，然後退到一邊坐下。

**◈ 身諸光明遍照祇樹給孤獨園。**
　　他身體散發出的各種光芒，照亮了整個祇樹給孤獨園。

**◈ 時，彼天子而說偈言：**
　　那時，那位天子說了偈頌（詩偈）說：

**◈ 「冥運持命去，故令人短壽，為老所侵迫，而無救護者。」**
　　「由於被無明（對真理的迷失）所掌控，生命不斷流逝，因此使人的壽命短暫，被衰老所逼迫，而沒有任何人能將其救護。」

**◈ 「覩斯老病死，令人大恐怖，唯作諸功德，樂住至樂所。」**
　　「看見這種衰老、疾病與死亡的現象，令人感到極大的恐懼；認為唯有積累各種功德，才能安住在至高快樂的境界中。」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「冥運持命去，故令人短壽，為老所侵迫，而無救護者。」**
　　「確實，由於被無明（對真理的迷失）所掌控，生命不斷流逝，因此使人的壽命短暫，被衰老所逼迫，而沒有任何人能將其救護。」

**◈ 「觀此有餘過，令人大恐怖，當斷世貪愛，無餘涅槃樂。」**
　　「觀察到生命中這些殘餘的缺陷（苦諦），會令人感到極大的恐懼；因此應當斷除對世間的貪愛（對感官享受與存在的執著），才能獲得完全沒有殘餘痛苦的涅槃（超越生死、絕對寧靜的解脫境界）之樂。」

**◈ 時，彼天子復說偈言：**
　　那時，那位天子再次說了偈頌說：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「長久以來看見那些修行者（婆羅門），最終達到了般涅槃（完全的解脫），一切恐懼都已經過去，永遠超越了世間的恩愛（對情愛與物質的執著）。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，那位天子聽聞佛陀所說的教法，心中感到歡喜並隨喜，頂禮佛陀的足下，隨即消失不再出現。

### [詞語註解（解釋重要佛教概念）]
1. **無明 (Avijjā)**：指對四聖諦（苦、集、滅、道）的無知，是十二因緣的第一環，也是導致輪迴與痛苦的根本原因。
2. **功德 (Puñña)**：指透過善行、布施、持戒所積累的正面能量。在天子最初的看法中，功德被視為逃避老病死的手段，但佛陀指出真正的解脫需透過斷除貪愛。
3. **貪愛 (Taṇhā)**：指對感官快感的渴求、對存在的渴求或對不存在的渴求，是導致「集（苦因）」的核心動力。
4. **涅槃 (Nibbāna)**：指貪、嗔、痴三毒完全熄滅的狀態，是超越生死輪迴、絕對寧靜且不再受苦的解脫境界。

### [現代實作指南]
本經文揭示了從「恐懼」到「解脫」的認知轉向。現代人常面對中年危機、疾病或對死亡的焦慮，可採取以下實踐步驟：

1. **正視恐懼（觀照老病死）**：
   ➔ 不要刻意逃避衰老或疾病的現實。當恐懼感升起時，不要試圖用物質消費或短暫的娛樂來掩蓋，而是靜靜觀察這種「恐懼」在身體上的感受（如胸口緊縮、不安）。

2. **辨識「功德」的局限（區分暫時快樂與永恆解脫）**：
   ➔ 意識到單純的「做善事」或「追求成功」雖然能帶來暫時的心理安慰（類似天子的功德觀），但無法根除對死亡的根本恐懼。將追求「外在獎賞」轉向追求「內在覺醒」。

3. **分析貪愛的運作（斷世貪愛）**：
   ➔ 當我們對某樣東西產生強烈執著（貪愛）時，試著問自己：「如果這件東西消失了，我的恐懼來自哪裡？」意識到恐懼源於「想要掌控不可掌控之物」的執著。

4. **練習「放下」的寧靜（體驗微型涅槃）**：
   ➔ 在日常生活中，練習短暫地放下所有身份（如：我不是主管、我不是父母、我不是某人的孩子），僅僅作為一個「覺知者」存在。這種短暫的「無我」體驗，能讓我們預演涅槃中那種超越執著的寧靜感。

### [段落大意]
一位天子向佛陀表達對生命短暫、衰老與死亡的恐懼，並認為積累功德是唯一的救贖。佛陀則指引他，真正的解脫不在於積累功德，而在於觀察生命的苦諦並斷除對世間的貪愛，如此才能進入無餘涅槃的永恆快樂中。天子領悟後隨喜消失。
