# 雜阿含經 第 1300 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，釋提桓因於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。

時，釋提桓因說偈問佛：

「何法命不知？　　何法命不覺？ 　何法鏁於命？　　何法為命縛？」

爾時，世尊說偈答言：

「色者命不知，　　諸行命不覺， 　身鏁於其命，　　受縛於命者。」

釋提桓因復說偈言：

「色者非為命，　　諸佛之所說， 　云何而得熟？　　於彼甚深藏， 　云何段肉住？　　云何知命身？」

爾時，世尊說偈答言：

「迦羅邏為初，　　迦羅邏生胞， 　胞生於肉段，　　肉段生堅厚， 　堅厚生肢節，　　及諸毛髮等， 　色等諸情根，　　漸次成形體， 　因母飲食等，　　長養彼胎身。」

爾時，釋提桓因聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：一時，佛住舍衛國祇樹給孤獨園。**
　　我是這樣聽佛陀說的：在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，釋提桓因於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。**
　　那時，釋提桓因（天主因陀羅）在深夜時分來到佛陀這裡，頂禮佛足後，退到一邊坐下。他的身體散發出光明，遍照整個祇樹給孤獨園。

**◈ 時，釋提桓因說偈問佛：「何法命不知？　　何法命不覺？ 　何法鏁於命？　　何法為命縛？」**
　　那時，釋提桓因用偈頌問佛陀：「什麼樣的法（現象）使得生命處於不知狀態？什麼樣的法使得生命處於不覺狀態？什麼樣的法將生命束縛？什麼樣的法成為生命的枷鎖？」

**◈ 爾時，世尊說偈答言：「色者命不知，　　諸行命不覺， 　身鏁於其命，　　受縛於命者。」**
　　那時，世尊用偈頌回答說：「色（物質身體）使得生命處於不知狀態，諸行（心的造作）使得生命處於不覺狀態，身體被其生命（的執著）所束縛，而受（感受）則被生命（的輪迴）所囚禁。」

**◈ 釋提桓因復說偈言：「色者非為命，　　諸佛之所說， 　云何而得熟？　　於彼甚深藏， 　云何段肉住？　　云何知命身？」**
　　釋提桓因再次用偈頌說：「諸佛所說，色（物質身體）並非真正的生命，那麼（胎兒）是如何成熟發育的？在母體深處的子宮中，一塊肉段是如何生存的？又是如何知道生命身體的形成？」

**◈ 爾時，世尊說偈答言：「迦羅邏為初，　　迦羅邏生胞， 　胞生於肉段，　　肉段生堅厚， 　堅厚生肢節，　　及諸毛髮等， 　色等諸情根，　　漸次成形體， 　因母飲食等，　　長養彼胎身。」**
　　那時，世尊用偈頌回答說：「首先是迦羅邏（受精卵/最初的生命物質），迦羅邏產生胞衣，胞衣產生肉段，肉段逐漸變得堅厚，堅厚後生出肢體關節，以及毛髮等，接著色（物質身體）等諸種情根（感官）漸次形成形體，並依賴母親的飲食等營養，來長養這個胎兒之身。」

**◈ 爾時，釋提桓因聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，釋提桓因聽聞佛陀所說的教法，心生法喜並隨喜，頂禮佛足後，立即消失不見。

### [詞語註解]
1. **色 (Rūpa)**：指物質身體及其構成的物理現象。在本經中，佛陀強調物質身體並非永恆的「命」，而是因緣和合的產物。
2. **行 (Saṅkhāra)**：指心的造作、意志或心理傾向。它是導致生命在無明中不斷輪迴、不覺醒的主因。
3. **受 (Vedanā)**：指對刺激產生的感受（苦、樂、捨）。當人執著於感受時，便被生命之苦所囚禁。
4. **迦羅邏 (Kalala)**：原始佛教關於胚胎發育的第一階段，指受精後極微小、如油滴般的物質，是形體構成的起點。
5. **情根 (Indriya)**：指感知世界的器官（如眼、耳、鼻、舌、身、意），是物質身體與意識接觸外界的媒介。

### [現代實作指南]
本經揭示了「身體 $\neq$ 生命」的深刻洞察。在現代生活中，我們常將「我」與「我的身體」等同視之，導致對衰老、疾病與死亡產生極大的恐懼。可採取以下實作步驟：

1. **去標籤化觀察 (De-identification)**：
   當你感到身體疲累或疼痛時，試著將觀察對象從「我好累」轉變為「色（物質身體）正在呈現疲累的狀態」。透過這種轉念，將「自我」從物質身體中抽離，減少對身體變化的情緒反應。

2. **觀察「行」的運作**：
   當你對某件事產生強烈執著（如：渴望獲得某物）時，意識到這是「行（心的造作）」在運作。提醒自己：這種渴望是導致「不覺」的枷鎖，而非真實的生命需求。

3. **體悟因緣生滅**：
   思考經文中描述的胎兒發育過程 ➔ 從微小的物質 → 胞衣 → 肉段 → 肢體。意識到目前的身體僅僅是物質營養與因緣的暫時組合。當我們體會到身體是「漸次而成」的過程而非「固定不變」的實體，能有效降低對物質外貌的過度執著。

### [段落大意]
釋提桓因詢問佛陀關於生命被束縛與不覺的原因，佛陀指出色（物質身體）與行（心的造作）是導致無明與束縛的主因。隨後，佛陀詳細說明胎兒從迦羅邏開始，經由肉段、肢體到情根漸次形成的生物過程，以此證明身體僅是因緣和合的物質產物，而非永恆的生命實體。
