# 雜阿含經 第 1270 經

### [原文]
如是我聞：

一時，佛住王舍城山谷精舍。

時，有拘迦尼，是光明天女，容色絕妙，於後夜時來詣佛所，稽首佛足，身諸光明遍照山谷。

時，拘迦尼天女而說偈言：

「其心不為惡，　　及身口世間， 
五欲悉虛空，　　正智正繫念。 
不習近眾苦，　　非義和合者。」

佛告天女：「如是，如是。

「其心不為惡，　　及身口世間， 
五欲悉虛空，　　正智正繫念。 
不習近眾苦，　　非義和合者。」

時，拘迦那娑天女聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

爾時，世尊夜過晨朝，入於僧中，敷尼師壇，於大眾前坐，告諸比丘：「昨日夜後，有拘迦那天女，容色絕妙，來詣我所，稽首我足，退坐一面而說偈言：

「其心不為惡，　　及身口世間， 
五欲悉虛空，　　正智正繫念， 
不習近眾苦，　　非義和合者。」

我即答言：「如是，天女！如是，天女！

「其心不為惡，　　及身口世間， 
五欲悉虛空，　　正智正繫念。 
不習近眾苦，　　非義和合者。

「說是語時，拘迦尼天女聞我所說，歡喜隨喜，稽首我足，即沒不現。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **五欲 (Pañca-kāma)**：指對五種感官享受的貪愛，通常指財富、美色、名聲、飲食、睡眠。在經文中，「悉虛空」是指對這些欲望的執著心完全消失，達到一種空靈、不被牽引的狀態。
2. **正智 (Sammā-ñāṇa)**：正確的知見與智慧，能洞察事物真實的本質（如無常、苦、無我），而非基於偏見或妄想的認知。
3. **正繫念 (Sammā-sati/upatthāna)**：正確的覺知與專注。指能持續地將心繫於當下，清楚地觀察身心狀態，不被雜念帶走。
4. **非義和合 (Adhamma-samāgama)**：指不基於正法（Dhamma）而建立的社交關係或結合。例如為了利益、權力或感官快感而與他人結交，而非為了修行或真理而同行。

### [現代實作指南]
本經文提供了一套簡潔的「清淨生活清單」，可用於現代人的心理調適與行為管理：

1. **身口意三道的自我審核**：
   - **意 (心)**：每日回顧，是否有出於貪婪、嗔恨或愚癡的惡念？
   - **口 (言)**：今日是否說了謊言、惡言或無意義的廢話？
   - **身 (行)**：今日的行為是否對他人造成傷害或違背道德？
   - **實作**：在睡前進行 5 分鐘的「三道回顧」，將意識聚焦於修正而非自責。

2. **五欲的「虛空化」練習**：
   - 當強烈的欲望（如對名聲的渴望或對物質的執著）升起時，不要壓抑，而是觀察它。
   - 告訴自己：「這個欲望像雲一樣，雖然存在，但本質是空（虛空）的，它不定義我。」
   - ➔ 將「我想要」轉念為「我觀察到心中升起了一個想要...的念頭」。

3. **社交圈的「正義」篩選**：
   - 檢視目前的社交關係：哪些關係是基於「互惠利益」或「感官快感」（非義和合）？哪些關係是基於「共同成長」或「正向價值」（義和合）？
   - **實作**：減少與會誘導自己產生負面情緒或不良習慣的人接觸（不習近眾苦），增加與能激發正念、正見之人的交流。

4. **正智與正念的同步**：
   - 在面對衝突時，先運用「正繫念」覺察到憤怒的升起 ➔ 再運用「正智」思考這場衝突的因緣與無常 ➔ 最後選擇不造惡的反應方式。

### [段落大意]
本經記述拘迦尼天女深夜造訪佛陀，以偈頌讚嘆修行者應具備的特質：內心與身口不造惡、對五欲不執著、具備正智與正念，且遠離痛苦之源與不正當的社交。佛陀對此表示認同，並於次日將此對話分享給比丘眾，強調清淨心與正向社交對修行的重要性。
