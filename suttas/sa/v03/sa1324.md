# 雜阿含經 第 1324 經

### [原文]
如是我聞：

一時，佛在摩竭提國人間遊行，到針毛鬼住處夜宿。爾時，針毛鬼會諸鬼神，集在一處。

時，有炎鬼見世尊在針毛鬼住處夜宿。見已，往詣針毛鬼所，語針毛鬼言：「聚落主！汝今大得善利，今如來、應、等正覺於汝室宿。」

針毛鬼言：「今當試看，為是如來、為非？」

時，針毛鬼與諸鬼神集會已，還歸自舍，束身衝佛。爾時，世尊却身避之。如是再三，束身衝佛，佛亦再三却身避之。

爾時，針毛鬼言：「沙門怖耶？」

佛言：「聚落主！我不怖也，但汝觸惡。」

針毛鬼言：「今有所問，當為我說，能令我喜者善，不能令我喜者，當壞汝心，裂汝胸，令汝熱血從其面出，捉汝兩手擲恒水彼岸。」

佛告針毛鬼：「聚落主！我不見諸天、魔、梵、沙門、婆羅門、天神、世人能壞如來、應、等正覺心者，能裂其胸者，能令熱血從面出者，執其兩臂擲著恒水彼岸者。汝今但問，當為汝說。令汝歡喜。」

時，針毛鬼說偈問佛：

「一切貪恚心，　　以何為其因， 　不樂身毛竪，　　恐怖從何起？ 　意念諸覺想，　　為從何所起， 　猶如新生兒，　　依倚於乳母。」

爾時，世尊說偈答言：

「愛生自身長，　　如尼拘律樹， 　展轉相拘引，　　如籐綿叢林， 　若知彼所因，　　當令鬼覺悟， 　度生死海流，　　不復重增有。」

爾時，針毛鬼聞世尊說偈，心得歡喜，向佛悔過，受持三歸。

佛說此經已，針毛鬼聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛在摩竭提國人間遊行，到針毛鬼住處夜宿。爾時，針毛鬼會諸鬼神，集在一處。**
　　在某一個時間，佛陀在摩竭提國的人間遊行，來到一個名為「針毛鬼」的鬼神居住之處過夜。那時，針毛鬼召集了眾多鬼神，聚集在同一個地方。

**◈ 時，有炎鬼見世尊在針毛鬼住處夜宿。見已，往詣針毛鬼所，語針毛鬼言：「聚落主！汝今大得善利，今如來、應、等正覺於汝室宿。」**
　　這時，有一位炎鬼看見世尊在針毛鬼的住處過夜。看見之後，炎鬼前往針毛鬼那裡，對針毛鬼說：「聚落主！你現在獲得了極大的善利（福德利益），因為如來（成就不變真理而來到世間的人）、應供（應受世間供養的人）、正遍知（對真理有全面覺知的人）現在就住在你的房間裡。」

**◈ 針毛鬼言：「今當試看，為是如來、為非？」**
　　針毛鬼說：「我現在要去試探看看，他究竟是不是如來，還是另有其人？」

**◈ 時，針毛鬼與諸鬼神集會已，還歸自舍，束身衝佛。爾時，世尊却身避之。如是再三，束身衝佛，佛亦再三却身避之。**
　　那時，針毛鬼與諸鬼神集會結束後，回到自己的住處，猛然縮身衝向佛陀。這時，世尊向後退身避開。如此重複了三次，針毛鬼三次縮身衝向佛陀，佛陀也三次退身避開。

**◈ 爾時，針毛鬼言：「沙門怖耶？」**
　　那時，針毛鬼問道：「這位沙門（出家修行者）是在害怕嗎？」

**◈ 佛言：「聚落主！我不怖也，但汝觸惡。」**
　　佛陀回答：「聚落主！我並不害怕，只是你的觸碰（接觸）是不善的。」

**◈ 針毛鬼言：「今有所問，當為我說，能令我喜者善，不能令我喜者，當壞汝心，裂汝胸，令汝熱血從其面出，捉汝兩手擲恒水彼岸。」**
　　針毛鬼說：「我現在有問題要問，請你為我解答。如果能讓我滿意歡喜那就好；如果不能讓我歡喜，我就要摧毀你的心，撕裂你的胸膛，讓你滾燙的血液從胸口流出，並抓住你的兩隻手將你扔到恆河水的彼岸。」

**◈ 佛告針毛鬼：「聚落主！我不見諸天、魔、梵、沙門、婆羅門、天神、世人能壞如來、應、等正覺心者，能裂其胸者，能令熱血從面出者，執其兩臂擲著恒水彼岸者。汝今但問，當為汝說。令汝歡喜。」**
　　佛陀告訴針毛鬼：「聚落主！我看不到有任何天神、魔王、梵天、沙門、婆羅門、天神或世間之人，能夠摧毀如來（成就不變真理而來到世間的人）、應供、正遍知的心，能撕裂其胸膛，能令熱血從胸口流出，或能抓住其兩臂將其扔到恆河彼岸的。你現在儘管發問，我將為你解說，讓你感到歡喜。」

**◈ 時，針毛鬼說偈問佛：「一切貪恚心，以何為其因，不樂身毛竪，恐怖從何起？意念諸覺想，為從何所起，猶如新生兒，依倚於乳母。」**
　　這時，針毛鬼用偈頌問佛：「所有貪婪與憤怒的心，是以什麼為因緣而生的？為什麼會感到不快樂、身體起雞皮疙瘩，恐懼又是從何而起？心中那些覺知與想（想像與認知）的念頭，又是從什麼地方生起的？就像新生兒依賴乳母一樣（如此強烈地依附）。」

**◈ 爾時，世尊說偈答言：「愛生自身長，如尼拘律樹，展轉相拘引，如籐綿叢林，若知彼所因，當令鬼覺悟，度生死海流，不復重增有。」**
　　那時，世尊用偈頌回答說：「愛（貪愛，對感官欲望的渴求）在自身中生起並成長，就像尼拘律樹（一種分枝繁茂的樹）一樣，不斷地轉向、相互糾纏牽引，如同藤蔓交織的叢林。如果能知道這個因緣，就能讓鬼神覺悟，度過生死輪迴的海洋，不再增加新的有（存在，即輪迴的生存狀態）。」

**◈ 爾時，針毛鬼聞世尊說偈，心得歡喜，向佛悔過，受持三歸。**
　　那時，針毛鬼聽聞世尊的偈頌，內心感到歡喜，向佛陀懺悔過錯，並受持三歸（皈依佛、法、僧）。

**◈ 佛說此經已，針毛鬼聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，針毛鬼聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **如來、應、等正覺 (Tathāgata, Arahat, Sammasambuddha)**：佛陀的三種稱號。如來指成就不變真理而來到世間的人；應供指配得上世間最高供養的人；等正覺（正遍知）指對一切真理有全面且正確覺知的人。
2. **愛 (Taṇhā)**：在十二因緣中，指對感官快感的渴求或對生存的執著，是導致苦與輪迴的核心動力。
3. **有 (Bhava)**：十二因緣中的「有」，指業力所導致的生存狀態或存在形式，是出生（生）的前因。
4. **三歸 (Tisarana)**：皈依佛、皈依法、皈依僧，是進入佛教修行的基本門檻，代表將生命交付於覺悟的導師與真理。
5. **尼拘律樹 (Nikujala tree)**：一種分枝極其繁茂且容易交織的樹木，在此經中被用作比喻，形容「貪愛」如何從一個念頭開始，隨後迅速擴散並將心靈層層綑綁。

### [現代實作指南]
本經文揭示了情緒（貪、恚、恐）的根源在於「愛（貪愛/渴求）」。在現代生活中，我們可以透過以下步驟實踐轉念：

1. **識別「藤蔓」的萌芽**：
   當你感到強烈的焦慮（如經文中的「身毛竪」）或憤怒時，不要試圖直接壓制情緒，而是向內觀察：我現在在「渴求」什麼？（例如：渴求對方的認同、渴求事情按我的意願發展、渴求掌控權）。這個「渴求」就是那顆尼拘律樹的種子。

2. **覺察「糾纏」的過程**：
   觀察貪愛如何像藤蔓一樣擴散。例如：從「我想要這個東西」 ➔ 「如果沒有它我就不快樂」 ➔ 「為什麼我還沒有它」 ➔ 「這太不公平了（憤怒）」。當你意識到情緒是在「展轉相拘引」時，你就從被動的受害者變成了主動的觀察者。

3. **切斷依附的慣性**：
   經文中提到「猶如新生兒依倚於乳母」，這比喻我們對某些習慣、想法或人的病態依賴。實作時，試著對自己說：「這個念頭只是暫時的感受，我不需要依附它才能生存。」透過這種「去認同化」的練習，減少對特定結果的執著。

4. **以「知因」產生解脫**：
   當你明確意識到「恐懼與憤怒 ➔ 源於貪愛 ➔ 貪愛源於對無常的抗拒」時，這種對因果的理智認知會產生一種冷靜的力量，使你不再被情緒的風暴捲走，進而達到內心的寧靜。

### [段落大意]
佛陀在摩竭提國逗留時，遭到針毛鬼的挑釁與威脅。佛陀以不動心的威儀與智慧，回答了針毛鬼關於貪婪、憤怒與恐懼之根源的疑問。佛陀指出，一切痛苦皆源於「貪愛」的生起與擴張，其糾纏之深如藤蔓叢林。針毛鬼聞法後深受觸動，從傲慢轉為謙卑，最終皈依三寶。
