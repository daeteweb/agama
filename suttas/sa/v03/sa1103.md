# 雜阿含經 第 1103 經

### [原文]
如是我聞：

一時，佛住王舍城多眾踐蹈曠野中，與六百比丘眾俱，為諸比丘說六觸入處集、六觸集、六觸滅。

時，魔波旬作是念：「今沙門瞿曇住王舍城多眾踐蹈曠野，為六百比丘說六觸入處是集法、是滅法，我今當往，為作留難。」化作壯士，大身勇盛，力能動地，來詣佛所。

彼諸比丘遙見壯士，身大勇盛，見生怖畏，身毛皆竪，共相謂言：「彼為何等，形狀可畏？」

爾時，世尊告諸比丘：「此是惡魔，欲作嬈亂。」

爾時，世尊即說偈言：

「色聲香味觸，　　及第六諸法， 　愛念適可意，　　世間唯有此。 　此是最惡貪，　　能繫著凡夫， 　超越斯等者，　　是佛聖弟子。 　度於魔境界，　　如日無雲翳。」

時，魔波旬作是念：「沙門瞿曇已知我心。」內懷憂慼，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城多眾踐蹈曠野中，與六百比丘眾俱，為諸比丘說六觸入處集、六觸集、六觸滅。**
　　在某一個時間，佛住在王舍城的「多眾踐蹈曠野」中，與六百位比丘（出家修行者）在一起，為這些比丘解說關於六種觸（感官接觸）的入處集（產生原因）、六觸集（聚集過程）以及六觸滅（熄滅之法）。

**◈ 時，魔波旬作是念：「今沙門瞿曇住王舍城多眾踐蹈曠野，為六百比丘說六觸入處是集法、是滅法，我今當往，為作留難。」**
　　那時，魔波旬心裡這麼想：「現在沙門（修行者）瞿曇住在王舍城的多眾踐蹈曠野，正為六百位比丘說明六種觸（感官接觸）是如何聚集而生、又是如何熄滅的，我現在應該過去，為他製造一些困難與阻礙。」

**◈ 化作壯士，大身勇盛，力能動地，來詣佛所。**
　　於是魔波旬幻化成一名壯士，身體強壯勇猛，力量大到能震動大地，來到佛陀所在之處。

**◈ 彼諸比丘遙見壯士，身大勇盛，見生怖畏，身毛皆竪，共相謂言：「彼為何等，形狀可畏？」**
　　那些比丘從遠處看到這名壯士，身體如此巨大且勇猛，心中產生了恐懼，全身汗毛直豎，彼此詢問道：「那個人是誰？形貌如此可畏？」

**◈ 爾時，世尊告諸比丘：「此是惡魔，欲作嬈亂。」**
　　那時，世尊告訴在場的眾位比丘說：「這是一位惡魔，想要製造混亂與騷擾。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即說了一首偈頌：

**◈ 「色聲香味觸，及第六諸法，愛念適可意，世間唯有此。」**
　　「色（物質身體與視覺對象）、聲音、氣味、味道、觸（身體接觸），以及第六種對象（心法/意識對象），若對這些產生愛（貪愛）與念（執著的思考）而感到滿意，世間僅有這些東西（能讓人受苦）。」

**◈ 「此是最惡貪，能繫著凡夫，超越斯等者，是佛聖弟子。」**
　　「這是最惡劣的貪欲，能夠將凡夫（尚未覺悟的眾生）束縛住；而能超越這些感官誘惑的人，才是佛陀真正的聖弟子。」

**◈ 「度於魔境界，如日無雲翳。」**
　　「能跨越魔之境界（煩惱的束縛），就像太陽沒有雲霧遮蔽一樣（光明清淨）。」

**◈ 時，魔波旬作是念：「沙門瞿曇已知我心。」**
　　那時，魔波旬心裡這麼想：「沙門瞿曇已經看穿了我的心思。」

**◈ 內懷憂慼，即沒不現。**
　　他內心感到憂慮與沮喪，隨即消失不見。

### [詞語註解（解釋重要佛教概念）]
1. **六觸 (Phassa)**：指六種感官（眼、耳、鼻、舌、身、意）與其對應的對象（色、聲、香、味、觸、法）接觸，並產生意識的過程。這是苦受產生的關鍵環節。
2. **魔波旬 (Māra)**：在原始佛教中，魔不僅指外在的惡魔，更象徵內心的貪欲、恐懼、執著以及所有阻礙修行解脫的心理力量。
3. **凡夫 (Puthujjana)**：指尚未證得聖果、仍被無明與煩惱所束縛，在輪迴中受苦的普通眾生。
4. **第六諸法 (Dhamma)**：指心意識所能認知的所有對象，包括記憶、概念、思想、情緒等心理現象。

### [現代實作指南]
本經文揭示了「恐懼」與「貪愛」其實是同一種心理機制（魔）的不同面相。我們可以將其轉化為以下生活實踐：

1. **識別「心理壯士」**：
   當你在生活中遇到令人恐懼的壓力、威脅或突如其來的焦慮時（如經中比丘見壯士而怖畏），請意識到這就是你的「內在魔波旬」在幻化形狀。恐懼往往是感官對外部刺激的過度反應。
   ➔ **實作**：當恐懼升起時，對自己說：「這是一個幻化出的壯士，它是我的心理反應，而非事實本身。」

2. **覺察感官的「適可意」**：
   經文提到色聲香味觸及第六法若「適可意」便會產生貪愛。這提醒我們，痛苦並非來自對象本身，而來自我們對對象的「滿意」與「想要持有」的念頭。
   ➔ **實作**：在享受美食或美景時，練習在「滿意」的瞬間加入覺知，觀察心中升起的「愛（貪愛）」與「取（執取）」，意識到這種滿意感是無常的。

3. **建立「無雲之日」的心境**：
   佛陀以「日無雲翳」比喻聖弟子。這意味著修行不是要消滅外部的魔（困難），而是讓內心不再被雲霧（貪、嗔、癡）遮蔽，使智慧之光自然顯現。
   ➔ **實作**：練習「不反應」技巧。面對外界的騷擾或內心的不安時，不急於推開或迎合，而是像觀察雲朵飄過一樣觀察自己的情緒，直到情緒自然熄滅。

### [段落大意]
佛陀在王舍城向比丘講述六觸的生滅法，魔波旬幻化成強壯勇猛的壯士試圖干擾，令比丘們心生恐懼。佛陀揭穿魔之真面目，並以偈頌教導：凡夫被感官對象的貪愛所束縛，唯有超越感官誘惑的人才能如無雲之日般清淨。魔波旬意識到被看穿後，憂心消失。
