# 雜阿含經 第 1177 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「譬如灰河，南岸極熱，多諸利刺，在於闇處，眾多罪人在於河中隨流漂沒。中有一人，不愚不癡，聰明黠慧，樂樂厭苦，樂生厭死，作如是念：『我今何緣在此灰河，南岸極熱，又多利刺，在闇冥處隨流漂沒？我當以手足方便，逆流而上。』漸見小明。其人默念：『今已疾強見此小明。』復運手足，勤加方便，遂見平地，即住於彼，觀察四方，見大石山，不斷不壞，亦不穿穴，即登而上。復見清涼八分之水，所謂冷、美、輕、軟、香、淨，飲時不噎，咽中不閡。飲已安身，即入其中，若浴若飲，離諸惱熱。

「然後復進大山上，見七種華，謂優鉢羅華、鉢曇摩華、拘牟頭華、分陀利華、修揵提華、彌離頭揵提花、阿提目多花。聞花香已，復上石山，見四層階堂，即坐其上；見五柱帳，即入其中，斂身正坐，種種枕褥，散花遍布，莊嚴妙好；而於其中自恣坐臥，涼風四湊，令身安隱，坐高林下，高聲唱言：『灰河眾生諸賢正士！如彼灰河，南岸極熱，多諸利刺，其處闇冥，求出於彼。』

「河中有聞聲者，乘聲問言：『何方得出？從何處出？』其中有言：『汝何須問何處得出？彼喚聲者亦自不知、不見從何而出，彼亦當復在此灰河，南岸極熱，多諸利刺，於闇冥中隨流來下，用問彼為？』

「如是，比丘！我說此譬，今當說義。灰者，謂三惡不善覺。云何三？欲覺、恚覺、害覺。河者，謂三愛——欲愛、色愛、無色愛。南岸極熱者，謂內、外六入處。多諸利刺者，謂五欲功德。闇冥處者，謂無明障閉慧眼。眾多人者，謂愚癡凡夫。流，謂生死河。中有一人不愚不癡者，謂菩薩摩訶薩。手足方便，逆流上者，謂精勤修學。微見小明者，謂得法忍。得平地者，謂持戒。觀四方者，謂見四真諦。大石山者，謂正見。八分水者，謂八聖道。七種花者，謂七覺分。四層堂者，謂四如意足。五柱悵者，謂信等五根。正身坐者，謂無餘涅槃。散花遍布者，謂諸禪、解脫、三昧、正受。自恣坐臥者，謂如來、應、等正覺。四方風吹者，謂四增心見法安樂住。舉聲唱喚者，謂轉法輪。彼有人問『諸賢正士何處去？何處出』者，謂舍利弗、目揵連等諸賢坐比丘。於中有言『汝何所問？彼亦不知不見有所出處，彼亦當復於此灰河，南岸極熱，多諸利刺，於闇冥處隨流來下』者，謂六師等諸邪見輩，所謂富蘭那迦葉、末伽梨瞿舍梨子、散闍耶毘羅胝子、阿耆多枳舍欽婆羅、伽拘羅迦氈延、尼揵連陀闍提弗多羅，及餘邪見輩。如是，比丘！大師為諸聲聞所作，我今已作，汝今當作所作，如前篋毒蛇說。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「譬如灰河，南岸極熱，多諸利刺，在於闇處，眾多罪人在於河中隨流漂沒。**
　　那時，世尊告訴在場的眾位比丘說：「譬如有一條灰色的河流，其南岸極其灼熱，且佈滿了許多尖銳的刺，處在黑暗之中，許多罪人就在這河中隨著水流漂浮沉沒。

**◈ 中有一人，不愚不癡，聰明黠慧，樂樂厭苦，樂生厭死，作如是念：『我今何緣在此灰河，南岸極熱，又多利刺，在闇冥處隨流漂沒？我當以手足方便，逆流而上。』**
　　其中有一個人不愚笨且不癡迷，聰明且機敏，追求快樂而厭惡痛苦，嚮往真正的生命而厭惡生死輪迴，他如此思考：『我現在為什麼會在這條灰河之中？這裡南岸極熱，且多利刺，在黑暗中隨流漂泊，我應該利用手腳的努力與方法，逆著水流向上游游去。』

**◈ 漸見小明。其人默念：『今已疾強見此小明。』**
　　他漸漸看到了一絲微弱的光芒。那人心中默念：『現在我已經努力地看到了這道微光。』

**◈ 復運手足，勤加方便，遂見平地，即住於彼，觀察四方，見大石山，不斷不壞，亦不穿穴，即登而上。**
　　他再次運作手腳，勤奮地採取方法，終於看見了一片平地，便停留在那里。他觀察四周，看見一座巨大的石山，堅固地不斷不壞，也沒有孔洞，於是便攀登而上。

**◈ 復見清涼八分之水，所謂冷、美、輕、軟、香、淨，飲時不噎，咽中不閡。飲已安身，即入其中，若浴若飲，離諸惱熱。**
　　接著他看見了具有八種特質的清涼之水，也就是冷、美、輕、軟、香、淨，飲用時不會哽噎，喉嚨也不會阻塞。飲水後安定身心，隨即進入水中，時而沐浴時而飲用，遠離了所有煩惱與灼熱。

**◈ 「然後復進大山上，見七種華，謂優鉢羅華、鉢曇摩華、拘牟頭華、分陀利華、修揵提華、彌離頭揵提花、阿提目多花。**
　　然後他繼續前進到大山上，看見了七種花，分別是優鉢羅花、鉢曇摩花、拘牟頭花、分陀利花、修揵提花、彌離頭揵提花以及阿提目多花。

**◈ 聞花香已，復上石山，見四層階堂，即坐其上；見五柱帳，即入其中，斂身正坐，種種枕褥，散花遍布，莊嚴妙好；**
　　在聞到花香之後，他再次登上石山，看見一座四層的階梯殿堂，便坐在其上；又看見一座由五根柱子撐起的帳幕，便進入其中，收斂身心端正地坐著，周圍有各種枕褥，散花遍布，莊嚴且美好。

**◈ 而於其中自恣坐臥，涼風四湊，令身安隱，坐高林下，高聲唱言：『灰河眾生諸賢正士！如彼灰河，南岸極熱，多諸利刺，其處闇冥，求出於彼。』**
　　他在帳幕中自在地坐臥，涼風從四方吹來，使身體安穩寧靜。他坐在高大的森林之下，高聲呼喊道：『灰河中的眾生，各位賢能的正士們！就像那條灰河，南岸極熱且多利刺，處在黑暗之中，請尋找方法從那裡脫離出來！』

**◈ 「河中有聞聲者，乘聲問言：『何方得出？從何處出？』**
　　河中有人聽到了聲音，順著聲音的方向詢問道：『從哪個方向能出去？從何處可以脫離？』

**◈ 其中有言：『汝何須問何處得出？彼喚聲者亦自不知、不見從何而出，彼亦當復在此灰河，南岸極熱，多諸利刺，於闇冥中隨流來下，用問彼為？』**
　　但其中也有人說：『你為什麼要問從何處出去？那個呼喊的人自己也不知道、沒看見是從哪裡出來的，他以前一定也在這條灰河中，在南岸極熱、多利刺的黑暗中隨流而下，詢問他有什麼用？』

**◈ 「如是，比丘！我說此譬，今當說義。灰者，謂三惡不善覺。云何三？欲覺、恚覺、害覺。**
　　比丘們！我說這個譬喻，現在為你們解說其中的義理。所謂的『灰』，是指三種不善的覺知（對對象的認知）：分別是貪欲的覺知、嗔恚的覺知以及害人的覺知。

**◈ 河者，謂三愛——欲愛、色愛、無色愛。**
　　所謂的『河』，是指三種貪愛（對生存的執著）：分別是對感官慾望的愛、對有色境界的愛以及對無色境界的愛。

**◈ 南岸極熱者，謂內、外六入處。**
　　所謂的『南岸極熱』，是指內在與外在的六入處（六種感官及其對象，如眼與色、耳與聲等）。

**◈ 多諸利刺者，謂五欲功德。**
　　所謂的『多諸利刺』，是指五欲（財、色、名、食、睡）所帶來的假象功德。

**◈ 闇冥處者，謂無明障閉慧眼。**
　　所謂的『闇冥處』，是指無明（對真理的無知）遮蔽了智慧之眼。

**◈ 眾多人者，謂愚癡凡夫。**
　　所謂的『眾多人』，是指愚癡的凡夫。

**◈ 流，謂生死河。**
　　所謂的『流』，是指生死輪迴的河流。

**◈ 中有一人不愚不癡者，謂菩薩摩訶薩。**
　　其中那個不愚不癡的人，是指菩薩摩訶薩（追求覺悟的大乘修行者）。

**◈ 手足方便，逆流上者，謂精勤修學。**
　　利用手腳的方法逆流而上，是指精進地修學佛法。

**◈ 微見小明者，謂得法忍。**
　　微弱地看到光芒，是指獲得了法忍（對佛法真理的初步體悟與忍耐）。

**◈ 得平地者，謂持戒。**
　　到達平地，是指持守戒律。

**◈ 觀四方者，謂見四真諦。**
　　觀察四方，是指見到了四聖諦（苦、集、滅、道）。

**◈ 大石山者，謂正見。**
　　巨大的石山，是指正見（正確的見地）。

**◈ 八分水者，謂八聖道。**
　　八分之水，是指八正道（正見、正思惟、正語、正業、正命、正精進、正念、正定）。

**◈ 七種花者，謂七覺分。**
　　七種花，是指七覺分（修行覺悟的七個階段）。

**◈ 四層堂者，謂四如意足。**
　　四層殿堂，是指四如意足（達成目標的四種心理能力）。

**◈ 五柱悵者，謂信等五根。**
　　五柱帳幕，是指信、精進、念、定、慧這五根（修行的五種基礎能力）。

**◈ 正身坐者，謂無餘涅槃。**
　　端正身心而坐，是指進入無餘涅槃（超越生死、絕對寧靜的解脫境界）。

**◈ 散花遍布者，謂諸禪、解脫、三昧、正受。**
　　散花遍布，是指各種禪定、解脫、三昧（專注之定）與正受。

**◈ 自恣坐臥者，謂如來、應、等正覺。**
　　自在地坐臥，是指如來、應供、等正覺（佛陀的境界）。

**◈ 四方風吹者，謂四增心見法安樂住。**
　　四方吹來的涼風，是指四種增長心靈的見地，使人安樂地住在法中。

**◈ 舉聲唱喚者，謂轉法輪。**
　　高聲呼喊，是指轉法輪（宣說佛法）。

**◈ 彼有人問『諸賢正士何處去？何處出』者，謂舍利弗、目揵連等諸賢坐比丘。**
　　那些詢問『賢正士們要去何處？從何處出』的人，是指舍利弗、目揵連等優秀的比丘。

**◈ 於中有言『汝何所問？彼亦不知不見有所出處，彼亦當復於此灰河，南岸極熱，多諸利刺，於闇冥處隨流來下』者，謂六師等諸邪見輩，所謂富蘭那迦葉、末伽梨瞿舍梨子、散闍耶毘羅胝子、阿耆多枳舍欽婆羅、伽拘羅迦氈延、尼揵連陀闍提弗多羅，及餘邪見輩。**
　　而那些說『你問什麼？他也不知道從哪裡出來，他以前一定也隨流而下』的人，是指六師等持有錯誤見解的人，包括富蘭那迦葉、末伽梨瞿舍梨子、散闍耶毘羅胝子、阿耆多枳舍欽婆羅、伽拘羅迦氈延、尼揵連陀闍提弗多羅以及其他邪見者。

**◈ 如是，比丘！大師為諸聲聞所作，我今已作，汝今當作所作，如前篋毒蛇說。」**
　　比丘們！過去的大導師為聲聞弟子們所做的事，我現在已經做完了，你們現在也應該去做該做的事，就像之前說過的『篋中毒蛇』的比喻一樣。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **六入處 (Saḷāyatana)**：指內六入（眼、耳、鼻、舌、身、意）與外六入（色、聲、香、味、觸、法）。本經將其比作「南岸極熱」，意指感官與對象的接觸是痛苦與煩惱的源頭。
2. **三愛 (Taṇhā)**：指欲愛（對感官快感的渴求）、色愛（對色界定境的渴求）、無色愛（對無色界定境的渴求）。這是導致輪迴的核心動力。
3. **法忍 (Dhammakhanti)**：對佛法真理的初步領悟，並能在面對困難或質疑時，堅定地忍耐並持有此見地。
4. **七覺分 (Satta Bojjhaṅga)**：覺悟的七個要素，包括：念、擇法、精進、喜、輕安、定、捨。
5. **四如意足 (Iddhipāda)**：達成任何目標所需的四種心理條件：欲（強烈的願望）、精進、心（專注）、審（審慎觀察）。
6. **五根 (Indriya)**：修行者必須培養的五種能力：信、精進、念、定、慧。

### [現代實作指南]
本經透過「灰河」的譬喻，將修行過程具象化為「逆流而上」的過程。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **識別「灰河」狀態 (覺察)**：
   - 當你感到內心焦慮、憤怒或強烈渴望某物時，意識到這就是「灰河」的灼熱感與利刺。
   - 練習將「欲覺（貪）」、「恚覺（嗔）」、「害覺（惡）」標記出來。例如：當你對同事感到憤怒時，在心中默念「這是恚覺」，將自己從情緒的隨流中抽離。

2. **採取「逆流」行動 (轉念)**：
   - 隨流是指：憤怒 → 爭吵 → 痛苦。
   - 逆流是指：憤怒 → 覺察 → 停止反應 → 觀察憤怒的無常。
   - 這種「不隨之流」的意識，就是本經中「手足方便」的現代實踐。

3. **建立「石山」與「平地」 (穩定)**：
   - **平地（持戒）**：在生活中設定簡單的底線（如：不說謊、不傷害他人），這能為心靈提供一個穩定的基礎，不再隨波逐流。
   - **石山（正見）**：時刻提醒自己「一切現象皆是無常且無我」，將此見地作為心中的定海神針，使其「不斷不壞」。

4. **追求「清涼水」 (正念)**：
   - 每天撥出 10-15 分鐘進行正念呼吸，體驗那種「冷、美、輕、軟」的心理狀態，用以對沖生活中的「惱熱」。

### [段落大意]
