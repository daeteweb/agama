# 雜阿含經 第 1251 經

### [原文]
如是我聞：

一時，佛在拘薩羅人間遊行，至那楞伽羅聚落……如上廣說，乃至彼彼所起求利。

佛告那提迦：「我見聚落邊有精舍，有比丘坐禪。我見已，作如是念：『今此尊者聚落人，此或沙彌，來往聲響作亂，障其禪思，覺其正受，於不到欲到、不獲欲獲、不證欲證而作留難。』那提迦！我不喜彼比丘住聚落精舍。

「那提迦！我見比丘住空閑處，仰臥吁咄。我見是已，而作是念：『令彼比丘覺寤睡眠，思空閑想。』那提迦！我亦不喜如是比丘住空閑處。

「那提迦！我復見比丘住空閑處，搖身坐睡。見已，作是念：『今此比丘於睡覺寤，不定得定，定心者得解脫。』是故，那提迦！我不喜如是比丘住空閑處。

「那提迦！我復見比丘住空閑處，端坐正受。我見已，作如是念：『令此比丘不解脫者，疾得解脫；已解脫者，令自防護，使不退失。』那提迦！我喜如是比丘住空閑處。

「那提迦！我復見比丘住空閑處，彼於後時，遠離空閑處集捨床臥具，還入聚落受床臥具。那提迦！我亦不喜如是比丘還入聚落。

「復次，那提迦！我見比丘住聚落精舍，名聞大德能感財利、衣被、飲食、湯藥、眾具。彼於後時，集捨利養聚落床座，至於空閑，床臥安止。那提迦！我喜如是比丘集捨利養聚落床臥，住於空閑。那提迦！比丘當如是學。」

佛說此經已，那提迦比丘歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛在拘薩羅人間遊行，至那楞伽羅聚落……如上廣說，乃至彼彼所起求利。**
　　在某一個時間，佛在拘薩羅國的人間遊行，來到那楞伽羅聚落……（中間省略部分）直到提到他們各自追求利益的情況。

**◈ 佛告那提迦：「我見聚落邊有精舍，有比丘坐禪。**
　　佛告訴那提迦：「我看見在聚落邊緣有一座精舍，有一位比丘在其中坐禪。

**◈ 我見已，作如是念：『今此尊者聚落人，此或沙彌，來往聲響作亂，障其禪思，覺其正受，於不到欲到、不獲欲獲、不證欲證而作留難。』**
　　我看見之後，心裡這樣想：『這位尊者住在聚落中，這裡的村民或沙彌來往的嘈雜聲響會造成干擾，阻礙他的禪定思考，使他從正受（正確的禪定專注狀態）中覺醒，導致他對於尚未到達的境界無法到達、尚未獲得的無法獲得、尚未證悟的無法證悟，而造成修行上的困難。』

**◈ 那提迦！我不喜彼比丘住聚落精舍。**
　　那提迦！我不贊同比丘住在聚落的精舍中。

**◈ 「那提迦！我見比丘住空閑處，仰臥吁咄。**
　　「那提迦！我看見有比丘住在空閑清靜之處，卻仰著身子嘆息懶散。

**◈ 我見是已，而作是念：『令彼比丘覺寤睡眠，思空閑想。』**
　　我看見之後，心裡這樣想：『應該讓這位比丘從睡眠中覺醒，真正思考空閑清靜的意義。』

**◈ 那提迦！我亦不喜如是比丘住空閑處。**
　　那提迦！對於這樣住在空閑處的比丘，我同樣不贊同。

**◈ 「那提迦！我復見比丘住空閑處，搖身坐睡。**
　　「那提迦！我又看見有比丘住在空閑處，卻在坐著時搖晃身體陷入睡眠。

**◈ 見已，作是念：『今此比丘於睡覺寤，不定得定，定心者得解脫。』**
　　看見之後，我心裡這樣想：『這位比丘應該從睡眠中覺醒，在不定的狀態中獲得定（心不散亂的專注力），因為唯有心定的人才能獲得解脫（從苦與輪迴中完全解脫）。』

**◈ 是故，那提迦！我不喜如是比丘住空閑處。**
　　因此，那提迦！我不贊同這樣住在空閑處的比丘。

**◈ 「那提迦！我復見比丘住空閑處，端坐正受。**
　　「那提迦！我又看見有比丘住在空閑處，端正地坐著進入正受（正確的禪定專注狀態）。

**◈ 我見已，作如是念：『令此比丘不解脫者，疾得解脫；已解脫者，令自防護，使不退失。』**
　　我看見之後，心裡這樣想：『願這位比丘若尚未解脫，能迅速獲得解脫（從苦與輪迴中完全解脫）；若已經解脫，能自我防護，使覺悟的境界不退失。』

**◈ 那提迦！我喜如是比丘住空閑處。**
　　那提迦！我贊同這樣住在空閑處的比丘。

**◈ 「那提迦！我復見比丘住空閑處，彼於後時，遠離空閑處集捨床臥具，還入聚落受床臥具。**
　　「那提迦！我又看見有比丘住在空閑處，但後來卻放棄了空閑處的床臥具，重新回到聚落中接受床臥具。

**◈ 那提迦！我亦不喜如是比丘還入聚落。**
　　那提迦！對於這樣重新回到聚落的比丘，我同樣不贊同。

**◈ 「復次，那提迦！我見比丘住聚落精舍，名聞大德能感財利、衣被、飲食、湯藥、眾具。**
　　「另外，那提迦！我看見有比丘住在聚落精舍，因為名聲顯赫且德行高尚，能獲得財物、衣服、飲食、藥品以及各種生活用品的供養。

**◈ 彼於後時，集捨利養聚落床座，至於空閑，床臥安止。**
　　但他在之後的時間裡，放棄了聚落中的利養與床座，前往空閑清靜之處安頓床臥。

**◈ 那提迦！我喜如是比丘集捨利養聚落床臥，住於空閑。**
　　那提迦！我贊同這樣放棄聚落利養而住在空閑處的比丘。

**◈ 那提迦！比丘當如是學。」**
　　那提迦！比丘應該這樣學習。」

**◈ 佛說此經已，那提迦比丘歡喜隨喜，作禮而去。**
　　佛陀講完這部經後，那提迦比丘感到歡喜並隨喜佛陀的教導，向佛陀行禮後才離開。

### [詞語註解]
1. **正受 (Samma-samādhi)**：指正確的禪定狀態，心意識完全集中於單一對象而不散亂，是進入深層覺悟的必要基礎。
2. **空閑處 (Viveka)**：指遠離喧囂、人群與物質干擾的清靜環境，旨在減少外在刺激以利於內在觀察。
3. **解脫 (Vimokkha)**：指心靈從貪、嗔、痴等煩惱以及輪迴的苦痛中完全解脫，達到絕對的自由與寧靜。
4. **定 (Samādhi)**：指心不散亂、專注於當下對象的能力。
5. **利養 (Dakshina)**：指世俗對修行者的供養，包括物質財富、名聲與尊重。

### [現代實作指南]
本經文探討的是「環境」與「心態」對修行成效的影響，可轉化為以下現代實踐技巧：

1. **環境管理 ➔ 建立「神聖空間」**：
   - 佛陀不贊同住在嘈雜聚落，是因為外在噪音會中斷深層專注。在現代生活中，我們應為自己設定一個「無干擾時段」或「專屬空間」（如關掉手機通知的30分鐘），避免在碎片化的環境中追求深層的心理平靜。

2. **警惕「偽清靜」 ➔ 區分孤獨與懶散**：
   - 佛陀批評住在空閑處卻仰臥嘆息或坐睡的比丘。這提醒我們，單純地「遠離人群」並不等於「修行」。若將孤獨變成逃避責任的藉口或懶散的溫床，這種清靜反而成了障礙。實作時應自問：我現在的安靜是在「覺察」還是在「昏沉」？

3. **目標導向的專注 ➔ 從「不定」走向「定」**：
   - 練習將注意力從外界的雜訊（如社交媒體、他人評價）移回內在。當發現心神不寧（不定）時，不要強求，而是溫柔地將意識帶回呼吸或當下的感受，直到心能安定（得定）。

4. **價值觀的轉移 ➔ 從「外在認可」轉向「內在自由」**：
   - 經文最後讚揚那些放棄名聲利養而選擇清靜處的比丘。這是一種「斷捨離」的實踐。在生活中，嘗試減少對他人讚美、職稱或物質標籤的依賴，將追求「被看見」的能量轉化為追求「看清真相」的能量。

### [段落大意]
佛陀透過對比四種比丘的狀態，說明修行環境與心態的關係：住在嘈雜聚落會受干擾而難以證悟；住在清靜處卻懶散或昏睡則毫無進展；唯有在清靜處端正修行、進入正受者才能迅速解脫。佛陀鼓勵比丘應捨棄世俗的利養與名聲，追求真正的內在清靜。
