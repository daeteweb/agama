# 雜阿含經 第 1163 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園……如上說。差別者，唯說異偈言：

「老死之所壞，　　身及所受滅， 
唯有惠施福，　　為隨己資糧。 
依於善攝護，　　及修禪功德， 
隨力而行施，　　錢財及飲食。 
於群則眠覺，　　非為空自活。」

佛說此經已，彼婆羅門聞佛所說，歡喜隨喜，作禮而去。

---

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園……如上說。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園……（前文內容）如同前面所說的一樣。

**◈ 差別者，唯說異偈言：**
　　不同之處在於，僅僅說了不同的偈頌（詩句）如下：

**◈ 「老死之所壞，身及所受滅，**
　　「當被老死（十二因緣中最後的出生、衰老與死亡）所摧毀時，色（物質身體）以及受（感受）都會隨之滅盡，

**◈ 唯有惠施福，為隨己資糧。**
　　唯有透過布施他人所獲得的福德，能成為跟隨自己、在輪迴中作為依託的資糧。

**◈ 依於善攝護，及修禪功德，**
　　依賴於對身心的良好攝護（正念與自律），以及修行禪定所獲得的功德，

**◈ 隨力而行施，錢財及飲食。**
　　根據自己的能力去實踐布施，將金錢以及食物分享給他人。

**◈ 於群則眠覺，非為空自活。」**
　　在眾生之中，無論是在睡眠或覺醒之間（指在世間生活時始終保持覺知），而不是虛度光陰、徒然地生存。」

**◈ 佛說此經已，彼婆羅門聞佛所說，歡喜隨喜，作禮而去。**
　　佛陀講完這部經後，那位婆羅門聽聞佛陀所說的教法，心中感到法喜並隨喜贊嘆，向佛陀行禮後才離開。

### [詞語註解]
1. **老死 (Jarāmaraṇa)**：十二因緣的最後一環，指生物必然經歷的衰老與死亡過程，是苦（Dukkha）的集中體現。
2. **色 (Rūpa)**：五蘊中的第一蘊，指構成身體的物質要素。
3. **受 (Vedanā)**：五蘊中的第二蘊，指對感官接觸後產生的苦、樂、捨（中性）的感受。
4. **布施 (Dāna)**：一種修行方法，透過捨棄對物質的執著，將財產、知識或無畏（安全感）分享給他人，以累積福德。
5. **攝護 (Saṃvara)**：指對感官（眼、耳、鼻、舌、身、意）的克制與守護，防止煩惱（Kilesa）隨之而起。

### [現代實作指南]
本經文強調在面對必然的死亡（無常）時，應如何經營生命。可轉化為以下實踐技巧：

1. **建立「生命資糧」意識**：
   ➔ 意識到物質財產無法隨身離開，但「善行」與「覺悟」是唯一能跨越時間的資產。練習將「擁有」轉化為「分享」，在幫助他人時體會自我執著的減少。

2. **實踐「隨力而行」**：
   ➔ 布施不在於金額大小，而在於「捨」的心念。在生活中嘗試小規模的分享（如分享知識、給予鼓勵或適度的捐助），重點在於不強求、不攀比，僅根據現有能力而為。

3. **在日常中「眠覺」**：
   ➔ 練習「正念生活」。在睡眠、起床、工作、社交等日常瑣事中，保持一種「我知道我在做什麼」的覺知狀態，避免陷入自動導航般的無意識生存（非為空自活）。

4. **身心攝護練習**：
   ➔ 當強烈的情緒（受）產生時，不要立即反應，而是先「攝護」住它。試著觀察感受的升起與滅去，不被其牽著走，這便是將「修禪功德」應用於生活的具體方式。

---

### [段落大意]
本段經文透過偈頌提醒眾生，物質身體與感受終將被老死摧毀，唯有布施的福德與禪定的功德能成為生命的真實資糧。佛陀勉勵修行者應在能力範圍內行善，並在世間生活中保持覺知，避免徒然虛度光陰。
