# 雜阿含經 第 1255 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「譬如有人有匕手劍，其刃廣利。有健士夫言：『我能以手以拳椎打汝劍，令其摧碎。』諸比丘！彼健士夫當能以手以拳椎打彼劍，令摧碎不？」

比丘白佛：「不能。世尊！彼匕手劍其刃廣利，非彼士夫能以手以拳椎打碎折，正足自困。

「如是，比丘！若沙門、婆羅門下至如[(殼-一)/牛]牛乳頃，於一切眾生修習慈心，若有諸惡鬼神欲往伺求其短，不能得其間便，正可反自傷耳。是故，諸比丘！當如是學：『數數下至如[(殼-一)/牛]牛乳頃，修習慈心。』」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「譬如有人有匕手劍，其刃廣利。**
　　那時，世尊告訴在場的眾位比丘說：「比方說，有人擁有一把短劍，其劍刃非常寬且鋒利。

**◈ 有健士夫言：『我能以手以拳椎打汝劍，令其摧碎。』**
　　有個強壯的男子說：『我可以用手用拳頭去擊打你的劍，讓它破碎。』

**◈ 諸比丘！彼健士夫當能以手以拳椎打彼劍，令摧碎不？」**
　　諸位比丘！那個強壯的男子能用手用拳頭擊碎那把劍嗎？」

**◈ 比丘白佛：「不能。世尊！彼匕手劍其刃廣利，非彼士夫能以手以拳椎打碎折，正足自困。」**
　　比丘們對佛陀恭敬地說道：「不能。世尊！那把短劍的刃口寬且鋒利，那個男子不可能用手或拳頭將其擊碎，反而只會讓自己受傷。」

**◈ 「如是，比丘！若沙門、婆羅門下至如[(殼-一)/牛]牛乳頃，於一切眾生修習慈心，**
　　「比丘們，就像這樣！如果沙門（出家修行者）或婆羅門，哪怕僅僅在極短的時間（如擠出一滴牛奶的時間）內，對所有眾生進行修行（Bhāvanā），培養慈心（對眾生生起願其得樂的慈悲心），

**◈ 若有諸惡鬼神欲往伺求其短，不能得其間便，正可反自傷耳。**
　　如果有各種惡鬼神想要伺機尋找他的缺點或弱點，將無法找到任何機會，反而會讓自己受傷。

**◈ 是故，諸比丘！當如是學：『數數下至如[(殼-一)/牛]牛乳頃，修習慈心。』」**
　　因此，諸位比丘！應該這樣學習：『經常地，哪怕僅僅在極短的時間（如擠出一滴牛奶的時間）內，修行（Bhāvanā）慈心。』」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **慈心 (Mettā)**：一種願他人獲得快樂、安適的無條件善意。在原始佛教中，慈心不僅是情感，更是一種透過修行（Bhāvanā）而建立的心理狀態，能產生強大的保護力。
2. **牛乳頃**：比喻極其短暫的時間。在古印度文獻中，常用擠出一滴牛奶的時間來形容瞬間或極短的時段。
3. **沙門 (Samaṇa)**：原指捨棄世俗生活、追求解脫的修行者，泛指當時印度各種出家修行團體的成員。
4. **伺求其短**：指惡意的意識或外在負面能量尋找對方的心理漏洞（如憤怒、恐懼、貪婪）以進行干擾或傷害。

### [現代實作指南]
本經文將「慈心」比喻為「鋒利的劍」，揭示了善意在心理層面具有強大的「防禦性」。在現代生活中，我們可以將其轉化為以下實踐技巧：

1. **建立「善意護盾」 (The Kindness Shield)**：
   ➔ 當你感覺到周圍環境充滿敵意，或面對一個讓你感到壓力、負能量沉重的人時，不要試圖用對抗（如憤怒或辯論）來回應。
   ➔ 嘗試在心中快速地對對方發出一個願望：「願你平安，願你遠離痛苦」。
   ➔ 這種轉念能迅速改變你的心理頻率，使你從「受害者」或「對抗者」轉變為「祝福者」，從而切斷負面情緒的連結，讓對方的惡意因找不到對應的「憤怒接點」而無法對你造成實質心理傷害。

2. **微量修行法 (Micro-Practice)**：
   ➔ 經文提到「牛乳頃」，強調修行不需要長時間的靜坐，而是在碎片時間的即時轉念。
   ➔ 實作：在等紅燈、電梯上升、或開啟電子郵件的 3 到 5 秒內，對隨機的一個對象（如路人、同事）心中默念一次慈心願文。
   ➔ 透過高頻率、短時間的微量練習，將慈心內化為一種本能的心理反應，而非刻意的努力。

3. **認知轉念 ➔ 從「防禦」到「轉化」**：
   ➔ 意識到「惡意」就像用拳頭擊打鋒利劍刃，傷害的最終承接者是發起惡意的人。
   ➔ 當你保持慈心時，你不需要主動攻擊，你的心理狀態本身就是一種保護。這能減少內心的恐懼感，增加面對衝突時的定力。

### [段落大意]
佛陀以「強壯男子無法用拳頭擊碎鋒利短劍，反而會自傷」為喻，說明修習慈心（Mettā）能產生強大的保護力。修行者即便在極短的時間內對眾生生起慈心，也能使惡意之靈或負面能量無法尋得漏洞而受挫，反而反噬自身。因此，佛陀勉勵比丘應經常練習短時間的慈心修行。
