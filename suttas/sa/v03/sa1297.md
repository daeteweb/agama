# 雜阿含經 第 1297 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「云何數所數？　　云何數不隱？ 　云何數中數？　　云何說言說？」

爾時，世尊說偈答言：

「佛法難測量，　　二流不顯現， 　若彼名及色，　　滅盡悉無餘， 　是名數所數，　　彼數不隱藏， 　是彼數中數，　　是則說名數。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

於時，天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出色的天子，在深夜時分來到佛陀這裡，頂禮佛陀的足下，然後退到一旁坐下。他的身體發出強烈的光芒，照亮了整個祇樹給孤獨園。

**◈ 時，彼天子說偈問佛：「云何數所數？云何數不隱？云何數中數？云何說言說？」**
　　那時，這位天子用詩偈的形式請問佛陀：「如何衡量那些可被衡量的事物？如何衡量那些不被遮蔽的真理？如何在衡量之中找到真正的衡量？又該如何用語言來描述這種衡量？」

**◈ 爾時，世尊說偈答言：「佛法難測量，二流不顯現，若彼名及色，滅盡悉無餘，是名數所數，彼數不隱藏，是彼數中數，是則說名數。」**
　　那時，世尊用詩偈回答說：「佛法（真理）是難以用常人的邏輯或數量來測量的，當兩種極端的流轉（對立狀態）不再顯現；如果那些名相（概念認知）以及色（物質身體）完全滅盡，沒有任何殘餘，這才叫做衡量了可衡量之物，這才是毫不隱藏的衡量，這才是衡量中的核心，這才是對『衡量』真正的描述。」

**◈ 時，彼天子復說偈言：「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　這時，那位天子再次說道：「我長期以來觀察那些婆羅門，他們最終達到了涅槃（超越生死、絕對寧靜的解脫境界），所有的恐懼都已過去，永遠超越了世間的貪愛與執著。」

**◈ 於時，天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，天子聽聞佛陀所說的教法，心生法喜並隨之歡喜，頂禮佛陀的足下，隨即消失不再出現。

### [詞語註解]
1. **色 (Rūpa)**：指物質身體或所有物質現象，是構成五蘊（身心五要素）中的物質部分。
2. **涅槃 (Nibbāna)**：超越生死、絕對寧靜的解脫境界，指熄滅了貪、嗔、痴等煩惱後的最終狀態。
3. **佛法 (Buddha-dhamma)**：佛陀所覺悟的宇宙真理以及導向解脫的教法。
4. **名 (Nāma)**：指心理上的名相、概念、定義或認知，與「色」相對，合稱「名色」。

### [現代實作指南]
本經文探討的是「衡量」與「超越衡量」的關係。在現代生活中，我們習慣用數量、標籤、定義（即經文中的「數」與「名」）來衡量成功、幸福或自我價值。

**實作技巧：從「衡量」轉向「覺知」**
1. **識別「名色」的束縛**：當你感到焦慮時，觀察你是否在用某種「名相」（例如：我是一個失敗者、這件事必須成功）來衡量自己。意識到這些標籤（名）和物質條件（色）只是暫時的組合。
2. **練習「不測量」的靜心**：每天抽出十分鐘，嘗試不對自己的感受進行「評分」或「定義」。不要說「這感覺很糟糕（負分）」或「這感覺很好（正分）」，僅僅觀察感受的升起與滅去。
3. **超越對立的「二流」**：當你陷入「對與錯」、「好與壞」的二元對立爭論時，提醒自己這正是「二流顯現」的狀態。試著跳出對立，觀察這場爭論背後的執著，從而達到內心的寧靜。
4. **放下對結果的執取**：效法經文中「超世恩愛」的精神，在努力工作或經營關係時，將焦點放在「過程的正念」而非「結果的衡量」，減少因追求特定數值（金錢、名聲、認可）而產生的恐懼。

### [段落大意]
一位天子請問佛陀關於真理如何衡量與描述的問題。佛陀回答指出，佛法超越量化，唯有當物質身體（色）與概念認知（名）完全滅盡，才能真正體會到不被遮蔽的真理。天子隨即感嘆涅槃境界能超越恐懼與貪愛，最後在法喜中消失。
