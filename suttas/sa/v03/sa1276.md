# 雜阿含經 第 1276 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有一天子，容色絕妙，來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。

時，彼天子而說偈言：

「愚癡人所行，　　不合於黠慧， 
　自所行惡行，　　為自惡知識。 
　所造眾惡行，　　終獲苦果報。」

爾時，世尊說偈答言：

「既作不善業，　　終則受諸惱， 
　造業雖歡喜，　　啼泣受其報。 
　造諸善業者，　　終則不熱惱， 
　歡喜而造業，　　安樂受其報。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 
　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 時，有一天子，容色絕妙，來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。**
　　那時，有一位天界的子息，容貌極其絕妙，來到佛陀所在之處，頂禮佛陀的足下，他的身體散發出各種光明，照亮了整個祇樹給孤獨園。

**◈ 時，彼天子而說偈言：**
　　那時，這位天子用偈頌說道：

**◈ 「愚癡人所行，不合於黠慧，**
　　「那些被無明遮蔽而愚癡的人所採取的身心行為，並不符合睿智的覺知。

**◈ 自所行惡行，為自惡知識。**
　　他們自己所造的惡行，反而成了他們自己最糟糕的陪伴與指引（惡知識）。

**◈ 所造眾惡行，終獲苦果報。」**
　　他們所造的種種惡業，最終必然會獲得痛苦的業報（Vipāka）。」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「既作不善業，終則受諸惱，**
　　「既然造了不善的業（Kamma），最終必然會承受種種煩惱與痛苦。

**◈ 造業雖歡喜，啼泣受其報。**
　　即使在造惡業的當下感到歡快，但在承受其業報時卻會啼哭流淚。

**◈ 造諸善業者，終則不熱惱，**
　　而那些造種種善業的人，最終不會感到心中燥熱煩惱。

**◈ 歡喜而造業，安樂受其報。」**
　　他們在歡喜中造善業，最終在安樂中承受其業報。」

**◈ 時，彼天子復說偈言：**
　　那時，這位天子再次用偈頌說道：

**◈ 「久見婆羅門，逮得般涅槃，**
　　「我長期觀察那些婆羅門，看到他們最終達到了般涅槃（超越生死、絕對寧靜的解脫境界）。

**◈ 一切怖已過，永超世恩愛。」**
　　所有的恐懼都已過去，永遠超越了世間的愛欲與執著。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，這位天子聽聞佛陀所說的教法，心生法喜並隨之歡喜，頂禮佛陀的足下，隨即消失不再出現。

### [詞語註解]
1. **業 (Kamma)**：指有意識的造作（身、口、意），是導致未來果報的根本原因。
2. **業報 (Vipāka)**：業所產生的結果。善業導致樂報，惡業導致苦報。
3. **惡知識 (Papamitta)**：指會誘導他人走向錯誤道路、增加煩惱的對象或環境。在本經中，指人自身的惡行成為了最壞的導師。
4. **涅槃 (Nibbāna)**：貪、嗔、癡三毒完全熄滅的狀態，是超越輪迴、絕對寧靜的解脫境界。
5. **世恩愛**：指對世間物質、情感、自我身分的執著與貪愛，這是導致輪迴與痛苦的主因。

### [現代實作指南]
本經揭示了「行為與結果」的時間差，以及「自我誘導」的心理機制。在現代生活中可採取以下實踐：

1. **覺察「即時快感」的陷阱**：
   許多惡行（如欺瞞、憤怒地攻擊他人）在當下能提供一種權力感或快感（造業雖歡喜），但這種快感是暫時的。當我們在衝動時，應提醒自己：現在的快感 ➔ 未來的啼泣。將注意力從「當下的滿足」移向「長遠的業報」。

2. **重新定義「知識（陪伴）」**：
   經文提到「惡行成為惡知識」。這意味著我們的習慣會塑造我們的性格。如果你習慣於抱怨與憤怒，這些行為本身就成了你的「惡友」，不斷強化你的負面心態。實作方法是：刻意建立「善知識」的習慣（如每日感恩、誠實對待他人），讓正向的行為成為指引自己的導師。

3. **練習「不熱惱」的心理狀態**：
   善業的特質是「不熱惱」。當我們採取利他或誠實的行動時，內心會產生一種清涼、坦蕩的感覺。在面對選擇時，可以問自己：「這個決定會讓我在未來感到不安（熱惱），還是感到坦然（安樂）？」選擇後者即是實踐正法。

### 段落大意 
一位天子到訪佛陀，透過偈頌討論業報法則。天子指出愚癡者以惡行作為自己的惡友，最終必受苦報；佛陀則補充說明，造惡業雖在當下可能歡喜，但果報卻是啼泣；而造善業則能帶來長久的安樂。最後天子讚嘆達到涅槃者已超越恐懼與世間愛欲，隨後法喜消失。
