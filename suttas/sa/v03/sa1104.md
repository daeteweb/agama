# 雜阿含經 第 1104 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

爾時，世尊告諸比丘：「若能受持七種受者，以是因緣得生天帝釋處。謂天帝釋本為人時，供養父母，及家諸尊長，和顏軟語，不惡口，不兩舌，常真實言；於慳悋世間，雖在居家而不慳惜，行解脫施，勤施，常樂行施，施會供養，等施一切。」爾時，世尊即說偈言：

「供養於父母，　　及家之尊長， 　柔和恭遜辭，　　離麁言兩舌。 　調伏慳悋心，　　常修真實語， 　彼三十三天，　　見行七法者。 　咸各作是言，　　當來生此天。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若能受持七種受者，以是因緣得生天帝釋處。**
　　「如果能夠接納並實踐七種行為（受），以此因緣（條件與原因而生）而生，就能夠在未來出生在天帝釋（帝釋天）的境界。

**◈ 謂天帝釋本為人時，供養父母，及家諸尊長，**
　　也就是說，天帝釋在原本作為人類的時候，孝養父母以及家中所有年長的長輩。

**◈ 和顏軟語，不惡口，不兩舌，常真實言；**
　　臉色溫和且言語柔和，不說惡毒的話，不說挑撥離間的兩舌之詞，並且經常說真實的話。

**◈ 於慳悋世間，雖在居家而不慳惜，**
　　在這個充滿吝嗇與貪婪的世間，雖然生活在居家環境中，卻不吝嗇自己的財產。

**◈ 行解脫施，勤施，常樂行施，施會供養，等施一切。」**
　　實踐解脫（從對物質的執著中解脫）的布施，勤奮地布施，經常以快樂的心情行布施，在集會中供養，並且平等地布施給一切眾生。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即說了偈頌（詩歌形式的教法）說：

**◈ 「供養於父母，及家之尊長，**
　　「孝養父母，以及家中的長輩。

**◈ 柔和恭遜辭，離麁言兩舌。**
　　使用柔和且恭敬的言語，遠離粗魯的惡口與挑撥的兩舌。

**◈ 調伏慳悋心，常修真實語，**
　　調伏內心的吝嗇與貪婪，經常修行說真實話的習慣。

**◈ 彼三十三天，見行七法者。**
　　那三十三天的天眾，看到有人實踐這七種法（行為）時。

**◈ 咸各作是言，當來生此天。」**
　　全都會這麼說：『這個人將來會出生在這個天界。』」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **因緣 (Paṭicca-samuppāda)**：指事物之所以產生，必須具備「因」（直接原因）與「緣」（間接條件）的共同作用。本經中，實踐七種美德為「因」，而生於天界為「果」。
2. **兩舌 (Pisuna-vaca)**：指為了挑撥他人關係而對不同的人說不同的話，是正語（Right Speech）中所禁止的行為。
3. **慳悋 (Macchariya)**：指吝嗇、貪婪，不願分享自己擁有之物的心理狀態，是導致心靈狹隘的煩惱。
4. **解脫施 (Vimokkha-dāna)**：指在布施時，心不執著於「我施」、「我所施」以及「受施者」，透過給予來斷除對物質的貪著，從而達到心靈的解脫。

### [現代實作指南]
本經將「天界」的出生條件簡化為日常生活中可操作的七項美德。在現代生活中，可將其轉化為以下心理實踐：

1. **建立感恩關係（孝養長輩）**：
   ➔ 實作：每天花五分鐘思考父母或長輩對自己的照顧，並以具體的行動（如詢問健康、協助家務）表達感謝，將「供養」轉化為「關注與陪伴」。

2. **正向溝通訓練（和顏軟語、離惡口兩舌）**：
   ➔ 轉念：在說話前進行「三秒停頓」，自問：「這句話是否真實？是否溫和？是否會造成分裂？」若答案為否，則選擇沉默或重新措辭。

3. **對抗匱乏感（調伏慳悋心）**：
   ➔ 實作：練習「微小布施」。不一定要捐大筆金錢，可以是分享知識、給予讚美或在公共空間讓行。透過「給予」來打破「我不夠」的匱乏感，體會分享帶來的心理滿足。

4. **平等心練習（等施一切）**：
   ➔ 轉念：在幫助他人時，不區分對方的社會地位、種族或對自己的好壞。嘗試對不喜歡的人展現基本的善意，以訓練心靈的寬廣度。

### [段落大意]
本經記述佛陀教導比丘，若能實踐孝養長輩、溫和誠實的言語以及無私平等的布施這七種美德，能以此因緣在未來出生於天帝釋的境界。佛陀以偈頌強調調伏吝嗇心與實踐正語的重要性，說明善行將獲得天界眾生的認可。
