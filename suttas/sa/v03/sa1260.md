# 雜阿含經 第 1260 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「過去世時，有一猫狸，飢渴羸瘦，於孔穴中伺求鼠子。若鼠子出，當取食之。有時鼠子出穴遊戲，時，彼猫狸疾取吞之。鼠子身小，生入腹中；入腹中已，食其內藏，食內藏時，猫狸迷悶，東西狂走，空宅、塜間，不知何止，遂至於死。

「如是，比丘！有愚癡人依聚落住，晨朝著衣持鉢，入村乞食，不善護身，不守根門，心不繫念，見諸女人，起不正思惟，而取色相，發貪欲心，貪欲發已，欲火熾燃，燒其身心；燒身心已，馳心狂逸，不樂精舍、不樂空閑、不樂樹下，為惡不善心侵食內法，捨戒退減，此愚癡人長夜常得不饒益苦。是故，比丘！當如是學：『善護其身，守諸根門，繫心正念，入村乞食。』當如是學。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「過去世時，有一猫狸，飢渴羸瘦，於孔穴中伺求鼠子。**
　　那時，世尊告訴在場的眾位比丘說：「在過去的世間，有一隻貓，飢餓口渴且身體瘦弱，在洞穴口伺機尋找小老鼠。

**◈ 若鼠子出，當取食之。**
　　牠心想，如果小老鼠出來，就打算將牠捕捉來吃。

**◈ 有時鼠子出穴遊戲，時，彼猫狸疾取吞之。**
　　有一次小老鼠出洞玩耍，那時，那隻貓迅速地將牠吞掉。

**◈ 鼠子身小，生入腹中；入腹中已，食其內藏，**
　　小老鼠身體小，活著被吞進肚子裡；進入肚子之後，小老鼠開始啃食貓的內臟。

**◈ 食內藏時，猫狸迷悶，東西狂走，空宅、塜間，不知何止，遂至於死。**
　　在內臟被啃食的時候，貓感到迷亂困惑，在東西方向瘋狂奔跑，跑過空屋、跑過墳場，不知道該停在何處，最後導致死亡。

**◈ 「如是，比丘！有愚癡人依聚落住，晨朝著衣持鉢，入村乞食，**
　　比丘們！就像這樣，有些愚癡的人住在村落中，每天早晨穿上袈裟、拿著缽，進入村莊乞食。

**◈ 不善護身，不守根門，心不繫念，**
　　他們沒有善於保護自己的身體，沒有守住根門（眼、耳、鼻、舌、身、意這六個感官入口），心也不繫於正念（對當下覺知）。

**◈ 見諸女人，起不正思惟，而取色相，發貪欲心，**
　　當他們看見許多女人，產生了不正的思惟，執著於色（物質身體）的相貌，發起了貪欲之心。

**◈ 貪欲發已，欲火熾燃，燒其身心；**
　　貪欲一旦發起，就像欲火熾熱燃燒，燒灼著他的身心。

**◈ 燒身心已，馳心狂逸，不樂精舍、不樂空閑、不樂樹下，**
　　在燒灼身心之後，心變得奔馳狂亂，不再喜歡精舍（修行處）、不再喜歡清靜空閑之處、不再喜歡在樹下禪修。

**◈ 為惡不善心侵食內法，捨戒退減，**
　　被惡劣不善的心念侵蝕內在的法（修行成果與正法），導致捨棄戒律，修行退轉。

**◈ 此愚癡人長夜常得不饒益苦。**
　　這個愚癡的人在輪迴（Saṃsāra）的長夜中，經常承受沒有利益的痛苦。

**◈ 是故，比丘！當如是學：『善護其身，守諸根門，繫心正念，入村乞食。』當如是學。」**
　　因此，比丘們！應該這樣學習：「要善於保護身體，守住所有的根門（感官入口），將心繫於正念（覺知），然後才進入村莊乞食。」應該這樣學習。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **根門 (Indriya-dvāra)**：指六根（眼、耳、鼻、舌、身、意）與外界接觸的入口。守根門是指在感官接觸對象時，不讓心隨之起貪或嗔的反應。
2. **色 (Rūpa)**：在此指物質身體的形態與相貌。
3. **正念 (Sati)**：對當下經驗的清晰覺知，不隨之起妄想或情緒反應。
4. **內法 (Internal Dhamma)**：指修行者在心中建立的定力、智慧以及對正法的體悟。
5. **長夜 (Long night)**：佛教比喻輪迴（Saṃsāra）的漫長過程，充滿無明與痛苦。

### [現代實作指南]
本經以「貓吞鼠」比喻「貪欲侵蝕心靈」，提醒我們感官失控會導致內在崩潰。在現代生活中，可採取以下實作步驟：

1. **識別「觸發點」 (Trigger Identification)**：
   ➔ 當你在社交媒體看到吸引人的照片，或在現實中遇到令你心動的人時，意識到這就是「根門」被觸發的瞬間。
2. **建立「緩衝區」 (Creating a Buffer)**：
   ➔ 在「看到（色相）」與「產生貪欲」之間插入一個覺知。告訴自己：「現在，眼睛看到了對象，心產生了吸引力」。將其視為一種生理現象，而非「我」的渴望。
3. **觀察「欲火」的過程 (Observing the Fire)**：
   ➔ 觀察貪欲如何像火一樣燃燒：心跳加快、注意力集中在對象身上、開始幻想。意識到這種狀態會讓你失去對生活的掌控力（如同貓的狂走）。
4. **回歸「正念錨點」 (Returning to the Anchor)**：
   ➔ 當心開始奔馳時，立即將注意力移回呼吸或腳掌觸地的感覺。用「繫心正念」將心從對象身上拉回來，恢復內心的寧靜。
5. **評估「內在損耗」 (Evaluating the Cost)**：
   ➔ 思考：為了短暫的感官快感，我是否犧牲了專注力、內心的平靜或對目標的堅持？意識到「不守根門」會導致精神內耗，進而產生對生活的不滿與痛苦。

### [段落大意]
佛陀以一隻貓因吞食小老鼠而被反噬內臟而死的故事，比喻修行者若不守護感官入口（根門），在面對物質色相時起貪欲，將被欲火燒灼身心，導致修行退轉並在輪迴中受苦。因此，佛陀教導比丘應在生活中保持正念，善護感官，以避免內在法被侵蝕。
