# 雜阿含經 第 1059 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有四十法成就，如鐵槍投水，身壞命終，下生惡趣泥犁中。何等為四十法？謂手自殺生，教人令殺，讚歎殺生，見人殺生心隨歡喜，乃至自行邪見，教人令行，讚歎邪見，見行邪見心隨歡喜，是名四十法成就。如鐵槍投水，身壞命終，下生惡趣泥犁中。有四十法成就，如鐵槍鑽空，身壞命終，上生天上。何等為四十？謂不殺生，教人不殺，口常讚歎不殺功德，見不殺者心隨歡喜；乃至自行正見，教人令行，亦常讚歎正見功德，見人行者心隨歡喜，是名四十法成就。如鐵槍鑽空，身壞命終，上生天上。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「有四十法成就，如鐵槍投水，身壞命終，下生惡趣泥犁中。**
　　「若有四十種不善的行為（法）成就，就像鐵槍投進水中迅速下沉一樣，在物質身體（色）毀壞、生命結束之後，將會墮生在惡趣的泥犁（地獄）之中。

**◈ 何等為四十法？**
　　這四十種不善的行為是什麼呢？

**◈ 謂手自殺生，教人令殺，讚歎殺生，見人殺生心隨歡喜，**
　　指的是親手殺害生命、教唆他人殺生、讚美殺生的行為，以及看到他人殺生時，內心隨之感到歡喜；

**◈ 乃至自行邪見，教人令行，讚歎邪見，見行邪見心隨歡喜，是名四十法成就。**
　　直到於自身持有邪見（錯誤的認知與見解）、教唆他人實踐邪見、讚美邪見的行為，以及看到他人實踐邪見時，內心隨之感到歡喜，這就稱為四十種不善法成就。

**◈ 如鐵槍投水，身壞命終，下生惡趣泥犁中。**
　　（如此行為）就像鐵槍投進水中，在身體毀壞、生命結束之後，將會墮生在惡趣的泥犁（地獄）之中。

**◈ 有四十法成就，如鐵槍鑽空，身壞命終，上生天上。**
　　反之，若有四十種善的行為（法）成就，就像鐵槍鑽向天空輕盈上升一樣，在身體毀壞、生命結束之後，將會上升而生在天界之中。

**◈ 何等為四十？**
　　這四十種善行為是什麼呢？

**◈ 謂不殺生，教人不殺，口常讚歎不殺功德，見不殺者心隨歡喜；**
　　指的是不殺害生命、教導他人不殺生、口中經常讚美不殺生的功德，以及看到不殺生的人時，內心隨之感到歡喜；

**◈ 乃至自行正見，教人令行，亦常讚歎正見功德，見人行者心隨歡喜，是名四十法成就。**
　　直到於自身持有正見（正確的認知與見解）、教導他人實踐正見、也經常讚美正見的功德，以及看到他人實踐正見時，內心隨之感到歡喜，這就稱為四十種善法成就。

**◈ 如鐵槍鑽空，身壞命終，上生天上。」**
　　（如此行為）就像鐵槍鑽向天空，在身體毀壞、生命結束之後，將會上升而生在天界之中。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **泥犁 (Niraya)**：佛教宇宙觀中的地獄，是承受極大痛苦的惡趣之首，由強烈的貪、嗔、癡及惡業（Kamma）導致。
2. **邪見 (Micchā-diṭṭhi)**：錯誤的見解。在原始佛教中，最嚴重的邪見是指否認因果報應、否認業力作用或否認四聖諦的認知。
3. **正見 (Sammā-diṭṭhi)**：正確的見解。指對四聖諦（苦、集、滅、道）的正確認知，是八正道的第一步，也是修行解脫的指南。
4. **功德 (Puñña)**：指透過善行（身、口、意）所累積的正向心理能量與業力，能帶來快樂的果報。

### [現代實作指南]
本經揭示了一個重要的心理機制：**「行為 ➔ 影響 ➔ 認同 ➔ 強化」**。善惡不僅在於個人的直接行動，更在於我們如何看待與推廣這些行為。

**1. 覺察「隨歡喜」的心理陷阱：**
現代人常在社群媒體上看到他人攻擊、嘲諷或傷害他人（類似於「殺生」或「邪見」）時，若感到快感或隨之附和（心隨歡喜），這在心理上等同於參與該惡行。
➔ **實作：** 當你發現自己對他人的負面行為感到「爽快」時，立即意識到這正在累積負面的心理慣性，嘗試將注意力轉向該行為可能造成的傷害。

**2. 建立「正向強化」的循環：**
經文提到「讚歎」與「歡喜」能產生巨大的善業。這意味著對善行的肯定具有強大的推動力。
➔ **實作：** 刻意練習「發現他人的善」。當看到他人展現誠實、慈悲或正確的認知時，給予具體的讚美或在心中表達認同。這不僅能鼓勵他人，更能將自己的心境調整至「正見」的頻率。

**3. 從「自身」到「環境」的擴散：**
修行不應僅止於「我不殺生」或「我持有正見」，而應擴展到教導與鼓勵。
➔ **實作：** 在生活中扮演「正向影響者」。不強求說教，而是透過自身的實踐（自行正見）與對善行的支持（讚歎功德），營造一個正向的微環境。

### [段落大意]
本經以「鐵槍投水」與「鐵槍鑽空」為喻，說明四十種不善法（親自實踐、教唆、讚美、隨喜惡行）會導致墮入泥犁，而四十種善法（親自實踐、教導、讚美、隨喜善行）則能導向天界，強調身口意三業在實踐與認同上的因果影響。
