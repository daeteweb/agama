# 雜阿含經 第 1245 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「捨身惡行者，能得身惡行斷；不得身惡行斷者，我不說彼捨身惡行。以彼能得身惡行斷故，是故我說彼捨身惡行。身惡行者，不以義饒益安樂。眾生離身惡行，以義饒益，得安樂故，是故我說捨身惡行；口、意惡行亦如是說。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

---

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「捨身惡行者，能得身惡行斷；**
　　「那些捨棄身體惡行（如殺生、偷盜、邪淫等不正確的身體行為）的人，能夠真正斷除身體上的惡行；

**◈ 不得身惡行斷者，我不說彼捨身惡行。**
　　對於那些無法斷除身體惡行的人，我不稱他們為捨棄了身體惡行。

**◈ 以彼能得身惡行斷故，是故我說彼捨身惡行。**
　　正因為他們能夠確實地斷除身體惡行，所以我才說他們捨棄了身體惡行。

**◈ 身惡行者，不以義饒益安樂。**
　　那些從事身體惡行的人，無法獲得正義的利益（Attha）與真正的安樂。

**◈ 眾生離身惡行，以義饒益，得安樂故，是故我說捨身惡行；**
　　眾生如果能遠離身體惡行，就能獲得正義的利益並得到安樂，因此我才教導要捨棄身體惡行；

**◈ 口、意惡行亦如是說。」**
　　關於口頭的惡行（如妄語、兩舌、惡口、綺語）與意念的惡行（如貪欲、瞋恚、邪見），也是同樣的道理。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **身惡行 (Kāya-duccarita)**：指透過身體行為造成的不善業，在原始佛教中主要指殺生、偷盜、以及不合正道的性行為。
2. **義 (Attha)**：在此處指「利益」、「目的」或「正當的價值」。指行為能帶來實質且正向的結果。
3. **斷 (Nirodha/Cutting off)**：指徹底停止某種行為模式，使其不再產生，從而消除該行為所帶來的業報。
4. **口、意惡行 (Vacī-duccarita / Mano-duccarita)**：口惡行指言語上的傷害與欺騙；意惡行指心中產生的貪婪、憤怒與錯誤的見解。

### [現代實作指南]
本經文強調的是「行為與結果」的邏輯關係。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **行為覺察（正念觀察）**：
   當你準備採取某個行動（身）、說某句話（口）或產生某個念頭（意）時，先停頓三秒，觀察這個衝動是否屬於「惡行」（即：是否會對他人造成傷害，或對自己產生長遠的負面影響）。

2. **利益分析（義饒益評估）**：
   問自己：「這個行為能帶來真正的、長久的安樂（義饒益），還是僅僅是短暫的快感？」
   ➔ 例子：憤怒時想辱罵他人 → 短暫快感（宣洩） → 長遠結果（關係破裂、內心不安） → 結論：不具義饒益。

3. **果斷截斷（實踐「斷」）**：
   一旦意識到該行為不具義饒益，立即採取「捨」的行動。
   ➔ 轉念技巧：將「我想做這件事」轉化為「我選擇不這樣做，以獲得真正的安樂」。

4. **正向替代**：
   以「身、口、意」的三好行為替代惡行。
   ➔ 身體 → 幫助他人 ➔ 口頭 → 說鼓勵的話 ➔ 意念 → 練習慈心。

---

### 段落大意 
本經闡述了捨棄身、口、意三種惡行的必要性。佛陀指出，唯有真正斷除惡行的人才稱得上是「捨棄」，而惡行無法帶來真正的利益與安樂。反之，遠離惡行能使眾生獲得正義的利益與內心的寧靜。
