# 雜阿含經 第 1290 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。

時，彼天子而說偈言：

「廣無過於地，　　深無踰於海， 　高無過須彌，　　大士無毘紐。」

爾時，世尊說偈答言：

「廣無過於愛，　　深無踰於腹， 　高莫過憍慢，　　大士無勝佛。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

於是，天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出眾的天子，在深夜時分來到佛陀這裡，頂禮佛陀的足下，然後退到一邊坐下。他身體散發出的光明，遍照了整個祇樹給孤獨園。

**◈ 時，彼天子而說偈言：**
　　這時，那位天子以詩偈的形式說道：

**◈ 「廣無過於地，深無踰於海，高無過須彌，大士無毘紐。」**
　　「在寬廣方面，沒有比大地更寬的；在深邃方面，沒有比大海更深的；在高度方面，沒有比須彌山更高的；而在偉大的修行者中，沒有比毘紐（印度教主神）更偉大的。」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊以詩偈的形式回答說：

**◈ 「廣無過於愛，深無踰於腹，高莫過憍慢，大士無勝佛。」**
　　「在寬廣方面，沒有比愛（對世間的貪愛）更廣的；在深邃方面，沒有比腹（對物質欲望的貪婪）更深的；在高度方面，沒有比憍慢（傲慢與自大）更高的；而在偉大的修行者中，沒有能勝過佛陀的。」

**◈ 時，彼天子復說偈言：**
　　這時，那位天子再次以詩偈的形式說道：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「經過長久時間，終於見到了像婆羅門（指覺悟的聖者）般的人，達到了般涅槃（超越生死、絕對寧靜的解脫境界），所有恐懼都已超越，永遠超脫了世間的恩愛執著。」

**◈ 於是，天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　於是，天子聽聞佛陀所說的教法，心生法喜並隨之歡喜，頂禮佛陀的足下，隨即消失不再出現。

### [詞語註解]
1. **愛 (Taṇhā)**：指對感官快感或生存的強烈渴求，是導致輪迴苦因的核心動力。
2. **腹 (Kudha/Lobha)**：在此處以「腹」隱喻貪欲，指對物質、感官享受永不滿足的貪婪心。
3. **憍慢 (Māna)**：指一種自以為比他人優越的心理狀態，是阻礙修行、遮蔽真理的煩惱。
4. **毘紐 (Viṣṇu)**：印度教的三位主神之一，在當時的文化背景中代表著世間權能與偉大的頂峰。
5. **般涅槃 (Parinibbāna)**：指徹底熄滅煩惱之火，不再受生於輪迴，達到絕對寂靜的解脫狀態。

### [現代實作指南]
本經文透過「物質空間」與「心理狀態」的對比，揭示了內心煩惱的強大與危險。在現代生活中，我們可以透過以下方式實踐：

1. **覺察「愛」的擴張性**：
   當我們覺得「只要擁有這個我就滿足了」時，請意識到「愛（貪愛）」的特質就像大地一樣廣闊，它會不斷地將欲望擴張到新的領域。實作方法 ➔ 每當產生強烈渴望時，對自己說：「這又是貪愛的擴張，而非真正的需求。」

2. **識別「腹」的深不見底**：
   物質消費主義讓我們陷入「深不見底的腹」中。實作方法 ➔ 練習「知足（Santutthi）」，在進食或購物前，區分「生理需求」與「心理貪婪」，意識到貪婪的深淵是無法用物質填滿的。

3. **瓦解「憍慢」的高度**：
   當我們在職場或社交中感到優越感時，即是「憍慢」在升高。實作方法 ➔ 練習「對等視角」，提醒自己所有成就皆由因緣而生，並非單純的自我能力，將高度轉化為對他人的慈悲。

4. **從對比中尋找方向**：
   天子最初將偉大定義為「物理的巨大」或「神格的權能」，但佛陀將其重新定義為「對煩惱的超越」。實作方法 ➔ 將成功的定義從「獲取更多（廣、深、高）」轉向「放下更多（解脫、寧靜）」。

### [段落大意]
一位天子以大地、大海、須彌山與毘紐神來比喻世間之最，佛陀則以貪愛、貪婪、傲慢與佛陀的覺悟來回應，指出內心煩惱的強大程度遠超物質世界，而真正的偉大在於超越這些煩惱而達到的涅槃境界。天子聞後法喜，頂禮後消失。
