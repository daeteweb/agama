# 雜阿含經 第 1261 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「譬如木杵，常用不止，日夜消減。如是，比丘！若沙門、婆羅門從本以來，不閉根門，食不知量，初夜、後夜不勤覺悟修習善法，當知是輩終日損減，不增善法，如彼木杵。

「諸比丘！譬如優鉢羅、鉢曇摩、拘牟頭、分陀利生於水中，長於水中，隨水增長。如是，沙門、婆羅門善閉根門，飲食知量，初夜、後夜精勤覺悟，當知是等善根功德日夜增長，終不退減。當如是學：『善閉根門，飲食知量，初夜、後夜精勤覺悟，功德善法日夜增長。』當如是學。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「譬如木杵，常用不止，日夜消減。**
　　「就像一個木製的杵，如果不停地使用，日夜之間會逐漸磨損減少。

**◈ 如是，比丘！若沙門、婆羅門從本以來，不閉根門，食不知量，初夜、後夜不勤覺悟修習善法，當知是輩終日損減，不增善法，如彼木杵。」**
　　「比丘們！同樣地，如果沙門或婆羅門從開始修行以來，不善於閉根門（守護感官之門，不讓貪欲與嗔心隨外界刺激而起），飲食沒有節制，且在夜晚的初段與後段不精勤地保持覺悟並修行善法，應當了解，這些人的功德將會像那根木杵一樣，終日損減，而無法增加善法。」

**◈ 「諸比丘！譬如優鉢羅、鉢曇摩、拘牟頭、分陀利生於水中，長於水中，隨水增長。**
　　「比丘們！就像優鉢羅（青蓮）、鉢曇摩（紅蓮）、拘牟頭（白蓮）、分陀利（白蓮/紅蓮類）這些蓮花，生長在水中，隨著水的滋養而日益增長。

**◈ 如是，沙門、婆羅門善閉根門，飲食知量，初夜、後夜精勤覺悟，當知是等善根功德日夜增長，終不退減。**
　　「同樣地，如果沙門或婆羅門能善於閉根門（守護感官之門），飲食知量（適度飲食），且在夜晚的初段與後段精勤地保持覺悟，應當了解，這些人的善根（能導向解脫的良善心理種子）與功德將會像蓮花一樣日夜增長，最終不會退減。

**◈ 當如是學：『善閉根門，飲食知量，初夜、後夜精勤覺悟，功德善法日夜增長。』當如是學。」**
　　「你們應該這樣學習：『要善於守護感官之門，飲食適量，在夜晚初段與後段精勤覺悟，如此才能使功德與善法日夜增長。』你們應該這樣學習。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **閉根門 (Indriya-saṃvara)**：指對六根（眼、耳、鼻、舌、身、意）的守護。當感官接觸到外界對象時，不隨之產生貪著或厭惡的反應，防止煩惱進入心中。
2. **飲食知量 (Bhojane mattaññutā)**：指在飲食上保持適度，不因貪食而過量，也不因過度禁食而損害健康，以維持身體適合修行的狀態。
3. **初夜、後夜 (First and last watches)**：古代將夜晚分為三段。初夜與後夜是修行者最容易產生倦怠或散漫的時間，因此強調在此時段需精勤覺悟。
4. **善根 (Kusala-mūla)**：指能產生正向結果、導向解脫的心理傾向或種子，如不貪、不嗔、不痴。

### [現代實作指南]
本經文提供了一套關於「能量管理」與「心智守門」的實踐方法，可轉化為以下生活技巧：

1. **感官的「防火牆」機制（實作閉根門）**：
   ➔ **覺察觸發點**：在刷手機、社交媒體或面對衝突時，意識到「眼根」或「耳根」正在接收資訊。
   ➔ **建立緩衝區**：在接收資訊與產生反應（如憤怒、嫉妒）之間，刻意創造一個呼吸的間隙。告訴自己：「我看到了這個訊息，但我不需要立即對其產生情緒反應。」

2. **生理狀態的適度調節（實作飲食知量）**：
   ➔ **正念飲食**：在用餐時觀察飢餓感，而非僅僅滿足口腹之欲。
   ➔ **避免過飽**：過量飲食會導致身體沉重、精神昏沉（食後嗜睡），直接影響專注力與覺悟能力。嘗試在八分飽時停止，以保持心智的敏銳。

3. **關鍵時間的覺醒訓練（實作初夜、後夜精勤）**：
   ➔ **睡前與醒後的覺察**：在入睡前（初夜）與剛醒來（後夜）時，不要立即陷入雜念或手機螢幕中。
   ➔ **微修行**：利用這兩個時間點進行簡單的呼吸觀察或回顧今日的善行，將心智從「自動導航」模式切換到「覺悟」模式，防止心靈能量的損耗。

### 段落大意 
佛陀以「木杵磨損」比喻不守感官、飲食無度且懈怠的人，其功德會日漸消減；以「蓮花隨水增長」比喻能守護感官、飲食適量且精勤覺悟的人，其善根功德會日夜增長。勉勵比丘應透過自律與覺悟來累積善法。
