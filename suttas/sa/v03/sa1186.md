# 雜阿含經 第 1186 經

### [原文]
如是我聞：

一時，佛住迦毘羅衛國尼拘律園中。

時，有縈髻羅豆婆遮婆羅門，本俗人時，為佛善知識，來詣佛所，面相問訊慰勞已，退坐一面。而說偈言：

「外身縈髻者，　　是但名縈髻； 　內心縈髻者，　　是結縛眾生。 　今請問瞿曇，　　云何解縈髻？」

爾時，世尊說偈答言：

「受持於淨戒，　　內心修正覺， 　專精勤方便，　　是則解縈髻。」

時，縈髻婆羅門聞佛所說，歡喜隨喜，從座起而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住迦毘羅衛國尼拘律園中。**
　　在某一個時間，佛陀住在迦毘羅衛國的尼拘律園中。

**◈ 時，有縈髻羅豆婆遮婆羅門，本俗人時，為佛善知識，來詣佛所，面相問訊慰勞已，退坐一面。**
　　那時，有一位名叫縈髻（頭髮盤結）的羅豆婆遮婆羅門，他在尚未出家、仍是俗人時，就是佛陀的善知識（指能引導修行的人）。他來到佛陀這裡，在面對面詢問近況並表達慰問之後，退到一邊坐下。

**◈ 而說偈言：「外身縈髻者，是但名縈髻；內心縈髻者，是結縛眾生。今請問瞿曇，云何解縈髻？」**
　　接著他用偈頌說道：「身體外在盤結頭髮的人，那僅僅是名為『縈髻』的稱呼；而內心被煩惱（Kilesa）結縛的人，才是真正將眾生束縛在輪迴（Saṃsāra）中的原因。現在我想請問瞿曇（佛陀），應該如何才能解開內心的結縛？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「受持於淨戒，內心修正覺，專精勤方便，是則解縈髻。」**
　　「持守清淨的戒律（Sīla），在內心修正覺知（正念與正定），專注且精進地運用修行的方法（Upāya），這就是解開內心結縛的方法。」

**◈ 時，縈髻婆羅門聞佛所說，歡喜隨喜，從座起而去。**
　　那時，縈髻婆羅門聽聞佛陀所說的教法，心生法喜並依照教導去實踐，隨後從座位上起身離開。

### [詞語註解（解釋重要佛教概念）]
1. **煩惱 (Kilesa)**：指污染心靈、導致痛苦與錯誤認知的心理狀態，如貪、嗔、痴。
2. **輪迴 (Saṃsāra)**：指眾生在生死之間不斷循環、受業報驅使而遷徙的過程。
3. **戒 (Sīla)**：指修行者為了防止惡行、安定心靈而遵守的道德準則與行為規範。
4. **方便 (Upāya)**：指為了引導不同根器的人達到解脫，而採取靈活且適當的教學或修行手段。

### [現代實作指南]
本經文將「盤結的頭髮」比喻為「內心的執著與煩惱」。在現代生活中，我們可以透過以下步驟實踐「解結」：

1. **區分「形式」與「實質」**：
   ➔ 覺察自己是否過度在意外在的標籤（如職稱、名聲、外貌），如同「外身縈髻」僅是名號。意識到真正的痛苦來自於內心的「結」（如對某人的恨意、對失敗的恐懼）。

2. **建立行為底線（受持淨戒）**：
   ➔ 當內心混亂時，先從簡單的行為規範開始。例如：決定在接下來的一週內不說謊、不批評他人。當行為趨於清淨，內心的躁動會減少，為後續的覺察創造空間。

3. **練習覺知修正（內心修正覺）**：
   ➔ 採取「正念觀察」。當憤怒或焦慮（內心的結）升起時，不立即反應，而是對自己說：「現在，憤怒升起了」。透過這種覺知，將自己從情緒的結縛中抽離。

4. **尋找適合的工具（專精勤方便）**：
   ➔ 認清每個人解結的方式不同。有人適合透過靜坐（定），有人適合透過閱讀經論（慧），有人適合透過服務他人（慈）。找到最適合自己的修行方法並持之以恆地練習。

### [段落大意]
一名婆羅門以外在盤結的頭髮比喻內心的煩惱結縛，詢問佛陀解脫之法。佛陀教導應透過持戒、修正覺知以及精進運用修行方便，方能解開內心的束縛。
