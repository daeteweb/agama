# 雜阿含經 第 1010 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「誰縛於世間，　　誰調伏令解？ 　斷除何等法，　　說名得涅槃？」

爾時，世尊說偈答言：

「欲能縛世間，　　調伏欲解脫， 　斷除愛欲者，　　說名得涅槃。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出眾的天子，在深夜時分來到佛陀這裡，頂禮佛陀的足部，然後退到一邊坐下。他的身體散發出各種光明，遍照整個祇樹給孤獨園。

**◈ 時，彼天子說偈問佛：**
　　那時，那位天子用偈頌（詩歌形式）請問佛陀：

**◈ 「誰縛於世間，誰調伏令解？斷除何等法，說名得涅槃？」**
　　「是誰將眾生束縛在世間的輪迴之中？又是誰能調伏內心使之獲得解脫？必須斷除什麼樣的法（現象或心理狀態），才能被稱作達到了涅槃（超越生死、絕對寧靜的解脫境界）？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「欲能縛世間，調伏欲解脫，斷除愛欲者，說名得涅槃。」**
　　「正是欲（對感官的渴望）能將眾生束縛在世間；而調伏這份欲求，才能獲得解脫。那些能夠斷除愛欲（對對象的貪愛與執著）的人，才被稱作達到了涅槃（超越生死、絕對寧靜的解脫境界）。」

**◈ 時，彼天子復說偈言：**
　　那時，那位天子再次說偈頌道：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「我長期以來觀察那些婆羅門，看到他們最終達到了般涅槃（完全的解脫），所有的恐懼都已經過去，永遠超越了世間的恩愛（情感上的執著與牽絆）。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，那位天子聽聞佛陀所說的教法，心生法喜並隨喜讚嘆，頂禮佛陀的足部，隨即消失不再出現。

### [詞語註解]
1. **涅槃 (Nibbāna)**：指熄滅貪、嗔、痴三毒的狀態，是超越生死輪迴、絕對寧靜且不再受苦的解脫境界。
2. **欲 (Taṇhā/Chanda)**：在此指對感官享受的渴求或對生存的貪戀，是導致眾生在世間輪迴、受苦的核心原因（即十二因緣中的「愛」）。
3. **調伏 (Damana)**：指透過修行（Bhāvanā）來觀察、控制並轉化內心的衝動與煩惱，使其不再主導行為。
4. **般涅槃 (Parinibbāna)**：通常指阿羅漢或佛陀在肉身死亡時，由於已斷除所有煩惱，不再投生，達成最終且完全的解脫。

### [現代實作指南]
本經文揭示了「束縛」與「解脫」的關鍵在於「欲（渴望）」。在現代生活中，我們可以透過以下步驟實踐：

1. **識別「束縛感」**：
   當你感到焦慮、不滿足或強烈地想要得到某樣東西（如名聲、金錢、他人的認同）時，請意識到這就是「欲」在起作用。這種「必須擁有」的感覺，就是經文中所說的「縛於世間」的心理體驗。

2. **觀察「欲」的運作 ➔ 轉念**：
   不要試圖強行壓抑慾望，而是採取「觀察者」的視角。
   - **原念**：「我一定要得到那個職位，否則我會很失敗。」（被欲束縛）
   - **轉念**：「我現在心中升起了一種對權力的渴望，這種渴望讓我感到緊張。這只是心的造作（行），並非我的本質。」

3. **練習「微小解脫」**：
   在日常生活中練習「短暫放棄」。例如：在極其想喝咖啡或刷手機的衝動升起時，刻意等待三分鐘，觀察那種「渴望」如何升起、達到頂峰，然後逐漸消退。透過這種方式，你是在實踐「調伏」，體驗到即使不滿足慾望，內心依然可以保持平靜。

4. **定義自己的「涅槃」**：
   將涅槃理解為「不再被情緒牽著走」的狀態。每當你能從「我想要」轉變為「我覺知到我想」，你就體驗到了一次微小的解脫。

段落大意 
一位天子請問佛陀，是什麼將眾生束縛在世間，以及如何才能獲得解脫。佛陀回答，正是「欲（貪愛）」將眾生束縛，而唯有調伏並斷除愛欲，才能達到涅槃的解脫境界。天子聞後深信不疑，隨喜頂禮後消失。
