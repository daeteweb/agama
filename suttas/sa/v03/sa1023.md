# 雜阿含經 第 1023 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。時，有尊者叵求那住東園鹿母講堂，疾病困篤。

尊者阿難往詣佛所，稽首禮足，退住一面，白佛言：「世尊！尊者叵求那住東園鹿母講堂，疾病困篤，如是病比丘多有死者。善哉！世尊！願至東園鹿母講堂尊者叵求那所，以哀愍故！」

爾時，世尊默然而許，至日晡時，從禪覺，往詣東園鹿母講堂，至尊者叵求那房，敷座而坐，為尊者叵求那種種說法，示、教、照、喜；示、教、照、喜已，從坐起去。

尊者叵求那，世尊去後，尋即命終。當命終時，諸根喜悅，顏貌清淨，膚色鮮白。

時，尊者阿難供養尊者叵求那舍利已，往詣佛所，稽首佛足，却住一面，白佛言：「世尊！尊者叵求那，世尊來後，尋便命終，臨命終時，諸根喜悅，膚色清淨，鮮白光澤。不審世尊！彼當生何趣？云何受生，後世云何？」

佛告阿難：「若有比丘先未病時，未斷五下分結，若覺病起，其身苦患，心不調適，生分微弱，得聞大師教授、教誡種種說法，彼聞法已，斷五下分結。阿難！是則大師教授設法福利。

「復次，阿難！若有比丘先未病時，未斷五下分結，然後病起，身遭苦患，生分轉微，不蒙大師教授、教誡說法，然遇諸餘多聞大德修梵行者教授、教誡說法，得聞法已，斷五下分結。阿難！是名教授、教誡聽法福利。

「復次，阿難！若比丘先未病時，不斷五下分結，乃至生分微弱，不聞大師教授、教誡說法，復不聞餘多聞大德諸梵行者教授、教誡說法，然彼先所受法，獨靜思惟，稱量觀察，得斷五下分結。阿難！是名思惟觀察先所聞法所得福利。

「復次，阿難！若有比丘先未病時，斷五下分結，不得無上愛盡解脫，不起諸漏，心善解脫，然後得病，身遭苦患，生分微弱，得聞大師教授、教誡說法，得無上愛盡解脫，不起諸漏，離欲解脫。阿難！是名大師說法福利。

「復次，阿難！若有比丘先未病時，斷五下分結，不得無上愛盡解脫，不起諸漏，離欲解脫，覺身病起，極遭苦患，不得大師教授、教誡說法，然得諸餘多聞大德諸梵行者教授、教誡說法，得無上愛盡解脫，不起諸漏，離欲解脫。阿難！是名教授教誡聞法福利。

「復次，阿難！若有比丘先未病時，斷五下分結，不得無上愛盡解脫，不起諸漏，離欲解脫。其身病起，極生苦患，不得大師教授、教誡說法，不得諸餘多聞大德教授、教誡說法，然先所聞法，獨一靜處，思惟稱量觀察，得無上愛盡解脫，不起諸漏，離欲解脫。阿難！是名思惟先所聞法所得福利。

「何緣叵求那比丘不得諸根欣悅，色貌清淨，膚體鮮澤？叵求那比丘先未病時，未斷五下分結，彼親從大師聞教授、教誡說法，斷五下分結，世尊為彼尊者叵求那受阿那含記。」

佛說此經已，尊者阿難聞佛所說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。時，有尊者叵求那住東園鹿母講堂，疾病困篤。**
　　在某一個時間，佛住在舍衛國的祇樹給孤獨園。那時，有一位名叫叵求那的尊者住在東園的鹿母講堂，病情非常嚴重。

**◈ 尊者阿難往詣佛所，稽首禮足，退住一面，白佛言：「世尊！尊者叵求那住東園鹿母講堂，疾病困篤，如是病比丘多有死者。善哉！世尊！願至東園鹿母講堂尊者叵求那所，以哀愍故！」**
　　尊者阿難走到佛陀面前，低頭頂足行禮，然後退到一邊，對佛陀恭敬地說道：「世尊！尊者叵求那住在東園鹿母講堂，病情危急，像這樣患病的比丘（出家修行者）很多都因此去世了。世尊，如果您能出於慈悲心，前往東園鹿母講堂看望尊者叵求那，那就太好了！」

**◈ 爾時，世尊默然而許，至日晡時，從禪覺，往詣東園鹿母講堂，至尊者叵求那房，敷座而坐，為尊者叵求那種種說法，示、教、照、喜；示、教、照、喜已，從坐起去。**
　　那時，世尊默默地答應了。到了傍晚時分，佛陀從禪定中覺醒，前往東園鹿母講堂，來到尊者叵求那的房間，鋪好座位坐下，為尊者叵求那講解各種法（真理），給予指引、教導、啟發與鼓勵；在完成指引、教導、啟發與鼓勵後，佛陀便從座位上起身離開。

**◈ 尊者叵求那，世尊去後，尋即命終。當命終時，諸根喜悅，顏貌清淨，膚色鮮白。**
　　尊者叵求那在世尊離開後，很快就停止了呼吸而去世。在命終之時，他的感官功能（諸根）顯得喜悅，面容清淨，皮膚色澤鮮白。

**◈ 時，尊者阿難供養尊者叵求那舍利已，往詣佛所，稽首佛足，却住一面，白佛言：「世尊！尊者叵求那，世尊來後，尋便命終，臨命終時，諸根喜悅，膚色清淨，鮮白光澤。不審世尊！彼當生何趣？云何受生，後世云何？」**
　　這時，尊者阿難在處理完尊者叵求那的舍利（骨灰）供養後，走到佛陀面前，低頭頂足行禮，退到一邊，對佛陀恭敬地說道：「世尊！尊者叵求那在世尊來訪後不久就去世了，臨終時諸根喜悅，膚色清淨且有鮮白光澤。我不確定，世尊，他將會投生到什麼樣的境界（趣）？會如何受生？後世的情況會是如何？」

**◈ 佛告阿難：「若有比丘先未病時，未斷五下分結，若覺病起，其身苦患，心不調適，生分微弱，得聞大師教授、教誡種種說法，彼聞法已，斷五下分結。阿難！是則大師教授設法福利。**
　　佛陀告訴阿難：「如果有一位比丘在還沒生病之前，尚未斷除五種下分結（即對自我、對法、對儀軌的執著，以及對欲界與色界的貪愛）。當他感覺到疾病發生，身體遭受痛苦，心境不寧，生命力微弱時，如果能聽到大師（佛陀）的教授與教誡，聽聞種種說法，他在聽法之後斷除了五種下分結。阿難！這就是大師親自教授所帶來的法益。

**◈ 「復次，阿難！若有比丘先未病時，未斷五下分結，然後病起，身遭苦患，生分轉微，不蒙大師教授、教誡說法，然遇諸餘多聞大德修梵行者教授、教誡說法，得聞法已，斷五下分結。阿難！是名教授、教誡聽法福利。**
　　另外，阿難！如果有一位比丘在還沒生病之前，尚未斷除五種下分結，隨後生病，身體遭受痛苦，生命力更加微弱，雖然沒有機會聽到大師的教授與教誡，但遇到了其他博學多聞且修持梵行（清淨修行）的大德長老，聽聞他們的教授與教誡，在聽法之後斷除了五種下分結。阿難！這就叫做聽從教授教誡而獲得的法益。

**◈ 「復次，阿難！若比丘先未病時，不斷五下分結，乃至生分微弱，不聞大師教授、教誡說法，復不聞餘多聞大德諸梵行者教授、教誡說法，然彼先所受法，獨靜思惟，稱量觀察，得斷五下分結。阿難！是名思惟觀察先所聞法所得福利。**
　　另外，阿難！如果一位比丘在還沒生病之前，沒有斷除五種下分結，直到生命力微弱，既沒有聽到大師的教授教誡，也沒有聽到其他博學大德的教授教誡，但他對先前所學的法（真理）獨自靜心思惟，經過稱量與觀察，最終斷除了五種下分結。阿難！這就叫做透過思惟觀察先前所聞之法而獲得的法益。

**◈ 「復次，阿難！若有比丘先未病時，斷五下分結，不得無上愛盡解脫，不起諸漏，心善解脫，然後得病，身遭苦患，生分微弱，得聞大師教授、教誡說法，得無上愛盡解脫，不起諸漏，離欲解脫。阿難！是名大師說法福利。**
　　另外，阿難！如果有一位比丘在還沒生病之前，已經斷除了五種下分結，但尚未獲得最高層次的愛盡解脫（即完全斷除貪愛而達到阿羅漢果），尚未止息所有漏（煩惱的流出），心尚未完全解脫。隨後他生病，身體遭受痛苦，生命力微弱，此時聽聞大師的教授與教誡，進而獲得最高層次的愛盡解脫，不再產生任何漏，達到離欲的解脫。阿難！這就是大師說法所帶來的法益。

**◈ 「復次，阿難！若有比丘先未病時，斷五下分結，不得無上愛盡解脫，不起諸漏，離欲解脫，覺身病起，極遭苦患，不得大師教授、教誡說法，然得諸餘多聞大德諸梵行者教授、教誡說法，得無上愛盡解脫，不起諸漏，離欲解脫。阿難！是名教授教誡聞法福利。**
　　另外，阿難！如果有一位比丘在還沒生病之前，已經斷除了五種下分結，但尚未獲得最高層次的愛盡解脫，尚未止息所有漏，尚未達到離欲解脫。當他感覺到身體生病，遭受極大痛苦，雖然沒有機會聽到大師的教授教誡，但聽到了其他博學大德的教授教誡，進而獲得最高層次的愛盡解脫，不再產生任何漏，達到離欲的解脫。阿難！這就叫做聽從教授教誡而獲得的法益。

**◈ 「復次，阿難！若有比丘先未病時，斷五下分結，不得無上愛盡解脫，不起諸漏，離欲解脫。其身病起，極生苦患，不得大師教授、教誡說法，不得諸餘多聞大德教授、教誡說法，然先所聞法，獨一靜處，思惟稱量觀察，得無上愛盡解脫，不起諸漏，離欲解脫。阿難！是名思惟先所聞法所得福利。**
　　另外，阿難！如果有一位比丘在還沒生病之前，已經斷除了五種下分結，但尚未獲得最高層次的愛盡解脫，尚未止息所有漏，尚未達到離欲解脫。當他身體生病，遭受極大痛苦，既沒有聽到大師的教授教誡，也沒有聽到其他博學大德的教授教誡，但他對先前所聞的法，在獨自靜處中進行思惟、稱量與觀察，進而獲得最高層次的愛盡解脫，不再產生任何漏，達到離欲的解脫。阿難！這就叫做透過思惟先前所聞之法而獲得的法益。

**◈ 「何緣叵求那比丘不得諸根欣悅，色貌清淨，膚體鮮澤？叵求那比丘先未病時，未斷五下分結，彼親從大師聞教授、教誡說法，斷五下分結，世尊為彼尊者叵求那受阿那含記。」**
　　為什麼叵求那比丘在去世時會諸根欣悅，面貌清淨，皮膚鮮澤呢？因為叵求那比丘在還沒生病之前，尚未斷除五種下分結，後來他親自聽聞大師的教授與教誡說法，斷除了五種下分結。因此，世尊為尊者叵求那給予了「阿那含（不還果，即不再回到欲界受生）」的預言（記）。

**◈ 佛說此經已，尊者阿難聞佛所說，歡喜隨喜，作禮而去。**
　　佛陀講完這部經後，尊者阿難聽聞佛陀所說的教法，心生法喜並依照教導去實踐，向佛陀行禮後才離開。

### [詞語註解]
1. **五下分結 (Orambhagiya-samyojana)**：指將眾生束縛在欲界（Kāma-loka）的五種根本煩惱，包括：身見（執著有一個恆常的自我）、疑（對正法產生懷疑）、戒禁見（執著於錯誤的儀軌與禁忌）、堪欲（對感官慾望的貪愛）、以及對欲界存在的執著。斷除這五項結使者可達到不還果（Anāgāmī）。
2. **漏 (Āsava)**：原意為「流出」，比喻煩惱像滲漏的水一樣不斷流出，污染心靈。主要分為有漏（Kilesa）與無漏（Nirāmisa），包含欲漏、有漏（對存在的執著）與無明漏。
3. **愛盡解脫 (Rāga-nirodha-vimokkha)**：指完全斷除貪愛（Taṇhā）而達到的解脫狀態，即阿羅漢（Arahant）的境界，不再受輪迴之苦。
4. **阿那含 (Anāgāmī)**：意為「不還」。指已斷除五下分結的修行者，死後不再投生於欲界，而是在淨處天（Suddhāvāsa）中證得涅槃。
5. **記 (Prediction/Anumodana)**：佛陀根據修行者的根基與進度，對其將來必然能證得某種果位（如阿羅漢或不還果）所作的正式肯定與預言。

### [現代實作指南]
本經文揭示了在面對生命終結或極端身體痛苦（疾病）時，心靈轉化與解脫的三種路徑。我們可以將其轉化為現代生活中的心理實踐：

1. **主動尋求正向指引 (對應「大師/大德教授」)**：
   ➔ 當我們陷入人生低谷或面對不可逆的疾病時，不要陷入孤立的絕望。尋找具有智慧、能提供客觀真理（而非單純情感安慰）的導師或精神指引。透過聆聽關於「無常」與「無我」的真理，將注意力從「身體的痛苦」轉移到「對痛苦本質的觀察」上。

2. **深化既有知識的內化 (對應「獨靜思惟」)**：
   ➔ 在無法獲得外部指引的孤獨時刻，練習「回溯思惟」。將過去讀過的正向書籍、學習過的正念技巧或體悟過的真理重新拿出來，在靜心中進行「稱量觀察」。問自己：「這個痛苦是否也是無常的？」、「這個受苦的『我』究竟是什麼？」。將知識轉化為體悟，在孤獨中完成心靈的自救。

3. **接納身體與心靈的脫鉤 (對應「諸根喜悅」)**：
   ➔ 練習將「身體的病痛」與「心靈的狀態」分開看待。身體雖然在承受苦患（色蘊的變異），但心靈可以透過對法的體悟保持清淨與喜悅。實作方法是：在疼痛發生時，觀察疼痛作為一種「受（感受）」的生起與滅去，而不將其定義為「我的痛苦」，從而減少心理上的二次傷害。

### 段落大意 
本經描述尊者叵求那在病重臨終前，因佛陀的指引而斷除五下分結，以清淨喜悅之相命終，並獲佛陀預言為不還果。佛陀藉此向阿難說明，比丘在病中可透過三種方式獲得解脫：一是聽聞佛陀說法，二是聽聞大德說法，三是獨自思惟先前所聞之法。這三種路徑分別適用於尚未斷五下分結者（達到不還果）以及已斷五下分結者（達到阿羅漢果），強調了正法指引與自我思惟在生命末期對心靈解脫的決定性作用。
