# 雜阿含經 第 1296 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。

時，彼天子白佛言：「世尊！拘屢陀王女修波羅提沙今日生子。」

佛告天子：「此則不善，非是善。」

時，彼天子即說偈言：

「人生子為樂，　　世間有子歡， 　父母年老衰，　　子則能奉養， 　瞿曇何故說，　　生子為不善。」

爾時，世尊說偈答言：

「當知恒無常，　　純空陰非子， 　生子常得苦，　　愚者說言樂。 　是故我說言，　　生子非為善， 　非善為善像，　　念像不可念。 　實苦貌似樂，　　放逸所踐蹈。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

於是，天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出眾的天子，在深夜時分來到佛陀這裡，頂禮佛陀的足下，然後退到一邊坐下。他身體散發出的光明遍照了整個祇樹給孤獨園。

**◈ 時，彼天子白佛言：「世尊！拘屢陀王女修波羅提沙今日生子。」**
　　這時，那位天子對佛陀恭敬地說道：「世尊！拘屢陀王的女兒修波羅提沙今天生孩子了。」

**◈ 佛告天子：「此則不善，非是善。」**
　　佛陀告訴天子：「這件事（對解脫而言）並不吉祥，並非真正的善（有益於脫離苦海之法）。」

**◈ 時，彼天子即說偈言：「人生子為樂，世間有子歡，父母年老衰，子則能奉養，瞿曇何故說，生子為不善。」**
　　這時，天子立即用偈頌說道：「人們認為生孩子是快樂的，世間的人因為有了孩子而歡喜；當父母年老衰弱時，孩子能夠提供奉養。瞿曇（佛陀）為什麼說生孩子是不善的呢？」

**◈ 爾時，世尊說偈答言：「當知恒無常，純空陰非子，生子常得苦，愚者說言樂。」**
　　那時，世尊用偈頌回答說：「應當了解，一切現象恆常處於無常（變遷不居）的狀態，構成身心的五種要素（色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識））本質上是空（沒有永恆實體）的，並沒有一個真實永恆的『孩子』存在。生孩子實際上會帶來恆久的苦，只有愚笨的人才會說這是快樂。」

**◈ 「是故我說言，生子非為善，非善為善像，念像不可念。實苦貌似樂，放逸所踐蹈。」**
　　「因此我才說，生孩子並非真正的善。將不善的事物誤認為是善，就像是追求一個虛假的幻像，而幻像是不可執著的。這件事本質上是苦，只是外貌看起來像快樂，是那些放逸（不修行、隨波逐流）的人所追求的道路。」

**◈ 時，彼天子復說偈言：「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　這時，天子再次用偈頌說道：「我長時間觀察這位婆羅門（佛陀），他終於達到了超越生死、絕對寧靜的解脫境界（涅槃），所有的恐懼都已過去，永遠超越了世間的愛欲與執著。」

**◈ 於是，天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　於是，天子聽聞佛陀所說的教法，內心感到法喜並隨喜，頂禮佛陀的足下，隨即消失不見。

### [詞語註解]
1. **不善 (Akusala)**：在佛教術語中，不善並非指道德上的「邪惡」，而是指「不能導致解脫」或「會增加執著與苦痛」的狀態。
2. **無常 (Anicca)**：指一切由因緣和合而成的現象，都在不斷地生起與滅盡，沒有任何永恆不變的實體。
3. **陰 (Khandha)**：即「蘊」，指構成生命個體的五種要素（五蘊）。此處「純空陰」強調五蘊的本質是空性，沒有一個恆常的「我」或「子」。
4. **放逸 (Pamāda)**：指心靈對正念的缺失，隨順感官欲望而生活，缺乏對生命真相的覺察。
5. **般涅槃 (Parinibbāna)**：指完全熄滅煩惱之火，徹底脫離輪迴生死、不再受苦的終極境界。

### [現代實作指南]
本經文探討的是「愛與執著」的深刻矛盾。在現代生活中，我們常將「擁有」或「繁衍」視為幸福的指標，但佛陀提醒我們，這種幸福是建立在「無常」的基礎之上的。

**實作技巧：**
1. **區分「慈愛」與「執著」**：
   ➔ **執著**：認為孩子是「我的」，期望孩子符合我的願望，將孩子視為填補自我空虛的工具。這種愛在孩子改變或離去時會轉化為巨大的痛苦。
   ➔ **慈愛**：認同孩子是一個獨立的生命個體，以無條件的關懷支持其成長，但不對其擁有所有權。

2. **練習「無常觀」**：
   當我們在生活中感受到強烈的依戀（如對親人、財產的依戀）時，試著在心中默念：「這一切都是由因緣構成的，它在變遷，沒有永恆的實體。」這並非要我們變得冷漠，而是讓我們在愛情的同時，保持一份覺知，避免在失去時陷入崩潰。

3. **覺察「貌似樂」的陷阱**：
   反思生活中那些讓我們感到短暫快樂但隨後帶來更多焦慮的事物（例如：追求名聲、物質消費）。問自己：「這是我真正的內在平安，還是僅僅是一個『善像』（虛假的快樂幻像）？」

### [段落大意]
一位天子告知佛陀某王女生子的喜訊，佛陀卻指出生子對解脫而言並非善事。佛陀解釋，由於五蘊本質空且無常，對孩子的依戀實則將導致長久的苦痛，世人將其誤認為樂是因為處於放逸之中。天子聞後體悟佛陀已超越世間愛欲、證得涅槃，心生歡喜而頂禮消失。
