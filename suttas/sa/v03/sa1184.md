# 雜阿含經 第 1184 經

### [原文]
如是我聞：

一時，佛在拘薩羅人間遊行，宿於孫陀利河側。

爾時，世尊剃髮未久，於後夜時，結跏趺坐，正身思惟，繫念在前，以衣覆頭。

時，孫陀利河側有婆羅門住止，夜起持祠，餘食不盡，時至河邊，欲求大德婆羅門以奉之。

爾時，世尊聞河邊婆羅門聲。聞已，謦咳作聲，却衣現頭。

時，孫陀利河側婆羅門見佛已，作是念：「是剃頭沙門，非婆羅門。」欲持食還去。彼婆羅門復作是念：「非獨沙門是剃頭者，婆羅門中亦有剃頭，應往至彼，問其所生。」時，孫陀利河側婆羅門詣世尊所，而問之言：「為何姓生？」

爾時，世尊即說偈言：

「汝莫問所生，　　但當問所行。 　刻木為鑽燧，　　亦能生於火； 　下賤種姓中，　　生堅固牟尼。 　智慧有慚愧，　　精進善調伏， 　究竟大明際，　　清淨修梵行。 　而今正是時，　　應奉施餘食。」

時，孫陀利河側婆羅門復說偈言：

「我今吉良日，　　求福修供養， 　遇得見大士，　　三時最勝尊， 　若不見佛者，　　當更施餘人。」

爾時，孫陀利河側婆羅門轉得信心，即持餘食以奉世尊，世尊不受，以說偈得故，如上因說偈而得食廣說。

孫陀利河側婆羅門白佛言：「世尊！今此施食當置何所？」

佛告婆羅門：「我不見諸天、魔、梵、沙門、婆羅門、天神、世人有能食此食，令身安隱者。汝持此食去著無虫水中，及少生草地。」

時，婆羅門即持此食著無虫水中，水即烟起涌沸，啾啾作聲，如燒鐵丸投之冷水，烟起涌沸，啾啾作聲。如是彼食著無虫水中，烟起涌沸，啾啾作聲。孫陀利河側婆羅門心欲恐怖，身毛皆竪，謂為災變，馳走上岸，集聚乾木，供養祠火，令息災恠。

世尊見彼集聚乾木，供養祠火，望息災恠。見已。即說偈言：

「婆羅門祠火，　　焚燒乾草木， 　莫呼是淨道，　　能却諸災患。 　此則惡供養，　　而謂為黠慧； 　作如是因緣，　　外道取修淨。 　汝今棄薪火，　　起內火熾然， 　常修不放逸，　　常富於供養。 　處處興淨信，　　廣施設大會， 　心意為束薪，　　瞋恚黑烟起， 　妄語為塵味，　　口舌為木杓， 　胸懷燃火處，　　欲火常熾然。 　當善自調伏，　　消滅士夫火， 　正信為大河，　　淨戒為度濟， 　澄淨清流水，　　智者之所歎！ 　人中淨天德，　　當於中洗浴， 　涉水不著身，　　安樂度彼岸。 　正法為深淵，　　福德為下濟， 　澄淨水充滿，　　智者所讚嘆！ 　人中天淨德，　　當於中洗浴， 　涉水不著身，　　安樂度彼岸。 　真諦善調御，　　攝護修梵行， 　慈悲為苦行，　　真實心清淨， 　沐浴以正法，　　智者所稱歎。」

爾時，孫陀利河側婆羅門聞佛所說，歡喜隨喜，復道而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛在拘薩羅人間遊行，宿於孫陀利河側。**
　　在某一個時間，佛在拘薩羅國境內遊行，夜晚住在孫陀利河邊。

**◈ 爾時，世尊剃髮未久，於後夜時，結跏趺坐，正身思惟，繫念在前，以衣覆頭。**
　　那時，世尊剛剃髮不久，在深夜時分，採取結跏趺坐（雙腿盤繞的禪定坐姿），調整身體姿勢進入深層的思考與觀照，將正念（對當下的覺知）專注於前方，並用衣服覆蓋頭部。

**◈ 時，孫陀利河側有婆羅門住止，夜起持祠，餘食不盡，時至河邊，欲求大德婆羅門以奉之。**
　　當時，住在孫陀利河邊的一位婆羅門在深夜起身舉行祭祀，祭祀後的剩餘食物沒有吃完，於是來到河邊，想要尋找一位德高望重的婆羅門將食物供養給他。

**◈ 爾時，世尊聞河邊婆羅門聲。聞已，謦咳作聲，却衣現頭。**
　　那時，世尊聽到了河邊婆羅門的聲音。聽完後，世尊輕輕咳嗽發聲，移開覆頭的衣服，露出頭部。

**◈ 時，孫陀利河側婆羅門見佛已，作是念：「是剃頭沙門，非婆羅門。」欲持食還去。**
　　那時，孫陀利河邊的婆羅門看到佛後心想：「這是一個剃頭的沙門（出家修行者），不是婆羅門。」於是打算拿著食物離開。

**◈ 彼婆羅門復作是念：「非獨沙門是剃頭者，婆羅門中亦有剃頭，應往至彼，問其所生。」時，孫陀利河側婆羅門詣世尊所，而問之言：「為何姓生？」**
　　但這位婆羅門隨即又想：「並非只有沙門才剃頭，婆羅門階級中也有剃頭的人，我應該走到他面前，詢問他的出身。」於是，這位婆羅門走到世尊面前，詢問道：「您出生於哪個種姓？」

**◈ 爾時，世尊即說偈言：「汝莫問所生，但當問所行。」**
　　那時，世尊立即以偈頌回答：「你不要詢問我的出身種姓，而應該詢問我的行為操守。」

**◈ 「刻木為鑽燧，亦能生於火；下賤種姓中，生堅固牟尼。」**
　　「就像用木頭刻成鑽火棒，也能摩擦出火來；同樣地，即使在被視為低賤的種姓中，也能誕生出意志堅定、覺悟的牟尼（聖者）。」

**◈ 「智慧有慚愧，精進善調伏，究竟大明際，清淨修梵行。」**
　　「具備智慧且心中有慚愧心（對過錯的自覺），透過精進（持續不懈的努力）來善於調伏內心，最終能達到大光明之境，清淨地實踐梵行（清淨且不染污的修行生活）。」

**◈ 「而今正是時，應奉施餘食。」**
　　「現在正是時候，你應該將剩餘的食物供養出來。」

**◈ 時，孫陀利河側婆羅門復說偈言：「我今吉良日，求福修供養，遇得見大士，三時最勝尊，若不見佛者，當更施餘人。」**
　　那時，孫陀利河邊的婆羅門再次以偈頌說道：「我今天真是幸運的日子，正想尋求福德而進行供養，竟然遇見了如此偉大的聖者，這位在三世中最殊勝的尊者。如果今天沒見到佛，我本來打算將食物施捨給其他人。」

**◈ 爾時，孫陀利河側婆羅門轉得信心，即持餘食以奉世尊，世尊不受，以說偈得故，如上因說偈而得食廣說。**
　　那時，孫陀利河邊的婆羅門轉而產生了信心，立即持著剩餘的食物供養世尊。但世尊並沒有接受，而是透過說偈頌來引導（如前文所述，透過對話讓對方領悟後才得食的廣義說法）。

**◈ 孫陀利河側婆羅門白佛言：「世尊！今此施食當置何所？」**
　　孫陀利河邊的婆羅門對佛陀恭敬地說道：「世尊！現在這份施捨的食物應該放在什麼地方？」

**◈ 佛告婆羅門：「我不見諸天、魔、梵、沙門、婆羅門、天神、世人有能食此食，令身安隱者。汝持此食去著無虫水中，及少生草地。」**
　　佛告訴婆羅門：「我看不到無論是天人、魔王、梵天、沙門、婆羅門、天神或是凡人，有誰能吃下這份（祭祀剩餘的）食物而能讓身體安穩。你請將這份食物拿去放在沒有蟲子的水中，或是少有雜草的地上。」

**◈ 時，婆羅門即持此食著無虫水中，水即烟起涌沸，啾啾作聲，如燒鐵丸投之冷水，烟起涌沸，啾啾作聲。**
　　那時，婆羅門立即將食物放入沒有蟲子的水中，水竟然立刻冒煙沸騰，發出啾啾的聲響，就像將燒紅的鐵丸投入冷水之中一樣，冒煙沸騰，發出啾啾聲。

**◈ 如是彼食著無虫水中，烟起涌沸，啾啾作聲。孫陀利河側婆羅門心欲恐怖，身毛皆竪，謂為災變，馳走上岸，集聚乾木，供養祠火，令息災恠。**
　　就這樣，食物放入水中後，水面煙起沸騰，發出啾啾聲。孫陀利河邊的婆羅門內心感到極其恐怖，全身汗毛豎起，認為發生了災難，趕緊跑上岸，收集乾木頭來供養祭祀之火，希望能消除災禍。

**◈ 世尊見彼集聚乾木，供養祠火，望息災恠。見已。即說偈言：**
　　世尊看到他收集乾木、供養祭火以期望消除災禍，見到後立即說偈頌：

**◈ 「婆羅門祠火，焚燒乾草木，莫呼是淨道，能却諸災患。」**
　　「婆羅門舉行祭火儀式，焚燒乾草木，不要稱之為清淨的道路，以為這樣能除去各種災患。」

**◈ 「此則惡供養，而謂為黠慧；作如是因緣，外道取修淨。」**
　　「這種做法其實是錯誤的供養，卻被誤以為是聰明睿智；外道正是以此類因緣（儀式）來追求所謂的清淨。」

**◈ 「汝今棄薪火，起內火熾然，常修不放逸，常富於供養。」**
　　「你現在應該捨棄外部的薪火，而在內心燃起（覺悟的）火，經常修行不放逸（時刻警覺，不讓心隨業力漂流），如此才能在精神上富足於供養。」

**◈ 「處處興淨信，廣施設大會，心意為束薪，瞋恚黑烟起，妄語為塵味，口舌為木杓，胸懷燃火處，欲火常熾然。」**
　　「（如果你不修行），雖然處處建立淨信、舉辦盛大集會，但你的心意就像是捆綁的柴薪，瞋恚（憤怒）會化作黑煙升起，妄語（謊言）則是塵垢的味道，口舌就像攪動火堆的木杓，而胸懷則是燃火之處，貪欲之火在其中經常熾熱燃燒。」

**◈ 「當善自調伏，消滅士夫火，正信為大河，淨戒為度濟，澄淨清流水，智者之所歎！」**
　　「應該善於調伏自己的內心，熄滅凡夫的煩惱之火。將正信（正確的信仰）視為大河，將淨戒（清淨的戒律）視為渡河的船隻，這才是澄淨清澈的流水，是智者所讚嘆的！」

**◈ 「人中淨天德，當於中洗浴，涉水不著身，安樂度彼岸。」**
　　「在人間擁有如天人般清淨的德行，應在其中洗浴，涉水而過而身體不被染污，安樂地到達彼岸（涅槃）。」

**◈ 「正法為深淵，福德為下濟，澄淨水充滿，智者所讚嘆！」**
　　「將正法（佛陀的真理）視為深邃的淵池，將福德視為下方的基底，其中充滿了澄淨的水，這是智者所讚嘆的！」

**◈ 「人中天淨德，當於中洗浴，涉水不著身，安樂度彼岸。」**
　　「在人間擁有如天人般清淨的德行，應在其中洗浴，涉水而過而身體不被染污，安樂地到達彼岸（涅槃）。」

**◈ 「真諦善調御，攝護修梵行，慈悲為苦行，真實心清淨，沐浴以正法，智者所稱歎。」**
　　「善於調御真理，攝護並實踐梵行（清淨的修行生活），將慈悲心視為真正的苦行，使真實的心變得清淨，用正法來沐浴心靈，這才是智者所稱讚的。」

**◈ 爾時，孫陀利河側婆羅門聞佛所說，歡喜隨喜，復道而去。**
　　那時，孫陀利河邊的婆羅門聽聞佛陀的教導，心中感到歡喜並隨喜，隨後沿路離開。

### [詞語註解（解釋重要佛教概念）]
1. **牟尼 (Muni)**：原意為「聖者」或「沉默者」，指透過深刻的冥想與自律，達到內心寂靜並覺悟真理的人。
2. **梵行 (Brahmacariya)**：指清淨的修行生活，不僅包含禁欲，更核心的是遠離一切染污心（貪、瞋、癡），追求精神上的絕對純淨。
3. **不放逸 (Appamāda)**：指時刻保持覺知，不讓心在散亂或貪欲中漂流，是所有修行中最重要的基礎，確保修行者能持續精進。
4. **彼岸 (Param)**：比喻從充滿苦難與輪迴的此岸（Samsāra），透過修行到達絕對寧靜、不再受苦的涅槃（Nibbāna）境界。
5. **外道 (Samaññā/Tīrthika)**：指不認同佛法緣起論，而採取其他修行路徑（如極端苦行或單純儀式祭祀）的修行者。

### [現代實作指南]
本經文提供了一個深刻的轉念視角：**將「外部儀式」轉化為「內部心靈工程」**。

1. **識別「外部薪火」**：
   - 現代人常追求外部的「淨化」或「救贖」，例如：迷信符咒、單純的捐款以求福報、或依賴某種形式的儀式感來緩解焦慮。這些就像經文中的「焚燒乾草木」，雖有形式，但無法根除內心的災患。
   - **實作**：當你感到不安而想尋求外部安慰時，試著問自己：「我現在是在燃燒外部的薪火，還是在處理內心的火？」

2. **覺察「內在之火」**：
   - 經文將「瞋恚」比作黑煙，「妄語」比作塵味，「欲火」比作熾熱之火。
   - **實作**：練習「情緒標記法」。當憤怒升起時，在心中默念：「黑煙升起了」；當強烈的慾望產生時，默念：「欲火在燃燒」。透過這種比喻，將自己與情緒抽離，從「我是憤怒的」變成「我觀察到憤怒的黑煙」。

3. **建立「正信之河」與「淨戒之船」**：
   - 真正的洗滌不是用水，而是用「正法」與「慈悲」。
   - **實作**：
     - **正信**：每天花五分鐘思考一個真理（如：無常），讓心不再執著於暫時的得失。
     - **淨戒**：設定一個簡單的「清淨目標」（例如：今天不說任何損人利己的妄語），將此作為渡河的船，保護自己不被世俗的染污所觸及。

4. **從「種姓」轉向「行為」**：
   - 佛陀強調「莫問所生，但當問所行」。
   - **實作**：停止用標籤（如：學歷、職位、家庭背景、國籍）來定義他人或自己。將關注點移至「當下的行為」➔ 「這個行為是否帶來慈悲？」➔ 「這個行為是否符合正法？」。

### [段落大意]
本經敘述佛陀在孫陀利河邊與一名婆羅門的對話。婆羅門起初以種姓區分看待佛陀，但佛陀以偈頌指出，人的價值在於行為而非出身，唯有透過智慧、精進與梵行才能獲得真正的清淨。隨後，佛陀以食物入水沸騰的奇蹟，破除婆羅門對外部祭祀火儀式的迷信，教導其應熄滅內心的瞋恚與貪欲之火，以正信、淨戒與慈悲作為洗滌心靈的法水，方能安樂地度向涅槃彼岸。
