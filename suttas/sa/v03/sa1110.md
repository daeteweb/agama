# 雜阿含經 第 1110 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「過去世時，有天帝釋，天、阿修羅對陣欲戰。釋提桓因語三十三天眾言：『今日諸天與阿修羅軍戰，諸天得勝，阿修羅不如者，當生擒毘摩質多羅阿修羅王，以五繫縛，將還天宮。』毘摩質多羅阿修羅王告阿修羅眾：『今日諸天、阿修羅共戰，若阿修羅勝，諸天不如者，當生擒釋提桓因，以五繫縛，將還阿修羅宮。』當其戰時，諸天得勝，阿修羅不如。

「時，彼諸天捉得毘摩質多羅阿修羅王，以五繫縛，將還天宮，縛在帝釋斷法殿前門下。帝釋從此門入出之時，毘摩質多羅阿修羅縛在門側瞋恚罵詈。

「時，帝釋御者見阿修羅王，身被五縛，在於門側，帝釋出入之時輒瞋恚罵詈。見已即便說偈，白帝釋言：

「『釋今為畏彼，　　為力不足耶， 　　能忍阿修羅，　　面前而罵辱。』

「釋即答言：

「『不以畏故忍，　　亦非力不足， 　　何有黠慧人，　　而與愚夫對。』

「御者復白言：

「『若但行忍者，　　於事則有闕， 　　愚癡者當言，　　畏怖故行忍。 　　是故當苦治，　　以智制愚癡。』

「帝釋答言：

「『我常觀察彼，　　制彼愚夫者， 　　見愚者瞋盛，　　智以靜默伏。 　　非力而為力，　　是彼愚癡力， 　　愚癡違遠法，　　於道則無有。 　　若使有大力，　　能忍於劣者， 　　是則為上忍。　　無力何有忍？ 　　於他極罵辱，　　大力者能忍， 　　是則為上忍。　　無力何所忍？ 　　於己及他人，　　善護大恐怖， 　　知彼瞋恚盛，　　還自守靜默。 　　於二義俱備，　　自利亦利他。 　　謂言愚夫者，　　以不見法故，　　愚夫謂勝忍，　　重增其惡口， 　　未知忍彼罵，　　於彼常得勝。 　　於勝己行忍，　　是名恐怖忍。 　　於等者行忍，　　是名忍諍忍。 　　於劣者行忍，　　是則為上忍。』」

佛告諸比丘：「釋提桓因於三十三天為自在王，常行忍辱，讚歎於忍。汝等比丘正信非家，出家學道，亦應如是行忍，讚歎於忍，應當學！」

佛說經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「過去世時，有天帝釋，天、阿修羅對陣欲戰。**
　　那時，世尊告訴在場的眾位比丘說：「在過去的世間，有天帝釋（釋提桓因），當時天眾與阿修羅（憤怒的半神）對陣準備開戰。

**◈ 釋提桓因語三十三天眾言：『今日諸天與阿修羅軍戰，諸天得勝，阿修羅不如者，當生擒毘摩質多羅阿修羅王，以五繫縛，將還天宮。』**
　　釋提桓因對三十三天的天眾說：『今天天軍與阿修羅軍交戰，如果天軍獲勝，而阿修羅軍失敗，就要將毘摩質多羅阿修羅王生擒，用五種繩索將他束縛，帶回天宮。』

**◈ 毘摩質多羅阿修羅王告阿修羅眾：『今日諸天、阿修羅共戰，若阿修羅勝，諸天不如者，當生擒釋提桓因，以五繫縛，將還阿修羅宮。』**
　　毘摩質多羅阿修羅王則對阿修羅眾說：『今天天眾與阿修羅共戰，如果阿修羅獲勝，而天眾失敗，就要將釋提桓因生擒，用五種繩索將他束縛，帶回阿修羅宮。』

**◈ 當其戰時，諸天得勝，阿修羅不如。**
　　當戰爭爆發時，天眾取得了勝利，而阿修羅軍則戰敗了。

**◈ 時，彼諸天捉得毘摩質多羅阿修羅王，以五繫縛，將還天宮，縛在帝釋斷法殿前門下。**
　　於是，天眾捉住了毘摩質多羅阿修羅王，用五種繩索將他束縛，帶回天宮，將他綁在帝釋管理法律的斷法殿前門口。

**◈ 帝釋從此門入出之時，毘摩質多羅阿修羅縛在門側瞋恚罵詈。**
　　每當帝釋從這扇門進出時，被綁在門邊的毘摩質多羅阿修羅就憤怒地辱罵他。

**◈ 時，帝釋御者見阿修羅王，身被五縛，在於門側，帝釋出入之時輒瞋恚罵詈。**
　　那時，帝釋的馬車御者看到阿修羅王被五種繩索束縛在門邊，且在帝釋出入時總是憤怒地辱罵。

**◈ 見已即便說偈，白帝釋言：『釋今為畏彼，為力不足耶，能忍阿修羅，面前而罵辱。』**
　　御者見狀立刻用詩偈對帝釋說：「帝釋您現在是因為害怕他，還是因為力量不足，才會忍受阿修羅在您面前如此辱罵？」

**◈ 釋即答言：『不以畏故忍，亦非力不足，何有黠慧人，而與愚夫對。』**
　　帝釋立即回答說：「我並非因為恐懼而忍耐，也不是因為力量不足，而是因為聰明且具智慧的人，不會與愚笨的人（被憤怒遮蔽心智的人）爭論。」

**◈ 御者復白言：『若但行忍者，於事則有闕，愚癡者當言，畏怖故行忍。是故當苦治，以智制愚癡。』**
　　御者再次說道：「如果僅僅是忍耐，在處理事情上會有缺失，因為愚癡的人（缺乏智慧且執著的人）會說您是因為恐懼才忍耐。因此應該採取嚴厲的手段，用智慧來制伏愚癡。」

**◈ 帝釋答言：『我常觀察彼，制彼愚夫者，見愚者瞋盛，智以靜默伏。**
　　帝釋回答說：「我經常觀察他，要制伏那個愚笨的人，看到他瞋心（憤怒的心理造作）如此強烈，智慧的做法是以靜默來降伏他。

**◈ 非力而為力，是彼愚癡力，愚癡違遠法，於道則無有。**
　　那種以為用暴力對抗才是力量的想法，其實是愚癡者的力量，而愚癡的人違背了正法（真理），在解脫之路上毫無進展。

**◈ 若使有大力，能忍於劣者，是則為上忍。無力何有忍？**
　　如果一個人擁有強大的內在力量，卻能忍受比自己低劣的人（在心靈層次較低的人），這才叫做『上忍』（最高層次的忍辱）。如果沒有力量，那種忍耐算什麼忍耐？

**◈ 於他極罵辱，大力者能忍，是則為上忍。無力何所忍？**
　　面對他人極其嚴重的辱罵，強大的人能夠忍受，這才是最高層次的忍辱。沒有力量的人，又能忍受什麼？

**◈ 於己及他人，善護大恐怖，知彼瞋恚盛，還自守靜默。**
　　對於自己和他人，能善於防護內心的恐懼與不安，知道對方的瞋心強烈，反而能保持自身的靜默。

**◈ 於二義俱備，自利亦利他。**
　　這樣做能同時具備兩種意義：既能讓自己獲益（保持心平氣和），也能讓對方獲益（不增加惡業）。

**◈ 謂言愚夫者，以不見法故，愚夫謂勝忍，重增其惡口，未知忍彼罵，於彼常得勝。**
　　那些稱呼他人為愚夫的人，是因為看不見法（真理）的緣故；而愚夫以為忍耐是失敗，因此會增加惡口辱罵，卻不知道忍受辱罵的人，實際上永遠是對方的勝者。

**◈ 於勝己行忍，是名恐怖忍。於等者行忍，是名忍諍忍。於劣者行忍，是則為上忍。』」**
　　面對比自己強大的人而忍耐，這叫做『恐怖忍』（因恐懼而忍）；面對與自己相當的人而忍耐，這叫做『忍諍忍』（為了避免爭執而忍）；而面對比自己低劣的人而忍耐，這才是最高層次的『上忍』。」

**◈ 佛告諸比丘：「釋提桓因於三十三天為自在王，常行忍辱，讚歎於忍。**
　　佛陀告訴比丘們：「釋提桓因在三十三天中作為自在王，經常實踐忍辱，並讚歎忍辱的功德。

**◈ 汝等比丘正信非家，出家學道，亦應如是行忍，讚歎於忍，應當學！」**
　　你們這些比丘，因為正信而離開家庭，出家學習解脫之道，也應該像這樣實踐忍辱，讚歎忍辱的價值，應當學習！」

**◈ 佛說經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **忍辱 (Khanti)**：指在面對逆境、痛苦或辱罵時，心不隨之起瞋心，保持平靜與不動搖的心理能力。
2. **阿修羅 (Asura)**：佛教宇宙觀中的一種眾生，具有天人的神通但瞋心極重，經常與天眾爭鬥。
3. **上忍**：最高層次的忍耐。指在擁有絕對優勢（力量、地位或智慧）的情況下，依然能以慈悲與靜默面對低劣者的挑釁，而非因恐懼或無能而忍受。
4. **瞋恚 (Kodha)**：一種強烈的厭惡與憤怒情緒，屬於五蓋之一，會遮蔽智慧並導致惡業。
5. **正法 (Saddhamma)**：符合宇宙真理、能導向解脫的教法。

### [現代實作指南]
本經文提供了一套處理「被低層次者挑釁」的心理機制，可轉化為以下實踐步驟：

1. **重新定義「強弱」**：
   ➔ 傳統觀念認為「反擊」是強，「忍耐」是弱。
   ➔ 實作轉念：將「反擊」視為被對方的情緒牽著走（被動），將「靜默」視為掌控自己心念的能力（主動）。真正的強大是「能反擊而選擇不反擊」。

2. **區分忍耐的動機**：
   ➔ **恐懼忍**：因為害怕對方而不敢發聲 ➔ 內心仍有壓抑與恐懼。
   ➔ **妥協忍**：為了維持表面和平而忍受 ➔ 內心仍有不滿。
   ➔ **智慧忍（上忍）**：意識到對方的憤怒是其內在痛苦的投射，對其產生憐憫而非對立 ➔ 內心完全平靜。

3. **實作靜默法（The Power of Silence）**：
   ➔ 當遭遇辱罵時，在心中默念：「對方的瞋心正在折磨他自己，我不需要參與這場痛苦。」
   ➔ 觀察自己的呼吸，將注意力從對方的「惡口」移回自己的「覺知」。
   ➔ 意識到：當你不再給予對方的憤怒任何反應時，對方的憤怒就失去了支撐點，這才是最高效的「制伏」。

### [段落大意]
本經透過天帝釋與阿修羅王的衝突，闡述「忍辱」的層次。帝釋面對被俘虜阿修羅王的辱罵保持靜默，並向御者解釋：真正的強大不在於以力制力，而是在擁有優勢時仍能忍受低劣者的挑釁。佛陀藉此勉勵比丘，修行者應學習這種最高層次的「上忍」，以智慧化解瞋心，達到自利利他的境界。
