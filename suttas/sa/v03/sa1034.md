# 雜阿含經 第 1034 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。

時，有長壽童子，是樹提長者孫子，身嬰重病。

爾時，世尊聞長壽童子身嬰重病，晨朝著衣持鉢，入王舍城乞食，次第到長壽童子舍。長壽童子遙見世尊，扶床欲起……乃至說三受，如叉摩修多羅廣說，乃至「病苦但增無損。是故，童子！當如是學：『於佛不壞淨，於法、僧不壞淨，聖戒成就。』當如是學。」

童子白佛言：「世尊！如世尊說四不壞淨，我今悉有，我常於佛不壞淨，於法、僧不壞淨，聖戒成就。」

佛告童子：「汝當依四不壞淨，於上修習六明分想。何等為六？謂一切行無常想、無常苦想、苦無我想、觀食想、一切世間不可樂想、死想。」

童子白佛言：「如世尊說，依四不壞淨，修習六明分想，我今悉有。然我作是念：『我命終後，不知我祖父樹提長者當云何？』」

爾時，樹提長者語長壽童子言：「汝於我所，故念且停。汝今且聽世尊說法，思惟憶念，可得長夜福利安樂饒益。」

時，長壽童子言：「我於一切諸行當作無常想、無常苦想、苦無我想、觀食想、一切世間不可樂想、死想，常現在前。」

佛告童子：「汝今自記斯陀含果。」

長壽童子白佛言：「世尊！唯願世尊住我舍食。」爾時世尊默然而許。

長壽童子即辦種種淨美飲食，恭敬供養。世尊食已，復為童子種種說法，示、教、照、喜已，從坐起而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 時，有長壽童子，是樹提長者孫子，身嬰重病。**
　　那時，有一位名叫長壽的童子，他是樹提長者的孫子，身體患了嚴重的疾病。

**◈ 爾時，世尊聞長壽童子身嬰重病，晨朝著衣持鉢，入王舍城乞食，次第到長壽童子舍。**
　　那時，世尊聽聞長壽童子身患重病，在早晨穿好衣服、拿著缽，進入王舍城乞食，依序走到了長壽童子的家中。

**◈ 長壽童子遙見世尊，扶床欲起……乃至說三受，如叉摩修多羅廣說，乃至「病苦但增無損。是故，童子！當如是學：『於佛不壞淨，於法、僧不壞淨，聖戒成就。』當如是學。」**
　　長壽童子從遠處看見世尊，扶著床想要起身……（中間省略部分，提及三種感受，如同叉摩修多羅所廣泛說明的那樣），直到說到：「病苦只會增加而不會減少。因此，童子！你應該這樣學習：『對佛陀保持不毀壞的清淨信心，對法（真理）與僧伽（僧團）保持不毀壞的清淨信心，並成就聖者的戒律。』你應該這樣學習。」

**◈ 童子白佛言：「世尊！如世尊說四不壞淨，我今悉有，我常於佛不壞淨，於法、僧不壞淨，聖戒成就。」**
　　童子對佛陀恭敬地說道：「世尊！如同世尊所說的四種不毀壞的清淨信心，我現在全部具足。我經常對佛陀不毀壞清淨，對法（真理）與僧伽（僧團）不毀壞清淨，且成就了聖者的戒律。」

**◈ 佛告童子：「汝當依四不壞淨，於上修習六明分想。何等為六？謂一切行無常想、無常苦想、苦無我想、觀食想、一切世間不可樂想、死想。」**
　　佛陀告訴童子：「你應該依據這四種不毀壞的清淨信心，在更高層次地修習六種明分想（覺悟的觀想）。什麼是六種？指的是：對於所有因緣而生的現象（一切行）皆是無常的觀想、因為無常而導致苦（不滿足）的觀想、因為是苦所以其中沒有永恆不變的自我（無我）的觀想、對於飲食之目的（僅為維持生命以修行）的觀想、對於整個世間都沒有真正永恆快樂的觀想，以及對於死亡必然到來的觀想。」

**◈ 童子白佛言：「如世尊說，依四不壞淨，修習六明分想，我今悉有。然我作是念：『我命終後，不知我祖父樹提長者當云何？』」**
　　童子對佛陀恭敬地說道：「如同世尊所說，依據四不壞淨而修習六明分想，我現在全部具足。然而我心中在想：『在我生命結束之後，不知道我的祖父樹提長者會如何？』」

**◈ 爾時，樹提長者語長壽童子言：「汝於我所，故念且停。汝今且聽世尊說法，思惟憶念，可得長夜福利安樂饒益。」**
　　那時，樹提長者對長壽童子說：「你對我的掛念可以停止了。你現在且聽世尊說法，思惟並憶念，可以獲得長久的福德、安樂與饒益。」

**◈ 時，長壽童子言：「我於一切諸行當作無常想、無常苦想、苦無我想、觀食想、一切世間不可樂想、死想，常現在前。」**
　　那時，長壽童子說：「我對於一切諸行（因緣而生的現象）將會恆常保持無常想、無常苦想、苦無我想、觀食想、一切世間不可樂想以及死想，讓這些觀想常現在前。」

**◈ 佛告童子：「汝今自記斯陀含果。」**
　　佛陀告訴童子：「你現在可以自我認定已證得斯陀含（一次還來）果位。」

**◈ 長壽童子白佛言：「世尊！唯願世尊住我舍食。」爾時世尊默然而許。**
　　長壽童子對佛陀恭敬地說道：「世尊！唯願世尊留在我的家中用餐。」那時世尊默然地答應了。

**◈ 長壽童子即辦種種淨美飲食，恭敬供養。世尊食已，復為童子種種說法，示、教、照、喜已，從坐起而去。**
　　長壽童子立刻準備了各種純淨美味的飲食，恭敬地供養。世尊用餐完畢後，再次為童子說了種種法，在示導、教導、照耀與給予歡喜之後，從座位上起身離開了。

### [詞語註解]
1. **四不壞淨 (Cattāri asammisā saddhā)**：指對佛、法、僧三寶的堅定信心，以及對聖戒的持守，這四者在面對困難或誘惑時不會毀壞，是進入聖道之基礎。
2. **六明分想 (Cha-vimamsa)**：一種系統性的觀想修行，旨在透過邏輯遞進地體悟真理：
    *   **一切行無常想**：體悟所有有為法（因緣合成之物）皆在變遷。
    *   **無常苦想**：體悟因事物變遷而產生的不滿足感即是苦。
    *   **苦無我想**：體悟既然一切皆是苦且無常，則不存在一個永恆不變的「我」。
    *   **觀食想**：反思飲食僅是為了維持生命以修行，而非為了口腹之欲。
    *   **一切世間不可樂想**：體悟世間之樂皆是暫時且有條件的，並非真正的樂。
    *   **死想**：正視死亡的必然性，以此激發修行之緊迫感。
3. **斯陀含 (Sakadāgāmī)**：意為「一次還來者」。在四果位中屬於第二階段，已斷除貪、嗔、癡中的部分粗顯煩惱，僅需再回到人間一次即可證得阿羅漢果。
4. **諸行 (Sankhāra)**：指所有由因緣而生、處於不斷變遷中的現象或心理造作。

### [現代實作指南]
本經文提供了一套在面對「極端壓力」（如重病、死亡威脅）時的心理轉化路徑，可轉化為以下實踐步驟：

1. **建立心理基石（對應四不壞淨）**：
    *   在危機中，首先尋找一個絕對信任的價值體系或導師（如佛法），建立內心的安全感與定力，避免在恐慌中崩潰。

2. **執行邏輯遞進的認知重構（對應六明分想）**：
    *   **第一步：接受變動** ➔ 當面對病痛或失敗時，告訴自己：「這件事（行）本身就是無常的，變動是自然法則。」
    *   **第二步：定義痛苦** ➔ 意識到痛苦源於「希望無常之物能恆常」的執著。
    *   **第三步：解構自我** ➔ 觀察痛苦時，問自己：「這個痛苦是『我』嗎？還是僅僅是一種感受（受）在運作？」藉此降低對「我受苦」的認同感。
    *   **第四步：簡化需求** ➔ 練習「觀食想」，將生活中的消費與飲食簡化為「維持功能」而非「追求快感」。
    *   **第五步：降低期待** ➔ 體認到世間沒有完美的快樂，減少對外部環境的依賴。
    *   **第六步：正視終點** ➔ 每日花三分鐘思考死亡，這並非消極，而是為了剔除生命中不重要的雜質，專注於當下的覺醒。

3. **處理情感依附（對應樹提長者的教導）**：
    *   當我們因擔心他人而無法專注於自身成長時，應意識到：最好的關懷是讓自己先獲得內心的安樂與智慧，而非在共同的焦慮中沉淪。

### [段落大意]
本經敘述佛陀探訪重病之長壽童子，指導其建立對三寶與戒律的堅定信心（四不壞淨），並修習從無常、苦、無我到死想的六種觀想（六明分想）。童子在樹提長者的鼓勵下，放下對親人的執著，專注於對諸行無常的體悟，最終證得斯陀含果位。
