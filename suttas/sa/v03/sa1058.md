# 雜阿含經 第 1058 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「三十法成就者，如鐵鉾鑽水，身壞命終，下生惡趣泥犁中。何等為三十法？謂自手殺生，教人令殺，讚歎殺生，乃至自行邪見，復以邪見教人令行，常復讚歎行邪見者，是名三十法。如鐵鉾鑽水，身壞命終，下生惡趣泥犁中。有三十法成就者，如鐵鉾鑽空，身壞命終，上生天上。何等為三十法？謂自不殺生，教人不殺，常復讚歎不殺功德；乃至自行正見，復以正見教人令行，常復讚歎正見功德，是名三十法成就。如鐵鉾鑽空，身壞命終，上生天上。」

諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「三十法成就者，如鐵鉾鑽水，身壞命終，下生惡趣泥犁中。**
　　「那些成就了三十種惡行的人，就像鐵鑽頭鑽入水中（毫不費力地沉沒），在身體毀壞、生命結束之後，會下生到惡道中的泥犁（地獄）之中。

**◈ 何等為三十法？**
　　所謂的三十種惡行是指什麼呢？

**◈ 謂自手殺生，教人令殺，讚歎殺生，**
　　也就是指：親手殺害生命、教唆他人殺生、讚歎殺生的行為，

**◈ 乃至自行邪見，復以邪見教人令行，常復讚歎行邪見者，是名三十法。**
　　直到於自身持有邪見（錯誤的見解），又用這種邪見教導他人去實踐，並且經常讚歎持有邪見的人，這就稱為成就了三十種惡行。

**◈ 如鐵鉾鑽水，身壞命終，下生惡趣泥犁中。**
　　這樣的人就像鐵鑽頭鑽入水中，在身體毀壞、生命結束後，會下生到惡道中的泥犁（地獄）之中。

**◈ 有三十法成就者，如鐵鉾鑽空，身壞命終，上生天上。**
　　而那些成就了三十種善行的人，就像鐵鑽頭鑽入虛空（輕盈上升），在身體毀壞、生命結束後，會上生到天界之中。

**◈ 何等為三十法？**
　　所謂的三十種善行是指什麼呢？

**◈ 謂自不殺生，教人不殺，常復讚歎不殺功德；**
　　也就是指：自身不殺生、教導他人不殺生，並且經常讚歎不殺生的功德；

**◈ 乃至自行正見，復以正見教人令行，常復讚歎正見功德，是名三十法成就。**
　　直到於自身持有正見（正確的見解），又用這種正見教導他人去實踐，並且經常讚歎正見的功德，這就稱為成就了三十種善行。

**◈ 如鐵鉾鑽空，身壞命終，上生天上。」**
　　這樣的人就像鐵鑽頭鑽入虛空，在身體毀壞、生命結束後，會上生到天界之中。」

**◈ 諸比丘聞佛所說，歡喜奉行。**
　　眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **泥犁 (Niraya)**：梵語意為「無光」或「無喜」，指佛教宇宙觀中最痛苦的惡道地獄，是因極重惡業而導致的輪迴去處。
2. **邪見 (Micchā-diṭṭhi)**：指錯誤的見解或歪曲的認知，特別是指否認因果報應、否認業力作用的見解，這被視為最危險的心理狀態，因為它會導向錯誤的行為。
3. **正見 (Sammā-diṭṭhi)**：八正道之首，指正確的見解。在原始佛教中，正見首先是指對四聖諦（苦、集、滅、道）的正確認知，以及對因果律的認同。
4. **惡趣 (Duggati)**：指不幸福的輪迴狀態，通常包括地獄、餓鬼、畜生三道。

### [現代實作指南]
本經揭示了一個重要的「影響力乘數」邏輯：一個人的業力不僅來自於「親自實踐」，還來自於「教導他人」與「讚歎鼓勵」。在現代生活中，我們可以將其轉化為以下實踐：

1. **意識影響力的擴散**：
   ➔ 意識到我們在社群媒體上的「讚（Like）」或「分享」其實就是一種「讚歎」與「教導」。當我們讚美暴力、仇恨或錯誤的價值觀時，實際上是在加強該惡行的能量，並將自己與該業力連結。
   ➔ 實作：在按下分享或讚美前，自問：「這個行為是在鼓勵正見（正確的認知）還是邪見（偏激或錯誤的認知）？」

2. **建立正向的價值鏈**：
   ➔ 善行不應僅止於個人不作惡。本經鼓勵我們透過「教導」與「讚歎」來擴大善的影響力。
   ➔ 實作：主動讚美他人表現出的慈悲、誠實或理智。當你讚歎一個人的正見時，你不僅在鼓勵對方，也在強化自身的正見。

3. **審視認知底層（正見 vs 邪見）**：
   ➔ 行為是認知的產物。殺生或不殺生，底層在於對「生命」的認知。
   ➔ 實作：練習「因果觀察」。當面對衝突時，試著思考：「如果我採取這個行動，會產生什麼樣的連鎖反應？」將注意力從「我想贏」轉向「這會造成什麼果報」，這就是將邪見轉化為正見的過程。

### [段落大意]
本經說明了行為、教導與讚歎三者如何共同構成業力的累積。成就「三十種惡行」（親自作惡、教人作惡、讚歎作惡）會導致身後墮入泥犁地獄，如同鐵鑽入水般迅速沉淪；而成就「三十種善行」（親自行善、教人行善、讚歎行善）則會導致身後升至天界，如同鐵鑽入空般輕盈上升。核心在於強調正見與邪見對生命去向的決定性影響。
