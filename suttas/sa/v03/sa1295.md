# 雜阿含經 第 1295 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「車從何處起？　　誰能轉於車？ 　車轉至何所，　　何故壞磨滅？」

爾時，世尊說偈答言：

「車從諸業起，　　心識能於車， 　隨因而轉至，　　因壞車則亡。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

於是，天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出眾的天子，在深夜時分來到佛陀所在之處，恭敬地向佛陀的足下頂禮，然後退到一邊坐下。他的身體散發出光明，照亮了整個祇樹給孤獨園。

**◈ 時，彼天子說偈問佛：**
　　那時，這位天子用偈頌的形式請問佛陀：

**◈ 「車從何處起？誰能轉於車？車轉至何所，何故壞磨滅？」**
　　「這輛代表生命輪迴的車（輪迴之車）是從何處開始運行的？是誰在驅動這輛車轉動？這輛車會轉向何方？又是因為什麼原因而毀壞磨滅？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「車從諸業起，心識能於車，隨因而轉至，因壞車則亡。」**
　　「這輛輪迴之車是由各種業（Kamma，由心、口、身造作的行為）所引起的；是識（Viññāṇa，意識）在驅動這輛車；它隨著因緣（Paṭicca-samuppāda，條件的組合）而轉向不同的處所；而當這些因緣毀壞時，這輛車也就隨之消失。」

**◈ 時，彼天子復說偈言：**
　　那時，這位天子再次說偈頌道：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「我曾見到許多婆羅門，最終達到了涅槃（Nibbāna，超越生死、絕對寧靜的解脫境界），所有的恐懼都已消除，永遠超越了世間的貪愛與執著。」

**◈ 於是，天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　於是，天子聽聞佛陀所說的教法後，內心感到歡喜並隨喜，向佛陀的足下頂禮，隨即消失不見。

### [詞語註解]
1. **業 (Kamma)**：指由意圖（思）所驅動的身體、言語、心靈行為，是導致輪迴與果報的核心動力。
2. **識 (Viññāṇa)**：五蘊中的一種，指對對象的覺知或意識，在十二因緣中，識是承接無明與行，導向名色產生的關鍵環節。
3. **因緣 (Paṭicca-samuppāda)**：指事物並非獨立存在，而是依據特定的條件（緣）而生起，條件消失則現象滅除的運作規律。
4. **涅槃 (Nibbāna)**：指熄滅貪、嗔、痴三毒，徹底終結輪迴苦痛，進入絕對寧靜且不生不滅的解脫狀態。

### [現代實作指南]
本經以「車」比喻生命的輪迴與運作，我們可以將其轉化為對日常心理慣性的觀察：

1. **識別「輪迴之車」的啟動（業的觀察）**：
   當你發現自己陷入重複的負面情緒模式（例如：每次被批評就感到憤怒 → 反擊 → 後悔），這就是你的「輪迴之車」在運轉。請意識到，啟動這輛車的是過去累積的「業」（習慣性的反應模式）。

2. **覺察「駕駛員」（識的觀察）**：
   在情緒爆發的瞬間，試著抽離，觀察那個「正在感受憤怒」的意識（識）。問自己：「現在是誰在驅動這輛憤怒之車？」當你能將「我很憤怒」轉變為「我意識到心中升起了一股憤怒」，你便從駕駛座移到了觀察座，打破了自動運轉的慣性。

3. **切斷「燃料」與「路徑」（因緣的轉化）**：
   輪迴之車依因緣而轉。憤怒的「因」可能是對自我的執著，「緣」可能是對方的言語。實作技巧是：在「觸（接觸）」與「受（感受）」之間創造一個空隙，不讓「愛（貪愛）」或「取（執取）」接手。當你不再對該情境產生執著，驅動車輛的燃料就消失了，輪迴之車自然停止。

4. **追求內在的「寧靜點」（涅槃的實踐）**：
   練習在日常生活中尋找短暫的「無我」狀態。當你全然地專注於當下，不評判、不渴求、不排斥，那一刻的平靜就是微小的涅槃體驗。透過不斷練習，將這種「超越恩愛（執著）」的狀態擴展，減少對外在認同的依賴。

### [段落大意]
一位天子以「車」比喻生命輪迴，請問佛陀輪迴的起因、驅動力與終結。佛陀開示：輪迴之車由「業」引起，由「識」驅動，隨「因緣」而轉，隨「因緣毀壞」而滅。天子感悟到解脫涅槃能超越世間執著與恐懼，隨後歡喜頂禮並消失。
