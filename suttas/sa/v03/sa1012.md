# 雜阿含經 第 1012 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「誰隱彼世間，　　誰繫於世間， 　誰憶於眾生，　　誰建眾生幢？」

爾時，世尊說偈答言：

「無明覆世間，　　愛結縛眾生， 　隱覆憶眾生，　　我慢眾生幢。」

時，彼天子即復說偈而問佛言：

「誰無有覆蓋，　　誰復無愛結， 　誰即出隱覆，　　誰不建慢幢？」

爾時，世尊說偈答言：

「如來等正覺，　　正智心解脫， 　不為無明覆，　　亦無愛結繫， 　超出於隱覆，　　摧滅我慢幢。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有一天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。身諸光明遍照祇樹給孤獨園。**
　　那時，有一位天子，容貌極其絕妙，在深夜時分來到佛陀這裡，頂禮佛陀的足部，然後退到一邊坐下。他身體散發出的光芒遍照整個祇樹給孤獨園。

**◈ 時，彼天子說偈問佛：**
　　那時，這位天子用偈頌的形式請問佛陀：

**◈ 「誰隱彼世間，誰繫於世間，誰憶於眾生，誰建眾生幢？」**
　　「是什麼遮蔽了這個世間？是什麼將眾生繫縛在世間之中？是什麼讓眾生陷入迷惘與憶念？又是什麼讓眾生建立起傲慢的旗幟（我慢）？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「無明覆世間，愛結縛眾生，隱覆憶眾生，我慢眾生幢。」**
　　「無明（對真理的無知）遮蔽了世間，愛（對感官與存在的貪愛）結縛了眾生，這種遮蔽讓眾生陷入迷惘，而我慢（對自我的執著與傲慢）則成了眾生所建立的旗幟。」

**◈ 時，彼天子即復說偈而問佛言：**
　　那時，這位天子立刻再次用偈頌請問佛陀：

**◈ 「誰無有覆蓋，誰復無愛結，誰即出隱覆，誰不建慢幢？」**
　　「誰能沒有這種遮蔽？誰能不再有貪愛的繫縛？誰能從遮蔽中解脫出來？誰能不再建立傲慢的旗幟？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「如來等正覺，正智心解脫，不為無明覆，亦無愛結繫，超出於隱覆，摧滅我慢幢。」**
　　「如來（成就不變真理而來到世間的人）等正覺者，以正智使心獲得解脫（Vimokkha），不會被無明（對真理的無知）所遮蔽，也沒有貪愛的繫縛，超越了所有遮蔽，摧毀了我慢的旗幟。」

**◈ 時，彼天子復說偈言：**
　　那時，這位天子再次說偈頌道：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「我久仰這位婆羅門（指佛陀），終於達到了涅槃（超越生死、絕對寧靜的解脫境界），所有的恐懼都已過去，永遠超越了世間的恩愛與執著。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，這位天子聽聞佛陀所說的教法，心中生起法喜並隨喜，頂禮佛陀的足部，隨即消失不再出現。

### [詞語註解]
1. **無明 (Avijjā)**：指對四聖諦以及緣起真理的無知，是導致輪迴與苦受的最根本原因。
2. **愛 (Taṇhā)**：指對感官快感的渴求、對存在的貪欲或對不存在的厭惡，是將眾生繫縛在輪迴中的直接動力。
3. **我慢 (Māna)**：指基於對「我」之實體的錯誤認知而產生的傲慢、自大或對自我價值的執著。
4. **解脫 (Vimokkha)**：指心靈從貪、嗔、痴三毒以及五蘊的束縛中完全釋放，不再受制於因緣的牽引。
5. **涅槃 (Nibbāna)**：指貪愛與仇恨的熄滅，是超越生死輪迴、絕對寧靜且不再受苦的最高境界。

### [現代實作指南]
本經文揭示了人類痛苦的三個核心層次：**認知盲區（無明）➔ 情緒牽引（愛）➔ 自我膨脹（我慢）**。在現代生活中，我們可以透過以下步驟實作轉念：

1. **識別「遮蔽」 (對抗無明)**：
   ➔ 當你感到憤怒或焦慮時，意識到這是一種「遮蔽」。問自己：「我現在看到的真相是什麼？還是我只看到了我想看到的（或我害怕看到的）？」透過覺察，將注意力從「對象」轉向「觀察自己的反應」，打破無明的覆蓋。

2. **觀察「繫縛」 (對抗愛/貪愛)**：
   ➔ 當你強烈渴望某件物品或某人的認同時，意識到這是一種「繫縛」。練習將「想要」轉化為「觀察想要」。告訴自己：「這份渴望正在產生，它試圖將我繫縛在對結果的執著中。」透過這種距離感，降低貪愛對心靈的控制力。

3. **放下「旗幟」 (對抗我慢)**：
   ➔ 當你感到被冒犯、或在成就中感到優越時，意識到你正在建立「我慢之幢」。試著將「我」這個主體淡化，將成就視為眾多因緣（環境、他人、時機）的共同結果，而非單一自我的功勞。將「我比他強」轉化為「我們都在學習」。

### [段落大意]
一位天子請問佛陀，是什麼遮蔽並繫縛了眾生，以及誰能從中解脫。佛陀回答：無明遮蔽世間，貪愛繫縛眾生，而我慢則讓眾生執著於自我。唯有如來等正覺者，能以正智獲得解脫，摧毀我慢並超越無明與貪愛。天子聞後深信不疑，讚嘆佛陀證得涅槃，隨即歡喜消失。
