# 雜阿含經 第 1307 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有赤馬天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。

時，彼赤馬天子白佛言：「世尊！頗有能行過世界邊，至不生、不老、不死處不？」

佛告赤馬：「無有能過世界邊，至不生、不老、不死處者。」

赤馬天子白佛言：「奇哉！世尊！善說斯義。如世尊說言：『無過世界邊，至不生、不老、不死處者。』所以者何？世尊！我自憶宿命，名曰赤馬，作外道仙人，得神通，離諸愛欲。我時，作是念：『我有如是捷疾神足，如健士夫，以利箭橫射過多羅樹影之頃，能登一須彌，至一須彌，足躡東海，超至西海。』我時，作是念：『我今成就如是捷疾神力，今日寧可求世界邊。』作是念已，即便發行，唯除食息便利，減節睡眠，常行百歲，於彼命終，竟不能得過世界邊，至不生、不老、不死之處。」

佛告赤馬：「我今但以一尋之身，說於世界、世界集、世界滅、世界滅道跡。赤馬天子！何等為世間？謂五受陰。何等為五？色受陰、受受陰、想受陰、行受陰、識受陰，是名世間。何等為色集？謂當來有愛，貪、喜俱，彼彼染著，是名世間集。云何為世間滅？若彼當來有愛，貪、喜俱，彼彼染著無餘斷、捨、離、盡、無欲、滅、息、沒，是名世間滅。何等為世間滅道跡？謂八聖道，正見、正志、正語、正業、正命、正方便、正念、正定，是名世間滅道跡。

「赤馬！了知世間，斷世間；了知世間集，斷世間集；了知世間滅，證世間滅；了知世間滅道跡，修彼滅道跡。赤馬！若比丘於世間苦若知、若斷，世間集若知、若斷，世間滅若知、若證，世間滅道跡若知、若修。赤馬！是名得世界邊，度世間愛。」

爾時，世尊重說偈言：

「未曾遠遊行，　　而得世界邊， 
　無得世界邊，　　終不盡苦邊。 
　以是故牟尼，　　能知世界邊， 
　善解世界邊，　　諸梵行已立。 
　於彼世界邊，　　平等覺知者， 
　是名賢聖行，　　度世間彼岸。」

是時，赤馬天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有赤馬天子，容色絕妙，於後夜時來詣佛所，稽首佛足，退坐一面。其身光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出眾的赤馬天子，在深夜時分來到佛陀面前，恭敬地向佛陀的足下頂禮，然後退到一邊坐下。他身體散發出的光明遍照了整個祇樹給孤獨園。

**◈ 時，彼赤馬天子白佛言：「世尊！頗有能行過世界邊，至不生、不老、不死處不？」**
　　這時，赤馬天子對佛陀恭敬地說道：「世尊！請問是否有人能夠走到世界的邊緣，到達一個不再經歷出生、不再衰老、不再死亡的地方？」

**◈ 佛告赤馬：「無有能過世界邊，至不生、不老、不死處者。」**
　　佛陀告訴赤馬天子：「沒有任何人能夠透過空間的移動走到世界的邊緣，進而到達不生、不老、不死的地方。」

**◈ 赤馬天子白佛言：「奇哉！世尊！善說斯義。如世尊說言：『無過世界邊，至不生、不老、不死處者。』**
　　赤馬天子對佛陀恭敬地說道：「太奇妙了！世尊！您說得太正確了。正如世尊所說，確實沒有人能走到世界的邊緣而到達不生、不老、不死之處。」

**◈ 所以者何？世尊！我自憶宿命，名曰赤馬，作外道仙人，得神通，離諸愛欲。**
　　為什麼這麼說呢？世尊！我回憶起自己的前世，當時名叫赤馬，是一名外道仙人，獲得了神通，且遠離了各種愛欲。

**◈ 我時，作是念：『我有如是捷疾神足，如健士夫，以利箭橫射過多羅樹影之頃，能登一須彌，至一須彌，足躡東海，超至西海。』**
　　我當時這樣想：『我擁有如此快捷的神速，就像一名強壯的弓箭手，在利箭射過多羅樹陰的極短時間內，我就能登上一個須彌山，到達另一個須彌山，足踏東海，瞬間跨越至西海。』

**◈ 我時，作是念：『我今成就如是捷疾神力，今日寧可求世界邊。』**
　　我當時又想：『我現在成就了如此快捷的神力，今天應該可以尋找到世界的邊緣。』

**◈ 作是念已，即便發行，唯除食息便利，減節睡眠，常行百歲，於彼命終，竟不能得過世界邊，至不生、不老、不死之處。」**
　　有了這個念頭後，我立即出發，除了必要的飲食與呼吸，極力減少睡眠，持續行走了一百年。直到生命結束，我依然無法走到世界的邊緣，也沒能到達不生、不老、不死的地方。」

**◈ 佛告赤馬：「我今但以一尋之身，說於世界、世界集、世界滅、世界滅道跡。**
　　佛陀告訴赤馬天子：「我現在僅僅以這一個尋（約兩公尺）之長的肉身，為你說明什麼是世界、世界的集（苦因）、世界的滅（苦止）以及世界滅的道跡（解脫之道）。」

**◈ 赤馬天子！何等為世間？謂五受陰。何等為五？色受陰、受受陰、想受陰、行受陰、識受陰，是名世間。**
　　赤馬天子！什麼叫做世間？指的是構成身心的五種要素（五蘊）：色（物質身體）、受（感受）、想（想像與認知）、行（心的造作）、識（意識），這就稱為世間。

**◈ 何等為色集？謂當來有愛，貪、喜俱，彼彼染著，是名世間集。**
　　什麼叫做世間的集（苦因）？指的是未來產生的貪愛，以及隨之而來的貪婪與喜悅，並對這些對象產生染著（執著），這就稱為世間集。

**◈ 云何為世間滅？若彼當來有愛，貪、喜俱，彼彼染著無餘斷、捨、離、盡、無欲、滅、息、沒，是名世間滅。**
　　什麼叫做世間的滅（苦止）？如果那些未來的貪愛、貪婪、喜悅以及染著，被完全地斷除、捨棄、分離、盡除，達到無欲、熄滅、平息、沒入的狀態，這就稱為世間滅。

**◈ 何等為世間滅道跡？謂八聖道，正見、正志、正語、正業、正命、正方便、正念、正定，是名世間滅道跡。**
　　什麼叫做世間滅的道跡（解脫之道）？指的是八正道：正見、正思惟（正志）、正語、正業、正命、正精進（正方便）、正念、正定，這就稱為世間滅道跡。

**◈ 「赤馬！了知世間，斷世間；了知世間集，斷世間集；了知世間滅，證世間滅；了知世間滅道跡，修彼滅道跡。**
　　赤馬！徹底了解世間（五蘊），就能斷除對世間的執著；徹底了解世間集（貪愛），就能斷除苦因；徹底了解世間滅（苦止），就能證悟解脫；徹底了解世間滅道跡（八正道），就能修行這條解脫之道。」

**◈ 赤馬！若比丘於世間苦若知、若斷，世間集若知、若斷，世間滅若知、若證，世間滅道跡若知、若修。赤馬！是名得世界邊，度世間愛。」**
　　赤馬！如果比丘對於世間的苦若能認知、若能斷除，對於世間集若能認知、若能斷除，對於世間滅若能認知、若能證悟，對於世間滅道跡若能認知、若能修行。赤馬！這才叫做真正到達了世界的邊緣，度過了世間的貪愛。」

**◈ 爾時，世尊重說偈言：**
　　那時，世尊再次說偈頌道：

**◈ 「未曾遠遊行，而得世界邊，**
　　不需要在遙遠的空間中遊行，就能到達世界的邊緣，

**◈ 無得世界邊，終不盡苦邊。**
　　若不能在心靈上體悟世界的邊緣，終究無法結束痛苦的邊際。

**◈ 以是故牟尼，能知世界邊，**
　　因此，牟尼（佛陀）能夠認知世界的邊緣，

**◈ 善解世界邊，諸梵行已立。**
　　善於理解世界的邊緣，建立了圓滿的清淨修行（梵行）。

**◈ 於彼世界邊，平等覺知者，**
　　在那個世界的邊緣，能以平等心覺知的人，

**◈ 是名賢聖行，度世間彼岸。」**
　　這才叫做賢聖的修行，真正度到了世間的彼岸。」

**◈ 是時，赤馬天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　這時，赤馬天子聽聞佛陀所說的教法，心生法喜並隨喜讚嘆，向佛陀的足下頂禮後，隨即消失不見。

### [詞語註解]
1. **五受陰 (Pañca-upādānakkhandhā)**：即五蘊。此處加「受」字，強調這五種要素（色、受、想、行、識）是被眾生執取（upādāna）而產生的感受與認同。
2. **世界邊 (Loka-cchāda/Loka-kanta)**：字面上指世界的邊緣，但在本經中被轉化為隱喻，指涉輪迴（Saṃsāra）的盡頭，即苦的終結。
3. **世間集 (Loka-samudaya)**：對應四聖諦中的「集諦」，指導致生命在世間輪迴的根本原因，即貪愛（Taṇhā）。
4. **世間滅 (Loka-nirodha)**：對應四聖諦中的「滅諦」，指貪愛完全熄滅後的涅槃狀態。
5. **世間滅道跡 (Loka-nirodha-marga)**：對應四聖諦中的「道諦」，即通往苦止的具體路徑（八正道）。
6. **梵行 (Brahmacariya)**：指清淨的修行生活，旨在追求解脫與覺悟。

### [現代實作指南]
本經的核心在於將「尋找解脫」的方向從**外部空間**轉向**內部心靈**。赤馬天子代表了許多追求極端體驗或依賴特殊能力（如神通、外部環境改變）來獲取平安的人。

**實作步驟：**
1. **停止「外部尋找」的慣性** ➔ 當我們感到痛苦或不安時，直覺往往是想「逃離」現狀（如換工作、搬家、尋找某個理想之地）。請意識到，只要「執著」還在，無論走到哪裡，痛苦都會隨之而行。
2. **將「世界」微縮至「五蘊」** ➔ 練習將注意力從外部環境移回自身。當情緒起伏時，試著分析：
    *   這是**色**（身體的不適）？
    *   這是**受**（單純的苦、樂、捨感受）？
    *   這是**想**（大腦對此感受的標籤與定義）？
    *   這是**行**（內心產生的反應或衝動）？
    *   這是**識**（意識對此現象的覺知）？
3. **識別「集」的發生** ➔ 在分析五蘊的過程中，觀察是否出現了「愛（Craving）」➔ 例如：希望快樂能持續（貪），希望痛苦快點消失（嗔）。認出這個「愛」就是導致你感到被困在世間的「集」。
4. **實踐「滅道跡」** ➔ 運用八正道中的「正念」與「正定」，不對五蘊的變動做反應，僅僅觀察而不執取。當你不再認同五蘊為「我」，你就在當下觸碰到了「世界的邊緣」。

### [段落大意]
赤馬天子曾以神通行走百年試圖尋找不生不死的「世界邊緣」而失敗。佛陀教導他，真正的世界邊緣並非地理位置，而是對構成身心的五蘊（世間）、貪愛（集）、熄滅（滅）及八正道（道）的徹底認知與實踐。透過斷除對五蘊的執著與貪愛，才能真正度過世間彼岸，終結痛苦。
