# 雜阿含經 第 1151 經

### [原文]
如是我聞：

一時，佛住舍衛國。

時，有年少阿修羅來詣佛所，於佛面前麁惡不善語，瞋罵呵責。

爾時，世尊即說偈言：

「不怒勝瞋恚，　　不善以善伏， 　惠施伏慳貪，　　真言壞妄語。 　不罵亦不虐，　　常住賢聖心， 　惡人住瞋恨，　　不動如山石。 　起瞋恚能持，　　勝制狂馬車， 　我說善御士，　　非彼攝繩者。」

時，年少阿修羅白佛言：「瞿曇！我今悔過，如愚如癡，不辯不善，於瞿曇面前訶罵毀辱。」如是懺悔已，時，阿修羅聞佛所說，歡喜隨喜，作禮而去。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國。**
　　在某一個時間，佛陀住在舍衛國。

**◈ 時，有年少阿修羅來詣佛所，於佛面前麁惡不善語，瞋罵呵責。**
　　那時，有一位年輕的阿修羅來到佛陀這裡，在佛陀面前使用粗魯惡劣且不善的言語，憤怒地辱罵與呵斥。

**◈ 爾時，世尊即說偈言：**
　　那時，世尊立刻說了偈頌（詩偈）道：

**◈ 「不怒勝瞋恚，不善以善伏，**
　　「以不憤怒來勝過瞋恚（憤怒的心理狀態），以善行來降伏不善（惡行），

**◈ 惠施伏慳貪，真言壞妄語。**
　　以布施（給予）來降伏慳貪（吝嗇與貪婪），以真實的話語來破除妄語（謊言）。

**◈ 不罵亦不虐，常住賢聖心，**
　　不辱罵也不虐待他人，心靈常時保持在賢聖（清淨覺悟）的狀態，

**◈ 惡人住瞋恨，不動如山石。**
　　即使面對處於瞋恨之中的惡人，內心依然像山石一樣不動搖。

**◈ 起瞋恚能持，勝制狂馬車，**
　　能夠控制心中升起的瞋恚（憤怒），這比控制一輛狂奔的馬車更勝一籌。

**◈ 我說善御士，非彼攝繩者。」**
　　我所說的優秀御者（能駕馭心念的人），並非僅僅是能拉住繩索的人。」

**◈ 時，年少阿修羅白佛言：「瞿曇！我今悔過，如愚如癡，不辯不善，於瞿曇面前訶罵毀辱。」**
　　這時，年輕的阿修羅對佛陀恭敬地說道：「瞿曇！我現在後悔自己的過錯，我之前的行為就像愚笨且痴迷的人一樣，言語不通達且不善，竟然在瞿曇面前呵斥辱罵與毀謗。」

**◈ 如是懺悔已，時，阿修羅聞佛所說，歡喜隨喜，作禮而去。**
　　這樣懺悔之後，這位阿修羅聽聞佛陀所說的教法，心生法喜並隨喜（認同並歡喜），向佛陀行禮後才離開。

### [詞語註解（解釋重要佛教概念）]
1. **瞋恚 (Kodha/Dosa)**：一種強烈的厭惡、憤怒或反感心理，是導致心靈不安與造惡的根源之一。
2. **慳貪 (Macchariya/Lobha)**：包含「慳」（吝嗇，不願分享）與「貪」（強烈渴望擁有），兩者共同構成對物質或名利的執著。
3. **妄語 (Musāvāda)**：故意說出與事實不符的話，旨在欺騙他人，屬於十不善業之一。
4. **賢聖心**：指一種擺脫了貪、瞋、癡等染污，處於覺悟、慈悲且平靜的心理狀態。
5. **御士 (Sārathi)**：原指駕車的人，在此經文中被用作比喻，指能夠運用正念（Sati）來駕馭自身心念、不被情緒牽著走的人。

### [現代實作指南]
本經文提供了處理人際衝突與情緒管理的高度實操技巧，可轉化為以下生活實踐：

1. **非對等反應法（以善伏不善）**：
   ➔ 當面對他人的憤怒、辱罵或惡意時，直覺反應通常是「反擊」或「防禦」。實作時，請意識到「憤怒 ➔ 憤怒」只會擴大衝突。嘗試將反應模式改為「憤怒 ➔ 平靜」或「憤怒 ➔ 慈悲」，這能迅速切斷衝突的能量迴路。

2. **建立「心理山石」的定力**：
   ➔ 練習在被批評時，將注意力從「對方的言語」移回「自己的呼吸」。將對方的辱罵視為一種「外部的聲音現象」，而非「對我人格的定義」。當你能像山石一樣不動搖，對方的憤怒將失去依附對象，最終會自行消散。

3. **區分「控制」與「駕馭」**：
   ➔ 經文提到「非彼攝繩者」，意指真正的控制不是強行壓抑（像拉繩子一樣死撐），而是像優秀的御者一樣，理解馬（情緒）的習性，用智慧引導它。
   ➔ **實作技巧**：當憤怒升起時，不要對自己說「我不准生氣」（壓抑），而要說「我注意到現在心中升起了憤怒」（覺察）。覺察即是駕馭的開始。

### 段落大意 
本經描述一名年輕阿修羅以粗魯言語辱罵佛陀，佛陀以平靜心面對，並透過偈頌教導以不怒勝瞋、以善伏不善、以真言破妄語的修行原則，強調控制心念比駕馭狂馬更困難且重要。阿修羅聞後深受觸動，由瞋恨轉為懺悔與歡喜。
