# 雜阿含經 第 1322 經

### [原文]
如是我聞：

一時，佛在摩竭提國人間遊行，與大眾俱，到富那婆藪鬼子母住處宿。

爾時，世尊為諸比丘說四聖諦相應法，所謂苦聖諦、苦集聖諦、苦滅聖諦、苦滅道跡聖諦。

爾時，富那婆藪鬼母，兒富那婆藪及鬼女欝多羅，二鬼小兒夜啼。時，富那婆藪鬼母教其男女故，而說偈言：

「汝富那婆藪，　　欝多羅莫啼， 　令我得聽聞，　　如來所說法。 　非父母能令，　　其子解脫苦， 　聞如來說法，　　其苦得解脫。 　世人隨愛欲，　　為眾苦所迫， 　如來為說法，　　令破壞生死。 　我今欲聞法，　　汝等當默然， 　時富那婆藪，　　鬼女欝多羅。 　悉受其母語，　　默然而靜聽， 　語母言善哉，　　我亦樂聞法。 　此正覺世尊，　　於摩竭勝山， 　為諸眾生類，　　演說脫苦法。 　說苦及苦因，　　苦滅滅苦道， 　從此四聖諦，　　安隱趣涅槃。 　母今但善聽，　　世尊所說法。」

時，富那婆藪鬼母即說偈言：

「奇哉智慧子，　　善能隨我心， 　汝富那婆藪，　　善歎佛導師。 　汝富那婆藪，　　及汝欝多羅， 　當生隨喜心，　　我已見聖諦。」

時，富那婆藪鬼母說是偈時，鬼子男女隨喜默然。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛在摩竭提國人間遊行，與大眾俱，到富那婆藪鬼子母住處宿。**
　　在某一個時間，佛陀在摩竭提國的人間遊行，與大眾在一起，來到富那婆藪鬼子母（Yakshini）的住處住宿。

**◈ 爾時，世尊為諸比丘說四聖諦相應法，所謂苦聖諦、苦集聖諦、苦滅聖諦、苦滅道跡聖諦。**
　　那時，世尊為眾位比丘解說關於四聖諦（四個關於苦的真理）相應的法，即是：苦聖諦（關於苦的真理）、苦集聖諦（關於苦之原因的真理）、苦滅聖諦（關於苦之止息的真理）、以及苦滅道跡聖諦（關於導向苦之止息的解脫之道的真理）。

**◈ 爾時，富那婆藪鬼母，兒富那婆藪及鬼女欝多羅，二鬼小兒夜啼。**
　　那時，富那婆藪鬼母的孩子富那婆藪以及鬼女欝多羅，這兩個鬼小兒在夜晚啼哭。

**◈ 時，富那婆藪鬼母教其男女故，而說偈言：**
　　那時，富那婆藪鬼母為了教導她的子女，而說了偈頌（詩偈）說：

**◈ 「汝富那婆藪，欝多羅莫啼，令我得聽聞，如來所說法。」**
　　「你富那婆藪，以及欝多羅，不要再啼哭，讓我也能聽聞如來（成就不變真理而來到世間的人）所說的法。」

**◈ 「非父母能令，其子解脫苦，聞如來說法，其苦得解脫。」**
　　「父母並不能讓孩子從苦（Dukkha，不滿足與不安的狀態）中解脫，只有聽聞如來所說的法，才能讓苦獲得解脫。」

**◈ 「世人隨愛欲，為眾苦所迫，如來為說法，令破壞生死。」**
　　「世間的人隨順著愛欲（貪愛），而被種種的苦所逼迫；如來為眾生說法，是為了讓眾生破壞生死的輪迴（Saṃsāra）。」

**◈ 「我今欲聞法，汝等當默然，時富那婆藪，鬼女欝多羅。」**
　　「我現在想要聽法，你們應該保持沉默。這時富那婆藪與鬼女欝多羅，」

**◈ 「悉受其母語，默然而靜聽，語母言善哉，我亦樂聞法。」**
　　「全部接受了母親的話，保持沉默地靜心聆聽，並對母親說：『好極了！我也很樂意聽法。』」

**◈ 「此正覺世尊，於摩竭勝山，為諸眾生類，演說脫苦法。」**
　　「這位正覺的世尊，在摩竭勝山，為所有類型的眾生，演說脫離苦的法。」

**◈ 「說苦及苦因，苦滅滅苦道，從此四聖諦，安隱趣涅槃。」**
　　「說到苦以及苦的原因（集）、苦的止息（滅）以及滅苦的道路（道）；透過這四聖諦，能安穩地趨向涅槃（超越生死、絕對寧靜的解脫境界）。」

**◈ 「母今但善聽，世尊所說法。」**
　　「母親現在請專心地聽世尊所說的法。」

**◈ 時，富那婆藪鬼母即說偈言：**
　　那時，富那婆藪鬼母立即說了偈頌說：

**◈ 「奇哉智慧子，善能隨我心，汝富那婆藪，善歎佛導師。」**
　　「真是奇妙的智慧孩子，能如此契合我的心意。你富那婆藪，能很好地讚歎佛陀這位導師。」

**◈ 「汝富那婆藪，及汝欝多羅，當生隨喜心，我已見聖諦。」**
　　「你富那婆藪以及你欝多羅，應該生起隨喜心（對他人善行感到快樂），因為我已經見到了聖諦。」

**◈ 時，富那婆藪鬼母說是偈時，鬼子男女隨喜默然。**
　　那時，富那婆藪鬼母說完這首偈頌，鬼子男女隨喜地保持沉默。

### [詞語註解]
1. **四聖諦 (Cattāri Ariyasaccāni)**：佛教的核心教義，包含苦（人生本質的苦）、集（苦的起因即貪愛）、滅（苦的止息即涅槃）、道（達成止息的八正道）。
2. **涅槃 (Nibbāna)**：指熄滅貪、嗔、痴三毒，徹底脫離輪迴生死，進入絕對寧靜與自由的境界。
3. **輪迴 (Saṃsāra)**：指眾生在六道中不斷出生、死亡、再出生的循環過程，其動力來自於業（Kamma）與無明。
4. **隨喜 (Anumodanā)**：指看到他人行善或獲得正法時，內心生起歡喜且不嫉妒的心情，是一種正向的心理修行。

### [現代實作指南]
本經文透過鬼母與孩子的對話，揭示了「正法」如何平息情緒動盪（夜啼）並轉化為智慧。在現代生活中，可採取以下實作步驟：

1. **識別「夜啼」的心理狀態**：
   當我們感到焦慮、不安或情緒崩潰（如同經中的夜啼）時，意識到這正是「苦（Dukkha）」的顯現。不要試圖強行壓抑，而是先覺察到：「我現在正處於不安的狀態中」。

2. **從「愛欲」中抽離**：
   經文提到「世人隨愛欲，為眾苦所迫」。當我們痛苦時，通常是因為我們執著於某個結果（愛欲/貪愛）。練習問自己：「我現在執著於什麼？我是否在要求現實必須符合我的願望？」➔ 意識到執著即是苦因。

3. **建立「聽法」的靜心空間**：
   鬼母要求孩子「默然」以聽法。在情緒激動時，練習「暫停 ➔ 呼吸 ➔ 聆聽」。停止對外界的抱怨或內心的爭辯，將注意力轉向客觀的真理（如：一切現象皆無常），讓心靈從混亂趨向平靜。

4. **實踐「隨喜」以轉化嫉妒與匱乏**：
   當看到他人獲得成功或覺悟時，練習「隨喜」。將「為什麼他有而我沒有」的匱乏感，轉化為「他能獲得，說明這條路可行」的喜悅感。這能有效打破內心的孤立感，將心靈導向開闊的狀態。

### [段落大意]
本段記載佛陀在摩竭提國逗留於鬼子母住處時，為比丘演說四聖諦。鬼子母藉此機會教導啼哭的子女，說明父母無法救子脫苦，唯有聽聞佛法、體悟四聖諦才能破除生死輪迴，趨向涅槃。子女隨即領悟並讚歎佛法，鬼母亦對子女的智慧與隨喜心表示讚許。
