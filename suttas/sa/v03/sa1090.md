# 雜阿含經 第 1090 經

### [原文]
如是我聞：

一時，佛住王舍城毘婆羅山七葉樹林石室中。

爾時，世尊夜起露地，或坐或經行。至後夜時，洗足入室，安身臥息，右脇著地，足足相累，繫念明相，正念正智，作起覺想。

時，魔波旬作是念：「沙門瞿曇住王舍城毘婆羅山七葉樹林石室中，夜起露地若坐若行，至後夜時，洗足入室而坐，右脇臥息，足足相累，繫念明相，正念正智，作起覺想。我今當往，為作留難。」化作年少，往住佛前，而說偈言：

「為因我故眠，　　為是後邊故， 　多有錢財寶，　　何故守空閑？ 　獨一無等侶，　　而著於睡眠。」

爾時，世尊作是念：「惡魔波旬欲作嬈亂。」即說偈言：

「不因汝故眠，　　非為最後邊， 　亦無多錢財，　　唯集無憂寶。 　哀愍世間故，　　右脇而臥息， 　覺亦不疑惑，　　眠亦不恐怖。 　若晝若復夜，　　無增亦無損， 　為哀眾生眠，　　故無有損減。 　正復以百槍，　　貫身常掘動，　　猶得安隱眠，　　已離內槍故。」

時，魔波旬作是念：「沙門瞿曇已知我心。」內懷憂慼，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城毘婆羅山七葉樹林石室中。**
　　在某一個時間，佛陀住在王舍城的毘婆羅山七葉樹林石室之中。

**◈ 爾時，世尊夜起露地，或坐或經行。**
　　那時，世尊在夜晚起床，在露天的地方，時而靜坐，時而經行（走禪修行）。

**◈ 至後夜時，洗足入室，安身臥息，右脇著地，足足相累，繫念明相，正念正智，作起覺想。**
　　直到後夜時分，世尊洗淨雙足進入室內，安穩地躺下休息，身體右側貼地（獅子臥），雙足重疊，保持著清明且連續的覺知，處於正念（正確的覺察）與正智（正確的理解）之中，心中保持著隨時能覺醒的意識。

**◈ 時，魔波旬作是念：「沙門瞿曇住王舍城毘婆羅山七葉樹林石室中，夜起露地若坐若行，至後夜時，洗足入室而坐，右脇臥息，足足相累，繫念明相，正念正智，作起覺想。我今當往，為作留難。」**
　　這時，魔波旬（象徵內在煩惱與外在誘惑的魔王）心想：「沙門瞿曇現在住在王舍城的毘婆羅山七葉樹林石室中，夜晚在露天處坐或行，後夜洗足入室，右側臥息，雙足相累，保持清明覺知，正念正智，準備覺醒。我現在應該前往，為他製造困難與阻礙。」

**◈ 化作年少，往住佛前，而說偈言：**
　　魔波旬幻化成一名年輕人的模樣，走到佛陀面前，說出以下偈頌：

**◈ 「為因我故眠，為是後邊故，多有錢財寶，何故守空閑？獨一無等侶，而著於睡眠。」**
　　「你是因為我的緣故而睡眠，是為了追求最後的終結（死亡或休息）嗎？你有這麼多錢財寶物，為什麼要守著這份清閑？你獨自一人沒有伴侶，卻如此執著於睡眠。」

**◈ 爾時，世尊作是念：「惡魔波旬欲作嬈亂。」**
　　那時，世尊心想：「惡魔波旬想要製造混亂與干擾。」

**◈ 即說偈言：「不因汝故眠，非為最後邊，亦無多錢財，唯集無憂寶。**
　　世尊隨即說出偈頌：「我並非因為你的緣故而睡眠，也不是為了追求最後的終結；我也沒有許多錢財，我唯一收集的是『無憂寶』（即解脫的智慧）。

**◈ 哀愍世間故，右脇而臥息，覺亦不疑惑，眠亦不恐怖。**
　　因為憐憫世間眾生，我才採取右側臥息的姿勢休息。我在覺醒時沒有疑惑，在睡眠時沒有恐怖。

**◈ 若晝若復夜，無增亦無損，為哀眾生眠，故無有損減。**
　　無論是白天還是夜晚，我的心境都沒有增加或損減；我是為了憐憫眾生而休息，因此不會有任何損失。

**◈ 正復以百槍，貫身常掘動，猶得安隱眠，已離內槍故。」**
　　即便現在用一百支長槍貫穿我的身體並不斷攪動，我依然能安穩地睡眠，因為我已經脫離了『內槍』（即內在的貪嗔痴煩惱）。」

**◈ 時，魔波旬作是念：「沙門瞿曇已知我心。」內懷憂慼，即沒不現。**
　　這時，魔波旬心想：「沙門瞿曇已經知道我的心意了。」他內心感到憂慮與沮喪，隨即消失不見。

### [詞語註解]
1. **魔波旬 (Māra)**：在原始佛教中，魔不僅是指神話中的魔王，更象徵著妨礙修行、誘發貪欲與恐懼的內在心理力量（如五蓋）。
2. **正念 (Sati)**：對當下經驗的持續覺察，不隨之起舞，亦不刻意排斥。
3. **正智 (Sampajañña)**：在覺察的基礎上，能正確理解當前狀態的本質與目的。
4. **內槍**：比喻內在的煩惱（Kilesa），如貪、嗔、痴。這些煩惱如同利槍般不斷刺痛心靈，使人無法獲得真正的安寧。
5. **無憂寶**：指透過修行而獲得的解脫智慧與涅槃境界，這種財富不隨時間流逝而損毀，亦不被盜取。

### [現代實作指南]
本經文描述了佛陀在面對外界誘惑與干擾時，如何保持內在的定力。在現代生活中，我們可以將其轉化為以下實踐：

1. **識別「內在魔波旬」**：
   當我們試圖專注於工作、學習或休息時，心中常會出現「魔波旬」的聲音。例如：在休息時感到罪惡感（「你怎麼能這麼懶散？」），或在專注時感到分心（「去看看手機吧」）。意識到這些聲音僅是「幻化」的干擾，而非真實的自我。

2. **建立「覺醒的休息」**：
   學習佛陀的「作起覺想」。休息不代表完全的昏沉，而是一種「有意識的恢復」。在睡前或休息時，保持對呼吸的輕微覺察，讓心靈在放鬆的同時不失去清明，這樣能有效減少噩夢或不安感。

3. **區分「外在痛苦」與「內在痛苦」**：
   佛陀提到即便身體被百槍貫穿也能安眠，是因為他已離「內槍」。這提示我們：生活中的壓力（外槍）雖然不可避免，但如果我們能透過轉念，減少對壓力的執著與憤怒（內槍），我們依然能在混亂的環境中找到內心的安寧。

4. **轉念技巧 ➔ 從「匱乏」轉向「富足」**：
   魔波旬用「錢財」與「孤單」來誘惑佛陀。當我們感到生活匱乏或孤單時，試著問自己：「我心中是否擁有『無憂寶』（如：善良、覺知、平靜）？」將關注點從外在的比較轉向內在的心理資產。

### [段落大意]
本段描述佛陀在王舍城修行時，即便在休息時刻仍保持正念正智。魔波旬幻化成青年，試圖以財富、陪伴與睡眠的誘惑來干擾佛陀。佛陀以偈頌回應，表明其休息是出於對眾生的憐憫，且因已根除內在煩惱（內槍），故能處於絕對的安穩與清明之中。魔波旬意識到佛陀已洞悉其心，最終沮喪消失。
