# 雜阿含經 第 1168 經

### [原文]
如是我聞：

一時，佛住拘睒彌國瞿師羅園。

爾時，世尊告諸比丘：「譬如[麩-夫+黃]麥著四衢道頭，有六壯夫執杖共打，須臾塵碎，有第七人執杖重打。諸比丘！於意云何？如[麩-夫+黃]麥聚，六人共打，七人重打，當極碎不？」

諸比丘白佛言：「如是，世尊！」

佛告諸比丘：「如是愚癡士夫六觸入處之所搥打。何等為六？謂眼觸入處，常所搥打。耳、鼻、舌、身、意觸入處，常所搥打，彼愚癡士夫為六觸入處之所搥打，猶復念求當來世有，如第七人重打令碎。

「比丘！若言是我，是則動搖；言是我所，是則動搖。未來當有，是則動搖；未來當無，是則動搖。當復有色，是則動搖；當復無色，是則動搖。當復有想，是則動搖；當復無想，是則動搖；當復非有想非無想，是則動搖。動搖故病，動搖故癰，動搖故刺，動搖故著。正觀察動搖故苦者，得不動搖心，多修習住，繫念正知。

「如動搖。如是思量虛誑，有行因愛。言我，是則為愛；言我所，是則為愛。言當來有，是則為愛；言當來無，是則為愛。當有色，是則為愛；當無色，是則為愛。當有想，是則為愛；當無想，是則為愛；當非想非非想，是則為愛。愛故為病，愛故為癰，愛故為刺。若善思觀察愛生苦者，當多住離愛心，正念正智。

「諸比丘！過去世時，阿修羅興軍與帝釋鬪。時，天帝釋告三十三天：『今日諸天、阿修羅苦戰，若諸天勝，阿修羅不如者，當生執阿修羅，縛以五繫，送還天宮。』阿修羅語其眾言：『今阿修羅軍與諸天戰，若阿修羅勝，諸天不如者，當生執帝釋，縛以五繫，還歸阿修羅宮。』當其戰諍，諸天得勝，阿修羅不如。時，三十三天生執毘摩質多羅阿修羅王，縛以五繫，還歸天宮。

「爾時，毘摩質多羅阿修羅王身被五繫，置於正法殿上，以種種天五欲樂而娛樂之。毘摩質多羅阿修羅王作是念：『唯阿修羅賢善聰慧，諸天雖善，我今且當還歸阿修羅宮。』作是念時，即自見身被五繫縛，諸天五欲自然化沒。

「毘摩質多羅阿修羅王復作是念：『諸天賢善，智慧明徹，阿修羅雖善，我今且當住此天宮。』作是念時，即自見身五縛得解，諸天五欲自然還出。

「毘摩質多羅阿修羅王乃至有如是微細之縛，魔波旬縛轉細。於是心動搖時，魔即隨縛；心不動搖，魔即隨解。是故，諸比丘！多住不動搖心，正念正智，應當學！」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住拘睒彌國瞿師羅園。**
　　在某一個時間，佛陀住在拘睒彌國的瞿師羅園。

**◈ 爾時，世尊告諸比丘：「譬如[麩-夫+黃]麥著四衢道頭，有六壯夫執杖共打，須臾塵碎，有第七人執杖重打。諸比丘！於意云何？如[麩-夫+黃]麥聚，六人共打，七人重打，當極碎不？」**
　　那時，世尊告訴在場的眾位比丘說：「譬如一些黃色的穀物掉在十字路口，有六個壯漢拿著棍子一起敲打，很快就碎成了粉末，接著又有第七個人拿著棍子再次重擊。諸位比丘！你們的看法如何？像這樣的一堆穀物，被六個人敲打，又被第七個人重擊，應該會被徹底擊碎吧？」

**◈ 諸比丘白佛言：「如是，世尊！」**
　　眾比丘對佛陀恭敬地說道：「確實如此，世尊！」

**◈ 佛告諸比丘：「如是愚癡士夫六觸入處之所搥打。何等為六？謂眼觸入處，常所搥打。耳、鼻、舌、身、意觸入處，常所搥打，彼愚癡士夫為六觸入處之所搥打，猶復念求當來世有，如第七人重打令碎。」**
　　佛陀告訴眾位比丘：「同樣地，愚癡的人也被六種觸（接觸）的入處所敲打。這六種是什麼？指的是眼部的觸（接觸）入處經常被敲打；耳、鼻、舌、身、意（心）的觸（接觸）入處也經常被敲打。那個愚癡的人已經被這六種觸（接觸）入處所敲打，卻還在心中渴望未來世能再次存在，這就像是第七個人再次重擊，使其徹底破碎。」

**◈ 「比丘！若言是我，是則動搖；言是我所，是則動搖。未來當有，是則動搖；未來當無，是則動搖。」**
　　「比丘！如果心中認為『這是我』，心就會產生動搖（不穩定）；認為『這是我的所有物』，心也會動搖。認為『未來將會存在』，心會動搖；認為『未來將不存在』，心也會動搖。」

**◈ 「當復有色，是則動搖；當復無色，是則動搖。當復有想，是則動搖；當復無想，是則動搖；當復非有想非無想，是則動搖。」**
　　「認為未來將會有色（物質身體），心會動搖；認為將沒有色（物質身體），心會動搖。認為將會有想（想像與認知），心會動搖；認為將沒有想（想像與認知），心會動搖；認為將處於非有想非無想的狀態，心也會動搖。」

**◈ 「動搖故病，動搖故癰，動搖故刺，動搖故著。正觀察動搖故苦者，得不動搖心，多修習住，繫念正知。」**
　　「因為心的動搖，所以產生病苦，產生腫瘤，產生如刺般的痛苦，產生執著。若能正確觀察到『因為動搖而導致痛苦』，就能獲得不動搖的心，應多加修習並安住於此，將正念與正知緊緊繫住。」

**◈ 「如動搖。如是思量虛誑，有行因愛。言我，是則為愛；言我所，是則為愛。」**
　　「就像這種動搖一樣。這種虛妄的思量，是由於愛（貪愛）而產生的行（心的造作）。說『我』，是因為愛（貪愛）；說『我的』，也是因為愛（貪愛）。」

**◈ 「言當來有，是則為愛；言當來無，是則為愛。當有色，是則為愛；當無色，是則為愛。當有想，是則為愛；當無想，是則為愛；當非想非非想，是則為愛。」**
　　「認為未來將存在，是因為愛（貪愛）；認為未來將不存在，也是因為愛（貪愛）。認為將有色（物質身體），是因為愛（貪愛）；認為將無色（物質身體），是因為愛（貪愛）。認為將有想（想像與認知），是因為愛（貪愛）；認為將無想（想像與認知），是因為愛（貪愛）；認為將處於非想非非想，也是因為愛（貪愛）。」

**◈ 「愛故為病，愛故為癰，愛故為刺。若善思觀察愛生苦者，當多住離愛心，正念正智。」**
　　「因為愛（貪愛）而導致病苦，因為愛（貪愛）而產生腫瘤，因為愛（貪愛）而產生如刺般的痛苦。如果能善於思考並觀察到『愛（貪愛）會產生痛苦』，就應多安住在遠離愛（貪愛）的心境中，保持正念與正智。」

**◈ 「諸比丘！過去世時，阿修羅興軍與帝釋鬪。時，天帝釋告三十三天：『今日諸天、阿修羅苦戰，若諸天勝，阿修羅不如者，當生執阿修羅，縛以五繫，送還天宮。』」**
　　「諸位比丘！在過去世時，阿修羅興起軍隊與帝釋天作戰。那時，帝釋天告訴三十三天的眾神說：『今天諸天與阿修羅激烈交戰，如果諸天獲勝，而阿修羅戰敗，我們就將俘虜阿修羅，用五種束縛將其綑綁，送回天宮。』」

**◈ 「阿修羅語其眾言：『今阿修羅軍與諸天戰，若阿修羅勝，諸天不如者，當生執帝釋，縛以五繫，還歸阿修羅宮。』」**
　　「阿修羅也對其軍隊說：『現在阿修羅軍與諸天作戰，如果阿修羅獲勝，而諸天戰敗，我們就將俘虜帝釋天，用五種束縛將其綑綁，帶回阿修羅宮。』」

**◈ 「當其戰諍，諸天得勝，阿修羅不如。時，三十三天生執毘摩質多羅阿修羅王，縛以五繫，還歸天宮。」**
　　「在交戰過程中，諸天獲勝，阿修羅戰敗。於是，三十三天的眾神俘虜了毘摩質多羅阿修羅王，用五種束縛將他綑綁，帶回天宮。」

**◈ 「爾時，毘摩質多羅阿修羅王身被五繫，置於正法殿上，以種種天五欲樂而娛樂之。」**
　　「那時，毘摩質多羅阿修羅王身體被五種束縛綑綁，被安置在正法殿上，諸天用種種天界的五欲樂（感官享受）來娛樂他。」

**◈ 「毘摩質多羅阿修羅王作是念：『唯阿修羅賢善聰慧，諸天雖善，我今且當還歸阿修羅宮。』作是念時，即自見身被五繫縛，諸天五欲自然化沒。」**
　　「毘摩質多羅阿修羅王心中這樣想：『只有阿修羅才是真正賢善聰慧的，諸天雖然也善，但我現在應該回到阿修羅宮。』當他這樣想時，立刻發現自己的身體被五種束縛緊緊綑綁，而原本的五欲樂也自然消失了。」

**◈ 「毘摩質多羅阿修羅王復作是念：『諸天賢善，智慧明徹，阿修羅雖善，我今且當住此天宮。』作是念時，即自見身五縛得解，諸天五欲自然還出。」**
　　「毘摩質多羅阿修羅王又這樣想：『諸天才是賢善的，智慧明徹，阿修羅雖然也善，但我現在應該留在這座天宮。』當他這樣想時，立刻發現身體的五種束縛被解開了，而五欲樂也自然重新出現。」

**◈ 「毘摩質多羅阿修羅王乃至有如是微細之縛，魔波旬縛轉細。於是心動搖時，魔即隨縛；心不動搖，魔即隨解。」**
　　「毘摩質多羅阿修羅王甚至有如此微細的束縛，而魔波旬的束縛則更加微細。因此，當心產生動搖（不穩定）時，魔就隨之將人束縛；當心不動搖時，束縛就隨之解開。」

**◈ 「是故，諸比丘！多住不動搖心，正念正智，應當學！」**
　　「所以，諸位比丘！應當多安住在不動搖的心境中，保持正念與正智，這才是應該學習的！」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解]
1. **六觸入處 (Saḷāyatana-phassa)**：指眼、耳、鼻、舌、身、意這六種感官與外界對象接觸的過程。在經文中，這被比喻為敲打穀物的棍子，意指凡夫在感官接觸中不斷產生反應，導致心靈受損與不安。
2. **動搖 (Vikampa)**：指心靈的不穩定、猶豫或對「我」的執著而產生的劇烈波動。這種不穩定源於對無常現象的錯誤認知（如認為有永恆的自我）。
3. **愛 (Taṇhā)**：即貪愛。這是十二因緣中的關鍵環節，是導致輪迴與痛苦的根本原因。經文指出，所有關於「我」與「未來」的妄想，本質上都是貪愛的表現。
4. **五繫 (Pañca-bandhana)**：象徵五種束縛或執著。在故事中，它具象化為繩索，但在法義上代表對五欲（色、聲、香、味、觸）的執著以及對自我身分的認同。
5. **魔波旬 (Māra)**：象徵內心的煩惱、誘惑與阻礙修行的力量。魔的束縛最為微細，指那些潛意識中極其細微的傲慢或貪欲。

### [現代實作指南]
本經文揭示了「認同感」如何成為我們心理束縛的根源。我們可以將其轉化為以下生活實踐：

1. **識別「我」的動搖**：
   - 當你在生活中感到焦慮、憤怒或不滿足時，觀察心中是否出現了「這是我（My identity）」或「這是我的（My possession）」的念頭。
   - 練習將「我感到很受傷」轉化為「現在有一個『受傷的感受』在發生」。透過將「我」抽離，減少心的動搖。

2. **覺察「第七次重擊」**：
   - 現代人常在感官刺激（六觸）後，陷入對未來的過度擔憂或幻想（第七人重打）。
   - 當你對一件事情反覆糾結時，提醒自己：「我正在進行第七次重擊」，意識到這種重複的思慮並非在解決問題，而是在增加痛苦。

3. **練習「心態轉換」的解縛**：
   - 效法阿修羅王的轉念：當他執著於「阿修羅最優秀」時，束縛出現；當他承認「他人（諸天）之長處」並放下傲慢時，束縛解開。
   - 在衝突中，嘗試從「我是對的」轉向「對方可能有其道理」，觀察這種心態轉變是否能立刻減輕你內心的壓抑感與束縛感。

4. **建立「不動搖心」的日常習慣**：
   - 每天抽出5分鐘，僅僅觀察呼吸。當雜念（如對未來的計畫、對過去的遺憾）出現時，不評判，僅僅標記它為「動搖」，然後溫柔地將注意力帶回呼吸。

### [段落大意]
本經以穀物被反覆敲打為喻，說明愚癡者受六根觸及對未來世的渴望而深受痛苦。佛陀指出，所有關於「我」與「我所」的執著，以及對未來存在狀態的妄想，皆源於「愛（貪愛）」，導致心靈動搖並產生病苦。透過阿修羅王在天宮中因傲慢而受縛、因謙卑而解縛的故事，強調心靈的穩定與正念正智是解脫微細束縛（魔波旬）的唯一途徑。
