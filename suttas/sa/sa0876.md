# 雜阿含經 第 876 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有四正斷。何等為四？一者斷斷，二者律儀斷，三者隨護斷，四者修斷。」

爾時世尊即說偈言：

「斷斷及律儀，　　隨護與修習， 　如此四正斷，　　諸佛之所說。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

---

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有四正斷。何等為四？**
　　那時，世尊告訴在場的眾位比丘說：「有四種正確斷除煩惱（內心的染污與執著）的方法。這四種是什麼呢？

**◈ 一者斷斷，二者律儀斷，三者隨護斷，四者修斷。」**
　　第一種是『斷斷』，即直接斷除已經產生的煩惱；第二種是『律儀斷』，即透過遵守戒律（律儀）來防止煩惱的生起；第三種是『隨護斷』，即透過正念隨時守護心念以斷除煩惱；第四種是『修斷』，即透過持續的修行（Bhāvanā）與觀照來徹底斷除。

**◈ 爾時世尊即說偈言：**
　　那時世尊隨即說了偈頌（詩歌形式的教法）：

**◈ 「斷斷及律儀，隨護與修習，如此四正斷，諸佛之所說。」**
　　直接斷除、律儀守戒、隨時守護以及持續修習，這四種正確的斷除方法，是所有佛陀所共同開示的。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **正斷 (Samyak-chheda)**：指正確地切斷煩惱與執著的過程，旨在消除導致輪迴（Saṃsāra）的根本原因。
2. **律儀 (Śīla/Discipline)**：指對戒律的遵守與自律。在斷除煩惱的過程中，律儀扮演「預防」的角色，避免因不當行為而產生新的煩惱。
3. **隨護 (Sampajañña/Mindfulness Guarding)**：指在感官接觸（觸）到對象時，能即時覺察並守護心念，不讓貪愛（Taṇhā）或嗔恨隨之生起。
4. **修習 (Bhāvanā/Cultivation)**：指透過系統性的修行，如定（Samādhi）與慧（Paññā）的訓練，從根源上轉化意識，達成最終的解脫（Vimokkha）。

### [現代實作指南]
本經提出的「四正斷」可轉化為一套完整的「情緒與行為管理系統」，適用於現代生活中的壓力與衝突處理：

1. **即時止損（對應：斷斷）** ➔ 當強烈的憤怒或焦慮爆發時，立即採取「停止」動作。例如：深呼吸三次，或暫時離開衝突現場，強行中斷負面情緒的遞增循環。
2. **建立邊界（對應：律儀斷）** ➔ 分析哪些環境或習慣容易誘發煩惱。例如：若發現社交媒體導致嫉妒心，則設定使用時間限制（建立律儀），從源頭減少煩惱產生的機會。
3. **覺察監控（對應：隨護斷）** ➔ 在日常生活中練習「正念」。當感覺到心中微小的不快（受）升起時，立刻標記它（例如：心裡默念「生起不快了」），在煩惱尚未擴大為行動前將其攔截。
4. **深度轉念（對應：修斷）** ➔ 定期反思煩惱的本質。思考該情緒是否基於對「我」的執著（無我 Anattā），體悟所有情緒皆是因緣而生且無常（Anicca），從而從認知層面徹底瓦解煩惱的根基。

### [段落大意]
本經由世尊開示四種正確斷除煩惱的方法：直接斷除、律儀預防、隨時守護與持續修習。這四者構成了從「事後處理」到「事前預防」，再到「即時監控」與「根源轉化」的完整修行路徑，是諸佛共同的教導。
