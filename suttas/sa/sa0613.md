# 雜阿含經 第 613 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有不善聚、善聚。何等為不善聚？謂三不善根，是名正說。所以者何？純不善積聚者，謂三不善根。云何為三？謂貪不善根、恚不善根、癡不善根。云何為善聚？謂四念處。所以者何？純善滿具者，謂四念處，是名善說。云何為四？謂身念處，受、心、法念處。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如三不善根，如是三惡行——身惡行、口惡行、意惡行，三想——欲想、恚想、害想，三覺——欲覺、恚覺、害覺，三界——欲界、恚界、害界。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有不善聚、善聚。**
　　那時，世尊告訴在場的眾位比丘說：「存在著『不善的聚集』與『善的聚集』。

**◈ 何等為不善聚？謂三不善根，是名正說。**
　　什麼被稱為不善的聚集？指的是三種不善的根源（三不善根），這樣說才是正確的。

**◈ 所以者何？純不善積聚者，謂三不善根。**
　　為什麼呢？因為純粹由不善而積聚起來的，就是指這三種不善的根源。

**◈ 云何為三？謂貪不善根、恚不善根、癡不善根。**
　　這三種是什麼？分別是貪（貪欲）的不善根、恚（瞋恚）的不善根、以及癡（愚癡）的不善根。

**◈ 云何為善聚？謂四念處。**
　　什麼被稱為善的聚集？指的是四念處（四種正念的確立）。

**◈ 所以者何？純善滿具者，謂四念處，是名善說。**
　　為什麼呢？因為純粹由善而圓滿具足的，就是指四念處，這樣說才是正確的。

**◈ 云何為四？謂身念處，受、心、法念處。」**
　　這四種是什麼？分別是身念處（對物質身體的覺知）、受念處（對感受的覺知）、心念處（對意識狀態的覺知）、以及法念處（對心理現象與真理的覺知）。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如三不善根，如是三惡行——身惡行、口惡行、意惡行，**
　　正如三種不善的根源一樣，也存在著三種惡行，即：身體的惡行、言語的惡行、以及意念的惡行。

**◈ 三想——欲想、恚想、害想，**
　　三種想（想像與認知），即：貪欲的想、瞋恚的想、以及傷害的想。

**◈ 三覺——欲覺、恚覺、害覺，**
　　三種覺（意識的覺知），即：貪欲的覺、瞋恚的覺、以及傷害的覺。

**◈ 三界——欲界、恚界、害界。**
　　三種界（心靈所處的境界），即：貪欲的境界、瞋恚的境界、以及傷害的境界。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **三不善根 (Akusala-mūla)**：指導致痛苦與輪迴的三種根本心理染污，即貪（對對象的強烈渴求）、恚（對不滿對象的排斥與憤怒）、癡（對四聖諦真理的無知與迷失）。
2. **四念處 (Satipaṭṭhāna)**：一種系統性的修行方法，透過對色（物質身體）、受（感受）、心（意識狀態）、法（心理現象與真理）的持續覺知，以破除對「我」的執著。
3. **三想 (Saññā)**：此處指基於三不善根而產生的認知模式，將對象標記為「可欲的」、「可恨的」或「可毀滅的」。
4. **三覺 (Viññāṇa)**：此處指在貪、恚、癡的驅動下，意識對對象所產生的覺知狀態。

### [現代實作指南]
本經文提供了一套簡單的「心理分類法」，可用於日常的情緒管理與自我覺察：

1. **識別「不善聚」的啟動**：
   當你感到不安、憤怒或強烈渴望時，立即在心中標記：「現在不善聚（三不善根）正在積聚」。
   ➔ 貪 ➔ 標記為「渴望」
   ➔ 恚 ➔ 標記為「排斥」
   ➔ 癡 ➔ 標記為「混亂/不清晰」

2. **切換至「善聚」的實作（四念處應用）**：
   一旦發現不善聚，不要試圖壓抑它，而是將注意力轉移到四念處的觀察上：
   ➔ **身念處**：感受憤怒時，身體哪個部位緊繃？（如：胸口悶、拳頭握緊）。
   ➔ **受念處**：這是一種「苦」的感受，還是「不快」的感受？
   ➔ **心念處**：觀察心現在是「躁動」的還是「沉重」的？
   ➔ **法念處**：意識到「這份情緒是因緣而生，並非永恆的自我」，體會其無常。

3. **阻斷惡行鏈條**：
   意識到「欲、恚、害」的想（認知）出現時，在它轉化為身、口、意之「惡行」前，透過上述的覺知將其截斷，防止不善根進一步擴大為不善的境界。

### [段落大意]
本經對比了「不善聚」與「善聚」。不善聚由貪、恚、癡三不善根構成，並進一步延伸至惡行、錯誤認知（想）、意識狀態（覺）與心靈境界（界）；而善聚則由四念處（身、受、心、法）構成。修行者應透過四念處的覺知，識別並轉化三不善根的積聚。
