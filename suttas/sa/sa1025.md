# 雜阿含經 第 1025 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。時，有異比丘年少新學，於此法、律出家未久，少知識，獨一客旅，無人供給，住邊聚落客僧房中，疾病困篤。

時，有眾多比丘詣佛所，稽首禮足，却坐一面，白佛言：「世尊！有一比丘年少新學，乃至疾病困篤，住邊聚落客僧房中。有是病比丘多死無活。善哉！世尊！往彼住處，以哀愍故！」

爾時，世尊默然而許。即日晡時從禪覺，至彼住處。

彼病比丘遙見世尊，扶床欲起。

佛告比丘：「息臥勿起！云何？比丘！苦患寧可忍不？……」如前差摩迦修多羅廣說如是三受，乃至「病苦但增不損。」

佛告病比丘：「我今問汝，隨意答我。汝得無變悔耶？」

病比丘白佛：「實有變悔。世尊！」

佛告病比丘：「汝得無犯戒耶？」

病比丘白佛言：「世尊！實不犯戒。」

佛告病比丘：「汝若不犯戒，何為變悔？」

病比丘白佛：「世尊！我年幼稚出家未久，於過人法勝妙知見未有所得，我作是念：『命終之時，知生何處？』故生變悔。」

佛告比丘：「我今問汝，隨意答我。云何？比丘！有眼故有眼識耶？」

比丘白佛：「如是，世尊！」

復問：「比丘！於意云何？有眼識故有眼觸、眼觸因緣生內受，若苦、若樂、不苦不樂耶？」

比丘白佛：「如是，世尊！」耳、鼻、舌、身、意亦如是說。

「云何？比丘！若無眼則無眼識耶？」

比丘白佛：「如是，世尊！」

復問：「比丘！若無眼識則無眼觸耶？若無眼觸，則無眼觸因緣生內受，若苦、若樂、不苦不樂耶？」

比丘白佛：「如是，世尊！」耳、鼻、舌、身、意亦如是說。

「是故，比丘！當善思惟如是法，得善命終，後世亦善。」爾時，世尊為病比丘種種說法，示、教、照、喜已，從坐起去。

時，病比丘，世尊去後，尋即命終。臨命終時，諸根喜悅，顏貌清淨，膚色鮮白。

時，眾多比丘詣佛所，稽首禮足，退坐一面，白佛言：「世尊！彼年少比丘疾病困篤，尊者今已命終。當命終時，諸根喜悅，顏貌清淨，膚色鮮白。云何？世尊！如是比丘當生何處？云何受生？後世云何？」

佛告諸比丘：「彼命過比丘是真寶物，聞我說法，分明解了，於法無畏，得般涅槃，汝等但當供養舍利。」

世尊爾時為彼比丘受第一記。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有異比丘年少新學，於此法、律出家未久，少知識，獨一客旅，無人供給，住邊聚落客僧房中，疾病困篤。**
　　那時，有一位年輕且剛開始學習的比丘，他出家學習這套法（真理）與律（戒律）的時間還不長，在僧團中沒有多少熟識的人，像個孤單的旅人，沒有人照顧供養，住在邊陲聚落的客僧房中，正遭受極其嚴重的疾病折磨。

**◈ 時，有眾多比丘詣佛所，稽首禮足，却坐一面，白佛言：「世尊！有一比丘年少新學，乃至疾病困篤，住邊聚落客僧房中。有是病比丘多死無活。善哉！世尊！往彼住處，以哀愍故！」**
　　這時，許多比丘來到佛陀面前，頂禮觸足後退到一邊坐下，對佛陀恭敬地說道：「世尊！有一位年輕新學的比丘，正處於疾病困苦之中，住在邊陲聚落的客僧房裡。像這樣病重的比丘，大多難以生存。世尊！請您出於慈悲心，前往他居住的地方探視他吧！」

**◈ 爾時，世尊默然而許。即日晡時從禪覺，至彼住處。**
　　那時，世尊默默地同意了。在當天傍晚，佛陀從禪定中覺醒，隨即前往該比丘居住的地方。

**◈ 彼病比丘遙見世尊，扶床欲起。**
　　那位病重的比丘從遠處看見世尊，試著扶著床想要起身。

**◈ 佛告比丘：「息臥勿起！云何？比丘！苦患寧可忍不？……」如前差摩迦修多羅廣說如是三受，乃至「病苦但增不損。」**
　　佛陀告訴比丘：「休息臥著就好，不必起身！比丘，你現在的病苦是否還能忍受？……」接著佛陀像之前對待摩迦修多羅那樣，詳細地解說關於三種受（感受）的性質，直到說明病苦雖然在增加，但並不會損害到真正的本質。

**◈ 佛告病比丘：「我今問汝，隨意答我。汝得無變悔耶？」**
　　佛陀問病比丘：「我現在問你，請隨意回答我。你心中是否產生了不安與悔恨？」

**◈ 病比丘白佛：「實有變悔。世尊！」**
　　病比丘對佛陀說：「世尊，我確實感到不安與悔恨！」

**◈ 佛告病比丘：「汝得無犯戒耶？」**
　　佛陀問病比丘：「你是否犯過戒律？」

**◈ 病比丘白佛言：「世尊！實不犯戒。」**
　　病比丘對佛陀說：「世尊！我確實沒有犯戒。」

**◈ 佛告病比丘：「汝若不犯戒，何為變悔？」**
　　佛陀問病比丘：「如果你沒有犯戒，那為什麼會感到不安與悔恨？」

**◈ 病比丘白佛：「世尊！我年幼稚出家未久，於過人法勝妙知見未有所得，我作是念：『命終之時，知生何處？』故生變悔。」**
　　病比丘對佛陀說：「世尊！我年紀輕且出家時間短，對於超越凡夫之法的殊勝妙見（正確的認知）還沒有獲得。我心中在想：『當生命結束之時，我不知道會投生到什麼地方？』因此才產生了不安與悔恨。」

**◈ 佛告比丘：「我今問汝，隨意答我。云何？比丘！有眼故有眼識耶？」**
　　佛陀告訴比丘：「我現在問你，請隨意回答我。比丘，是否因為有眼（物質身體的視覺器官），所以才會產生眼識（眼睛所產生的意識）？」

**◈ 比丘白佛：「如是，世尊！」**
　　比丘回答：「確實如此，世尊！」

**◈ 復問：「比丘！於意云何？有眼識故有眼觸、眼觸因緣生內受，若苦、若樂、不苦不樂耶？」**
　　佛陀又問：「比丘，你的看法如何？是否因為有眼識（意識），所以才會產生眼觸（眼睛與對象的接觸），而眼觸又是因緣而生（緣起），進而產生內在的受（感受），無論是苦的、樂的，或是既不苦也不樂的？」

**◈ 比丘白佛：「如是，世尊！」耳、鼻、舌、身、意亦如是說。**
　　比丘回答：「確實如此，世尊！」關於耳、鼻、舌、身、意（其餘五種感官）的運作，佛陀也以同樣的方式說明。

**◈ 「云何？比丘！若無眼則無眼識耶？」**
　　佛陀問：「比丘，如果沒有眼（物質器官），是不是就沒有眼識（意識）？」

**◈ 比丘白佛：「如是，世尊！」**
　　比丘回答：「確實如此，世尊！」

**◈ 復問：「比丘！若無眼識則無眼觸耶？若無眼觸，則無眼觸因緣生內受，若苦、若樂、不苦不樂耶？」**
　　佛陀又問：「比丘，如果沒有眼識（意識），是不是就沒有眼觸（接觸）？如果沒有眼觸，是不是就不會因緣而生（緣起）出內在的受（感受），無論是苦、樂或不苦不樂？」

**◈ 比丘白佛：「如是，世尊！」耳、鼻、舌、身、意亦如是說。**
　　比丘回答：「確實如此，世尊！」關於耳、鼻、舌、身、意（其餘五種感官）的運作，佛陀也以同樣的方式說明。

**◈ 「是故，比丘！當善思惟如是法，得善命終，後世亦善。」**
　　佛陀說：「因此，比丘！你應該好好思惟這種緣起法（因緣而生的道理），如此就能獲得安詳的死亡，且後世也會是好的。」

**◈ 爾時，世尊為病比丘種種說法，示、教、照、喜已，從坐起去。**
　　那時，世尊為病比丘說了各種法，在給予指引、教導、啟發與鼓勵之後，從座位上起身離開。

**◈ 時，病比丘，世尊去後，尋即命終。臨命終時，諸根喜悅，顏貌清淨，膚色鮮白。**
　　這時，病比丘在世尊離開後不久就命終了。在臨終之時，他的感官狀態喜悅，面容清淨，皮膚色澤鮮白。

**◈ 時，眾多比丘詣佛所，稽首禮足，退坐一面，白佛言：「世尊！彼年少比丘疾病困篤，尊者今已命終。當命終時，諸根喜悅，顏貌清淨，膚色鮮白。云何？世尊！如是比丘當生何處？云何受生？後世云何？」**
　　這時，許多比丘來到佛陀面前，頂禮觸足後退到一邊坐下，對佛陀說：「世尊！那位病重的年輕比丘已經命終了。他在臨終時，感官喜悅，面容清淨，膚色鮮白。世尊，這樣的比丘將會投生到何處？如何受生？後世的情況會如何？」

**◈ 佛告諸比丘：「彼命過比丘是真寶物，聞我說法，分明解了，於法無畏，得般涅槃，汝等但當供養舍利。」**
　　佛陀告訴諸比丘：「那位去世的比丘是真正的寶物。他聽聞我說法後，分明地理解了真理，對法不再有恐懼，已經達到了涅槃（超越生死、絕對寧靜的解脫境界），你們只需要供養他的舍利即可。」

**◈ 世尊爾時為彼比丘受第一記。**
　　世尊在那時為這位比丘給予了「第一記」（預言他未來將會成佛）。

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **眼識 (Cakkhu-viññāṇa)**：識（意識）的一種，特指眼睛與視覺對象接觸時所產生的覺知功能。
2. **觸 (Phassa)**：指色（物質身體/感官）、識（意識）與法（對象）三者會合的過程。它是產生「受（感受）」的直接原因。
3. **受 (Vedanā)**：對觸覺產生的心理反應，分為苦受、樂受、不苦不樂受。
4. **緣起 (Paṭicca-samuppāda)**：指一切現象皆是由特定的條件（因緣）組合而生，沒有獨立永恆的實體。本經中展示了「感官 ➔ 意識 ➔ 接觸 ➔ 感受」的因果鏈條。
5. **第一記 (Vyākaraṇa)**：佛陀對某位修行者所做的正式預言，確認該人未來必然會成就佛果。
6. **涅槃 (Nibbāna)**：熄滅貪、嗔、痴，徹底解脫輪迴之苦的絕對寧靜境界。

### [現代實作指南]
本經提供了一套強大的「解構痛苦」邏輯，可用於處理現代人的生理疼痛、焦慮或對死亡的恐懼：

1. **觀察「感受」的組成**：
   當我們感到痛苦（如病痛或心理壓力）時，習慣上會認為「我在痛苦」。請嘗試將其拆解為：
   - **物質基礎**：身體的某個部位（如：眼睛、皮膚）。
   - **覺知過程**：意識到該處有刺激（如：眼識、身識）。
   - **接觸發生**：刺激與意識的會合（觸）。
   - **感受產生**：最終產生的「苦」或「不適」（受）。

2. **運用「無條件則無結果」的邏輯轉念**：
   - 意識到：痛苦並非一個獨立的實體，而是由一系列條件構成的。
   - 轉念練習：告訴自己「如果沒有這個物質器官的接觸，就不會有這個感受」。這並非否認痛苦，而是將痛苦從「我的本質」轉化為「暫時的條件組合」。

3. **消除對未知的恐懼**：
   - 比丘的恐懼來自於「不知道死後去哪」。佛陀透過讓他理解「感受的消亡」，將其注意力從「未來的投生」拉回到「當下的運作機制」。
   - 實作：當對未來感到不安時，練習觀察當下不安感是如何產生的（觸 ➔ 受），意識到不安感本身也是緣起而生，而非永恆不變的。

### [段落大意]
本經記述佛陀探視一名病重且對死後投生感到恐懼的年輕比丘。佛陀透過邏輯分析，指導比丘理解感官、意識、接觸與感受之間的緣起關係：若無感官則無意識，無意識則無接觸，無接觸則無感受。比丘由此體悟到痛苦的虛幻與消亡，在心靈平靜中安詳離世，最終被佛陀讚嘆為得涅槃之寶物，並獲授成佛預言。
