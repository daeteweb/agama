# 雜阿含經 第 601 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。

時，彼天子說偈問佛：

「薩羅小流注，　　當於何反流， 　生死之徑路，　　於何而不轉， 　世間諸苦樂，　　何由滅無餘？」

爾時，世尊說偈答言：

「眼耳鼻舌身，　　及彼意入處， 　名色滅無餘，　　薩羅小還流， 　生死道不轉，　　苦樂滅無餘。」

時，彼天子復說偈言：

「久見婆羅門，　　逮得般涅槃， 　一切怖已過，　　永超世恩愛。」

時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 時，有天子容色絕妙，於後夜時來詣佛所，稽首佛足，身諸光明遍照祇樹給孤獨園。**
　　那時，有一位容貌極其出色的天子，在深夜時分來到佛陀這裡，頂禮佛陀的足部，他的身體發出強烈的光明，照亮了整個祇樹給孤獨園。

**◈ 時，彼天子說偈問佛：**
　　那時，這位天子用偈頌（詩歌形式）請問佛陀：

**◈ 「薩羅小流注，當於何反流，生死之徑路，於何而不轉，世間諸苦樂，何由滅無餘？」**
　　「就像薩羅小溪的流水一樣，（生命的流轉）應當在何處才能反轉而回？輪迴 (Saṃsāra) 的生死道路，在何處才能停止不再轉動？世間所有的苦與樂，要如何才能徹底滅盡而不再殘留？」

**◈ 爾時，世尊說偈答言：**
　　那時，世尊用偈頌回答說：

**◈ 「眼耳鼻舌身，及彼意入處，名色滅無餘，薩羅小還流，生死道不轉，苦樂滅無餘。」**
　　「當眼、耳、鼻、舌、身（物質感官），以及意（意識感官）這六個入處（感官接觸之門）不再運作，且名色（名：心理現象；色：物質身體）徹底滅盡而不再殘留時，（生命的）薩羅小溪便會反轉而回，輪迴 (Saṃsāra) 的生死道路將不再轉動，所有的苦與樂也會隨之徹底滅盡。」

**◈ 時，彼天子復說偈言：**
　　那時，這位天子再次說偈頌道：

**◈ 「久見婆羅門，逮得般涅槃，一切怖已過，永超世恩愛。」**
　　「我久仰這位婆羅門（指佛陀），終於見到了達到超越生死、絕對寧靜的解脫境界 (涅槃) 的人。他已經超越了一切恐懼，永遠超脫了世間的貪愛與執著。」

**◈ 時，彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。**
　　那時，這位天子聽聞佛陀所說的教法，心中感到法喜並隨喜讚嘆，頂禮佛陀的足部，隨即消失不再出現。

### [詞語註解（解釋重要佛教概念）]
1. **入處 (Ayatana)**：指感官與外界對象接觸的門戶。分為內入處（眼、耳、鼻、舌、身、意）與外入處（色、聲、香、味、觸、法）。
2. **名色 (Nāma-rūpa)**：十二因緣中的關鍵環節。「名」指心理組成（如受、想、行、識），「色」指物質身體。兩者共同構成個體的生命存在。
3. **輪迴 (Saṃsāra)**：指眾生在生死之間不斷循環流轉的過程，其動力源於無明與貪愛。
4. **涅槃 (Nibbāna)**：指熄滅一切貪、嗔、癡的煩惱，從而終結輪迴、達到絕對寧靜與自由的解脫狀態。

### [現代實作指南]
本經文揭示了苦樂的根源在於「感官的運作」與「名色的執著」。在現代生活中，我們可以透過以下步驟實踐：

1. **覺察感官門戶 (入處實作)**：
   ➔ 當強烈的情緒（如憤怒或焦慮）升起時，立即將注意力移回「入處」。
   ➔ 詢問自己：現在是透過哪個門戶在運作？（例如：是因為「眼」看到了不順心的畫面，或是「耳」聽到了刺耳的話語？）
   ➔ 意識到「接觸 ➔ 感受 ➔ 反應」的過程，在「接觸」的瞬間即止，不讓其演變成後續的心理造作。

2. **解構名色 (去中心化實作)**：
   ➔ 當我們感到「我很痛苦」時，嘗試將其拆解為「名」與「色」。
   ➔ 「色」：觀察身體的生理反應（如心跳加快、胸口悶）。
   ➔ 「名」：觀察心理的標籤（如「這是不公平的」、「他怎麼能這樣」）。
   ➔ 意識到這些只是暫時出現的現象，而非一個永恆的「我」。

3. **反轉流向 (轉念技巧)**：
   ➔ 將原本「追求快感 ➔ 產生執著 ➔ 失去後痛苦」的流向，轉化為「觀察快感 ➔ 意識到其無常 ➔ 放下執著」。
   ➔ 透過這種觀察，將生命從「追求滿足」的輪迴流向，轉向「覺知清明」的解脫流向。

### [段落大意]
一位天子請問佛陀如何能終結輪迴的流轉並滅盡世間苦樂。佛陀回答指出，當六入處（感官門戶）停止運作且名色（身心組成）徹底滅盡時，即可停止輪迴並滅除苦樂。天子聞後讚嘆佛陀已證得涅槃，超脫世間貪愛，隨即歡喜消失。
