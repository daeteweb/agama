# 雜阿含經 第 878 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有四正斷。何等為四？一者斷斷，二者律儀斷，三者隨護斷，四者修斷。云何為斷斷？謂比丘已起惡不善法斷，生欲、方便、精勤、心攝受，是為斷斷。云何律儀斷？未起惡不善法不起，生欲、方便、精勤、攝受，是名律儀斷。云何隨護斷？未起善法令起，生欲、方便、精勤、攝受，是名隨護斷。云何修斷？已起善法增益修習，生欲、方便、精勤、攝受，是名修斷。」

爾時，世尊即說偈言：

「斷斷及律儀，　　隨護與修習， 　如此四正斷，　　諸佛之所說。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有四正斷。何等為四？**
　　那時，世尊告訴在場的眾位比丘說：「有四種正確的斷除與修行方法（四正斷）。這四種方法是什麼呢？

**◈ 一者斷斷，二者律儀斷，三者隨護斷，四者修斷。**
　　第一種是『斷除已生的惡法』（斷斷），第二種是『防止未生的惡法』（律儀斷），第三種是『培育未生的善法』（隨護斷），第四種是『增益已生的善法』（修斷）。

**◈ 云何為斷斷？謂比丘已起惡不善法斷，生欲、方便、精勤、心攝受，是為斷斷。**
　　什麼是『斷除已生的惡法』？是指比丘在惡不善法（心中已產生的負面狀態）已經興起時，將其斷除。為了達成此目標，需要生起願望（生欲）、運用適當的方法（方便）、付出努力（精勤），以及讓心專注攝受，這就稱為『斷除已生的惡法』。

**◈ 云何律儀斷？未起惡不善法不起，生欲、方便、精勤、攝受，是名律儀斷。**
　　什麼是『防止未生的惡法』？是指在惡不善法尚未興起時，使其不興起。這同樣需要生起願望、運用方法、付出努力以及心之攝受，這就稱為『防止未生的惡法』。

**◈ 云何隨護斷？未起善法令起，生欲、方便、精勤、攝受，是名隨護斷。**
　　什麼是『培育未生的善法』？是指在善法尚未興起時，使其興起。這需要生起願望、運用方法、付出努力以及心之攝受，這就稱為『培育未生的善法』。

**◈ 云何修斷？已起善法增益修習，生欲、方便、精勤、攝受，是名修斷。」**
　　什麼是『增益已生的善法』？是指在善法已經興起時，使其進一步增益與修習。這需要生起願望、運用方法、付出努力以及心之攝受，這就稱為『增益已生的善法』。」

**◈ 爾時，世尊即說偈言：**
　　那時，世尊隨即用偈頌總結道：

**◈ 「斷斷及律儀，隨護與修習，如此四正斷，諸佛之所說。」**
　　「斷除已生的惡法、防止未生的惡法，培育未生的善法、增益已生的善法。這四種正確的斷除與修行方法，是諸位佛陀所教導的。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **四正斷 (Sammāppahāna)**：指四種正確的努力方向，旨在清除心靈的染污並建立正向的心理狀態。在其他經典中常被稱為「四正勤」。
2. **惡不善法 (Akusala Dhamma)**：指會導致痛苦、增加煩惱或造成傷害的心理狀態，如貪、嗔、痴。
3. **生欲 (Chanda)**：在此非指貪欲，而是指「志向」或「成就某事的願望」，是啟動修行的心理動力。
4. **方便 (Upāya)**：指適當的手段、技巧或方法，使修行能有效地達成目標。
5. **精勤 (Viriya)**：指不懈怠的努力與恆心。
6. **攝受 (Sampajāna/Samādhi context)**：指心的專注、覺知與掌控，使心不散亂地維持在目標上。

### [現代實作指南]
本經提出的「四正斷」實際上是一套完整的「心理品質管理系統」，可將其轉化為以下生活實踐步驟：

1. **第一階段：止損（斷斷 ➔ 處理已發生的負面情緒）**
   - **實作**：當你發現自己正處於憤怒、嫉妒或憂鬱中時，立即覺察（心攝受）。
   - **轉念**：告訴自己「這個情緒正在傷害我」，生起想要停止它的願望（生欲），運用深呼吸或抽離法（方便），堅定地將其切斷（精勤）。

2. **第二階段：預防（律儀斷 ➔ 建立心理防線）**
   - **實作**：分析哪些環境或對象容易誘發你的負面情緒（例如：特定的社交媒體、某個衝突對象）。
   - **轉念**：在誘因出現前，提前設定心理界限，決定「我不允許這個模式再次發生」，從而防止惡法興起。

3. **第三階段：啟動（隨護斷 ➔ 培養正向習慣）**
   - **實作**：找出你尚未養成但對生活有益的品質（例如：耐心、感恩、專注）。
   - **轉念**：設定小目標（生欲），尋找簡單的練習方式（方便），每天花五分鐘練習（精勤），讓善法在心中萌芽。

4. **第四階段：深化（修斷 ➔ 強化正向特質）**
   - **實作**：當你發現自己表現出善意或專注時，不要視為理所當然。
   - **轉念**：意識到「這種狀態對我是有益的」，並嘗試在更多場景中重複這種正向體驗，將其從「偶然的表現」轉化為「穩定的性格」。

### [段落大意]
本經詳細說明了「四正斷」的修行路徑：針對惡法，需在已生時斷除、未生時防止；針對善法，需在未生時培育、已生時增益。每一步都需結合願望、方法、努力與專注四項要素，旨在透過系統性的心理管理，清除染污並建立純淨的心靈狀態。
