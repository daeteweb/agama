# 雜阿含經 第 1049 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「殺生有三種，謂從貪生故、從恚生故、從癡生。乃至邪見亦三種，從貪生、從恚生、從癡生。離殺生亦有三種，不貪生、不恚生、不癡生。乃至離邪見亦三種，不貪生、不恚生、不癡生。」

佛說是經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「殺生有三種，謂從貪生故、從恚生故、從癡生。**
　　「殺生（剝奪生命的行为）有三種動機：一種是源於貪（對欲望的執著）而產生，一種是源於恚（憤怒與厭惡）而產生，一種是源於癡（對真理的無知與迷妄）而產生。

**◈ 乃至邪見亦三種，從貪生、從恚生、從癡生。**
　　甚至於邪見（錯誤的認知與見解）同樣有三種動機：源於貪（欲望執著）而產生、源於恚（憤怒厭惡）而產生、源於癡（無知迷妄）而產生。

**◈ 離殺生亦有三種，不貪生、不恚生、不癡生。**
　　而遠離殺生的狀態同樣有三種：因為不貪（不再被欲望驅使）而不殺生、因為不恚（不再被憤怒驅使）而不殺生、因為不癡（不再被無知驅使）而不殺生。

**◈ 乃至離邪見亦三種，不貪生、不恚生、不癡生。」**
　　甚至於遠離邪見的狀態同樣有三種：因為不貪（不再被欲望驅使）而遠離邪見、因為不恚（不再被憤怒驅使）而遠離邪見、因為不癡（不再被無知驅使）而遠離邪見。」

**◈ 佛說是經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **貪 (Lobha)**：對感官快感或物質對象的強烈渴望與執著，是導致行為失衡的心理根源之一。
2. **恚 (Dosa)**：對不順心之物產生的厭惡、憤怒或排斥心理，是導致傷害行為的心理根源之一。
3. **癡 (Moha)**：對事物本質（如無常、無我）的無知與迷妄，無法正確辨識真理，是所有煩惱的根本。
4. **邪見 (Micchā-diṭṭhi)**：違背正法、不符合緣起真理的錯誤見解，會導致錯誤的行為與痛苦的業報。

### [現代實作指南]
本經揭示了行為（殺生）與認知（邪見）背後的心理驅動力。在現代生活中，我們可以透過以下步驟實踐「根源覺察法」：

1. **行為前覺察（動機分析）**：
   當你產生想要攻擊他人、傷害生物，或持有偏激、歧視性的看法時，立即暫停並詢問自己：「這個衝動是從哪裡來的？」
   ➔ **貪**：我是為了滿足自己的私慾或獲利而想這麼做嗎？
   ➔ **恚**：我是因為對對方感到憤怒、討厭或想報復而這麼做嗎？
   ➔ **癡**：我是因為不了解事實真相、盲目跟風或陷入偏見而這麼做嗎？

2. **轉念實作（根源對治）**：
   一旦識別出動機，採取對應的心理轉換：
   ➔ **對治貪** ➔ 練習「捨」：意識到欲望的暫時性，將焦點從「獲取」轉向「分享」。
   ➔ **對治恚** ➔ 練習「慈」：意識到對方同樣在受苦，將憤怒轉化為對生命的同情。
   ➔ **對治癡** ➔ 練習「覺」：透過閱讀正法或邏輯分析，打破單一的認知框架，尋找事實的真相。

3. **建立正向迴路**：
   意識到「不殺生」與「正見」並非單純的禁令，而是「不貪、不恚、不癡」的自然結果。當心靈不再被三毒驅使，正確的行為將會自然發生。

### 段落大意 
本經闡述了殺生與邪見這兩種負面行為/認知，皆是由於「貪、恚、癡」三毒所驅使而產生；反之，若能根除這三種心理毒素，自然能遠離殺生並消除邪見。
