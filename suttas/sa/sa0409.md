# 雜阿含經 第 409 經

### [原文]
如是我聞：

一時，佛住王舍城迦蘭陀竹園。爾時，有眾多比丘集於食堂，或有貪覺覺者，或瞋覺覺者，或害覺覺者。

爾時，世尊知諸比丘心之所念，往詣食堂，敷坐具於眾前坐，告諸比丘：「汝等莫起貪覺覺，莫起恚覺覺，莫起害覺覺。所以者何？此諸覺非義饒益，非法饒益，非梵行饒益，非智、非正覺，不向涅槃。汝等當起苦聖諦覺、苦集聖諦覺、苦滅聖諦覺、苦滅道跡聖諦覺。所以者何？此四聖諦覺，義饒益、法饒益、梵行饒益、正智、正覺、向於涅槃。是故，諸比丘！於四聖諦當勤方便，起增上欲，正智正念，精進修學。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住王舍城迦蘭陀竹園。**
　　在某一個時間，佛陀住在王舍城的迦蘭陀竹園。

**◈ 爾時，有眾多比丘集於食堂，或有貪覺覺者，或瞋覺覺者，或害覺覺者。**
　　那時，有許多比丘聚集在食堂中，其中有些人產生了貪婪的覺知，有些人產生了瞋恨的覺知，有些人則產生了想要傷害他人的覺知。

**◈ 爾時，世尊知諸比丘心之所念，往詣食堂，敷坐具於眾前坐，告諸比丘：**
　　那時，世尊知道眾位比丘心中在想什麼，於是前往食堂，在眾人面前鋪好坐具坐下，世尊告訴在場的眾位比丘說：

**◈ 「汝等莫起貪覺覺，莫起恚覺覺，莫起害覺覺。**
　　「你們不要生起貪婪的覺知，不要生起瞋恨的覺知，也不要生起傷害的覺知。

**◈ 所以者何？此諸覺非義饒益，非法饒益，非梵行饒益，非智、非正覺，不向涅槃。**
　　為什麼呢？這些覺知對於世俗的利益沒有幫助，對於法（真理）沒有幫助，對於梵行（清淨的修行生活）沒有幫助，既不是智慧，也不是正覺（正確的覺悟），且無法導向涅槃（超越生死、絕對寧靜的解脫境界）。

**◈ 汝等當起苦聖諦覺、苦集聖諦覺、苦滅聖諦覺、苦滅道跡聖諦覺。**
　　你們應該生起對苦聖諦（關於苦的真理）的覺知、對苦集聖諦（關於苦之因的真理）的覺知、對苦滅聖諦（關於苦之止的真理）的覺知，以及對苦滅道跡聖諦（關於解脫之道的真理）的覺知。

**◈ 所以者何？此四聖諦覺，義饒益、法饒益、梵行饒益、正智、正覺、向於涅槃。**
　　為什麼呢？這四聖諦的覺知，對世俗利益有幫助，對法（真理）有幫助，對梵行（清淨的修行生活）有幫助，是正確的智慧與正確的覺悟，且能導向涅槃（超越生死、絕對寧靜的解脫境界）。

**◈ 是故，諸比丘！於四聖諦當勤方便，起增上欲，正智正念，精進修學。」**
　　因此，比丘們！對於四聖諦應當勤於尋找實踐的方法，生起追求更高境界的願望（增上欲），保持正確的智慧與正確的念覺，精進地修學。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **貪、瞋、害 (Lobha, Dosa, Vyāpāda)**：指三種不善的心理狀態。貪是指對對象的強烈渴求與執著；瞋是指對不滿意對象的厭惡與憤怒；害是指想要摧毀或傷害他人的惡意。
2. **梵行 (Brahmacariya)**：原意為「梵天般的行持」，在原始佛教中指清淨的修行生活，旨在遠離感官欲望與染污，追求精神的純淨。
3. **四聖諦 (Cattāri Ariyasaccāni)**：佛法核心的四項真理。包括：苦（人生本質的苦與不滿足）、集（苦的產生原因，即貪愛與無明）、滅（苦的完全止息，即涅槃）、道（導致苦止息的八正道修行路徑）。
4. **增上欲 (Chanda)**：一種正向的願望或志向，指修行者渴望達到更高精神境界、追求解脫的強烈動力。

### [現代實作指南]
本經文的核心在於「覺知的替換」。當我們在生活中產生負面情緒時，並非要強行壓抑，而是將覺知的對象從「情緒本身」轉移到「真理的分析」上。

**實作步驟：**
1. **覺察負面觸發**：當心中升起「想要佔有（貪）」、「感到憤怒（瞋）」或「想反擊對方（害）」的念頭時，立即在心中標記：「現在出現了貪/瞋/害的覺知」。
2. **評估損益**：快速問自己：「這個念頭能帶來長久的利益嗎？它能讓我的心變得清淨嗎？」根據經文，這類覺知「非義饒益」，即對生活品質與心靈成長毫無幫助。
3. **啟動聖諦分析（轉念技巧）**：
   - **觀苦**：意識到目前的憤怒或貪婪本身就是一種「苦（Dukkha）」，是一種心靈的不安與煎熬。
   - **觀集**：分析這個情緒背後的「集（原因）」，例如：「是因為我執著於某個結果，所以才感到憤怒」。
   - **觀滅**：想像如果放下這個執著，心靈將會恢復到什麼樣的寧靜狀態（滅）。
   - **觀道**：決定採取正確的行動（道），例如深呼吸、練習正念，或用慈悲心看待對方。
4. **建立正向動力**：將對物質或權力的渴望（貪），轉化為對「內心平靜與智慧」的渴望（增上欲），將注意力從外部對象移回內心的修煉。

### [段落大意]
佛陀發現比丘們在食堂中生起了貪、瞋、害等不善的覺知，遂告誡他們這些心態對世俗、真理與修行均無益處，且無法導向涅槃。佛陀指導比丘應將覺知轉向四聖諦（苦、集、滅、道），因為這才是正確的智慧，能帶來真正的利益並導向解脫。佛陀勉勵比丘應以正智正念，精進修學四聖諦。
