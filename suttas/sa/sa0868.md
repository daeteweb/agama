# 雜阿含經 第 868 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘……如上說。差別者：「彼如是知、如是見，欲有漏心解脫、有有漏心解脫、無明漏心解脫，解脫知見：『我生已盡，梵行已立，所作已作，自知不受後有。』若不解脫，而以彼法，欲法、念法、樂法取中般涅槃；若不爾者，取生般涅槃；若不爾者，取有行般涅槃；若不爾者，取無行般涅槃；若不爾者，取上流般涅槃；若不爾者，彼以欲法、念法、樂法生自性光音天；若不爾者，生無量光天；若不爾者，生少光天。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘……如上說。**
　　那時，世尊告訴在場的眾位比丘說……（內容如前文所述）。

**◈ 差別者：「彼如是知、如是見，欲有漏心解脫、有有漏心解脫、無明漏心解脫，解脫知見：**
　　其中的差異在於：「那個人如此地知道、如此地見地，希望從對感官慾望的染污（欲漏）、對生存存在的染污（有漏）、以及對真理的無知染污（無明漏）中獲得心的解脫，並且獲得了這種解脫的知見：

**◈ 『我生已盡，梵行已立，所作已作，自知不受後有。』**
　　『我的輪迴出生已經結束，清淨的修行（梵行）已經建立，應該完成的修行任務已經全部完成，我自知不再會於後世輪迴出生。』

**◈ 若不解脫，而以彼法，欲法、念法、樂法取中般涅槃；**
　　如果沒有達到完全的解脫，而是依據那種法，透過對慾望的法、對念想的法、對快樂的法，而進入一種中介狀態的涅槃（中般涅槃）；

**◈ 若不爾者，取生般涅槃；**
　　如果不是如此，則進入一種與出生相關的涅槃（生般涅槃）；

**◈ 若不爾者，取有行般涅槃；**
　　如果不是如此，則進入一種具有心行造作的涅槃（有行般涅槃）；

**◈ 若不爾者，取無行般涅槃；**
　　如果不是如此，則進入一種沒有心行造作的涅槃（無行般涅槃）；

**◈ 若不爾者，取上流般涅槃；**
　　如果不是如此，則進入一種高層次的涅槃（上流般涅槃）；

**◈ 若不爾者，彼以欲法、念法、樂法生自性光音天；**
　　如果還是不如此，那個人則依據對慾望的法、對念想的法、對快樂的法，而投生到自性光音天；

**◈ 若不爾者，生無量光天；**
　　如果不是如此，則投生到無量光天；

**◈ 若不爾者，生少光天。」**
　　如果不是如此，則投生到少光天。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **有漏 (Āsava)**：指像漏水一樣不斷流出的染污，使眾生被束縛在輪迴中。分為三類：欲漏（對感官慾望的執著）、有漏（對存在與生存的執著）、無明漏（對四聖諦等真理的無知）。
2. **梵行 (Brahmacariya)**：原指婆羅門的清淨生活，在佛教中指為了追求解脫而實踐的清淨修行生活，包括戒律的持守與心的淨化。
3. **不受後有 (Anupapajjā)**：指徹底斷除輪迴的因緣，在這次生命結束後，不再投生於任何六道世界。
4. **般涅槃 (Nibbāna - provisional)**：此處提到的「中、生、有行、無行、上流」般涅槃，並非指最終的絕對涅槃，而是指在深度禪定中達到的一種暫時的、類似寂滅的心理狀態，或是一種尚未完全斷除漏盡的暫時解脫。
5. **光音天 (Ābhāsvara)**：屬於色界高層的定居天界，其居民擁有自發的光芒，處於極高的禪定狀態。

### [現代實作指南]
本經文揭示了一個核心邏輯：**心靈的最終去向，取決於你「解脫」的程度以及你「執著」的對象。**

1. **識別內心的「漏」 (Leakage Detection)**：
   - **欲漏** ➔ 當你感到強烈渴望某物時，意識到這是一種「漏」，它會讓你心不安寧。
   - **有漏** ➔ 當你過度執著於「我必須成為某種樣子」或「我必須存在」時，意識到這是對生存的執著。
   - **無明漏** ➔ 當你對痛苦的根源感到困惑，或習慣性地以錯誤的認知看待世界時，意識到這是無明。

2. **區分「暫時的寧靜」與「真正的解脫」**：
   - 許多人追求的「心靈平靜」或「正向思考」其實類似經文中的「欲法、念法、樂法」所達到的暫時狀態（如暫時的涅槃或投生高天）。
   - **轉念技巧**：不要滿足於「感覺很好」或「暫時沒壓力」，而要問自己：「我是否已經看穿了這個快樂的無常本質？」

3. **建立「所作已作」的完結感**：
   - 在生活中練習「全然地完成」。當一件事情結束時，在心中默念：「此項任務已完成，我不再對其結果產生執著。」這是在模擬「所作已作」的心理狀態，減少對未來的牽掛（後有）。

### [段落大意]
本段經文說明了修行者根據對「三漏」（欲、有、無明）之解脫程度的不同，將導致截然不同的結果。完全解脫者將終結輪迴，不再投生；而未能完全解脫但具有一定定力者，則會根據其對慾望、念想或快樂的執著程度，進入不同層次的暫時涅槃狀態，或投生至色界的高層天界（如光音天）。
