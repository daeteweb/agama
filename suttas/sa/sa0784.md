# 雜阿含經 第 784 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「有邪、有正。諦聽，善思，當為汝說。何等為邪？謂邪見，乃至邪定。何等為正？謂正見，乃至正定。何等為正見？謂說有施、有說、有齋，有善行、有惡行，有善惡行果報，有此世、有他世，有父母、有眾生生，有阿羅漢善到、善向，有此世、他世自知作證具足住：『我生已盡，梵行已立，所作已作，自知不受後有。』何等為正志？謂出要志、無恚志、不害志。何等為正語？謂離妄語、離兩舌、離惡口、離綺語。何等為正業？謂離殺、盜、婬。何等為正命？謂如法求衣服、飲食、臥具、湯藥，非不如法。何等為正方便？謂欲、精進、方便、出離、勤競、堪能常行不退。何等為正念？謂念隨順，念不妄、不虛。何等為正定？謂住心不亂、堅固、攝持、寂止、三昧、一心。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：「有邪、有正。諦聽，善思，當為汝說。**
　　那時，世尊告訴在場的眾位比丘說：「世間存在著『邪（錯誤的）』與『正（正確的）』兩種方向。請你們仔細聽，好好思考，我現在為你們解說。

**◈ 何等為邪？謂邪見，乃至邪定。**
　　什麼是邪？指的是從錯誤的見解（邪見），一直到錯誤的禪定（邪定）。

**◈ 何等為正？謂正見，乃至正定。**
　　什麼是正？指的是從正確的見解（正見），一直到正確的禪定（正定）。

**◈ 何等為正見？**
　　什麼是正見（正確的見解）？

**◈ 謂說有施、有說、有齋，有善行、有惡行，有善惡行果報，有此世、有他世，有父母、有眾生生，**
　　指的是認同布施、認同正法教說、認同齋戒，認同有善行與惡行之分，並認同善惡行為會產生相應的業報（Vipāka），認同有此生（現世）以及來世（他世），認同父母的存在，以及眾生輪迴出生的事實。

**◈ 有阿羅漢善到、善向，有此世、他世自知作證具足住：『我生已盡，梵行已立，所作已作，自知不受後有。』**
　　認同阿羅漢（已解脫的聖者）確實能達到解脫的境界，且修行者能朝向該目標前進；認同在現世或來世中，有人能自覺地證悟並安住於：「我的輪迴出生已結束，清淨的修行（梵行）已建立，應做的事已完成，自知不再受後世的出生。」

**◈ 何等為正志？謂出要志、無恚志、不害志。**
　　什麼是正志（正確的志向/正思惟）？指的是想要脫離苦海的出離志向、沒有嗔恨的無恚志向、以及不傷害他人的不害志向。

**◈ 何等為正語？謂離妄語、離兩舌、離惡口、離綺語。**
　　什麼是正語（正確的言語）？指的是遠離虛妄的謊言、遠離挑撥離間的兩舌、遠離粗暴的惡口、以及遠離花言巧語的綺語。

**◈ 何等為正業？謂離殺、盜、婬。**
　　什麼是正業（正確的行為）？指的是遠離殺生、遠離偷盜、以及遠離不正當的性行為。

**◈ 何等為正命？謂如法求衣服、飲食、臥具、湯藥，非不如法。**
　　什麼是正命（正確的生活方式）？指的是以符合法義的方式獲取衣服、飲食、臥具與藥品，而非以不正當的手段獲取。

**◈ 何等為正方便？謂欲、精進、方便、出離、勤競、堪能常行不退。**
　　什麼是正方便（正確的努力/正精進）？指的是擁有修行之欲、精進不懈、善用方法、追求出離、勤奮努力，且能恆常實踐而不退轉。

**◈ 何等為正念？謂念隨順，念不妄、不虛。**
　　什麼是正念（正確的覺知）？指的是覺知能隨順法義，且覺知內容不虛妄、不空洞。

**◈ 何等為正定？謂住心不亂、堅固、攝持、寂止、三昧、一心。」**
　　什麼是正定（正確的禪定）？指的是心念安住而不紊亂、堅固不動、能攝持心神、寂靜止息，進入三昧（心之專注）的單一一心狀態。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **業報 (Vipāka)**：指行為（業）所產生的結果。善行帶來快樂的果報，惡行帶來痛苦的果報，這是因果律的體現。
2. **阿羅漢 (Arahant)**：指已斷除一切煩惱、不再輪迴的聖者，達到了完全的解脫。
3. **梵行 (Brahmacariya)**：原意為清淨的生活方式，在解脫脈絡中指為了追求涅槃而實踐的清淨修行。
4. **三昧 (Samādhi)**：指心意識高度集中、不再散亂的定境，是產生智慧的前提。
5. **正方便 (Sammā-vāyāma)**：即八正道中的「正精進」，指正確的努力方向，旨在止惡、除惡、生善、增上。

### [現代實作指南]
本經文詳細定義了「正」的具體操作路徑，可將其轉化為現代生活的「正向生活清單」：

1. **認知校準（正見）**：
   ➔ 練習承認「行為有後果」。當面對選擇時，不採取短視的快感，而是思考該行為在長遠時間線上會帶來什麼樣的業報。
2. **意圖淨化（正志）**：
   ➔ 在產生憤怒或想攻擊他人時，立即將意圖轉向「不害」與「無恚」。問自己：「我的這個念頭是為了解決問題，還是為了宣洩情緒？」
3. **溝通修煉（正語）**：
   ➔ 實踐「真、善、利」原則。在說話前檢查：這句話是否真實？是否溫和？是否對對方有益？若不滿足，則選擇沉默。
4. **生活自律（正業、正命）**：
   ➔ 審視自己的獲利方式。確保職業不建立在傷害他人、欺騙或剝削之上，追求「如法」的生存，即誠實且正義的獲利。
5. **心智訓練（正方便、正念、正定）**：
   ➔ 建立微小的修行習慣。例如：每天抽出5分鐘練習「單一一心」，專注於呼吸，不對雜念做評判，僅僅是覺知其存在（正念），並在面對困難時提醒自己「不退轉」的決心（正方便）。

### [段落大意]
本經由佛陀詳細區分「邪」與「正」的不同，並將「正」具體化為八正道的實踐路徑：從建立正確的因果見解（正見）開始，延伸至純淨的志向（正志）、言語（正語）、行為（正業）、生活方式（正命）、努力方向（正方便）、覺知能力（正念），最終達到心靈的絕對安定（正定）。
