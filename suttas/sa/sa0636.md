# 雜阿含經 第 636 經

### [原文]
如是我聞：

一時，佛住巴連弗邑鷄林精舍。

爾時，世尊告諸比丘：「當為汝說修四念處。何等為修四念處？若比丘！如來、應、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊出興于世，演說正法，上語亦善，中語亦善，下語亦善，善義善味，純一滿淨，梵行顯示。若族姓子、族姓女從佛聞法，得淨信心。如是修學，見在家和合欲樂之過，煩惱結縛，樂居空閑，出家學道，不樂在家，處於非家，欲一向清淨，盡其形壽，純一滿淨，鮮白梵行：『我當剃除鬚髮，著袈裟衣，正信非家，出家學道。』作是思惟已，即便放捨錢財親屬，剃除鬚髮，著袈裟衣，正信非家，出家學道，正其身行，護口四過，正命清淨，習賢聖戒，守諸根門，護心正念。眼見色時，不取形相，若於眼根住不律儀，世間貪憂、惡不善法常漏於心，而今於眼起正律儀；耳、鼻、舌、身、意起正律儀，亦復如是。

「彼以賢聖戒律成就，善攝根門，來往周旋，顧視屈伸，坐臥眠覺語默，住智正智。彼成就如此聖戒，守護根門，正智正念，寂靜遠離，空處、樹下、閑房獨坐，正身正念，繫心安住。斷世貪憂，離貪欲，淨除貪欲；斷世瞋恚、睡眠、掉悔、疑蓋，離瞋恚、睡眠、掉悔、疑蓋，淨除瞋恚、睡眠、掉悔、疑蓋。斷除五蓋惱心，慧力羸、諸障閡分、不趣涅槃者，是故，內身身觀念住，精勤方便，正智正念，調伏世間貪憂；如是外身、內外身，受、心、法法觀念住，亦如是說。是名比丘修四念處。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住巴連弗邑鷄林精舍。**
　　在某一個時間，佛陀住在巴連弗邑的鷄林精舍中。

**◈ 爾時，世尊告諸比丘：「當為汝說修四念處。何等為修四念處？**
　　那時，世尊告訴在場的眾位比丘說：「我現在為你們解說如何修行四念處（對身體、感受、心念、心理現象的四種正念觀察）。什麼叫做修行四念處呢？

**◈ 若比丘！如來、應、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊出興于世，演說正法，**
　　比丘們！當如來（成就不變真理而來到世間的人）、應供、等正覺、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛世尊出現在這個世上，並演說正法（正確的真理教法）時，

**◈ 上語亦善，中語亦善，下語亦善，善義善味，純一滿淨，梵行顯示。**
　　無論是對高層次的人、中層次的人或低層次的人，佛陀的教導都同樣適切且善巧，其義理深奧且易於領悟，內容純淨無雜，親身示範了清淨的梵行（純潔的修行生活）。

**◈ 若族姓子、族姓女從佛聞法，得淨信心。**
　　如果有來自貴族家庭的男子或女子從佛陀那裡聽聞佛法，並產生了純淨的信心。

**◈ 如是修學，見在家和合欲樂之過，煩惱結縛，樂居空閑，出家學道，不樂在家，處於非家，欲一向清淨，盡其形壽，純一滿淨，鮮白梵行：**
　　他們如此修學，便能看見在家生活中追求男女情愛與感官享樂的過失，體悟到那是煩惱（內心的染污與執著）的結縛。因此，他們嚮往清靜空閑的生活，決定出家學道，不再樂於在家中生活，而選擇處於非家（出家）的狀態，希望在餘下的生命中追求絕對的清淨，實踐純一且潔白的梵行。

**◈ 『我當剃除鬚髮，著袈裟衣，正信非家，出家學道。』**
　　他們心想：「我應該剃除鬚髮，穿上袈裟，虔誠地選擇非家生活，出家修行。」

**◈ 作是思惟已，即便放捨錢財親屬，剃除鬚髮，著袈裟衣，正信非家，出家學道，**
　　在這樣思考之後，他們立即捨棄錢財與親屬，剃除鬚髮，穿上袈裟，虔誠地選擇非家生活，正式出家學道。

**◈ 正其身行，護口四過，正命清淨，習賢聖戒，守諸根門，護心正念。**
　　他們端正自己的身體行為，防止口頭上的四種過失（妄語、兩舌、惡口、綺語），維持清淨的正命（正當的生活方式），習以為常地遵守賢聖的戒律，守護六根門（眼、耳、鼻、舌、身、意），並在心中保持正念。

**◈ 眼見色時，不取形相，**
　　當眼睛看見色（物質身體與現象）的時候，不會執著於外在的形相。

**◈ 若於眼根住不律儀，世間貪憂、惡不善法常漏於心，而今於眼起正律儀；**
　　如果眼睛在接觸外界時缺乏律儀（自律與警覺），世間的貪愛與憂慮等不善法會經常從心中滲漏而出；而現在，他們在眼睛的接觸中建立了正律儀。

**◈ 耳、鼻、舌、身、意起正律儀，亦復如是。**
　　對於耳朵、鼻子、舌頭、身體、以及意識的接觸，同樣地建立了正律儀。

**◈ 「彼以賢聖戒律成就，善攝根門，來往周旋，顧視屈伸，坐臥眠覺語默，住智正智。**
　　這樣的人以賢聖的戒律成就自身，善於攝受根門。無論是在行走往來、顧盼視察、屈身伸展，或是坐、臥、睡眠、覺醒、說話、沉默的每一個瞬間，都處於正智（正確的覺知）之中。

**◈ 彼成就如此聖戒，守護根門，正智正念，寂靜遠離，空處、樹下、閑房獨坐，正身正念，繫心安住。**
　　他們成就了這樣的聖戒，守護根門，具備正智與正念，在寂靜遠離喧囂的地方，如空曠之處、樹下或閑靜的房間獨自坐著，端正身體，保持正念，使心專注安住。

**◈ 斷世貪憂，離貪欲，淨除貪欲；**
　　他們斷除世間的貪愛與憂慮，遠離貪欲，並將貪欲徹底淨除。

**◈ 斷世瞋恚、睡眠、掉悔、疑蓋，離瞋恚、睡眠、掉悔、疑蓋，淨除瞋恚、睡眠、掉悔、疑蓋。**
　　他們斷除世間的瞋恚（憤怒）、睡眠（昏沉）、掉悔（掉舉與悔恨）、以及疑蓋（疑惑的遮蔽）；遠離並淨除這些瞋恚、睡眠、掉悔與疑蓋。

**◈ 斷除五蓋惱心，慧力羸、諸障閡分、不趣涅槃者，**
　　他們斷除了五蓋（貪欲、瞋恚、睡眠、掉悔、疑蓋）這五種惱心，克服了智慧力不足的問題，除去了所有阻礙前進的因素，不再受困於不趨向涅槃（超越生死、絕對寧靜的解脫境界）的狀態。

**◈ 是故，內身身觀念住，精勤方便，正智正念，調伏世間貪憂；**
　　因此，他們對內在的身體（內身）進行觀照並安住其中，以精勤的手段與正智正念，調伏世間的貪愛與憂慮。

**◈ 如是外身、內外身，受、心、法法觀念住，亦如是說。是名比丘修四念處。」**
　　同樣地，對於外在的身體（外身）、內外身體，以及受（感受）、心（意識狀態）、法（心理現象與真理）進行觀照並安住其中，也是如此。比丘們，這就叫做修行四念處。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **四念處 (Satipaṭṭhāna)**：佛教修行中建立正念的四個基礎，分別是：身念處（觀察身體）、受念處（觀察感受）、心念處（觀察心念）、法念處（觀察心理現象與真理）。
2. **五蓋 (Pañca Nīvaraṇā)**：遮蔽心靈、阻礙禪定與智慧的五種障礙，包括：貪欲 (Kāmacchanda)、瞋恚 (Vyāpāda)、睡眠/昏沉 (Thīna-middha)、掉悔/躁動 (Uddhacca-kukkucca)、以及懷疑 (Vicikicchā)。
3. **根門 (Indriya)**：指感官的入口，包括眼、耳、鼻、舌、身、意。守護根門是指在感官接觸外界時，不讓心隨之起貪或瞋。
4. **律儀 (Saja-sampajañña/Samyama)**：指在行為與心念上保持自律、警覺，不隨意地被欲望或情緒牽引。
5. **梵行 (Brahmacariya)**：原意為「如梵天般的生活」，指清淨、無染、遠離感官欲望的修行生活。

### [現代實作指南]
本經文提供了一套從「認知」到「行動」再到「深層心靈淨化」的完整路徑，現代人可將其轉化為以下實踐步驟：

1. **覺察欲望的代價 (認知階段)**：
   - 練習觀察生活中追求「快感」或「物質滿足」後的空虛感。意識到過度的執著（如對名聲、感情的強烈渴求）往往帶來的是不安與憂慮（煩惱結縛），而非真正的快樂。

2. **建立感官的「防火牆」 (律儀實踐)**：
   - **視覺律儀**：當在社交媒體看到令人羨慕或憤怒的資訊時，練習「不取形相」，即意識到這只是一個影像，不讓心隨之產生「我也要」或「他怎麼這樣」的強烈反應。
   - **聽覺/語言律儀**：在說話前停頓一秒，檢查是否包含「口四過」（謊言、挑撥、惡言、花言巧語），將對話轉向真實且有益的方向。

3. **將正念融入微小動作 (生活化修行)**：
   - 不要將修行僅限於靜坐。在行走、伸展、甚至喝水、打字等日常動作中，練習「住智正智」➔ 意識到「我現在正在行走」、「我現在正在呼吸」，將心從對未來的焦慮或對過去的悔恨中拉回當下。

4. **清理心靈的五大垃圾 (心理除垢)**：
   - 當情緒升起時，嘗試給它貼標籤：
     - 想要某物 ➔ 「這是貪欲」
     - 感到憤怒 ➔ 「這是瞋恚」
     - 精神渙散/疲憊 ➔ 「這是睡眠/昏沉」
     - 心神不寧/自責 ➔ 「這是掉悔」
     - 對目標猶豫不決 ➔ 「這是疑蓋」
   - 僅僅是「標記」而不「認同」，即可削弱這些障礙對心靈的控制力。

5. **由內而外的觀照 (四念處實作)**：
   - **身**：感受呼吸的起伏或身體的緊繃感。
   - **受**：覺察當下的感受是舒服、不舒服，還是中性的。
   - **心**：觀察心現在是躁動的還是平靜的。
   - **法**：思考這個情緒是如何產生的（緣起），以及它是否無常。

### [段落大意]
本經詳細描述了一位修行者從聽聞佛法、產生信心，到決定出家並實踐清淨生活的完整過程。重點在於透過「守護根門」與「正律儀」來防止煩惱滲漏，並在日常生活的每個動作中保持正智。最終透過斷除五蓋，在寂靜中實踐四念處（身、受、心、法）的觀照，以調伏貪憂，趨向涅槃的解脫境界。
