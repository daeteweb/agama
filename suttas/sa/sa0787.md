# 雜阿含經 第 787 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「向邪者違於法，不樂於法，向正者樂於法，不違於法。何等為向邪者違於法，不樂於法？謂邪見人身業如所見，口業如所見，若思、若欲、若願、若為，彼皆隨順，一切得不愛果，不念、不可意果。所以者何？以見惡故，謂邪見。邪見者，起邪志、邪語、邪業、邪命、邪方便、邪念、邪定，是向邪者違於法，不樂於法。

「何等為向正者樂於法，不違於法？謂正見人若身業隨所見，若口業、若思、若欲、若願、若為，悉皆隨順，得可愛、可念、可意果。所以者何？以見正故？謂正見。正見者，能起正志、正語、正業、正命、正方便、正念、正定，是名向正者樂於法，不違於法。」

佛說是經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的。

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「向邪者違於法，不樂於法，向正者樂於法，不違於法。**
　　「傾向於邪見（錯誤認知）的人違背法（真理），對法不感到快樂；而傾向於正見（正確認知）的人則樂於法，不違背法。

**◈ 何等為向邪者違於法，不樂於法？**
　　什麼樣的情況叫做傾向於邪見而違背法、對法不感到快樂呢？

**◈ 謂邪見人身業如所見，口業如所見，若思、若欲、若願、若為，彼皆隨順，一切得不愛果，不念、不可意果。**
　　也就是指持有邪見（錯誤認知）的人，其身業（身體的行為）與口業（言語的行為）都隨其錯誤的見解而行；無論是思考、欲望、願望或實際行動，全部都隨順於這種錯誤的認知，因此，他們所獲得的所有業報（行為的結果）都是不令人喜愛的，是不值得記憶、不令人滿意的結果。

**◈ 所以者何？以見惡故，謂邪見。**
　　為什麼會這樣呢？因為他們的見解是惡劣的，這就稱為邪見。

**◈ 邪見者，起邪志、邪語、邪業、邪命、邪方便、邪念、邪定，是向邪者違於法，不樂於法。**
　　持有邪見的人，會進而產生錯誤的志向（邪志）、錯誤的言語（邪語）、錯誤的行為（邪業）、錯誤的生活方式（邪命）、錯誤的修行方法（邪方便）、錯誤的覺知（邪念）以及錯誤的專注（邪定），這就是所謂的傾向於邪見而違背法、對法不感到快樂。

**◈ 「何等為向正者樂於法，不違於法？**
　　那麼，什麼樣的情況叫做傾向於正見而樂於法、不違背法呢？

**◈ 謂正見人若身業隨所見，若口業、若思、若欲、若願、若為，悉皆隨順，得可愛、可念、可意果。**
　　也就是指持有正見（正確認知）的人，其身業（身體的行為）隨其正確的見解而行，口業（言語的行為）、思考、欲望、願望或實際行動，全部都隨順於這種正確的認知，因此，他們能獲得令人喜愛、值得記憶且令人滿意的業報（行為的結果）。

**◈ 所以者何？以見正故？謂正見。**
　　為什麼會這樣呢？因為他們的見解是正確的，這就稱為正見。

**◈ 正見者，能起正志、正語、正業、正命、正方便、正念、正定，是名向正者樂於法，不違於法。」**
　　持有正見的人，能進而產生正確的志向（正思惟）、正確的言語（正語）、正確的行為（正業）、正確的生活方式（正命）、正確的修行方法（正精進）、正確的覺知（正念）以及正確的專注（正定），這就稱為傾向於正見而樂於法、不違背法。」

**◈ 佛說是經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **正見 (Sammā-diṭṭhi)**：八正道之首，指對四聖諦（苦、集、滅、道）以及因果律的正確認知，是所有修行正確方向的基石。
2. **邪見 (Micchā-diṭṭhi)**：對真理的錯誤認知，例如否認因果報應、認為行為沒有結果，或對世界的本質產生誤解。
3. **身業/口業 (Kamma)**：業指「造作」。身業是指透過身體行動產生的造作，口業是指透過言語產生的造作。
4. **業報 (Vipāka)**：行為之後所產生的結果。正向的造作帶來正向的果報，反之亦然。
5. **正方便 (Sammā-vāyāma)**：即「正精進」，指努力防止未生之惡法生起，斷除已生之惡法，並令未生之善法生起，維持已生之善法。

### [現代實作指南]
本經文揭示了「認知 ➔ 行為 ➔ 結果」的心理連鎖反應。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **審視底層認知（檢查「見」）**：
   當我們面臨衝突或壓力時，先問自己：「我現在對這件事的看法是基於事實，還是基於偏見（邪見）？」
   ➔ 例如：將「對方故意針對我」視為預設（邪見），會導致憤怒的行為；將「對方可能正處於壓力中」視為可能性（正見），則會導向理解的行為。

2. **覺察行為的隨順性（觀察「業」）**：
   觀察自己的言語（口業）和行動（身業）是否在隨順某種負面情緒或錯誤認知。
   ➔ 當發現自己想說尖酸刻薄的話時，意識到這是「邪見」在驅動「口業」。

3. **正向干預（建立「正方便」）**：
   刻意練習將認知轉向「正見」。
   ➔ 練習將注意力從「我想要什麼（貪愛）」轉向「這件事的因果是什麼」。當認知改變，隨之而來的志向（正志）與行動（正業）會自然轉化。

4. **驗證結果（體會「果」）**：
   記錄在採取「正見」行動後，內心的平靜程度與人際關係的改善情況，透過正向回饋來強化正見的習慣。

### [段落大意]
本經闡述了「見解」如何決定一個人的生命品質。持有邪見者，其身口意之行為皆隨順錯誤認知，最終導致不滿意的業報，並陷入錯誤的八道循環；而持有正見者，其行為隨順真理，能獲得滿意的結果，並能建立起完整的八正道修行路徑。
