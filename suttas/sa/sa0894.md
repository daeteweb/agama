# 雜阿含經 第 894 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「於我世間，於世間及世間集不如是知者，我終不得於諸天、魔、梵、沙門、婆羅門及諸世間，為解脫、為出、為離，離顛倒想，亦不名阿耨多羅三藐三菩提。以我於世間及世間集如實知故，是故我於諸天、世人、魔、梵、沙門、婆羅門及餘眾生，為得解脫、為出、為離，心離顛倒，具足住，得成阿耨多羅三藐三菩提。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

如是世間世間集、世間滅，世間集、世間出，世間集、世間滅、世間味、世間患、世間出，世間集、世間滅、世間出，世間集、世間滅道跡，世間集、世間滅道跡、世間滅道跡，世間集、世間滅、世間味、世間患、世間出，世間集、世間滅、世間集道跡、世間滅道跡、世間味、世間患、世間出。

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「於我世間，於世間及世間集不如是知者，**
　　「如果我對於『世間』（指受苦的眾生與環境）以及『世間集』（指導致世間苦因的貪愛與執著）沒有如此正確的認知，

**◈ 我終不得於諸天、魔、梵、沙門、婆羅門及諸世間，為解脫、為出、為離，離顛倒想，亦不名阿耨多羅三藐三菩提。**
　　我最終無法在諸天、魔、梵、沙門、婆羅門以及所有世間眾生之中，獲得解脫（從束縛中釋放）、出離（脫離輪迴）與遠離，無法擺脫顛倒的錯誤認知，也無法被稱為獲得最高、最圓滿的智慧與覺悟 (阿耨多羅三藐三菩提)。

**◈ 以我於世間及世間集如實知故，**
　　正是因為我對世間以及世間集（苦因）有如實的正確認知，

**◈ 是故我於諸天、世人、魔、梵、沙門、婆羅門及餘眾生，為得解脫、為出、為離，心離顛倒，具足住，得成阿耨多羅三藐三菩提。」**
　　所以我在諸天、世人、魔、梵、沙門、婆羅門以及其他眾生之中，能夠獲得解脫、出離與遠離，內心脫離了顛倒的認知，安住在圓滿的狀態中，成就了最高、最圓滿的智慧與覺悟 (阿耨多羅三藐三菩提)。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀講完這部經後，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

**◈ 如是世間世間集、世間滅，世間集、世間出，世間集、世間滅、世間味、世間患、世間出，世間集、世間滅、世間出，世間集、世間滅道跡，世間集、世間滅道跡、世間滅道跡，世間集、世間滅、世間味、世間患、世間出，世間集、世間滅、世間集道跡、世間滅道跡、世間味、世間患、世間出。**
　　就這樣，佛陀詳細地說明了：世間（苦）、世間集（苦因）、世間滅（苦止）；世間集（苦因）、世間出（出離）；世間集（苦因）、世間滅（苦止）、世間味（世俗的誘惑）、世間患（世俗的危險）、世間出（出離）；世間集（苦因）、世間滅（苦止）、世間出（出離）；世間集（苦因）、世間滅（苦止）的道跡（修行路徑）；世間集（苦因）、世間滅（苦止）的道跡（修行路徑）、世間滅（苦止）的道跡（修行路徑）；世間集（苦因）、世間滅（苦止）、世間味（世俗的誘惑）、世間患（世俗的危險）、世間出（出離）；世間集（苦因）、世間滅（苦止）、世間集（苦因）的道跡（修行路徑）、世間滅（苦止）的道跡（修行路徑）、世間味（世俗的誘惑）、世間患（世俗的危險）、世間出（出離）。

### [詞語註解（解釋重要佛教概念）]
1. **世間 (Loka)**：在原始佛教中，世間不僅指物理空間，更指由五蘊（色、受、想、行、識）構成的經驗世界，以及在其中受苦、輪迴的眾生狀態。
2. **世間集 (Loka-samudaya)**：對應四聖諦中的「集諦」，指導致世間苦難的根本原因，主要是指無明（對真理的不認知）與貪愛（對感官快感的渴求）。
3. **世間滅 (Loka-nirodha)**：對應四聖諦中的「滅諦」，指苦因被完全消除後的狀態，即涅槃 (Nibbāna)。
4. **世間味 (Loka-rasa)**：指世間感官所能體驗到的快感、名聲、權力等吸引力，是導致眾生執著於輪迴的誘因。
5. **世間患 (Loka-ādīnava)**：指世間本質上的缺陷與危險，如生老病死、離別、不得所欲等必然的苦痛。
6. **道跡 (Magga-paṭhipada)**：指通往滅苦的具體路徑，即八正道。

### [現代實作指南]
本經的核心在於「正確的認知」是解脫的前提。在現代生活中，我們可以將其轉化為以下實踐步驟：

1. **辨識「世間味」與「世間患」** ➔ 當我們追求某項物質或成就（如升職、名聲、消費）時，意識到這就是「世間味」。隨即提醒自己，這些快感伴隨著維持它的壓力、失去它的恐懼以及最終的無常，這就是「世間患」。
2. **分析「世間集」的運作** ➔ 當心中產生強烈渴望時，觀察這個渴望是如何從「無明」（認為這能帶來永恆快樂）➔ 「愛」（強烈想要）➔ 「取」（執著追求）而產生的。
3. **練習「離顛倒想」** ➔ 顛倒想是指將「無常」看作「常」、「苦」看作「樂」、「無我」看作「我」。實作方法是：在面對挫折時，告訴自己「這現象本來就是無常的」，將對結果的執著轉化為對過程的覺察。
4. **建立「道跡」的習慣** ➔ 每天撥出短時間練習正念（正念），觀察身體的感受（受）與心靈的造作（行），不對其做價值判斷，僅僅是如實地知道，這便是走在「世間滅道跡」上的初步實踐。

### [段落大意]
佛陀強調，若不能如實認知世間的本質及其苦因（世間集），就無法在任何生命層次中獲得真正的解脫與最高覺悟。佛陀透過對世間的苦、因、滅、道，以及世間的誘惑（味）與危險（患）的詳細分析，指明了從顛倒認知走向覺悟的必然路徑。
