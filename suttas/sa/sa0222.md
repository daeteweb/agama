# 雜阿含經 第 222 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「當知一切知法、一切識法。諦聽，善思，當為汝說。

「云何一切知法、一切識法？諸比丘！眼是知法、識法，若色、眼識、眼觸、眼觸因緣生受，內覺若苦、若樂、不苦不樂，彼一切是知法、識法。耳、鼻、舌、身、意亦復如是。」

佛說此經已，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽聞的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　有一次，佛住在舍衛國的祇樹給孤獨園中。

**◈ 爾時，世尊告諸比丘：「當知一切知法、一切識法。諦聽，善思，當為汝說。**
　　那時候，世尊告訴所有的比丘：「你們應當了解如何去知曉一切法、辨識一切法。請仔細聆聽並深入思考，我將為你們說明。」

**◈ 「云何一切知法、一切識法？諸比丘！眼是知法、識法，若色、眼識、眼觸、眼觸因緣生受，內覺若苦、若樂、不苦不樂，彼一切是知法、識法。耳、鼻、舌、身、意亦復如是。」**
　　「如何才算是知曉一切法、辨識一切法呢？比丘們！眼睛就是知法與識法的方式；例如當色（物質對象）、眼識（視覺意識）與眼觸（視覺接觸）發生，並由這些接觸因緣而產生的受（感受），無論內在覺察到的是苦的、樂的、或是非苦非樂的感受，這一切過程都是在知曉與辨識法。耳朵、鼻子、舌頭、身體以及意念，也是同樣的道理。」

**◈ 佛說此經已，諸比丘聞佛所說，歡喜奉行。**
　　佛陀說完這部經後，所有的比丘聽聞了佛陀的教導，皆感到歡喜並遵從修行。

### [詞語註解（解釋重要佛教概念）]
1. **知法、識法 (Vijānāti Dhamma)**：指透過感官與對象的接觸，去覺察與辨識現象（法）是如何運作、如何生起的過程。
2. **色 (Rūpa)**：指物質性的存在或感官所接觸的外在對象（如顏色、聲音、氣味等）。
3. **觸 (Phassa)**：指感官、對象與意識三者結合時產生的「接觸」作用，是感受產生的前提。
4. **受 (Vedanā)**：指在接觸之後，內心所產生的即時「感受」，分為苦（不適）、樂（舒適）或不苦不樂（中性）。

### [現代實作指南]
本經教導我們透過觀察「感官運作的連鎖反應」來理解世界的真相。你可以嘗試以下練習：

1. **拆解感官鏈條**：當你感到情緒波動時（例如看到一張令你不悅的照片），不要直接陷入情緒，而是將過程拆解為：
   → 看到對象（色）
   → 視覺意識產生（眼識）
   → 目光與影像接觸（觸）
   → 隨即產生的不舒服感（受）
2. **觀察「受」的無常性**：練習在感受出現時，僅僅將其標記為「苦受」、「樂受」或「不苦不樂受」。透過這種方式，你會發現所謂的「我」，其實只是這一連串因緣（色、識、觸、受）不斷流動的過程。
3. **去中心化練習**：當你意識到「這一切都是在知法與識法」時，你就不會再把感受誤認為是「我的情緒」，從而獲得心理上的空間與解脫感。

### [段落大意]
佛陀說明感官（眼耳鼻舌身意）與其對象、意識及接觸後產生的感受，即是「知法」與「識法」的過程。透過觀察這些因緣生起的感受，比丘能理解現象的運作規律。
