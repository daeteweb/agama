# 雜阿含經 第 13 經

### [原文]
如是我聞：

一時，佛住舍衛國祇樹給孤獨園。

爾時，世尊告諸比丘：「若眾生於色不味者，則不染於色；以眾生於色味故，則有染著。如是眾生於受、想、行、識不味者，彼眾生則不染於識；以眾生味受、想、行、識故，彼眾生染著於識。

「諸比丘！若色於眾生不為患者，彼諸眾生不應厭色，以色為眾生患故，彼諸眾生則厭於色。如是受、想、行、識不為患者，彼諸眾生不應厭識；以受、想、行、識為眾生患故，彼諸眾生則厭於識。

「諸比丘！若色於眾生無出離者，彼諸眾生不應出離於色；以色於眾生有出離故，彼諸眾生出離於色。如是受、想、行、識於眾生無出離者，彼諸眾生不應出離於識。以受、想、行、識於眾生有出離故，彼諸眾生出離於識。

「諸比丘！若我於此五受陰不如實知味是味、患是患、離是離者，我於諸天、若魔、若梵、沙門、婆羅門、天、人眾中，不脫、不出、不離，永住顛倒，亦不能自證得阿耨多羅三藐三菩提。

「諸比丘！我以如實知此五受陰味是味、患是患、離是離故，我於諸天、若魔、若梵、沙門、婆羅門、天、人眾中，自證得脫、得出、得離、得解脫結縛，永不住顛倒，亦能自證得阿耨多羅三藐三菩提。」

時，諸比丘聞佛所說，歡喜奉行。

### [白話翻譯]
**◈ 如是我聞：**
　　我是這樣聽佛陀說的：

**◈ 一時，佛住舍衛國祇樹給孤獨園。**
　　在某一個時間，佛陀住在舍衛國的祇樹給孤獨園。

**◈ 爾時，世尊告諸比丘：**
　　那時，世尊告訴在場的眾位比丘說：

**◈ 「若眾生於色不味者，則不染於色；以眾生於色味故，則有染著。**
　　「如果眾生對於色（物質身體）不感到吸引（不味），就不會被色（物質身體）所染污；正因為眾生覺得色（物質身體）有吸引力，所以才會產生染著。

**◈ 如是眾生於受、想、行、識不味者，彼眾生則不染於識；以眾生味受、想、行、識故，彼眾生染著於識。**
　　同樣地，如果眾生對於受（感受）、想（想像與認知）、行（心的造作）、識（意識）不感到吸引，就不會被識（意識）所染污；正因為眾生覺得受（感受）、想（想像與認知）、行（心的造作）、識（意識）有吸引力，所以才會染著於識（意識）。

**◈ 「諸比丘！若色於眾生不為患者，彼諸眾生不應厭色，以色為眾生患故，彼諸眾生則厭於色。**
　　「諸比丘！如果色（物質身體）對眾生而言不是一種苦患，眾生就不應該厭離色（物質身體）；正因為色（物質身體）對眾生而言是苦患，所以眾生才會厭離色（物質身體）。

**◈ 如是受、想、行、識不為患者，彼諸眾生不應厭識；以受、想、行、識為眾生患故，彼諸眾生則厭於識。**
　　同樣地，如果受（感受）、想（想像與認知）、行（心的造作）、識（意識）對眾生而言不是苦患，眾生就不應該厭離識（意識）；正因為受（感受）、想（想像與認知）、行（心的造作）、識（意識）對眾生而言是苦患，所以眾生才會厭離識（意識）。

**◈ 「諸比丘！若色於眾生無出離者，彼諸眾生不應出離於色；以色於眾生有出離故，彼諸眾生出離於色。**
　　「諸比丘！如果色（物質身體）對眾生而言沒有出離（解脫）的可能，眾生就不應該試圖出離色（物質身體）；正因為色（物質身體）對眾生而言是有出離之道的，所以眾生才能出離色（物質身體）。

**◈ 如是受、想、行、識於眾生無出離者，彼諸眾生不應出離於識。以受、想、行、識於眾生有出離故，彼諸眾生出離於識。**
　　同樣地，如果受（感受）、想（想像與認知）、行（心的造作）、識（意識）對眾生而言沒有出離（解脫）的可能，眾生就不應該試圖出離識（意識）；正因為受（感受）、想（想像與認知）、行（心的造作）、識（意識）對眾生而言是有出離之道的，所以眾生才能出離識（意識）。

**◈ 「諸比丘！若我於此五受陰不如實知味是味、患是患、離是離者，**
　　「諸比丘！如果我對於這五受陰（構成身心的五種要素）不能如實地知道什麼是吸引力（味）、什麼是苦患（患）、什麼是出離（離）的話，

**◈ 我於諸天、若魔、若梵、沙門、婆羅門、天、人眾中，不脫、不出、不離，永住顛倒，亦不能自證得阿耨多羅三藐三菩提。**
　　我在諸天、魔王、梵天、沙門、婆羅門以及天人眾之中，將無法脫離、無法超出、無法離棄，永遠處在顛倒（錯誤認知）的狀態中，也無法自證獲得最高、最圓滿的智慧與覺悟 (阿耨多羅三藐三菩提)。

**◈ 「諸比丘！我以如實知此五受陰味是味、患是患、離是離故，**
　　「諸比丘！正因為我能如實地知道這五受陰（構成身心的五種要素）的吸引力是吸引力、苦患是苦患、出離是出離，

**◈ 我於諸天、若魔、若梵、沙門、婆羅門、天、人眾中，自證得脫、得出、得離、得解脫結縛，永不住顛倒，亦能自證得阿耨多羅三藐三菩提。」**
　　我在諸天、魔王、梵天、沙門、婆羅門以及天人眾之中，才能自證獲得脫離、超出、離棄，並獲得解脫結縛，不再處於顛倒（錯誤認知）的狀態，且能自證獲得最高、最圓滿的智慧與覺悟 (阿耨多羅三藐三菩提)。」

**◈ 時，諸比丘聞佛所說，歡喜奉行。**
　　那時，眾比丘聽聞佛陀所說的教法，心生法喜並依照教導去實踐。

### [詞語註解（解釋重要佛教概念）]
1. **味 (Rasa)**：指事物呈現出的吸引力或快感，是導致眾生產生貪愛與染著的直接原因。
2. **患 (Ādīnava)**：指事物內在的缺陷、危險或必然導致的苦受。在五蘊中，指其無常、不可掌控且帶來痛苦的本質。
3. **離 (Nissaraṇa)**：指從染著中解脫的途徑或狀態，即透過覺知事物的「患」而產生的厭離心，進而達成出離。
4. **五受陰 (Pañcupādānakkhandhā)**：即五蘊（色、受、想、行、識）在被執取（受）之後的狀態，指眾生將這五種要素誤認為「我」而產生的執著。
5. **顛倒 (Viparyāsa)**：指對現實的錯誤認知，例如將無常視為常恆、將苦視為樂、將無我視為我。

### [現代實作指南]
本經文提供了一套認知轉化模型，可用於處理現代生活中的執著與情緒困擾：

1. **識別「味」（吸引力分析）**：
   當你對某件事物（如名聲、物質、特定關係）產生強烈渴望時，先意識到這是一種「味」。承認它有吸引力，但不要立即陷入其中。
   ➔ 實作：問自己「我現在被這個事物的什麼特質吸引了？」

2. **洞察「患」（風險分析）**：
   在享受吸引力的同時，理性分析其必然帶來的副作用或無常性。例如，追求名聲的「味」是成就感，但其「患」是壓力、對評價的恐懼以及名聲消失時的失落。
   ➔ 實作：問自己「如果我執著於此，未來會帶來什麼樣的痛苦或束縛？」

3. **尋找「離」（解脫路徑）**：
   當「患」大於「味」時，自然會產生一種不願被束縛的心理。這不是強迫的厭惡，而是基於事實的「不再需要」。
   ➔ 實作：練習將注意力從「對象的吸引力」轉移到「觀察吸引力產生的過程」，從而獲得心理空間。

4. **對抗「顛倒」**：
   提醒自己，身體的衰老、情緒的起伏、認知的偏差（五蘊）都不是永恆的自我。透過不斷重複「味 ➔ 患 ➔ 離」的觀察，打破將暫時快感視為永恆幸福的顛倒認知。

### [段落大意]
本經闡述了對五蘊（色、受、想、行、識）認知的三個階段：首先是感知其吸引力（味）導致染著，其次是體悟其內在缺陷（患）導致厭離，最後是發現出離之道（離）達成解脫。佛陀強調，唯有如實知曉這三個面向，才能打破顛倒認知，從各種生命狀態中解脫，最終證得最高圓滿的覺悟。
